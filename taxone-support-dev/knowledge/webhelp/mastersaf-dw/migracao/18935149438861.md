## Passo-a-Passo

1\. Erro

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/18935063000077)

2\. O programa em java que roda a atualização das TFIX espera um padrão no TNSnames, se estiver fora desse padrão acaba não reconhecendo corretamente os dados que estão sendo passados e não conecta com o banco.

Exemplo de um TNS incorreto.

(HOST=HOST.TESTE.COM) (PORT=1521)) (CONNECT_DATA= (SERVER=DEDICATED) (SERVICE_NAME=MSAF) (FAILOVER_MODE= (TYPE=SELECT) (METHOD=BASIC))))

3\. No TNSnames é necessário colocar um espaço entre os sinais de iguais (=), um espaço a frente e outro atrás conforme exemplo abaixo.

Exemplo do TNS correto:

(HOST = HOST.TESTE.COM) (PORT = 1521)) (CONNECT_DATA = (SERVER = DEDICATED) (SERVICE_NAME = MSAF) (FAILOVER_MODE = (TYPE = SELECT) (METHOD = BASIC))))
