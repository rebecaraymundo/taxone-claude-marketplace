Introdução

"Esse artigo tem por objetivo mostrar os passos para a correção do erro:

_
NOK - NonParserException: cvc-complex-type.2.4.a: Invalid content was found starting with element '{"http://www.reinf.esocial.gov.br/schemas/evt4020PagtoBeneficiarioPJ/v2_01_02":**endExt**}'. One of '{"http://www.reinf.esocial.gov.br/schemas/evt4020PagtoBeneficiarioPJ/v2_01_02":**frmTribut**}' is expected. - null

_Deve ser verificado se o campo 'Forma de Tributação' no respectivo 'Controle de Tributos' (Safx53) está preenchido.
