# Introdução

Este artigo tem por objetivo demonstrar com corrigir o erro de conexão para novos usuários cadastrados no powerlock.

## Passo-a-Passo

1\. Erro: Após criar um novo usuário no PowerLock e exibir a mensagem no primeiro login:

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414727973005/mceclip0.png)

2\. Certifique-se que o usuário criado está vinculado a uma CONEXÃO no meu Profile.

3\. Após realizar o vinculo das conexões, o login poderá ser feito normalmente. ![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414735329933/mceclip1.png)
