## Introdução

O objetivo deste artigo é demonstrar a resolução do erro 90403: "O campo Data Inventário está fora do período solicitado".

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10998313086989)

## Passo a passo

Este erro ocorre quando o campo Data de Inventário (DATA_INVENTARIO) não compreende o período parametrizado na programação de importação no Job Servidor. Para corrigir o problema, basta informar uma data período que compreenda tal período.

a Verifique se a data nestá dentro do período parametrizado na importação da SAFX52.
