# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 90385: O Campo Codigo de Pais nao esta cadastrado

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414652528653/mceclip0.png)

2\. O problema ocorre pois o código do país está incorreto no arquivo ou não está cadastrado no menu BÁSICOS > DATAWAREHOUSE > Manutenção > Cadastros > País

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414652536461/mceclip1.png)

3\. No exemplo acima o COD_PAIS estava incorreto, para solucionar o problema basta alterar para o código correto.

Os cadastros de COD_PAIS são importados através da TACES05, a Thomson Reuters possui alguns códigos nela já para que o sistema não inicie sem dados, porém a manutenção desses dados são de responsabilidade do cliente.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414668005517/mceclip2.png)
