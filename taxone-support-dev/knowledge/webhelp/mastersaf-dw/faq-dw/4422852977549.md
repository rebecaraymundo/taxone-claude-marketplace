# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 90001: Codigo de Empresa nao esta preenchido

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4422831551501)

2\. O erro ocorre devido ao campo COD_EMPRESA estar com valor @ ou em branco

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4422861213453)

3\. Para solucionar o problema basta informar a empresa correspondente aos dados que está importando.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4422831578765)
