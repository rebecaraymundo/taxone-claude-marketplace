## Introdução

Este artigo tem como objetivo demonstrar os procedimentos para cadastro de diretórios mapeados no servidor de banco de dados, a serem utilizados em módulos, para salvar arquivos gerados.

## Passo-a-passo

1\. Mapeamento da pasta no servidor de banco de dados

O mapeamento da pasta deve ser previamente criado pelo DBA responsável pela base de dados do cliente.

Importante que a pasta mapeada seja acessível para o usuário que irá utilizar o recurso e também que o banco de dados possua permissão para salvar arquivos nesta pasta.

Ainda, deve-se atentar para que haja espaço suficiente na unidade de disco da pasta, para salvar os arquivos gerados.

Também é importante que após o usuário realizar a cópia e utilização do arquivo, o arquivo seja eliminado da pasta do servidor, a fim de evitar problemas com esgotamento de espaço na unidade.
