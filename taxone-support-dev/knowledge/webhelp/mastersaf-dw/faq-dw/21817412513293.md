## PROBLEMA

Ao acessar o DW, não é possível encontrar o estabelecimento para gerar o relatório Retenções de ISSQN, este cenário é único quando a empresa é sua própria centralizadora.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21817399729165)

## SOLUÇÃO PROPOSTA

1\. Opção é acrescentar essa empresa via banco de dados.

Para que este estabelecimento fique disponível no relatório, realizar um UPDATE


> ( SELECT * FROM PRT_CENTRAL_ESTAB FOR UPDATE)


Com o comando 'FOR UPDATE' executado, eu inclua o estabelecimento no relatório via banco de dados:
![](/attachments/token/CJtUSzv4RBppqlXArObvCxtzo/?name=image.png)

*O ultimo campo precisa de algum caractere, podendo ser apenas um "espaço".

Após o UPDATE o estabelecimento será exibido dentro do Relatório.

![](/attachments/token/CLXH8WiWqbe6OTEKjTrl2AnCx/?name=image.png)

**Para este procedimento, recomendamos a abertura de um ticket na Thomson Reuters.**
