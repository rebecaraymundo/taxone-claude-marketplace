## Introdução

O objetivo deste artigo é demonstrar a resolução do erro 92840: O Código Especificador da Situação Tributária informado - CEST Não Encontrado.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10981983710093)

## Passo a passo

Este erro ocorre pela falta de dados na tabela PRT_CLASS_CEST que é alimentada pela TACES94. Para sanar, basta realizar a importação da TACES94.
