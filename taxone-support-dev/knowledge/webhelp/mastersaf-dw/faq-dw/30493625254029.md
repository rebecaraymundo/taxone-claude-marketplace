## Problema:

O usuário deseja saber quais funcionalidades o Powerlock oferece ao se conectar com o Active Directory via LDAP.

## Causa Raiz:

A dúvida surgiu devido à implementação da conexão LDAP no Powerlock e a necessidade de entender como essa integração afeta as operações diárias relacionadas à gestão de usuários e acesso a sistemas.



## Resolução:

Após consulta com o time técnico, confirmamos que o Powerlock, ao se conectar com o Active Directory via LDAP, é capaz de realizar as seguintes tarefas:

  * Validação de usuário e senha.


  * Gestão de perfis, determinando se o usuário tem ou não acesso ao sistema Mastersaf.


  * Associação do usuário a grupos específicos.
