Este artigo tem por objetivo demonstrar como corrigir o erro ORA-04036: PGA memory used by the instance exceeds PGA_AGGREGATE_LIMIT

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/6264335511693/mceclip0.png)

2\. O problema ocorre pois algumas variáveis do banco de dados podem estar divergentes do recomendado pela Thomson Reuters.

3\. Para solucionar o problema solicite ao DBA realizar a alteração do memory_target do seu banco para Memory_target=10G e realizar o seguinte script:

alter system set pga_aggregate_limit=0 scope=both;
