# Geração do Registro I250 para o Livro A e R no DW.

## Problema:

O usuário enfrenta dificuldades na geração do registro I250 para o Livro A no DW, enquanto o Livro R é gerado corretamente.

## Causa Raiz:

A geração do registro I250 para o Livro A não estava ocorrendo porque é necessária a importação da SAFX101 para que os registros sejam gerados corretamente. Além disso, o encerramento realizado pelo DW não estava gerando o Livro A devido às especificidades do sistema que pega exclusivamente da SAFX101 para essa geração.

## Resolução:

  * Verifique se a SAFX101 está corretamente carregada e importada no sistema.


  * Na parametrização dos Dados Iniciais, ajuste a "Gerar Plano de Contas" para o período correto conforme necessário.


  * Realize um novo teste de geração dos registros para o Livro A e confirme se os registros I250 estão sendo gerados conforme esperado.


## Próximos Passos:

Após implementar a solução, monitore os próximos encerramentos para garantir que o Livro A está sendo gerado corretamente. Caso encontre novas dificuldades, não hesite em abrir um novo ticket de suporte.

## Dicas e Recomendações:

Considere sempre verificar as atualizações e patches disponíveis para o sistema DW e mantenha sua equipe informada sobre as mudanças.
