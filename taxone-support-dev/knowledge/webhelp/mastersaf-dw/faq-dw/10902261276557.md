# [Básicos][Job Servidor] DW - SAFX03 - Erro 90127: A Data de Vencimento nao pode ser menor que a Data de Emissao do Documento

Este artigo tem por objetivo demonstrar como corrigir o erro 90994: Erro na conversao de dados para numerico.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10902218264589)

2\. O problema ocorre pois a data de vencimento está menor que a data de emissão:

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10902226803213)

3\. Para solucionar o problema basta ajustar o campo DATA_VENCTO para uma data igual ou menor que a data de emissão:

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10902229209869)
