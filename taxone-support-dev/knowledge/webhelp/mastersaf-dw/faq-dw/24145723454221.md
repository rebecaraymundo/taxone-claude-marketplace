Este artigo tem como finalidade orientar sobre motivo e solução do erro 104 no MastersafDW

Ao realizar carga de SAFX o sistema apresenta a seguinte msg de erro:

![](/attachments/token/Sb7p0nrQFObWrGzuK9m0dROUW/?name=image.png)

Esta mensagem é apresentada por falta de memória no Banco de Dados e deve ser analisada pelo DBA para solução.

Caso o erro seja recorrente deve ser seguido o seguinte fluxo:

1 - Boas Práticas no Banco.
2 - PAD's (Após retorno abrir demanda TechOps)
