# Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro 90468: Não foi possível recuperar para o estabelecimento o grupo de centralização do Código de Operação

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415145796493/mceclip0.png)

2\. O problema ocorre pois o estabelecimento preenchido no arquivo ainda não possui relacionamento entre tabelas e grupos.

Veja que para o estabelecimento em questão, não possui nenhuma relacionamento.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415145828109/mceclip1.png)

3\. Para corrigir o problema, basta realizar o relacionamento entre tabelas e grupos para o estabelecimento.
