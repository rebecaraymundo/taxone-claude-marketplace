## Problema:

Usuários questionam se é possível criar mais de um usuário administrador no sistema Mastersaf DW.

## Causa Raiz:

A dúvida surgiu devido à necessidade de ter múltiplos administradores para gerenciar o sistema de forma mais eficiente.

## Resolução:

  * Acesse o sistema Powerlock.
  * Navegue até **Maintenance > Administrators...**
  * Crie o novo usuário administrador conforme necessário.
  * Para atribuir responsabilidades ao novo administrador, vá em **Actions > Administrators > Assignment**.
  * Após a atribuição, o novo administrador poderá realizar as tarefas administrativas, exceto criar novos administradores.
