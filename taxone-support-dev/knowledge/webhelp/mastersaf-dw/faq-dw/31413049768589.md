# Configuração de Permissões para Compilação de Objetos PL/SQL com Opção "NATIVE"

## Problema:

O cliente precisa de assistência para configurar as permissões no sistema operacional para a compilação de objetos PL/SQL com a opção "NATIVE" devido a questões de performance.

## Causa Raiz:

A necessidade de configuração surge para garantir a performance ideal do sistema, conforme recomendado em nosso manual de instalação. Configurações não recomendadas podem não ser suportadas, o que pode levar a problemas operacionais.

## Resolução:

  * Acesse o manual de instalação disponível no nosso site.


  * Siga as instruções detalhadas para a configuração das permissões no sistema operacional.


  * Compile todos os objetos necessários com o parâmetro "NATIVE" conforme instruído.


  * Consulte a documentação adicional sobre a funcionalidade de compilação em: [Oracle Documentation](<https://docs.oracle.com/cd/B19306_01/appdev.102/b14261/tuning.htm#LNPLS01209>).
