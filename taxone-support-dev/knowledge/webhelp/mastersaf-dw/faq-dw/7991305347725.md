# Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro 91274: O campo Data do Registro de Exportação deve ser preenchido.

## Passo-a-Passo

1.Erro:

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/7991181392013/mceclip0.png)

2\. O problema ocorre pois o campo DAT_RE não esta preenchido ou esta preenchido com @

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/7991257095821/mceclip1.png)

3\. Para resolver o problema, basta preencher o campo DAT_RE com a data correta.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/7991283235469/mceclip2.png)
