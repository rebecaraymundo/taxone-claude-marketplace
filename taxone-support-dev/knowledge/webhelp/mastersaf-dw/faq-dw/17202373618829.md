## **Palavras-Chave**

ORA- 06502:PL/SQL : numeric or value error : character string buffer too small

## Introdução

A mensagem de erro abaixo, é apresentada devido ao campo ‘Núm. Recibo da Última Declaração’ conter a quantidade acima de 41 caracteres, conforme o leiaute do Guia Prático do EFD Contribuições solicita.
