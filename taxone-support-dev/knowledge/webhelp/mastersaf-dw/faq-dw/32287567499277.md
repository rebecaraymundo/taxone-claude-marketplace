# [Básicos][Gerais]  DW- É possível criar uma conta que não seja MASTERSAF para aplicar Patchs?

- **Atualizado:** 2025-04-11

---

O cliente teria um usuário de banco de dados que seria "vazio" e teria permissão para realizar alterações no owner do MastersafDW e o patch seria aplicado neste "owner vazio" que apontaria para os owner a ser atualizado. É possivel?

**Entendemos ser possível criar uma conta diferente para aplicar patches desde que tenha todas as permissões do usuário. Lembrando que esta definição da segurança on premisses fica à cargo da empresa contratante.**
