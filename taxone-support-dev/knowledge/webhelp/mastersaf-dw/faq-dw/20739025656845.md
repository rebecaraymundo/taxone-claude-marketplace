Erro 90150 Unidade de medida não cadastrada.

Realizar o cadastro da unidade de medida criticada com uma data de validade maior que a data fiscal da nota em questão.

Módulo DW no menu Manutenção - Código - Medida.

Abaixo o caminho no DW.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/20739016227597)
