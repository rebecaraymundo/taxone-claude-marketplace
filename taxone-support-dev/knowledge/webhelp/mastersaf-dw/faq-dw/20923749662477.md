# Introdução

Este artigo tem como objetivo demonstrar como criar um Perfil de Geração do EFD Contribuições

# Módulo e Menu

  * _Acessar Modulo_** _SPED > EFD Contribuicoes_**
  * _Acessar Menu_** _Parametros > Perfil_**


# Passos

_Clique no botão "Novo"_

![](https://t12927461.p.clickup-attachments.com/t12927461/9ce2a538-1bce-44a3-9ca1-c8d6c48efec5/image.png)

_Selecione um dos Layouts abaixo. Neste exemplo, utilizaremos a partir da versão 1.32_

![](https://t12927461.p.clickup-attachments.com/t12927461/80105a50-c2b9-462d-b95f-19426ba0c778/image.png)

_Digite um código e nome do perfil_

![](https://t12927461.p.clickup-attachments.com/t12927461/e729bcc5-2d6c-4f5e-b462-bfc3612f36ca/image.png)

![](https://t12927461.p.clickup-attachments.com/t12927461/955902a0-0aec-4b0c-aea0-8e90c840a98b/image.png)

_Selecione na tabela da esquerda o bloco que deseja marcar._

![](https://t12927461.p.clickup-attachments.com/t12927461/de02b428-94f5-4426-bf9f-d9f51cde854a/image.png)

_Na tabela da direita, marque os registros que devem ser gerados no meio magnético do EFD Contribuições. Neste exemplo estamos marcando todos os registros conforme abaixo, utilizamos "selecionar todos"_

![](https://t12927461.p.clickup-attachments.com/t12927461/96d13dbf-c80d-4427-83a6-fb4d11d395cb/image.png)

_O passo acima deve ser efetuado para cada bloco abaixo_

![](https://t12927461.p.clickup-attachments.com/t12927461/a755bc9d-fbae-42aa-91b9-4caa37a9735f/image.png)

_Após finalizar as marcações acima, clique no botão "confirma"_

![](https://t12927461.p.clickup-attachments.com/t12927461/316654f1-0d6b-4bff-8657-b142b9db939d/image.png)
