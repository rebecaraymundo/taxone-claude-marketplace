##

#### Palavras-Chave

Lançamento, INSS, Dados, Informação, Nula, GPS

#### Evidências

MENSAGEM DE ERRO: Tentativa de inserir informação(ões) nula(s)!

#### Causa-Raiz

O cadastro da tela de Manutenção de Dados de Retenção do INSS, está sendo realizado manualmente.

#### Solução

Essa tela não pode ser preenchida manualmente, essas informações são geradas automaticamente pela Nota Fiscal, é possível efetuar apenas alterações não todo o cadastro. Conforme o Manual Operacional:

Manutenção de Dados de Retenção: Nesta rotina é possível alterar/consultar os dados gerados de retenção de INSS.

Apenas após o processo de geração é possível preencher esta tela.
