# Erro na Importação das Tabelas Fixas

## Problema:

Após atualização do patch o sistema não conclui a atualização das tabelas fixas.

## Causa Raiz:

A causa deste problema pode-se atribuir a arquivos corrompidos no momento da extração dos arquivos do pacote.

## Resolução:

Extrair novamente todos os arquivos do pacote de atualização para uma _nova pasta,_ seguir com processo de atualização, utilizando a opção 'Atualizar TFIX'.
