# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 90085: O Numero de CGC/CPF deve ser preenchido para este Tipo de Registro

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414448006285/mceclip0.png)

2\. Quando o campo IND_CONTEM_COD possui algum desses valores (1, 2 ou 3) o campo CPF_CGC não pode ser nulo ou ter o valor **@**

No exemplo abaixo o campo CPF_CGC está com o valor @

**![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414427866765/mceclip1.png)**

3\. Ajustar o arquivo para um valor de CPF/CGC valido e o problema será sanado

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414435399181/mceclip2.png)
