# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 91769: O Campo de Data de Nascimento Autonomo com formato invalido

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414447882381/mceclip0.png)

2\. Verificar os campos DATA_NASC_AUTONOMO e DATA_MOLESTIA se estão como **00000000** ou com "**@** " ou em com data no formato **YYYYMMDD**

No exemplo abaixo o campo DATA_MOLESTIA está com valor **01**

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414435228685/mceclip1.png)

3\. Ajustar o arquivo para um dos três valores aceitáveis e o problema será sanado.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414435241869/mceclip2.png)
