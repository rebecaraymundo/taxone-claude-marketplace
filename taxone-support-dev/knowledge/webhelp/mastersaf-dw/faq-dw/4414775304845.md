# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 93385: Campo Produtor Rural invalido. Preencher com S ou N

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414775268493/mceclip0.png)

2\. O problema ocorre pois o campo IND_PRODUTOR_RUAL não possui algum dos valores pré-definidos:

Informar se a pessoa fisica/jurídica é um Produtor Rural, de acordo com:
S – Se a pessoa fis/jur for um produtor rural;
N – Se a pessoa fis/jur não for um produtor rural.

O objetivo do campo é simplesmente identificar quando a pessoa física/jurídica é um Produtor Rural, independente da condição de ter ou não uma inscrição estadual.

Este campo foi criado para atender à DMD – Bahia (Declaração da Movimentação de Produtos com ICMS Diferido), cuja orientação é para que todos os produtores rurais, sejam eles inscritos ou não no Cadastro do ICMS, sejam considerados como “não inscritos”.

Veja que no arquivo o campo está com o valor "1"

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414775287053/mceclip1.png)

3\. Para corrigir o problema, basta alterar o conteúdo do campo para um dos valores validos.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414775299725/mceclip2.png)
