# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 91768: Funcao nao existe para o codigo informado

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414736678541/mceclip0.png)

2\. O problema ocorre pois o campo COD_FUNCAO_AUTONOMO não está cadastrado na tela ata Warehouse > Manutenção > Códigos > Funcao

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414743073293/mceclip1.png)

3\. Basta cadastrar o código na tela e o problema será solucionado.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414743121933/mceclip2.png)
