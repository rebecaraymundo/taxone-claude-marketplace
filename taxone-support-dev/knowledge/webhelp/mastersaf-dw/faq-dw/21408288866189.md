## Introdução

_"Esse artigo tem por objetivo sanar o erro quando o Estabelecimento não é demonstrado para seleção na opção de Geração_

_"Arquivos de Entradas de Serviços ( Const. Civil / Utilities/ Telecom) nos Módulos Municipais._

## Solução

Nessa opção são listados somente os estabelecimentos que não pertencem ao município a ser gerado e que o Código do Estabelecimento esteja cadastrado

no arquivo de Licenciamento.

_![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21408288853773)_

Para o Municipio ser demonstrado para Seleção é necessário que o estabelecimento tenha inscrições municipais eventuais cadastradas para o município

a ser gerado em:

Básicos > MasterSAF DW menu Manutenção > Cadastros > Inscrições Municipais.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21408323901837)

No arquivo XML de licenciamento, devem constar os campos abaixo:

· Tag outras_inscricoes

· Código do Estabelecimento
