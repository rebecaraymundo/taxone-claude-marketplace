# Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro 90124: O código da pessoa Física/Jurídica não esta cadastrado.

## Passo-a-Passo

1\. Erro:

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/7991468808589/mceclip0.png)

2\. O problema ocorre pois o código da pessoa física/jurídica que consta na SAFX não esta cadastrado no sistema.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/7991507034893/mceclip1.png)

3\. Para resolver o problema, basta cadastrar a pessoa física/jurídica através do Data Warehouse ou importando a SAFX04.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/7991547808909/mceclip2.png)
