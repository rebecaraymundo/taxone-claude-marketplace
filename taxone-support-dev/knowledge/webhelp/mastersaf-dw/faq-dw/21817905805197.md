## PROBLEMA

O campo Valor reajustado deve ser preenchido com o valor igual ao resultado da expressão (Valor Líquido/0,65)

## SOLUÇÃO PROPOSTA

Este erro ocorre na Importação da SAFX279, isso ocorre porque o analista arredonda o valor, já o sistema utiliza apenas duas casas decimais.


Um exemplo:

Valor informado pelo Analista: 108548.22

Valor Líquido: 7055636 /0,65 = 108548,25

Para casos como este, recomendamos utilizar o próprio Excel, acrescentando 4 casas decimais para saber o valor exato, como no exemplo abaixo, como fica ,61 o sistema arredonda para 25.
