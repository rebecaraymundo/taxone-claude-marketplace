# Resolução de Erro na Importação de Dados com o Tipo de Código EAN na SAFX23 para SPED Fiscal

## Problema:

Erro na importação da SAFX23 para o SPED Fiscal devido à falta do cadastro do tipo de código EAN.

## Causa Raiz:

A causa do problema é a falta de informação sobre o código de barras no campo 40-Código de Barras da SAFX2013, que é necessário para o preenchimento do campo 04-COD_BARRA do registro 0200 do SPED-EFD.

## Resolução:

  * Acesse a tela da SAFX2013 no sistema.


  * Localize o campo 40-Código de Barras.


  * Preencha este campo com o código de barras correto que deve ser usado.


  * Salve as alterações e realize a importação novamente.


  * Verifique se o erro persiste. Se tudo estiver correto, a importação deve ocorrer sem erros.


## Próximos Passos:

Após realizar os passos acima, monitore as próximas importações para garantir que o erro não se repita. Caso o problema persista, entre em contato com o suporte técnico para uma análise mais detalhada.

## Dicas e Recomendações:

É importante sempre verificar se todos os campos obrigatórios estão preenchidos corretamente antes de realizar a importação de dados. Manter o sistema atualizado com as últimas instruções e manuais também é crucial para evitar problemas de incompatibilidade de informações.
