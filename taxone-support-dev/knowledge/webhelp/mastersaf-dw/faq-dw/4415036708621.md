# Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro 90009: O campo Data de Movimento não esta preenchido ou preenchido com zeros

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415056521613/mceclip0.png)

2\. O problema ocorre pois o campo DATA_OPERACAO não esta preenchido ou esta preenchido com zeros.

Veja que no arquivo o campo esta com "@"

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415056570125/mceclip1.png)

3\. Para corrigir o problema, basta alterar o conteúdo para uma data válida "AAAAMMDD"

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415056651789/mceclip2.png)
