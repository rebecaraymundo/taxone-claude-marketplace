# Verificação de Transferência de Dados entre Bases DW/OneSource e TAXONE

## Problema:

O cliente precisa de uma garantia de que todos os dados existentes na Base do DW/OneSource foram de fato transferidos para a Base TAXONE após o upload.

## Causa Raiz:

Atualmente, não existe uma ferramenta específica de comparação de bases ou um log de evidências que confirme que os registros e quantidades são os mesmos entre as bases DW e TAXONE.

## Resolução:

Embora não tenhamos uma ferramenta específica para comparar diretamente as bases, o processo de transferência de dados é feito por meio de um DUMP, que é uma cópia integral do Banco de Dados. Isso garante que todos os dados enviados à nossa equipe sejam adicionados ao TAXONE. Portanto, você pode confiar que a transferência foi completa e correta.

## Próximos Passos:

Após a transferência de dados, é recomendável que o cliente faça uma verificação manual dos dados mais críticos para garantir que todas as informações necessárias estejam presentes na Base TAXONE. Qualquer discrepância deve ser reportada ao suporte técnico para investigação adicional.

## Dicas e Recomendações:

Para futuras transferências, considere manter um registro local dos dados que serão transferidos e verificar esses registros após o processo. Isso pode ajudar a identificar rapidamente qualquer problema ou discrepância nos dados transferidos.
