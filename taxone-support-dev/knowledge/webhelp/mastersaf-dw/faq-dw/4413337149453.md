Alguns erros podem ocorrer no Mastersaf DW caso as configurações do arquivo safdw.ini estejam incorretas. Esse arquivo armazena os dados de conexão com o banco de dados.

Alguns exemplos de erros:

![Nao_foi_poss_vel_estabelecer_conexao_com_a_base_de_seguran_a.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4413351613325/Nao_foi_poss_vel_estabelecer_conexao_com_a_base_de_seguran_a.png)

![Array_boundary_exceeded_at_line_24_in_function_of_carrega_prt_pwrlock.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4413337032717/Array_boundary_exceeded_at_line_24_in_function_of_carrega_prt_pwrlock.png)

Para corrigir o problema, é necessário recriar o arquivo. Para isso, siga os passos:

1 - Encontre o arquivo **safdw.ini** na pasta de executáveis do DW **e renomeie** :

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5037655711885/mceclip0.png)

2 - Inicie o DW. Será apresentada a seguinte mensagem, clique em Sim:

![2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4413340887949/2.png)

3 - Informe os dados de conexão com o banco de dados:

![3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4413351637901/3.png)

4 - Clique em Gravar.

Feito isso, será criado um novo arquivo safdw.ini na pasta de executáveis com os dados corretos.

Palavras chave: erro ao abrir DW, erro abertura, array boundary exceeded
