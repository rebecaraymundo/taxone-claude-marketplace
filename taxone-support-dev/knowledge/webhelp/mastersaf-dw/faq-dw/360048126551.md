## Introdução

Este artigo tem por objetivo mostrar como realizar a exportação das parametrizações do sistema feitas em um determinado ambiente/estabelecimento, a fim de que sejam copiadas para outros ambientes (ou estabelecimentos).

## Passo-a-passo

**1 -** Para copiar/exportar a parametrização de um ambiente existente para outro, devemos utilizar a funcionalidade de "Cópia de Parametrizações", no módulo Ferramentas:

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415029996173/mceclip0.png)

**2 -** Após, basta escolher as parametrizações desejadas e executar:

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415030246669/mceclip1.png)

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415030268301/mceclip2.png)![](http://www.mastersaf.com.br/HELPDW/V2R010/Conteudo/Demais_Empresas/Mastersaf_DW/DW/KBA_FAQ/B%C3%A1sicos/Ferramentas/KBA_-_Como_gerar_uma_c%C3%B3pia_\(backup\)_das_parametriza%C3%A7%C3%B5es_de_um_ambiente_existente\(2\)_2_00.gif)

**3 -** Após a geração, devemos escolher um local para salvar o arquivo gerado e finalizar o backup das parametrizações:
