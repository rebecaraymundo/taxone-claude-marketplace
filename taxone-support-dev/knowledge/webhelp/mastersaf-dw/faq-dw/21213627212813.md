## Introdução

Este artigo tem por objetivo demonstrar uma rotina alternativa ao painel de controle de eventos para geração e envio do arquivo XML. Utilizando essa opção o usuário ganha celeridade no fluxo de geração e envio dos eventos com origem externa. Ela foi criada para atender principalmente o grande volume de eventos.

Importante ressaltar que essa opção é exclusiva para eventos com origem externa, e destina-se a geração e envio ao OneSource, ou seja, as outras opções, como por exemplo retificação de evento, deve ser executada a partir do painel de controle de eventos.

## Passo a passo

**1** \- Importação dos Arquivos Externos e Geração Prévia dos Movimentos

Os arquivos devem ser previamente importados para o MasterSAF-DW, e deve ser efetuado a geração prévia dos movimentos para o período.

Para importação dos arquivos e posterior geração prévia dos movimentos, pode ser verificado os seguintes artigos:

[TAX ONE - Importação de Arquivos Externos (JCP) XML dos Bancos](<https://suporte.thomsonreuters.com.br/hc/pt-br/articles/21162751356685>), e

[TAX ONE - Geração e envio de eventos relativos a arquivos externos (JCP) XML dos bancos](<https://suporte.thomsonreuters.com.br/hc/pt-br/articles/21165659218445>).
