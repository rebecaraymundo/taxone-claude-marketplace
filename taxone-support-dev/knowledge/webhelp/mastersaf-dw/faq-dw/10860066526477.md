## Introdução

_"Esse artigo tem por objetivo mostrar a geração do parâmetro "Modelo Msaf x Modelo" para todos os Módulos Municipais que utilizam este De x Para._

## Passo-a-passo

O objetivo deste parâmetro é realizar o relacionamento entre os modelos informados no documento fiscal e os modelos solicitados para geração dos validadores.

**1 - Documento Fiscal**

Verificar qual Tipo de Documento foi informada no Documento Fiscal:

Módulo: |  Básicos > Data Warehouse
---|---
Menu: |  Manutenção > Documento Fiscal > Doc. Fiscal de Serviço

![55.PNG](https://suporte.thomsonreuters.com.br/hc/article_attachments/10859905853069)

**2 - Parâmetro "_Modelo Msaf x Modelo_ _"_**

Módulo: |  Municipal > Parâmetros para Municipio
---|---
Menu: |  Parâmetros> _Modelo Msaf x Modelo_ __

Selecionar a UF e o Municipio

![56.PNG](https://suporte.thomsonreuters.com.br/hc/article_attachments/10860004248461)

**Validade: I** nformar a validade do relacionamento entre os modelos. A validade é do tipo “a partir de”. Essa data também será utilizada para buscar o grupo do modelo.

**Modelo MasterSAF:** Nesse campo o usuário irá selecionar os modelos utilizados nas notas fiscais cadastradas no MasterSAF DW.

**Modelo:** Selecionar o modelo através de uma lista com os modelos solicitados pelo validador em questão. O validador será identificado através do município previamente selecionado.
