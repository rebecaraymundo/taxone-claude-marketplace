# [Básicos][Gerais]  DW - Erro de registros na tabela EPC_REG_AJT_M105_M505 sem referência correspondente na tabela EPC_REG_AJT_M100_M500

**PROBLEMA:**

Durante validação da base de dados o usuário se depara com mensagem que registros na tabela EPC_REG_AJT_M105_M505 sem referência correspondente na tabela EPC_REG_AJT_M100_M500.

**CAUSA RAIZ:**

Dentro da lógica de relacionamentos, os registros da tabela EPC_REG_AJT_M105_M505 são filhos da tabela EPC_REG_AJT_M100_M500, e se não possuem referência é como se os não possuíssem uma capa.

É importante validar e ter certeza se realmente não possuem referência. Realizar essa validação vai ajudar você a tomar uma decisão sem comprometer a integridade dos dados do SPED.

**SOLUÇÃO:**

O problema reportado é referente a Base de Dados do ambiente do Cliente e neste caso a responsabilidade de correção é do Parceiro Fiscal ou do próprio cliente. A Thomson Reuters não tem responsabilidade/autonomia para interceder nas informações na base do cliente.

O correto é o Parceiro/Cliente corrigir as informações e gerar o DUMP para solicitação de Migração de DW para Tax One.
