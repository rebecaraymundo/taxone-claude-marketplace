## Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro Erro 90012: Conta contabil não esta cadastrada ou data de validade posterior a data de lançamento

## Passo-a-Passo

1.Erro:

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10301768093325)

2\. O erro ocorre pois a conta contábil não esta cadastrada ou esta com uma data de validade posterior a data do lançamento

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10301921304717)

3\. Para solucionar basta cadastrar a conta em DATA WAREHOUSE.

**Importante que a conta seja cadastrada com a data de validade menor que a data de lançamento.
