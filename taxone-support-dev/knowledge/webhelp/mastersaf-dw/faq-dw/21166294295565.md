## Introdução

Este artigo tem por objetivo demonstrar os processos de geração e envio para o OneSource, dos eventos com origem externa dos bancos (JCP).

Os arquivos devem ser previamente importados para a solução MasterSAF-DW. Para essa rotina temos o seguinte artigo: [DW - Importação de Arquivos Externos (JCP) XML dos Bancos](<https://suporte.thomsonreuters.com.br/hc/pt-br/articles/21159608571277>).
