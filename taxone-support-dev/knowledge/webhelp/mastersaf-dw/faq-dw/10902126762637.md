Este artigo tem por objetivo demonstrar como corrigir o erro 90994: Erro na conversao de dados para numerico.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10901617004429)

2\. O problema ocorre pois o valor informado em um dos campos abaixo está com algum caractere especial:

VLR_MOVTO, VLR_MOVTO_CONV, VLR_TOT_DOCTO, VLR_RET, VLR_N_RET, QTDE_PARCELA

No exemplo abaixo o campo VLR_MOVTO_CONV está com a letra "!" ao invés de "@" para informar nulo.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10901693273741)

3\. Para solucionar o problema basta ajustar o campo para numérico ou então nulo "@"

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10901726085517)
