Este artigo tem por objetivo demonstrar como corrigir o erro 93605: O campo Valor Liquido deve ser preenchido, quando a natureza de rendimento for igual a 19001 ou 19009

## Passo-a-Passo

1\. Erro

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21493829964173)

2\. Conforme o manual de layout de SAFX, este campo deve ser preenchido e com valor maior que zero, quando a natureza de rendimento for igual a **‘19001‘** ou **‘19009’**. Caso a natureza for diferente das citadas, o campo não deve ser preenchido.

No exemplo abaixo o IND_NAT_REND está diferente dos citados acima e o campo está preenchido.![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21493823033869)

3\. Para solucionar o problema, basta informar o IND_NAT_REND igual a 19001 ou 19009.

Ou informar @ no campo VLR_LIQ, para que leve a informação nula.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21493798028301)
