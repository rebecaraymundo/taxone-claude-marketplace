# Introdução

Este artigo tem como objetivo demonstrar como realizar a manutenção ou inserção de dados na Tabela de Dependentes sem Vínculo.

# Modulo e Menu

_Modulo: Básicos > Data Warehouse_

_Menu: Manutenção > Cadastros > Pessoas Físicas/Jurídicas_

# Passo a passo

_Ao abrir um cadastro de pessoa física, ficará habilitado o botão abaixo "Seleciona Dependentes sem vinculo"_
