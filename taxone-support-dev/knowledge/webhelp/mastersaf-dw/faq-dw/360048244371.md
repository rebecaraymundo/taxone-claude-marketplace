# [Básicos][Segurança] DW - Como liberar empresas para os usuários

- **ID:** 360048244371

- **Criado:** 2020-08-18
- **Atualizado:** 2024-04-09

---

Para liberar as empresas para os usuários, faça o seguinte

1 - Acesse o menu Básicos > Segurança

2 - Informe o usuário Admin e a senha.

3 - Clique em Users.

4 - Selecione o usuário que deseja liberar as empresas.

5 - Clique em User/Co.

6 - Selecione a Empresa que deseja liberar para o usuário.

7 - Clique em Save.

8 - Se desejar liberar a Empresa para mais de um usuário, selecione-os no quadro Replicar para Usuários.
