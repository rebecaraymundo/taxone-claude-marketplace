# [Básicos][Gerais] DW - Resolução de Erros nos Códigos de Unidade Federativa e Município no Mastersaf DW/TAXONE

## Problema:

O cliente enfrenta problemas ao tentar utilizar os códigos de Unidade Federativa e Município corretos na plataforma Mastersaf DW/TAXONE, resultando em erros de cadastro e validação.

## Causa Raiz:

O erro ocorre devido a uma confusão entre o código de Unidade Federativa (UF) e o código dos municípios dentro do estado de São Paulo. A tabela de Estados (Tfix04) utiliza o código 26 para São Paulo, enquanto a tabela de Municípios utiliza o código 35 para os municípios dentro do estado.

## Resolução:

  * Acesse o sistema Mastersaf DW/TAXONE.
  * Navegue até: BÁSICOS > DATAWAREHOUSE > Manutenção > Cadastros > Município.
  * Verifique e confirme que os códigos de municípios estão corretos conforme a Tabela do IBGE (Taces06).
  * Na safx07, certifique-se de utilizar o código 35 para municípios dentro de São Paulo nos campos 117 e 122.
  * Confirme que o campo UF_Origem e UF_Destino estão preenchidos com a sigla "SP" para São Paulo.
