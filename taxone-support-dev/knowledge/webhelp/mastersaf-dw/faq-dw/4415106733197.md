# Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro 90098: Os campos Código do Histórico e Histórico Complementar são relacionados, pelo menos um deve ser informado.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415106535821/mceclip0.png)

2\. O problema ocorre pois os campos HISTPADRAO e HISTCOMPL estão ambos vazios.

Veja que no arquivo, ambos estão com "@".

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415106664717/mceclip1.png)

3\. Para corrigir o problema, basta preencher um dos dois campos com as informações válidas.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415099568269/mceclip2.png)
