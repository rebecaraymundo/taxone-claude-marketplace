Algumas funcionalidades no Mastersaf DW utilizam o Diretório do Servidor. Esse cadastro é realizado no menu abaixo:

_Ferramentas > Parâmetros do Sistema > Cadastro de Diretórios do Servidor_

Para cadastrar, acesse a tela e faça o seguinte:

1 - Clique em Novo registro;

2 - Selecione o módulo para o qual o diretório será criado. No exemplo, utilizamos o Job Servidor;

3 - Defina um apelido para o diretório. Pode ser o nome da pasta destino ou qualquer outro;

4 - Preencha o local de destino do diretório. **Esse deve ser um local físico no servidor de banco de dados.**

5 - Clique em Salvar;

6 - Após salvar, será criado o script que deve ser executado no banco de dados;

ATENÇÃO! O script deve ser executado na Command Window e por usuário SYSDBA.

7 - Após execução dos scripts, as colunas Directory Criado e Permissão mudarão para Sim.
