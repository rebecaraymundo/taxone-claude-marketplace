# Introdução

Este artigo tem por objetivo detalhar o significado da sigla EXCLDF na descrição no job de importação

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414742652557/mceclip0.png)

2\. 'EXCLDF', é a descrição que o sistema dá à exclusões de documentos na tabela de processo (LOG_PROCESSO).

3\. Para descobrir o que foi realizado basta realizar um select nesta tabela ou pesquisar as descrições com os caracteres 'EXCLDF' nas consultas de LOG de processo.
