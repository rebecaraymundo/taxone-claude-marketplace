## PROBLEMA]

Erro Oracle ORA-01756 ao salvar alterações ou inclusões dentro das manutenções do módulo.

## SOLUÇÃO PROPOSTA

Este erro está associado as aspas simples utilizada dentro de arquivos para importação no banco de dados. Quando você converte os dados em texto no excel, é possível observar desta forma:

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21826106657037)

O Banco de dados ORACLE não interpreta essas aspas simples.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21826151929485)

Para cenários assim, segue abaixo as orientações da ORACLE sobre o que é necessário realizar.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21826123020173)

Portanto, se é necessário a utilização das aspas, ou utiliza-se duas aspas simples ou duas aspas duplas, mas nunca apenas 1 aspas no arquivo que será consumido pelo banco de dados.
