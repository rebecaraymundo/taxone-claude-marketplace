# Como Exportar Dados para CSV Quando Excede o Limite de Linhas do Excel na Família 4000, evento R-4020.

###

## Problema:

Usuário enfrenta problemas para gerar relatório da Família 4000 no Excel devido ao excesso de linhas, que ultrapassa o limite máximo permitido pelo software.

## Causa Raiz:

O problema ocorre porque o Excel possui um limite de 1.000.000 de linhas, e os dados do relatório da Família 4000, evento R-4020 "excedem" esse número.

## Resolução:

Para apoiá-los, segue uma consulta, para o cliente rodar na base e depois salvar em csv.

**Importante se atentar aos dados de empresa, estabelecimento e período que devem ser alterados para os dados do cliente.**

Segue abaixo:

SELECT oc.* , pagto.*, info.*, ret.*, ext.*
FROM Reinf_Pger_R4020_estab estab,
reinf_pger_apur apur,
reinf_pger_r4020_oc oc,
reinf_pger_r4020_ident_pgto pagto,
reinf_pger_r4020_info_pgto info,
reinf_pger_r4020_retencoes ret,
reinf_pger_r4020_info_pgto_ext ext
WHERE oc.id_r4020_estab = estab.id_r4020_estab
and estab.id_pger_apur = apur.id_pger_apur
and oc.id_r4020_oc = pagto.id_r4020_oc
and pagto.id_r4020_ident_pgto = info.id_r4020_ident_pgto
and ret.id_r4020_info_pgto (+) = info.id_r4020_info_pgto
and ext.id_r4020_info_pgto (+)= info.id_r4020_info_pgto
and apur.cod_empresa in ( '076') ----trocar pelo do cliente
and apur.cod_estab in ( 'REINFP') ----trocar pelo do cliente
and ( apur.dat_apur = to_date('30/09/2025', 'dd/mm/yyyy') );----trocar pelo do cliente


## Próximos Passos:

Após exportar os dados para CSV, verifique se todas as informações necessárias estão presentes e corretas. Faça também backups regulares dos seus dados exportados para evitar perdas.

## Dicas e Recomendações:

Considere utilizar softwares ou bancos de dados que suportem grandes volumes de dados, caso a necessidade de relatórios volumosos seja uma constante. Isso pode ajudar a evitar problemas futuros com limitações de software como Excel.
