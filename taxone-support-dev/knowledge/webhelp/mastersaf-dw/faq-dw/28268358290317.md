Para realizar a Importação de Registros XML Família 4000, siga as orientações:

Para a importação de arquivos externos no SPED > EFD -REINF > REINF> Arquivos Externos >Importação, orientamos que SEMPRE utilize arquivo de extensão CSV ao invés de XML, pois normalmente trata-se de grandes volumes.

Ao importar um arquivo CSV, desprezaremos a primeira linha, por se tratar do header. E a partir da segunda linha, cada linha deve conter as informações de um XML.

O formato XML terá uma finalidade segundaria, onde deve-se utilizar apenas para subida individual de registros, caso necessite.
