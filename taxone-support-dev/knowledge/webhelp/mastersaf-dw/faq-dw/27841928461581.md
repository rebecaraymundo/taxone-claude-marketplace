# Resolução de Erro na Geração do Evento R-4020 com Beneficiários Estrangeiros

## Problema:

Ocorre um erro na geração do evento R-4020 para beneficiários estrangeiros, mesmo após consultas ao FAQ DW.

![09-07-2024 10-23-59.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/28261022890637)

## Causa Raiz:

O erro é causado por um campo específico que não está sendo preenchido corretamente. Campo de nome "**Relação Fonte Pagadora/Beneficiário** " via tela de cadastro Pessoas Físicas/Jurídicas.

## Resolução:

  * Acesse a área de entrada de dados via módulo Data Warehouse, menu: Manutenção > Cadastros > Pessoas Físicas/Jurídicas.


  * Localize o campo indicado "Relação Fonte Pagadora/Beneficiário"


  * Preencha o campo corretamente com as informações do beneficiário estrangeiro.


  * Salve as alterações realizadas.


  * Realize novamente a geração do evento R-4020.


## Próximos Passos:

Após a implementação da solução, verifique se o erro foi resolvido realizando uma nova geração do evento.

## Dicas e Recomendações:

Assegure-se de revisar todos os campos necessários para a geração de eventos envolvendo beneficiários estrangeiros para evitar erros recorrentes.
