Este artigo tem por objetivo demonstrar como corrigir o erro 90381: O Campo Codigo de Centro de Custo nao esta cadastrado.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/6437563173645)

2\. O problema ocorre pois o código informado no campo CENTRO_CUSTO não está cadastro na tela Data Warehouse > Manutenção > Códigos > Centro de Custo

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/6437597216525)

3\. Para solucionar o problema basta informar um código valido no arquivo TXT.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/6437607712269)

Ou cadastrar o código informado no arquivo TXT na tela Data Warehouse > Manutenção > Códigos > Centro de Custo

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/6437654264461)
