## PROBLEMA

Ao importar XML de Licenciamento, meu estabelecimento aparece como "Não Licenciado".

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21815736042125)

## SOLUÇÃO PROPOSTA

Este caso o sistema esta mostrando desta forma porque este CNPJ trata-se de um **estabelecimento** , e estabelecimentos não são cadastrados via XML de Licenciamentos, devem ser cadastrados no módulo "Parâmetros".

Acesse o DW > Básicos > Parâmetros > Preliminares > Empresa/Estabelecimento

Ao acessar a página abaixo, basta cadastrar o Estabelecimento para uso.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21815707175309)

Qualquer dúvida, abrir chamado no Zendesk para solicitar apoio ao time de Suporte.
