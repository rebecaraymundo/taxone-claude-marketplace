_"_ Este artigo tem por objetivo demonstrar quais critérios de geração do Diferencial de alíquota com Base Dupla no módulo Report Fiscal''

_Primeiro verifique se a UF do estabelecimento que será apurado, está contemplando o cálculo do diferencial de alíquota com Base Dupla._

_![](https://suporte.thomsonreuters.com.br/hc/article_attachments/20738704952077)_

RS - Instrução Normativa RE Nº 39/2016:

Valor do DIFAL = [Base de Cálculo – Valor do ICMS
\--------------------------------------- x Alíquota de Destino] – Valor do ICMS
1 – Alíquota de Destino

• GO - Decreto 8.519, 29 de Dezembro de 2015:

Valor do DIFAL = Base de Cálculo
\---------------------------- x Diferença de Alíquota ICMS/100
1 – Alíquota Interna/100

• PR - Decreto 5.603, 29 de Novembro de 2016 e MG - Decreto 46.930, 30 de Dezembro de 2015:

Valor do DIFAL = Base de Cálculo - ICMS Origem
[ ( -------------------------------------- ) x Alíquota Interna/100 ] – ICMS Origem
1 – Alíquota Interna/100

• SP - Lei n° 17.470/21 e o Decreto n° 66.559/22:

Valor do DIFAL = Base de Cálculo - Valor do ICMS
[ ( -------------------------------------- ) x Alíquota Destino ] – Valor do ICMS
1 – Alíquota Destino

O Valor do DIFAL é demonstrado na Apuração do ICMS através de dois lançamentos, um a Crédito e outro a Débito:

Valor do Débito = Base de Cálculo - Valor do ICMS
[ ( -------------------------------------- ) x Alíquota Destino ]
1 – Alíquota Destino

O Valor Crédito = Valor do ICMS

**Obs.:** Para as UF’s de MG e PR a fórmula é a mesma.

• BA - Lei n° 14.415/21:

Valor do DIFAL = Base de Cálculo - Valor do ICMS
[ ( -------------------------------------- ) x Alíquota Destino ] – Valor do ICMS
1 – Alíquota Destino

• SC - Medida Provisória n° 250, de 31 de janeiro de 2022:

Valor do DIFAL = Base de Cálculo
[ ( -------------------------------------- ) x Diferença de Alíquota ICMS/100
1 – Alíquota Interna/100

• ES - Lei n° 11.623/22: Valor do DIFAL = Base de Cálculo - Valor do ICMS
[ ( -------------------------------------- ) x Alíquota Destino ] – Valor do ICMS
1 – Alíquota Destino

• PA - Lei n° 8.315: Valor do DIFAL = Base de Cálculo - Valor do ICMS
[ ( -------------------------------------- ) x Alíquota Destino ] – Valor do ICMS
1 – Alíquota Destino

• MA - Lei n° 11.867: Valor do DIFAL = Base de Cálculo – Valor do ICMS
( -----------------------------------------------) x Dif. de Alíquota ICMS/100
1 – Alíquota Interna/100

_Para isso, consulte o roteiro de utilização do módulo ICMS, tópico “Procedimentos para Cálculo do Diferencial de Alíquota”._
