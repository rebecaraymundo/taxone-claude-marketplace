# [Básicos][Segurança] DW - Redefinir senha Powerlock/Segurança

- **ID:** 360047766932
- **URL:** https://suporte.thomsonreuters.com.br/hc/pt-br/articles/360047766932--B%C3%A1sicos-Seguran%C3%A7a-DW-Redefinir-senha-Powerlock-Seguran%C3%A7a
- **Criado:** 2020-08-18
- **Atualizado:** 2021-10-13

---

Para redefinir a senha do módulo Segurança (Powerlock), execute o script abaixo no banco de dados:


    _update PL_ADMIN set ADM_PWD = 'powerlock' where ADM_LOGIN = 'admin';_
    _commit;_

A senha será redefinida para o padrão 'powerlock', minúsculo e sem as aspas.
