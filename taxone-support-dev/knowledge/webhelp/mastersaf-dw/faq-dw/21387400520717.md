Este artigo tem por objetivo demonstrar como corrigir o erro 91861: Centro de Custo nao encontrado.

## Passo-a-Passo

1\. Erro

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21387427064205)

2\. O problema ocorre pois o campo COD_CUSTO não possui cadastro na tabela X2003_CENTRO_CUSTO

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21387409430541)

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21387384525837)

3\. Para solucionar o problema você pode ajustar o COD_CUSTO para um código existente na tabela X2003_CENTRO_CUSTO ou então cadastrar manualmente na tela Dataware House >>Manutencao >> Codigos >> Centro de Custo

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21387427066509)

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21387409432333)
