# Introdução

Este artigo te por objetivo demonstrar como corrigir o erro 90034: Código do produto não cadastrado

## Passo-a-Passo

1.Erro:

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10021615704973)

2\. O problema ocorre pois o código do produto que consta na SAFX, não esta cadastrado no sistema.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10021604979981)

3\. Para resolver o problema, basta cadastrar o produto através do Data Warehouse ou importando a SAFX2013.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10021703246093)
