# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 90091: O Indicador de Conteudo de Codigo de CGC/CPF deve ser preenchido de acordo com : <1>, <2>, <3> ou <4>

## Passo-a-Passo

1.Erro![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4413817202061/mceclip0.png)

2\. Verificar o campo IND_CONTEM_COD no arquivo txt se está em branco ou com "@" ou então não possui as seguintes informações validas:

Informar o Indicador do Conteúdo do Código, de acordo com:
1 - Se no Campo de Código (02) o conteúdo for um Código para Pessoa Física/Jurídica que tenha CGC/CPF;
2 - Se no Campo de Código (02) o conteúdo é um CPF;
3 - Se no Campo de Código (02) o conteúdo é um CGC;
4 - Se a Pessoa Física/Jurídica não tenha CPF/CGC - casos especiais.

No exemplo abaixo o arquivo está com a informação diferente do que se espera, por isso o erro ocorre

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4413802596621/mceclip1.png)

3\. Ajustar o IND_CONTEM_COD para o conteúdo correto e o problema será sanado
