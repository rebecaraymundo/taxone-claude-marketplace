# Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro 90011: Conta contábil deve ser preenchida.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415109551757/mceclip0.png)

2\. O problema ocorre pois a conta contábil não esta preenchida ou esta preenchida com "@".

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415109599757/mceclip2.png)

3\. Para corrigir o problema, basta preencher a conta contábil corretamente.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415109564685/mceclip1.png)
