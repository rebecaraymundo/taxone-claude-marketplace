# Introdução

Este artigo tem por objetivo demonstrar com corrigir o erro "Usuário ou Senha Inválido" para novos usuários cadastrados no powerlock.

## Passo-a-Passo

1\. Erro: Após criar um novo usuário no PowerLock e exibir a mensagem no primeiro login:

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4413813180557)

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4413820071565)

2\. Certifique-se que o usuário criado está vinculado a um GRUPO, CONEXÃO ou APLICAÇÕES no meu Profile.
Exemplo: Selecionar o conteúdo do lado Direito e arrastar para o usuário criado do lado Esquerdo.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4413813202061)

3\. Após o vinculo, no próximo login será permitido a troca da senha inicial

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4413813211789)
