# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 91683: A data de validade da pessoa fisica/juridica e anterior a data de inicio de vigencia do codigo de atividade economica informada.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414667723917/mceclip0.png)

2\. O problema ocorre pois o código do campo COD_ATIVIDADE possui uma data maior que o campo DATA_X04

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414652215437/mceclip1.png)

3\. Para verificar a data de validade do COD_ATIVIDADE basta acessar o menu BÁSICOS > PARÂMETROS > Preliminares > Atividade Econômica

No código do exemplo o inicio de validade do COD_ATIVIDADE é 01/01/2020

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414667731341/mceclip2.png)

Basta alterar a DATA_X04 para igual ou maior que a data de validade ou então procurar um COD_ATIVIDADE valido para esse Cliente/Fornecedor

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414659419789/mceclip3.png)
