# Introdução

Este artigo tem como objetivo demonstrar como gerar o informe de rendimento para Código de DARF 0916.

# Módulo e Menu

  * _Modulo**Básicos > Data Warehouse**_
  * Menu _**Manutenção > Retenções > Controle de Tributos**_
  * _Modulo**Federal > Obrigações de Tributos Federais**_
  * _Menu**Rendimentos > Rendimentos Outros > Emissão da Declaração de Rendimentos (Grandes Volumes) > Por Pessoa Física**_

 _No Modulo Básicos > Data Warehouse > Manutenção > Retenções > Controle de Tributos verifique se o documento está com os campos abaixo preenchidos._
