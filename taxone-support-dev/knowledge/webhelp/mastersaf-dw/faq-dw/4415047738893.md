# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro "Falta valor obrigatório para Coluna. Por favor, entre com o valor." ao tentar fazer a alteração do cadastro de Pessoa Física/Jurídica.

## Passo-a-Passo

Realizar um select na tabela X04_PESSOA_FIS_JUR e verificar se o campo IND_PRODUTOR_RURAL está em branco ou nula.
Se estiver, favor realizar um update para "N" e verificar se a situação continua.
