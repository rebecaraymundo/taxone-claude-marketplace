# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 92622: O Campo de Data de Prestacao de Servico com formato invalido

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414447973133/mceclip0.png)

2\. Verificar os campos DATA_PREST_SERV se estão como **00000000** ou com "**@** " ou em com data no formato **YYYYMMDD**

No exemplo abaixo o campo DATA_PREST_SERV está com valor 0202

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414447980301/mceclip1.png)

3\. Ajustar o arquivo para um dos três valores aceitáveis e o problema será sanado

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414435342093/mceclip2.png)
