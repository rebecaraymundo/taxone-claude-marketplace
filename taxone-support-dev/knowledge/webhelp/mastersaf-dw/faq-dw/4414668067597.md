# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 90549: O Codigo do Municipio nao esta cadastrado ou com informacao alfanumerica no codigo.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414668025357/mceclip0.png)

2\. Esse erro pode ocorrer por dois motivos:

\- Campo UF está nulo ou de um estado diferente do COD_MUNICIPIO

\- COD_MUNICIPIO não está cadastrado na tela BÁSICOS > DATAWAREHOUSE > Manutenção > Cadastros > Municipio

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414668032781/mceclip1.png)

3\. Para solucionar o problema basta cadastrar o COD_MUNICIPIO na tela mencionada no item 2 ou verificar se o campo UF está correto para o municipio.

No exemplo acima o campo UF estava nulo (@), basta alterar o campo UF para o estado do municipio.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414652586381/mceclip2.png)
