# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 91381: O campo Categoria do Trabalho nao esta cadastrado.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414730567437/mceclip0.png)

2\. O problema ocorre pois o campo IND_IDENTIF_FISCAL não possui algum dos valores pré-definidos:

Informar o Indicativo do Número de Identificação Fiscal. Preencher com:
1 - Beneficiário com NIF;
2 - Beneficiário dispensado do NIF;
3 - País não exige NIF.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414730673037/mceclip1.png)

3\. Basta alterar o conteúdo do campo para um dos valores validos.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414744334989/mceclip2.png)
