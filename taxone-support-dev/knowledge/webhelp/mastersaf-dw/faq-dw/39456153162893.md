# Excluir dados antigos de empresas não licenciadas no Mastersaf DW

## Problema

Usuários do Mastersaf DW precisam excluir dados antigos de empresas que não fazem mais parte do grupo há mais de 5 anos, mas essas empresas não estão licenciadas, tornando impossível realizar a exclusão diretamente pela interface do sistema.

## Causa Raiz

O sistema Mastersaf DW exige que as empresas estejam licenciadas para acessar as funcionalidades de exclusão de dados pela interface padrão. Sem o licenciamento adequado, as opções de exclusão não ficam disponíveis para o usuário.

## Resolução

Siga os passos abaixo para abordar o problema:

  1. Verifique com seu time interno ou com Customer Success (CS) a viabilidade de licenciar as empresas no Mastersaf DW para habilitar a funcionalidade de exclusão.
  2. Considere incluir o licenciamento dessas empresas como parte do planejamento de migração do Mastersaf DW para o Onesource Tax One.
  3. Entre em contato com o responsável pelo atendimento do CS, caso haja dúvidas ou necessidade de suporte adicional.


## Próximos Passos

Após seguir os passos acima e realizar o licenciamento, utilize a funcionalidade de exclusão pela interface padrão do Mastersaf DW para apagar os dados antigos das empresas. Caso esteja em processo de migração para o Tax One, garanta que os dados a serem transferidos estejam corretos e atualizados.
