Este artigo tem por objetivo demonstrar como corrigir o erro 90209: O Campo de Data de Validade com formato invalido.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10442157661197)

2\. O problema ocorre pois o valor informado no campo DATA_MOVTO está fora do formato AAAAMMDD (ANO-MES-DIA)

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10442147422093)

3\. Para solucionar o problema basta corrigir o formato da data no campo DATA_MOVTO:

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10442200722701)
