## Introdução

Esse artigo tem por objetivo mostrar as causas e soluções para sanar o erro _Não foi possível estabelecer a conexão com a base de segurança:__Oracle library OCI.DLL could not be loaded_.

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414606999693/mceclip0.png)

## Causas

Este erro ocorre quando há algum problema com a instalação do Oracle Client. Nesses casos, é bem provável que as versões entre o Powerbuilder (Mastersaf DW) e o Oracle Client estejam incompatíveis entre si. Por esse motivo, é mandatório que o Cliente seja 32 bits (isso independente da estrutura do Sistema Operacional).

É importante destacar que o Mastersaf DW foi desenvolvido em uma arquitetura 32 bits e só pode ter seus componentes (software) carregados caso o Oracle Client também seja 32 bits.

## Passo-a-passo

Para solucionar este problema, é necessário remover qualquer vestígio de antigas instalações de Oracle Client da máquina e instalar a versão 32 bits corretamente (isso inclui a remoção das chaves dos Clients antigos no Registro do Windows).

Lembrando que a variável do Windows "PATH" deve estar apontando corretamente para o diretório do Client Oracle 32 bits, conforme imagem abaixo:
