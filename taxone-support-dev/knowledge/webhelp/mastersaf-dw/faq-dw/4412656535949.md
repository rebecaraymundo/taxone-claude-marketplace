## Introdução

_"Esse artigo tem por objetivo mostrar os requisitos para geração do Registro 0200 do SPED Contribuições para o produto Mastersaf DW"._

## Passo-a-passo

_Para os casos em que o registro 0200 do SPED esteja sendo gerado de forma incompleta deve ser verificados os seguintes parâmetros:_

_1- Marcar a opção**" Preparação dos Dados de Documento Fiscais EPCDF / EPCNE) Expressa"** em Geração dos Registros._

_**![2-_Gera__o_de_Arquivo.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4412663510541/2-_Gera__o_de_Arquivo.png)**_

_2- E em**Dados Iniciais** marcar a opção **replicar para os estabelecimentos** e na sequência gerar o arquivo novamente._

![1-_Dados_Iniciais.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4412648527373/1-_Dados_Iniciais.png)
