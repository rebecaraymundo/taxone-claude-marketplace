# [Básicos][Job Servidor] DW -  Mastersaf DW - Job Servidor - Resolvendo Problemas de Substituição de Registros e Vinculação de Notas Fiscais nos Módulos SAFX309 e SAFX311

**Problema:**

Durante a importação de registros no módulo SAFX309, um registro está sendo substituído pelo segundo ao invés de serem inseridos separadamente. Além disso, no módulo SAFX311, não há um campo para inserir o código de ajuste, o que faz com que a interface vincule todas as notas fiscais ao primeiro código de ajuste encontrado na SAFX309.

**Causa Raiz:**

O problema ocorre porque o campo NUM_DISCRIMINACAO, que é um sequencial chave, está com a mesma informação em ambos os registros na SAFX309. Além disso, o módulo SAFX311 não tem o campo necessário para discriminar o código de ajuste específico.

**Resolução:**

No módulo SAFX309, ajuste o campo NUM_DISCRIMINACAO para valores distintos em cada registro de forma sequencial. Verifique se o campo NUM_DISCRIMINACAO na SAFX311 está configurado para corresponder ao valor do mesmo campo na SAFX309 correspondente. Após realizar esses ajustes, faça uma nova importação e verifique se cada registro e nota fiscal está sendo vinculado corretamente.
