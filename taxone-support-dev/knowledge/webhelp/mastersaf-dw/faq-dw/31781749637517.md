## Problema:

O usuário enfrenta um erro ao tentar aplicar um patch de atualização na aplicação Mastersaf, que foi recentemente vinculada a um banco de dados na Oracle Cloud Infrastructure (OCI). O erro indica uma falha na configuração do Oracle Notification Service (ONS).

## Causa Raiz:

O erro ocorre porque o executável de atualização tenta configurar o Fast Application Notification (FAN), que é um recurso de alta disponibilidade que depende do ONS configurado para Oracle RAC, o qual não é aplicável ao ambiente atual do usuário que usa uma única instância na OCI.

## Resolução:

  * Localize o arquivo do pacote de atualização no seu sistema.
  * Substitua o arquivo original pelo arquivo atualizado que desativa o FAN. Este arquivo pode ser fornecido pelo suporte técnico.
  * Abra o prompt de comando e navegue até o diretório onde o arquivo de atualização está localizado.
  * Execute o comando de atualização com o parâmetro adicional para desativar o FAN:




    java -Doracle.jdbc.fanEnabled=false -jar -Xms512m -Xmx768m ApplyPatchAcumulativo.jar

  * Verifique se a atualização foi aplicada sem erros.
