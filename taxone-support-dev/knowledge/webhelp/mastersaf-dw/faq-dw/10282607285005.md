## Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro Erro 90129: O campo Tipo de Movimento de Entrada/Saída não esta preenchido ou com informação inválida

## Passo-a-Passo

1.Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10282495422093)

2\. O erro ocorre porque o campo normal ou devolução não esta preenchido ou esta com informação inválida.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10282567070733)

3\. Para solucionar basta preencher o campo com os dados corretos 1 (Normal) ou 2 (Devolução).
