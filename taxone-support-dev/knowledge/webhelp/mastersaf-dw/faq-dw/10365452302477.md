Este artigo tem por objetivo demonstrar como corrigir o erro 93185: Para o indicador de Lancamento igual a <X> (Lancamento Extemporaneo) o campo Data de Lancamento Extemporaneo deve ser preenchido

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10365360800909)

2\. O problema ocorre pois o código informado no campo TIPO_LANCTO está igual a "X - Extemporâneo" e o campo DAT_LANCTO_EXTEMP está sem data

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10365383841677)

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10365356049165)

3\. Para solucionar o problema deve-se verificar se o lançamento realmente é extemporâneo e então informar a data no campo DAT_LANCTO_EXTEMP, caso contrario deve-se alterar o campo TIPO_LANCTO para "N - Normal" ou "E - Encerrado"

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10365389268365)
