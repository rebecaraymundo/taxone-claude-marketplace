#### Causa-raiz

Uma vez que não há uma legislação fixa para estes amparos, o SUBITEM cumpre este papel, podendo ser cadastrados SUBITENS para determinado código de amparo, de acordo com a necessidade do cliente.

#### Solução proposta

Para os lançamentos terminados por ***99, onde cita-se que sejam definidos pelo CONTRIBUINTE, para que a geração automática seja realizada, deve-se vincular o código de amparo à um código de SUBITEM.

Para realizar tal vinculação ao código de amparo ***99, basta selecioná-lo na tela de lançamentos complementares e realizar o processo normalmente.
