# Ao gerar o SPED FISCAL, apresentou o seguinte erro: Erro ocorrido no processamento de documentos fiscais.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/37361920026893)

O erro esta relacionado a problemas com a versão dos executáveis utilizados no software de gestão fiscal ou configurações inadequadas.

  * Verifique o Manual de Boas Práticas, fornecido pelo software de gestão fiscal, para garantir que todas as configurações estão corretas.
  * Realize a Substituição dos Executáveis conforme orientado no manual, que pode ser encontrado em anexo ao ticket de suporte.
  * Após realizar essas atualizações, tente gerar novamente a EFD Fiscal e verifique se o erro persiste.
