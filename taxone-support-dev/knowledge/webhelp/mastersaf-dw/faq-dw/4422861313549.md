# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 90002: Codigo de Estabelecimento deve ser preenchido

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4422831652237)

2.O erro ocorre devido ao campo COD_ESTAB estar com valor @ ou em branco

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4422861317517)

3\. Para solucionar o problema basta informar a empresa correspondente aos dados que está importando.
