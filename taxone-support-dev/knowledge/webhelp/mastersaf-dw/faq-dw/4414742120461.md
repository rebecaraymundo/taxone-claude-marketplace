# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 90418: O Codigo de CBO nao esta cadastrado

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414742009229/mceclip0.png)

2\. O problema ocorre pois o campo COD_CBO não está cadastrado na tela Data Warehouse > Manutenção > Códigos > Código CBO

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414742030349/mceclip1.png)

3\. O código não estava cadastrado, basta realizar o cadastro na tela e o problema será sanado

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414735861133/mceclip3.png)
