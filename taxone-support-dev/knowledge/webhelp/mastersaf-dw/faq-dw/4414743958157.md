# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro [Básicos][Job Servidor] DW - SAFX04 - Erro 91899: Campo com conteudo invalido. Este campo somente podera conter a informacao Sim se a pessoa for juridica. Verificar campo 06 (CPF_CGC).

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414743887117/mceclip0.png)

2\. O problema ocorre pois o campo IND_PESSOA_LIVRE está igual a "S" e o campo CPF_CGC está com conteúdo de CPF ao invés de CNPJ

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414730320909/mceclip2.png)

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414737383821/mceclip1.png)

3\. Verificar se o indicador está correto, caso esteja é necessário alterar o campo CPF_CGC para um CNPJ valido, caso o indicador esteja incorreto basta altera-lo para "N"
