# Resolução de Problemas de Execução do Manager.exe no Mastersaf DW via Windows Remote Desktop

## Problema:

O cliente enfrenta problemas ao tentar executar o arquivo manager.exe no Mastersaf DW através do Windows Remote Desktop. A aplicação apresenta a tela de logon, mas é encerrada logo em seguida.

## Causa Raiz:

A causa raiz do problema está relacionada a um erro com uma DLL específica, conforme indicado no Visualizador de Eventos do Windows. A falha persiste mesmo após adicionar a conta como administrador local.

## Resolução:

  * Acesse as propriedades do arquivo executável (manager.exe).


  * Navegue até a aba de compatibilidade.


  * Marque a opção "Executar este programa em modo de compatibilidade".


  * Selecione uma versão anterior do Windows na qual o programa funcionava corretamente.


  * Clique em "Aplicar" e depois em "OK".


  * Tente abrir a aplicação novamente.
