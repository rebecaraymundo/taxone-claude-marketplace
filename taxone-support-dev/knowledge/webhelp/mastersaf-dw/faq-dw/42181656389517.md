## Problema

Ao tentar importar o arquivo SAFX82, ocorre o erro de situação “I”, mesmo quando o cadastro do BEM está definido como Componente.

## Causa Raiz

Esse erro normalmente ocorre quando o campo **"Não Apropriar Crédito dos Bens Componentes adquiridos a partir de:"** não está devidamente preenchido ou apresenta uma data incompatível com o cadastro do bem e a data de operação informada na SAFX82. O sistema realiza validações entre o status do ativo, o tipo do bem (Componente) e as datas cadastradas, resultando no erro caso alguma dessas condições não seja atendida.

## Resolução

  1. Acesse o sistema e navegue até o caminho: **Estadual > Ativo Permanente > Cadastros > Parâmetros de Estabelecimento**.
  2. Localize o campo **"Não Apropriar Crédito dos Bens Componentes adquiridos a partir de:"**.
  3. Verifique se este campo está preenchido:
     * Caso esteja em branco, preencha com a data correta conforme sua operação.
     * Certifique-se de que a data informada seja compatível com a data de operação da SAFX82 e com o cadastro do bem como Componente (TIPO_BEM = '2').
  4. Salve as alterações realizadas.
  5. Tente realizar novamente a importação da SAFX82.
