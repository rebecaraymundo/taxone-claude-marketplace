# Resolução de Divergência nos Tipos de Serviços em Relatórios do REINF

## Problema:

Usuário relata que o relatório "Listagem de Serviços Sujeitos à Retenção Previdenciária" está apresentando divergências nos tipos de serviços, especificamente com a inclusão do tipo "R", que está duplicando registros.

## Causa Raiz:

A raiz do problema foi identificada como uma parametrização desnecessária no sistema, que estava permitindo a inclusão do tipo "R" para um serviço que não deveria ser listado nessa categoria no relatório.

## Resolução:

  * Acesse o módulo REINF Parâmetros.


  * Navegue até Retenções Previdenciárias.


  * Selecione Identificador do Tipo de Serviço Por Código de Serviço.


  * Verifique a parametrização e exclua o tipo "R" se não for necessário para sua empresa.


  * Refaça a emissão do relatório para confirmar que a divergência foi corrigida.


No retório quando é gerado como:

S - é para indicar parametrização do Identificador de Tpo de Serviço por Tipo de Serviço. Essa parametrização é para geração dos eventos R-2010 e R-2020.

R - é para indicar parametrização do Identificador de Tpo de Repasse por Tipo de Serviço. Essa parametrização é para geração dos eventos R-2040. Se o cliente não entrega esse tipo de evento não precisa essa parametrização .


## Próximos Passos:

Após realizar as correções, monitore os próximos relatórios emitidos para garantir que o problema não se repita. Caso a divergência persista, entre em contato com o suporte técnico para uma análise mais aprofundada.

## Dicas e Recomendações:

É importante revisar regularmente as parametrizações no sistema para garantir que todas estão corretas e atualizadas, minimizando assim a chance de erros em relatórios e declarações. Mantenha-se atualizado com as últimas versões do software e consulte o manual do REINF regularmente para quaisquer atualizações ou mudanças.
