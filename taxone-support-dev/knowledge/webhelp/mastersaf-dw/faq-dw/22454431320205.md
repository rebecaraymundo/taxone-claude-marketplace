Este artigo tem como objetivo dar orientações caso cliente não encontre o arquivo Msaf.exe na pasta de instalação do DW.

Durante a instalação do Patch 210 do MastersafDW, o Manual de Instalação orienta que na pasta haverá um arquivo chamado Msaf.exe

![](https://suporte.thomsonreuters.com.br/attachments/token/rHWBFEBpyTyXrSUMdzIvbRaTL/?name=image001.png)

Caso algum cliente queixar que arquivo não esteja presente na pasta orientar o cliente a utilizar o arquivo:
**ApplyPatchAcumulativo.exe
