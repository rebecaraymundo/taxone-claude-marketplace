## Problema:

Erro ao importar arquivo: "O Codigo da Pessoa Fisica/Juridica não está cadastrado".

## Causa Raiz:

O erro ocorre porque o código de Pessoa Física/Jurídica necessário para a importação do arquivo não está registrado no sistema no módulo DW.

## Resolução:

Para resolver esse problema, siga os passos abaixo para cadastrar o código de Pessoa Física/Jurídica:

  * Acesse o módulo DW.


  * Navegue até o menu **Manutenção** e selecione **Cadastros**.


  * Escolha a opção **Pessoa Física/Jurídica**.


  * Preencha os campos necessários como indicado na imagem anexa ao ticket de suporte.


  * Salve as alterações para concluir o cadastro.


## Próximos Passos:

Após completar o cadastro, tente novamente importar o arquivo. Se o erro persistir, entre em contato com o suporte técnico para assistência adicional.

## Dicas e Recomendações:

Verifique sempre se todos os campos obrigatórios estão preenchidos antes de salvar o cadastro, para evitar erros de importação. Mantenha os cadastros atualizados para garantir a eficiência das operações no sistema.
