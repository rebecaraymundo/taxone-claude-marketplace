## Problema:

Após a aplicação de um patch de atualização, o programa Mastersaf DW entra em looping e a tela fica congelada no processamento, impedindo o avanço no uso do software.

## Causa Raiz:

A causa raiz do problema pode estar relacionada à incompatibilidade da versão do Oracle utilizada ou à configuração incorreta após a atualização do patch. O problema pode também ser exacerbado por configurações de segurança como o antivírus ou pela falta de arquivos dll necessários no sistema operacional.

## Resolução:

  * Navegue até a pasta "Executáveis" onde o Mastersaf DW está instalado.


  * Localize e abra o arquivo "SAFDW.ini".


  * No campo [profiles] sob DBMS, adicione a palavra TRACE após ORA, conforme mostrado em uma imagem de exemplo enviada anteriormente.


  * Salve as alterações e abra o Mastersaf DW. O software pedirá que você salve um log de cada tela acessada até que ocorra o erro.


  * Quando o erro ocorrer, salve o log em um local seguro e envie pelo ticket de suporte.


  * Feche o Mastersaf DW e remova o TRACE do arquivo SAFDW.ini.


  * Verifique também se o Oracle Client está instalado corretamente na máquina, e se as dll necessárias estão presentes nas pastas 'System32' e 'SysWOW64' do Windows.


  * Desative temporariamente o antivírus para verificar se ele está bloqueando a conexão do software com o banco de dados.


## Próximos Passos:

Após seguir os passos de resolução, monitore o sistema para verificar se o problema foi completamente resolvido. Se o erro persistir, reabra o ticket de suporte para uma investigação mais detalhada.

## Dicas e Recomendações:

Assegure-se de que todas as atualizações de software sejam compatíveis com a infraestrutura existente antes de aplicá-las. Mantenha o Oracle Client e os arquivos dll do sistema operacional sempre atualizados. Considere configurar exceções no antivírus para softwares críticos como o Mastersaf DW para evitar interrupções no funcionamento.
