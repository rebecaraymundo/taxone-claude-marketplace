## Introdução

Este artigo tem por objetivo demonstrar como resolver o erro Grupo de Pessoa Fisica/Juridica deve ser preenchido no módulo Data Warehouse.

## Passo-a-passo

1) Ao tentar cadastrar um novo controle de tributos no caminho Manutenção>Retenções>Controle de Tributos, informando a data o campo de Pessoa Fis/Jur deve ser preenchido automaticamente.
![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10813218335501)

2) Quando não ocorre, pode ocorrer o seguinte erro.
![blobid0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10813303324173)

3) validar a forma que a data do computador está configurada, sempre deixar no formato dd/mm/yyyy
