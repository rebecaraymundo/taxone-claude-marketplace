Este artigo tem por objetivo demonstrar como corrigir o erro Erro 91316: Erro na validação do CFOP

1.Erro:

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10300954825101)

2\. O erro acontece pois o campo do CFOP esta preenchido com informação inválida

3\. Para solucionar o erro, basta preencher o CFOP com uma informação válida.
