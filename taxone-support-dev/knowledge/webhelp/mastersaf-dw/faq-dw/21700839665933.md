# **Solução para o Erro SAFX07 - Erro 90971: Código do Município de ISSQN não cadastrado**

Ao se deparar com o erro SAFX07 - Erro 90971, referente ao campo "código do município de ISSQN não cadastrado", é essencial seguir alguns passos específicos para corrigir essa questão dentro do sistema Mastersaf DW. Este erro pode ocorrer durante o processo de cadastro ou utilização de informações relacionadas ao ISSQN (Imposto Sobre Serviços de Qualquer Natureza).

## Identificando a Origem do Problema

O erro SAFX07 - Erro 90971 geralmente ocorre quando o código do município do ISSQN não está devidamente registrado na tabela de Município ISS dentro do Mastersaf DW.

## Solução Passo a Passo

Para resolver essa questão, siga as orientações abaixo:

### Passo 1: Acesso ao Módulo de Manutenção do Mastersaf DW

  1. Acesse o Mastersaf DW.
  2. Navegue até a seção de **Básicos > DW > Manutenção > Cadastros > Município ISS > Manutenção**.


### Passo 2: Cadastro do Código do Município do ISSQN

  1. Dentro do módulo de manutenção, localize a opção para inserir ou editar informações referentes ao Município do ISSQN.
  2. Insira o código do município de acordo com a Tabela de Município (SAFX2097) no campo designado para tal.


### Passo 3: Utilize o Manual Operacional como Referência

  1. Caso haja dúvidas sobre quais informações devem ser cadastradas nessa tabela específica, utilize o manual operacional do sistema como guia.
  2. O manual conterá informações detalhadas sobre os campos e dados necessários para o cadastro correto do código do município do ISSQN.


### Passo 4: Verificação e Testes

  1. Após inserir ou editar o código do município do ISSQN, verifique se o problema foi resolvido.
  2. Faça testes no sistema para garantir que o erro SAFX07 - Erro 90971 não está mais sendo exibido.


## Conclusão

Corrigir o erro SAFX07 - Erro 90971 relacionado ao campo "código do município de ISSQN não cadastrado" demanda a inserção correta dessas informações na tabela de Município ISS do Mastersaf DW. Ao seguir os passos acima e utilizar o manual operacional como referência, será possível resolver esse problema e garantir a consistência dos dados fiscais dentro do sistema.
