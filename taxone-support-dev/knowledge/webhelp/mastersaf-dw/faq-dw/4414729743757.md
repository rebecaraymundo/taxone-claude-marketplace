# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 91871: O conteudo do campo Incentivo Cultural esta invalido.

## Passo-a-Passo

1\. Erro

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414743323789/mceclip2.png)

2\. O problema ocorre pois o campo IND_SIMPLES_NAC não possui algum dos valores pré-definidos no manual de layout:

Informar o Incentivo Cultural, com “S” ou “N”.

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414743314061/mceclip0.png)

3.Para solucionar o problema basta informar o valor "S" ou "N", no exemplo acima foi informado o valor "2"

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414736916237/mceclip1.png)
