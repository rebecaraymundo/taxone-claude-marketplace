## Problema:

O usuário enfrenta dificuldades com o sistema não enviando e-mails ao final do processo de informe de rendimentos e não gravando arquivos em formato PDF.

## Resolução:

Para resolver o problema, siga os passos abaixo:

  * Verifique se a versão do Java instalada é a Amazon Corretto 17.
  * Abra o arquivo JMsafDW.bat que está na pasta de executáveis do DW e adicione um pause na segunda linha, como mostrado:




    java -jar JMsafDW.jar %1 %2 %3 pause

  * Tente realizar o processo de geração novamente.
  * Ao final da geração, um Prompt de Comando (CMD) permanecerá aberto exibindo qualquer erro que possa ter ocorrido.
  * Envie um print da tela de erro para análise, se necessário.
