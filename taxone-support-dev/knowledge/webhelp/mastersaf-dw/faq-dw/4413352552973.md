# [Básicos][Job Servidor] DW - Importação de SAFX travada

- **Criado:** 2021-11-11
- **Atualizado:** 2023-05-24

---

Favor verificar se os seguintes parâmetros estão marcados:

**1 -** Parâmetro de DataMart automático para o estabelecimento em questão:

Ferramentas > Parâmetros do Sistema > Parâmetro de DataMart Automático (verifique se a primeira coluna está marcada)

**2 -** Processamento da importação em Multi Thread:

Ferramentas > Parâmetros do Sistema > Parâmetro de DataMart Automático (verifique se o campo Importação da SAFX01, 07, 08 e 09 por processamento em threads está marcado)

**Caso algum desses parâmetros esteja marcado** , solicite ao DBA que verifique o campo VALUE da seguinte consulta:


    select * from gv$parameter where name = 'job_queue_processes'

O ideal é que o valor no campo VALUE seja de pelo menos 2000.
