## Introdução

_"Esse artigo tem por objetivo mostrar a geração do parâmetro "Série Msaf x Série" para todos os Módulos Municipais que utilizam este De x Para._

## Passo-a-passo

O objetivo deste parâmetro é realizar o relacionamento entre as séries informadas no Documento Fiscal (SAFX07) e as séries solicitadas para geração dos validadores municipais.

**1 - Documento Fiscal**

Verificar qual Série foi informada no Documento Fiscal:

Módulo: |  Básicos > Data Warehouse
---|---
Menu: |  Manutenção > Documento Fiscal > Doc. Fiscal de Serviço

![95.PNG](https://suporte.thomsonreuters.com.br/hc/article_attachments/10859228635917)

**2 - Parâmetro _Série Msaf x Série"_**

Módulo: |  Municipal > Parâmetros para Municipio
---|---
Menu: |  Parâmetros > Série Msaf x Série

Selecionar a UF e o Municipio

![96.PNG](https://suporte.thomsonreuters.com.br/hc/article_attachments/10859385858573)

**Validade:** Informar a validade do relacionamento entre as séries. A validade é do tipo “a partir de”.

**Série MSAF:** Nesse campo o usuário irá digitar a série utilizada nas notas fiscais cadastradas no MasterSAF DW.

**Série:** Nesse campo o usuário irá selecionar as séries através de uma lista de acordo com o validador. O validador será identificado através do município previamente selecionado.
