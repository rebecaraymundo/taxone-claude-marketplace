## Introdução

Este artigo tem por finalidade demonstrar a geração de arquivos de integração no formato MOV0001 para integração de lançamentos contábeis para o Onesource ECF.

## Passo-a-passo

1\. Lançamentos Contábeis

Os registros de lançamentos contábeis são necessários para compor o arquivo TXT gerado.

Módulo: Data Warehouse

Caminho: Manutenção // Contabilidade // Diário Geral // Lançamentos Contábeis // Por Estabelecimento

3\. Geração dos registros:

Módulo: ECF - Escrituração Contábil

Caminho: Geração // Geração dos Dados![](http://www.mastersaf.com.br/HELPDW/V2R010/Conteudo/Demais_Empresas/Mastersaf_DW/DW/KBA_FAQ/Conex%C3%A3o_Onesource_BR/ECF/KBA_-_GERA%C3%87%C3%83O_MOV0001_3_01.gif)

Considerando a geração, poder ser realizada para inclusão ou exclusão. No sentido de geração de dados para exclusão, temos que nos ater ao principio de que para se gerar as informações de exclusão, primeiramente deve ser realizada a exclusão dos registros de lançamentos contábeis, os quais farão parte do arquivo gerado, para que possa ser gerado o arquivo com leiaute de exclusão.
