## Introdução

_"Esse artigo tem por objetivo mostrar o modelo de Arquivo de Licenciamento quando o Estabelecimento não é cadastrado no Município._

_Quando o validador possui a opção de geração Construção Civil/Utilities permite que os usuários desses ramos possam gerar o ISS a partir de um ou mais estabelecimentos que não são do município desse Validador, desde que licenciados corretamente para o município da mesma_

_O licenciamento será para o Município portador do Validador, mas o Estabelecimento do Municipio que será gerada a obrigação deverá ser identificado no XML_

## Modelo

No arquivo XML de licenciamento, devem constar os campos abaixo:

· Tag outras_inscricoes

· Código do Estabelecimento
