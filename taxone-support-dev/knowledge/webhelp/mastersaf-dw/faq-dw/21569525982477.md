Este artigo tem por objetivo demonstrar como corrigir o erro 91943: Codigo de Aglutinacao nao cadastrado

## Passo-a-Passo

1\. Erro

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21569531120013)

2\. O problema ocorre pois o código informado no campo COD_CONTA_AGLUT não está cadastrado na tabela X2102_CONTAS_AGLUT

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21569517382285)

3\. Para solucionar o problema é necessário realizar uma das opções abaixo:

3.1 Cadastrar o código na tela: SPED > ECD - ESCRITURAÇÃO CONTÁBIL DIGITAL > Manutenção > Códigos de Aglutinação (B. Patrimonial/D. Resultado/DLPA/DMPL)

3.2 Importar a SAFX2102

3.3 Verificar se o código informado está correto e então corrigir no arquivo

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21569558065037)
