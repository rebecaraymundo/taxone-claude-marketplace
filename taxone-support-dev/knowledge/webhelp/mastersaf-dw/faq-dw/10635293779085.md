## Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro 90163: O Campo Código NBM não está cadastrado.

## Passo-a-passo

**1)** Erro:

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10635032425485)
**2)** O problema ocorre, porque o campo COD_NBM está preenchido com um código que não está cadastrado no sistema.
![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10635060398349)

3) O Código do NBM deve ser cadastrado utilizando a SAFX2043 ou cadastrar o código manualmente na tela Básicos > Data Warehouse > Manutenção > Códigos > Código NBM:
![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10635286027405)
