## Introdução

Este artigo tem por objetivo demonstrar os principais dados que são verificados no documento fiscal, a fim de definir seu código de situação, no Campo 06 (COS_SIT) do Registro C100, envolvendo documento fiscal extemporâneo.

![mceclip6.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/11045395115917)

## Passo a passo

**1** \- Documento regular extemporâneo - COD_SIT = 01.

O principal critério que irá definir se um documento fiscal é extemporâneo, será o preenchimento do campo '**Dt. Extemp** '.

É muito importante consultar o [Manual de Leiaute SAFX](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Tecnicos/Manual_Layout.htm>), para correto preenchimento dos campos referentes às datas informadas no documento fiscal, bem como as particularidades definidas por cada UF, quanto aos documentos extemporâneos.

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/11045560497805)

**2** \- Documento cancelado extemporâneo - COD_SIT = 03.

Além do preenchimento do campo 'Dt. Extemp', este documento estará indicando que é cancelado, através da marcação do indicador '**Sit. Cancelado** '.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/11045602484493)

**3** \- Documento Fiscal Complementar extemporâneo - COD_SIT = 07.

Além do preenchimento do campo 'Dt. Extemp', este documento estará indicando a situação especial '**5 - NF Complementar ou CT-e de Complemento de Valores (modelo: 57)** '.

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/11045663730701)
