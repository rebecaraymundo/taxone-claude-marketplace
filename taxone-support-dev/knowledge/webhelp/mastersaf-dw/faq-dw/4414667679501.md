# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 93167: CPF do campo (CPF do Proprietario da Fazenda SP - Sao Paulo) invalido

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414659299597/mceclip0.png)

2\. O problema ocorre pois o campo CPF_SP está com um CPF incorreto para o estado de SP.

No exemplo abaixo está um CNPJ no lugar de um CPF.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414659306381/mceclip1.png)

3\. Existem 2 soluções para o problema, alterar o campo para **@** ou então cadastrar um CPF válido de São Paulo.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414652140685/mceclip2.png)
