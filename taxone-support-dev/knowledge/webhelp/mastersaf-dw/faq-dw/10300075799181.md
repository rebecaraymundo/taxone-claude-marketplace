## Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro Erro 90128: O campo Data de Emissão do documento não esta preenchido ou esta preenchido com zeros

## Passo-a-Passo

1.Erro:

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10300014842381)

2\. O erro ocorre pois o campo Data de Emissão do documento não esta preenchido ou esta preenchido com zeros.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10300030370061)

3\. Para solucionar, basta preencher o campo data de emissão corretamente AAAMMDD.
