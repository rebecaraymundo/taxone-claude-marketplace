## Problema:

O usuário precisa localizar onde os logs de cada usuário são armazenados no sistema Mastersaf e também necessita conhecer as URLs e portas que devem ser liberadas para as aplicações Mastersaf, a fim de evitar bloqueios de acesso à internet.

## Causa Raiz:

Os logs de usuários no sistema Mastersaf são armazenados dentro do banco de dados na tabela PL_AUD, e não em arquivos separados.  A aplicação não utiliza conexão com a internet, porém se o cliente utiliza a mensageria, será necessitará liberar as URLs específica para funcionamento correto.

## Resolução:

  * Acesse o menu Access Log Report dentro do Powerlock para gerar um relatório de acessos da aplicação, onde você encontrará as informações sobre logs na tabela PL_AUD.


  * Para as configurações de rede, caso utilize a mensageria (onesource) libere as seguintes URLs necessárias para comunicação externa:

ENVIO: <https://reinf.receita.economia.gov.br/recepcao/lotes>
CONSULTA: https://reinf.receita.economia.gov.br/consulta/lotes/
