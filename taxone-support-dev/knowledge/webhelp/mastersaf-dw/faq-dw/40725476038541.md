## **Problema**

Durante a geração do arquivo ISIMP, linhas duplicadas foram identificadas. O sistema gerou o mesmo registro como "regulado" e "não regulado" para o mesmo agente.

## **Causa Raiz**

O problema ocorreu devido à ativação do parâmetro de "Tratamento para agente regulado e não regulado" na tela de geração do arquivo. Esse parâmetro havia sido implementado previamente para atender demandas específicas.

## **Resolução**

Siga os passos abaixo para resolver o problema:

  1. Acesse a tela de geração do arquivo ISIMP.
  2. Identifique o parâmetro "Tratamento de agente regulado e não regulado".
  3. Desative esse parâmetro, caso não seja necessário para o seu processo.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40725448980749)

4\. Realize a geração do arquivo novamente.

5\. Verifique se as duplicidades foram eliminadas.

## **Próximos Passos**

Após realizar a correção, é recomendado:

  * Revisar os arquivos gerados para garantir que não há mais duplicidades.
  * Manter o monitoramento durante as próximas gerações de arquivos para identificar possíveis inconsistências.
