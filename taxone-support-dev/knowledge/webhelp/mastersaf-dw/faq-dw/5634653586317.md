Este artigo tem por objetivo demonstrar como corrigir o erro 90209: O Campo de Data de Validade com formato invalido.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5634580061709/mceclip0.png)

2\. O erro ocorre devido ao campo DATA_OPERACAO estar com o formato de data diferente de AAAAMMDD

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5634581149965/mceclip1.png)

3\. Para solucionar o problema basta informar a data no formato AAAAMMDD que o problema é solucionado

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5634598547853/mceclip2.png)
