# Introdução

Este artigo tem por objetivo demonstrar como resolver o erro 90088: O indicador de pessoa Física/Jurídica não esta preenchido ou esta preenchido com informação inválida.

## Passo-a-Passo

1\. Erro:

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/7992436189069/mceclip0.png)

2\. O problema ocorre pois o campo IND_FIS_JUR não esta preenchido ou esta com uma informação inválida. O campo deve ser preenchido com 1, 2, 3, 4 ou 5.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/7992495388813/mceclip1.png)

3\. Para resolver, basta preencher o campo com um valor valido (1, 2, 3, 4 ou5).
