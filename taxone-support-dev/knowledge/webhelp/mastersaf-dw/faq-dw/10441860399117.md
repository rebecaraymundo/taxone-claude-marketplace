Este artigo tem por objetivo demonstrar como corrigir o erro 90009: O Campo Data de Movimento nao esta preenchido ou preenchido com zeros.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10441784411277)

2\. O problema ocorre pois o valor informado no campo DATA_MOVTO está Nulo (@) ou preenchido com 00000000

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10441837402253)

3\. Para solucionar o problema basta informar no campo DATA_MOVTO uma data valida

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10441839067917)
