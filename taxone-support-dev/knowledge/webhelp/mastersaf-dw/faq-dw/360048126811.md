## Passo-a-passo

O Dashboard se encontra tela inicial do Mastersaf DW (Manager). A finalidade desta tela é demonstrar graficamente os status das importações das SAFX feitas pela rotina de Importação Batch.

Para utilizá-lo, basta parametrizar os dados que se deseja visualizar, conforme abaixo:

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415011665293/mceclip2.png)

![mceclip4.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415027150733/mceclip4.png)

![mceclip6.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415027163661/mceclip6.png)

**Obs.:** Lembrando que o Dashboard não exibe os dados das importações manuais, somente da Importação Batch.
