# Resolução de Problemas de Importação de Tabela no Job Servidor

## Problema:

Usuário enfrenta um erro ao tentar importar dados da tabela safx188 através do menu Job Servidor, recebendo a mensagem “Saldo Mensal não encontrado”.

## Causa Raiz:

O problema ocorre devido à divergência nas datas de operação entre a tabela safx188 e a tabela X02_SALDOS. As datas dessas tabelas devem estar alinhadas para permitir a importação correta dos dados.

## Resolução:

  * Acesse o menu Job Servidor e localize a tabela safx188.


  * Verifique a data de operação registrada na safx188.


  * Ajuste a data de operação na safx188 para que corresponda à data do saldo registrada na tabela X02_SALDOS.


  * Tente importar novamente os dados após ajustar as datas.


## Próximos Passos:

Após realizar os ajustes e a importação ter sido executada com sucesso, verifique se todos os dados foram importados corretamente. Monitore o sistema para garantir que não haja novas ocorrências do problema.

## Dicas e Recomendações:

É importante manter as datas de operação alinhadas entre as tabelas relacionadas para evitar erros de importação. Sempre verifique as datas antes de iniciar o processo de importação. Consulte o manual de layout para entender melhor como as tabelas estão relacionadas.
