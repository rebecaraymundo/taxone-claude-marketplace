## Problema:

Usuário enfrenta erro ao tentar salvar o relatório de Imposto de Renda (IR) em formato PDF.

## Causa Raiz:

O problema ocorre devido à presença de múltiplas versões do Java instaladas no sistema (Java 11 e Java 17), causando conflitos no reconhecimento da versão adequada para a execução do software.

## Resolução:

  * Identifique todas as versões do Java instaladas no seu sistema.
  * Desinstale a versão mais antiga do Java (neste caso, Java 11).
  * Caso necessite das duas versões, realizar o apontamento do DW para o Java 17.
  * Reinicie o sistema para garantir que as alterações surtam efeito.
  * Tente salvar o relatório de IR em PDF novamente.


## Próximos Passos:

Após seguir os passos de resolução, verifique se o relatório de IR pode ser salvo em PDF corretamente. Caso o problema persista, entre em contato com o suporte técnico para assistência adicional.

## Dicas e Recomendações:

Mantenha seu sistema sempre atualizado e verifique regularmente se existem múltiplas versões de softwares que possam causar conflitos. Isso ajudará a evitar problemas semelhantes no futuro e garantirá o bom funcionamento de suas aplicações.
