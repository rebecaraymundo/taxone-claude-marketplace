Este artigo tem por objetivo demonstrar como corrigir o erro 93600: Campo Data da Escrituracao Contabil deve ser informada

## Passo-a-Passo

1\. Erro

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21505278312973)

2\. Esse erro ocorre pois o campo IND_NAT_REND está com o código 12052 e o campo DATA_ESCR_CONT está nulo ou em branco

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21505316377485)

3\. Para solucionar o problema basta informar uma data valida, no formato AAAAMMDD, no campo DATA_ESCR_CONT

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21505323522317)
