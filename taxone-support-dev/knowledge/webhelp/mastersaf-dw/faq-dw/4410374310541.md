## Introdução

Este artigo tem a finalidade de demonstrar a parametrização que permite salvar os arquivos TXT gerados em um diretório previamente parametrizado.

## Passo-a-passo

1\. Parametrizar os sinônimos de banco de dados

Solicitar do DBA responsável pelo banco de dados do MasterSAF DW que realize a parametrização de sinônimos, a fim de garantir a comunicação com as tabelas de integração no banco de dados do Onesource ECF, conforme descrito no manual de configuração do Onesource ECF:
