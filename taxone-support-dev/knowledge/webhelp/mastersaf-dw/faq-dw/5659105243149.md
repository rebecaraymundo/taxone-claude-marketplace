Este artigo tem por objetivo demonstrar como corrigir o erro 90383: O Campo Codigo de Historico Padrao nao esta cadastrado

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5659056922381)

2\. O problema ocorre pois o código informado no campo HISTPADRAO não está cadastro na tela Data Warehouse > Manutenção > Códigos > Histórico Padrão

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5659059110541)

3\. Para solucionar o problema basta informar um código valido no arquivo TXT

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5659126815757)

Ou cadastrar o código informado no arquivo TXT na tela Data Warehouse > Manutenção > Códigos > Histórico Padrão

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5659077975437)
