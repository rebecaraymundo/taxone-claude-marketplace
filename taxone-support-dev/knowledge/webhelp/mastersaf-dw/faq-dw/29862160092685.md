## Problema:

O usuário precisa restringir o acesso a alguns menus, exemplo no módulo Data Warehouse, especificamente os menus Carga e Importação, sem desabilitar o módulo Controle para certos usuários.

## Causa Raiz:

A necessidade de restringir o acesso a certos menus surge de uma política interna de segurança para controlar o acesso a informações sensíveis, sem limitar completamente o acesso ao sistema.
