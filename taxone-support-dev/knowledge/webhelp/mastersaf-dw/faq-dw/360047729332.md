# [SPED][SPED Contábil]DW - Moeda Funcional do Registro 0000

- **ID:** 360047729332
- **URL:** https://suporte.thomsonreuters.com.br/hc/pt-br/articles/360047729332--SPED-SPED-Cont%C3%A1bil-DW-Moeda-Funcional-do-Registro-0000
- **Criado:** 2020-08-18
- **Atualizado:** 2023-11-01

---

Para o preenchimento do campo 19 (IDENT_MF) do registro 0000 referente à Moeda Funcional.

É necessário acessar o módulo ECD, menu Parâmetros > Dados Iniciais, selecionar o estabelecimento centralizador, após será possível definir se é utilizado ou não os valores em moeda funcional, a partir da opção 'ECD com Movimentação em Moeda Funcional'.
