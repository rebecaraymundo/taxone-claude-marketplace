## Introdução

Esse artigo tem por objetivo mostrar uma das soluções para o erro ORA-0001: Restrição exclusiva (PK_JOB_IMPORTACAO) violada.

## Passo-a-passo

Basta executar o script abaixo:


    UPDATE IDENT_SUBSTITUTO
    SET PROX_IDENT_SUBST = (SELECT MAX(NUM_JOB) + 1 FROM JOB_IMPORTACAO)
    WHERE NOME_COLUNA = 'NUM_JOB'
    AND NOME_TABELA = 'JOB_IMPORTACAO';
