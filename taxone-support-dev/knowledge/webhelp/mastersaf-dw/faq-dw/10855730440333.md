Introdução:

Este artigo tem por objetivo demonstrar a correção para o seguinte erro:

Erro para gerar o livro de entradas ICMS - Não existem estabelecimentos cadastrados para esta obrigação
Menu DataMART -> Emissão Livro de Entrada -> Modelo P1A -> Ajuste SINIEF



Palavras-Chave

Apuração ICMS, P1A, Livro 103

Evidências

N/A



## Causa-Raiz

Geração do Livro de ICMS Tipo 103 Registros de Entrada Modelo P1A

## Solução

Caso seja apresentada a mensagem de erro abaixo:

![1-Geracao_do_livro_data_mart.PNG](https://suporte.thomsonreuters.com.br/hc/article_attachments/10854679404045)

![19B0FE17.PNG](https://suporte.thomsonreuters.com.br/hc/article_attachments/10854703041165)

Deve ser realizado os seguintes passos:

**1** \- Parametrizar as**Obrigações Fiscais**

**Modulo Estadual > Controle de Obrigações Estaduais**

**![3-_Controle_de_obriga__es_.jpeg](https://suporte.thomsonreuters.com.br/hc/article_attachments/10854849933837)**

Acesse a aba **Obrigações / Obrigações Fiscais / Por Estabelecimentos**

**![4-_Controle_de_obriga__es_Por_estabeleicmento_.jpeg](https://suporte.thomsonreuters.com.br/hc/article_attachments/10854883629069)**

![Obrigacao_Fiscal_por_estabelecimento.PNG](https://suporte.thomsonreuters.com.br/hc/article_attachments/10855567277581)

**2** \- Na sequência parametrizado o **Calendários fiscal.**

**Obrigações / Calendario de Obrigações/ Por Estabelecimentos**

**![5-_Calendario_de_Obriga__es.jpeg](https://suporte.thomsonreuters.com.br/hc/article_attachments/10854948500109)**

Parametrize o **livro 103 - Registro de Entradas -Modelo P1A**

![Calendario_Fiscal.PNG](https://suporte.thomsonreuters.com.br/hc/article_attachments/10855569851533)

Após este procedimento gere novamente o Livro Modelo P1A através do **Modulo ICMS / DATA MART/ Emissão Livros de Entrada /Modelo P1A /Ajuste SINIEF/** e este será gerado corretamente.

![Livro_P1A.PNG](https://suporte.thomsonreuters.com.br/hc/article_attachments/10855631172493)
