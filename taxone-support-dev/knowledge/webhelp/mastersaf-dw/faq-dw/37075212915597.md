# [SPED][SPED Contábil]DW - Resolução de Problemas com Encerramento Trimestral no Mastersaf DW

Problema:

O cliente enfrenta dificuldades para encerrar as contas no término de cada trimestre no software Mastersaf DW.

Causa Raiz:

A causa do problema é a configuração inadequada da data de encerramento para o exercício social e a geração do SPED trimestralmente.

Resolução:

Acesse o módulo de SPED Contábil na aplicação Mastersaf DW.

Navegue até **Parâmetros** e **Dados Iniciais**.

Confirme se o parâmetro "Dia e Mês do Encerramento do Exercício Social" está definido corretamente para cada trimestre, e não apenas para o final do ano (31/12).

Ajuste a data para cada trimestre necessário (31/03, 30/06, 30/09, 31/12).

Após ajustar, vá em **Geração** , selecione **Meio Magnético** e depois **Gerar SPED**.

Confirme a seleção da opção de geração trimestral durante o processo.

Próximos Passos:

Após realizar os procedimentos de encerramento trimestral, monitorar a correta geração dos arquivos SPED e verificar se todos os dados estão corretamente registrados. Caso haja inconsistências, repetir o processo de ajuste dos parâmetros de datas.

Dicas e Recomendações:

É importante sempre verificar e ajustar as datas de encerramento antes do encerramento de cada trimestre para garantir que a geração do SPED seja precisa. Verifique sempre a última versão do software para evitar problemas operacionais. Consulte os documentos de suporte sobre o encerramento trimestral para mais detalhes.
