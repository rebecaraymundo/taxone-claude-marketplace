# Introdução

Este artigo visa demonstrar como retificar os eventos R-4010/R-4020 - Cenário de Juros sobre Capital (JCP)

# Passos

Deve ser importado novo arquivo XML*, através da funcionalidade de Importação de Registros XML Família 4000 (em SPED > EFD-REINF > REINF > Arquivos Externos > Importação), de um evento importado e já enviado para o FISCO .

Após a importação, efetuar a geração do arquivo retificador através da tela de Painel de Controle de Eventos (SPED > EFD > REINF > Painel > Painel de Controle de Eventos), selecionando o evento original através do botão “Reprocessar Evento”.

*O arquivo XML importado deve conter as mesmas informações do arquivo original (mesmo evento, período e beneficiário), porém o campo ID deve ser diferente do anterior.
