# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 90009:O Campo Data de Movimento nao esta preenchido ou preenchido com zeros.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5634421070861/mceclip0.png)

2\. O erro ocorre devido ao campo DATA_OPERACAO estar com valor @, em branco ou 00000000

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5634384013069/mceclip1.png)

3\. Para solucionar o problema basta informar uma data no formato AAAAMMDD.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5634462334733/mceclip2.png)
