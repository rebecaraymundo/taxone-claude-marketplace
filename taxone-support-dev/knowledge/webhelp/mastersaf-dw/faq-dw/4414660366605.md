# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 91167: Classe de Empresa deve ser preenchido com <01> ou <02> ou <03> ou <04>

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414668704013/mceclip0.png)

2\. O problema ocorre pois o campo IND_CLASSE_EMP não possui algum dos valores pré-definidos no manual de layout:

Informar a Classe de Empresa, de acordo com:
01 - EPP (Empresa de Pequeno Porte);
02 - Pequena Empresa;
03 - Empresa Normal;
04 – Microempresa (ME);
05 – MEI (Micro Empreendedor Individual).

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414653276173/mceclip1.png)

3\. Para solucionar o problema basta informar um dos 5 códigos pré-definidos no campo, no exemplo acima o código informado no arquivo não está dentro dos 5.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414668706701/mceclip2.png)
