**Introdução**

_"Esse artigo tem por objetivo mostrar os passos para demonstrar quando há valores apurado de IPI no SPED Fiscal. O Registro E510 deve ser preenchido com os valores consolidados do Imposto sobre Produtos Industrializados (IPI), de acordo com o período de apuração"._

**Solução**

Quando não realizada a apuração do IPI, o registro com detalhamento dos valores não é apresentado;

|E500|0|01112022|31112022|
|E520|0|9400,03|1446,94|0|0|0|7953,09|
|E990|20|

A correção é realizada conforme passo a passo a seguir;

Módulo |  Federal - IPI
---|---
Menu: |  Data Mart > Apuração do IPI

Informe os dados do Estabelecimento de Geração para identificar o período:
