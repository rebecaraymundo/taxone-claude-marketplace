## Introdução

_"Esse artigo tem por objetivo mostrar a geração do parâmetro "Tipo Docto Msaf x Tipo Docto" para todos os Módulos Municipais que utilizam este De x Para._

## Passo-a-passo

O objetivo deste parâmetro é realizar o relacionamento entre os tipos de documento informados no documento fiscal e os tipos de documento solicitados para geração dos validadores.

**1 - Documento Fiscal**

Verificar qual Tipo de Documento foi informada no Documento Fiscal:

Módulo: |  Básicos > Data Warehouse
---|---
Menu: |  Manutenção > Documento Fiscal > Doc. Fiscal de Serviço

![50.PNG](https://suporte.thomsonreuters.com.br/hc/article_attachments/10859654585101)

**2 - Parâmetro _Tipo Docto Msaf x Tipo Docto "_**

Módulo: |  Municipal > Parâmetros para Municipio
---|---
Menu: |  Parâmetros> _Tipo Docto Msaf x Tipo Docto_

Selecionar a UF e o Municipio

![51.PNG](https://suporte.thomsonreuters.com.br/hc/article_attachments/10859735026957)

**Validade:** Informar a validade do relacionamento entre os tipos de documentos. A validade é do tipo “a partir de”. Essa data também será utilizada para buscar o grupo do tipo de documento.

**Tipo Documento MasterSAF:** Nesse campo o usuário irá selecionar os tipos de documentos utilizados nas notas fiscais cadastradas no MasterSAF DW.

**Tipo Documento:** Nesse campo o usuário irá selecionar através de uma lista os tipos de documento solicitados para a geração do validador. O validador será identificado através do município previamente selecionado.
