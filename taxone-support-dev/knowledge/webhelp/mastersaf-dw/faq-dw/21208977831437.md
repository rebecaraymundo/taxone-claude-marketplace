# Introdução

Este artigo tem como objetivo demonstrar como inserir um novo código de receita.

# Módulo e Menu

  * _Modulo_** _Básicos > Ferramentas_**
  * _Menu_** _Tabelas Internas > Manter > Tabelas_**


# Passos

_No modulo e menu acima citados, pesquise pela tabela "_**_código de receita_** _"._

![](https://t12927461.p.clickup-attachments.com/t12927461/09cebacb-f15d-4bb1-b5b3-e699acccd7db/image.png)

![](https://t12927461.p.clickup-attachments.com/t12927461/c36ef5bd-ebd0-4957-8d06-ec3a6cc4f5d9/image.png)

_Clique no botão "_**_novo_** _" no canto superior esquerdo._

![](https://t12927461.p.clickup-attachments.com/t12927461/618dcbe6-286e-445d-aacb-7b7672024914/image.png)

_A nova linha para digitação aparecerá em branco_

![](https://t12927461.p.clickup-attachments.com/t12927461/c10f63db-4b79-46ac-a3fe-fee8f7c7be19/image.png)

_Preencha o código de receita e a descrição. Neste exemplo estamos incluindo o código 596007._

![](https://t12927461.p.clickup-attachments.com/t12927461/06b4a256-2f7f-4b64-a3bd-e780e9409872/image.png)

_Preencha o código do tributo desejado_

![](https://t12927461.p.clickup-attachments.com/t12927461/f9feacd3-82d5-4368-94e6-463d8d9dcf77/image.png)

_Coloque a periodicidade desejada_

![](https://t12927461.p.clickup-attachments.com/t12927461/b549c07c-590e-48f6-a792-101cd1f669da/image.png)

_Preencha o campo "_**_inicio validade_** _" e deixe o campo "_**_fim validade_** _" em branco_

![](https://t12927461.p.clickup-attachments.com/t12927461/5661cac9-1072-418a-b7d9-48131b396a1b/image.png)

_Preencha os campos abaixo conforme desejado_

![](https://t12927461.p.clickup-attachments.com/t12927461/f4d8ffcd-3869-410c-9c85-4f78f1a466b2/image.png)

_Após preenchimento, clique no botão "_**_confirma_** _" no canto superior esquerdo._

![](https://t12927461.p.clickup-attachments.com/t12927461/90877376-8d24-4c8e-98f6-c165c420fde7/image.png)
