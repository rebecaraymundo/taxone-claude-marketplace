Este artigo tem por objetivo demonstrar como corrigir o erro 91865: Entidade Responsavel deve ser preenchida

## Passo-a-Passo

1\. Erro

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21278946890637)

2\. O problema ocorre pois o campo COD_ENTIDADE_RESP não está preenchido ou está com informação nula (@)

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21278938555021)

3\. Para solucionar o problema basta ajustar o campo COD_ENTIDADE_RESP no arquivo txt com um valor valido, se a VERSAO_REF for igual a 1.00 os códigos devem ser: '00', '10', '20'

se a VERSAO_REF for igual a 3.00 os códigos devem ser: '00', '10', '20','1', '2','3','4','5','6','7','8','9'

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21278930262285)
