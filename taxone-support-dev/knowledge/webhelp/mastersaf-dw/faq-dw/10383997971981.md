Este artigo tem por objetivo demonstrar como corrigir o erro 90110: O Indicador de Debito/Credito do Saldo Final nao confere

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10383923020045)

2\. O problema ocorre pois o valor informado em IND_SALDO_FIM não reflete o resultado do calculo

VLR_SALDO_FIM = VLR_SALDO_INI + VLR_TOT_DEB - VLR_TOT_CRE

**Se o resultado for negativo o indicado deve ser "C", se o resultado for positivo o indicador deve ser "D"**

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10383923307917)

3\. Para solucionar o problema basta informar no campo IND_SALDO_FIM = "C" ou IND_SALDO_FIM = "D"

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10383908347917)
