Este artigo tem por objetivo demonstrar como corrigir o erro 90477: Grupo de Centralizacao de Codigo de Historico Padrao nao esta preenchido ou invalido na data ou nao esta cadastrado ou Data nao preenchida.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5658894526861)

2\. O problema ocorre pois o campo HISTPADRAO está preenchido, porém não existe relacionamento de Tabela x Grupo para o estabelecimento informado na SAFX

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5658895450381)

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5658885230093)

3\. Realizar o relacionamento entre Tabelas x Grupos para a tabela Histórico Padrão no menu Parâmetros > Funcionais > Relac. entre Tabelas e Grupos

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5658924090509)
