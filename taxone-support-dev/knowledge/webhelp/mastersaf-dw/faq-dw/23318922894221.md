Objetivo desse artigo é manter ativo o comunicado referente a atualização necessária da versão Java para o funcionamento correto da aplicação MastersasDW.

A partir da release******290.0.0****(05/02/2024),** liberaremos a Atualização do produto com o Java 17 (Amazon Corretto 17).

Clientes ** _on-premise_****** que por ventura ainda não estejam atualizados, precisam obrigatoriamente se adequar com a atualização Java 17.

Ressaltamos que a não atualização para o JAVA 17 pode impactar diretamente a atualização de seu patch nas funcionalidade de:

  * **_Tabelas Fixas/Acessórias_**
  * **_Integração com LDAP_**
  * **_Validador FINTELISS_**** _
_**
  * **_Relatórios de Conferência REINF_**
  * **_Relatórios de Tributos Federais_**
  * **_Informe de Rendimentos_**


O objetivo desta evolução é mitigar problemas relacionados à segurança, além de promover uma melhor experiência de nossas soluções.
