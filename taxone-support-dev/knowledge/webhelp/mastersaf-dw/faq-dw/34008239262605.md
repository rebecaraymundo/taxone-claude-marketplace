Após a aplicação de Patch, é possível que o DW apresente a mensagem abaixo:

![image \(44\).png](https://atendimentotr.zendesk.com/attachments/token/g0T0ddWtiHyuOTCBz7ouHXaVv/?name=image+%2844%29.png&lotus_request=true)

Neste caso, o cliente usa Citrix para acesso as bases.

A seguir, estão os passos solicitados ao cliente antes de abrir demanda para DEV:

1º Substituição de executáves;

2º Executar o manager.exe que está na mesma pasta dos executáveis;

3º Verificar se o pl_admin.exe existe no diretório de executáveis;

4º Realizar o processo de TRACE.

Após isso, acionar o DEV, que provavelmente vai solicitar o seguinte:

"Solicitar para o cliente executar direto pela pasta de Executáveis do DW ou solicitar a configuração correta do Atalho no Citrix do Manager do DW."
