Este artigo tem por objetivo demonstrar como corrigir o erro 90468: Nao foi possivel recuperar para o estabelecimento o grupo de centralizacao do Codigo de Operacao.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5658715672333)

2\. Esse erro ocorre pois o campo COD_OPERACAO está preenchido e para o estabelecimento informado na SAFX não existe relacionamento de tabelas x grupos para a tabela de "Operação"

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5658747187213)

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5658752961677)

3\. Solução é associar a tabela "Operação" para o estabelecimento que está importando a SAFX no menu Parâmetros > Funcionais > Relac. entre Tabelas e Grupos

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/5658744875277)
