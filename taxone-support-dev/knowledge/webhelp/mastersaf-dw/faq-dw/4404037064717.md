## Introdução

_"Essa FAQ tem por objetivo demonstrar a funcionalidade do Parâmetro**'** 'Utilizar Somente DATA MART para períodos anteriores'' na Geração dos Movimentos de Entradas?"_

## Passo-a-passo

Módulo: Estadual > Ressarcimento de ICMS-ST PR

Menu: Geração > NPF 003/20 > Geração dos Movimentos de Entradas

Através desta opção é possível definir como será feita a recuperação dos dados de notas de períodos anteriores.

Se Marcado o sistema irá recuperar os dados de notas de períodos anteriores, apenas com base no DATA MART (períodos equalizados);

Se Desmarcado: A recuperação será com base no DATA MART, porém quando não existirem mais dados, e a pesquisa não atingir a quantidade total das saídas, e nem a data limite informada, então serão recuperados os dados das tabelas “normais” (tabelas X07/X08).
