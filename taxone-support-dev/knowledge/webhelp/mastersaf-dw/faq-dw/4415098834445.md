# Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro 90386: Data de movimento informada no registro esta fora do periodo definido para importação.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415098774285/mceclip0.png)

2\. O problema ocorre pois a data contida na SAFX esta fora do periodo definido no JOB SERVIDOR.

Veja que no arquivo a data que consta é 11/04/2020

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415104568333/mceclip1.png)

E no JOB SERVIDOR a data esta 01/01/2021 a 30/10/2021

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415109354381/mceclip2.png)

3\. Para corrigir o problema, basta informar um periodo que compreenda a data informada na safx.

Exemplo: 01/01/2000 - 30/10/2021
