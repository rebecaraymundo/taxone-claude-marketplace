# Introdução

Este artigo tem como objetivo demonstrar como gerar ou retirar o Relatório de Associação das Aglutinações.

# Passos

## Modulo e Menu

_Modulo: SPED > ECD_

_Menu: Manutenção > Códigos de Aglutinação (B. Patrimonial/D. Resultado/DLPA/DMPL)_

_Escolher a aba do tipo de Demonstração Financeira desejada, utilizar o botão "_**__Relatório__** _" em destaque no print abaixo._
