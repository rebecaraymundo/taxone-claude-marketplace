# [Básicos][Job Servidor] DW - SAFX279- Erro 93261: O campo Natureza do Rendimento deve ser preenchido com uma das opcoes validas: 19001 ou 19009

Este artigo tem por objetivo demonstrar como corrigir o erro 93261: O campo Natureza do Rendimento deve ser preenchido com uma das opcoes validas: 19001 ou 19009

## Passo-a-Passo

1\. Erro

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21496835322637)

2\. Esse erro ocorre pois o código informado no campo IND_NAR_REND não está cadastrado na tela BÁSICOS > FERRAMENTAS > Tabelas Internas > Manter > Tabelas > Tabela de Natureza de Rendimentos (EFD-REINF)

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21496827870349)

3\. Para solucionar o problema realize a importacao da tabela TACES101- Tabela de Natureza de Rendimentos (EFD-REINF) ou então cadastre na tela BÁSICOS > FERRAMENTAS > Tabelas Internas > Manter > Tabelas > Tabela de Natureza de Rendimentos (EFD-REINF)

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21496820934157)
