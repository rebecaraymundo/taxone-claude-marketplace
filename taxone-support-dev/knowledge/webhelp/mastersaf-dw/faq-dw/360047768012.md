## Introdução

Neste artigo veremos como sanar o erro _ORA-29532: chamada Java finalizada por exceção Java não detectada: oracle.aurora.vm.IdNotFoundException: [...] user or role id does not exist_.

####

## Causas

A principal causa desse erro é a falta do componente Java no banco de dados. A crítica geralmente ocorre durante a atualização dos patches/releases.

## Solução

Solicitar que ao DBA que instale o componente Java no banco de dados Oracle (há a possibilidade de o mesmo estar apenas desabilitado) em seguida, realizar novamente o procedimento de atualização.
