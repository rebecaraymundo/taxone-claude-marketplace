Este artigo tem por objetivo demonstrar como corrigir o erro 93186: O campo Data de Lancamento Extemporaneo deve ser preenchido apenas quando o indicador de Lancamento for igual a <X> \- Lancamento Extemporaneo

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10365587117581)

2\. O problema ocorre pois o código informado no campo TIPO_LANCTO está igual a "N - Normal" e o campo DAT_LANCTO_EXTEMP está com data preenchida

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10365612750221)

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10365613277197)

3\. Para solucionar o problema deve-se verificar se o lançamento realmente é normal e então retirar a data no campo DAT_LANCTO_EXTEMP, caso contrario deve-se alterar o campo TIPO_LANCTO para "X - Extemporâneo"

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10365592583437)

![mceclip4.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10365593135117)
