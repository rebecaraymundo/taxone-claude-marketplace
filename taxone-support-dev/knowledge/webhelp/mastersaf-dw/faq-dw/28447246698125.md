## Problema:

O usuário precisa reportar para a auditoria uma relação detalhada de atividades dos usuários no sistema, incluindo alterações em notas fiscais e geração de guias para pagamento, mas não conseguiu encontrar essa funcionalidade no sistema atual.

## Causa Raiz:

A funcionalidade atual do sistema não suporta o detalhamento de atividades por usuário até o nível desejado. É possível acessar informações a nível de módulo, mas não as modificações específicas em campos de dados.

## Resolução:

  * Acesse a tela de configuração para ativar a opção de rastreamento de notas.


  * Ative esta opção para que ela comece a rastrear as alterações feitas a partir deste momento.


  * Considere que essa funcionalidade não grava todos os campos alterados, apenas os principais.


  * Para uma solução mais customizada, entre em contato com uma consultoria homologada que possa desenvolver um script ou outra solução no banco de dados para atender a necessidade específica.


## Próximos Passos:

Após ativar o rastreamento de notas, monitore as alterações para garantir que o rastreamento está funcionando conforme o esperado. Se precisar de uma solução mais detalhada, inicie o contato com uma consultoria homologada para discutir opções customizadas.

## Dicas e Recomendações:

Para evitar futuros problemas com auditorias, considere manter a funcionalidade de rastreamento de notas sempre ativa. Além disso, faça revisões periódicas das configurações do sistema para verificar se há novas atualizações ou funcionalidades que possam ser benéficas para a gestão de auditorias internas.
