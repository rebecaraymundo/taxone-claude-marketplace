# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 90063: Numero de CPF informado nao e valido, conforme regras da Receita Federal.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414436429453/mceclip0.png)

2\. O problema ocorre pois o CPF do cliente está incorreto, no exemplo abaixo o campo IND_CONTEM_COD está como 2 indicando que o conteúdo é um CPF, porém se analisarmos o arquivo veremos que o conteúdo do campo é um CNPJ.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414455329165/mceclip1.png)

3\. Neste é existem duas soluções, é necessário verificar qual conteúdo é o correto e alterar o outro, se o CNPJ está correto o campo IND_CONTEM_COD deve ser alterado para 3 ou 1

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414448971149/mceclip2.png)

Caso o campo IND_CONTEM_COD esteja correto, é necessário ajustar o campo CPF_CGC

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414448979725/mceclip3.png)
