**Introdução**

Este artigo tem por objetivo demonstrar os principais pontos para geração dos registros K010, conforme o Guia Prático EFD ICMS IPI vigência a partir de janeiro/2023.

**Solução**

Módulo: |  SPED > EFD - Escrituração Fiscal Digital
---|---
Menu: |  Parâmetros > Dados Iniciais

Ao acessar os Dados Iniciais > Bloco K - Controle da Produção e do Estoque, certifique o campo selecionado para geração.

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10917361912205)

Indicador de tipo de leiaute adotado:

0 - Leiaute simplificado

1 - Leiaute completo

2 - Leiaute restrito aos saldos de estoque

Este registro indica o tipo de leiaute que o contribuinte adotou na informação para geração do bloco K.

|K001|0|

|K010|0|

|K100|01012023|31012023|

Campo 01 (REG) - Valor Válido: [K010]

Validação: registro obrigatório se o campo 02 (IND_MOV) do registro K001 estiver informado com “0 - Bloco com dados informados”, ou seja, se; |K001|0| o registro K010 é obrigatório.
