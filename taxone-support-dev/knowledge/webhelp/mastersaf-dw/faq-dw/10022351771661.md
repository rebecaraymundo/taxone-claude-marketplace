# Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro 91287: O campo Data da DDE/DSE é inválido

## Passo-a-Passo

1.Erro:

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10022123802765)

2\. O problema ocorre pois o campo Data da DDE/DSE não esta preenchido ou esta preenchido incorretamente:

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10022210062093)

3\. Para resolver, basta preencher o campo com uma data válida.
