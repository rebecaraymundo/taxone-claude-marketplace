# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 91453: O campo Indicador de Fome Zero deve ser nulo, @ , S ou N .

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414660684813)

2\. O problema ocorre pois o campo IND_FOME_ZERO não possui algum dos valores pré-definidos no manual de layout:

Informar o Participante do Programa Fome Zero, de acordo com:
S - Pessoa Física Jurídica participa do Programa Fome Zero;
N - Pessoa Física Jurídica não participa do Programa Fome Zero.
Atendimento ao Ato COTEPE nr 70/05 - Registro C200.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414672922253)

3\. Para solucionar o problema basta informar o valor "S" ou "N", no exemplo acima foi informado o valor "1"

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414660689933)
