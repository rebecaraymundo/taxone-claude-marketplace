Introdução

**Este artigo tem como objetivo demonstrar como vincular o cadastro do Perfil A ou B no Estabelecimento para a geração da EFD SPED ICMS/IPI.**

Após ter realizado o cadastro do Leiaute e definido o Perfil de geração, é necessário realizar o vinculado deste cadastro no estabelecimento para geração do arquivo magnético.
 _Obs. É possível realizar o vinculo em mais de um estabalecimento._

Acessando; Módulo e Menu

Acessar Modulo: SPED > EFD - ESCRITURAÇÃO FISCAL DIGITAL

Acessar Menu Parâmetros > Perfil por Estabalecimento

O Sistema vai abrir a opção para selecionar o Perfil (Cod./Desc.) cadastrado.

No exemplo abaixo estarei utilizando o perfil cadastrado.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21712314933389)

Logo, será habilitada a caixa para selecionar os ESTABELECIMENTOS para vincular o PERFIL de geração;

Após, clique em CONFIRMAR e em seguida poderá realizar a GERAÇÃO da EFD SPED ICMS/IPI.

Ao realizar a geração para o estabelecimento que foi vinculado, o PERFIL será exibido;
