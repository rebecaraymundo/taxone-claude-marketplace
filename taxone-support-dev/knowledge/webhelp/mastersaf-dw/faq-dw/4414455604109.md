# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 90084: O Numero de CGC/CPF nao deve ser preenchido para este Registro

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414436496013/mceclip0.png)

2\. O problema ocorre pois o campo IND_CONTEM_COD está com o valor "4 - Se a Pessoa Física/Jurídica não tenha CPF/CGC - casos especiais." e o campo CPF_CGC está preenchido, conforme o exemplo abaixo

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414436542989/mceclip1.png)

3\. Nesse caso é necessário avaliar qual informação está correta, caso o IND_CONTEM_COD está incorreto deve-se aplicar um dos outros 3 códigos conforme o manual de layout:

1 - Se no Campo de Código (02) o conteúdo for um Código para Pessoa Física/Jurídica que tenha CGC/CPF;
2 - Se no Campo de Código (02) o conteúdo é um CPF;
3 - Se no Campo de Código (02) o conteúdo é um CGC;

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414455533197/mceclip2.png)

Caso o campo CPF_CGC está incorreto, basta alterar para **@** ou deixar em branco

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414436717581/mceclip3.png)
