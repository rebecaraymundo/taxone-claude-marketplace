# Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro 91937: Expressão e Ordem finalizada sem ponto-e-virgula ;

## Passo-a-Passo

1\. Erro

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414728585741/mceclip1.png)

2\. O problema ocorre pois o campo EXPRESSAO_ORD esta finalizando com outro caracter que não é ponto-e-virgula ;

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414742139917/mceclip2.png)

3\. Para solucionar, basta retirar o caractere que esta finalizando a expressão e colocar ponto-e-virgula.

Nesse caso, o campo finalizava com aspas, bastou retirar as aspas.
