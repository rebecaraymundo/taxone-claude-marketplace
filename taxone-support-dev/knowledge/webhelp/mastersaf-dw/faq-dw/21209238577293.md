# Introdução

Este artigo tem como objetivo demonstrar como gerar o Registro C870 do EFD Contribuições.

# Módulo e Menu

  * _Modulo_** _SPED > EFD Contribuicoes_**
  * _Menu_** _Parametros > Dados Iniciais_**
  * _Modulo_** _Básicos > Data Warehouse_**
  * _Menu_** _Manutencao > Cupom Fiscal Eletronico > Capa e Detalhe_**


# Passos

_Em Básicos > Manutenção > Cupom Fiscal Eletrônico verifique se na aba detalhes do cupom fiscal, estão preenchidos os campos Valor Base PIS, Valor PIS, CST PIS, Valor Base COFINS, Valor COFINS, CST COFINS._

![](https://t12927461.p.clickup-attachments.com/t12927461/d1c92d2d-348c-4b92-a8e7-bd52119636df/image.png)

_No modulo SPED > EFD Contribuições > Parâmetros, verifique e marque o parâmetro "Demonstrar Registros de Detalhamento por Código de Item" em Dados iniciais e salve._

![](https://t12927461.p.clickup-attachments.com/t12927461/734d7b83-8029-4799-99c0-0e42702e63a0/image.png)

_Efetue a Geração dos Registros e Meio Magnético e verifique se o Registro C870 foi criado._

![](https://t12927461.p.clickup-attachments.com/t12927461/bf9d4225-9875-46ab-8667-727be46c7e07/image.png)
