# Regras aplicáveis na geração do campo 17 IND_FRT no registro C100.

## Problema

Um usuário reportou que a modalidade de frete para uma nota fiscal específica está divergente entre o arquivo SPED e o sistema SAP. No SPED, a modalidade aparece como "9", enquanto no SAP está registrada como "3".

## Causa Raiz

A discrepância ocorre devido à forma como o sistema DW (Data Warehouse) está mapeando a modalidade de frete para o SPED.

Se a data fiscal do documento for maior ou igual á “01/01/2012” então gerar o campo 17 (IND_FRT) do registro C100 conforme a regra:


Se a modalidade do frete (Campo 72 da SAFX07) for = 1 (Frete por conta do Emitente), então gerar o registro 17 do SPED Fiscal com o valor “0”.
Se a modalidade do frete (Campo 72 da SAFX07) for = 2 (Frete por conta do Destinatário), então gerar o registro 17 do SPED Fiscal com o valor “1”.
Se a modalidade do frete (Campo 72 da SAFX07) for = 0 (Outros), então gerar o registro 17 do SPED Fiscal com o valor “2”.
Para a modalidade de frete (Campo 72 da SAFX07) = 9 (Sem cobrança de Frete), continuar gerando SPED Fiscal com o valor “9”.

Para as opções 3 e 4 é possível se basear através do de_para e Manual de Layout, ou seja, quando for a opção 3 na nota, irá gravar 3 no SPED, o mesmo irá ocorrer para a opção 4.

##

## Resolução

  * Verificar a configuração atual de mapeamento de modalidade de frete no DW.


  * Corrigir o mapeamento para garantir que a modalidade de frete no DW corresponda à modalidade registrada no SAP.


  * Realizar uma limpeza na base de dados do DW para remover quaisquer registros incorretos.


  * Executar uma nova equalização para atualizar os dados conforme as novas configurações.


  * Gerar novamente o arquivo SPED para verificar se a modalidade de frete está correta.


## Próximos Passos

Após realizar as correções, é recomendável monitorar os próximos arquivos SPED gerados para assegurar que a incongruência entre os sistemas não persista. Caso a discrepância continue, é aconselhável revisitar as configurações de mapeamento e consultar o suporte técnico.

## Dicas e Recomendações

É importante realizar verificações periódicas das configurações de mapeamento no DW para evitar discrepâncias futuras. Manter uma documentação detalhada das configurações e das modificações realizadas pode facilitar o diagnóstico e a resolução de problemas semelhantes no futuro. Para mais informações, consulte o manual de integração DW-SAP disponível na central de ajuda.
