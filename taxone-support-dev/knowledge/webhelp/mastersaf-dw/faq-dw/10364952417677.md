Este artigo tem por objetivo demonstrar como corrigir o erro 90135: O Campo Codigo de Tipo de Servico nao esta cadastrado

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10364395165325)

2\. O problema ocorre pois o código informado no campo COD_SERVICO não está cadastro na tela BÁSICOS > DATA WAREHOUSE > Manutenção > Códigos > Serviços ou então a DATA_OPERACAO é menor que a data inicial de validade do COD_SERVICO cadastrado no sistema.

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10364852531597)

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10364564119821)

No exemplo do erro, a data de operação é do dia 06/01/2022 e o COD_SERVICO tem a validade maior, portante não é possível que esse COD_SERVICO seja vinculado a esse lançamento contábil.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10364850503053)

3\. Para solucionar o problema:

\- Informar um código valido no arquivo TXT;

\- Cadastrar o COD_SERVICO na tela BÁSICOS > DATA WAREHOUSE > Manutenção > Códigos > Serviços;

\- Verificar a data de validade do serviço no cadastro e corrigir;
