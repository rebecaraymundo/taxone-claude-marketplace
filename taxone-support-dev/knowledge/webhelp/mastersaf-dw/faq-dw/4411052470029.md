## Introdução

O relatório de **Validação das Retenções Previdenciárias** , foi criado para conferir os dados que são utilizados na geração dos eventos R-2010 e R-2020. Neste artigo você irá conferir o passo-a-passo para geração deste relatório.

## Passo-a-passo

**1)** No módulo SPED > REINF, acesse o menu Relatórios > Previdenciário > Validação das Retenções Previdenciárias.
![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4411039281165)

**2)** Informe a competência do REINF que deseja analisar nos campos**Data Inicial** e **Final** , selecione o **Tipo de Relatório** : Listagem de Serviços Sujeitos a Retenção Previdenciária, selecione **Origem** : Entrada de Serviço (para os R-2010) ou Saída de Serviço (para os R-2020), selecione os **Estabelecimentos** e clique em **OK**.

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4411052353933)

3) Será gerada uma página para cada código de Tipo de Serviço do REINF. Os dados exibidos neste relatórios são os mesmos que serão utilizados na geração dos eventos R-2010 e R-2020 originais e retificadores.
![mceclip4.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4411039501709)

Obs. Este relatório pode ser exportado para Excel, para facilitar a conferência dos dados (Arquivo > Salva Como)
