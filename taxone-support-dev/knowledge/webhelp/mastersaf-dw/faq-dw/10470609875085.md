Este artigo tem por objetivo demonstrar como corrigir o erro 90209: O Campo de Data de Validade com formato invalido.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10470603890445)

2\. O problema ocorre pois o valor informado no campo DATA_EMISSAO ou DATA_VENCTO está com formato diferente de AAAAMMDD (ANO-MES-DIA)

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10470600704013)

3\. Para solucionar o problema basta informar uma data no formato valido no campo DATA_EMISSAO ou DATA_VENCTO

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10470604459917)
