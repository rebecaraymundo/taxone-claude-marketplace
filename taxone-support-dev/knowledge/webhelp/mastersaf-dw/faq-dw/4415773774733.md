# Introdução

Este artigo tem como objetivo demonstrar a geração do registro 0205.

# Passo a Passo

## Cadastro do Produto

Módulo: Data Warehouse

Menu: Manutenção > Códigos > Produtos

No cadastro do produto atual é necessário informar os campos do grupo "Código Anterior Utilizado pelo Produto".

A geração irá ocorrer no primeiro período em que ocorrer movimento a partir da data de alteração de código informada.
