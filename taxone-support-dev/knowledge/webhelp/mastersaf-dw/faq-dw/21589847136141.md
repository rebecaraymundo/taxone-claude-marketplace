Erro: 90617 - Bem de Origem não cadastrado

Quando o erro ocorre é necessário validar os campos

Validade do bem que deve ser menor que a data_trab, verificar também os valores de:

Código do Bem, Código do Incorporador, Código do Estabelecimento, Código da Empresa

Todos esses valores devem ser iguais para validar a existência do bem cadastrado.
