# Resolução de Erro na Limpeza ao Importar e Erro ao Tentar Alterar Documentos

## Problema:

O usuário enfrenta um erro ao tentar realizar a limpeza de dados durante a importação e também encontra erros ao tentar realizar alterações em documentos no sistema.

## Causa Raiz:

O problema ocorre devido à ausência de registro do estabelecimento na tabela de Datamart Automático (PRT_DATAMART_AUTOM) no banco de dados.

## Resolução:

Para corrigir o erro, é necessário inserir um registro para o estabelecimento na tabela mencionada. Siga os passos abaixo:

  * Acesse o banco de dados do sistema.


  * Execute o seguinte comando SQL, substituindo 'EMPRESA' pelo código da empresa e 'ESTAB' pelo código do estabelecimento:




    insert into PRT_DATAMART_AUTOM (COD_EMPRESA, COD_ESTAB, IND_AUTOMATICO, IND_FECEP) values ('EMPRESA', 'ESTAB', 'N', 'N');

  * Após executar o comando, verifique se o erro foi resolvido tentando realizar a importação e as alterações novamente.
