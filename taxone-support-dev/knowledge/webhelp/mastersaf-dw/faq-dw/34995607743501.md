## Problema:

Após a atualização do software Mastersaf DW para a versão 303.0.1, ocorreu um erro relacionado ao type "T_SAFX08_NO_VIRTUAL" durante a execução do comando SQL para verificar objetos inválidos.

## Causa Raiz:

O erro ocorreu porque o type "T_SAFX08_NO_VIRTUAL" mencionado não faz parte dos components standard do software, indicando que é um objeto customizado criado especificamente para o ambiente do usuário.

## Resolução:

Siga os passos abaixo para resolver o erro:

  * Identifique e revise a personalização associada ao type "T_SAFX08_NO_VIRTUAL".
  * Consulte a equipe de desenvolvimento ou a equipe técnica que realizou as customizações no sistema para entender melhor a funcionalidade e o propósito do type.
  * Certifique-se de que todas as dependências desse type estejam atualizadas e compatíveis com a versão atual do Mastersaf DW.
  * Execute novamente o comando SQL para verificar objetos inválidos após realizar as correções necessárias.
