Este artigo tem por objetivo demonstrar como corrigir o erro 90209: O Campo de Data de Validade com formato invalido.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10365781789709)

2\. O problema ocorre pois o código informado no campo DAT_LANCTO_EXTEMP não está dentro do formato de data esperado pelo sistema: AAAAMMDD (ANO MES DIA)

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10365759172109)

3\. Para solucionar o problema basta informar a data com a formatação correta no campo DAT_LANCTO_EXTEMP

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10365807576077)
