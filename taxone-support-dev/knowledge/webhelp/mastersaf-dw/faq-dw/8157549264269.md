# [Básicos][Job Servidor] DW - SAFX07 - Erro 91440: UF do armazem de origem da coleta deve ser informada para que seja possivel validar o municipio.

Erro no log de importação:

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/8157544951821/mceclip0.png)

O problema ocorre pois o campo UF_ARMAZ_ORIG está nulo na SAFX07. O campo UF_ARMAZ_ORIG deve ser preenchido com a UF do município mencionado no campo COD_MUNIC_ARMAZ_ORIG.
