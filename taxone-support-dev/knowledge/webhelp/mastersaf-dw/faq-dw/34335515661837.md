# Geração de Informes de Rendimento para IRRF de Auto Retenção.

## Problema:

Usuário necessita gerar Informes de Rendimento para IRRF de Auto Retenção (Retenção no Recebimento) e encontrou dificuldades para executar o procedimento.

## Causa Raiz:

A dificuldade na geração dos Informes de Rendimento para IRRF de Auto Retenção ocorre devido ao desconhecimento de que o sistema Mastersaf DW lê apenas a tabela SAFX53 para essa finalidade, enquanto a tabela SAFX283 é utilizada para outras operações relacionadas ao REINF.

## Resolução:

  * Acesse o sistema Mastersaf DW.


  * Navegue até **Básicos Data Warehouse Manutenção Retenções Controle de tributos**.


  * Insira manualmente ou importe os dados necessários na tabela **SAFX53**.


  * Para gerar os Informes de Rendimento, vá até **Federal Obrigações de Tributos Federais Rendimentos**.


  * Execute a geração dos Informes de Rendimento e verifique se os dados estão corretos.


## Próximos Passos:

Após a geração dos Informes, é recomendável que o usuário verifique meticulosamente se todos os dados estão corretos e completos. Se encontrar qualquer inconsistência, é necessário repetir o processo de entrada de dados no Controle de Tributos e regerar os Informes.

## Dicas e Recomendações:

É importante manter o sistema Mastersaf DW sempre atualizado para a última versão disponível para evitar problemas de compatibilidade e aproveitar as melhorias e correções implementadas. Além disso, realizar backups regulares dos dados inseridos garante uma camada adicional de segurança para as informações da empresa.
