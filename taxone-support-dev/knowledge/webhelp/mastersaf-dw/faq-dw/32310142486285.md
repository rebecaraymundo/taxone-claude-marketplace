# [SPED][EFD - REINF]DW - Resolução do erro ao 'Gerar e Enviar XML' de evento R-4020.

- **ID:** 32310142486285
- **URL:** https://suporte.thomsonreuters.com.br/hc/pt-br/articles/32310142486285--SPED-EFD-REINF-DW-Resolu%C3%A7%C3%A3o-do-erro-ao-Gerar-e-Enviar-XML-de-evento-R-4020
- **Labels:** R-4020, R4020, NOK - NonParserException, DWERRXSD
- **Criado:** 2024-11-30
- **Atualizado:** 2025-11-03

---

Problema:

Ao 'Gerar e Enviar XML' de evento R-4020 está apresentando as seguintes informações:

Código da Mensagem:

DWERRXSD

Descrição da Mensagem:

NOK - NonParserException: cvc-complex-type.2.4.a: Invalid content was found starting with element '{http://www.reinf.esocial.gov.br/schemas/evt4020PagtoBeneficiarioPJ/v2_01_02:endExt}'.
One of '{http://www.reinf.esocial.gov.br/schemas/evt4020PagtoBeneficiarioPJ/v2_01_02:frmTribut}' is expected. - null

Causa Raiz:

Essa situação ocorrer por conta da falta de informações para compor campos obrigatórios do respectivo evento. Para estes casos geralmente é apresentado uma mensagem de alerta, por ocasião da 'Geração Prévia' dos 'Movimentos' indicando a ausência das informações.

Resolução:

Para correção para essa mensagem específico, deve ser informado a 'Forma de Tributação' no respectivo 'Controle de Tributos'. O acesso é através do módulo Data Warehouse > Manutenção > Retenções.

Próximos Passos:

Após realizar a correção, monitore o envio do evento para garantir que os dados estejam corretos no eCAC.

Dicas e Recomendações:

Uma forma de identificar essa situação e encontrar maiores detalhes com a especificação das inconsistências, pode ser através do log da 'Geração Prévia' dos 'Movimentos', disponível no módulo EFD - Reinf > REINF > Geração Prévia > Movimentos > LOG.
