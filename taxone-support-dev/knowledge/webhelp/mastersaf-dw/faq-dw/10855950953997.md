## Passo a passo

O Acrobat DC foi testado em plataformas Microsoft Windows 7 de 64 bits, Microsoft Windows Vista de 64 bits, Windows XP de 64 bits e Windows Server 2003 de 64 bits. A configuração se dá pelos passos abaixo:

1) Após a instalação do Acrobat, esta deve ser renomeada para "Acrobat PDFWriter", conforme abaixo:

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/14439015669901)

![](http://www.mastersaf.com.br/HELPDW/V2R010/Conteudo/Demais_Empresas/Mastersaf_DW/DW/KBA_FAQ/Manager/KBA_-_Configura%C3%A7%C3%A3o_da_impressora_Acrobat_PDFWriter\(1\)_1_02.gif)

2) Acionar o botão de preferência de impressão, em seguida, alterar o campo "Adobe PDF Output Folder" para Documents\\*.pdf e as opções que estão marcadas no print:

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/14439053430797)

3) Nessa figura indica onde o programa irá gravar o arquivo PDF. Toda vez que o usuário mudar o local de saída do arquivo o sistema irá mudar automaticamente o direcionamento indicado pelo usuário:

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/14439087788557)![](http://www.mastersaf.com.br/HELPDW/V2R010/Conteudo/Demais_Empresas/Mastersaf_DW/DW/KBA_FAQ/Manager/KBA_-_Configura%C3%A7%C3%A3o_da_impressora_Acrobat_PDFWriter\(1\)_2_01.gif)

4) Direcionar para C:\temp (caso queira utilizar Informe de Rendimento por e-mail):

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/14439152667405)

Obs.: Atentar para o perfil dessas configurações, deve ser feita para todos que usarem o MasterSAF DW.
