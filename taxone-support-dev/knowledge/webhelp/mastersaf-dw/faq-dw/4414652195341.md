# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 90384: O Campo Codigo de Atividade Economica nao esta cadastrado

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414652165773/mceclip0.png)

2\. O problema ocorre pois o campo COD_ATIVIDADE possui um código que não está cadastrado dentro do DW.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414659329549/mceclip1.png)

3\. Para solucionar o problema existem 3 soluções:

1) Verificar se o código que está no arquivo é valido, caso não esteja corrigir;

2) Ajustar a TACES01 manualmente e importar para o sistema novamente;

3) Cadastrar manualmente o código no menu BÁSICOS > PARÂMETROS > Preliminares > Atividade Econômica

Para o exemplo demonstrado o código estava faltando no ambiente e então foi cadastrado através da tela.

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414667707533/mceclip3.png)
