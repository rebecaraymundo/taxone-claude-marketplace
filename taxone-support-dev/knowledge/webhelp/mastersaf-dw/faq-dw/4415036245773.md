# Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro 90002: O Código do Estabelecimento deve ser preenchido

## Passo-a-Passo

1\. Erro

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415055204365/mceclip2.png)

2\. O problema ocorre pois o campo COD_ESTAB não esta preenchido, veja que no arquivo o campo esta com @

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415055269645/mceclip3.png)

3\. Para corrigir, basta preencher o campo com o Código do Estabelecimento.
