# Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro 91275: O campo Data da nota Fiscal deve ser preenchido

## Passo-a-Passo

1\. Erro:

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/7992045910157/mceclip0.png)

2\. O problema ocorre pois o campo DAT_NF não esta preenchido ou esta preenchido com @.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/7992108348941/mceclip1.png)

3\. Para resolver o problema, basta preencher o campo com a data correta da NF.
