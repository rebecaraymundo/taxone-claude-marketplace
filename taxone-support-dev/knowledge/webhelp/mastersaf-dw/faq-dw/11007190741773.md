# Introdução

Este artigo tem como objetivo demonstrar como resolver o erro "_Não aparece estabelecimento para selecionar na geração do relatório sintético/analítico da ECD_ ".

# Modulo e Menu

_Modulo: SPED > ECD _

_Menu 1: Parâmetros > Controle de Obrigação_

_Menu 2: Relatórios > Relatório Sintético/Analítico_

# Erro:

_"Não aparece estabelecimento para selecionar na geração do relatório sintético/analítico da ECD"_

![](https://t12927461.p.clickup-attachments.com/t12927461/02b1f4e2-3ac8-4cb6-8fce-b894ce7ae843/image.png)

# Solução:

Deve ser verificado se a periodicidade do relatório é a mesma cadastrada no Controle de obrigação.

Neste exemplo a periodicidade do Controle de Obrigação foi configurado para **Anual** , e a tentativa de geração do relatório foi periodicidade **Mensal**. Neste caso, para gerar o relatório também deve ser selecionada a opção **Anual,** desta forma, o estabelecimento aparecerá para seleção.
