## Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro 91170: Indicador de Fabricação pelo Estabelecimento deve ser preenchido com <S> ou <N>.

## Passo-a-passo

**1)** Erro:
![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10637388988813)
**2)** O problema ocorre porque o campo DATA_PRODUTO está com a data no formato invertido. A data deve ser preenchida no formado AAAAMMDD ou seja 20210925:

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10637417598989)
