**Artigo: Resolução do Erro ORA-20100 TRG_RESUME_IMPORTACAO.TRIGGER - ERRO - BANCO: ORA-01400**

Prezado Usuário,

O erro ORA-20100 acompanhado por "TRG_RESUME_IMPORTACAO.TRIGGER - ERRO - BANCO: ORA-01400:C" é um problema comumente relacionado a restrições de banco de dados Oracle, indicando uma violação de integridade ou inserção de valor NULL em uma coluna que não permite tal ação. Para resolver esse problema, siga as instruções abaixo:

**Aplicação do Manual de Boas Práticas:**

  1. Acesse o Manual de Boas Práticas através do seguinte caminho no Zendesk: _Mastersaf DW > Manuais > Manuais Técnicos > Manuais > Manual de Boas Práticas_, ou pelo link [Manual de Boas Práticas](<https://webhelp.thomsonreuters.com.br/Site/download/MasterSAF_DW/pads/Manual-de-Boas-Praticas.pdf>).

  2. Siga as orientações do Manual de Boas Práticas para implementar as práticas recomendadas na gestão e manutenção do banco de dados. Essas boas práticas podem ajudar a prevenir e corrigir problemas como o erro ORA-01400.


**Procedimento de Análise e Diagnóstico do Sistema de Banco de Dados:**

Caso o erro persista após a aplicação do Manual de Boas Práticas, siga os passos abaixo para executar a ferramenta de análise e diagnóstico de sistema de banco de dados:

  1. Extraia todo o conteúdo do arquivo .zip enviado.
  2. Descompacte o arquivo “openjfx-11.0.2_windows-x64_bin-sdk.zip”.
  3. Verifique a versão do Java no Sistema Operacional utilizando o comando `java -version` no prompt do DOS.
  4. Execute o arquivo `StartPads_Java1.8.bat` (para versão Java 1.8) ou `StartPads_Java_11.0.bat` (para versão Java 11.0). Preencha os campos com os dados do banco de dados e clique em "Conectar".
  5. Após a execução, envie os arquivos .HTML e .log gerados para análise adicional.


Segue o link para o manual operacional do programa: [Manual do Usuário - Ferramenta de Análise e Diagnóstico](<https://webhelp.thomsonreuters.com.br/Site/download/MasterSAF_DW/pads/ONESOURCE_PADS_Manual_do_Usuario.pdf>).

Essas orientações visam a aplicação de boas práticas e uma análise mais detalhada do banco de dados, permitindo identificar e resolver o erro ORA-20100 - ORA-01400:C de forma mais eficaz.

Se persistirem dúvidas ou necessitar de assistência adicional, recomendamos entrar em contato com o suporte técnico especializado para um acompanhamento mais direcionado.
