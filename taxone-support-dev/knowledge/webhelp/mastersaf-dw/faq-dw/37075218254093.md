Como resolver o erro na Listagem de Sub Apuração do ICMS

O módulo de ICMS - Listagem de Sub Apuração apresenta uma tela de erro após gerar o relatório

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/37677781411853)

Este tipo de erro geralmente acontece depois de uma atualização de Patch e está relacionado com a necessidade de atualização manual dos executáveis do sistema.

Após realizar a atualização dos executáveis e testar o relatório, monitorize o módulo durante alguns dias para assegurar que o problema foi totalmente resolvido.

Caso o erro retorne, entre em contato com o suporte técnico para uma investigação mais aprofundada.
