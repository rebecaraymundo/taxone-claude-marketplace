## Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 90009: O campo Data de Movimento não esta preenchido ou esta preenchido com zeros.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/6513343900045/mceclip0.png)

2\. O problema ocorre pois a data de movimento não esta preenchida ou esta preenchida com zeros.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/6516297665293/mceclip1.png)

3\. Para solucionar o problema basta preencher o campo com a data correta AAAAMMDD.
