## PROBLEMA

Após instalação do DW, não é possível abrir o DW, aparece o erro abaixo:

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21811932333837)

## SOLUÇÃO PROPOSTA

Este erro costuma ocorrer quando o client Oracle não é compatível com o banco de dados, portanto, desinstale o cliente Oracle e reinstale-o novamente, certificando-se de instalar a versão de 32 bits apropriada para seu banco de dados (ou o cliente 32 e 64). Se você conseguir se conectar usando SQL + (sqlplus.exe) após instalar o cliente, você deverá conseguir se conectar com o app do DW.
