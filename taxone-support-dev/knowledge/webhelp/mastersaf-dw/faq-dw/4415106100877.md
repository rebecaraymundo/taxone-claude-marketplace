# Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro 90100: O campo Debito/Credito deve ser preenchido com D ou C.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415110010765/mceclip0.png)

2\. O problema ocorre pois o campo IND_DEB_CRE não possui algum dos valores pré-definidos:

D - Se o Lançamento foi a Débito da Conta;

C - Se o Lançamento foi a Crédito da Conta.

Veja que no arquivo o valor esta com "@"

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415106009101/mceclip1.png)

3\. Para corrigir o problema, basta alterar o conteúdo do campo para um dos valores válidos.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415110040717/mceclip2.png)
