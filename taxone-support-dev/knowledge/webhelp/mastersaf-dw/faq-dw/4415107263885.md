# Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro 90101: O campo Valor do Lançamento Contábil deve ser preenchido

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415107159181/mceclip0.png)

2\. O problema ocorre pois o Valor do Lançamento Contábil não esta cadastrado ou esta preenchido com zero.

Veja que no arquivo, o campo esta preenchido com zero.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415107234445/mceclip1.png)

3\. Para corrigir o problema, basta preencher o campo VLR_LANCTO com um valor válido.
