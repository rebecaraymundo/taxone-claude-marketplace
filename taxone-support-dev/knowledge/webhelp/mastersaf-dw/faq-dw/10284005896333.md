## Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro Erro 90088: O indicador de pessoa Física/Jurídica não esta preenchida ou com informação inválida

1.Erro:

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10283913109773)

2\. O erro ocorre pois o indicador de pessoa Física/Jurídica não esta preenchido ou com informação inválida

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10283951228301)

3\. Para resolver basta preencher o campo com a informação correta: 1, 2, 3, 4 ou 5.
