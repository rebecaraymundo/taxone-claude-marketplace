# [Federal][OBRIGAÇÕES TRIBUTOS FEDERAIS]DW - Como Corrigir o Erro "Falta Valor Obrigatório para "coluna". Por favor, entre com o valor" - Parâmetros DARF/GPS

- **Criado:** 2020-08-18
- **Atualizado:** 2023-11-01

---

Para sanar o erro “Falta valor obrigatório para “coluna”. Por favor, entre com o valor.”, na inclusão de novos dados nos Parâmetros DARF/GPS (Módulo OTF), deverá ser marcado e desmarcado os parâmetros: “Considerar Estorno a partir da Data de Retificação” e “Gera Retenções de Cooperativa como Estab. (Avulso)” e após isso clicar no ícone "Confirmar".
