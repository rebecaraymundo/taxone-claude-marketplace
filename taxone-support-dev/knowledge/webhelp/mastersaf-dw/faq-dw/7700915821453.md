Ao gerar o XML do evento R-2099 retorna esta mensagem:

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/7700756090125)

Este problema já esta corrigido utilizando o patch atualizado, porém se aparecer algum caso verificar:

Com o XML do R-2099 com status aguardando geração efetuar o seguinte select:

select * from reinf_pger_r2099_compl order by id_r2099_compl for update

A ultima coluna desta tabela é a ind_evt_aquis, se estiver sem preenchimento ocasiona o erro do print acima, pois ela precisa estar = S se o periodo tiver eventos R-2055 e N se não tiver.
Paliativamente pode preencher esta coluna com S ou N (verificar com o cliente se tem R-2055 no periodo) e posteriormente investigar o causador.
