# Introdução:

Este artigo tem por objetivo demonstrar o motivo de não carregar o código natureza da receita na tela Identificador de Natureza de Receita

## Passo-a-Passo

Parâmetros> Apuração > Identificador de Natureza de Receita

O campo Cód. Natureza da receita, são carregado pela TACES 72, serão demonstrado quando data inicial esta de acordo com tabela.

Exemplo: TACES 72 com CST 04, com validade inicial 01012011, na aplicação também deverá ter essa data para buscar código correto.
