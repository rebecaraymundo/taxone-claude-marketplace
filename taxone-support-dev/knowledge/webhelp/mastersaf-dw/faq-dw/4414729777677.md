# Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro "Falta valor obrigatório para 'coluna'" ao atualizar o campo no cadastro de pessoa física/jurídica.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414729755661/mceclip0.png)

2\. O problema ocorre pois o campo IND_PRODUTOR_RURAL está em branco ou nulo.

3.Para corrigir realizar um select na tabela X04_PESSOA_FIS_JUR e verificar se o campo IND_PRODUTOR_RURAL está em branco ou nula. Se estiver, realizar um update para **"N".**
