## Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro 90366: O Campo Código NCM não está cadastrado.

## Passo-a-passo

**1)** Erro:
![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10636106951437)**2)** O problema ocorre, porque o campo COD_NBM está preenchido com um código que não está cadastrado no sistema.
![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10636108960653)

3) O Código do NBM deve ser cadastrado utilizando a SAFX2045 ou cadastrar o código manualmente na tela Básicos > Data Warehouse > Manutenção > Códigos > Código NCM:
![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10636155807757)
