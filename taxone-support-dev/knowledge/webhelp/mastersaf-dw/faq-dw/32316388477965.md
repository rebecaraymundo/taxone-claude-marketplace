# [SPED][EFD - REINF]DW - Resolução de erro na Geração Prévia dos Eventos R-4010 e R-4020: ORA-01722 Invalid Number

Problema:

Usuários enfrentam erro "ORA-01722: invalid number" ao tentar realizar a pré-geração dos eventos R-4010 e R-4020 no sistema.

Causa Raiz:

O erro é causado por um dado inválido na 'Natureza de Rendimento' dentro do 'Controle de Tributos', que não é aceito pelo sistema. Por exemplo alguma caractere especial. Este campo deve ser numérico, conforme é exigido pela obrigação.

Resolução:

Acesse o 'Controle de Tributos', através do módulo Data Warehouse > Manutenção > Retenções.

Verifique e corrija as informações inválidas no campo 'Natureza de Rendimento'.

Salve as alterações realizadas.

Proceda com uma nova 'Geração Prévia' dos 'Movimentos' para testar se o problema foi resolvido.

Próximos Passos:

Após ajustar as informações e realizar uma nova pré-geração, monitore se o erro persiste. Caso o problema continue, entre em contato com o suporte técnico para uma análise mais aprofundada.

Dicas e Recomendações:

Regularmente revise e valide as informações inseridas no sistema para evitar erros de preenchimento que podem causar falhas durante a geração de eventos. Considere realizar treinamentos periódicos ou revisões de procedimentos para assegurar a qualidade dos dados.

Os 'Relatório de Conferência de Tributos Federais', do módulo EFD - Reinf > Relatórios > Federais, pode auxiliar na análise e identificação de alguma informação inválida.
