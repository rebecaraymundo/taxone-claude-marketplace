# Introdução

Este artigo tem por objetivo orientar a solução para a seguinte crítica, apresentada por ocasião da importação do arquivo texto no validador - " Não é um valor válido para o campo domicílio fiscal". - Segmento A – Informações Econômico-Fiscais/Identificação do Contribuinte/Apuração do Imposto.

1- Mensagem de erro ao importar o arquivo meio magnético no validador.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/29097325143181)

2 - Para correção é necessário realizar a parametrização " Dados Mensais da Declaração".

_Módulo:_ | _Estadual > GIAM TO_
---|---
_Menu:_ | _Obrigações > Dados Mensais da Declaração_

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/29097294497805)

Após a parametrização realizar a geração completa e transmitir o arquivo no portal.
