## Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro Erro 90129: O campo Tipo de Movimento de Entrada/Saída não esta preenchido ou com informação inválida

## Passo-a-Passo

1\. Erro:

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10034816006157)

2\. O problema ocorre pois o campo Tipo de Movimento não esta preenchido, ou esta preenchido com informação inválida.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10034837451149)

3\. Para solucionar o problema, basta preencher o campo com uma das informações corretas (1,2,3,4,5 ou 9).
