## Introdução

Neste artigo veremos como fazer um módulo que sumiu/não está disponível aparecer no Manager do Mastersaf DW.

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414101102989/mceclip3.png)

![](http://www.mastersaf.com.br/HELPDW/V2R010/Conteudo/Demais_Empresas/Mastersaf_DW/DW/KBA_FAQ/Manager/KBA_-__O_m%C3%B3dulo_n%C3%A3o_est%C3%A1_dispon%C3%ADvel_no_Manager_do_MasterSAF_DW._Como_resolver_1_00.gif)

## Causas

Esse problema ocorre por falta do registro no banco de dados que faz referência ao módulo, mais precisamente na tabela SAF_MANAGER.

![mceclip5.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414110896013/mceclip5.png)

Isso pode ocorrer quando um novo módulo é incluído no produto e ocorre algum problema na atualização, no momento em que os scripts de atualização do banco são rodados. O mesmo pode acontecer em cópias (DUMPs) de bancos mal-sucedidos.

## Solução

Será necessário inserir o registro via script de banco de dados. Para isso, basta utilizar um script como abaixo:


    insert into saf_manager (IDENT_ITEM_MENU, DESC_ITEM_MENU, IND_TIPO_MENU, IND_PARAMETROS, APLICACAO, IDENT_ITEM_PAI, SEQ_ORDENACAO, ICONE, IND_APRES_SEM_LIC, DESC_RESUM_ITEM_MENU, PATH, VERSAO, COD_MODULO_LIC, IND_TAX_ONE, IND_DW)
    values (600, 'Adamantina', '3', 'S', 'safmunic.exe', 7, 445, 'cdrom.ico', 'S', 'Consiste na entrega das informações sobre os serviços prestados, tomados e bancários no município de Adamantina.', '', '', 'MUN ADAMANTINA', 'S', 'S');
    commit;

Obs.: O campo IDENT_ITEM_MENU é o identificador único do campo, logo não se pode utilizar um número já existente em outro registro na tabela SAF_MANAGER. Portanto, deve ser utilizado O próximo número disponível. No exemplo em questão, o último número utilizado foi 599.

![mceclip6.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414110926477/mceclip6.png)

Após a execução do script, podemos reiniciar o MasterSAF DW, que o módulo se apresenta disponível:

**![](http://www.mastersaf.com.br/HELPDW/V2R010/Conteudo/Demais_Empresas/Mastersaf_DW/DW/KBA_FAQ/circles-55x55-09.png)Observação:** Cada módulo possui seus próprios dados na tabela SAF_MANAGER. O script deverá ser adequado ao módulo que precisa ser inserido, ou seja, o script nunca será igual para todos os casos. Para conseguir os dados, basta consultar um outro ambiente que não esteja com esse problema (QA, DEV, etc.). Se esse problema ocorrer com todos os ambientes, será necessário abrir um ticket solicitando.
