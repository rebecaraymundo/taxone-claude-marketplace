# Como Gerar Relatório de Conferência de Documentos Fiscais com Detalhamento de INSS

## Problema:

O usuário não consegue visualizar os valores de base INSS e INSS retido no relatório 'Conferência de Documentos Fiscais (INSS/ISS/PIS/COFINS)' no Mastersaf para notas de mercadorias.

## Causa Raiz:

O relatório 'Conferência de Documentos Fiscais (INSS/ISS/PIS/COFINS)' é projetado apenas para notas de serviços, o que significa que notas de mercadorias não serão exibidas nesse relatório específico.

## Resolução:

Siga os passos abaixo para gerar o relatório correto:

  * Acesse o Mastersaf DW/TAXONE.


  * No menu lateral, selecione a opção de relatórios 'Via Servidor'.


  * Escolha o relatório 'Detalhamento por Produto'.


  * Execute o relatório para o período desejado.


  * Verifique se as informações de base INSS e INSS retido estão sendo exibidas corretamente.


## Próximos Passos:

Após gerar o relatório, verifique se todas as informações estão corretas e completas. Caso encontre alguma inconsistência, entre em contato com o suporte para assistência adicional.

## Dicas e Recomendações:

Para evitar confusões futuras, sempre verifique a classificação da nota (serviço ou mercadoria) antes de selecionar o relatório. Isso ajudará a garantir que você está utilizando o relatório correto para o tipo de documento fiscal que deseja analisar.
