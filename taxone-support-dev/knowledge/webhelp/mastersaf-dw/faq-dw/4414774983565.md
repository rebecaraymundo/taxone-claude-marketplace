# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 93048: O campo Natureza da Atividade deve ser nulo, 0 ou 1.

## Passo-a-Passo

1\. Erro

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414774917133/mceclip1.png)

2\. Esse erro pode ocorrer por dois motivos;

\- O código informado não está cadastrado na tela BÁSICOS > DATAWAREHOUSE > Manutenção > Códigos > Setor

\- Não existe relacionamento entre tabela Setor e grupo que o estabelecimento pertence na tela BÁSICOS > PARAMETROS > FUNCIONAIS > Relac. entre Tabelas e Grupos

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414775337357/mceclip0.png)

3\. Para o caso desse estabelecimento existe o cadastro na tela BÁSICOS > DATAWAREHOUSE > Manutenção > Códigos > Setor

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414766264845/mceclip2.png)

Mas não existe o relacionamento do estabelecimento com a tabela setor, para solucionar o problema basta associar a tabela Setor para o grupo.

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414766273293/mceclip3.png)
