## Introdução

O objetivo deste artigo é explicar e tentar solucionar o erro 90999: ORA-01843: not a valid month.

## Solução

Normalmente esse erro é causado por alguma informação inválida em campos de data, como sequencias de zeros (00000000). Para sanar, é necessário verificar todos os campos que armazenem algum tipo de data, e se possuir uma sequência de zeros, este deve ser corrigido.

Observação: Lembrando que de acordo com o Manual de Layout, caso seja necessário anular um campo por falta de informação, este deve ser preenchido com @, caso contrário, poderá ocasionar erros como o do código 90999.
