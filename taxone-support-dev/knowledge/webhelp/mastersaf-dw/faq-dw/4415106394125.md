# Introdução

Este artigo tem por objetivo demonstrar como corrigir o erro 90099: O campo Arquivamento é obrigatório - deve ser preenchido

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415106313869/mceclip0.png)

2\. O problema ocorre pois o campo ARQUIVAMENTO não esta preenchido ou esta com "@".

Veja que no arquivo, o campo esta em branco.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4415099515917/mceclip1.png)

3\. Para corrigir o problema, basta realizar o preenchimento do campo com as informações corretas.
