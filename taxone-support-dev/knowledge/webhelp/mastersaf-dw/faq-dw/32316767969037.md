# [SPED][EFD - ECD]DW - Relatório de Conferência Contas x Referencial não está sendo gerado

Problema:

O relatório "Conferência Contas x Referencial" não está abrindo na aba de processos gerados após sua geração no sistema.

Causa Raiz:

A causa identificada para a não abertura do relatório é a ausência de dados para os parâmetros selecionados ao gerar o relatório. Sem dados para processar, o sistema não gera o arquivo do relatório.

Através do log do processo de geração é possível verificar que há uma mensagem, descrevendo que não foram encontrados dados para os parâmetros selecionados.

Resolução:

Verifique se os parâmetros selecionados para a geração do relatório estão corretos e se há dados disponíveis para esses parâmetros.

Vá até a aba de geração de relatórios e selecione novamente o relatório "Conferência Contas x Referencial".

Selecione os parâmetros adequados que você sabe que contêm dados.

Gere o relatório novamente.

Se o relatório for gerado, ele deverá aparecer na aba de processos gerados para visualização e download.

Próximos Passos:

Após seguir os passos acima, se o problema persistir, entre em contato com o suporte técnico fornecendo detalhes adicionais do problema para uma investigação mais aprofundada.

Dicas e Recomendações:

Este relatório é para apoio ao usuário, com objetivo de efetuar algumas validações de inconsistência de dados e ou parâmetros. Sendo assim, só deverá ser gerado se houver dados 'inconsistentes" enquadrados nessas validações.

Assegure-se de revisar o Manual Funcional do módulo para entender melhor os requisitos de dados para cada tipo de relatório. Isso pode evitar problemas de geração de relatórios no futuro devido à seleção incorreta de parâmetros.
