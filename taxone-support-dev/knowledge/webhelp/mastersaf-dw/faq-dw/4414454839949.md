# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 90506: O Campo Codigo de Pessoa Fisica/Juridica deve ser preenchido e acordo com o Indicador de Conteudo do Codigo

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414448421389/mceclip0.png)

2\. Quando o campo IND_CONTEM_COD é igual a 2 ou 3 o campo COD_FIS_JUR precisam ser iguais.

No exemplo abaixo o arquivo está com os campos diferentes

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414454834957/mceclip1.png)

3\. Ajustar o arquivo para que os campos fiquem iguais e o problema será sanado

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414435893261/mceclip2.png)
