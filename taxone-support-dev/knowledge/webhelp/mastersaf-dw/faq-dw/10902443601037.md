Este artigo tem por objetivo demonstrar como corrigir o erro 90994: Erro na conversao de dados para numerico.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10902399457549)

2\. O problema ocorre pois o campo COD_DOCTO está com "@" ou em branco

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10902402781069)

3\. Para solucionar o problema basta ajustar o campo COD_DOCTO com um valor valido.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/10902418262541)
