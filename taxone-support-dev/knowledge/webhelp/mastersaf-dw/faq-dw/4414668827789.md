# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 91168: Indicador de Contribuinte deve ser preenchido com <S> ou <N>

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414660460429/mceclip0.png)

2\. O problema ocorre pois o campo IND_PFJ_CONTRIB não possui algum dos valores pré-definidos no manual de layout:

Informar a Pessoa Física Contribuinte, de acordo com:
S - Se for Pessoa Física Contribuinte;
N - Se não for Pessoa Física Contribuinte.

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414653424269/mceclip1.png)

3\. Para solucionar o problema basta informar o valor "S" ou "N", no exemplo acima foi informado o valor "1"

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414653424653/mceclip2.png)
