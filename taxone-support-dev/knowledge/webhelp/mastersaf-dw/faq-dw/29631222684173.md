## Problema:

Usuário enfrenta erro ao tentar cadastrar pessoa jurídica estrangeira para importação de notas fiscais, utilizando o sistema SAFX 07.

## Causa Raiz:

A causa do problema está relacionada ao uso de um CNPJ inválido durante o processo de cadastro no sistema.

## Resolução:

Siga os passos abaixo para corrigir o erro no cadastro de pessoa jurídica estrangeira:

  * Acesse o sistema e vá até: Básicos > Data Warehouse > Manutenção > Cadastros > Pessoa Física/Jurídica.
  * No campo 'CNPJ', insira um código válido. Caso o cadastro não possua um CNPJ válido, selecione no campo 'Indicador do Código' a opção 'Especial'. Isso permitirá deixar o campo CNPJ em branco.
  * Salve as alterações e tente novamente a importação das notas fiscais.
