Este artigo tem por objetivo demonstrar como corrigir o erro 93603: Campo Data Escrituracao Contabil deve ser preenchido quando a natureza de rendimento igual a 12052

## Passo-a-Passo

1\. Erro

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21505038097165)

2\. O erro ocorre pois o campo DATA_ESCR_CONT está preenchido e o campo IND_NAT_REND é diferente de 12052

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21504991824525)

3\. Para corrigir o erro, basta alterar o campo DATA_ESCR_CONT para @ (nulo) ou então ajustar o código do campo IND_NAT_REND para 12052

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21505007462413)
