# Introdução:

Este artigo tem por objetivo demonstrar como corrigir o erro 92843: O campo Indicacao de Inclusao no Processo Conciliacao de NFS de Corretores no Gerenciador de ISS deve ser preenchido com: S, N ou @.

## Passo-a-Passo

1\. Erro

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414765883661/mceclip0.png)

2\. O problema ocorre pois o campo IND_CONCILIACAO não possui algum dos valores pré-definidos:

Informar se a Pessoa Fis/Jur tem perfil de seguradora, de acordo com:
S – Sim
N – Não.
Este campo foi criado, para ser utilizado no módulo Gerenciador de ISS, diferenciando empresas com perfil de seguradoras e de outros segmentos.

Veja que no arquivo o campo está com o valor "0"

![mceclip1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414774345741/mceclip1.png)

3\. Para corrigir o problema, basta alterar o conteúdo do campo para um dos valores validos.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414774366477/mceclip2.png)
