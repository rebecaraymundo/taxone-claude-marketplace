Campo 24 - Modalidade de Frete no Sistema SIMP

**Problema**

O campo 24 - Modalidade de Frete está sendo gerado incorretamente quando a modalidade de frete da nota fiscal é "3" ou "4.
Conforme manual do SIMP, para os códigos "3" e "4", devem ser gerados no arquivo do SIMP os valores "13" e "14", respectivamente.

**Causa Raiz**

Siga os passos abaixo para garantir que a funcionalidade corrigida esteja implementada corretamente:

Agora, quando a modalidade de frete na nota fiscal for "3 - Transporte próprio estipulado pelo remetente" ou "4 - Transporte próprio estipulado pelo destinatário", o sistema gerará corretamente os códigos "13" e "14" respectivamente, em conformidade com o manual do SIMP. Esta correção garante o correto preenchimento das informações de frete nas declarações SIMP.

**Resolução**

Verifique se o sistema esta com a versão dos objetos referente ao modulo I SIMP atualizado com a ultima versão disponibilizada.

Após a atualização, o sistema deverá gerar automaticamente os códigos "13" e "14" para as modalidades de frete "3" e "4", respectivamente, ao gerar os arquivos SIMP.

**Dicas e Recomendações**

Para evitar problemas futuros relacionados ao preenchimento automático de campos no sistema SIMP:

Mantenha o sistema sempre atualizado, garantindo que as correções mais recentes sejam aplicadas. Realize verificações periódicas das versões instaladas.

Leia atentamente o manual do SIMP para entender os requisitos de preenchimento dos arquivos e evitar inconsistências.

Se surgir qualquer erro ou comportamento inesperado, entre em contato com o suporte técnico imediatamente para análise e resolução.
