Passo a passo para fazer a geração do registro E111 no SPED Fiscal com a regra para código de ajuste:

  1. Acesse _Speds > SPED Fiscal._
  2. Para gerar o registro E111 corretamente quando o código de ajuste for da tabela 5.1.1, siga a regra: o terceiro caractere do código deve ser igual a "0".


![SPED E111.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/30190023887117)

  1. Verifique os códigos de ajuste disponíveis na Apuração do ICMS conforme a tabela 5.1.1.
