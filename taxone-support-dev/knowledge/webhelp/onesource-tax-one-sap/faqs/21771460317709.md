  1. Acesse  _Obrigações Estaduais > DECLAN RJ_
  2. Clique em ![](https://suporte.thomsonreuters.com.br/hc/article_attachments/25243798510477), e selecione**"Configurações dos registros 0300 e 0400".**
  3. Na tela **Configurar Operação** , clique em**![](https://suporte.thomsonreuters.com.br/hc/article_attachments/25243798536973)** , e a tela **Adicionar Operação** é aberta.
  4. No parâmetro **Base de cálculo** , selecione o filtro desejado (opcional o preenchimento).
     1. Esse campo será considerado filtro dos documentos fiscais:
        1. NULO (standard – não realiza o filtro da base de cálculo, considera todos os documentos de acordo com os demais filtros configurados.
        2. Base Tributada ICMS (considerar os documentos que possuem base de tributada do ICMS maior que zero).
        3. Base Outras ICMS (considerar somente os documentos que possuem base outras do ICMS maior que zero).
        4. Base Isentas ICMS (considerar somente os documentos que possuem base isentas do ICMS maior que zero).
        5. Base Outras e Base Isentas ICMS (considerar os documentos que possuem uma das duas base, base outras do ICMS e/ou base isentas do ICMS maiores que zero).
