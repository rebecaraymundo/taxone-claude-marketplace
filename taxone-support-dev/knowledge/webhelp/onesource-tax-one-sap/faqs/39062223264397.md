1. Acesse  _Gerencial / Licenças._


### **Problema**

Ao realizar a importação do arquivo XML de Licenciamento é apresentada a seguinte mensagem:

Erro ao realizar upload: Arquivo de Licença Inválido!

### **Resolução**

Deve ser avaliado a estrutura do arquivo XML importado, onde este deve obedecer a estrutura conforme exemplo abaixo:

![Captura de tela 2025-12-10 094128.jpg](https://suporte.thomsonreuters.com.br/hc/article_attachments/41812554475021)

### **Próximos Passos**

Caso mesmo assim a mensagem "Erro ao realizar upload: Arquivo de Licença Inválido!" persista deverá ser verificado com o time de Contratos!
