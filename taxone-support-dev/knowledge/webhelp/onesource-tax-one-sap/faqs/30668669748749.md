O bloqueio de edição ocorre devido a uma atualização no sistema TAX ONE FOR SAP, que aumentou as restrições de segurança para garantir a integridade dos documentos que podem ser apresentados em mais de uma apuração.

Passo a passo para resolver o problema de não conseguir editar documentos retidos no ONESOURCE TAX ONE FOR SAP para apurações de impostos retidos (INSS, PCC, IRRF ou ISS):

  1. Verifique se todas as apurações relacionadas ao documento que deseja editar estão abertas. Acesse o documento retido e clique no botão "Informações" para identificar se alguma apuração está encerrada.
  2. Caso alguma apuração esteja encerrada, será necessário reabri-la para permitir a edição do documento retido.
  3. Após realizar as alterações necessárias, salve as informações e feche adequadamente todas as apurações que foram reabertas para evitar inconsistências futuras.
  4. Monitore o sistema para verificar se o problema foi resolvido. Caso persista, entre em contato com o suporte técnico.
