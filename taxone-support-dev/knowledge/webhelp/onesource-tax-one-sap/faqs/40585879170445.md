# Erro na Edição do Campo "Considera lançamentos com retenção zerada na apuração"

## Problema

Os usuários enfrentaram um erro ao tentar editar o campo "Considera lançamentos com retenção zerada na apuração" no menu Apuração impostos retidos Apuração ISS.

## Causa Raiz

O problema foi causado por inconsistências no componente relacionado à interface do usuário, que impediam a edição do campo mencionado.

## Resolução

O erro foi corrigido com a liberação de uma atualização para o componente responsável pela funcionalidade:

  * **iss-ui_tax4b** \- versão `1.150.1`


Após aplicar a atualização, o campo voltou a funcionar corretamente, permitindo que os usuários realizem as alterações necessárias.

## Próximos Passos

Após a aplicação da atualização, recomendamos que os usuários:

  * Verifiquem se o campo "Considera lançamentos com retenção zerada na apuração" está funcionando conforme esperado.
  * Realizem testes em diferentes cenários para garantir que o problema foi completamente resolvido.
  * Relatem qualquer comportamento inesperado ao suporte técnico.
