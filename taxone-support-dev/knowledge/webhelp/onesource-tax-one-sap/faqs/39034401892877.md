## Problema

Usuário deseja saber o valor de uma licença adicional do módulo Import no sistema Tax One For SAP e busca instruções sobre como acessar essa informação.

## Causa Raiz

Informações sobre a aquisição ou valores de módulos adicionais estão disponíveis exclusivamente com o gerente de conta do cliente, sendo necessário contatar essa pessoa para obter os detalhes.

## Resolução

Siga os passos abaixo para obter informações sobre o valor de uma licença adicional do módulo Import no sistema Tax One For SAP:

  1. Entre em contato com o gerente de conta responsável pelo cliente. Essa é a pessoa habilitada para fornecer informações relacionadas à aquisição e valores de novos módulos.
  2. Se você não souber quem é o gerente de conta responsável, poderá verificar internamente com alguém responsável pelo contrato.
  3. Aguarde o retorno do gerente de conta. Eles fornecerão um orçamento ou informações sobre os próximos passos para aquisição.


## Próximos Passos

Após receber os detalhes de valores e condições do módulo Import do gerente de conta:

  1. Confirme com sua equipe interna a necessidade da aquisição da licença adicional.
  2. Acompanhe a evolução do processo de aquisição diretamente com o gerente de conta.
  3. Certifique-se de testar o novo módulo Import após sua implementação, caso seja adquirido.


## Dicas e Recomendações

  * Mantenha um registro atualizado dos contatos do gerente de conta responsável para facilitar futuros atendimentos.
  * Avalie cuidadosamente a necessidade de licenças adicionais antes de prosseguir com a compra, garantindo que o investimento atenda às demandas reais do seu negócio.
  * Entre em contato com o suporte para quaisquer dúvidas técnicas sobre o módulo Import e sua configuração após a aquisição.
  * Considere acessar nossa base de conhecimento regularmente para verificar atualizações ou artigos relacionados ao produto Tax One For SAP.
