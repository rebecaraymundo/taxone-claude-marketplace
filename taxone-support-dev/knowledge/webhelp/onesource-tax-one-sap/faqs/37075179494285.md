# [Speds][Sped Fiscal]TAX ONE FOR SAP - Geração do Registro 0015.

- **ID:** 37075179494285
- **URL:** https://suporte.thomsonreuters.com.br/hc/pt-br/articles/37075179494285--Speds-Sped-Fiscal-TAX-ONE-FOR-SAP-Gera%C3%A7%C3%A3o-do-Registro-0015
- **Criado:** 2025-06-02
- **Atualizado:** 2025-12-08

---

**Problema:**

O Registro 0015 do Sped Fiscal não está sendo gerado no sistema SAP TDF/DRC.

**Causa Raiz:**

A geração do Registro 0015 da EFD ICMS IPI é controlada por uma configuração padrão dentro do módulo SAP TDF/DRC, que é independente do aplicativo Tax One for SAP. Este registro pode deixar de ser gerado devido a configurações ou atualizações internas no SAP TDF/DRC.

![](/attachments/token/5pRvyBCRJwKQai429YCO2koLp/?name=image.png)

**Resolução:**

Verifique se as configurações do SAP TDF/DRC estão corretas para a geração do Registro 0015.

Consulte a documentação do sistema SAP TDF/DRC para entender os requisitos necessários para a geração deste registro.

Se não conseguir resolver, abra um chamado de suporte diretamente com a equipe do SAP para assistência especializada.

**Próximos Passos:**

Após ajustar as configurações ou receber suporte do SAP, monitore as próximas gerações do Sped Fiscal para garantir que o Registro 0015 esteja sendo incluído conforme esperado.

**Dicas e Recomendações:**

Verifique regularmente as atualizações de sistema do SAP TDF/DRC e mantenha as configurações alinhadas com as melhores práticas e recomendações do fornecedor. Isso pode ajudar a evitar futuros problemas de geração de registros fiscais e garantir a conformidade contínua com os requisitos fiscais.
