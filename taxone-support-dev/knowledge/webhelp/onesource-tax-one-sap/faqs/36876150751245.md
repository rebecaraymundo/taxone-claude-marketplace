# [Speds][Sped Fiscal]TAX ONE FOR SAP - Geração de Registro C113.

- **ID:** 36876150751245
- **URL:** https://suporte.thomsonreuters.com.br/hc/pt-br/articles/36876150751245--Speds-Sped-Fiscal-TAX-ONE-FOR-SAP-Gera%C3%A7%C3%A3o-de-Registro-C113
- **Criado:** 2025-05-24
- **Atualizado:** 2025-12-08

---

**Problema:**

Notas de entrada referentes à transferência de imobilizado (CFOP 1552/2552) estão gerando registro no Bloco C113 do SPED Fiscal, quando não deveriam.

**Causa Raiz:**

O Registro C113 da EFD ICMS IPI é gerado de forma padrão pelo SAP TDF/DRC, sem influência direta do Tax One for SAP. Este comportamento acontece devido à configuração padrão na fundamentação legal do produto.

**Resolução:**

Acesse o sistema SAP TDF/DRC.

Navegue até a configuração dos registros do SPED Fiscal.

Verifique e ajuste a configuração do Registro C113 para que não reconheça as notas de entrada com CFOP 1552/2552 como obrigatoriedade de geração deste registro.

Salve as alterações e teste para garantir que o registro não está mais sendo gerado para esses CFOPs.

**Próximos Passos:**

Após realizar as alterações, monitore os próximos lançamentos para assegurar que o registro C113 não esteja sendo mais gerado para as notas com CFOP 1552/2552. Caso o problema persista, entre em contato com o suporte técnico para uma análise mais detalhada.

**Dicas e Recomendações:**

É importante revisar periodicamente as configurações de registro do SPED Fiscal no SAP TDF/DRC para garantir que estão alinhadas com as operações da empresa e a legislação vigente. Mantenha-se atualizado com as mudanças na legislação tributária para evitar inconsistências nos registros fiscais.
