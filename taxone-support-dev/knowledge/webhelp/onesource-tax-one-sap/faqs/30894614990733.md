# Configuração de Tipos de Serviço para Geração de Arquivos de Construção Civil no GISS ONLINE

## Problema:

Usuários do GISS Online estão encontrando um problema onde tipos de serviço não relacionados à construção civil estão sendo incorretamente incluídos nos arquivos dos layouts H6 e I3.

## Causa Raiz:

O sistema estava configurado para incluir tipos de serviço como limpeza (códigos 7.09 e 07.10) nos arquivos de construção civil, o que é inadequado devido à natureza específica desses serviços.

## Resolução:

Criada configuração na tela de Parâmetro "Configurar Código Tipo de Serviço para Construção Civil"
para o usuário identificar quais tipos de serviço servirão de parâmetro para que a aplicação gere os arquivos de construção civil modelos H6 e I3. O sistema ao ler o parâmetro irá identificar que para aquele código apontado pelo usuário, a aplicação deve gerar somente os arquivos dos leiautes H6 e I3. Para os demais leiautes, a aplicação deverá verificar esse parâmetro e não gerar dados para documentos com esses tipos de serviço apontados.

## Próximos Passos:

Após aplicar as configurações, monitorize a geração dos arquivos durante os próximos ciclos de envio para garantir que apenas os tipos de serviço corretos estão sendo incluídos nos arquivos de construção civil. Caso encontre alguma inconsistência, entre em contato com o suporte técnico.

## Dicas e Recomendações:

Mantenha sempre a documentação de configuração atualizada e revise periodicamente as configurações para evitar futuros problemas. Em caso de dúvidas ou necessidade de suporte adicional, não hesite em contatar nossa equipe de suporte técnico.
