Quando o erro "Single-row query returns more than one: SQL Error" ocorrer ao consultar a view - wrapper::NF_IMPOSTO_RETIDO no módulo Apuração de Impostos Retido > ISS, a seguinte orientação deve ser seguida:

Para a execução do SELECT, é necessário incluir o parâmetro HINT, conforme o exemplo abaixo:

SELECT * FROM "tax4b.wrapper::NF_IMPOSTO_RETIDO" WHERE NUM_LCTO IN ('1900000735', '1900000206') WITH HINT(NO_USE_HEX_PLAN).

Este erro é comum quando o cliente utiliza a plataforma BTP e pode estar relacionado à necessidade de ajustes na codificação, como a utilização de tabelas ANSI. Incluir o parâmetro HINT no SELECT ajuda a garantir que a consulta seja executada corretamente.
