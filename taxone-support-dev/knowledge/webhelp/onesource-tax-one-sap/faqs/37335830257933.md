# ROTEIRO

O cache do PIS/COFINS é ativado/desativado a partir de dois parâmetros que devem sempre ser alterados em conjunto. Um deles está no sinonimo "tax4b.koine::CD_UTIL.PARAM" e o outro na tabela "tax4b.piscof.cache::CACHE.CONFIG".

  * Cache - ATIVO


\- No sinonimo "tax4b.koine::CD_UTIL.PARAM", devemos inserir o seguinte item:

name: '/PISCOF/CACHE'

value: 'X'

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/37335830229773)

\- Na tabela "tax4b.piscof.cache::CACHE.CONFIG" o campo 'LIDADO' deve ser marcado com 'S'.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/37335830231437)

  * Cache - INATIVO


\- No sinonimo "tax4b.koine::CD_UTIL.PARAM", devemos inserir o seguinte item:

name: '/PISCOF/CACHE'

value: ''

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/37335830241293)

\- Na tabela "tax4b.piscof.cache::CACHE.CONFIG" o campo 'LIDADO' deve ser marcado com 'N'.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/37335845157773)
