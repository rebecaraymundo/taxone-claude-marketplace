## Problema

O código BAE22 - Aquisição de serviços de Comunicação/Telecomunicação, válido para operações no estado da Bahia, está listado na tabela do SPED, mas não é apresentado nas configurações do sistema Tax One.

## Causa Raiz

O código BAE22 possui uma validade inicial que determina o período em que pode ser utilizado. Tentativas de aplicação em períodos fora da validade inicial impedem que o código seja exibido. Além disso, a ausência de atualização dos objetos e configurações do sistema pode levar ao problema reportado.

## Resolução

Siga os passos abaixo para habilitar o código BAE22 no sistema Tax One:

  1. Certifique-se de que o período da apuração corresponde à validade inicial do código BAE22 (07/2024), ou utilize um período posterior.
  2. Atualize os objetos relacionados no sistema Tax One para as versões mais recentes disponíveis:
     * **tax4b.dbobjects**
     * **tax4b.cfg_filtro-backend**
     * **tax4b.cfg_filtro-db**
     * **tax4b.cfg_filtro-ui**
  3. Realize testes detalhados para garantir que o código seja exibido nas configurações do registro 1400.
  4. Caso os testes ainda indiquem que o código não está disponível, capture e envie as seguintes evidências ao suporte técnico:
     * Captura de tela detalhada do passo a passo do cenário reportado, incluindo período de apuração, estrutura organizacional, status e estabelecimento.
     * Versão atualizada dos objetos do sistema, especialmente `Dbobjects` e `Configuração de filtro`.


## Próximos Passos

Após atualizar os objetos e testar para um período válido, confirme se o código BAE22 está habilitado. Caso o problema persista, encaminhe as evidências listadas acima ao suporte técnico para análise aprofundada.

## Dicas e Recomendações

  * Antes de realizar o teste, verifique a validade dos códigos diretamente na tabela do SPED. Isso evitará erros relacionados a períodos incorretos.
  * Mantenha os objetos do sistema Tax One sempre atualizados para garantir acesso às configurações mais recentes.
  * Ao reportar problemas ao suporte, inclua o máximo de informações e evidências para acelerar o processo de análise.
  * Consulte regularmente documentações adicionais ou artigos relacionados na Base de Conhecimento para evitar problemas similares.
