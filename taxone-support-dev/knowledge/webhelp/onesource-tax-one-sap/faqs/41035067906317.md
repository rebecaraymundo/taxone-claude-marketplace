# Como reconstruir o cálculo do FCI (Ficha de Conteúdo de Importação) no T14SAP

###

## Problema

O usuário precisa recriar cálculos históricos do FCI realizados em um sistema legado e deseja saber se o T14SAP possui funcionalidades para importar dados históricos ou reconstruir esses cálculos.

## Causa Raiz

O T14SAP não possui, em sua configuração padrão, funcionalidades nativas para importação direta de dados históricos do FCI provenientes de sistemas legados. O sistema é projetado para trabalhar com informações disponíveis dentro do ambiente SAP ou inseridas manualmente.

## Resolução

Siga as instruções abaixo para tratar a necessidade:

  1. Verifique as configurações gerais do aplicativo do FCI no T14SAP. Certifique-se de que os parâmetros estão ajustados para permitir uma busca mais abrangente de dados, como períodos e limitação de meses de notas fiscais e ordens de produção.
  2. Consulte o **Manual Funcional do aplicativo FCI** para entender os formatos de entrada de dados aceitos pelo sistema.
  3. Caso seja necessário inserir dados históricos manualmente, organize as informações em um formato claro, como tabelas ou planilhas, conforme especificado no manual.
  4. Realize a entrada dos dados históricos diretamente no ambiente do T14SAP, utilizando as funcionalidades disponíveis para cálculo do FCI.
  5. Se os dados históricos forem volumosos ou complexos, avalie com a equipe de TI ou consultoria especializada a viabilidade de desenvolver uma integração personalizada entre o sistema legado e o T14SAP.


## Próximos Passos

Após concluir a inserção ou reconstrução dos dados históricos:

  1. Realize testes para validar os cálculos do FCI no T14SAP e garantir que os resultados estão corretos.
  2. Documente o processo utilizado para reconstrução, caso seja necessário repetir o procedimento no futuro.
  3. Entre em contato com o suporte da Thomson Reuters caso surjam dúvidas ou seja necessário suporte adicional.


## Dicas e Recomendações

  * Consulte regularmente o **Manual Funcional do aplicativo FCI** para garantir que você está utilizando todas as funcionalidades disponíveis.
  * Considere organizar um cronograma de migração de dados históricos para evitar sobrecarga de trabalho em períodos fiscais movimentados.
  * Para evitar problemas futuros, mantenha um backup organizado dos cálculos e dados do FCI, especialmente aqueles provenientes de sistemas legados.
  * Entre em contato com um especialista em integração de sistemas caso seja necessário importar grandes volumes de dados de um sistema legado.


Para mais informações sobre o T14SAP e suas funcionalidades, recomendamos acessar os guias e recursos disponíveis no site oficial da Thomson Reuters ou entrar em contato com o suporte técnico.
