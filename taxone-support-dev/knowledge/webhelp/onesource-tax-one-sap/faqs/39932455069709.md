# [Gestão de Dados][Nota Fiscal] ONESOURCE TAX ONE FOR SAP - Notas não sendo apresentados no modulo no Nota Fiscal

- **ID:** 39932455069709
- **URL:** https://suporte.thomsonreuters.com.br/hc/pt-br/articles/39932455069709--Gest%C3%A3o-de-Dados-Nota-Fiscal-ONESOURCE-TAX-ONE-FOR-SAP-Notas-n%C3%A3o-sendo-apresentados-no-modulo-no-Nota-Fiscal
- **Criado:** 2025-09-29
- **Atualizado:** 2025-09-29

---

**Problema**

Ao acessar o módulo Nota Fiscal os documentos não são apresentados para nenhum empresa e período.

**Resolução**

Deve ser verificado a estrutura organizacional cadastrada possui espaço em branco **exemplo:** "TESTE LTDA', o modulo ODATA não aceita espaços em branco como parâmetro em suas views e acaba falhando as consultas.
Para isto poderá ser verificado a seguinte view e shadow:

Shadow: "/TMF/D_CENTR_EFD"

View Standard SAP - sap.glo.tmflocbr.ctr/EMPRESA_CENTRAL_EFD

### **Próximos Passos**

Após realizar os ajustes recomendados, monitore o sistema para assegurar que as notas fiscais estão aparecendo corretamente no módulo Gestão de Dados / Nota Fiscal .

Caso o problema se repita, revisite as configurações e caso persista, consulte o suporte técnico novamente.

### Dicas e Recomendações:

Mantenha os objetos e configurações do Tax One FOR SAP atualizados para garantir o pleno funcionamento do sistema.
