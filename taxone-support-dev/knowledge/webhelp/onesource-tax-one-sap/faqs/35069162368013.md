Passo a passo para resolver divergências em valores de Guia de Pagamento com Ajuste Automatizado no ONESOURCE TAX ONE FOR SAP (ICMS-ST - Cálculo por Dentro = SIM):

1.Acesse o menu _Apuração de impostos indiretos > ICMS ST_.

2\. Verifique se a regra de cálculo **"Cálculo por Dentro = SIM"** está ativada para o ajuste automatizado do ICMS-ST.

3\. Utilize a fórmula para calcular o valor correto: [base de cálculo / (1 - alíquota do ajuste sem percentual)] x alíquota do ajuste. Exemplo: Se a base tributada do ICMS ST é R$ 1000, o cálculo seria (1000 / (1-0.08)) x 8% = R$ 86,96.

4\. Confira se o valor calculado pela fórmula é o mesmo valor gerado pelo ajuste automatizado e refletido na Guia de Arrecadação.

![ICMS.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/37403998114189)

5\. Monitore os próximos pagamentos para garantir que as divergências foram resolvidas. Caso o problema persista, entre em contato com o suporte técnico.

6\. Revise periodicamente os parâmetros de cálculo no sistema para garantir conformidade com as normas fiscais e necessidades da empresa. Consulte o manual do usuário para mais informações.
