# [Obrigações Federais][IRPJ] ONESOURCE Tax One for SAP - Obrigações Federais - IRPJ - Resolução de Erro no Encerramento da Apuração do IRPJ no TaxOne

## Problema:

Erro ao tentar encerrar a apuração do IRPJ no software TaxOne, com apresentação de mensagem de erro na tela.

## Causa Raiz:

O problema foi identificado como uma falha na regra de conversão do limite de incentivo no módulo de Obrigações Federais IRPJ. Em situações específicas, era possível cadastrar um limite de incentivo vazio, levando a um erro de conversão numérica que afetava todas as funções relacionadas.

## Resolução:

A correção aplicada envolveu ajustar a regra de conversão para que, ao detectar um limite de incentivo vazio, o sistema automaticamente defina o valor padrão como 0%. Esse ajuste permite que o usuário faça as alterações necessárias sem que ocorra o erro.

Para aplicar a correção no seu sistema, por favor, atualize o módulo para a versão **irpj-db_tax4b 1.150.2** , ou superior.

## Próximos Passos:

Após atualizar o módulo, verifique se o erro persiste ao tentar encerrar a apuração do IRPJ. Caso o problema continue, entre em contato com o suporte técnico.

## Dicas e Recomendações:

É recomendável sempre manter o software atualizado para a última versão disponível para evitar problemas de compatibilidade e erros operacionais. Além disso, verifique se todos os campos de incentivos estão corretamente preenchidos antes de realizar o encerramento de apurações para evitar erros.
