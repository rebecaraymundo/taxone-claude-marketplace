# Correção da Exibição de Valores no Relatório Analítico de IRRF

## Problema:

O número fixo "1" estava sendo exibido no lugar do valor correto do Imposto de Renda Retido na Fonte (IRRF) na coluna Imposto da aba Documentos Extemporâneos do Relatório Analítico - XLS.

## Causa Raiz:

A exibição incorreta do valor do IRRF foi causada por uma versão anterior do componente irrf-js que não estava processando corretamente os dados de retenção de impostos.

## Resolução:

A correção foi implementada com a atualização do componente irrf-js para a versão 1.149.1. Para usuários que enfrentam este problema, siga estes passos para garantir que o sistema esteja atualizado:

  1. Acesse o sistema e vá até a seção de atualizações do software.
  2. Procure pela versão 1.149.1 do componente irrf-js.
  3. Instale a atualização disponível.
  4. Reinicie o sistema para que as mudanças tenham efeito completo.


## Próximos Passos:

Após atualizar o componente, é recomendável exportar novamente o Relatório Analítico - XLS para verificar se a exibição dos valores de IRRF está correta. Se o problema persistir, entre em contato com o suporte técnico.

## Dicas e Recomendações:

Mantenha seu sistema sempre atualizado para evitar problemas de exibição e processamento de dados. Verifique regularmente por novas atualizações e instale-as para garantir o funcionamento otimizado do sistema.
