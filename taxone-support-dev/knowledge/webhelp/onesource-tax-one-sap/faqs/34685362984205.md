# Utilizando Filtros de Lançamento Contábeis no TAX ONE FOR SAP

## Problema:

Usuário não consegue aplicar dois filtros de lançamento contábeis utilizando o Tipo de Filtro "CONTÉM" no sistema TAX ONE FOR SAP ao qual é apresentado a mensagem abaixo:

![lançamento contabeis.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/37551980837773)

Não é possível realizar o filtro utilizando o CONTÉM duas vezes

![filtro pis cofins.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/37551980839053)

Mensagem informativa apresentada pelo sistema:

![Piscofins.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/37552021670797)

## Causa Raiz:

A mensagem **"Múltiplos Filtros somente podem ser inseridos para os tipos de filtro 'Igual'"** não se trata de um erro de sistema, mas sim de uma informação sobre a utilização do filtro de "CONTEM" ao qual esta condição deve ser aplicada apenas uma vez.
O sistema utiliza esta regra para que desempenho d IRPJ não seja afetado.

## Resolução:

  * Acesse o módulo de lançamentos contábeis no sistema TAX ONE FOR SAP.
  * Insira o filtro desejado utilizando o Tipo de Filtro "CONTÉM".
  * Certifique-se de que este é o único filtro do tipo "CONTÉM" que está sendo aplicado.
  * Se necessário aplicar outro filtro, escolha um Tipo de Filtro diferente, como "IGUAL" ou "DIFERENTE".
  * Execute o filtro para ver os resultados.


## Próximos Passos:

Após aplicar o filtro corretamente, verifique se os resultados estão de acordo com o esperado. Caso encontre alguma inconsistência, entre em contato com o suporte técnico para uma análise mais detalhada.

## Dicas e Recomendações:

Para uma melhor performance do sistema e precisão nos resultados, é recomendado revisar os tipos de filtros aplicados em suas buscas e consultas no sistema TAX ONE FOR SAP. Utilizar o filtro "CONTÉM" de maneira limitada ajuda a evitar sobrecargas e mantém a eficiência do cálculo de impostos.
