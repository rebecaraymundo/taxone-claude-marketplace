Passo a passo para verificar o motivo das notas não estarem sendo apresentadas no Painel de Notas:

  1. Para que as Notas Fiscais sejam apresentadas no Painel de Nota Fiscal, deve ser avaliado o campo DT_E_S (Data de Entrada e Saída), ao qual este deve ser preenchido com uma Data Válida.
  2. O preenchimento com zeros não será considerado. **![Nota Fiscal.PNG](https://suporte.thomsonreuters.com.br/hc/article_attachments/26738160474509)**
  3. A informação referente ao campo DT_E_S poderá ser veriicada no TDF através da transação SE38 - ITMF_REPDAT_FILL.
