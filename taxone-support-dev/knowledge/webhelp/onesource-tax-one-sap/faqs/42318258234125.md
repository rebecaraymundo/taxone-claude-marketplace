# [Obrigações Municipais]TAX ONE FOR SAP - Configuração do Tipo de Nota Fiscal para o Município de São José dos Pinhais no Módulo ISS ONLINE TXT

## Problema

Ao tentar configurar o tipo de nota fiscal no sistema, o campo de seleção não apresenta nenhuma informação para o município de São José dos Pinhais.

## Causa Raiz

O sistema possui configuração de tipo de nota fiscal apenas para municípios específicos. Para São José dos Pinhais, essa configuração não estava previamente habilitada, pois o atendimento a esse município depende da utilização do módulo correto e da análise do layout/documentação municipal.

## Resolução

  1. Confirme se o município em questão é São José dos Pinhais.
  2. Verifique se o módulo utilizado para geração do arquivo é o **ISS ONLINE TXT**. Caso não seja, altere para esse módulo conforme orientação da equipe técnica.
  3. Caso necessário, obtenha o layout/manual da prefeitura de São José dos Pinhais para validar os tipos de nota e os campos obrigatórios.
  4. Após confirmar o uso do módulo **ISS ONLINE TXT** , realize a geração do arquivo normalmente pela ferramenta.
  5. Se o campo de tipo de nota continuar em branco, confirme junto ao suporte se há atualização pendente no sistema ou se há necessidade de customização.


## Próximos Passos

  * Valide o arquivo gerado pelo sistema junto ao layout exigido pela prefeitura de São José dos Pinhais.
  * Realize testes de envio do arquivo para garantir que a configuração está de acordo com a obrigação municipal.
  * Em caso de dúvidas sobre o preenchimento ou obrigatoriedade de campos, consulte a equipe de suporte ou o manual do município.


## Dicas e Recomendações

  * Sempre utilize o módulo correto para cada município, conforme orientação da equipe técnica.
  * Antes de configurar novos municípios, verifique se existe documentação ou layout atualizado da prefeitura para evitar divergências.
  * Para municípios que não aparecem na lista de configuração automática, encaminhe o manual municipal para avaliação da equipe técnica.
  * Em caso de dúvidas, consulte a base de conhecimento ou entre em contato com o suporte.
  * Para mais informações sobre configurações de notas fiscais e obrigações municipais, acesse os demais artigos da nossa base de conhecimento.
