Passo a passo para fazer a cópia das configurações:

  1. Acesse  _Gestão de Dados > Cópia de configurações._

  2. Selecione o **Aplicativo **e a **Configuração**.


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/25213443566605)

Observação importante: se executado uma vez, as configurações serão realizadas normalmente. Se executado uma segunda vez para a mesma estrutura de destino, as configurações serão duplicadas, pois cada configuração tem um ID interno de controle, portanto, quem estiver realizando este processo tem que ficar atento as estruturas de destino. Caso seja configurado em duplicidade por engano, o delete tem que ser realizado em cada uma das estruturas organizacionais, manualmente.

Para o aplicativo **ICMS **estão disponíveis no momento as cópias das configurações (serão disponibilizadas outras opções aos poucos):

\- Ajustes automatizados

\- Guia de recolhimento do ICMS PRÓPRIO

  1. Selecione a **Configuração **a ser copiada.

  2. Selecionar a **Unidade Organizacional** (origem), ou seja, da estrutura que pretende copiar as configurações

  3. O campo **Vigente em** é preenchido com o período atual.

  4. Selecionar a **Unidade Organizacional** (destino), ou seja, a estrutura que pretende configurar, copiando da origem (pode selecionar 1 ou mais ao mesmo tempo).

  5. No campo **Vigência inicial em** informe a partir de qual ano e mês será válida essas configurações.

  6. Após o preenchimento, clique em **Iniciar cópia**.

  7. A seguinte tela é apresentada mostrando as **Tarefas em andamento** :


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/25213443580557)

  1. Para cada** Estrutura Organizacional** é apresentado um **Status**. Após finalizado, é apresentado o status “**Nova parametrização salva** ”.

  2. Quando for solicitada a cópia para várias estruturas, aguarde a finalização de todas para visualizar o **Status**.


Para o aplicativo **PIS / COFINS **estão disponíveis no momento as cópias das configurações:

  * Ajustes automatizados
  * Apuração de controle de crédito sobre bens do ativo imobilizado
  * Atos cooperados
  * Configuração de busca de lançamento contábil
  * Controle de créditos
  * Detalhamento das receitas isentas, sem incidência, sujeitas a alíquota zero ou com suspensão
  * Gerais
  * Guia de recolhimento
  * PIS COFINS cumulativo
  * Rateio
