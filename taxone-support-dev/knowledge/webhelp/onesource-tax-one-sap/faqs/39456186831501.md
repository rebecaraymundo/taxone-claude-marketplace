# Resolução de Problemas de Contabilização de Guias de ICMS-ST e FECP-ST no SAP

## Problema:

Usuários estão enfrentando problemas recorrentes ao tentar contabilizar guias de ICMS-ST e FECP-ST em algumas UFs no SAP, com o sistema retornando a mensagem "Nenhuma parametrização encontrada para o código de receita: 100048".

## Causa Raiz:

O problema ocorre devido a uma discrepância nos códigos de receita configurados no app de contabilização automática. Se o código de receita configurado não corresponder exatamente ao código da guia de recolhimento, a mensagem de erro será exibida.

## Resolução:

  1. Verificar a configuração do documento de arrecadação da contabilização automática e assegurar que o 'Código da Receita' está correto, ou seja, deve ser **100048**.
  2. Realizar um novo teste de contabilização para a respectiva guia de recolhimento.
  3. Caso o problema persista, atualize todos os objetos pertinentes aos aplicativos de contabilização automática e do ICMS-ST e realize um novo teste.
  4. Se ainda enfrentar problemas, encaminhar as evidências atualizadas do teste, incluindo a configuração da guia de recolhimento do ICMS-ST para análise detalhada.


## Próximos Passos:

Após implementar a solução, continue monitorando os processos de contabilização das guias de ICMS-ST e FECP-ST para garantir que o problema foi resolvido completamente. Caso o erro persista, entre em contato novamente com o Suporte.

## Dicas e Recomendações:

Mantenha sempre atualizada a configuração dos códigos de receita no SAP para evitar problemas de contabilização. Consulte regularmente a documentação dos aplicativos de contabilização automática para quaisquer atualizações ou dicas adicionais.
