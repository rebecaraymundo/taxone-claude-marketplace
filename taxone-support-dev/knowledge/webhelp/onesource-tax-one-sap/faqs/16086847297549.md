  1. Ao selecionar a linha do livro, e na sequência o Código de ajuste, o sistema apresentará automaticamente os códigos vigentes para o período, sem a necessidade de digitar o início ou o código de ajuste completo.

  1. Quando é digitado o Código de ajuste para busca, o sistema apresenta todos os códigos disponíveis, mesmo os encerrados, pois é realizado o encerramento da vigência e não a exclusão do código em tabela do sistema, ou seja, para verificar os códigos de ajustes vigentes no sistema não há necessidade de digitar a informação no campo pesquisa, pois o sistema carregará automaticamente os códigos de ajustes vigentes.
