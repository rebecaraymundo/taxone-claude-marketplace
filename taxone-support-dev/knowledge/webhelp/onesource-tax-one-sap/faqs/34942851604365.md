Quando o módulo Gerencial Licenciamento não apresentar informações ou exibir a mensagem "Nenhuma Licença encontrada", verifique os seguintes pontos:

  1. Confirme se o upload do XML de Licenciamento foi realizado no ambiente.
  2. Solicite o resultado das seguintes consultas SQL:
     * SELECT * from "tax4b.koine::CD_LICENCA.LICENCA_MODULO"
     * SELECT * from "tax4b.koine::CD_LICENCA.LICENCA";
  3. Verifique se foi realizado o procedimento de Undeploy de algum objeto e avalie a sequência realizada. Quando um Undeploy é feito, é necessário realizar o procedimento para todos os objetos dependentes na ordem correta.
  4. Cada caso deve ser avaliado individualmente, pois alguns aplicativos podem causar o redeploy de todo o sistema. Exemplos incluem: wrapper, koine e dbobjects. É recomendado evitar fazer Undeploy de dbobjects.
