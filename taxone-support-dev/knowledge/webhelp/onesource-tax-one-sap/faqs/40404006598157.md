# Notas Fiscais desaparecidas da apuração do INSS

## Problema:

Usuário reportou que algumas notas fiscais desapareceram da apuração de INSS do período de março de 2025.

## Causa Raiz:

O problema foi identificado como decorrente de uma incompatibilidade na versão de objetos relacionados ao cálculo do INSS no sistema. A versão desatualizada dos objetos estava causando inconsistência na apuração das notas fiscais.

## Resolução:

Siga os passos abaixo para resolver o problema:

  1. Certifique-se de que o cliente enviou todos os dados necessários, incluindo os resultados dos SELECTs relevantes.
  2. Verifique se o ambiente do cliente está atualizado com as versões corretas dos objetos do INSS.
  3. Solicite ao cliente que atualize os objetos para as versões indicadas:
     * inss-db_1.147.1
     * inss-ui_1.139.2
  4. Após a atualização dos objetos, realize testes para verificar se as notas fiscais aparecem corretamente na apuração do INSS.


## Próximos Passos:

Após a implementação da solução apresentada:

  1. Confirme com o cliente se as notas fiscais ausentes agora estão aparecendo na apuração do INSS.
  2. Realize testes no ambiente do cliente para validar o funcionamento do sistema e a correção do problema.
  3. Documente o processo de atualização e os resultados dos testes no ticket.
  4. Oriente o cliente a monitorar o sistema por alguns dias e informar qualquer anormalidade.


## Dicas e Recomendações:

  * Mantenha os objetos do sistema sempre atualizados para evitar incompatibilidades.
  * Estabeleça um cronograma de atualização preventiva para os sistemas utilizados, garantindo que estejam na versão mais recente.
  * Realize backups regulares antes de implementar qualquer atualização no sistema.
  * Em caso de dúvida sobre os procedimentos de apuração do INSS ou configurações do sistema, consulte nossa base de conhecimento ou entre em contato com o suporte.
  * Para evitar problemas relacionados a desaparecimento de notas fiscais, verifique periodicamente se todos os documentos emitidos estão sendo corretamente registrados na apuração.
