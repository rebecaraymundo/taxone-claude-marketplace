## INTRODUÇÃO

Solução para Notas Fiscais Não Aparecendo no Convênio 115 de Filial Específica

### Problema

Ao realizar a apuração do Convênio 115 para uma filial específica, os documentos fiscais não estão sendo demontrados.

### Causa Raiz

O problema pode ser causado por uma falta de configuração adequada para a geração do Convênio 115, especificamente a configuração de forma de apresentação dos documentos.

### Resolução

Verificar se as configurações de geração do Convênio 115 estão corretas e completas para a filial e período em questão.

Realizar a configuração adequada para o período que falta na tela de configuração do Convênio 115.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/38753913483277)

Confirmar a configuração e realizar uma nova geração do convênio para verificar se o problema foi solucionado.

### Próximos Passos

Após realizar os ajustes recomendados, monitore o sistema para assegurar que as notas fiscais estão aparecendo corretamente na apuração do Convênio 115.

Caso o problema se repita, revisite as configurações e caso persista, consulte o suporte técnico novamente.

### Dicas e Recomendações:

Garanta que todas as filiais que necessitem, tenham as configurações necessárias válida para o período de apuração.

Considere realizar auditorias periódicas nas configurações do sistema para prevenir a ocorrência de erros na apuração de documentos fiscais.
