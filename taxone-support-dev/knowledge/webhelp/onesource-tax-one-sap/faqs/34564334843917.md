Se você está enfrentando dificuldades para salvar informações na tela de Configurações da DIRF, é possível verificar o motivo pelo qual as informações não estão sendo salvas. Para isso, você pode realizar uma consulta no banco de dados utilizando o seguinte comando SQL:

SELECT * FROM "tax4b.dirf::CD_PARAMETROS.CONFIGURACOES".

Nota: Este comando SQL deve ser executado no container do banco de dados da DIRF. Após executar a consulta, você pode enviar o resultado para o time de suporte técnico para uma análise mais detalhada.

Para enviar o resultado, é necessário abrir um ticket de suporte. Além disso, é recomendável realizar verificações periódicas das configurações e das versões dos objetos de apuração da DIRF para prevenir problemas futuros na geração de arquivos. Manter o sistema atualizado é essencial para garantir o bom funcionamento do aplicativo DIRF.
