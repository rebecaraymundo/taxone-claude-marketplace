**Problema**

Não esta gerando o valor do campo 07 - VL_OPER_AQUIS no registro F130 no arquivo .txt do SPED Contribuições.

**Resolução**

A Shadow D_BESENCAR alimenta de forma correta tanto é gravada e o registro F130 do SPED.

É possível observar que o campo 07(VL_OPER_AQUIS) esta sendo gravado.

![F130.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/39125630220685)


O campo VL_OPER é fonte de dados tanto para VL_OPER quanto para VL_OPER_AQ conforme é possível observar no nó U_SHADOW_ERP da view BENS_INCORP_AT_IMOB do TDF.

![VL_OPER.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/39125630222221)

Nó U_SHADOW_ERP da view BENS_INCORP_AT_IMOB do TDF.
