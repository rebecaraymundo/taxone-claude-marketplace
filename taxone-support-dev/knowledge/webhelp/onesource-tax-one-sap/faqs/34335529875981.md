Para resolver o problema de geração incorreta de arquivos complementares para documentos extemporâneos no aplicativo EGOVERNO, siga os seguintes passos:

1\. Verifique se existem informações cadastradas na apuração de ISS em Documentos de arrecadação - Documentos extemporâneos. A ausência de dados cadastrados pode resultar na geração de um arquivo vazio ou com informações incorretas.

![Captura de tela 2025-02-19 072750.gif](https://suporte.thomsonreuters.com.br/hc/article_attachments/34392491782029)

2\. Confira se as versões dos objetos de apuração de ISS (Iss-db, iss-odata, iss-ui, iss-js) estão atualizadas. Caso não estejam, atualize-os para garantir a correta funcionalidade do sistema.

3\. Após atualizar os objetos, regenere o arquivo complementar e verifique se as informações contidas estão corretas.

4\. Continue monitorando a geração dos próximos arquivos complementares para assegurar que o problema foi resolvido. Se o problema persistir, abra um novo ticket de suporte para uma análise mais detalhada.

5\. Realize verificações periódicas das configurações e das versões dos objetos de apuração de ISS para prevenir futuros problemas na geração de arquivos.
