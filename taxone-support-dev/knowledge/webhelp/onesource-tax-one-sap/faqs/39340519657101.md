# [Apuração de impostos indiretos][ICMS]TAX ONE FOR SAP - Como parametrizar o código RJREC00001 no Registro 1400 para filiais do estado do RJ

## Problema:

O código RJREC00001, necessário para o preenchimento do registro 1400 no estado do RJ, não está disponível para uso no aplicativo de parametrização fiscal Tax One for SAP.

## Causa Raiz:

O código RJREC00001, válido a partir de maio de 2023, é referente à receita bruta anual do estabelecimento e deve ser informado apenas na declaração de dezembro. O sistema atual utiliza fontes como notas fiscais e cupons fiscais para a automatização, e o registro com este código específico depende do carregamento manual de dados via planilha XLSX.

## Resolução:

  1. Certifique-se de que o código RJREC00001 está sendo utilizado para a receita bruta anual do estabelecimento.
  2. Para incluir o registro 1400 com este código, prepare os dados em um arquivo XLSX (tabela shadow).
  3. Utilize a funcionalidade de carga de dados via planilha XLSX disponível para o DRC/TDF no sistema Tax One for SAP.
  4. Envie os dados para o sistema e valide os registros gerados conforme as orientações do manual de preenchimento do registro 1400.


## Próximos Passos:

Após carregar os dados na tabela shadow e gerar os registros, revise os documentos e certifique-se de que o código RJREC00001 foi aplicado corretamente na filial correspondente do estado do RJ. Após finalizar, envie o relatório para a declaração de dezembro e mantenha monitoramento constante para garantir que o preenchimento está alinhado com as orientações fiscais.

## Dicas e Recomendações:

  * Leia atentamente o manual de orientação de preenchimento do registro 1400, especialmente as seções específicas para o estado do RJ.
  * Organize previamente os dados em planilhas XLSX para facilitar o carregamento no sistema.
  * Mantenha a versão dos componentes do sistema Tax One for SAP atualizada para evitar erros de compatibilidade.
  * Considere criar relatórios periódicos que ajudem na identificação de falhas ou inconsistências nos registros.
  * Em caso de dúvidas, consulte a equipe de suporte da Thomson Reuters para orientações adicionais.


Para informações adicionais sobre a funcionalidade de carga de dados via XLSX, consulte a documentação oficial do Tax One for SAP ou acesse a base de conhecimento da Thomson Reuters.
