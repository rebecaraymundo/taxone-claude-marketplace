O processo de Ordens de Produção e Leitura de Views no ONESOURCE TAX ONE FOR SAP é realizado da seguinte forma:

  1. Acesse _Obrigação Estadual > FCI _no sistema.
  2. O ONESOURCE TAX ONE FOR SAP utiliza as seguintes VIEWS STANDARDs da SAP:
     * NF_DOCUMENTO_ITEM_IMPOSTO
     * ITEM
     * CONSUMO_ESPECIFICO
  3. As configurações gerais para as Ordens de Produção são realizadas dentro do app FCI.


![FCI.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/28453546946829)

  1. Após as configurações, o sistema consome as seguintes views para obter os dados:
     * 1º Produto Produzido:
       * SELECT * FROM "_SYS_BIC"."sap.glo.tmflocbr.ctr/ITENS_PRODUZIDOS"
     * 2º Os insumos que foram utilizados para produzir esse item:
       * SELECT * FROM "_SYS_BIC"."sap.glo.tmflocbr.ctr/INSUMOS_CONSUMIDOS".
