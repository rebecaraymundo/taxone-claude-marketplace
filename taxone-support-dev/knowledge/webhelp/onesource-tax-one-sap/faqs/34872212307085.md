# [Sped][Sped Fiscal]TAX ONE FOR SAP - Atualização para a Versão 19 do SPED ICMS IPI.

- **Criado:** 2025-03-10
- **Atualizado:** 2025-12-08

---

**Problema:**

O cliente precisa atualizar o sistema TAX ONE for SAP para a versão 19 do SPED ICMS IPI, para estar em conformidade com as novas versões que serão implementadas a partir de 2025.

**Causa Raiz:**

A necessidade de atualização surgiu devido à disponibilização de um novo leiaute (código 019) para o ano de 2025, o qual foi comunicado mas ainda não foi implementado pelo cliente.

**Resolução:**

Acesse o repositório de atualizações do TAX ONE for SAP.

Localize o objeto correspondente ao leiaute 019, que foi disponibilizado em 15/01/2025.

Siga as instruções de atualização fornecidas no Informativo V01.R131 para implementar a versão mais recente do objeto.

Confirme a atualização e teste o sistema para garantir que a nova versão está funcionando corretamente.

**Próximos Passos:** Após a atualização, continue monitorando as comunicações do suporte para futuras atualizações e melhorias. Verifique periodicamente se há novos leiautes ou requisitos legais que possam afetar o sistema.

**Dicas e Recomendações:** Sempre mantenha seu sistema atualizado com os objetos mais recentes disponibilizados para evitar problemas de conformidade fiscal. Fique atento às comunicações do suporte e participe de treinamentos oferecidos para entender melhor as atualizações e como elas afetam seu trabalho.
