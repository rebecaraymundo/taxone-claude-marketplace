**Problema:**

Dúvida em relação à geração dos registros 1100 e 1105 (Informações sobre Exportação) no produto TAX ONE FOR SAP.
**Resolução:**

Os dados de exportação não são standard SAP, cabendo um software complementar para gerenciar essas informações.

Deve ser avaliado um tratamento para popular as tabelas shadow disponibilizadas pela SAP.

**Próximos Passos**

O cliente possuiu o OSGT? Se, sim deverá ser retornado a informação para o time de Suporte seguir na avaliação do tema.

Caso não tenha, seguir com as orientações informadas acima.
