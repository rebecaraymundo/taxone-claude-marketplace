Passo a passo para validação e configuração de unidade de medida no ONESOURCE TAX ONE FOR SAP:

  1. **Validação inicial:**

     * O sistema valida a unidade de medida da nota fiscal em relação ao inventário.
     * Se as unidades de medida forem iguais, a validação é aprovada (OK).
  2. **Verificação adicional:**

     * Se as unidades de medida forem diferentes, o sistema realiza uma verificação adicional:
       * Primeiro, verifica a configuração específica no sistema.
       * Caso não exista configuração específica, o sistema consulta a view FATOR_CONVERSAO_UNIDADE.
  3. **Necessidade de conversão:**

     * Em qualquer um dos passos acima, é necessário que exista uma conversão de unidade de medida configurada para que o sistema possa realizar a verificação corretamente.
  4. **Configuração padrão do SAP:**

     * Mesmo que não haja diferença na unidade de medida, o fator de conversão deve estar configurado (configuração padrão do SAP).
     * Essa configuração é essencial para o funcionamento correto do sistema.
  5. **Regra de configuração:**
     * O fator de conversão deve estar configurado na view padrão FATOR_CONVERSAO_UNIDADE ou em configuração específica do sistema.
     * Se o fator de conversão não existir na view padrão, é necessário realizar a configuração manualmente no ONESOURCE TAX ONE FOR SAP.
  6. **Importância da configuração:**
     * A configuração correta é essencial para que o sistema identifique se há ou não diferença na unidade de medida.
     * Sem essa configuração, o sistema pode não funcionar corretamente, resultando em erros ou inconsistências.


### Dicas e recomendações:

  * Garantir que os fatores de conversão de unidade de medida estejam corretamente configurados é crucial para o funcionamento adequado do sistema.
  * Siga os passos descritos acima para assegurar que todas as unidades de medida sejam validadas e convertidas corretamente.
  * Em caso de dúvidas, consulte a documentação disponível na Central de Ajuda ou entre em contato com o suporte.
