# [Obrigações estaduais][DECLAN RJ] ONESOURCE TAX ONE FOR SAP - DECLAN RJ - Como aplicar o filtro 'Base de cálculo' para os registros 0300 e 0400?

  1. Acesse  _Obrigações estaduais > DECLAN RJ > Botão Configurações > Registros 0300 e Registros 0400_
  2. Na configuração da DECLAN RJ referente aos "Registros 0300 e 0400", é possível incluir um filtro de "Base de cálculo". Este filtro é opcional e, ao ser configurado, determina quais documentos fiscais serão considerados com base nos critérios estabelecidos:
     1. **NULO (padrão):** Não aplica o filtro de base de cálculo e considera todos os documentos fiscais conforme os outros filtros previamente configurados.
     2. **Base Tributada ICMS:** Inclui apenas os documentos fiscais que apresentam valor de base de cálculo tributada do ICMS maior que zero.
     3. **Base Outras ICMS:** Considera somente os documentos fiscais com valor de base de cálculo 'Outras' do ICMS maior que zero.
     4. **Base Isentas ICMS:** Filtra e considera apenas os documentos fiscais que indicam valor de base de cálculo isenta do ICMS maior que zero.
     5. **Base Outras e Base Isentas ICMS:** Inclui os documentos fiscais que possuem valor na base de cálculo 'Outras' do ICMS e/ou base de cálculo isenta do ICMS, ambas maiores que zero.
  3. A parametrização desse filtro deve ser realizada conforme a necessidade do cliente.
