Para o Tax One For SAP calcular corretamente o valor do fator de crédito CIAP e considerar todas as notas fiscais parametrizadas, siga as orientações:

  1. Para considerar "SAÍDAS BASE OUTRAS", o sistema só inclui essas notas quando a configuração “considerar notas com outras bases em saídas tributadas” estiver definida como “SIM”. Se os CSTs estiverem parametrizados, o sistema filtra apenas pelas notas com os CSTs especificados. Se nenhum CST estiver especificado, todas as notas são consideradas. Os CFOPs utilizados devem ser os mesmos que estão parametrizados nas operações de saídas.
  2. "DEVOLUÇÕES" são consideradas apenas se as configurações relevantes estiverem definidas como “SIM”. Os CFOPs parametrizados nas operações de devoluções são considerados.
  3. Cálculo das SAÍDAS TOTAIS: O sistema filtra as notas usando os CFOPs parametrizados na operação de SAÍDAS através do campo COD_NAT da view NF_DOCUMENTO_ITEM_IMPOSTO e pelo CST ICMS. Se nenhum CST estiver configurado, todos são considerados. O cálculo para o Fator CIAP é realizado da seguinte forma:
(SAÍDAS TRIBUTADAS + EXPORTAÇÃO + SAÍDAS BASE OUTRAS - DEVOLUÇÕES) / (SAÍDAS TOTAIS - DEVOLUÇÕES)
  4. Para auxiliar na conferência das informações, pode-se utilizar o relatório exportável "CIAP - Relatório do Fator de Crédito".
  5. Após realizar as alterações recomendadas, é aconselhável executar um novo cálculo para garantir que o valor do fator de crédito CIAP esteja correto.
  6. Monitore os resultados e entre em contato com o suporte caso haja discrepâncias.
  7. É importante manter as configurações do Tax One For SAP atualizadas de acordo com as operações fiscais da sua empresa e verificar periodicamente se todos os CFOPs necessários estão corretamente parametrizados para evitar problemas nos cálculos fiscais.
