  1. Acesse: _Gestão de Dados > Documento de Arrecadação_
     * Ao enviar o documento de arrecadação para a Sefaz é apresentada a seguinte mensagem de erro: _" TypeError: Cannot read property 'api_url' of undefined"_


  1. Para este cenário é necessário ser avaliado dentro do Cockpit XSA o conteúdo do campo 'USER PROVIDER'.


![3_documento de arrecadacao.PNG](https://suporte.thomsonreuters.com.br/hc/article_attachments/24489514501645)
