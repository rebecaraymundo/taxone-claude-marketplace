# Procedimentos para Inserir Texto na Contabilização Automática e Manual no SAP

### Problema:

Usuários enfrentam dificuldades para inserir textos na contabilização automática e manual no SAP, gerando dúvidas sobre o momento e local correto de inserção.

### Causa Raiz:

Em que momento o texto deve ser inserido antes ou depois da contabilização automática?
Em qual tabela é gravado os textos?
Edição de textos é somente para os casos de ajustes manuais?

### Resolução:

  * Para contabilização manual, habilite e informe o texto no campo “Identificação do cenário” antes de realizar a contabilização. Este campo é utilizado no histórico do documento contabilizado.


  * Para contabilização automática, o texto também deve ser inserido antes de iniciar o processo de contabilização.


  * Os textos são registrados e podem ser consultados na transação FB03 no SAP.

### Próximos Passos:

Após inserir o texto conforme as instruções, prossiga com a contabilização. Monitore o histórico do documento para garantir que o texto foi adicionado corretamente.

### Dicas e Recomendações:

Considere revisar regularmente os procedimentos de contabilização para garantir que estão atualizados e alinhados com as necessidades do negócio. Lembre-se de verificar as versões atuais dos componentes do sistema, como contab-db, contab-backend e contab-ui, para garantir compatibilidade e performance adequada.
