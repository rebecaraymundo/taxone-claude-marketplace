# [Apuração de impostos retidos] ONESOURCE TAX ONE FOR SAP - IRRF - Comportamento do Campo Código de Retenção(Cód. Receita) na Tela Documento Retido

**Problema:**

O cliente enfrenta um problema onde alterações no campo "Código de Retenção" na tela Documento Retido, onde não estão sendo mantidas após a edição do registro.

**Causa Raiz:**

Identificamos que o problema ocorre devido a uma falha no comportamento do campo "Código de Retenção", que não preserva o valor alterado após ser editado. Isso se deve a uma questão de configuração na exibição dos dados na tela.

**Resolução:**

O campo "**Cód. receita** " da tela "**Documento retido** " refletirá a alteração no campo 'CHAVE_IRF' da shadow "/TMF/D_NF_IMPWH".

![1-iss retido.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/37711866632845)![2 iss .png](https://suporte.thomsonreuters.com.br/hc/article_attachments/37711910156685)

Porém a determinação do código de receita na apuração dos impostos retidos do TAX14SS é definida baseado na configuração que relaciona a **categoria IRF/código IRF** ao **código da receita.**

![3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/37711910161165)

Portanto, mesmo que corrija, a alteração do campo Cód. Receita da shadow, via tela Documento retido, a mudança não será refletida na apuração. Pois o documento continurá com a mesma categoria a código de imposto.

A tela não permite, pois a categoria é campo chave:

![4.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/37711866637837)

**Próximos Passos:**

Após realizar os entendimentos conforme descrito acima, é recomendável monitorar o sistema para garantir que a correção foi efetiva e que não há outras incidências relacionadas. Caso o problema persista, entre em contato com o suporte técnico para mais assistência.

**Dicas e Recomendações:**

É importante sempre verificar as configurações de campos editáveis para garantir que as alterações sejam refletidas corretamente em todas as telas e relatórios. Mantenha os sistemas sempre atualizados com os últimos patches de segurança e melhorias de funcionalidades disponibilizados para evitar problemas similares no futuro.
