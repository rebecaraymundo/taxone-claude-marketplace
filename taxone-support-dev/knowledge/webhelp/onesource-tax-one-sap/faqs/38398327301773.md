## Problema

Usuários enfrentaram um problema onde ao adicionar um novo item na tabela T002 - Códigos de Atividades, o último registro existente desaparecia.

## Causa Raiz

A causa deste problema foi identificada como uma falha na atualização do módulo de backend do aplicativo SIMP, relacionada ao gerenciamento de registros da tabela.

## Resolução

Uma correção foi aplicada ao módulo backend do aplicativo SIMP com a atualização para a versão 1.143.1.

Com essa atualização, todos os registros agora permanecem visíveis na tabela após a inclusão de novos itens.

Para aplicar a correção, siga os passos abaixo:

  * Verifique a versão atual do seu módulo SIMP e confirme se a correção é aplicável.

  * Instale a atualização tax4b.i_simp-backend_1.143.1.mtar disponibilizada por nossa equipe de suporte.

  * Reinicie o sistema aplicativo SIMP para garantir que as mudanças tenham efeito.


## Próximos Passos

Após realizar a atualização, monitore o sistema para garantir que o problema não persista. Caso necessite de assistência adicional, entre em contato com a equipe de suporte da Thomson Reuters.

## Dicas e Recomendações

É recomendável manter sempre o aplicativo SIMP atualizado com as últimas versões do software. Isso garantirá a otimização de recursos e a correção de erros prévios, contribuindo para a estabilidade do sistema.
