# [Obrigações Municipais]TAX ONE FOR SAP - Resolução de Problema com a Exibição do Município no Aplicativo ISS

## Problema:

O campo 'Município' não apresenta o município de São Paulo configurado no aplicativo ISS, impactando nos testes unitários.

## Causa Raiz:

A situação ocorre devido a uma configuração específica no uso dos códigos de municípios no ERP do cliente, onde o terceiro caractere do campo 'TAXJURCODE' na tabela 'J_1BTXJURT' foi suprimido, desviando do padrão esperado que é ‘UF + ESPAÇO "EM BRANCO" + CÓDIGO IBGE COMPLETO’. Esse tratamento impacta na busca das informações pelo Tax One for SAP.

## Resolução:

  1. Acesse o SAP e verifique o formato dos códigos municipais na tabela 'J_1BTXJURT'.
  2. Garanta que o formato esteja como 'UF + ESPAÇO EM BRANCO + CÓDIGO IBGE COMPLETO', por exemplo: 'SP 3550308'.
  3. Caso necessário, corrija os códigos para se adequarem ao padrão mencionado.
  4. Teste as configurações realizadas acessando o aplicativo ISS novamente.


## Próximos Passos:

Após realizar as alterações, monitorar o sistema para garantir que o problema foi totalmente resolvido. Se o problema persistir, entre em contato com o suporte técnico para uma análise mais aprofundada.

## Dicas e Recomendações:

Mantenha sempre as configurações do ERP atualizadas e em conformidade com os padrões estabelecidos para evitar problemas de compatibilidade com os aplicativos. Considere realizar verificações periódicas nas configurações para prevenir futuros problemas similares.
