# Atualização Correta dos Componentes usando o Instalador Automático

## Problema:

Release x Repositório

Identificar quais versões precisam ser atualizadas para que todos os componentes fiquem alinhados.

## Causa Raiz:

As liberações dos aplicativos são feitas pelos serviços liberados durante uma sprint (15 dias). Atualmente não é listado todos os micro serviços existentes dentro da Release, apenas os que tiveram alterações. Se foi realizado um desenvolvimento apenas no micro serviço DB, apenas esse serviço estará no documento de release.

Para o exemplo utilizado no chamado, que é o Difal DB os outros aplicativos Difal, e seus pré-requisitos que devem estar instalados, são os disponibilizados na própria release 119 ou em releases anteriores (maior versão disponível anterior a versão 119), pois no momento dessa liberação, foi com essas versões que a obrigação foi testada e validada.

A recomendação é que a instalação seja sempre feita pelo atualizador. Isso garante que a versão mais atual do aplicativo, com suas melhorias e correções, esteja instalada, além de assegurar que seus pré-requisitos também estejam atualizados, garantindo assim o funcionamento do produto. Qualquer instalação realizada manualmente, utilizando pacotes disponibilizados nas releases 119 ou 120 (atualmente o produto trabalha nas versões 123), pode sim impactar negativamente a utilização do produto. Isso ocorre porque, sem o uso do instalador, há um risco considerável de falhas não mapeadas ao não aplicar os pré-requisitos de um aplicativo, como é o caso do Difal DB.

**Nota** : Para liberações mais recente, verificar Informativo do Produto TAX ONE FOR SAP através da Central de Ajuda do ZENDESK WebHelp.

## Resolução:

  * Acesse o instalador automático fornecido pela empresa.
  * Execute o instalador. Este processo garantirá que todos os componentes necessários sejam atualizados para a última versão disponível.
  * O instalador também irá verificar e instalar quaisquer pré-requisitos necessários para o funcionamento adequado dos componentes atualizados.
  * Confirme a conclusão da atualização após o término do processo de instalação.


## Próximos Passos:

Após a atualização, é importante monitorar o desempenho dos aplicativos para garantir que tudo está funcionando como esperado. Em caso de qualquer irregularidade, entre em contato com o suporte técnico imediatamente.

## Dicas e Recomendações:

Recomenda-se sempre utilizar o instalador automático para atualizações, pois isso minimiza o risco de erros e garante que todos os componentes e pré-requisitos estejam corretamente atualizados. Evite fazer atualizações manuais, a menos que seja estritamente necessário e você tenha conhecimento técnico suficiente para gerenciar as dependências.
