Para visualizar os Códigos de Receita:

  1. Acesse _Apuração Impostos Retidos > PIS/COFINS/CSLL - Apresentação Códigos de Receitas > Configurações Gerais_
  2. Os códigos de receitas são apresentados de acordo com as as tabelas do SAP T059Z e T059ZT. Além disso são filtrados os campos LAND='BR'e SPRAS='P' para que seja apresentado no sistema TAX ONE FOR SAP.![1-pcc.PNG](https://suporte.thomsonreuters.com.br/hc/article_attachments/20878271062669)
