## Problema de Apuração PIS/COFINS Não Disponível no TaxOne For SAP

### Problema:

Usuário não consegue visualizar os valores para o fechamento fiscal dos meses de abril e maio de 2025 no sistema TaxOne For SAP.

### Causa Raiz:

A indisponibilidade dos meses de Abril e Maio para visualização e processamento no sistema ocorre porque o mês anterior (março) foi reaberto. Para que um novo mês seja liberado, o mês anterior precisa estar encerrado.

### Resolução:

  * Verifique se o mês de Março no sistema está devidamente encerrado.
  * Se Março estiver reaberto, proceda com o encerramento deste mês.
  * Uma vez que Março esteja encerrado, Abril será automaticamente apresentado e quando Abril estiver encerrado e Maio ficará disponível para processamento e assim sucessivamente.
  * Caso encontre dificuldades para encerrar o mês de março, contate o suporte técnico para assistência adicional.


### Dicas e Recomendações:

Para evitar problemas de visualização e apuração futuros, sempre certifique-se de que todos os meses estão corretamente encerrados antes de avançar para o próximo período. Isso garante a continuidade e a integridade dos dados no sistema TaxOne For SAP.
