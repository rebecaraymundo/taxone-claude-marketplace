## Problema

HERMES - Erros na aplicação BTP x DRC/S4

## Resolução

Seguem os passos para verificar e corrigir possíveis erros na replicação do ONESOURCE TAX ONE FOR SAP - HERMES:

  1. Verifique se as triggers para a tabela foram criadas corretamente, conforme demonstrado no portal [https://onesource4sap-repository.thomsonreuters.com/content/raw/installer/hana_cloud/triggers_drc.sql](<https://onesource4sap-repository.thomsonreuters.com/content/raw/installer/hana_cloud/triggers_drc.sql> "Hyperlink \[https://onesource4sap-repository.thomsonreuters.com/content/raw/installer/hana_cloud/triggers_drc.sql\]").
  2. Com as triggers criadas, acesse os logs para verificar se estão ocorrendo erros de transporte na execução da replicação da tabela, considerando a seguinte instrução SQL:
     * SELECT * FROM "tax4b.hermes::CD_LOG.DELTA" WHERE TABLE_NAME = '/TMF/D_DEM_DOCS';7


## Próximos Passos:

Caso o cenario persista, verifique o seguinte passos:

  *     * A comunicação com o Cloud Connector está funcionando?
    * No ambiente S/4 está chegando a requisição corretamente?
    * No S/4 está apresentando algum erro no recebimento?
    * ##

Efetue o select abaixo e envie para a equipe de Suporte efetuar as devidas analise:
    * SELECT * FROM "tax4b.koine::CD_AGNE.LOG" WHERE app = 'HERMES';
