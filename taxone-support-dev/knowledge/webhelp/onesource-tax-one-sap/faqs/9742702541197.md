Não existem views da SAP disponibilizadas diretamente para Obrigações Estaduais no caminho "Content >>SAP >> glo >> tmflocbr".

O Produto ONESOURCE TAX ONE FOR SAP SOLUTION desenvolveu as views das Obrigações Estaduais que consomem as views da SAP.

Para acessar essas views do ONESOURCE TAX ONE FOR SAP SOLUTION é necessário incluir as conexões para cada uma das aplicações no HANA, ou seja, para cada uma das obrigações estaduais temos uma conexão específica no HANA.

Para incluir essas conexões no HANA, o Basis tem que consultar o XSA do ambiente e incluí-la. Dentro dessas conexões haverão diversas views que são consumidas por cada uma das Obrigações Estaduais, por isso, é necessário o conhecimento técnico para saber as conexões entre essas views.

Para realizar a conferência das Obrigações Estaduais a recomendação é utilizar o title de [Apuração de ICMS](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One4SAP/Web_Help_ONESOURCE_TAX_ONE_4SAP/Manuais_Funcionais/Apuracao_impostos_indiretos/ICMS/ICMS.htm>), relatórios ou até mesmo a criação de relatórios (Gestão de dados).
