## Causa Raiz

Erro no encerramento do processo de apuração de ISS, que apresenta status "em processamento" sem concluir.

## Resolução

Para avaliar de forma aprofundada o tema, deverá ser solicitado:

  1. Utilize os comandos listados abaixo e envie para suporte técnico para análise:


SELECT * FROM "tax4b.iss::CD_STATUS.LOG_EXECUCAO" WHERE mandt = '?' AND empresa = '?' AND ano = '?' AND periodo = '?';
SELECT * FROM "tax4b.iss::CD_STATUS.STATUS_OBRIGACAO" WHERE mandt = '?' AND empresa = '?' AND ano = '?' AND periodo = '?';

## Próximos Passos

Encaminhe os devidos resultados dos selects para o time de Suporte Técnico avaliar.
