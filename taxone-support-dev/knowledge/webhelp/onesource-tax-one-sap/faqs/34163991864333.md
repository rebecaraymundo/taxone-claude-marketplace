# Correção do Campo CNPJ/CPF Transportador no Registro 45

## Problema:

Usuários relatam que o campo CNPJ/CPF Transportador (Campo 12 do Registro 45) está sendo preenchido incorretamente com '0' ou vazio quando deveria estar em branco ou conter um número válido, resultando em erros de layout nos arquivos SCANC.

## Causa Raiz:

A causa raiz do problema foi identificada como um mapeamento incorreto no sistema que gerencia o preenchimento dos campos CNPJ/CPF Transportador no Registro 45, gerando assim erros de layout.

## Resolução:

  * Atualize os seguintes objetos para corrigir o mapeamento:


  *     * tax4b.scanc-backend_1.131.1.mtar


  * Verifique no validador se o campo 12 do Registro 45 está sendo preenchido corretamente e não consta mais o '0' ou vazio indevido.


## Próximos Passos:

Após a atualização dos objetos, é crucial realizar testes para garantir que o campo CNPJ/CPF Transportador esteja sendo preenchido corretamente. Caso o erro persista, entre em contato com o suporte técnico para mais investigações.

## Dicas e Recomendações:

Assegure-se de manter seu sistema sempre atualizado com as últimas correções e patches de segurança para evitar problemas similares no futuro. Consulte regularmente a documentação do sistema para entender as melhores práticas de preenchimento de dados e configuração.
