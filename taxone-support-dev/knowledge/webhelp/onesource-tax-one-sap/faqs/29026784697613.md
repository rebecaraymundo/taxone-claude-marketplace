  1. Acesse: _Apuração Impostos Retidos > ISS_

  2. No módulo de apuração do ISS, ao tentar contabilizar "Parcialmente", é apresentada a mensagem: "Há Guias não Estornadas. Favor Estornar!"

  3. A aplicação permite contabilizar os Municípios selecionados. No entanto, quando se tenta contabilizar por Município, a aplicação exige que as guias sejam estornadas primeiro.

  4. **A apuração do ISS possui o seguinte comportamento:**

A opção "Contabilizar Todos" e "Estornar Todos" foi criada para que, após todos os encerramentos, esses processos sejam realizados de uma única vez.

A função dos botões “**Contabilizar Todos** ” e “**Estornar Todos”** é contabilizar ou estornar todos os documentos de arrecadação das apurações encerradas. Caso haja alguma contabilização no período, o sistema possui uma trava para garantir que valores não sejam contabilizados em duplicidade. Por isso, apresentamos a mensagem informando que há guias a serem estornadas.

No caso de encerramento parcial, é recomendado que a contabilização seja realizada individualmente dentro de cada apuração. Alternativamente, é possível vincular a contabilização ao encerramento, eliminando a necessidade de contabilizar individualmente.

Esse procedimento pode ser configurado no módulo _Gestão de Dados > Contabilização Automática._
