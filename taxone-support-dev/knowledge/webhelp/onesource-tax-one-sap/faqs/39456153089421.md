## Problema

Ao tentar acessar as configurações da apuração do ISS, a tela para selecionar o tipo de taxa de juros não abre ao clicar no ícone "Editar".

## Causa Raiz

Este problema pode estar relacionado ao cache presente no navegador utilizado, que pode impedir o carregamento adequado da funcionalidade.

## Resolução

Siga os passos abaixo para solucionar o problema:

  1. Primeiramente, limpe o cache do navegador. Caso não saiba como realizar esse procedimento, consulte as instruções específicas para o navegador que você utilizou (Google Chrome, Mozilla Firefox, etc.).
  2. Depois de limpar o cache, atualize a página e tente novamente acessar as configurações da apuração do ISS.
  3. Clique no ícone "Editar" e confirme se a tela para configurar a taxa de juros está sendo exibida corretamente.


## Próximos Passos

Após realizar os procedimentos acima e verificar que o problema foi resolvido:

  1. Configure o tipo de taxa de juros conforme sua necessidade.
  2. Monitore o comportamento da funcionalidade para garantir que ela esteja operando normalmente.
  3. Caso o problema persista, entre novamente em contato com o suporte para uma análise mais aprofundada.


## Dicas e Recomendações

  * Mantenha seu navegador atualizado para garantir o funcionamento adequado de sistemas e ferramentas. Navegadores desatualizados podem causar incompatibilidades.
  * Se possível, configure uma limpeza periódica de cache no navegador para evitar problemas de carregamento.
  * Consulte nossa base de conhecimento para encontrar artigos sobre outros possíveis problemas e soluções.
