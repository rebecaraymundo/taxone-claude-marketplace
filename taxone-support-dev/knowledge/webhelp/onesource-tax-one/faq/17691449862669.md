Nota: Este erro é identificado ao realizar uma carga no Interdados Oracle Cloud e no Log Processamento, quando na coluna Identificador é apresentado 'CONECTOR INTERDADOS TAXONE' e na coluna Mensagem apresentar a informação 'ERRO AO ENVIAR DADOS PARA O TAXONE'.

Para visualizar a mensagem do erro completa, clique nos três pontinho (...) da coluna Info.

“Erro: java.lang.RuntimeException:com.tr.interfaces.erp.taxone.exceptions.InterdadosException: LOGIN/PASSWORD TAXONE INVÁLIDO OU EXPIRADO com.tr.interfaces.erp.taxone.routes.SafxToQueueRoute$

SendToTaxOneProcessor.lambda

$process$0(SafxToQueueRoute.java:213)”

Para resolver este problema, **verifique se a senha do Tax One está expirada**. Para alterar a senha, siga o procedimento abaixo:

  1. Acesse o produto Interdados, menu **Opções > Parâmetros > Parâmetros de Configuração**.
  2. No fim da tela de Parâmetros de Configuração, no quadro **API Envio de dados** , altere a **Senha **e clique em Salvar.
  3. Realize uma nova carga.
