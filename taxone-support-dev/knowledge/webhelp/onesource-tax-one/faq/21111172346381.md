As origens das informações para geração da tabela SAFX13 são:

**Tabelas:**

  * ANLA
  * ANLB
  * ANLC
  * ANLP
  * ANLZ
  * RSEG
  * J_1BNFLIN
  * J_1BNFDOC


**Processos:**

  * ZSAFE036 – ARQUIVO DE CADASTRO DE BEM (Processo tratado pela Interface)
  * ZSAFE082 – ARQUIVO DE BENS DO ATIVO CIAP (Não tratado pela Interface)


**Parâmetros:**

  * Classe de Imobilizado: Parametrizar os códigos de Classe de Imobilizado do Ativo, é utilizada para geração do Cadastro de Bens. Somente serão gerados registros dos lançamentos que estiverem associados às classes inseridas nesta parametrização.
  * Classe Imob. Andamento CIAP: Parametrizar os códigos de Classe de Imobilizado do Ativo. É utilizada para geração do Cadastro de Bens. Somente serão gerados registros para essas tabelas, os lançamentos que estiverem associados às classes inseridas nesta parametrização.
