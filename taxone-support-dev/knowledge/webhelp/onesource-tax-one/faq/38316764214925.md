### **Lote Preso no Status FILA_TAXONE: Erro de Configuração Firm ID**

**Problema:** Um lote de integração enviado fica permanentemente no status **FILA_TAXONE** e não avança para o processamento final no sistema, dando a impressão de que a importação parou.

**Causa Raiz:** O problema é causado pelo preenchimento incorreto do campo **"Firm ID Tax One"** no cadastro das empresas no OBI. Este campo só deve ser preenchido se o ambiente possuir a funcionalidade de BPO, que permite unificar o acesso a mais de um ambiente em um único ambiente do Tax One. Se o campo for preenchido em ambientes padrão, o sistema tenta aplicar uma lógica de BPO, não a encontra, e o lote trava na fila.

**Resolução:** Para resolver o problema, o campo **Firm ID** deve ser limpo.

  1. Acesse o módulo de cadastros no OBI.
  2. Navegue até o cadastro das empresas envolvidas na integração.
  3. Localize o campo **"Firm ID Tax One"** e apague completamente o seu conteúdo, deixando-o vazio.
  4. Salve as alterações realizadas.
  5. Reenvie a carga para validar se o lote foi processado corretamente.
