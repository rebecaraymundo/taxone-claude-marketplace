Nota: Este erro ocorre no momento em que é realizada alguma carga no Interdados Oracle Cloud V17R01 e a mesma é finalizada com a mensagem de erro: '**Erro ao enviar dados para o TaxOne'** e, ao verificar o Log de Processamento, a mensagem encontrada é:

Erro: com.tr.interfaces.erp.taxone.exceptions.InterdadosException: 500 null.

Para solucionar o erro, siga o procedimento abaixo:

  1. Acesse o Interdados, clique em **Opções > Parâmetros > Parâmetros de Configuração**.
  2. No fim da página, na guia **API Envio de Dados** , verifique se a URL do campo **API URL** está configurada com uma barra (/) no final, conforme o exemplo:
**https://safx.prod.taxone.thomsonreuters.com**
     1. Se houver, remova essa barra (/) deixando a URL e clique em Salvar.
  3. Refaça a carga.
