Corrigindo o Erro "DWERRXSD" no Evento R2010 do Mastersaf DW/TAX ONE

Problema:

Erro "DWERRXSD" ao tentar enviar o evento R2010 para um fornecedor de transporte no sistema Mastersaf DW/TAX ONE.

Causa Raiz:

O erro ocorre devido à falta da tag "indCPRB" no XML do evento. A regra de geração desta tag exige que, se houver pelo menos uma nota fiscal com a alíquota de INSS igual a 3,5% (campo VLR_ALIQ_INSS), a tag "indCPRB" deverá ser gerada com valor ‘1’, caso contrário, o valor será ‘0’.

Resolução:

Verifique se todas as notas fiscais relevantes possuem a alíquota de INSS corretamente preenchida.

Equalize o período contábil no sistema.

Realize uma nova geração prévia do evento R2010.

Se o erro persistir, entre em contato com o suporte técnico para mais assistência.

Próximos Passos:

Após realizar a correção e uma nova tentativa de envio do evento, observe se o erro persiste. Se tudo ocorrer bem, continue com os próximos envios. Caso contrário, notifique o suporte técnico com as informações atualizadas.

Dicas e Recomendações:

É essencial manter as informações de alíquota de INSS atualizadas e corretas para evitar erros durante o envio de eventos. Recomendamos verificar regularmente a configuração dos dados fiscais, principalmente antes de executar eventos importantes.
