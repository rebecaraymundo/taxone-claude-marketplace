**PROBLEMA:**

Ao Integrar um XML usuário recebe mensagem de erro não mapeado ao formatar resultado.

**CAUSA RAIZ:**

O erro normalmente esta associado ao formato incorreto de uma data presente no XML que o sistema está tentando processar, mas também pode ocorrer em outros campos.


**SOLUÇÃO:**


**Verifique o formato do conteúdo do campo no XML** : O sistema vai apresentar o conteúdo preenchido no campo. Se for data ele requer que o formato de data esteja conforme especificado no manual de layout. É muito comum o campo estar preenchido com o texto '20250312T1624170300' trazendo a informação completa com data e horário e deve ser alterado para o formato correto.

Se preferir, o usuário poderá realizar a exportação do XML e do Processamento e encaminhar para que seja feita uma validação das informações pelo Time de Suporte.
