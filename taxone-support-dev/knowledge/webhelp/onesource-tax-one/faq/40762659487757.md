## Problema:

O usuário enfrenta problemas com o reprocessamento de itens na interface OBI, onde o sistema acaba reenviando toda a interface ao invés de apenas os itens com erro e a reciclagem não funciona para a safx52.

## Causa Raiz:

A interface OBI foi configurada para reprocessar e reenviar o lote inteiro de dados, não permitindo a seleção de registros específicos com erro para reprocessamento. Esta configuração é uma limitação atual do sistema.


A Reciclagem é um serviço do painel de integração responsável por reprocessar automaticamente os lotes que apresentaram falhas no processamento inicial, permitindo uma nova tentativa de execução após a correção dos erros identificados.

**Somente os tipos de SAFX listados a seguir estão habilitados para o processo de reciclagem no sistema:**

SAFX01 - Arquivo Contábil;
SAFX04 - Arquivo de Cadastro de Pessoas Físicas/Jurídicas;
SAFX2013 - Tabela de Produtos;
SAFX07 - Arquivo de Notas Fiscais;
SAFX08 - Itens Notas Fiscais Mercadorias e Produtos;
SAFX09 - Itens Notas Fiscais de Serviços;
SAFX112 - Observações da Nota Fiscal;
SAFX113 - Ajuste/Outros Valores do Lançamento Fiscal.

**Atenção:** Qualquer tentativa de reciclagem de SAFX fora desta lista não será permitida pelo sistema.

## Resolução:

  1. Acesse o sistema OBI.
  2. Verifique se a interface que você está trabalhando permite reciclagem. Se sim, utilize essa funcionalidade para reprocessar somente os dados necessários.
  3. Se a reciclagem não estiver disponível, será necessário reenviar todo o lote, conforme a configuração atual do sistema.
  4. Para sugerir mudanças ou melhorias no processo, acesse o portal Customer Influencer e registre sua sugestão. Link para Customer Influencer: <https://customerinfluencer.thomsonreuters.com>
