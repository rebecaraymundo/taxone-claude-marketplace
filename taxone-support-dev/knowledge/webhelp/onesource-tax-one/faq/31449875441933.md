**Para gerar o Convênio 115, siga as orientações a seguir:**

Verifique se a tabela SAFX431 (Itens Negativos de Documentos Fiscais de Utilities) foi importada corretamente.

**Nota:** Esta tabela é crucial para o carregamento dos itens negativos dos documentos fiscais (Utilities), contendo todas as informações necessárias para a elaboração do Convênio ICMS 115. Além dos itens negativos, os itens normais (não negativos) que foram consolidados em outros itens da nota original também devem ser incluídos.

**Atenção:** As informações contidas nesta tabela são destinadas exclusivamente para o Convênio ICMS 115, visando reproduzir os itens exatamente como aparecem no documento original, incluindo os itens negativos, quando aplicável.

  * Esta tabela deve ser utilizada apenas nos casos em que os dados foram carregados na SAFX43 com os valores dos itens já consolidados, seja por dedução ou adição, conforme necessário.
  * Adicionalmente, é essencial preencher o campo “N. Item Real” na SAFX43.


**Nota:** Para documentos fiscais que não apresentam itens com valores negativos e cujos dados foram importados para a SAFX43 contendo todos os itens originais, o uso desta tabela não é necessário.

**Nota:** Desde a implementação do Convênio ICMS 60/2015, esta tabela passou a ser padrão para a geração do arquivo magnético do Convênio 115, uma vez que a legislação começou a considerar a exibição de descontos como itens negativos no arquivo magnético.

Na elaboração do Convênio ICMS 60 (referente ao Convênio 115), é necessário que os valores de Descontos/Acréscimos sejam reproduzidos conforme constam na nota fiscal. Estas informações são derivadas da tabela SAFX431:

  * Os valores negativos devem ser reportados apenas nos itens consolidados (SAFX431), visto que a capa (SAFX42) e o item (SAFX43) dos documentos fiscais de utilities são empregados na geração de outras obrigações fiscais, nas quais os valores negativos não são aceitos, devendo ser considerados apenas na geração do Convênio 60/15. Este procedimento garante que todas as informações necessárias sejam apresentadas de forma correta e conforme os requisitos legais do Convênio ICMS 115.
