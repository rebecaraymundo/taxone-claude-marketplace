Sim, para editar um Fluxo é necessário clicar no botão NOVO. Rolando a tela para baixo será possível visualizar os Fluxos Criados e selecionando um deles será possível fazer sua edição. O botão editar não fica ativo se mais de um Fluxo for selecionado simultaneamente. É permitido ao usuário editar um Fluxo, adicionando ou excluindo tarefas. Não é permitido editar um Fluxo para eliminar todas as tarefas que existem nele. A Edição de Fluxo permite inclusive a edição do novo dado inicialmente ao Fluxo.
