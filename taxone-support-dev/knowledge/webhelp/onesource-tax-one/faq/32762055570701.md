# Resolução de Problemas de Acesso a Empresas no Sistema TAX One para Usuários Específicos

## Problema:

Usuários novos não conseguem visualizar as empresas no sistema TAX One mesmo com a configuração de acesso definida como "ALL".

## Causa Raiz:

O problema ocorre devido a uma limitação técnica do sistema TAX One, que não suporta logins de e-mail com mais de 40 caracteres. Esta limitação impede que os usuários com e-mails longos consigam acessar as empresas no sistema.

## Resolução:

  * Verifique o número de caracteres do e-mail do usuário. Se for superior a 40 caracteres, será necessário criar um novo login com menos caracteres.


  * Informe ao usuário para realizar o login com o novo usuário criado.


  * Teste para garantir que as empresas estão agora visíveis para o usuário.


## Próximos Passos:

Após realizar a mudança de login, monitore o acesso do usuário para garantir que não há mais problemas relacionados à visualização das empresas. Caso o problema persista, entre em contato com o suporte técnico para uma análise mais aprofundada.

## Dicas e Recomendações:

Para evitar problemas futuros com a limitação de caracteres, considere revisar e ajustar os e-mails de usuários que estão próximos ao limite de 40 caracteres. Isso garantirá uma experiência de usuário mais fluida e sem interrupções. Além disso, sempre verifique se todas as configurações de SSO estão corretamente ativadas para evitar desativações acidentais que possam impactar o acesso dos usuários.
