## Problema:

Os códigos de IRRF e PCC gerados para o Reinf apresentam dificuldades na geração de DCTF/MIT, enquanto outros códigos como PIS, COFINS e IPI não estão sendo incluídos corretamente.

## Causa Raiz:

A configuração atual no sistema não permite que alguns códigos específicos de DARF que são utilizados no Reinf sejam excluídos automaticamente para a geração do MIT.

## Resolução:

  * Acesse o sistema pela opção 'Básicos'.
  * Vá para 'Data Warehouse' e então 'Manutenção'.
  * Clique em 'Códigos - Código do Darf'.
  * Na página que aparece, localize o código de DARF que precisa ser ajustado (exemplo: 5952 para o Reinf).
  * Marque a opção "**DCTF (Desprezar)**" para o código de DARF selecionado.
  * Salve as alterações realizadas e prossiga com a geração do MIT.


## Próximos Passos:

Após realizar a configuração, monitore os próximos relatórios de DCTF/MIT gerados para garantir que todos os códigos necessários estejam sendo processados corretamente. Caso o problema persista, entre em contato com o suporte técnico para mais assistência.

## Dicas e Recomendações:

Recomenda-se consultar regularmente o manual do sistema ou as FAQs disponíveis no portal de suporte para espairecimentos adicionais sobre a configuração de códigos tributários. Ademais, manter o sistema sempre atualizado garante menor incidência de incompatibilidades ou erros na geração de dos relatórios fiscais.
