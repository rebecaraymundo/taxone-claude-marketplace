Passo a passo para corrigir o erro 90063 - 'Numero de CPF informado nao e valido, conforme regras da Receita Federal:

  1. Ao importar a SAFX07, este erro pode ocorrer se o campo CNPJ_ARMAZ_DEST contiver o código do fornecedor, em vez do CNPJ.
  2. Na SAFX, verifique se o campo CNPJ_ARMAZ_DEST está preenchido com o código do fornecedor. Se sim, altere para **@**.
Nota: O mesmo erro pode acontecer para o campo CNPJ_ARMAZ_ORIG. A solução é a mesma para ambos os campos.
