## Problema:

Usuários estão enfrentando problemas ao tentarem importar uma grande quantidade de usuários via planilha no Tax One, com o sistema exibindo o status "Complete with errors" sem especificar o erro concretamente.

## Causa Raiz:

O erro ocorre devido a inconsistências nos dados da planilha importada, o que leva o sistema a concluir a importação com erros, mas sem especificar as falhas na interface principal.

## Resolução:

Para resolver este problema e identificar os erros específicos, siga os passos abaixo:

  1. Após a tentativa de importação, baixe o arquivo com o status "Complete with errors".
  2. Abra o arquivo baixado e consulte a última coluna. Esta coluna destaca o erro encontrado durante a importação.
  3. Corrija os erros conforme indicado na planilha e tente realizar a importação novamente.
