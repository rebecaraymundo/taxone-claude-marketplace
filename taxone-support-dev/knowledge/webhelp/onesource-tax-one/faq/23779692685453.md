Quando executada a carga SAFX07 de saída (SAFX07s ou procedimento 57) verificou-se que o valor dos campos abaixo estão duplicados para todas as Notas:

  * VLR_BASE_PIS
  * VLR_PIS
  * VLR_BASE_COFINS
  * VLR_COFINS


A Oracle Cloud padronizou a criação de apenas um lançamento para cada imposto na integração com a Thomson Reuters, conforme o documento "Latin America Cloud Local Solution for Brazil (Doc ID 2291252.1) - File Name: LACLS_FISCAL_SOLUTION_PATH_V1".
