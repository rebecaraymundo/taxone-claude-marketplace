Exemplo de espaçamento entre todos os caracteres de uma SAFX:

0 0 1 0 1 2 4 D E S C R I C A O N O T A F I S C A L

Para um cenário como este, verifique a codificação que o SAP está configurado:

  1. Abra o SAP Logon.
  2. Clique com o botão direito do mouse na linha do ambiente que está apresentando o problema.
  3. Selecione a opção "Características".
  4. Na tela de características do ambiente, localize a opção "Code Page".
  5. Verifique se a codificação está configurada como "ANSI". Caso não esteja, altere para "ANSI".
  6. Clique em "OK" para salvar a alteração.


Após seguir esses passos, o espaçamento indesejado entre os caracteres nos arquivos TXT deve ser corrigido. A configuração da codificação como ANSI no ambiente SAP resolve esse tipo de problema de exibição de caracteres.

É importante ressaltar que essa configuração pode variar entre ambientes e usuários, o que explica por que em alguns casos o problema ocorre e em outros não.
