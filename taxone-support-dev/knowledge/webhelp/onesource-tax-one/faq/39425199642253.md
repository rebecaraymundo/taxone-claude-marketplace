O objetivo deste artigo é orientar o consultor\cliente como corrigir o problema credenciais de logon no Oracle Cloud após executar uma carga a mensagem do log apresenta “ERRO AO CRIAR JOB ORACLE CLOUD: NOVA TENTATIVA INICIADA”



### Problema:

Os clientes enfrentam dificuldades para integrar suas operações no INTERDADOS devido à questões relacionadas com as credenciais da Oracle Cloud.

A causa do problema está geralmente ligada à expiração das credenciais usadas para logar no Oracle Cloud conforme a mensagem “ERRO AO CRIAR JOB ORACLE CLOUD: NOVA TENTATIVA INICIADA”.
![](https://suporte.thomsonreuters.com.br/hc/article_attachments/39825716548749)

Após clicar no botão Info o cliente tem a confirmação sobre a credencial através da mensagem abaixo:
_**Client received SOAP Fault from server: java.lang.SecurityException: Failed to log into BI Publisher: invalid username or password. Please see the server log to find more detail regarding exact cause of the failure.**_

Se elas não forem atualizadas no sistema INTERDADOS, pode-se encontrar erros durante a geração dos dados.

### Resolução:

  1. Basta acessar o interdados, ir em Opções e depois clicar em Propriedades.
  2. Já em Configuração de Propriedades, coloque a palavra “CLOUD.HOST” no filtro.
  3. A coluna Valor é mostrada a URL do Oracle Cloud do cliente. Basta copiar a mesma e colocar em um Browser do Google Chrome ou Microsoft Edge.
  4. Acessando a URL, inserir seu usuário e senha. Se a senha estiver expirada, o sistema irá notificar e pedirá para ser alterada.
  5. Após atualizar sua senha pela URL, ir ao INTERDADOS clicar em Opções > Parâmetros de Configuração > INTEGRAÇÃO ORACLE CLOUD.
  6. Insira a nova senha no campo Senha Oracle Cloud e clique em Salvar.
  7. Realize uma nova carga para validar se o problema foi resolvido.


### Próximos Passos:

Após seguir os passos anteriores, verifique se o processo de carga no INTERDADOS é concluído sem erros. Caso continue enfrentando problemas, entre em contato com o suporte técnico para mais assistência.

### Dicas e Recomendações:

Recomenda-se verificar regularmente a validade das suas credenciais Oracle Cloud e atualizá-las no INTERDADOS para evitar interrupções no serviço. Além disso, manter as configurações de integração revisadas ajuda a prevenir problemas futuros.
