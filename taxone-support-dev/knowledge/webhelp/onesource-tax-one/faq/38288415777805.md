### **Envio de Dump via SFTP para Migração do DW para o Tax One.**

**Problema:** É necessário enviar o arquivo de dump do DW para o ambiente de produção, utilizando o protocolo SFTP, para iniciar o processo de migração para o Tax One.

**Resolução:** Para realizar o envio de forma correta e segura, siga os passos abaixo:

  1. **Geração de Credenciais (Pré-requisito):**
     * É obrigatório **abrir um ticket de suporte** solicitando a geração do usuário e senha para o envio do dump via SFTP. As credenciais são temporárias e específicas para o seu processo.
  2. **Configuração e Envio do Arquivo:**
     * Baixe e utilize o software de transferência SFTP **FileZilla**.
     * Configure os dados de acesso:
       * **Servidor:** `oms-sftp-prd.thomsonreuters.com`
       * **Porta:** `22`
       * **Usuário/Senha:** Utilize as credenciais geradas pelo time de suporte.
     * Conecte-se ao servidor e envie o arquivo dump **diretamente para a pasta raiz** (`/`).


**Próximos Passos e Recomendações:**

  * Após concluir o envio, **avise o time de suporte** no ticket original para que eles possam validar o arquivo e iniciar o processamento da carga.
