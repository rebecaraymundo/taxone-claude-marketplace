O Dump "CL_GUI_FRONTEND_SERVICES====CM014" ocorre quando o local de geração do arquivo SAFX (parâmetro P_PATH) não está acessível ou configurado incorretamente.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/29434463230349)

Passo a passo para solucionar o Dump "CL_GUI_FRONTEND_SERVICES====CM014" ao executar Job de extração e envio de SAFX para o WebService:

  1. Acesse as parametrizações do Job.
  2. No campo P_UNIX, modifique o local onde será realizada a gravação do arquivo para o sistema Unix.
  3. No campo P_PATH, informe um destino válido para gravação do arquivo. Uma opção é utilizar o mesmo caminho informado nos Parâmetros Gerais, aba Configuração path Unix. Outra opção é verificar o caminho correto através da transação AL11.
  4. Após ajustar os parâmetros P_UNIX e P_PATH corretamente, execute novamente o Job.
