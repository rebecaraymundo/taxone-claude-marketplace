Abaixo seguem os requisitos básicos sugeridos para implantação do V17 em ambiente on premise:

**Hardware (Sizing)**

  * **Aplicação** > Recomendado:
    * Processador: 4 processadores
    * Memória: 8 Gb para bases menores. Até 16Gb para maiores
    * HD: 60 Gb
    * Sistema Operacional: Linux Kernel 2.6 (ou superior) ou Windows Server
    * Java 11 ( Large Pages habilitado )
    * Tomcat 10.1.8


  * **Banco de Dados** > Recomendado:
    * Processador: 8 processadores
    * Memória: 64 Gb
    * HD: 500 Gb para bases menores. Até 1Tb para maiores
    * Sistema Operacional: Linux Kernel 2.6 (ou superior) ou Windows Server


Nota: Recomenda-se que o servidor seja dedicado ao Interdados V17 e que o servidor do banco de dados seja uma máquina separada.
