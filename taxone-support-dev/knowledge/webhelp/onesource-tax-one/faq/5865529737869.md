Acesse esta FAQ no [web help do produto](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/WebHelp/mergedProjects/FAQs/Administration.htm#adm_06>).

Esse erro pode ocorrer quando o módulo "Administration" é acessado com o Tax One configurado no idioma português. Para resolver, acesse o módulo de Administration utilizando o idioma em Inglês.

Notas:

  * O comportamento observado é esperado, pois o módulo "Administration" deve ser utilizado no idioma inglês. Este módulo não possui localização para outros idiomas, e os valores não são traduzidos.
  * Não há funcionalidade de localização para este módulo, e conforme informações obtidas junto à plataforma, não há previsão de inclusão dessa característica no roadmap.
  * Anteriormente, existia um erro que impedia até mesmo o cadastro de usuários quando o idioma estava configurado para português, mas esse problema específico já foi corrigido.
