# [Básicos][Job Servidor] TAX ONE  - Correção de Erro de Importação no Sistema SAFX devido a Espaços em Branco no Código do Bem

## Problema:

Erro ao importar dados na SAFX82 informando que o Bem não está cadastrado, mesmo com o cadastro feito corretamente na SAFX13 devido a espaços em branco no código do bem.

## Causa Raiz:

O problema ocorre porque há uma inconsistência entre os espaços em branco no código do bem no momento da criação do cadastro (SAFX13) e o momento da importação dos dados (SAFX82). Os espaços adicionais são interpretados como diferenças no código, levando ao erro de importação.

## Resolução:

  * Acesse a SAFX13 e localize o item cadastrado com o Código do Bem (11).
  * Analise o código do bem e remova todos os espaços em branco desnecessários – tanto anteriores, intermediários quanto posteriores ao código.
  * Salve as alterações após fazer a correção.
  * Tente importar novamente os dados na SAFX82 e verifique se o erro persiste.


## Próximos Passos:

Após ajustar os espaços no código do bem e reimportar os dados, monitore o sistema para confirmar que o erro foi resolvido completamente. Caso o problema persista, entre em contato com o suporte técnico para uma análise mais detalhada.

## Dicas e Recomendações:

Sempre revise os campos digitados para evitar espaços em branco desnecessários que podem causar erros. Considere utilizar funcionalidades de trim(corte de espaços excedentes) em sua rotina de cadastro, se disponível.

Consulte regularmente a documentação do sistema para melhores práticas de cadastro e importação de dados.
