O erro 90124: 'O código da pessoa Física/Jurídica não esta cadastrado' é gerado quando o sistema não consegue encontrar um código de pessoa física ou jurídica na ferramenta ou quando há uma discrepância no campo IND_FIS_JUR, indicando se o cadastro é de uma pessoa física ou jurídica.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/17113266123021)

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/17113314295693)

Para corrigir o erro, é necessário cadastrar a pessoa física ou jurídica no Data Warehouse. Isso pode ser feito manualmente ou importando os dados corretos através da SAFX04.

Se o cadastro já existe, mas o campo IND_FIS_JUR está incorreto, ajuste-o para que corresponda ao tipo de entidade correto (física ou jurídica) conforme registrado no sistema.
