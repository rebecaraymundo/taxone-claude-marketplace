# [Básicos][Job Servidor] TAX ONE - Resolução de Erro na Importação de SAFX2002: "Código da Conta Sintética não Cadastrado"

## Problema:

Erro ao tentar importar a SAFX2002 no Tax One, com a mensagem: "Código da Conta Sintética não cadastrado", mesmo utilizando um arquivo apenas com contas sintéticas.

## Causa Raiz:

O erro ocorre porque o código de conta sintética usado no campo sintético não corresponde a um código previamente cadastrado como conta no sistema. Isso resulta no sistema tentando importar e classificar a mesma conta como sintética simultaneamente, sem um registro prévio válido.

## Resolução:

  * Verifique o arquivo SAFX2002 que está tentando importar, focando nos códigos de conta e de conta sintética.


  * Certifique-se de que cada conta sintética utilizada (Campo 6 - COD_CONTA_SINT) tenha um correspondente já cadastrado como uma conta normal no sistema.


  * Se necessário, crie um registro "normal" para a conta antes de tentar importá-la como uma conta sintética.


  * Após ajustar o arquivo conforme necessário, tente realizar a importação novamente.


## Próximos Passos:

Após corrigir e importar o arquivo, verifique se o erro persiste. Em caso afirmativo, entre em contato com o suporte técnico para uma análise mais aprofundada.

## Dicas e Recomendações:

Para evitar esse tipo de erro no futuro, sempre confira se os códigos das contas e das contas sintéticas estão corretamente cadastrados no sistema antes de proceder com a importação de arquivos. Considere criar uma checklist para revisar todos os códigos antes de importar grandes volumes de dados.
