# [Básicos][Data Warehouse] TAX ONE - Código NBM existente não aparece ao cadastrar produto, como resolver?

Para garantir a correta validação de um produto com um código NBM (Nomenclatura Brasileira de Mercadorias) já existente, é importante seguir o procedimento de verificação da data de início de validade do produto em relação à data do código NBM. O código NBM deve ter uma data anterior ou, no máximo, igual à data de início de validade do produto. Veja como proceder:

1\. Ao cadastrar um novo produto, verifique a data de início de validade do mesmo.

2\. Realize uma pesquisa do código NBM para assegurar que a data associada a esse código seja anterior ou igual à data de início de validade do produto que está sendo cadastrado.

Por exemplo, uma pesquisa com a data 01/01/2005 pode mostrar vários códigos, incluindo o 8470155 com a data 01/01/2004, o que é aceitável.
Por outro lado, se for realizada uma pesquisa com uma data inferior à data do código NBM, o código 8470155 não será localizado devido à **restrição** de data.
