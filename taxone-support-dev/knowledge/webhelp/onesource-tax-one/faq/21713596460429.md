Quando identificada a necessidade de alterar o mapeamento de campo de uma SAFX específica onde o botão SALVAR está desabilitado:

  1. Verifique se existe uma regra de mapeamento "De-Para" para o campo que deseja alterar. Para isso:
  2. Dê um duplo clique sobre o campo em questão, observe a tela que se abre e localize o campo "**Regra De-Para"**.
  3. Se o campo "Regra De-Para" estiver em branco:
     1. Insira a regra específica para o campo em questão, se disponível.
     2. Caso não exista uma regra específica, digite "0 – Sem regra".
  4. Após inserir a regra apropriada ou "0 – Sem regra", o botão **SALVAR** será habilitado.
