Como lidar com códigos de serviço vetados em municípios específicos

**Problema:**

Durante a validação de arquivos na prefeitura, o usuário encontrou o erro "Código de serviço vetado". Esse problema ocorreu em um município específico, enquanto outros municípios não apresentaram o mesmo erro.

**Causa Raiz:**

O erro ocorre porque determinados códigos de serviço são vetados em certas prefeituras, conforme regulamentos municipais. Como resultado, esses códigos não devem ser considerados no meio magnético da declaração.

**Resolução:**

Siga as instruções abaixo para resolver o problema:

Acesse o agrupamento GissOnline no sistema TaxOne.

Procure pelo parâmetro relacionado a códigos de serviço vetados.

Ative a funcionalidade que impede que os códigos vetados sejam considerados em arquivos do meio magnético.

Teste a funcionalidade para assegurar que os códigos vetados são corretamente excluídos da declaração.

Caso já tenha eliminado manualmente os códigos vetados para o município específico, não será necessário realizar novas ações adicionais.

**Próximos Passos:**

Confirme com a prefeitura se a declaração gerada sem os códigos vetados foi validada sem erros.

Revisite os parâmetros configurados no agrupamento GissOnline periodicamente para garantir que estejam atualizados e atendendo às regulamentações locais.
