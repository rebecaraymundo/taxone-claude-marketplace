## Problema:

Usuário deseja saber se o sistema gera arquivos para o Convênio ICMS 201, quais tabelas são necessárias e se há um manual para geração desses arquivos.

## Causa Raiz:

O Convênio ICMS 201 possui especificidades que necessitam de tabelas específicas para a geração de arquivos. O usuário precisa de instruções claras para acessar e utilizar as informações necessárias.

## Resolução:

  1. Acesse a Central de Ajuda da plataforma e vá até a seção de Manuais.
  2. Consulte os manuais específicos para o Convênio ICMS 201:
     * **Manual Funcional Conv. ICMS 201 – Fatura** : Informações sobre faturas de serviços de telecomunicações.
     * **Manual Funcional Conv. ICMS 201 – Pré-Pago** : Informações sobre carregamento de créditos em terminais telefônicos pré-pagos.
  3. Para geração dos arquivos, é necessário que os dados estejam inseridos nas seguintes tabelas:
     * SAFX259 - Fatura de Serviços de Comunicação e de Telecomunicações
     * SAFX260 - Itens Fatura de Serviços de Comunicação e de Telecomunicações
     * SAFX261 - Carregamento de Créditos em Terminais Telefônicos Pré-pagos
  4. Gere os arquivos seguindo as instruções dos manuais para cada tipo de arquivo requerido pelo convênio.


## Próximos Passos:

Após a geração dos arquivos, verifique se todos os dados estão corretos e completos. Caso encontre qualquer discrepância, revise as informações nas tabelas mencionadas e gere os arquivos novamente. Documente quaisquer problemas ou alterações para futuras referências.

## Dicas e Recomendações:

Assegure-se de manter registros precisos e atualizados nas tabelas relevantes para evitar problemas na geração de arquivos. Consulte regularmente a Central de Ajuda para novas atualizações ou mudanças nas especificações do convênio.
