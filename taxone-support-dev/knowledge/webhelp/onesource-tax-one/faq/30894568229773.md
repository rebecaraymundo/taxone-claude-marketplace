# [SPED][EFD - ECD]TAX ONE - EFD ECD - Como resolver problemas na Geração do Registro I052 no módulo SPED ECD?

Se o registro I052 não for gerado no módulo SPED ECD, isso pode ocorrer se as demonstrações contábeis necessárias não foram geradas anteriormente ou se houve uma reimportação de dados que afetou a geração correta do arquivo. Para resolver, siga as orientações:

  1. Verifique se as demonstrações contábeis foram geradas corretamente no módulo ECD antes de tentar gerar o registro I052.
  2. Confirme se os saldos contábeis das contas aglutinadas foram importados corretamente.
  3. Caso tenha realizado alguma importação recente, certifique-se de gerar novamente os demonstrativos do mês e, em seguida, o arquivo SPED com o registro I052.
  4. Após seguir as etapas acima, se o problema persistir, forneça um exemplo de código de aglutinação que deveria ter sido gerado no I052 para análise adicional.


**Nota:** Para evitar problemas futuros com a geração de registros no SPED ECD, sempre confirme que todas as etapas preparatórias, como a geração de demonstrações contábeis e a importação de saldos, foram concluídas com sucesso antes de proceder à geração final do arquivo. Mantenha um registro das importações realizadas para facilitar a identificação e correção de eventuais problemas.
