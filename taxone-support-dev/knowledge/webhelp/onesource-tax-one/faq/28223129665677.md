Para gerar o EFD Contribuições de forma consolidada para NF-e (Modelo 55) e de forma individual para NFC-e (Modelo 65) em um arquivo que não excede 1GB, siga as orientações:

**Nota:** O sistema está configurado para consolidar todas as operações no registro C180, independentemente do modelo da nota fiscal e do tamanho do arquivo.

  1. Acesse o módulo Sped Contribuições.
  2. Navegue até o menu Parâmetros e selecione Dados Iniciais.
  3. Na aba Específico, localize o campo Indicador de Apuração na Escrituração.
  4. Altere a configuração para '1' para gerar registros consolidados
  5. Verifique se no Perfil está marcado C175. Se não estiver, marque essa opção.
  6. Gere novamente os registros e verifique se os modelos 55 são consolidados e os modelo 65 são individuais.
  7. Após realizar as alterações e gerar os registros, é importante verificar se o arquivo gerado atende às especificações necessárias. Caso contrário, ajuste as configurações conforme necessário e gere os registros novamente.


**Importante:** Regularmente revise as configurações de geração dos arquivos EFD para garantir que estão de acordo com as atualizações da legislação. Mantenha o sistema atualizado para evitar discrepâncias ou erros na geração dos arquivos.
