## Problema

O usuário necessita de orientação sobre os próximos passos para configuração e parametrização adequada do ambiente.

## Causa Raiz

É necessário realizar configurações específicas no Interdados V17 para que a comunicação com o OBI ocorra corretamente. Essas configurações não são automáticas e exigem intervenção manual do usuário ou administrador do sistema.

## Resolução

  1. Acesse o sistema Interdados.
  2. Navegue até **Opções** > **Parâmetros de configuração**.
  3. Na seção **Execução - Ferramenta** :

     * Altere o campo **Utilizar CIAM** para **SIM**.
     * Altere o campo **Tipo envio SAFX** para **OBI**.

![Configuração Execução - Ferramenta](https://atendimentotr.zendesk.com/attachments/token/6XJv4O4fIm5zidGBGLzN4dAuP/?name=image.png)

  4. Na seção **Web Service Tax One** :

     * Inclua a URL da API de integração: `https://<seuambiente>.obi.thomsonreuters.com/gateway/api/v1/integracao`
     * Verifique e mantenha corretamente os dados cadastrais relacionados ao CIAM (Caso não possua as cadastrais do CIAM, entre em contato com o suporte)

![Configuração Web Service Tax One](https://atendimentotr.zendesk.com/attachments/token/UYQ0EwGBAoz1JYHWl7a5tInaw/?name=image.png)

  5. Clique em **Salvar** para registrar as alterações realizadas.


## Próximos Passos

  * Após salvar as configurações, realize as validações na ferramenta para garantir que a comunicação entre Interdados V17 e OBI está funcionando corretamente.
  * Teste a integração enviando dados de exemplo e monitore eventuais mensagens de erro ou sucesso.
  * Caso ocorra alguma falha, verifique se todos os campos foram preenchidos corretamente conforme orientado acima.


## Dicas e Recomendações

  * Antes de realizar as configurações, certifique-se de que possui as credenciais e permissões necessárias para acessar e alterar os parâmetros do Interdados V17.
  * Mantenha um registro das alterações realizadas para facilitar o suporte futuro.
  * Se possível, realize as validações primeiro no ambiente de UAT (Homologação) antes de aplicar no ambiente de produção.
  * Para dúvidas adicionais, consulte a documentação oficial do Interdados V17 e do OBI ou entre em contato com o suporte.
  * Mantenha sempre sua equipe informada sobre as mudanças realizadas no ambiente de integração.
