Passo a passo para corrigir o **Erro 92843** \- O campo Indicação de Inclusão no Processo Conciliação de NFS de Corretores no Gerenciador de ISS deve ser preenchido com: S, N ou @.

  1. Informe se a **Pessoa Fis/Jur** (campo I**ND_CONCILIACAO)** tem perfil de seguradora:

     1. S – Sim

     2. N – Não

  2. Este campo foi criado para ser utilizado no módulo Gerenciador de ISS, diferenciando empresas com perfil de seguradoras e de outros segmentos.


Nota: O erro ocorre porque o campo I**ND_CONCILIACAO** não possui um dos valores pré-definidos no **[manual de layout](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Tecnicos/Manual_Layout.htm>). **Para corrigir o problema, basta alterar o conteúdo do campo para um dos valores citados acima.
