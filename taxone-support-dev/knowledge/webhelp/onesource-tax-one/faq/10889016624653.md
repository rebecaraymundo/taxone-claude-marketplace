Para corrigir o erro **90549 **\- O Código do Município não está cadastrado ou com informação alfanumérica no código, siga os passos abaixo:

Nota: Isto ocorre pois o código informado no campo **COD_MUNIC_ORIG** não está cadastrado em Básicos > Data Warehouse > Manutenção > Cadastros > Município ou o código é diferente do estado cadastrado na tabela **SAFX01**.

  1. Informe um código valido no campo COD_MUNIC_ORIG.

  2. Cadastre o COD_MUNIC_ORIG em Básicos > Data Warehouse > Manutenção > Cadastros > Município.

  3. Verifique se o COD_MUNIC_ORIG está correto no arquivo TXT.

  4. Verifique se o COD_MUNIC_ORIG pertece ao COD_ESTADO_ORIG informado no arquivo TXT.
