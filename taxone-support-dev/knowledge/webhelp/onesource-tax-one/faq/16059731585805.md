Normalmente, esse problema ocorre pois a data de movimento não está preenchida ou está preenchida com zeros.

Para corrigir o erro 90009: 'O campo Data de Movimento não esta preenchido ou esta preenchido com zeros.', siga a orientação abaixo:

  * Preencha o campo com a data correta AAAAMMDD.
