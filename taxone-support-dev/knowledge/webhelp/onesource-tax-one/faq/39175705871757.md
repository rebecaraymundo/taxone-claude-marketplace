# Erro ORA-01654 ao executar importações de SAFX.

**Problema:** Durante a importação de arquivos, como a SAFX07, ou ao tentar salvar informações manualmente, o sistema exibe o erro completo: **"ORA-01654: unable to extend index ... by 8 in tablespace"**.

**Causa Raiz:** Esse erro indica que o espaço disponível em um dos `tablespaces` do banco de dados está insuficiente. Isso impede a extensão de um índice necessário para o processo, especialmente em operações que envolvem um grande volume de dados.

**Resolução:** Como a gestão do banco de dados é de responsabilidade da Thomson Reuters, a solução é interna. O cliente deve **abrir um ticket de suporte** reportando o erro para que o time de DBA possa realizar o aumento de espaço no `tablespace` afetado.

**Próximos Passos:**

  * Após a abertura do ticket, o time técnico da Thomson Reuters irá atuar para solucionar a questão.
  * Não é necessário que o cliente realize qualquer procedimento no banco de dados.


**Dicas e Recomendações:**

  * Reportar o erro imediatamente permite uma resolução mais rápida.
  * Forneça todos os detalhes do erro e, se possível, anexe os logs da importação para agilizar a análise.
