# Passo a passo para resolver o Erro no Código de Receita no Data Warehouse:

Ao utilizar o Data Warehouse para acessar a tela de **Manutenção > Retenções > Controle de Tributo**, você pode se deparar com um erro relacionado ao código de receita. Esse problema ocorre quando o código de receita não está cadastrado inicialmente. Nesse caso, é necessário validar se o código se encontra na tabela Taces14. A seguir, apresentamos os passos para solucionar essa questão.

## **Verificação na Tabela Taces14**

  1. **Acesse a Tabela Taces14** :

     * Acesse o sistema e navegue até a tela que representa a visualização da Taces14: **Ferramentas > Tabelas Internas > Manter > Tabelas > Código de Receita>Relatório **onde é possível verificar os registros de receitas.![](https://suporte.thomsonreuters.com.br/hc/article_attachments/27701261036301)
     * Procure pelo código de receita em questão e verifique se ele está cadastrado na tabela.
     * Anote a data e o código de tributos associados a esse código de receita.
  2. **Impacto na Visualização do Código** :

     * Os valores informados na tela de seleção de código de receita impactam diretamente na visualização do código.
     * Certifique-se de que os valores estejam corretos para evitar problemas na exibição.


## **Cadastro do Código de Receita**

Caso o código de receita não seja encontrado na Tabela Taces14 e seja necessário para o uso, siga os passos abaixo para cadastrá-lo:

  1. **Acesse Básicos > Ferramentas > Tabelas Internas > Manter > Tabelas > Código de Receita
**

     * Abra o Tax One e navegue até a opção para cadastrar o código, clicando em novo.![](https://suporte.thomsonreuters.com.br/hc/article_attachments/27701245689997)
