Caso não consiga registrar o CST 98 em notas fiscais de entrada de ativo imobilizado no EFD Contribuições, mesmo seguindo os passos recomendados. O problema pode ser a confusão sobre a utilização do CST 98 para notas fiscais de entrada de ativo imobilizado. O CST 98 é normalmente utilizado para devoluções de vendas, enquanto os CSTs apropriados para ativo imobilizado são 70 e 98 quando escriturado.

Para resolver, siga as orientações:

  1. Verifique se a nota fiscal de entrada de ativo imobilizado utiliza o CFOP 1551.
  2. Confirme se o CST correto para a operação é 98, conforme orientação do manual do layout na EFD Contribuições para registros abertos e fechados.
  3. Na geração dos registros EFD, ajuste as notas para utilizar o CST 70, conforme apropriado.
  4. Se a nota contém itens sem direito à crédito, ajuste as configurações para não gerar NFS1s ENTRADAS SEM CRÉDITO durante a geração do arquivo.
  5. Após realizar as alterações, verifique se o arquivo EFD Contribuições gerado contém os registros corretos. Em caso de dúvidas, consulte o manual do usuário ou entre em contato com o suporte técnico.


**Nota:** Para evitar problemas futuros com a escrituração de notas fiscais de ativo imobilizado, sempre confira os CSTs aplicáveis e as regras de escrituração vigentes. Mantenha-se atualizado com as mudanças no guia prático da EFD Contribuições para garantir conformidade com as regulamentações fiscais.
