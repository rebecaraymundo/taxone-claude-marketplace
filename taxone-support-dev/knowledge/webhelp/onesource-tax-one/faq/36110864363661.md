# [Básicos][Report Fiscal] TAX ONE -  Resolução de Problemas com Geração de Relatório de Apoio à Apuração de ISS com Diferenças nos Itens das Notas Fiscais

## Problema:

O Relatório de Apoio à Apuração de ISS não estava gerando corretamente, apresentando algumas notas de forma analítica em vez de uma linha somada por nota fiscal conforme necessário.

## Causa Raiz:

O problema ocorreu devido a diferenças no campo "Alíquota INSS" entre os itens das notas fiscais, fazendo com que o relatório separasse os itens ao invés de agrupá-los por nota fiscal.

## Resolução:

  * Acesse o relatório problemático no sistema.


  * Navegue para a seção de parâmetros do relatório.


  * Localize o campo "Alíquota INSS" e remova-o dos parâmetros de geração do relatório.


  * Gere novamente o relatório para verificar se os itens de serviço estão agora corretamente agrupados por nota fiscal.
