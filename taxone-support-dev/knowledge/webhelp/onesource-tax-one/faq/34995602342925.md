Resolução de Problemas na Pré-Geração do Registro E115 no TAX ONE - SPED FISCAL

**Problema:**

O Registro E115 no TAX ONE - SPED EFD não está sendo gerado apesar de ter realizado todos os parâmetros de dados iniciais, inclusão de CFOPs e tentativa de geração do livro.

**Causa Raiz:**

A falha na geração do Registro E115 pode estar ligada à necessidade de parametrização por CST específico para o código SC850093, o qual é opcional e requer configuração se utilizado.

**Resolução:**

Para resolver o problema, siga os seguintes passos:

Acesse a configuração de parametrização por CST no TAX ONE - SPED EFD.

Verifique se existe parametrização ativa para o código SC850093.

Caso haja parametrização para esse código, modifique para o código SC850065 ou opte por não usar a parametrização de CST para nenhum código se esse ajustamento for aplicável.

Faça a pré-geração do Registro E115 novamente para verificar se os ajustes foram aplicados com sucesso.

**Próximos Passos:**

Após realizar os ajustes recomendados, monitore o sistema para qualquer sinal de inconsistência. Verifique se a geração do Registro E115 ocorre como esperado e, em caso de problemas persistirem, contate o suporte técnico para uma análise mais detalhada.
