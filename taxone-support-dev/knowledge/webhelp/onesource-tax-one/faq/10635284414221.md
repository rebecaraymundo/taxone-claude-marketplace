Passo a passo para corrigir o **Erro 90163** \- O Campo Código NBM não está cadastrado:

  1. Cadastre o **Código do NBM** utilizando a **SAFX2043,** ou cadastre manualmente em [Básicos > Data Warehouse > Manutenção > Códigos > Código NBM](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Basicos/Basicos_Data_Warehouse/Basicos_Data_Warehouse_Menu_Manutencao/Basicos_DW_Menu_Manutencao_Codigos.htm>).


Nota: O erro ocorre quando o campo **COD_NBM** está preenchido com um código que não está cadastrado no sistema.
