# Como corrigir o erro:

> **Erro:**
>  Erro no tratamento de dados das Notas Fiscais Emitidas (`MUN_SIMPLISS_CC_GERA_TOMAD.setDadosItem`)
> ORA-06512: at `USER_TAXONE_MRV.MUN_SIMPLISS_CC_GERA_TOMAD`, line 155
> Chave: `001-0002-01/04/2024-339778-000000676/U/`
> ORA-06502: PL/SQL: numeric or value error: character string buffer too small

### O erro ocorre devido a uma parametrização incorreta realizada.

### Abaixo explico o ponto que precisa ser ajustado para que o arquivo seja gerado com sucesso.

De acordo com o manual de layout, o campo **Es_Documento_tipo** deve ser preenchido apenas com os seguintes valores:


  * **1** – Nota Fiscal
  * **3** – Cupom Fiscal
  * **4** – Recibo


![](/attachments/token/8PG8OsCZcGEuDKeP48vBtpDpl/?name=image.png)
Caso tenha uma parametrização diferente disso será apontado erro.
