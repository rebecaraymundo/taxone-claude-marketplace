Segue informação sobre os principais dados que são verificados no documento fiscal, a fim de definir seu código de situação, no campo 06 (COS_SIT) do Registro C100, envolvendo documento fiscal extemporâneo.

![mceclip6.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/11045395115917)

1\. Documento regular extemporâneo - COD_SIT = 01.

O principal critério que irá definir se um documento fiscal é extemporâneo, será o preenchimento do campo '**Dt. Extemp** '.

É muito importante consultar o [Manual de Leiaute SAFX](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Tecnicos/Manual_Layout.htm>), para o correto preenchimento dos campos referente às datas informadas no documento fiscal, bem como as particularidades definidas por cada UF, quanto aos documentos extemporâneos.

![mceclip2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/11044906413069)

2\. Documento cancelado extemporâneo - COD_SIT = 03.

Além do preenchimento do campo 'Dt. Extemp', este documento deve indicar que é cancelado, através da marcação do indicador '**Sit. Cancelado** '.

![mceclip3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/11045044992013)

3\. Documento Fiscal Complementar extemporâneo - COD_SIT = 07.

Além do preenchimento do campo 'Dt. Extemp', este documento deve indicar a situação especial '**5 - NF Complementar ou CT-e de Complemento de Valores (modelo: 57)** '.

![mceclip4.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/11045230419981)
