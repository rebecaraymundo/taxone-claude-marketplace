Para garantir que o Código do Município seja gerado corretamente, é necessário verificar se o cadastro do Município MSaf foi realizado para o respectivo Município ISS. Caso não tenha sido feito, é preciso proceder com o cadastro.


Caminho para realizar o cadastro:

  * Básicos > Data Warehouse > Manutenção > Cadastros > Municípios ISS > Manutenção
