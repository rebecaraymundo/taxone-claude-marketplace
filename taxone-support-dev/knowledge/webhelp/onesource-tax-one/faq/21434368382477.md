Passo a passo para utilização do Parâmetro Específico p/ Cálculo das Operações de Saídas Tributadas (por UF):

**Cadastro do Estabelecimento**

1\. Verifique o cadastro do estabelecimento no módulo Básicos > Parâmetros, menu Preliminares > Empresa/Estabelecimento. Use o exemplo da UF PE para referência.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21434358623757)

**Geração do Livro de Apuração ICMS**

2\. Gere o Livro de Apuração do ICMS no módulo Estadual > ICMS, menu DATA MART > Apuração do ICMS > Geração / Emissão.


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21434375412621)

**Parâmetro Especifico p/ Cálculo das Operações de Saídas Tributadas (por UF)**

3\. Configure o Parâmetro Específico para o Cálculo das Operações de Saídas Tributadas (por UF) acessando o módulo Estadual > Ativo Permanente, menu Cadastros > Parâmetro Específico p/ Cálculo das Operações de Saídas Tributadas (por UF) > Por CFOP / Natureza de Operação.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21434368364429)

**Importante:** Essa parametrização não é obrigatória e é usada para atender a regras específicas de determinadas UFs, quando usado o Valor de Base de Cálculo no quadro Parâmetros Específico p/ Apuração das Saídas Tributadas na tela de Cálculo do Crédito - ICMS. Para casos não parametrizados, será utilizado o cálculo padrão.

**Cálculo de Crédito -ICMS**

4\. Realize o cálculo de crédito de ICMS no módulo Estadual > Ativo Permanente, menu Movimento > Cálculo de Crédito - ICMS.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21434395971213)

**Cálculo Geral dos Créditos**

5\. Verifique o Cálculo Geral dos Créditos no módulo Estadual > Ativo Permanente, menu Movimento > Manutenção Dados de Cálculo > Modelo C > Cálculo Geral dos Créditos.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21434381130765)
