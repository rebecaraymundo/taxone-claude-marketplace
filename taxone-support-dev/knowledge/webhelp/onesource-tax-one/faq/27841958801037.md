Passo a passo para resolver o erro de XML no evento R-4020 ao Enviar pelo Painel de Evento:

  1. Acesse o módulo de **Parâmetros** , no menu **Preliminares**.
  2. Selecione a opção **Empresa/Estabelecimento**.
  3. Localize o campo "**Natureza Jurídica** " e verifique o conteúdo inserido. Caso o código esteja incorreto ou desatualizado, atualize-o para um código válido com 4 dígitos numéricos, conforme a tabela do IBGE.
  4. Se a **natureza jurídica** não for mais necessária, considere removê-la do cadastro.
  5. Após ajustar o cadastro, gere previamente os eventos para criar novos R-4020 corretamente.


Próximos passos:

  1. Envie novamente o evento R-4020 e verifique se o erro foi resolvido.
  2. Se o erro persistir, entre em contato com o suporte técnico para obter mais assistência.


Dicas e recomendações:

  * Revise periodicamente os dados cadastrais da empresa para garantir que todas as informações estejam atualizadas e corretas, evitando erros futuros. Isso pode prevenir erros futuros em envios de eventos e outras operações no sistema.
