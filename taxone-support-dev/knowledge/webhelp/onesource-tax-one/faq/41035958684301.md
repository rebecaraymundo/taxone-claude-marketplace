As regras de de importação que são customizadas no Tax One devem seguir o Manual de Customização da Importação disponível no link:

[Job_Servidor_Menu_Importacao_Customizacao da Importacao](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/WebHelp/mergedProjects/Webhelp_ONESOURCE_TAXONE/Manuais_Funcionais/Basicos/Basicos_Job_Servidor/Job_Servidor_Menu_Importacao/Job_Menu_Importacao_Customizacao_Importacao.htm>)


O Fluxo de aplicação das regras segue da seguinte forma:

  1. As Cargas das tabelas são recepcionadas no Tax One e os registros são adicionados nas Tabelas Temporária;
  2. Antes de Executar a Importação o sistema realiza a Aplicação das Regras nos campos ainda na Tabela Temporária, alterando os registros seguindo as solicitações;
  3. Após o sistema Aplicar as Regras e salvar as alterações ainda na tabela temporária ocorre a validação dos campos para verificar erros e incompatibilidades;
  4. Após o Sistema Validar os campos já com as regras aplicadas é realizada a Importação dos dados para a Tabela Definitiva.


Se algum registro tiver erros na aplicação de alguma regra, o sistema só vai identificar na validação antes da importação e vai interromper (abortar) o processo e os registros não serão Importados.

Será necessário validar o log de importação para identificar e corrigis erros das regras.
