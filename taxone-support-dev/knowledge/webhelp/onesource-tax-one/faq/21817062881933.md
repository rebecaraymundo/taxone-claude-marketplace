# [Básicos][Report Fiscal] TAX ONE - A empresa não aparece no relatório de Manutenção das Retenções de ISSQN, como resolver?

Ao acessar o Tax One, não é possível encontrar o estabelecimento para gerar o relatório Retenções de ISSQN. Para resolver, é necessário centralizar os estabelecimentos. Siga os passos abaixo:

1\. Acesse Básicos > Report Fiscal > Obrigações Acessórias > ISS > Centralização de Estabelecimentos.

2\. Marque os estabelecimentos que deseja centralizar.

3\. Clique no botão "Executar" para iniciar o processo de centralização.

4\. Confirme a operação clicando no botão "Confirmar" localizado no topo da tela.


Após a centralização, os estabelecimentos deverão ser exibidos como opções disponíveis no relatório de Retenções de ISSQN.
