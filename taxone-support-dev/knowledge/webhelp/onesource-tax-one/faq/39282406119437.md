## Problema

A geração do arquivo de recomposição dos saldos contábeis não está sendo enviada para o sistema ONESOURCE, mesmo com as parametrizações previamente configuradas.

## Causa Raiz

O problema ocorre devido à configuração do campo "Considerar apenas contas contábeis e centros de custos movimentados na exportação dos cadastros" nas parametrizações do menu "Parâmetros". Quando ativado, este campo pode restringir os dados enviados, impedindo que a geração seja realizada corretamente.

## Resolução

Siga os passos abaixo para resolver o problema:

  1. Acesse o menu **"Parâmetros"**.
  2. Navegue até **"Parametrização para Recomposição dos Saldos Contábeis"**.
  3. Localize o campo chamado **"Considerar apenas contas contábeis e centros de custos movimentados na exportação dos cadastros"**.
  4. Se este campo estiver **marcado** , **desmarque-o**.
  5. Realize novamente o processo de geração do arquivo de recomposição dos saldos contábeis.


A geração deve ser enviada com sucesso após esta alteração.

## Próximos Passos

Após implementar a solução:

  1. Teste a geração novamente para confirmar que o problema foi resolvido.
  2. Verifique se os dados foram enviados corretamente para o sistema ONESOURCE.
  3. Continue monitorando o processo de integração para garantir que novas exportações sejam realizadas sem problemas.


## Dicas e Recomendações

  * Evite alterar configurações de parametrização sem compreender completamente os seus impactos no sistema.
  * Mantenha documentadas todas as alterações realizadas nas parametrizações, de modo que outros analistas ou usuários possam identificar rapidamente o histórico de mudanças.
  * Consulte a documentação oficial do sistema ONESOURCE ou entre em contato com o suporte técnico para orientações mais detalhadas sobre parametrizações.
  * Realize testes frequentes e validados para garantir que o sistema está operando conforme o esperado.
