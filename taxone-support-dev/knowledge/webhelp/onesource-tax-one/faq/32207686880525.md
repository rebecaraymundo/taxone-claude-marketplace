## Problema:

Usuário deseja alterar o código de um estabelecimento existente para um novo código no sistema TAXONE, indagando se existe algum recurso automático para tal mudança.

## Causa Raiz:

A plataforma TAXONE não possui funcionalidade automática para alteração de códigos de estabelecimentos. A mudança requer um processo manual que envolve exclusão e recriação do cadastro do estabelecimento.

## Resolução:

  * Exportar toda a movimentação financeira e administrativa do estabelecimento atual.
  * Excluir o cadastro do estabelecimento atual, incluindo todos os dados associados.
  * Criar um novo cadastro para o estabelecimento com o novo código desejado.
  * Ajustar o código do estabelecimento na movimentação exportada para corresponder ao novo cadastro.
  * Reimportar a movimentação ajustada para o sistema, sob o novo estabelecimento.


## Próximos Passos:

Após completar a reimportação dos dados, é recomendável verificar se todas as informações estão corretas e se refletem as operações do estabelecimento adequadamente. Monitorar as transações por alguns dias para garantir que tudo está funcionando como esperado.

## Dicas e Recomendações:

Devido à complexidade e à necessidade de precisão neste processo, recomenda-se solicitar o apoio de um parceiro homologado. Isso garantirá que a transição seja feita de maneira eficaz, minimizando riscos de erros ou perda de dados. Mantenha sempre um backup dos dados antes de iniciar o processo.
