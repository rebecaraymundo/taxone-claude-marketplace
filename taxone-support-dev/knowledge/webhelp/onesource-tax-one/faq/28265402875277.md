Devido as origens não terem os dados por completo, o campo COD_MUNICIPIO geralmente não era preenchido, causando erro na importação. Por isso, o MastersafDW e/ou TaxOne criou a funcionalidade para preencher este campo através do CEP, com isto a Interface deixou de tratar este campo.

Passo a passo para configurar o Código de Município (COD_MUNICIPIO) na SAFX04:

  1. Informar o **Código do Município** que será utilizado para gerar relatórios para as Zonas de Livre Comércio, Zona Franca de Manaus e para geração do campo 10 da DES-SP.
  2. O código informado deve estar de acordo com a **Tabela de Município do IBGE** (Taces06).


**Nota** : Quando o campo não é preenchido, a importação fará a geração automática através do CEP (campo 20), utilizando a tabela de CEP´s x Municípios IBGE (TACES 12). Este recurso só funcionará se o CEP for informado corretamente e estiver cadastrado na tabela de CEP´s.
