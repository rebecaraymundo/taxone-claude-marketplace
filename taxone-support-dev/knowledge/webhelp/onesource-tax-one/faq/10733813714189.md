# [SPED][EFD - REINF]TAX ONE - SPED - EFD - REINF - O que fazer quando o evento R-2010/R-2020 está gerando notas fiscais sem valores de INSS?

Acesse esta FAQ no [web help do produto](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/WebHelp/mergedProjects/FAQs/SPED_REINF.htm#sped_reinf_10>).

Para solucionar o problema de notas fiscais que utilizam serviços/produtos parametrizados no módulo REINF, porém, sem valores de INSS no qual não devem ser considerados na geração, siga os passos abaixo:

  1. Em [SPED > EFD - REINF > Parâmetros > Dados Iniciais](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/SPED/EFD_REINF/REINF_Parametros/REINF_Param_Dados_Iniciais.htm>), selecione o parâmetro Desconsiderar NFs sem Valor de Retenção Informado.
