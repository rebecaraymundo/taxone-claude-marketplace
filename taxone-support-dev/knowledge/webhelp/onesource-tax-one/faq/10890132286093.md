Passo a passo para gerar o parâmetro **Tipo Docto Msaf x Tipo Docto** para todos os Módulos Municipais que utilizam este De x Para:

Nota: O objetivo deste parâmetro é realizar o relacionamento entre os tipos de documento informados no documento fiscal e os tipos de documento solicitados para geração dos validadores.

**Documento Fiscal**

  1. Acesse [Básicos > Data Warehouse > Manutenção > Documento Fiscal > Doc. Fiscal de Serviço](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Basicos/Basicos_Data_Warehouse/Basicos_Data_Warehouse_Menu_Manutencao/Basicos_DW_Menu_Manu_Doc_Fiscal.htm>).

  2. Na **Capa da Nota Fiscal** , verifique qual **Tipo de Documento** foi informado.


**Parâmetro 'Tipo Docto Msaf x Tipo Docto'**

  1. Acesse [Municipal > Parâmetros para Município > Parâmetros> Tipo Docto Msaf x Tipo Docto. ](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Municipal/Parametros_Para_Municipio/Parametros_Municipio.htm>)

  2. Selecione a **UF **e o **Município.**

  3. Na tela **Tipo Docto Msaf x Tipo Documento** :

     1. Informe a **Validade **do relacionamento entre os tipos de documentos. A validade é do tipo “a partir de”. Essa data também é utilizada para buscar o grupo do tipo de documento.

     2. No campo **Tipo Documento MasterSAF** selecione os **tipos de documentos** utilizados nas notas fiscais cadastradas no Data Warehouse.

     3. No campo **Tipo Documento **selecione os tipos de documentos solicitados para a geração do validador. O validador é identificado através do município previamente selecionado.
