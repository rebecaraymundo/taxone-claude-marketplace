Acesse esta FAQ no[ web help do produto](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/microcontent/Federal_IPI.htm#federal_ipi_03>).

Passa a passo para fazer o encerramento do Status das Obrigações - Imposto sobre Produtos Industrializados:

  1. Acesse [Federal > Apuração IPI > Status das Obrigações](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Federal/IPI/IPI_Apuracao/IPI_Apura_Status.htm>).

  2. Preencha o campo **Situação **com a opção **Apuração Realizada.**

  3. Preencha o campo **Validade **com a opção **Válido.**
