Caso enfrente problemas com a geração incompleta de registros 0140 no arquivo de EFD Contribuições, onde apenas algumas filiais são geradas e outras não são. A causa pode ser a falta de marcação do registro C175 para estabelecimentos que possuem notas do modelo 65. Sem este registro, o sistema não gera todos os registros 0140 necessários. Para resolver, siga as orientações:

  1. Verifique se o estabelecimento utiliza notas do modelo 65.
  2. Acesse o Perfil do Estabelecimento em SPED > EFD Contribuições > Parâmetros > Perfil.
  3. Marque o registro C175 no perfil do estabelecimento e salve.![](https://suporte.thomsonreuters.com.br/hc/article_attachments/31638726876813)
  4. Faça uma nova geração dos registros e do meio magnético do arquivo de EFD Contribuições e verifique se todos os registros 0140 foram incluídos.
  5. Após aplicar a solução, é recomendado verificar se o arquivo de EFD Contribuições está completo e correto. Caso persistam problemas, entre em contato com o suporte técnico para mais assistência.


Importante: Sempre verifique as configurações de perfil dos estabelecimentos no Tax One, especialmente em relação aos registros obrigatórios para cada modelo de nota fiscal utilizado. Manter a documentação e as configurações atualizadas pode prevenir problemas na geração de arquivos obrigatórios.
