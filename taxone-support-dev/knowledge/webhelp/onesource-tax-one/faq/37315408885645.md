Problema:

Usuário relata que, mesmo após a carga de dados, o relatório de Balancete mostra como "Sem Movimento".

Causa Raiz:

A geração de relatórios requer que as tabelas específicas estejam corretamente carregadas.

Resolução:

  * Acesse o módulo Report Fiscal Gerenciais Contábil.
  * Utilize o relatório em Gerenciais >Contábil >Geral >**Balancete - Formato XLS** para visualizar os dados das tabelas SAFX01 e SAFX02.
