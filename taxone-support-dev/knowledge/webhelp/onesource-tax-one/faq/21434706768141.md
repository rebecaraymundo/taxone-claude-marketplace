Passo a passo para utilização do Parâmetro p/ Cálculo das Operações de Saídas Tributadas/Devoluções:

**Geração do Livro de Apuração ICMS**

1\. Gere o Livro de Apuração do ICMS acessando o módulo Estadual > ICMS, menu DATA MART > Apuração do ICMS > Geração / Emissão.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21434684781837)

**Parâmetro p/ Cálculo das Operações de Saídas Tributadas / Devoluções**

2\. Configure o parâmetro para o cálculo das operações de saídas tributadas/devoluções em Estadual > Ativo Permanente, menu Cadastros > Parâmetro p/ Cálculo das Operações de Saídas Tributadas / Devoluções > por CFOP / Por Extensões de CFOP.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21434691043725)

**Cálculo de Crédito -ICMS**

3\. Realize o cálculo de crédito de ICMS no módulo Estadual > Ativo Permanente, menu Movimento > Cálculo de Crédito - ICMS.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21434941740429)

**Cálculo Geral dos Créditos**

4\. Efetue o cálculo geral dos créditos no módulo Estadual > Ativo Permanente, menu Movimento > Manutenção Dados de Cálculo > Modelo C > Cálculo Geral dos Créditos.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21434684793485)
