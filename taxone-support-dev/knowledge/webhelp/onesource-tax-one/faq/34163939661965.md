# Como Importar Registros M220 para Apuração Geral PIS/COFINS no Módulo SPED EFD

**Problema:** Usuários enfrentam dificuldades para localizar a funcionalidade de UPLOAD e importar registros M220 no módulo SPED EFD – Escrituração Fiscal Digital das Contribuições.

**Causa Raiz:** A confusão geralmente ocorre devido à falta de clareza sobre a nomenclatura dos arquivos e os passos corretos para a importação, seguindo o layout específico requerido pelo sistema.

**Resolução:**

  * Verifique se o arquivo .txt está de acordo com o layout requerido, incluindo a codificação UTF-8.
  * O nome do arquivo deve seguir o padrão "EPCARQ_M220".
  * Carregue o arquivo para o sistema via SFTP.
  * Acesse o módulo SPED EFD e vá para a tela de apuração geral PIS/COFINS.
  * Selecione o arquivo carregado para importar os registros M220.


**Próximos Passos:** Após a importação, verifique se os registros M220 foram corretamente atualizados na plataforma. Realize testes de validação para garantir que os dados foram importados conforme esperado.

**Dicas e Recomendações:** Sempre salve o arquivo no formato correto e verifique a nomenclatura antes de tentar o upload. Consulte o manual do usuário disponível no WebHelp para esclarecimentos adicionais sobre o layout e procedimentos de importação.
