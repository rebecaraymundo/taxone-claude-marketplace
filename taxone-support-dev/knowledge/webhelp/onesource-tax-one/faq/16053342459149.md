Normalmente, esse problema ocorre pois o campo IND_DEB_CRE não possui algum dos valores pré-definidos:

D - Se o Lançamento foi a Débito da Conta;

C - Se o Lançamento foi a Crédito da Conta.

Para corrigir o erro 90100: 'O campo Debito/Credito deve ser preenchido com D ou C.', siga a orientação abaixo:

  * Altere o conteúdo do campo para um dos valores válidos.
