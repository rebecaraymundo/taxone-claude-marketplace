Para realizar a exclusão dos lançamentos complementares, siga as orientações:

1\. Acesse Básicos > Processos Customizados > Execução dos Processos Customizados.

Nota: Se o processo customizado não aparecer no módulo Básico, verifique no módulo "Utilitários".

2\. Clique em Incluir Processo Customizado.

3\. Selecione o processo desejado e clique em Confirmar.

4\. Após confirmado, clique em Executar.

Muita Atenção!
Vão ser excluídos todos os dados deste período selecionado de todas as tabelas informadas e também das filhas, recomenda-se fazer a exportação das mesmas via Job Servidor como Backup.
Disponibilizamos a opção ‘Somente Listar’ para conferir os dados que serão excluídos.

5\. Acesse a aba Processos e verifique os dados no relatório.

6\. Selecione a opção ‘Listar e Excluir’ para apagar os dados das tabelas.

7\. Após excluído, importe os dados que deseja novamente.
