A geração do relatório previdenciário depende das informações fornecidas nos itens das Notas Fiscais (NF). A ausência dos valores da 2ª alíquota de INSS no relatório pode ser decorrente de uma falta de parametrização adequada ou de dados incompletos nos itens da NF.

Passo a passo para corrigir os valores da 2ª alíquota de INSS em relatórios de serviços de retenção previdenciária:

  1. Verifique as informações inseridas nos itens da NF para garantir que todos os dados necessários, incluindo a 2ª alíquota de INSS, estão corretamente registrados.
  2. Acesse o sistema de geração de relatórios previdenciários.
  3. Navegue até as configurações de parametrização do relatório.
  4. Confirme se a opção para incluir a 2ª alíquota de INSS está ativada.
  5. Se não estiver, ajuste as configurações para incluir este valor no relatório.
  6. Gere novamente o relatório para verificar se os valores estão sendo exibidos corretamente.
  7. Após ajustar as configurações e gerar novamente o relatório, confira se todos os valores estão corretos. Caso o problema persista, entre em contato com o suporte técnico para uma investigação mais detalhada.


**Dicas e Recomendações:**

Recomendamos revisar regularmente as configurações de parametrização dos relatórios para garantir que todas as informações necessárias estão sendo incluídas. Mantenha os registros dos itens das NF sempre atualizados para evitar discrepâncias nos relatórios gerados.
