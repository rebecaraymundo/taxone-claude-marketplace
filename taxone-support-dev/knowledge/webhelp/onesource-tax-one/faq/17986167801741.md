Para corrigir o **erro 90029 - Código Fiscal de Operação (CFOP) não cadastrado, **na SAFX08, faça o seguinte:

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/19247365807117)

1\. Acesse [Data Warehouse > Manutenção > Código](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Basicos/Basicos_Data_Warehouse/Basicos_Data_Warehouse_Menu_Manutencao/Basicos_DW_Menu_Manutencao_Codigos.htm>).

2\. Selecione a opção **Código Fiscal**.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/17986077899533)

3\. Preencha os dados da tela para cadastrar o código desejado.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/19250624345229)
