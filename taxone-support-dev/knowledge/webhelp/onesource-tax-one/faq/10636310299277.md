Passo a passo para corrigir o **Erro 90369** \- O Campo IPI Controlado por Selo deve ser branco, S ou N:

  1. Preencha o campo **IND_CONTROLE_SELO** , conforme o [manual de layout](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Tecnicos/Manual_Layout.htm>):

     1. S - Controlado por Selo;

     2. N - Não controlado por Selo.


Nota: O erro ocorre quando o campo **IND_CONTROLE_SELO **não possui um dos valores definidos no manual de layout. Para corrigir o erro, altere o conteúdo do campo para um dos valores citados acima.
