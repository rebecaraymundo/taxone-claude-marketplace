## Problema:

O usuário está enfrentando dificuldades para identificar o mapeamento dos campos das tabelas SAFX do MasterSAF DW e Tax One.

## Causa Raiz:

A documentação necessária para o mapeamento dos campos não estava facilmente acessível ao usuário.

## Resolução:

  1. Acesse o link abaixo para baixar o manual de layout do MasterSAF DW: [Manual de Layout MasterSAF DW](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/WebHelp/mergedProjects/Webhelp_ONESOURCE_TAXONE/assets/zips/Manual_Layout_MastersafDW_T1.zip>)
  2. Descompacte o arquivo baixado para acessar o documento.
  3. Consulte a planilha disponível no manual para identificar o mapeamento dos campos necessários entre as tabelas do MasterSAF DW e os arquivos TXT.
