Para visualizar os detalhes da request, em caso de erro de aplicação, siga os passos abaixo:

Nota: Este procedimento é em sua maioria realizado pelo Basis ou ABAP.

  1. Acesse a transação **SE09**.
     1. Clique no botão "óculos” para pesquisar pelo número da request que está com erro.
  2. Clique no número da request.
     1. Clique no botão “folha” para verificar o detalhe do transporte da request.
  3. Será apresentada a tela de 'Síntese de todos os logs de transporte', clique no botão "folha" onde contém o erro.
  4. Envie o print da tela apresentada, evidenciando o detalhe do erro.
Exemplo de erro na aplicação.


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/25153766971661)

  1. Para extrair o log do SAP, acesse Sistema (Y) > Lista > Gravar > File local.
     1. Sugerimos a utilização do **Format.HTML** para melhor visualização, porém, a extração pode ser realizada em outros formatos.
