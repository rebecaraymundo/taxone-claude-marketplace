## Problema

Erro "/MSAFX/SAPLSAFE_WS, Include /MSAFX/LSAFE_WSU04: Syntax error na aplicação da Request Principal do pacote da versão **4104** da Interface OBTI (print do erro abaixo):

![Pasted image \(1\).png](/attachments/token/Ad9oGn9qWxYpHOVqiR3PCc0rK/?name=Pasted+image+%281%29.png&lotus_request=true)

O erro se repetiu na tentativa de aplicação da Request Principal do pacote da versão **4105.**

## Causa Raiz

De fato existia um problema com a Request Principal da versão**4104** , que impactava diretamente na aplicação subsequente da Request Principal da versão **4105**.

##

## Resolução

  1. Para ambientes onde a Request Principal da versão **4104** já tenha sido aplicada, causando o erro:
     1. Aplicar a Request para Webservice da versão **4105** (TRQK901123) este passo corrigirá o erro inicial.
     2. Aplicar os seguintes pacotes da versão **4105** :
        * Request Principal (TRQK901118).
        * Request Customizing (TDDK906235).
        * Request para Webservice (TRQK901123).
  2. Para ambientes onde a Request Principal da versão **4104** não foi aplicada:
     * Aplicar diretamente o pacote da versão **4105**.
