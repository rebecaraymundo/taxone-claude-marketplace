Passo a passo para solucionar o erro Enhancement-Point - "Must still be adjusted" na tentativa de realizar manutenção no programa Z (customizado):

Nota: Este erro ocorre devido a atualizações realizadas no ambiente SAP (Atualização de Versão, Aplicação de SP (Suport Package), SNotes (Notas SAP) e entre outros).

  1. Abra a transação SPAU_ENH.

  2. Verifique a opção "Enhancement Implementations w/o".

  3. Verifique se o ENHP está dentro da lista de objetos expandida.

  4. Abra o ENHP desejado.

  5. Clique na opção de Editar (Lápis/óculos no canto esquerdo superior), para mudar de 'View' para 'Modify' a tela.

  6. Selecione a linha do Enhancement Point que deseja alterar no ALV no centro da tela.

  7. Clique na opção "Editor Abap".

  8. Grave a modificação, "Sim" gravar dentro de uma request.
