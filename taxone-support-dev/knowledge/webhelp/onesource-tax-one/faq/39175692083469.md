## **Erro de Código de Operação não Cadastrado na Importação SAFX10**

**Problema:** Durante a importação de um arquivo SAFX10, o sistema retorna um erro porque um código de operação não foi identificado. O log do erro pode não ser claro sobre a origem do problema.

**Causa Raiz:** O valor informado no campo **COD_OPERACAO** do arquivo SAFX10 não está cadastrado no sistema Tax One. Sem o cadastro adequado, o sistema não consegue processar o registro, gerando o erro.

**Resolução:** Para solucionar o erro, é necessário cadastrar o código de operação que está faltando no sistema. Siga os passos abaixo:

  1. Acesse o módulo **DATA WAREHOUSE**.
  2. Navegue para **Manutenção Códigos Operação**.
  3. Cadastre o código de operação conforme a informação do arquivo SAFX10.
  4. Após o cadastro, execute a importação novamente.
