# [Estadual][Ativo Permanente] Como realizar a exclusão/limpeza dos dados de aquisição e alienações do Ativo Permanente (CIAP)?

Para realizar o processo de exclusão de aquisição ou alienação deve ser realizado diretamente pela interface(tela) do sistema.


É importante lembrar que, para excluir uma aquisição, é necessário primeiro verificar se existe alguma alienação associada ao item em questão.

Caso exista, é preciso excluí-la antes de proceder com a exclusão da aquisição.


Após realizar qualquer processo de exclusão, é essencial recalcular o crédito do ICMS para garantir que todas as informações estejam atualizadas e corretas.
