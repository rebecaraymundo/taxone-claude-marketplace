# [Básicos][Data Warehouse] TAX ONE - Relatórios não são gerados, como resolver?

Caso não consiga gerar relatórios gerenciais e customizados de períodos anteriores a maio de 2024, mesmo utilizando as funções de relatório customizado em utilitários, siga as orientações:

**Nota:** Os dados das tabelas temporárias do Datamart são limpos periodicamente, de acordo com um cronograma predefinido, o que pode resultar na ausência de dados para períodos anteriores ao estipulado.

  1. Verifique se a equalização do Datamart foi realizada para os períodos desejados. Caso contrário, será necessário realizar a equalização novamente.
  2. Acesse o sistema e vá até a seção 'Utilitários'.
  3. Selecione a opção 'Relatório Customizado'.
  4. Insira o período desejado para o relatório e confirme a equalização dos dados.
  5. Gere o relatório e verifique se as informações necessárias estão presentes.
  6. Após gerar o relatório, é importante verificar se todas as informações necessárias estão presentes. Caso encontre alguma discrepância ou falta de dados, pode ser necessário reprocessar a equalização para períodos específicos. Mantenha um controle dos períodos equalizados para evitar a necessidade de reprocessamento frequente.


**Importante:** Para evitar problemas com a falta de dados em relatórios futuros, recomendamos manter um registro dos períodos já equalizados e estar atento ao cronograma de limpeza das tabelas temporárias. Isso ajudará a garantir que todos os dados necessários estejam disponíveis quando precisar gerar relatórios de períodos anteriores.
