Normalmente, esse problema ocorre porque o código informado no campo TIPO_LANCTO está diferente dos valores esperados:

N – Normal,
E – Encerrado,
X – Extemporâneo.

Para corrigir o erro 93187: 'Indicador do tipo de lancamento deve ser <N>, <E> ou <X>.', siga a orientação abaixo:

  * Informe o valor válido no arquivo TXT.
