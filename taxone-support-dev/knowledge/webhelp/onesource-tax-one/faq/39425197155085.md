# Cadastro de multiplas Empresas no OBI para utilização do TDM (Tax Data Management)

## Problema:

Como gerenciar o cadastro de empresas no OBI para uso com o TDM, considerando a necessidade de cadastro de multiplos estabelecimentos, como por exemplo, mais de 1000 estabelecimentos.

## Causa Raiz:

O cadastro de empresa no OBI precisa ser feito manualmente no momento, o que pode ser trabalhoso com um grande número de estabelecimentos, já que o sistema requer um cadastro individualizado para cada local para funcionamento adequado com o TDM.

## Resolução:

  1. Acesse o cadastro de empresas no OBI.
  2. Manualmente, adicione cada estabelecimento como um novo registro no sistema.
  3. Paralelamente, registre uma sugestão de melhoria para automatização deste processo no [Portal Customer Influencer](<http://customerinfluencer.thomsonreuters.com/>).


### Dicas e Recomendações

Para sugestões de melhorias ou novas funcionalidades no sistema, envie suas ideias diretamente para a equipe de Produtos através do portal **Customer Influencer**. Dessa forma, sua solicitação poderá ser considerada nas próximas evoluções do produto.
