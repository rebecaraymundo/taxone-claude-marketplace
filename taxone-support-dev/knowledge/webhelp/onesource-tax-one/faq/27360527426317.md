Este artigo tem como objetivo apresentar as formas utilizadas para realizar autenticação de acesso no sistema Tax One.

O ONESOURCE Tax One conta com 3 processos de autenticação:

1 - Acesso default baseado na utilização de Usuário e Senha.

2 - Acesso com autenticação Multi-Factor Authentication (MFA).

3 - Acesso com autenticação Single Sign-On (SSO).

1) A utilização da autenticação default por usuário e senha é a principal, pela qual um Usuário ADM (Cliente) realiza o registro dos demais usuários que acessarão o sistema.

2) A ativação do MFA é feita no módulo Administration, em Setup > Multi-Factor Authentication. [Acesse a documentação da funcionalidade.](<https://helpcenter.thomsonreuters.com/platform-administration/Content/01%20-%20MFA%20Topics/Multifactor_Authentication_Overview.htm>)

3) Já o Single Sing-On (SSO) é um serviço contratado adicionalmente. Após a contratação, um profissional do time responsável acompanhará a implantação do SSO, enviando todas instruções necessárias e ficará disponível para solucionar dúvidas sobre a configuração.

Se o serviço ainda não tiver sido contratado, é necessário entrar em contato com o Account Manager responsável pelo cliente para adicioná-lo ao Tax One.

Se a contratação já tiver sido aprovada, informe isso na abertura do chamado, e o Time de Suporte dará o devido direcionamento.
