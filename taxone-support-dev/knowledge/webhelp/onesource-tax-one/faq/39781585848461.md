## Problema

Ao realizar a apuração do PIS/COFINS, o Código de Contribuição 32 (Contribuição Apurada por Substituição Tributária) não foi demonstrado na apuração, mesmo após ajustes realizados na nota fiscal.

## Causa Raiz

A situação ocorreu devido à falta de parametrização do Código de Contribuição 32 no menu de parâmetros de apuração do sistema. Essa pendência impede que o sistema inclua corretamente o código na apuração de PIS/COFINS.

## Resolução

Siga os passos abaixo para solucionar o problema: Acesse o menu **Parâmetros** no sistema, em **Apuração Identificador dos códigos de contribuição**. Inclua os dados do Código de Contribuição 32 conforme indicado no log do sistema. Caso não saiba quais dados incluir, consulte as evidências do log disponíveis na aba de geração de registros. Após realizar o ajuste, gere novamente os registros da EFD e o meio magnético. Verifique se o código de contribuição 32 foi incluído corretamente na apuração de PIS/COFINS.

## Próximos Passos

Após realizar os ajustes e incluir o Código de Contribuição 32: Confirme se o arquivo gerado está correto, visualizando a inserção do código na apuração. Realize testes adicionais, caso necessário, para garantir que todas as notas fiscais e contribuições foram parametrizadas corretamente. Certifique-se de realizar um backup do arquivo ajustado antes do envio oficial.

## Dicas e Recomendações

Para evitar problemas semelhantes no futuro, recomendamos: Realizar auditorias periódicas na parametrização dos códigos de contribuição do sistema. Manter a documentação das configurações atualizada para referência futura. Encaminhar logs detalhados em casos de erros, permitindo maior agilidade na identificação da causa raiz. Consultar a base de conhecimento da equipe ou o suporte técnico para esclarecer possíveis dúvidas antes de iniciar a apuração.

Para mais informações sobre o sistema e outros códigos de apuração, consulte o guia oficial ou entre em contato com nosso suporte.
