O objetivo deste artigo é orientar o usuário sobre como alterar a parametrização do BIP Dinâmico para Extratores no Interdados Oracle Cloud V17R01.



Passo a passo para atualização e utilização do Extrator Oracle Cloud para Interfaces:

  1. Antes de realizar o processo de troca de BIP Dinâmico para Extratores, verifique se o ambiente Oracle Cloud está apto a receber os extratores. Caso não tenha certeza, abra um chamado no suporte técnico da Thomson Reuters solicitando essa análise.
  2. Acesse a documentação da Oracle no My Oracle Support:


Link: [My Oracle Support](<https://support.oracle.com/> "Hyperlink \[https://support.oracle.com/\]")

  1. Analise a documentação da Oracle e siga o processo de parametrização do Manual Funcional/Extratores Oracle Cloud que mostra todo o processo para a parametrização:


Link do Manual: [Manual de Extratores Oracle Cloud](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Interfaces/Interface_Oracle_V17R01_Cloud/Webhelp_Interface_Oracle_V17R01_Cloud/Manuais_portugues/Manuais_funcionais/Extratores_Oracle.htm> "Hyperlink \[https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Interfaces/Interface_Oracle_V17R01_Cloud/Webhelp_Interface_Oracle_V17R01_Cloud/Manuais_portugues/Manuais_funcionais/Extratores_Oracle.htm\]")

  * A página 12 mostra o processo de parametrização do extrator do procedimento.
  * Na página 13, há informações sobre o filtro da parametrização, importante para buscar. dados no Oracle Cloud.


  1. Consulte também o Manual do Usuário, parte de Procedimentos, que explica a parametrização do extrator com exemplos de tela.


Link do Manual: [Procedimentos](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Interfaces/Interface_Oracle_V17R01_Cloud/Webhelp_Interface_Oracle_V17R01_Cloud/Manuais_portugues/Manuais_funcionais/Manual_de_usuario/V17R01_usuario_procedimentos.htm> "Hyperlink \[https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Interfaces/Interface_Oracle_V17R01_Cloud/Webhelp_Interface_Oracle_V17R01_Cloud/Manuais_portugues/Manuais_funcionais/Manual_de_usuario/V17R01_usuario_procedimentos.htm\]").

**Dicas e recomendações:**

  * Mantenha sempre a documentação atualizada e acessível.
  * Consulte os manuais funcionais e as documentações disponíveis no My Oracle Support para evitar problemas futuros e garantir a correta utilização dos extratores Oracle Cloud.
