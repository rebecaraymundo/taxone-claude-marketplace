Ao gerar o relatório, é apresentada a mensagem "Erro não tratado pelo sistema".


Esse problema ocorre por preenchimento inadequado dos campos de data.

Nesse caso, o campo **DAT_ESCR_EXTEMP** estava com o valor "0".

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/26782787567373)

Para fazer a correção, ajuste o valor do campo para **"@"**.

Recomendamos verificar os campos datas da SAFX, pois estes campos esperam os valores "00000000" ou "@" quando não há data para preenchê-los.
