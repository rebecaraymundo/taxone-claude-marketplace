# [SPED][EFD - Contribuições]TAX ONE - SPED - EFD Contribuições - Como corrigir a falta de geração do Registro 0150 para clientes sem identificação?

Se no arquivo gerado pelo TaxOne o registro 0150 não estiver sendo gerado em sua totalidade, resultando na falta de alguns clientes específicos, então é possível que a causa do problema esteja relacionada a uma configuração incorreta no cadastro de clientes, onde um campo específico não foi devidamente marcado, impedindo a geração adequada do registro.

Passo a passo para resolver a falta de geração do registro 0150 para clientes sem identificação no SPED EFD Contribuições do TaxOne:

  1. Acesse o cadastro de clientes no TaxOne através do caminho: _Básicos > Data Warehouse > Manutenção > Cadastros > Pessoas Físicas/Jurídicas._
  2. Localize o campo **"cliente SID"** e desmarque a opção.

  3. Salve as alterações feitas no cadastro.
  4. Realize uma nova geração dos registros e do meio magnético para verificar se o registro 0150 está sendo gerado corretamente.
  5. Após realizar as alterações e nova geração dos registros, é importante monitorar os próximos arquivos gerados para garantir que o problema foi completamente resolvido. Caso o problema persista, entre em contato com o suporte técnico para mais assistência.


**Dicas e Recomendações:**

É recomendável revisar regularmente as configurações de cadastro dos clientes para evitar problemas similares no futuro. Mantenha-se atualizado com as últimas versões e correções do sistema TaxOne para garantir o máximo aproveitamento das funcionalidades oferecidas.
