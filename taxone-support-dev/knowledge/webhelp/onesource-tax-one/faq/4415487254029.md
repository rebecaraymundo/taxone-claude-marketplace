Passo a passo para corrigir o **Erro 90084** -**** O Número de CGC/CPF não deve ser preenchido para este Registro:

  1. Verifique qual informação está correta nos campos **IND_CONTEM_COD **e **CPF_CGC.**

  2. Se o campo **IND_CONTEM_COD** estiver incorreto, informe um dos outros 3 códigos, conforme o [manual de layout](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Tecnicos/Manual_Layout.htm>):

     1. 1 - Se no Campo de Código (02) o conteúdo for um Código para Pessoa Física/Jurídica que tenha CGC/CPF

     2. 2 - Se no Campo de Código (02) o conteúdo é um CPF

     3. 3 - Se no Campo de Código (02) o conteúdo é um CGC

  3. Se o campo **CPF_CGC** estiver incorreto, altere a informação para @ ou deixe em branco.


Nota: O erro ocorre poque o campo **IND_CONTEM_COD **está com o valor "4 - Se a Pessoa Física/Jurídica não tenha CPF/CGC - casos especiais." e o campo **CPF_CGC** está preenchido.
