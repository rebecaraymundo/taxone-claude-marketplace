Atenção: A partir de 28 de agosto de 2023, os clientes que enviam dados para o ambiente utilizando Webservice devem alterar as URLs de envio de dados.

Antes de 28 de agosto, as URLs possuíam o seguinte formato:

API IMPORTAÇÃO DE DADOS: https://taxone-integrator.taxone.thomsonreuters.com

API ENVIO DE DADOS: https://safx.prod.taxone.thomsonreuters.com/api

**A partir do dia 28/08/2023, as URLs devem ser alteradas para o seguinte formato:**

  * **API IMPORTAÇÃO DE DADOS: https://prod01.taxone.thomsonreuters.com/taxone-integrator**
  * **API ENVIO DE DADOS: https://prod01.taxone.thomsonreuters.com/safx**


Para realizar a alteração das novas urls do Interdados V11R01 disponibilizada no dia 28/08/2023, siga as orientações abaixo:

  1. Se você não tiver acesso ao Banco de Dados, entre em contato com seu DBA e solicite a execução da query abaixo:
     1. Acesse o banco de dados e faça o update na tabela **select * from MSAFI_MSFTS_CONFIG**
  2. No campo **API_URI_DW_WS** , altere a URL de: https://taxone-integrator.taxone.thomsonreuters.com para:
     1. **https://prod01.taxone.thomsonreuters.com/taxone-integrator**
  3. No campo **API_URI_TAX_WS** , altere a URL de: https://safx.prod.taxone.thomsonreuters.com/api para:
     1. **https://prod01.taxone.thomsonreuters.com/safx**
  4. Após fazer a alteração, pare o serviço de cargas do Interdados.
  5. Baixe o arquivo: **interfaces-erp-taxone-V11.2023.01.jar** no [link](<https://webhelp.thomsonreuters.com.br/Site/download/interfaces/oracle/interfaces-erp-taxone-V11.2023.03.jar.zip>).
  6. Substitua o **arquivo .jar novo** que foi baixado, renomeie o arquivo existente na pasta Interdados\interfaces-erp-taxone.jar para **.old** e renomeie o interfaces-erp-taxone-V11.2023.01.jar para **interfaces-erp-taxone.jar**
  7. Suba o serviço de cargas novamente e realize um teste.
