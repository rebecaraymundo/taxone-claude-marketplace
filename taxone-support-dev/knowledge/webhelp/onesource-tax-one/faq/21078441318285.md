Este erro pode ocorrer quando um cliente que esteja utilizando o Ambiente Interdados V11R01 tenta fazer o processo de pré-instalação do Patch 046 sem ODI.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21078423969421)

Passo a passo para corrigir o erro "ERROR: ORA-01756: quoted string not properly terminated Disconnected from Oracle Database 23c Free, Release 12.0.0.0.0 – Developer-Release":

  1. Desinstale o Oracle Database 23c Free que está causando o erro, conforme indicado na mensagem de erro.

  2. Instale o Oracle Database 19c com o Oracle Client 12c, conforme recomendado no [Manual de Pré-Requisitos de Projeto do Interdados.](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Interfaces/Mastersaf_DW/WEBHELP_ORACLE_EBS_Interdados_V11R01/Interdados_V11R01_EBS/Manuais_Tecnicos/Interface_Interdados_EBS_V11R010_000_PRE_REQUISITOS_PROJETO.htm> "Hyperlink \[../Interdados_V11R01_EBS/Manuais_Tecnicos/Interface_Interdados_EBS_V11R010_000_PRE_REQUISITOS_PROJETO.htm\]")

  3. Após a instalação correta do Oracle Database 19c e Oracle Client 12c, reinicie o processo de pré-instalação do Patch 046 sem ODI do Interdados V11R01.
