Caso não consiga localizar o relatório "Analítico Serviço", isso se deve porque o relatório "Analítico Serviço" foi renomeado e teve seu formato alterado para melhorar o desempenho no sistema Tax One. Agora, ele se chama "Conferência de Documentos Fiscais (INSS/ISS/PIS/COFINS) - Formato XLS". Para resolver, siga as orientações:

  1. Acesse o sistema Tax One.
  2. Navegue até o menu Básicos > Report Fiscal > Gerenciais > Documentos Fiscais > Analíticos.
  3. Escolha a opção "Conferência de Documentos Fiscais (INSS/ISS/PIS/COFINS) - Formato XLS".
