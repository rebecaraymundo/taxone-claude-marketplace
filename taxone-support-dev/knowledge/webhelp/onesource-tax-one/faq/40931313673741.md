# [Interfaces] [SAP – Namespace/OBTI] [Geral] SAFX0000 não é gerada

- **Atualizado:** 2025-11-26

##

## Problema

SAFX0000 não é gerada após extrações, impedindo a finalização da importação dos dados para o Tax One.

## Causa Raiz

  * Parametrização da interface sem diretórios de trabalho para TR Files (pasta temporária) e sem ativação do "TR Files - Path Unix".
  * Falta de caminho "Path Unix" (pasta final) para onde os arquivos consolidados devem ser entregues.


## Resolução

  1. Configurar diretórios e timer na parametrização geral


  * Transação: /N/MSAFX/PARAMETROS.
  * Acessar: Parametrização Geral.
  * Preencher:
    * TR Files - Path Unix: definir pasta temporária de trabalho.
    * Path Unix: definir pasta final de destino.
  * Marcar o checkbox "TR Files - Path Unix".
  * Em "Timer - Importação Aut.", definir o intervalo (ex.: 600 segundos) garantindo tempo suficiente para consumo dos arquivos individuais antes da consolidação.

    * Caso o campo não esteja disponível para configuração, acesse a transação /n/MSAFX/SAFTR082 e marque o segundo checkbox da linha "TIMER_AUTOMATIC".


2\. Entender o fluxo após a configuração

  * A SAFX extrai para a pasta temporária (TR Files - Path Unix).
  * A interface move os arquivos gerados para a pasta final (Path Unix).
  * Após o tempo definido no "Timer - Importação Aut.", a SAFX0000 é criada na pasta temporária e então enviada à pasta final.


3\. Validar operação

  * Realizar uma extração de teste e aguardar o intervalo do timer.
  * Verificar se a SAFX0000 aparece primeiro na pasta temporária e, em seguida, na pasta final.
  * Confirmar que a pasta de extração original permaneça sem arquivos pendentes (vazia) antes da consolidação.


### Próximos Passos

  * Conferir permissões de leitura/escrita/execução nos diretórios configurados para o usuário do serviço SAP que executa a interface.
  * Executar ciclo completo de testes com múltiplas SAFX extraídas em sequência para validar concorrência e consolidação.


### Dicas e Recomendações

  * Garantir que a pasta de extração "fonte" permaneça vazia após o consumo; arquivos residuais bloqueiam a geração da SAFX0000.
  * Em incidentes, coletar: print de /N/MSAFX/PARAMETROS, listagem das pastas (antes/depois) e carimbo de data/hora em relação ao timer para acelerar o diagnóstico.
