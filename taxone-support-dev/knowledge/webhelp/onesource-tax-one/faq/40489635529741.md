# [Básicos][SFTP] Como acessar e transferir arquivos via área SFTP no TaxOne

## Problema:

Os usuários relataram dificuldades para acessar arquivos na área SFTP da Thomson, especificamente a ausência da pasta "Download", essencial para transferências e carga no TaxOne.

## Causa Raiz:

A pasta "Download" no diretório acessado via SFTP não é criada automaticamente ao realizar o acesso ao sistema. Ela é gerada apenas quando o sistema processa arquivos, emite relatórios ou realiza cargas de SAFX. A ausência da pasta "Download" ocorre porque não houve nenhuma interação que exigisse a criação dela até o momento.

## Resolução:

  1. Acesse o diretório raiz da área SFTP usando suas credenciais fornecidas.
  2. Para realizar uma carga de SAFX, insira o arquivo no formato .TXT ou .ZIP diretamente no diretório raiz.
  3. Após o processamento desse arquivo pelo sistema, a pasta "Download" será criada automaticamente e, nela, você encontrará o log da carga.
  4. Certifique-se de que os arquivos enviados estejam corretamente configurados para o processamento no TaxOne.
