Para gerar o evento de limpeza (base de produção restrita), que limpa os dados de eventos enviados do Onesource Tax One, Onesource REINF e Governo, siga os passos abaixo:

  1. Em [SPED > EFD - REINF > REINF > Painel > Painel de Controle de Eventos](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/SPED/EFD_REINF/Menu_REINF/MenuREINF_Painel.htm>) > Aba Limpeza da Base de Pré Produção, verifique se consta um XML aguardando geração, basta enviar ao Onesource e aguardar a limpeza.


Nota: Caso não apareça este XML aguardando geração, é necessário ter um R-1000 no Painel de Controle de Eventos. Dúvidas de como gerar o R-1000, acesse o artigo 'EFD - REINF - Como gerar o evento R-1000 - Informações do Contribuinte?'.

  * Com este XML do R-1000 aguardando geração, verifique se na Aba Limpeza da Base de Pré Produção apareceu o evento de limpeza para ser enviado ao Onesource REINF.
