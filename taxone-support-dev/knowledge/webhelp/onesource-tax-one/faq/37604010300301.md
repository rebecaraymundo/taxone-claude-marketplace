# O objetivo deste artigo é orientar o usuário à solucionar o problema de não geração do código de serviço

## Problema:

Código de Serviço não estava sendo gerado corretamente na SAFX09, mesmo após a regra 105 estar preenchida.

## Causa Raiz:

A configuração do mapeamento do campo "COD_SERVICO" foi identificada como a causa do problema, estando mapeada incorretamente, impedindo a geração do código de serviço.

## Resolução:

  * Navegue até Procedimentos SAFX09.
  * Clique no ícone do lápis para editar o procedimento.
  * Altere o campo "Mapeamento para campo COD_SERVICO" para "ITEM_NUMBER".
  * Clique em 'Salvar' para aplicar as mudanças.
  * Processe uma nova carga para verificar se o campo COD_SERVICO está sendo preenchido corretamente.


## Próximos Passos:

Após realizar a correção conforme indicado, é aconselhável monitorar as próximas cargas para garantir que o problema foi completamente resolvido. Caso o problema persista, entre em contato com o suporte para uma análise mais detalhada.

## Dicas e Recomendações:

É importante regularmente verificar as configurações dos mapeamentos para garantir que todos os campos estejam configurados corretamente, evitando erros futuros. Mantenha-se atualizado com as últimas instruções e práticas recomendadas fornecidas pelo suporte.
