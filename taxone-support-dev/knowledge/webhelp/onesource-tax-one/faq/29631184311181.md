Caso enfrente problemas no relatório de consistência da EFD REINF onde exibe dados apenas da Filial 002, mesmo que outras filiais estejam cadastradas. O problema pode ocorrer devido ao parâmetro "Multiempresas" na tela de Dados Iniciais do REINF. Dependendo de como este parâmetro está configurado, o relatório pode não exibir todos os estabelecimentos aos quais o usuário tem acesso. Para resolver, siga as orientações:

  1. Acesse a tela de Dados Iniciais do REINF.
  2. Verifique o parâmetro "Multiempresas".
  3. Se o parâmetro estiver desmarcado, marque-o para que o sistema exiba todos os estabelecimentos de todas as empresas que o usuário tem acesso.
  4. Se já estiver marcado e o problema persistir, entre em contato com o suporte técnico para uma análise mais detalhada.
  5. Após ajustar o parâmetro "Multiempresas", gere novamente o relatório de consistência da EFD REINF e verifique se os dados de todas as filiais estão sendo exibidos corretamente. Caso o problema persista, por favor, retorne este ticket dentro de 5 dias úteis.


**Importante:** Sempre verifique os parâmetros configurados no sistema antes de gerar relatórios importantes para assegurar que todos os dados necessários estejam sendo apresentados. Mantenha-se atualizado com as últimas configurações e funcionalidades do sistema através do nosso suporte ou base de conhecimento.
