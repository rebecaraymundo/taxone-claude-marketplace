Passo a passo para corrigir o **Erro 90384** -**** O Campo Código de Atividade Econômica não está cadastrado:

  1. Verifique se o **COD_ATIVIDADE** que está no arquivo é valido, caso não esteja, é necessário corrigir.

  2. Ajuste a **TACES01 **manualmente e importe para o sistema novamente.

  3. Também é possível cadastrar manualmente o código em [Básicos > Parâmetros> Preliminares > Atividade Econômica](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Basicos/Basicos_Parametros/Parametros_Menu_Preliminares/Param_Preli_Atividades_Economicas.htm>).


Nota: O erro ocorre quando o campo **COD_ATIVIDADE **possui um código que não está cadastrado no Tax One.
