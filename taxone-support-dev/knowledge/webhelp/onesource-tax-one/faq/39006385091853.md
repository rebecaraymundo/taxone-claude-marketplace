# Como executar o Sped Fiscal no Tax Automation para períodos diferentes dos pré-configurados no módulo, de forma Eventual:

### Acesse o módulo Basicos > Tax Automation

Primeiro, é necessário configurar a Rotina Fiscal. Para isso, acesse o menu **Rotinas Fiscais > Parâmetros**.

O período deve ser ajustado conforme os períodos definidos, para que posteriormente possa ser executado como uma rotina eventual.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/39029352203533)

Para adicionar a nossa rotina fiscal, é necessário primeiro criar um fluxo:

  1. Acesse o menu **Fluxo > Criar Fluxo > Novo**.
  2. Insira um nome para o fluxo.
  3. Adicione a tarefa desejada, neste caso, a EFD (Escrituração Fiscal Digital).
  4. Salve o fluxo criado.


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/39029325212941)

Após criar o fluxo, na mesma tela, localize o fluxo criado e clique em "Próximo Passo" para vinculá-lo aos estabelecimentos desejados.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/39029325214093)

##

Selecione os estabelecimentos desejados, defina a frequência e a data da próxima execução, e em seguida, salve as configurações.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/39029352209677)

##

Para que o fluxo possa ser executado de forma eventual, é necessário que ele seja executado pelo menos uma vez conforme parametrizado na Rotina Fiscal, por exemplo, para o mês anterior.

Para realizar a execução eventual, siga os passos abaixo:

  1. Acesse o menu **Dashboard > Dashboard > Dashboard**.
  2. Localize o fluxo que foi executado com os parâmetros definidos na Rotina Fiscal.


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/39029352212493)![](https://suporte.thomsonreuters.com.br/hc/article_attachments/39029325216909)
