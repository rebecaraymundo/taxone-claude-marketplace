Para corrigir o erro **90385** \- O campo Código de País não está cadastrado, siga os passos abaixo:

Verifique se o Código do País está correto no arquivo ou está cadastrado no menu Básicos > Data Warehouse > Manutenção > Cadastros > País.

Nota: Os cadastros de COD_PAIS são importados através da TACES05. A Thomson Reuters possui alguns códigos na tabela para que o sistema não inicie sem dados, porém, a manutenção desses dados são de responsabilidade do cliente.
