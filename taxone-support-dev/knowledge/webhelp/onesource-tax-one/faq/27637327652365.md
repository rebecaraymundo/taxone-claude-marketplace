Para entender como o sistema trata contas com e sem centro de custo durante o mesmo período, é importante compreender a lógica do sistema para interpretar os balancetes corretamente, sendo:

  1. Verifique se você possui saldos com e sem centro de custo para o mesmo período e conta.
  2. Se houver saldos para a mesma conta, tanto com centro de custo quanto sem, o sistema considerará apenas o saldo com centro de custo.
  3. O saldo sem centro de custo não será totalizado com o saldo com centro de custo para fins de relatório no balancete.
  4. Após verificar e entender como os saldos são considerados no balancete, mantenha a organização dos centros de custo atualizada para garantir a precisão dos relatórios financeiros. Revise periodicamente as configurações para assegurar que estão conforme o esperado.


**Atenção:** Para evitar confusões futuras, é recomendável manter uma documentação clara sobre quais contas possuem centros de custo associados e quais não possuem. Isso facilitará a interpretação dos balancetes e ajudará na tomada de decisões financeiras mais precisas. Para mais informações sobre como gerenciar centros de custo, visite nossa seção de ajuda ou entre em contato com o suporte.
