# [Básicos][Job Servidor] TAX ONE - SAFX01 - Como corrigir o erro 90054?

Acesse esta FAQ no [web help do produto](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/WebHelp/mergedProjects/FAQs/Basicos_JS_SAFX01.htm#basicos_js_safx01_15>).

Normalmente, esse problema ocorre porque o COD_INDICE_CONV informado não está cadastrado na tela Básicos > Data Warehouse > Manutenção > Códigos > Índice.

Para corrigir o erro 90054: 'Codigo do Indice de Conversao nao cadastrado.', siga a orientação abaixo:

  * Altere o código do TXT para um código já cadastrado no ambiente, ou
  * Realizar o cadastro do código informado no TXT na tela Básicos > Data Warehouse > Manutenção > Códigos > Índice.
