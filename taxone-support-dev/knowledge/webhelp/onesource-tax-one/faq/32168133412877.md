**ZSAFE108 - SAFX108 - Ordem de Produção - SAFX109 - item da Ordem de Produção**

Objetivo: Fornecer os dados para o cumprimento das obrigações fiscais relacionadas à desmontagem de produtos, conforme exigido pelos registros K210 e K215 do Bloco K.

Parametrizações Utilizadas:

  * TIPOS MOVTOS(K210/215/260/265 (BLOCO K)
  * TIPOS DE MATERIAIS


**ZSAFE108_V2 - SAFX108 - Ordem de Produção - SAFX109 - item da Ordem de Produção**

Objetivo: Fornecer os dados para o cumprimento das obrigações fiscais relacionadas a itens produzidos e insumos consumidos, conforme exigido pelos registros K230 e K235 do Bloco K.

Parametrizações Utilizadas:

  * TIPOS DE MOVIMENTO (BLOCO K)
  * TIPOS DE ORDEM (BLOCO K)
  * LISTA TÉC. INSUMO SUBSTITUTO (BLOCO K)
  * TIPOS DE OPERACAO (BLOCO K)
  * TIPOS DE MATERIAIS


**ZSAFE108_V3 - SAFX108 - Ordem de Produção - SAFX109 - item da Ordem de Produção**

Objetivo: Fornecer os dados para o cumprimento das obrigações fiscais relacionadas ao reprocessamento/reparo de produto/insumo e reprocessamento/reparo de mercadorias consumidas e/ou retornadas, conforme exigido pelos registros K260 e K265 do Bloco K.

Parametrizações Utilizadas:

  * TIPOS MOVTOS(K210/215/260/265 (BLOCO K)
  * LISTA TÉC. INSUMO SUBSTITUTO (BLOCO K)
  * TIPOS DE OPERACAO (BLOCO K)
  * TIPOS DE MATERIAIS
