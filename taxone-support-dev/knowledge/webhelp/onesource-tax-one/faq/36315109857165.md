## Problema:

O usuário criou um Fluxo no Tax Automation para uma empresa e deseja replicar o mesmo Fluxo para outras empresas, porém não encontra a opção de replicação disponível.

## Causa Raiz:

A plataforma Tax Automation suporta a criação e configuração de Fluxos de trabalho em nível de estabelecimento dentro de uma empresa, mas não oferece a funcionalidade de exportar um Fluxo já configurado para uso em diferentes empresas. Isso requer que o usuário configure novamente o Fluxo em cada nova empresa.

## Resolução:

  * Acesse o Tax Automation com suas credenciais.
  * Navegue até o estabelecimento onde o Fluxo foi originalmente criado e visualize as configurações de Fluxo.
  * Documente todas as configurações e etapas do Fluxo existente em detalhes.
  * Acesse cada uma das empresas desejadas onde o Fluxo será replicado.
  * Utilize o documento com as etapas do Fluxo original para configurar manualmente um novo Fluxo em cada empresa.
  * Confirme se cada Fluxo está configurado corretamente em cada empresa através de testes.


## Próximos Passos:

Após a configuração do Fluxo em cada nova empresa, é aconselhável monitorar de perto os resultados dos novos Fluxos durante as primeiras semanas para garantir que operem conforme esperado. Comunique-se com o suporte técnico para qualquer ajuste adicional necessário.
