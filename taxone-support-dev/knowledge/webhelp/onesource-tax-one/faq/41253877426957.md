## Problema

Ao tentar realizar a extração da SAFX04 no sistema Oracle, o usuário identificou um erro relacionado ao campo **IND_FIS_JUR,** o campo não foi gerado.

## Causa Raiz

O erro ocorreu porque a regra 18 estava presente no layout de parametrização do campo **IND_FIS_JUR.** Com o procedimento da SAFX04 via extrator **LACLS_ExtractSpedMasterData,** para o campo**IND_FIS_JUR, o campo segue a seguinte regra:**

Se PARTICIPANT_TYPE = 'SUPPLIER' e 'SUPPLIER_TYPE = 'SUPPLIER', Então 1

Se PARTICIPANT_TYPE = 'CUSTOMER', Então 2

Se PARTICIPANT_TYPE = 'SUPPLIER', então preencher o valor da tag SUPPLIER_TYPE.
Para demais registros, preencher o valor da tag PARTICIPANT_TYPE.

## Resolução

  1. Acesse o Interdados, navegue até Procedimentos > SAFX04 > Mais > Layout.
  2. No layout de parametrização do campo **IND_FIS_JUR** , **remova a regra 18** do campo 1, conforme orientado.
  3. Após realizar a alteração, reprocesse o **PST_ID** ou gere uma nova carga utilizando os mesmos dados para validar se o campo passará a ser preenchido.


## Próximos Passos

  * Verifique se o processamento da extração foi concluído sem erros após a alteração.
  * Realize testes adicionais, se necessário, para garantir que todas as informações estão corretas no arquivo gerado.
  * Em caso de dúvidas ou se o erro persistir, entre em contato com o suporte técnico para análise detalhada.
