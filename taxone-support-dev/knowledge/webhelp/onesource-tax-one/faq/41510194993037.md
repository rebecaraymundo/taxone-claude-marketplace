## Dúvida:

Qual processo seguir para identificar quais SAFX estão disponíveis para uso no Interdados

##

## Resolução:

  1. Acesse o O [Webhelp do Interdados V17](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Interfaces/Interface_Oracle_V17R01_Cloud/Webhelp_Interface_Oracle_V17R01_Cloud/index.htm>) e clique em "manuais Técnicos":

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/41510200275725)
  2. Clique em [Interdados - Processos](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Interfaces/Interface_Oracle_V17R01_Cloud/Webhelp_Interface_Oracle_V17R01_Cloud/Manuais_portugues/manuais_pdf_excel_ppt/Mastersaf_Interface_InterdadosV17R01_ERPCloudDW_V17R010_000_PROCESSOS.zip>) para baixar a planilha listando todas as SAFX disponíveis no DW/TaxOne:

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/41510194990861)

3\. Abra o arquivo baixado para visualizar todas as SAFX disponíveis. Entre as SAFX, veja a coluna "Status Interdados", nela serão mostrados três possíveis status:

  * **IMPLEMENTADO** : São SAFX que estão necessitam de customização para uso (mas as permitem) e possuem uma execução CORE, sendo necessário apenas a parametrização e execução das cargas iniciais para utilização;
  * **FAKE** : São SAFX que não possuem um processo CORE mas tem uma estrutura disponível para que seja feita a customização. Essas SAFX **precisam** ter sua View customizada pelo cliente ou pela consultoria para utilização;
  * **NÃO IMPLEMENTADO** : São SAFX que não estão disponíveis para uso no Interdados, nem via CTM, pois não há estrutura disponível para que seja feito o processo. Caso seja necessário a utilização de uma dessas SAFX, é necessário a abertura de um [Customer Influencer](<http://customerinfluencer.thomsonreuters.com/?vyemail=adriane.santos%40thomsonreuters.com&utm_campaign=br_corporate_20210608_onesource_comunicado_CustomerInfluencer&utm_medium=email&utm_source=Eloqua&chl=em&elq_mid=44788&elq_cid=5260030&elq_ename=BR_Corporate_Comunica%C3%83%C2%A7%C3%83%C2%A3o%20alertas%20e%20prazos_CustomerInfluencer_20210608&sfdccampaignid=&cid=>) e solicitar a implementação da SAFX.
