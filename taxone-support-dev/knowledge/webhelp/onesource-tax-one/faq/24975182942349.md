Passo a passo para gerar o relatório de conferência do Livro de Entrada e Saída em formato Excel:

  1. Acesse o módulo Básicos > Report Fiscal, menu Gerenciais > Documentos Fiscais > Conferência dos Movimentos p/ DataMart.
  2. Selecione o menu **Movimento de Entrada** ou o menu **Movimento de Saída**.


![tempsnip.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/24976170038029)

O objetivo deste módulo é emitir um relatório de conferência de ICMS e IPI incidentes nos movimentos fiscais de um determinado período.

**Sobre a emissão do Livro de Entrada > Ajuste SINIEF:**

  * É possível gerar o livro de acordo com a Resolução SEFCON N°. 4.055 de 29/05/2000. Para fazer a geração corretamente, é necessário habilitar o parâmetro **Cálculo do ICMS c/ Base na Resolução SEFCON nº 4055**.
  * É possível gerar o livro de acordo com o tratamento das Notas Fiscais de complemento do ICMS, conforme o RICMS - RJ, Decreto 27.427, art. 32 e 33. Para fazer a geração corretamente, é preciso habilitar o parâmetro **Tratamento das NF’s de Complemento de ICMS (P1/P2/P9).**
  * Na emissão deste relatório, o sistema realiza o tratamento das notas com complemento de CFOP cadastrado na tabela de extensões (SAFX 2081), sempre que houver. Os dois dígitos do complemento são anexados ao código do CFOP. Exemplo: 1.99.XX -> onde XX é o complemento.

**Sobre a emissão do Livro De Saída > Ajuste SINIEF:**

  * É possível gerar o livro de acordo com o tratamento das Notas Fiscais de complemento do ICMS, conforme o RICMS - RJ, Decreto 27.427, art. 32 e 33. Para fazer a geração corretamente, é preciso habilitar o parâmetro **Tratamento das NFs de Complemento de ICMS (P1/P2/P9)**.
  * É possível fazer a geração com CFOP de 3 ou 4 dígitos, de acordo com a seguinte regra:
    * Se a data de apuração é igual ou superior a 01/01/2003, se utiliza o CFOP de 4 dígitos; caso contrário, se utiliza o CFOP com 3 dígitos.
  * Na emissão deste livro, o sistema realiza o tratamento das notas fiscais com o complemento de CFOP cadastrado na Tabela de Extensões de CFOP (SAFX2081), sempre que houver. Os dois dígitos do complemento são anexados ao código do CFOP.
  * Para o Livro de Saída, é necessário fazer sua Geração e Emissão.
