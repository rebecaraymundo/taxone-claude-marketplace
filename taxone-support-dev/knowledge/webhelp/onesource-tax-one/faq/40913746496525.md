## Problema:

Ao realizar carga da namespace OBTI para o OBI a mesma é finalizada com status de erro:![](https://suporte.thomsonreuters.com.br/hc/article_attachments/41146497969165)

## Causa Raiz:

O sistema OBI possui uma limitação técnica de tamanho máximo para arquivos enviados através das interfaces SAFX, tanto na modalidade Sender, importação automática, quanto SFTP. Essa limitação é necessária para garantir o desempenho e a estabilidade da comunicação entre sistemas.

## Resolução:

Para esclarecer a capacidade máxima de envio de arquivos via OBI, siga os passos abaixo:

  1. O limite máximo para arquivos enviados pela modalidade **Sender** é de 5MB. Caso o arquivo exceda esse tamanho, ele deve ser dividido manualmente antes do envio, pois o sistema não realiza a divisão automática, para versões inferiores a 4109.
  2. Para ambientes com a versão 409 ou superior pode estar sendo utilizado o parâmetro Quebrar na Geração o .TXT por tamanho 5 MEGA (Aproximadamente)
  3. Caso ainda ocorra o erro no envio mantenha o parametro marcado e adicione no campo "ou MEGA:" o numero 4
