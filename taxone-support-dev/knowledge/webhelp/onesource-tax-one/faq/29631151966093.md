Caso não consiga visualizar a apuração para utilização ao gerar registros no módulo F600, o problema pode ser a não marcação das opções 1300/1700 que são necessárias para acessar as funcionalidades desejadas. Para resolver, siga as orientações:

  1. Acesse as configurações do seu perfil no sistema.
  2. Verifique se as opções 1300/1700 estão marcadas. Caso não estejam, marque-as.
  3. Salve as alterações e tente gerar os registros novamente.
  4. Após ajustar as configurações do perfil, monitore o funcionamento do módulo F600 para garantir que a apuração dos dados está sendo visualizada corretamente. Caso o problema persista, entre em contato com o suporte técnico.


**Nota:** Regularmente verifique as configurações do seu perfil para garantir que todas as opções necessárias para o seu trabalho estão ativadas. Isso pode prevenir problemas futuros e melhorar sua eficiência no uso do sistema.
