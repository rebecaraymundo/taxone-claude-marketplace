## Problema:

O usuário reportou que as informações estão sendo duplicadas no arquivo exportado para o SPED no sistema Mastersaf.

## Causa Raiz:

A causa do problema está na configuração de replicação dos dados para os estabelecimentos, e na marcação inadequada na opção "Preparação dos Dados de Documento Fiscais EPCDF / EPCNE".

## Resolução:

**Verifique a Configuração Inicial:** Acesse o menu de "Dados Iniciais" e certifique-se de que a opção de replicar para os estabelecimentos está corretamente ajustada.

**Preparação dos Dados:** Marque a opção "Preparação dos Dados de Documento Fiscais EPCDF / EPCNE" para garantir que os dados sejam preparados adequadamente antes da exportação.

**Revisão do Cadastro de Estabelecimentos:** Certifique-se de que cada estabelecimento tenha suas próprias informações definidas corretamente e que o CNPJ matriz esteja correto.

## Próximos Passos:

Após realizar as correções tendo sucesso na geração do arquivo SPED, recomendamos realizar testes de consistência nos dados. Caso o problema persista, entrar em contato com o suporte técnico para análise detalhada.
