# Resolução de Problemas de Atualização de Campo CFOP na SAFX07

## Problema:

O campo CFOP na tabela SAFX07 não está sendo atualizado corretamente durante o processo de ajuste em massa utilizando um arquivo .csv.

## Causa Raiz:

O problema ocorre devido a uma discrepância entre os nomes dos campos esperados no arquivo CSV e os campos na tabela SAFX07. Especificamente, o campo 'IND_FIS_JUR' estava sendo referenciado incorretamente como 'IDENT_FIS_JUR' no arquivo CSV.

## Resolução:

  * Verifique o nome dos campos no seu arquivo CSV com base na documentação atualizada.


  * Ajuste o nome do campo de 'IDENT_FIS_JUR' para 'IND_FIS_JUR' no arquivo CSV.


  * Realize uma nova importação do arquivo CSV para o sistema.


  * Se o processo ainda estiver demorando, renomeie o arquivo CSV para garantir que não haja conflitos de cache ou de nome de arquivo.


  * Execute o ajuste em massa novamente.
