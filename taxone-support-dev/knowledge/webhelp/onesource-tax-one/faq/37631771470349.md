### **Credenciais Únicas por Ambiente (OBI Gateway API)**

**Problema:** O usuário solicita a criação de credenciais adicionais para acesso aos endpoints do **OBI Gateway (API)**.

**Causa Raiz:** As políticas de acesso da plataforma exigem que apenas uma credencial de acesso (client_id e client_secret) seja gerada por ambiente (UAT e PROD).

**Resolução:** Não é possível criar um novo usuário com credenciais adicionais para acesso aos endpoints da API. A credencial única para o ambiente é enviada durante o processo de licenciamento do módulo de integração.

**Próximos Passos e Recomendações:**

  * A credencial única é a forma oficial de acesso e deve ser utilizada por todos os endpoints do ambiente.
  * Use a credencial que foi enviada ao responsável pelo licenciamento do módulo, pois ela é válida para todos os endpoints e anmbiente do **OBI**.
