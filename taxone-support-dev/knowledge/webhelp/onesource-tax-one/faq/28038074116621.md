Caso enfrente dificuldades com documentos no Tax One que não estão sendo enviados para o registro no REINF, o problema pode estar relacionado à utilização de Códigos de Receita não válidos para a natureza do rendimento utilizada, o que impede que os documentos sejam registrados corretamente no sistema REINF. Para resolver, siga as orientações:

  1. Verifique a natureza do rendimento e os códigos de receita associados na tabela MOR.
  2. Certifique-se de que os códigos de DARF utilizados estão de acordo com os códigos válidos destacados na tabela MOR para a natureza do rendimento específica.
  3. Corrija os códigos de receita conforme necessário e reenvie os documentos para o registro no REINF.
  4. Após ajustar os códigos de receita e reenviar os documentos, monitore o sistema para garantir que os documentos estão sendo registrados corretamente no REINF. Caso o problema persista, entre em contato com o suporte técnico para assistência adicional.


**Importante:** Mantenha sempre a tabela MOR atualizada e verifique-a antes de enviar documentos para o REINF. Isso evitará problemas de registro e garantirá que os processos ocorram sem erros.
