# Como Excluir Entradas em Massa na SAFX07

## Problema:

Usuário deseja realizar a exclusão em massa de um número de processo de entradas na SAFX07, mas não consegue encontrar uma opção específica para este procedimento.

## Causa Raiz:

A funcionalidade solicitada pelo usuário não está disponível diretamente. A plataforma permite a exclusão em massa, mas apenas de períodos específicos e não de um número de processo individual.

## Resolução:

  * Acesse o menu **Ferramentas** no sistema.


  * Escolha a opção **Inicialização**.


  * Selecione **Exclusão de Tabelas Definitivas**.


  * Escolha **Documentos Fiscais** para proceder com a exclusão em massa.


  * Confirme a exclusão para remover os dados desejados.


## Próximos Passos:

Após realizar a exclusão, é recomendável verificar se todos os dados desejados foram devidamente removidos. Caso encontre alguma inconsistência, entre em contato com o suporte para assistência adicional.

## Dicas e Recomendações:

Para evitar a necessidade de exclusões em massa no futuro, recomenda-se que revise regularmente os dados inseridos no sistema para garantir sua precisão e relevância. Utilize as ferramentas de filtragem e busca para gerenciar os processos de maneira mais eficiente.
