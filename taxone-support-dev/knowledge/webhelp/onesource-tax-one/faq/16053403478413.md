Normalmente, esse problema ocorre pois o campo ARQUIVAMENTO não esta preenchido ou está com "@".

Para corrigir o erro 90099: 'O campo Arquivamento é obrigatório - deve ser preenchido.', siga a orientação abaixo:

  * Preencha o campo com as informações corretas.

Veja que no arquivo, o campo esta em branco.
