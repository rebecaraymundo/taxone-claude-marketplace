# Usuário de Integração CIAM no Interdados V11

## Problema:

É possível aplicar o uso do CIAM no Interdados V11?

## Resposta:

No momento, ainda não é possível implementar o CIAM para usuários de integração com o TaxOne, pois essa funcionalidade não está disponível na interface Interdados V11.

## Dicas e Recomendações:

Altere a senha do usuário de integração do TaxOne antes que ela expire, a fim de evitar impactos na integração causados por falhas de autenticação.
