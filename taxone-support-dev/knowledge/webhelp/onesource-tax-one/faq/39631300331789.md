## Problema:

Os blocos referentes às notas fiscais não estão sendo gerados no EFD Contribuições, apresentando apenas a abertura e apurações.

## Causa Raiz:

O problema ocorre devido à configuração para geração das notas fiscais por capa. Identificamos um erro nesse modo de configuração que já está sendo endereçado para correção na próxima atualização.

## Resolução:

Enquanto aguardamos a correção na próxima atualização de software, recomendamos configurar temporariamente a geração de EFD Contribuições por item, ao invés de por capa. Siga estes passos para mudar a configuração:

  1. Acesse o painel de controle do software de geração do EFD Contribuições.
  2. Navegue até as configurações de geração de documentos.
  3. Selecione a opção "Por item" na configuração de geração de notas fiscais.
  4. Salve as alterações e gere novamente o EFD Contribuições.


## Próximos Passos:

Após realizar a alteração sugerida, verifique se os blocos das notas fiscais são gerados corretamente no EFD Contribuições. Caso o problema persista, ou se houver qualquer outra dúvida, entre em contato com nosso suporte técnico para obter mais assistência.

## Dicas e Recomendações:

É importante realizar verificação periódica das configurações de geração de documentos fiscais e tributários, para garantir que estejam alinhadas com as normativas vigentes e necessidades empresariais. Mantenha-se atento às atualizações do software para garantir que você esteja utilizando as funcionalidades mais recentes e corretas.
