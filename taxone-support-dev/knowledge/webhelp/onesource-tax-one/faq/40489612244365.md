# [Básicos][SFTP] Limite de Tamanho de Arquivo para Carga SAFX via SFTP (Máximo 1GB)

**Problema:** Arquivos de integração SAFX não estão sendo processados, pois excedem o limite máximo de tamanho permitido para envio via SFTP.

**Causa Raiz:** O limite máximo para o tamanho de qualquer arquivo individual enviado via SFTP para o processamento de cargas SAFX é de **1GB (Gigabyte)**. Arquivos que excedem esse limite não podem ser transferidos ou processados pelo sistema, impedindo a continuidade do processamento de outros arquivos na fila.

**Resolução:** Para garantir o processamento correto, siga estas etapas:

  1. **Verifique o Tamanho:** Verifique o tamanho do arquivo SAFX que está gerando a falha.
  2. **Divida o Arquivo:** Se o arquivo exceder **1GB** , ele deve ser dividido em arquivos menores. Cada arquivo resultante deve ter, no máximo, 1GB.
  3. **Remova o Arquivo Bloqueador:** Exclua ou mova o arquivo original excedente do diretório de importação SFTP para que o processamento dos demais arquivos não seja bloqueado.
  4. **Reenvie os Lotes:** Envie os novos arquivos divididos (com até 1GB cada) para o diretório SFTP.
