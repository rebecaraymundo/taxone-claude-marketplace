## Problema

Os Códigos de Retenção 597907, 596007 e 598707 não estão disponíveis no Data Warehouse (DW) para a correta gestão de tributos recolhidos separadamente, forçando o uso de um código de receita não corrente.

## Causa Raiz

A ausência dos códigos necessários no Controle de Tributos do DW deve-se à falta de atualização na TACES, que é uma tabela de apoio para códigos de receita. Essas tabelas são atualizadas e gerenciadas pelo cliente, mas podem requerer orientação para adição de novos códigos corretamente.

## Resolução

Siga os passos abaixo para adicionar novos códigos de receita no seu DW:

  * Acesse o menu de configurações do sistema.
  * Navegue até 'Básicos' 'Ferramentas' 'Tabelas Internas'.
  * Selecione 'Manter' 'Tabelas' 'Código de Receita'.
  * Na tela de manutenção, insira os novos códigos de receita conforme necessário.
  * Salve as alterações para atualizar a tabela de códigos no sistema.
