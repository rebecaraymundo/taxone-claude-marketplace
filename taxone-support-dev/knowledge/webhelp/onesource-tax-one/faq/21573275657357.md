Caso ocorra o erro "_SQL state [99999]; error code [17021]; Missing defines; nested exception is java.sql.SQLException: Missing defines – Carga 1000 SETUP INICIAL - ORG INFO_ " durante o processo da Carga 1000 SETUP INICIAL - ORG INFO no Interdados V17R01, siga o procedimento abaixo para solucioná-lo:

  1. A carga 1000 é a primeira carga realizada pelo cliente ou consultoria, que é um passo crucial para a instalação, configuração e parametrização do sistema.


Se a seguinte mensagem de erro aparecer:

"_Org.springframework.jdbc.UncategorizedSQLException: CallableStatementCallback; uncategorized SQLException for SQL []; SQL state [99999]; error code[17021]; Missing defines; nested exception is java.sql.SQLException: Missing defines._ "

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21573259728781)

  1. Para este erro específico, foi identificado que nas parametrizações no Interdados, o campo “Valor” deve ser inserido em letras Maiúsculas.
  2. Abra o Interdados, selecione **Opções\Propriedades**.

  3. Nas parametrizações abaixo, faça o preenchimento em letras maiúsculas e salve.


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21573303057037)

  1. Execute novamente a Carga 1000 SETUP INICIAL - ORG INFO. Após seguir esses passos, o erro mencionado não deve mais ocorrer.
