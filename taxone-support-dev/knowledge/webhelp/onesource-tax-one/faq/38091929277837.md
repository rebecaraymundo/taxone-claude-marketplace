# [Municipal][Nota JOSEENSE] TAX ONE Erro ao gerar o meio magnético do município de Campinas "Atividade não cadastrada"

Erro no log ao gerar o meio magnético para o município de Campinas: "Atividade não cadastrada".

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/38656667995277)

Esse erro ocorre devido à ausência de parametrização dos códigos de atividades no sistema.

Obs: A prefeitura disponibiliza uma tabela com os códigos de atividades vinculados aos códigos dos serviços. Para consultar essa tabela, basta acessar o link indicado no Manual de Layout do validador.

![](/attachments/token/VnfNVsy82Bu6u7Gwsxgd2kFsw/?name=image.png)

Para realizar a parametrização, siga o caminho: Municipal > Parâmetros Para Município > Parâmetros > Serviço Msaf x Atividade.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/38656694341901)

Selecione o código de atividade que corresponde ao código do serviço mencionado no documento fiscal.

Após confirmar o parâmetro, proceda com a geração do meio magnético.

Observamos que, após a parametrização, o campo foi preenchido automaticamente.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/38656667999629)
