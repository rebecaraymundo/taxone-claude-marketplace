# Como habilitar a geração dos registros no módulo SPED Contribuições

###

## Problema:

O usuário não conseguiu gerar registros dentro do módulo SPED Contribuições, mesmo com notas e o F100 já configurados. Não houve geração de dados para a apuração.

## Causa Raiz:

A opção _Expressa_ não estava marcada durante a configuração para geração dos registros no módulo. Essa opção inclui cálculos e otimiza a performance, sendo essencial para o processo.

## Resolução:

Siga os passos abaixo para habilitar a geração dos registros:

  1. Acesse o módulo SPED Contribuições.
  2. Na tela de Geração dos Registros, marque o **fleg Expressa**.
  3. Finalize a configuração e gere os registros novamente.
  4. Verifique se os dados do F100 estão sendo gerados corretamente.


Após marcar a opção _Expressa_ , o processo de geração dos registros estará habilitado e correto.

## Próximos Passos:

Certifique-se de que todos os registros necessários estão sendo gerados corretamente no arquivo gerado. Realize testes adicionais para validar e confirmar os dados da apuração antes da entrega.

## Dicas e Recomendações:

  * Mantenha o _fleg Expressa_ sempre marcado, pois ele impacta diretamente nos cálculos e melhora a performance do sistema durante a geração dos registros.
  * Revise todas as configurações do módulo antes de gerar novos arquivos SPED para evitar possíveis erros no processamento.
  * Consulte nossa base de conhecimento para mais artigos e informações úteis relacionadas ao SPED Contribuições.
