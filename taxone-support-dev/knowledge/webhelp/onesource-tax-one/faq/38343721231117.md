### **Procedimento para Contratação e Habilitação da API Tax Data Management (TDM)**

**Problema:** O usuário precisa realizar **consultas e recuperar dados** via **API Tax Data Management (TDM)** , mas a funcionalidade não está habilitada em seu sistema.

**Causa Raiz:** O **Tax Data Management (TDM)** é uma API e um serviço **contratado adicionalmente** , não fazendo parte do pacote padrão do sistema. Para que o acesso à API seja liberado, é necessário que a contratação do serviço seja confirmada e que a habilitação seja feita no ambiente do cliente.

**Resolução:** Para liberar o acesso à API TDM, que serve exclusivamente para consulta e recuperação de dados, siga os passos abaixo:

  1. **Confirmação e Contratação:** Entre em contato com o seu representante do nosso time de **Customer Success (CS)** para verificar se o serviço TDM já foi contratado para seu ambiente.
     * Caso o serviço não tenha sido contratado, o CS é o ponto de contato para dar seguimento ao processo de contratação.
  2. **Habilitação no Sistema:** Se o serviço TDM já estiver contratado, entre em contato com a nossa equipe de suporte para que possamos validar a contratação e liberar a habilitação técnica da API em seu sistema.


**Próximos Passos e Recomendações:**

  * Após a confirmação da contratação e a habilitação em seu ambiente, realize testes de **consulta** com a API para garantir que a conexão está funcionando corretamente.
  * Para mais detalhes sobre os endpoints e as formas de consulta, acesse a documentação oficial na Central de Ajuda:
    * [**Tax Data Management (TDM) - Central de Ajuda**](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Interfaces/OBI_Onesource_Brasil_Integration/web_help_OBI/TDM/Tax_Data_Management.htm>)
