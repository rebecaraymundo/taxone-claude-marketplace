Normalmente, esse erro ocorre pois o campo COD_OPERACAO está preenchido e para o estabelecimento informado na SAFX não existe relacionamento de tabelas x grupos para a tabela de "Operação".

Para corrigir o erro 90468: 'Nao foi possivel recuperar para o estabelecimento o grupo de centralizacao do Codigo de Operacao.', siga a orientação abaixo:

  * Associe a tabela "Operação" para o estabelecimento que está importando a SAFX no menu Básicos > Parâmetros > Funcionais > Relac. entre Tabelas e Grupos.
