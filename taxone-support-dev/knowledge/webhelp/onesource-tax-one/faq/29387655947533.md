Para realizar a reemissão do Livro Fiscal para vários períodos, siga as orientações:

**Nota:** Para a reemissão dos livros, é necessário previamente realizar a emissão dos meses que deseja fazer a reemissão.

  1. Acesse o menu DATA MART no sistema.
  2. Navegue até a opção 'Emissão Registro IPI' e selecione 'Emissão P8'.
  3. Realize a emissão para cada mês individualmente (de 04/2024 até 06/2024).
  4. Após a emissão mensal, retorne ao menu de Reemissão do Livro Fiscal e tente gerar o relatório para o período desejado novamente.
  5. Após seguir os passos acima, monitorar os relatórios gerados para confirmar que todos os meses desejados foram incluídos corretamente. Em caso de persistência do problema, entrar em contato com o suporte técnico para uma investigação mais detalhada.


**Nota:** Recomenda-se verificar se todos os períodos desejados contêm movimentos, pois períodos sem movimento podem não ser incluídos na reemissão automática. Para futuras reemissões, sempre confirme as seleções de período antes de iniciar o processo. Consulte regularmente o manual do usuário disponível na central de ajuda para dicas adicionais sobre a utilização do sistema.
