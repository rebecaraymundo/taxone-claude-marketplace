Para configurar a ativação e desativação de Exits de Regra no ambiente, execute os passos abaixo:

  1. Entre na transação **/N/MSAFX/MASTERSAF**
  2. Menu SAP > Mastersaf - Interface Nova Versão – Configuração > Definir Ativo / Inativo - Exit de Regra.
  3. Localize o processo referente a SAFX (Ex.: ZSAFE026 - Documentário Fiscal).
  4. Verifique o que está ativo, e:
     1. para desativar basta desmarcá-lo e salvar;
     2. para ativar basta marcar os campos desejados.
  5. Realize novamente a extração e verifique se o problema persiste.


Nota 1: No caso de documentário fiscal é necessário verificar as Exits para as três SAFX. Recomendamos um ABAP para realizar este procedimento.

Nota 2: É mais fácil localizar a exit pelo código, ao invés da denominação.
