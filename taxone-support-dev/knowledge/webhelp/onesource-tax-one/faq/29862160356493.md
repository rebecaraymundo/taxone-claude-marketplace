# [Básicos][Job Servidor] TAX ONE - Como resolver problema de duplicação de documentos devido a diferenças no campo "Série"?

Caso enfrente duplicação de documentos na importação mesmo tendo campos chaves idênticos, o problema pode ser a diferença nos valores do campo "série" (exemplo: "1" e "001"), que é um campo chave e do tipo VARCHAR, o que torna essas entradas diferentes para o sistema. Para resolver, siga as orientações:

  1. Verifique o campo "série" nos documentos que estão sendo importados.
  2. Garanta que os valores no campo "série" sejam consistentes em todas as importações. Por exemplo, se um documento foi importado com a série "1", todas as importações subsequentes desse documento devem usar a série "1" e não "001".
  3. Reimporte os documentos com os valores corretos no campo "série" para evitar duplicações.
