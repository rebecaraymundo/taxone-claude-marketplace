# Resolução de Erro na Geração do Código de Situação no Municipal de Monte Mor

###

Código de Situação, não esta sendo gerado no Meio Magnético!

## Causa Raiz:

O problema ocorreu devido à falta de parâmetro adequado na tela 'Parâmetros – Classificação de Serviços'. O campo COD_PARAM não estava configurado para '495', que é necessário para gerar o código de situação '0'.

## Resolução:

  1. Acesse o sistema e vá até a tela 'Parâmetros – Classificação de Serviços'.
  2. Verifique se o serviço e o município do prestador cadastrados na nota estão corretamente parametrizados.
  3. Insira ou atualize o COD_PARAM para '495'.
