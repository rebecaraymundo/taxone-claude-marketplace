Para resolver a situação em que as SAFX, que não têm um código de empresa associado, são consumidas via SFTP, mas não são visualizadas no sistema TAX ONE, siga a orientação abaixo:

**Nota:** Exemplos de SAFX sem código de empresa incluem: SAFX04, SAFX2012, entre outros.

1\. Identificação do Erro:
A SAFX é processada no SFTP, mas não é exibida no sistema TAXONE.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/15236762787341)

2\. Causa do Problema:
O erro ocorre quando não há um código de empresa configurado para as SAFX que estão sendo carregadas e que não possuem este código.

3\. Solução:
Acesse o menu 'Básicos > Ferramentas > Parâmetros do Sistema > Preferência.

Adicione o código da empresa desejada para que as cargas das SAFX específicas sejam realizadas com sucesso.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/15236859598861)
