# Alteração do Valor da Guia DARF no TaxOne

## Problema:

Usuário deseja saber se é possível alterar o valor de uma guia DARF antes de imprimir no sistema TaxOne, especificamente para um código de receita único.

## Causa Raiz:

O sistema TaxOne não permite a alteração do valor da guia DARF, pois os valores são obtidos diretamente da origem, que é o documento fiscal. Esta configuração garante a consistência dos dados fiscais.

## Resolução:

Infelizmente, **não é possível** alterar o valor da guia DARF para impressão no TaxOne, pois os valores são fixados conforme as informações do documento fiscal. A emissão da guia deve ser feita com o valor original especificado no documento.

Módulo: Federal > OTF > Outras Obrigações > DARF`s > Manutenção:

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/41381202234509)

## Próximos Passos:

Caso necessite emitir a guia com um valor diferente, será necessário revisar e ajustar o documento fiscal de origem que está gerando o valor da guia. Recomenda-se entrar em contato com o departamento fiscal para mais orientações.

## Dicas e Recomendações:

Para evitar discrepâncias, sempre verifique os valores nos documentos fiscais antes de proceder com a emissão de guias DARF. Em caso de dúvidas, consulte o suporte técnico ou o departamento fiscal responsável.
