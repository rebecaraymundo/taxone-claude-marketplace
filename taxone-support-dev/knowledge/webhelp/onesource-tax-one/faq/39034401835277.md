## Problema

O sistema não está gerando os registros de contribuições conforme esperado.

## Causa Raiz

O problema pode estar relacionado a configurações incorretas na base de dados, equalização da base ou parâmetros de geração inadequados.

## Resolução

  1. Certifique-se de que a base de dados está corretamente equalizada. Caso não esteja, execute os procedimentos de equalização recomendados pela documentação do sistema.
  2. Verifique os parâmetros configurados para a geração dos registros. Certifique-se de que estão correspondendo ao padrão necessário para o processo.
  3. Se aplicável, revise as opções de geração no sistema. Ajuste os parâmetros para incluir a opção de geração expressa, caso necessário.
  4. Teste a geração dos registros novamente após realizar as correções. Certifique-se de que os registros estão sendo gerados corretamente.
  5. Caso o problema persista, entre em contato com o suporte da solução para assistência adicional.


## Próximos Passos

Após seguir os passos acima, monitore o processo de geração de registros para garantir que os ajustes implementados foram eficazes. Realize testes adicionais em diferentes cenários, se necessário.
