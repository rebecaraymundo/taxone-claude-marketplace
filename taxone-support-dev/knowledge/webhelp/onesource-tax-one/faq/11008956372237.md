Para resolver o problema quando o **Registro K100 **está marcado no perfil, mas não é gerado no arquivo do SPED:


É necessário realizar a **Apuração do Livro ICMS (P9)** do período.

Atenção: O registro K100 trata-se do período de apuração do **ICMS/IPI** , que é o mesmo período demonstrado no bloco E100, por isso o livro deve ser gerado. Esta geração é feita apenas para abrir o período, nenhum valor na apuração é apurado.
