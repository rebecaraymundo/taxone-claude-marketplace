Quando **não** se tratar de uma NF de transferência de bem, verifique os seguintes parâmetros:


1\. Verifique o parâmetro no módulo Ativo Permanente, menu Cadastros > Parâmetros p/ Geração das Aquisições a partir das NFs > Por CFOP. ![](https://suporte.thomsonreuters.com.br/hc/article_attachments/18367153326989)

2\. Para o CFOP 1551/2551 e demais CFOPs de aquisição, esse parâmetro não pode estar configurado como **002 - Transferência - Entrada**.

\- Ajuste o parâmetro para **001 - Entrada por Aquisição**.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/18367306276621)

3\. Verifique o parâmetro no módulo Ativo Permanente, menu Cadastros > Tipo de Movimento.



4\. Esse parâmetro não pode estar configurado como**Entrada por Transferência.**

\- Ajuste o parâmetro para **Entrada por Aquisição**.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/18367510970893)

5\. Após ajustar esses parâmetros corretamente, a Geração das Aquisições a partir das NFs deverá ser realizada com sucesso.
