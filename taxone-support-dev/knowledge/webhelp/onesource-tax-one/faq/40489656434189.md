# [Básicos][OBI - OneSource Brasil Integration] Integração com a API do OBI: Uso de VPN

**Problema:** O usuário tem dúvidas sobre a necessidade ou a possibilidade de utilizar uma VPN (Rede Virtual Privada) para a comunicação e o tráfego de dados entre seu sistema (como o OBI Sender) e a API Gateway do OBI.

**Causa Raiz:** A confusão surge devido às práticas de segurança comuns em algumas empresas. No entanto, o canal oficial de comunicação para o OBI é público e não requer o uso de uma VPN.

**Resolução:** Não é necessário nem possível utilizar uma VPN para o tráfego de dados entre seu sistema e a API Gateway do OBI.
