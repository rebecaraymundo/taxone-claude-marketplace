#### Erro no Log- Para o serviço 1601, não foi localizada a parametrização de Serviços Lei 116 x Serviços Municipais. É necessário efetuar a parametrização no menu Parâmetros > Serviços > Serviços Lei 116 x Serviços Municipais x Código Atividade.


O erro ocorre devido à ausência de uma parametrização necessária no sistema. A configuração de serviços conforme a Lei 116/03 não foi devidamente relacionada aos serviços municipais e aos códigos de atividade, impedindo a geração correta do arquivo para validação.

Acesse o Modulo GISS ONLINE> Parâmetros por município

Navegue até o menu **Parâmetros Serviços Serviços Lei 116 x Serviços Municipais x Código Atividade**.

Realize a configuração dos serviços, conectando os códigos de atividade aos serviços municipais conforme exigido pela Lei 116.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/39030729018381)

Além disso verifique se no módulo **Básicos / Data Warehouse Menu: Códigos > Serviços**, o campo **"Código Serviço Lei 116/03"** está devidamente preenchido para os códigos de serviço que apresentaram erro no log.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/39030729019405)

Após realizar as parametrizações, gere um novo arquivo no Modulo GISS ONLINE > GISS Paulínia, no menu: Geração > Arquivo de entrada de Serviços.

Após as configurações o campo Cod Serviço, é devidamente preenchido.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/39030702495885)
