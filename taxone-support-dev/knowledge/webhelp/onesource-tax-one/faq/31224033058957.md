Caso enfrente dificuldades para parametrizar múltiplas naturezas para o mesmo CST devido à configuração do sistema que considera a conta contábil como chave nesse contexto. A configuração do sistema faz com que o campo "Conta Contábil" seja tratado como uma chave única quando a parametrização é feita por CST/CONTA CONTÁBIL, gerando conflitos ao tentar parametrizar múltiplas naturezas com o mesmo CST. Para resolver essa questão, siga os passos abaixo:

  1. Acesse o módulo de parametrização do sistema.
  2. Selecione a opção de parametrização por "CST/Alíquota".
  3. Insira o mesmo CST e a conta contábil desejada.
  4. Parametrize as diferentes naturezas necessárias.
  5. Salve as alterações e teste para garantir que as novas configurações estão funcionando corretamente.
  6. Após implementar a solução, monitore o sistema para garantir que a parametrização foi bem-sucedida e não está causando outros problemas operacionais. Documente qualquer ajuste adicional necessário e continue verificando o desempenho regularmente.


**Nota:** Considere revisar regularmente as configurações de parametrização para se adequar às mudanças nas operações fiscais e contábeis. Mantenha-se atualizado com as últimas práticas recomendadas e atualizações do sistema, acessando a central de ajuda e as políticas fiscais disponíveis para seu software.
