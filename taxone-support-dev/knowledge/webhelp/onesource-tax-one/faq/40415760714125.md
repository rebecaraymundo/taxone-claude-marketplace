SAFX52 (Processo ZSAFE031):
Objetivo: Baseia-se no estoque próprio da empresa para construir a extração SAFX52.

Para analisar se o conteúdo extraído está de acordo, sugerimos a utilização do relatório emitido através da transação MB5B.


Parametrizações Utilizadas:
• PARAMETRIZAÇÃO GERAL - Dentro da aba “Inventários”, devemos selecionar os campos que forem necessários de acordo com a designação de cada empresa.


SAFX52 (Processo ZSAFE031_V2):
Objetivo: Inclui na geração da SAFX52, tanto o estoque próprio quanto o estoque de terceiros.

Para analisar se o conteúdo extraído está de acordo, sugerimos a utilização do relatório emitido através da transação MB5B.


Parametrizações Utilizadas:
• PARAMETRIZAÇÃO GERAL - Dentro da aba “Inventários”, devemos selecionar os campos que forem necessários de acordo com a designação de cada empresa.


SAFX52 (Processo ZSAFE052_INV):
Objetivo: Inclui tanto o estoque próprio quanto o estoque de terceiros. Utiliza toda a seleção de valores/quantidades por meio de funções standards, disponibilizadas pela SAP e utilizadas nos relatórios standard.

Para analisar se o conteúdo extraído está de acordo, sugerimos a utilização do relatório emitido através da transação MB5B.


Parametrizações Utilizadas:
• PARAMETRIZAÇÃO GERAL - Dentro da aba “Inventários”, devemos selecionar os campos que forem necessários de acordo com a designação de cada empresa.

• CLASSE DE AVALIAÇÃO (GRP. CONTAG.) - Nesse parâmetro, iremos configurar para cada classe de avaliação utilizada, o grupo de contagem correspondente.

• TIPO DE MATERIAIS - Nesse parâmetro, iremos configurar para cada tipo de material SAP, as seguintes informações:

• Nat. Estoque (caso existir)

• Tipo de Material para solução fiscal (ver opções no match code)

• Class. Item para solução fiscal (ver opções no match code)

• COD. ESTOQUE ESPECIAL (GRP. CONTAG.) - Nesse parâmetro, iremos configurar para cada código de estoque especial utilizado, o grupo de contagem correspondente.
