## **Erro 90767: Código de Canal de Distribuição não Cadastrado**

**Problema:** A importação de um arquivo SAFX07 retorna o erro "O Codigo de Canal de Distribuição não está cadastrado", impedindo o processamento dos registros.

**Causa Raiz:** O erro ocorre porque o valor informado no campo `COD_CANAL_DISTRIBUICAO` do arquivo SAFX07 não existe no cadastro de **Canais de Distribuição** no sistema. Isso pode acontecer porque o código não foi previamente cadastrado, ou porque o campo está vazio, sendo que o padrão para a ausência de um código é o caractere `@`.

**Resolução:** Para solucionar o erro, verifique qual é a situação e siga as instruções:

  1. **Se o código de distribuição existe no seu arquivo, mas não no sistema:**
     * Acesse o módulo **Data Warehouse**.
     * Navegue até o menu **Manutenção > Cadastros > Canais de Distribuição/Obras**.
     * Cadastre o código de canal de distribuição que está faltando.
  2. **Se não houver um código de distribuição a ser informado:**
     * Preencha o campo `COD_CANAL_DISTRIBUICAO` do registro no arquivo com o caractere `"@"`.


Após a correção, execute a carga/importação do arquivo SAFX07 novamente.

**Próximos Passos:** Se o problema persistir com outros códigos ou se o erro continuar após a correção, verifique a integridade dos dados no arquivo de origem. Em seguida, abra um ticket de suporte com o arquivo em questão para uma análise mais detalhada.
