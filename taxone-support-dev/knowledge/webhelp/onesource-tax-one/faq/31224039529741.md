Dificuldades na geração do Relatório de Controle de Tributos, em que apenas os códigos IRRF 1708 e 0422 são incluídos, enquanto o código CSRF 5952 não é gerado, podem ocorrer devido à ausência de notas no SAP para o código CSRF 5952, o que impede a geração do relatório para este código de tributo.

Passo a passo para resolver o problema com a geração do Relatório de Controle de Tributos para o código CSRF 5952:

  1. Verifique a existência de notas no sistema SAP para o código CSRF 5952.
  2. Confirme se as notas estão no controle de tributos (SAFX53) com os valores de CSLL/PIS/COFINS no período de geração do relatório.
  3. Caso não haja notas, é necessário incluir as notas pertinentes ao CSRF 5952 no sistema TAXONE.
  4. Após a inclusão das notas, tente gerar o relatório novamente.
  5. Após seguir os passos acima, monitore a geração de relatórios para outros códigos de tributos para garantir que o problema foi completamente resolvido. Se o problema persistir, entre em contato com o suporte técnico.


**Dicas e Recomendações:**

É importante manter o sistema SAP atualizado com todas as notas fiscais relevantes para evitar problemas na geração de relatórios fiscais. Regularmente verifique a integridade dos dados no SAP para garantir que todas as informações necessárias estão sendo capturadas corretamente.
