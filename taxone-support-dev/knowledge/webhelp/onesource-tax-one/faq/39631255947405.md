## Problema:

O arquivo SAFX01 enviado para integração no sistema Taxone por meio do processo SFTP não foi consumido corretamente, e o cliente não conseguiu visualizar os registros após a carga.

## Causa Raiz:

O problema ocorreu devido a inconsistências no formato do arquivo base e na compactação utilizada durante o envio. O sistema exige que os arquivos tenham a extensão .txt e estejam compactados no formato .zip, seguindo padrões específicos de compressão compatíveis com o Java.

## Resolução:

  1. Certifique-se de que o arquivo .txt esteja salvo no formato correto, que deve ser **ANSI**. Caso o arquivo esteja em outro formato, como UTF-8 BOM, realize a conversão para ANSI antes de prosseguir.
  2. Para realizar a compactação, utilize a ferramenta **7-Zip** ou o compactador padrão do Windows.
  3. Certifique-se de compactar o arquivo para o **formato .zip**. Ondas compactações com extensões diferentes, como .7z ou .rar, não são suportadas.
  4. Envie o arquivo compactado para o sistema Taxone por meio do processo SFTP.
  5. Após o envio, confirme no sistema se o arquivo foi consumido corretamente e se os registros estão disponíveis.
