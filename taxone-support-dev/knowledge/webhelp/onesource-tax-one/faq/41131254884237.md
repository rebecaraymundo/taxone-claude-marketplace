# Como corrigir a TAG "CnpjIncorporacao" gerada incorretamente no MIT

###

## Problema

Para alguns registros do MIT, a TAG "CnpjIncorporacao" está sendo gerada com o CNPJ da sócia ostensiva, ao invés do CNPJ correto relacionado ao RET cadastrado.

## Causa Raiz

A causa do problema está relacionada à parametrização incorreta no sistema. Especificamente, o campo "Gera por matriz" estava configurado de forma inadequada para o cenário, causando a geração do CNPJ incorreto na TAG "CnpjIncorporacao".

## Resolução

Siga os passos abaixo para corrigir o problema:

  1. Acesse o sistema e navegue para: **Básicos Ferramentas Tabelas Internas Manter Tabelas Código de Receita**.
  2. Localize o código de receita relacionado ao problema (por exemplo, **411201**).
  3. Altere o campo "Gera por matriz" de **SIM** para **NÃO**.
  4. Certifique-se de que os parâmetros relacionados ao MIT estão configurados corretamente conforme indicado na documentação inicial.
  5. Efetue uma nova geração do MIT e do Meio Magnético.
  6. Verifique se a TAG "CnpjIncorporacao" está sendo gerada corretamente com o CNPJ esperado.

## Próximos Passos

Após aplicar as correções:

  1. Realize testes adicionais para garantir que o sistema está gerando os dados corretamente para todos os códigos de receita relacionados.
  2. Monitore o comportamento da geração do MIT nos próximos arquivos gerados para confirmar a resolução completa do problema.
  3. Documente qualquer ajuste adicional realizado para referência futura.


## Dicas e Recomendações

  * Mantenha a documentação de parametrizações atualizada e acessível para evitar erros futuros.
  * Antes de realizar alterações nos parâmetros, consulte a equipe responsável ou a documentação oficial para entender os impactos potenciais.
  * Realize testes em ambiente UAT antes de aplicar qualquer mudança no ambiente de produção.
  * Se o problema persistir, entre em contato com o suporte técnico para análise detalhada.


Para mais informações sobre o MIT e parametrizações no sistema, consulte o guia de usuário disponível no portal de suporte.
