A interface SAP Namespace trabalha com definições de melhores práticas e padrões estabelecidos pela SAP. Na Interface o processo para gerar a SAFX80 está aderente se o cliente utiliza o NEWGL (Consta tabelas standards de saldos).

Caso não tenha o Newgl ativo, a interface tem seleção de dados das contas contábeis para as tabelas BSIS e BSAS (Movimentos), porém essas tabelas não têm informações de saldos de contas contábeis.

Para esse cenário é necessário que o cliente providencie possíveis customizações em exits de regras para fazer a apuração dos saldos Contábeis anterior e Saldos Contábeis Atual.

A interface não pode "apurar" ou "calcular" esses saldos, pois nem Interface nem DW podem criar dados.
