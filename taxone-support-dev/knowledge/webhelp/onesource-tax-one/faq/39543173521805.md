# Como Gerar Saldos Contábeis para Períodos Atuais no Sistema TAX One

## Problema:

O usuário enfrenta um problema onde os saldos contábeis gerados pelo sistema TAX One estão mostrando dados apenas até 31/12/2024, apesar dos registros terem sido importados para o período de 01/01/2025 a 31/01/2025.

## Causa Raiz:

A causa raiz do problema é que foi importado apenas a tabela SAFX01 de lançamentos contábeis e a tabela de saldos contábeis SAFX02 não foi importada ou gerada via aplicação para o período solicitado de janeiro de 2025.

## Resolução:

  1. Acesse o sistema TAX One.
  2. Navegue até a opção SPED > ECD > Geração > Saldos.
  3. Consulte se a SAFX02 de janeiro de 2025 está presente e correta. Caso não esteja, você deve importá-la ou gerá-la neste local.
  4. Confira se os dados estão corretos após a geração/importação.


## Próximos Passos:

Após gerar ou importar a tabela de saldos correta, monitore os relatórios nos primeiros dias para garantir que a visualização dos saldos está sendo realizada de forma satisfatória. Em caso de discrepâncias, entre novamente em contato com o suporte técnico.

## Dicas e Recomendações:

É aconselhável verificar frequentemente a correta importação ou geração das tabelas de saldos contábeis, especialmente após o fechamento de cada período fiscal. Mantenha comunicação regular com o suporte técnico para esclarecer dúvidas ou relatar problemas inesperados.
