**GUIA DE INFORMAÇÃO MENSAL DE RORAIMA**

"O objetivo desse módulo é gerar a Guia de Informação Mensal do ICMS, que deve ser entregue pelos contribuintes enquadrados no regime normal de recolhimento ou de estimativa, inscritos no Cadastro Geral da Fazenda - CGF- RR".

Base legal; A GIM-RR está amparada pelo Decreto nº 4.335-E, art. 275 do RICMS.


Para que esta obrigação, em forma de relatório, seja gerada corretamente, observar os seguintes procedimentos [Premissas para geração da GIM - RR](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Estadual/GIM_RR/rot_safourr.htm>):


**PASSO A PASSO DE GERAÇÃO GIM RR - GUIA DE INFORMAÇÃO MENSAL DO ICMS**

Cadastrar os Códigos de Receita Estadual para o estado de Roraima.

Em Básicos > Data Warehouse > Manutenção > Códigos, selecione [Código da Receita Estadual. ](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Basicos/Basicos_Data_Warehouse/Basicos_Data_Warehouse_Menu_Manutencao/Basicos_DW_Menu_Manutencao_Codigos.htm>)

Esta informação é obrigatória para o cadastramento das Guias de ICMS, utilizadas para a geração do Quadro H.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/20553461271821)

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/20553461274893)

Após cadastrados os Códigos de Receita Estadual para o estado de Roraima, acessar;

Menu Parâmetros >Ativo Fixo ou Consumo > Por CFOP

Nesta tela devem ser marcados os CFOP’s das operações de entradas de mercadorias e de utilização de serviços de transportes e comunicação, do Estado, de outros estados e do exterior, destinadas ao ativo fixo ou consumo do estabelecimento. A partir destes CFOP’s são buscado na apuração do ICMS os valores para a geração do Quadro I.

Também devem ser marcados os CFOP’s que indiquem venda do ativo imobilizado. Neste caso, estes CFOP’s são excluídos do cálculo do Quadro F.

Exposto isto, estes parâmetros tem duas funções:
1\. Verificar CFOP’s de entrada a serem considerados no cálculo do Quadro I
2\. Verificar CFOP’s de entrada e de saída a serem desconsiderados no cálculo do Quadro F

Gerar o Livro de Apuração do ICMS (108 ou 165)

Movimento de Estoque: Este Quadro (J) tem a função de demonstrar os valores iniciais do estoque no ano corrente. Para isto, sempre os valores serão recuperados no dia 31/12 do ano anterior.


Para ser gerado, na tela de geração deve estar marcado o parâmetro para geração do Quadro J e escolhido através de qual tabela será a recuperação de dados.
Normalmente este Quadro é entregue na GIM de março.

**Geração do Relatório (Auxiliar para Preenchimento no site Sefaz RR)**


Período: Informar o período mensal de geração.

Gim Substitutiva: Marcar, caso a GIM seja substitutiva.

Geração p/ Inscrição Estadual Única: Caso a empresa tenha inscrição estadual única, o usuário deverá marcar este parâmetro. Ao ser marcado, só aparecerão os estabelecimentos centralizadores, e os dados devem ser gerados sempre com os parâmetros dos estabelecimentos centralizadores. Os dados são recuperados do estabelecimento centralizador e também de seus centralizados. Para isto, verificar:
Controle de Obrigações Estaduais → Obrigações → Empresas/Estabelecimentos com IEÚnica.
Gerar Quadro J (Estoque): o usuário pode marcar, caso queira que o Quadro J seja gerado. Quadro geralmente entregue em março, com as informações de inventário do ano anterior.


Estoque: caso o Quadro J seja marcado para geração, o usuário deverá indicar de qual tabela as informações são buscadas:
\- Por Produto → recuperar dados da safx52
\- Por NBM → recuperar dados da safx62


Local e Data: Campo para que o usuário digite texto livre, com letras e números. Campo de preenchimento obrigatório para a geração do Quadro M.
Exemplo de preenchimento: Boa Vista, 31 de março de 2010.


Representante Legal: escolher qual o responsável pelas informações. O mesmo deve ser cadastrado em:
Parâmetros → Requisitos Legais → Responsável por Informações.

Campo de preenchimento obrigatório para a geração do Quadro M.

Estabelecimentos: Marcar estabelecimento para geração.
