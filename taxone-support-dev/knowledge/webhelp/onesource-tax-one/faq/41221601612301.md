# [SPED][SPED Fiscal][Bloco E]  Erro na Geração de Valores do Registro E110

## Problema

O usuário relatou que os valores do Registro E110 não estão sendo gerados para um estabelecimento. Foi informado que a empresa não possui sub-apuração de ICMS.

## Causa Raiz

A causa do problema está relacionada à seleção do Registro 1900 no cadastro do perfil. Quando este registro está marcado, ele interfere na geração correta dos valores do Registro E110.

## Resolução

  1. Acesse o cadastro do perfil da empresa no sistema.
  2. Verifique se o Registro 1900 está selecionado.
  3. Se estiver selecionado, desmarque a opção.
  4. Refaça a geração do meio magnético.
  5. Confirme se os valores do Registro E110 foram populados corretamente.


## Próximos Passos

Após desmarcar o Registro 1900 e verificar a geração correta dos valores do Registro E110, o usuário deve monitorar as próximas gerações de arquivos para garantir que o problema não volte a ocorrer.
