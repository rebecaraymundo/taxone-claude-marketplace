# Correção de Erro de Layout na Submissão de NFS-e no Município de Descalvado

###

O erro ocorreu porque o código do serviço, exigido pela Lei 116/03, não foi incluído no arquivo gerado pelo sistema Tax One. Isso ocorreu devido à falta de parametrização correta no módulo de Parâmetros ISS.

## Resolução:

Acesse o módulo **Básicos Data Warehouse** no sistema Taxone / Selecione a opção **Códigos > Serviços** no menu.

  1. Localize os serviços usados nos documentos fiscais.
  2. Realize a parametrização inserindo o código do serviço conforme a Lei 116/03.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40966706902413)

## Próximos Passos:

Após realizar a parametrização, submeta novamente o arquivo da NFS-e no sistema da Prefeitura de Descalvado. Verifique se o envio é aceito sem erros. Em caso de persistência do problema, entre em contato com o suporte técnico.
