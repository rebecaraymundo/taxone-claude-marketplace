Para criar um Perfil de Geração no EFD Contribuições, siga as orientações:

1\. Em SPED > EFD Contribuições > Parâmetros > Perfil:

  * Clique no botão "+" (Novo).


![](https://t12927461.p.clickup-attachments.com/t12927461/a70d159b-7eb8-4a97-ad6a-70db587adc8b/image.png)

  * Selecione um dos Layouts disponíveis. Neste exemplo, utilizaremos a partir da versão 1.32


![](https://t12927461.p.clickup-attachments.com/t12927461/3c9ca789-2f65-4b7e-9610-862d8366d016/image.png)

  * Digite um código e nome do perfil.


![](https://t12927461.p.clickup-attachments.com/t12927461/4bc894fa-9a50-4681-a715-2a2fd4b018a8/image.png)

![](https://t12927461.p.clickup-attachments.com/t12927461/63d46d23-90b1-4e86-9748-8fe9e49df19c/image.png)

  * Na tabela Definição dos Blocos, selecione o bloco desejado.


![](https://t12927461.p.clickup-attachments.com/t12927461/846a1f20-9c6e-45e9-b235-12750f8737a7/image.png)

  * Na tabela Reg. Nível Descrição, marque os registros que devem ser gerados no meio magnético do EFD Contribuições. Neste exemplo estamos marcando todos os registros conforme abaixo, utilizamos "selecionar todos".


![](https://t12927461.p.clickup-attachments.com/t12927461/9c14416e-fd7f-428e-8ed9-84c6210d63ba/image.png)

Nota: O passo acima deve ser efetuado para cada bloco selecionado.

![](https://t12927461.p.clickup-attachments.com/t12927461/4daeeed6-8b10-4f7b-862f-7d57389c50b2/image.png)

  * Após finalizar as marcações acima, clique no botão "confirma".


![](https://t12927461.p.clickup-attachments.com/t12927461/6d3e8061-d91b-491c-a4a5-74a2a5b5c6c0/image.png)
