#

Este artigo tem por objetivo, demonstrar a geração do campo ImpRetido no arquivo ISS - Validador: ISS.WEB
Neste exemplo, o município de geração do arquivo é Vila Velha.

Iniciaremos demonstrando a geração da informação no campo ImpRetido = **I - Isento**

**CADASTRO DA PESSOA FÍSICA/JURÍDICA DA NOTA** : Básicos > Data Warehouse > Manutenção > Cadastros > Pessoas Físicas/Jurídicas


![KBA ISS Web ImpRetido - PRINT 1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/40877315019533)


com o documento fiscal:

**DOCUMENTO FISCAL** : Básicos > Data Warehouse > Manutenção > Documento > Doc. Fiscal de Mercadoria e Serviço

Doc Num: 123

![KBA ISS Web ImpRetido - PRINT 2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/40877330743437)

O item de serviço: 0097 - FISIOTERAPIA
![KBA ISS Web ImpRetido - PRINT 4.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/40877315021837)


**EQUALIZAÇÃO/LIMPEZA** : Básicos Data Warehouse > DATA MART > Equalização/Limpeza
Obs.: Importante fazer a equalização do período após a inserção de quaisquer dados de documentos fiscais.


**CLASSIFICAÇÃO DE SERVIÇOS POR PESSOA FIS/JUR** : Municipal > Parâmetros Para Município > Parâmetros > Classificação de Serviços > Pessoa Fis/Jur

![KBA ISS Web ImpRetido - PRINT 3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/40877315021965)


**GERAÇÃO DO MEIO MAGNÉTICO** : Municipal > Vila Velha > Obrigações >Meio Magnético

![KBA ISS Web ImpRetido - PRINT 5.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/40877315022349)

Veja que no arquivo foi gerado "I" no campo ImpRetido (que fica na posição 242)

![KBA ISS Web ImpRetido - PRINT 6.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/40877330751245)

Neste segundo exemplo, demonstraremos a geração do campo com a informação = S - Imposto Retido

![KBA ISS Web ImpRetido - PRINT 11.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/40877330757133)
