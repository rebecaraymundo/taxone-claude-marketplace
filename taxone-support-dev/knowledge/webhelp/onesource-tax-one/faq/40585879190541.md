# Correção do Código da Situação Tributária em Declarações de Serviço

###

## Causa Raiz:

O erro ocorreu devido à falta de parametrização correta no sistema para o código '473', que não estava ajustado para refletir a situação tributária correta quando há deduções aplicáveis e o município do estabelecimento é especificamente Palhoça.

## Resolução:

  1. Acesse o Módulo Municipal > Parâmetros por Município, menu: Parâmetros >Classificação de Serviços > Pessoa Fis/Jur
  2. Utilize o COD_PARAM (cod_classif) para a parametrização:
  3. Defina: Se Cod_classif = 473, Cod_municipio_Nf = Cod_municipio_Estab e Ind_tp_ret = 1, então cod_sit_tributaria deve ser igual a 04.
