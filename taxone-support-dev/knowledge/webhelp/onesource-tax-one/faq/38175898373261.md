### **Como Gerar o Relatório 'Detalhes do Usuário' e Consultar o Último Login**

**Problema:** Um usuário precisa de um relatório que contenha a lista completa de todos os usuários do sistema, incluindo a **última data de login** , mas não consegue extrair essa informação de forma direta.

**Causa Raiz:** As informações de detalhes e atividade dos usuários, como a data do último acesso, são centralizadas na funcionalidade de relatórios do sistema, e não são projetadas para extração direta via banco de dados.

**Resolução:** Para obter o relatório com o último acesso de todos os usuários, siga os passos abaixo:

  1. Acesse o módulo **Administration**.
  2. No menu, selecione **Reporting** e, em seguida, clique em **User Details**.
  3. Na tela de filtro, para garantir que todos os usuários sejam incluídos, deixe todos os campos de filtro em branco.
  4. Clique em **Search**.
  5. O relatório será gerado. A coluna **"Último Login"** exibirá a data da última atividade de acesso de cada usuário.


**Próximos Passos e Recomendações:**

  * Utilize o relatório de **User Details** regularmente para monitorar a atividade e a segurança dos usuários no sistema.
  * Este procedimento é a forma oficial de acessar essas informações, garantindo a integridade dos dados e a conformidade com as regras de acesso.
  * Após a geração do relatório, confirme se os dados atendem à sua necessidade de auditoria e segurança.
