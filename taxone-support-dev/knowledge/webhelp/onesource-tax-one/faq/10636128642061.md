Passo a passo para corrigir o **Erro 90366** \- O campo Código NCM não está cadastrado:

  1. Cadastre o** Código do NCM** manualmente em [Básicos > Data Warehouse > Manutenção > Códigos > Código NCM](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Basicos/Basicos_Data_Warehouse/Basicos_Data_Warehouse_Menu_Manutencao/Basicos_DW_Menu_Manutencao_Codigos.htm>). Também é possível realizar o cadastro utilizando a SAFX2045.


Nota: O erro ocorre quando o campo COD_NCM é preenchido com um código que não está cadastrado no sistema.
