Para fornecer orientações sobre a homologação de serviços SFTP para comunicação com o Tax One, é importante destacar que os clientes têm a liberdade de escolher a aplicação de SFTP que preferirem. O Tax One é compatível com uma variedade de ferramentas de SFTP, como FileZilla, WinSCP, WinPRO, FreeFTP, entre outras.


Para que a transferência de dados via SFTP seja bem-sucedida, é crucial que as credenciais de conexão sejam utilizadas corretamente. Essas credenciais são essenciais para estabelecer uma conexão segura e podem ser encontradas dentro do próprio sistema do Tax One.
