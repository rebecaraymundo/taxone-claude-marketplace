# [Interfaces] [SAP – Namespace/OBTI] [Web Service] Falha de conexão entre OBTI (WebService) com o Tax One

Problema

Ocorrem problemas ao tentar realizar cargas via Webservice. As cargas iniciadas estão paradas no SAP com um ícone de engrenagem, indicando que não foram finalizadas. Além disso, no sistema Tax One, não há registros dessas cargas, como se não tivessem sido enviadas.

Causa Raiz

Através da tela de "Parâmetros Gerais" aba "WebService" identificado que o "farol" do status da conexão esta vermelho, outra maneira de verificar é pela extração da tabela /MSAFX/SAFTB087, foi identificado que o campo USER_BLOCK estava marcado, o que indica que o usuário configurado para integração no SAP estava bloqueado. O bloqueio pode ter ocorrido após múltiplas tentativas de conexão sem sucesso, acionando o bloqueio automático do sistema devido à senha configurada estar incorreta e/ou expirada.

Resolução

  1. Acesse o Tax One com um usuário com perfil a administrador e desbloqueie o usuário de integração.
  2. Se necessário, altere a senha configurada para este usuário de integração no Tax One.
  3. Tente logar no Tax One com o usuário configurado no SAP para confirmar que o bloqueio foi removido e as credenciais funcionam.

Observação: Após o desbloqueio no Tax One, a conexão é restabelecida automaticamente. O desbloqueio na Interface OBTI também é efetuado e, na tela indicada abaixo, o "farol" do status da conexão deverá ficar verde.

  4. No SAP, acesse a transação /N/MSAFX/PARAMETROS - PARAMETRIZACAO GERAL >vá a aba "WebServices"; abaixo dois prints de exemplo para especificar o caminho da tela:

![](/attachments/token/Hl9j6IavfhzVe6aV1GmYypezy/?name=image.png)![](/attachments/token/kvcxvNaeRzy7wTTKiNK0Wg4X5/?name=image.png)
  5. Confira se o Status da Conexão mudou para verde, indicando que a conexão foi restaurada com sucesso.

  6. Caso seja necessário atualize os campos "Usuário" e "Senha" destacados na mesma tela acima com as credenciais do usuário integrador


Próximos Passos

  * Após executar as instruções de resolução, monitore as cargas e a integração com o Tax One para confirmar que estão ocorrendo sem novos erros.

  * Se o problema persistir, repita a verificação e ajuste das configurações


Dicas e Recomendações

  * Efetue a troca periódica da senha do usuário, respeitando seu prazo de expiração, com foco nos usados para integração via WebService.

  * Configure alertas/monitoramento para bloqueios de usuário, falhas de autenticação e quedas de conexão.

  * Após qualquer alteração de credenciais, teste a conexão e valide o "status"/farol na tela de parametrização.
