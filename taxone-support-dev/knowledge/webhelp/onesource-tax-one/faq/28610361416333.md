Quando os usuários não conseguem visualizar a opção de seleção de empresa após realizar o login na aplicação. Normalmente, este problema pode-se atribuir aos usuários que foram cadastrados com e-mail no nome de usuário, o que não é permitido pela configuração do sistema. Para resolver, siga as orientações:

1\. Solicite ao usuário Administrador que corrija os nomes de usuário, removendo os e-mails e deixando apenas o nome do usuário. Por exemplo:

  * **Errado:** SUPORTE_PRD_joao.silva@grupotr.com.br
  * **Correto:** SUPORTE_PRD_joao.silva


2\. Após a correção, basta efetuar novo acesso para confirmar que o problema foi resolvido.

3\. Após a correção dos nomes de usuário, os usuários afetados devem tentar acessar novamente a aplicação para verificar se a opção de empresa está sendo exibida corretamente. Caso o problema persista, entre em contato com o suporte técnico.

**Importante:** Sempre siga as diretrizes de formatação de nomes de usuário fornecidas no manual do sistema para evitar problemas de acesso e visualização de opções dentro da aplicação. Mantenha os dados de usuário sempre atualizados e em conformidade com as especificações técnicas.
