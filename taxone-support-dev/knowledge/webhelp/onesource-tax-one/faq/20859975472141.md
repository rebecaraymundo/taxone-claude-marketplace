Normalmente, esse problema ocorre porque o código informado no campo COD_SERVICO não está cadastro na tela Básicos> Data Warehouse > Manutenção > Códigos > Serviços ou a DATA_OPERACAO é menor que a data inicial de validade do COD_SERVICO cadastrado no sistema. Para corrigir o erro 90135: 'O Campo Código de Tipo de Serviço não esta cadastrado.', para solucionar o problema:

\- Informe um código valido no arquivo TXT;

\- Cadastre o COD_SERVICO na tela Básicos> Data Warehouse > Manutenção > Códigos > Serviços;

\- Verifique a data de validade do serviço no cadastro e realize o ajuste.
