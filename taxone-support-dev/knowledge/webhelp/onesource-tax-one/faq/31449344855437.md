A regra geral para o preenchimento do campo "COD_SIT" do Registro C100 devem ser verificados alguns campos da SAFX07, conforme segue:

  1. Se for nota eletrônica, verificar:
     * se a nota foi denegada à gravar “04”
     * se a nota foi ou inutilizada à gravar “05”
  2. Se for nota cancelada, verificar:
     * se a nota é extemporânea à gravar “03”
     * se a nota é normal (não extemp.) à gravar “02”
  3. Se for nota extemporânea, verificar:
     * se a nota é complementar à gravar “07”
     * se a nota é normal (não compl.) à gravar “01”
  4. Se for nota complementar:
     * gravar “06”
  5. Se nota tem Ato Normativo, Regime Especial etc...
     * gravar “08”
  6. Se a nota não atende a nenhuma das opções anteriores:
     * gravar “00”


**Nota:** Lembrando que existem particularidades para algumas UF's.
