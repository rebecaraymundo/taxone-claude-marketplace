Erro: Ao tentar excluir relacionamento no cadastro de pessoa física/Jurídica, é apresentada a mensagem

''Tentativa de excluir registro referenciado em outra tabela''

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21507000487181)

Esse erro ocorre quando o código da pessoa física/jurídica esta vinculado em algum lançamento contábil, Logo não é possível excluir esse relacionamento dentro do cadastro PF/PJ.

Para solucionar esse erro, é necessário excluir essa informação do lançamento, assim o sistema permite que o relacionamento também seja excluído dentro do cadastro.

PASSO A PASSO

Tentativa de exclusão do relacionamento.

Básicos> Data Warehouse> Manutenção> Cadastros> **Pessoas Físicas/Jurídicas**

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21507007743629)

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21506991222925)

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21507000491277)

Ao verificar os lançamentos nota-se que o código foi vinculado ao lançamento abaixo, por este motivo gera o erro ao tentar excluir

Básicos> Data Warehouse> Manutenção> Contabilidade> Diário Geral>Lançamentos Contábeis>Por Estabelecimento

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21506985012109)

Ao retirar o vínculo do lançamento contábil (SAFX01), é possível excluir ou alterar o relacionamento do cadastro PF/PJ.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21506976112013)

Básicos> Data Warehouse> Manutenção> Cadastros> **Pessoas Físicas/Jurídicas**
