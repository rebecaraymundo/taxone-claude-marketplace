Passo a passo para informar dados de um novo produto fabricado, que ainda não teve operaçãoes de venda, para fazer o cálculo do valor da parcela importada e conteúdo de importação:

  1. Antes de efetuar o Cálculo do Conteúdo de Importações, informe o**Valor Médio das Saídas** para o novo produto que ainda não teve operações de venda.
​​**Nota** : O cálculo será efetuado considerando o valor médio das saídas e os demais dados usados rotineiramente nos cálculos.


  2. Importe os dados no módulo Estadual > FCI - Ficha de Conteúdo de Importação, menu Obrigações > Manutenção > Movimentação de Produtos Sujeito ao FCI, ou faça a importação através da SAFX214.
​Atenção: Nessa tela, informe apenas os seguintes campos para o novo produto:

     1. **Estabelecimento
**
     2. **Data de Referência
**
     3. **Produto
**
     4. **Unid. de Medida FCI
**
     5. **Valor médio das saídas
** ​**Nota** : Os campos **Parcela Importada do Exterior** e **Percentual do Conteúdo de Importação** não devem ser informados, pois esses valores serão calculados com base nos dados informados.
