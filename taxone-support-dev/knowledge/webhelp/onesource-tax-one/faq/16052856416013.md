Normalmente, esse problema ocorre pois o código do estabelecimento não está preenchido no txt.

Para corrigir o erro 90002: 'Código do estabelecimento deve ser preenchido.', siga a orientação abaixo:

  * Preencha o campo COD_ESTAB corretamente.
