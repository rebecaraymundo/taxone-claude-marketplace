Passo a passo para incluir um novo bem ativo ou alterar os dados de um registro já existente:

  1. Acesse  [Básicos > Data Warehouse > Manutenção > Cadastro > Bem Patrimonial](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Basicos/Basicos_Data_Warehouse/Basicos_Data_Warehouse_Menu_Manutencao/Basicos_DW_Menu_Manu_Cadastros.htm>)

  2. Na tela **[Novo Bem Patrimonial](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Basicos/Basicos_Data_Warehouse/Basicos_Data_Warehouse_Menu_Manutencao/Basicos_DW_Menu_Manu_Cadastros.htm>) **inclua um novo bem ativo ou altere os dados de um registro já existente.

  3. Informe os dados solicitados nas respectivas pastas: Dados Principais, Custos e Especificações. Em caso de dúvidas sobre o preenchimento dos campos, confira o [Manual Operacional](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Basicos/Basicos_Data_Warehouse/Basicos_Data_Warehouse_Menu_Manutencao/Basicos_DW_Menu_Manu_Cadastros.htm>) da tela **Novo Bem Patrimonial.**
