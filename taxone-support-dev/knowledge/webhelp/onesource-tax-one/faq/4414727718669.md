O erro é apresentado quando se tenta recuperar um relatório diretamente na tela e o tamanho deste excede o limite estabelecido para visualização.

Para corrigir a mensagem de erro "Atenção, o limite para recuperação em tela do relatório foi atingido.", siga a orientação abaixo:

![mceclip0.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/4414735040013)

  * Quando a mensagem de erro aparecer, selecione a opção 'Não'.
  * Ao fazer isso, o relatório será processado e gerado no formato Excel.
  * Após a geração, o arquivo estará disponível para download via SFTP.
