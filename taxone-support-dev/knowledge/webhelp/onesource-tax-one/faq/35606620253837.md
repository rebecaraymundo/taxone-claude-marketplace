## Problema:

Usuário enfrenta problemas na importação da tabela SAFX48 no TaxOne, onde ocorre um erro de registro devido ao campo DAT_AVERBACAO estar com dados incorretos.

## Causa Raiz:

O problema ocorre devido ao valor do campo DAT_AVERBACAO estar configurado com zeros ('00000000'), quando deveria estar configurado com o valor '@' caso esteja sem dados.

## Resolução:

  * Verifique os registros que apresentaram erro e ajuste o valor do campo DAT_AVERBACAO para '@' caso esteja vazio.


  * Realize o teste novamente para garantir que os registros estão sendo importados corretamente.
