Resolução de Problema no Arquivo Magnético do ISS - CABREUVA

###  **Como Gerar Valores na Tag**`**< ISSDevido>**`**– Município de Cabreúva**

**Problema:**
O arquivo magnético do ISS para Cabreúva não apresenta as informações de **ISS devido** no validador, gerando erro no momento do upload no site da prefeitura.

* * *

#### **Passo a Passo para Correção**

**1\. Cadastro do Estabelecimento:**

  * Verifique se o estabelecimento está corretamente configurado no sistema.
  * Confirme os dados obrigatórios exigidos pelo município para cálculo do ISS.
