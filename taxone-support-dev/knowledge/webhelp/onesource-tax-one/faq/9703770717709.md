# [Estadual][Ressarcimento ICMS ST PR]TAX ONE - Qual a funcionalidade do Parâmetro 'Utilizar Somente DATA MART para períodos anteriores' na Geração dos Movimentos de Entradas?

Passo a passo para aplicar a funcionalidade do Parâmetro **Utilizar Somente DATA MART para períodos anteriores** na Geração dos Movimentos de Entradas - Ressarcimento ICMS-ST PR:

  1. Acesse [Estadual > Ressarcimento de ICMS-ST PR > Geração > NPF 003/20 > G](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Estadual/Ressarcimento_ICMS_ST_PR/Ressarcimento_ICMS_ST_PR_Geracao/RessarcPR_geracao_npf03320.htm>)[eração dos Movimentos de Entrada](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Estadual/Ressarcimento_ICMS_ST_PR/Ressarcimento_ICMS_ST_PR_Geracao/RessarcPR_geracao_npf03320.htm>).

  2. No campo **Utilizar Somente DATA MART para períodos anteriores** opte como será feita a recuperação dos dados de notas de períodos anteriores, nos casos em que este procedimento for necessário. Neste caso, o parâmetro é utilizado da seguinte forma:

     1. **Marcado** : A recuperação das notas de períodos anteriores é feita apenas com base no DATA MART.

     2. **Desmarcado** : A recuperação destas notas utiliza normalmente o DATA MART, no entanto, quando não existirem mais dados, e a pesquisa ainda foi finalizada (ainda não atingiu a quantidade total das saídas, e nem a data limite informada), então a pesquisa utilizará as tabelas “normais” (tabelas X07/X08).


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/25611873405965)Importante: Observe que a recuperação das notas do período é feita sempre através do DATA MART, independentemente do parâmetro. O parâmetro se aplica apenas à recuperação das notas de períodos anteriores. Quando o parâmetro for marcado, é importante que você tenha a certeza de que o DATA MART esteja equalizado para os períodos anteriores, o tempo suficiente que permita a identificação das entradas.
