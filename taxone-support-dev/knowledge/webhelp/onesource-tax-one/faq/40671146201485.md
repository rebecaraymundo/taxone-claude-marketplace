## Problema:

O usuário não consegue consumir/extrair dados nos endpoints das APIs no módulo OBI.

## Causa Raiz:

A causa raiz do problema foi a falta de ativação da opção TDM no cadastro das empresas nos ambientes especificados.

## Resolução:

  1. Acesse o cadastro das empresas no módulo OBI.
  2. Marque o flag "Tax Data Manager" (TDM) para ativar a visualização de dados.
  3. Após a ativação, os dados serão extraídos automaticamente na madrugada seguinte, à 1h da manhã.
  4. Realize a consulta nos endpoints das APIs no dia seguinte à ativação para verificar a disponibilidade dos dados.
