Para resolver o erro 93278 - 'Campo obrigatório deve ser preenchido', siga a orientação:

Nota: Ao lidar com erros durante a carga de dados em uma SAFX, é essencial seguir um processo de validação para identificar e corrigir os campos problemáticos.

1\. Quando um erro é apresentado durante a carga de uma SAFX, é importante verificar a mensagem de erro para entender quais campos estão sendo indicados como incorretos.

2\. Consulte o [manual de layout](<../Manuais_Tecnicos/Manual_Layout.htm> "Hyperlink \[../Manuais_Tecnicos/Manual_Layout.htm\]") para validar as características do campo que está causando o erro. O manual pode fornecer detalhes sobre os requisitos de preenchimento para cada campo.

Pontos Importantes:

  * Para evitar erros similares, é crucial validar sempre o preenchimento dos campos com base nas instruções do manual de layout.
  * Assegure-se de que todos os campos obrigatórios estejam preenchidos e que os dados estejam no formato correto antes de tentar uma nova carga.
