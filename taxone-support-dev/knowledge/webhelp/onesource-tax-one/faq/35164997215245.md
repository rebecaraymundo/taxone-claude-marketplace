# Resolução do Erro ao Gerar o Evento R2010 Extemporâneo no TAXONE

## Problema:

Usuário enfrenta erro ao tentar gerar o evento R2010 extemporâneo para uma nota fiscal migrada recentemente para o sistema TAXONE.

## Causa Raiz:

O erro é causado pela tentativa de geração do evento com a data da nota fiscal e a data fiscal registradas de forma incompatível com os requisitos do sistema para lançamentos extemporâneos originais.

## Resolução:

Siga os passos abaixo para resolver o problema:

  * Verifique a data de emissão do documento e a data fiscal.


  * No caso de um lançamento extemporâneo original, a data precisa ser superior ao período da emissão do documento.


  * Configure a geração para uma data adequada conforme mostra a correção indicada abaixo:
  * Ex. Emissão 01/07/2024 e Data Fiscal 05/12/2024 então a geração deverá ser da seguinte forma:

Período: 07/2024

Data Inicial de Entrada das NFs: 01/12/2024
Data Final de Entrada das NFs: 31/12/2024



  * Realize um novo teste de geração do evento e confirme se o problema foi resolvido.


## Próximos Passos:

Após ajustar as datas e gerar o evento com sucesso, monitore os próximos ciclos de geração do evento para assegurar que o problema não se repita. Caso ocorra qualquer outra discrepância, entre em contato com o suporte.

## Dicas e Recomendações:

Para evitar erros de geração futuros, sempre verifique se as datas estão configuradas de acordo com as diretrizes do sistema. Consulte o manual do usuário TAXONE ou a central de ajuda para mais informações sobre lançamentos extemporâneos no sistema.
