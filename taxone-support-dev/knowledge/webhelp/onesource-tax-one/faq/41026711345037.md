## Problema:

Durante a geração da extração da SAFX02 no ambiente do cliente a mesma não esta retornando com dados (em branco).

## Resolução:

Verificar em parâmetros de empresas e realizem algumas configurações se necessário.,

Em parâmetros de empresa, informar a empresa que esta realizando a extração com problema, clicar em buscar entrar na aba Contabilidade Geral.


![](/attachments/token/sVTdVwGRSKKWbAkW3gOVL12fu/?name=image.png)
![](/attachments/token/gSlyPjSxmAZN493VohRRTAPUf/?name=image.png)


Verificar se a conta alternativa esta marcada ou se a mesma deveria estar.

Verificar se o período esta informado, o padrão é utilizado 016

Verificar o tipo a ser colocado no NewGL:


Para saber se é New GL seguir o passo a passo:


  1. Acesse a transação SE37 e execute a função /MSAFX/CHECK_NEWGL.
  2. Essa função irá verificar se o New GL está ativo no sistema.
     1. Se o New GL estiver ativo, as origens dos dados são as tabelas FAGLFLEXA e GLT0.
     2. Se o New GL estiver desativado, as origens dos dados são as tabelas BSEG e BKPF.




Favor verificar também se as parametrizações abaixo foram realizadas no ambiente, caso necessário segue o manual segue o link: [Vol 11_Parametrizacao Interface](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Interfaces/Onesource_Brasil_Tax_Integration/WebHelp_OBTI/assets/ONESOURCE_BRASIL_TAX_INTEGRATION/Especialistas_Funcionais/OBTI_especialistasfuncionais_vol11.htm>)



### Parametrização de Contas Contábeis para extração (safx01/safx02)


Opção:

![](/attachments/token/WbZPEeF9ONJzYNySiNWRMSRcZ/?name=image.png)



  * Informe as contas contábeis passíveis de ajustes para seleção nas interfaces (safx01, safx02) integração SAP (NameSpace) com ONESOURCE ECF.
    * Os extratores para safx01/safx02 selecionam para o arquivo somente as contas contábeis com o indicador = ‘3’. As demais contas contábeis serão desprezadas da seleção.
    * Essa parametrização de contas contábeis (com o indicador = ‘3’ — contas para integrar ECF) é necessária somente para os clientes que integram diretamente as Interfaces Namespace do SAP com o ONESOURCE ECF.
    * A parametrização tem duas funcionalidades: informar contas para a integração safx147 e para as interfaces (safx01, safx02).
    * A interface de extração de dados para a safx147 não considera as contas informadas com o indicador = ‘3’ (contas para integrar ECF).
