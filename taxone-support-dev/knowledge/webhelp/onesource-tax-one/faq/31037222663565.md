# [SPED][EFD - ECD]TAX ONE - ECD - Como resolver duplicidade nos Registros J005 e J150 na Geração de ECD?

Para casos onde os Registros J005 e J150 estão sendo gerados em duplicidade para os anos de 2023 e 2024 quando apenas o ano de 2023 deveria estar sendo processado, isso pode ocorrer devido a uma configuração na geração das Demonstrações Contábeis que incluía o ano de 2024, mesmo que a intenção fosse gerar somente para 2023. Para resolver, siga as orientações:

  1. Acesse a tela de Geração Demonstrações Contábeis.
  2. Realize a geração das demonstrações para o período de **01/01/2023 a 31/12/2024** com a opção 'APENAS CONFERÊNCIA = SIM'.
  3. Em seguida, gere o meio magnético para o período de **01/01/2023 a 31/12/2023**.
  4. Realize novamente a Geração das Demonstrações Contábeis para o período de **01/01/2023 a 31/12/2023** com a opção 'APENAS CONFERÊNCIA = NÃO'.
  5. Por fim, gere o Meio Magnético para o período de **01/01/2023 a 31/12/2023**.
  6. Após a reconfiguração e geração dos dados, verifique se os registros não estão mais duplicados. Continue monitorando as gerações futuras para assegurar que o problema não se repita.


**Nota:** Assegure-se de verificar e confirmar os períodos de geração antes de executar o processo para evitar duplicações futuras. Em caso de dúvidas, consulte o suporte técnico.
