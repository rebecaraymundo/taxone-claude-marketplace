Caso enfrente dificuldades com a reordenação de parâmetros no SPED Contábil, resultando em registros com o mesmo número ou ordem incorreta após recalculação. O problema pode ser a existência de registros duplicados ou a inserção inadequada de novos registros sem a devida recalculação e validação das ordens existentes. Para resolver, siga as orientações:

  1. Acesse o módulo de Manutenção no SPED Contábil.
  2. Verifique se existem registros duplicados ou ordens que já estão sendo utilizadas.
  3. Exclua os registros duplicados ou altere as ordens dos registros conforme necessário.
  4. Reimporte os dados para a plataforma, garantindo que todos os registros estão corretos e únicos.
  5. Realize uma nova recalculação para verificar se a ordem dos registros está correta.
  6. Após realizar as correções, monitore os resultados para garantir que a parametrização esteja funcionando conforme o esperado. Se o problema persistir, considere entrar em contato com o suporte técnico para uma análise mais detalhada.


Nota: Verifique regularmente a parametrização de registros para evitar problemas futuros com duplicidade ou desordem nos registros. Sempre confira as ordens existentes antes de inserir novos registros para garantir a integridade dos dados.
