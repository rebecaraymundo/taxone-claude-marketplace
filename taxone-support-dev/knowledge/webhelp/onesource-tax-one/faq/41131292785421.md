## Problema

O cliente deseja realizar a importação automática de arquivos contendo ajustes complementares para apuração de EFD Contribuições, sem precisar selecionar manualmente os arquivos no sistema.

## Causa Raiz

O sistema TAX ONE não possui funcionalidade nativa para consumo automático de arquivos de ajustes complementares, devido à necessidade de gerar os registros da apuração antes de realizar a importação dos dados. A implementação desse recurso impactaria na estrutura do produto e poderia ocasionar problemas de performance.

## Resolução

Siga os passos abaixo para realizar a importação manual dos arquivos de ajustes complementares no sistema TAXONE:

  1. Acesse o módulo de apuração de PIS/COFINS no sistema TAXONE.
  2. Certifique-se de que os registros da apuração foram gerados para o mês corrente.
  3. Prepare o arquivo no formato **CSV** ou **TXT** , conforme especificado no manual técnico disponível no Web Help do produto.
  4. Selecione o arquivo manualmente na área de importação.
  5. Realize a importação do arquivo para que os ajustes complementares sejam incorporados à apuração.


Para mais detalhes sobre o layout dos arquivos, consulte o manual técnico acessando: _Manuais Técnicos Downloads Manuais Técnicos/ Documento Upload do EFD Contribuições_.

## Próximos Passos

Após realizar a importação, revise os dados importados para garantir que os ajustes foram incorporados corretamente na apuração. Caso necessário, execute testes ou verificações adicionais no sistema.

Se houver dúvidas ou dificuldades com o procedimento, entre em contato com o suporte técnico da Thomson Reuters.

## Dicas e Recomendações

  * Utilize sempre o manual técnico para verificar o layout correto dos arquivos a serem importados.
  * Considere enviar sugestões para melhorias no sistema, como a funcionalidade de importação automática, através do [Portal Customer Influencer](<http://customerinfluencer.thomsonreuters.com/>).
  * Mantenha o sistema atualizado para garantir acesso às últimas funcionalidades e correções.
  * Para grandes volumes de dados, organize e valide os arquivos antes da importação, reduzindo o risco de erros.
