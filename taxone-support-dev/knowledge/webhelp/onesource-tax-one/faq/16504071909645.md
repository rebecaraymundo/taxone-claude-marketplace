A **SAFX147** pode gerar informações contábeis e fiscais, porém existe uma distinção na parametrização para cada caso, sendo:

  * Para documentos contábeis é necessário parametrizar:
    * Tipos de Documento
    * Conta x Fornecedor
  * Para gerar informação fiscal, é necessário parametrizar:
    * Tipos de Notas Fiscais
    * Tipos de Impostos
    * Código da Natureza da Operação


Além desses parâmetros, em Parametrização Geral na aba Parâmetros Gerais temos as opções abaixo.

  * SAFX147 – Cálculo Pis/Cofins Alternativo: Opção para calcular o valor de PIS/COFINS utilizando alíquota parametrizada (/MSAFX/SAFTB046 - MasterSAF - Fornecedor x Conta).
  * SAFX147 – Outras Oper. Crédito – Não seleciona dados fiscais (SFX147): Inibe a seleção de dados fiscais, mantendo somente os dados contábeis quando existe cenários com os dois lançamentos.


O parâmetro **SAFX147 – Cálculo Pis/Cofins Alternativo** , quando marcado, verifica o relacionamento do Código do Fornecedor parametrizado no parâmetro Conta x Fornecedor, e só será carregado na Interface se no lançamento da BSEG o mesmo código informado no campo for KUNNR ou LIFNR.
