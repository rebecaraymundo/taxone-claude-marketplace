# [Básicos][Job Servidor] TAX ONE  -  Manutenção em Campos de Tabelas SAFX07 e SAFX08 para Notas Fiscais

## Problema:

O usuário precisa de assistência na manutenção de campos específicos nas tabelas SAFX07 e SAFX08 para cerca de 3.000 notas fiscais, especialmente o campo BASE_OUTR_IPI, que consta na tabela SAFX07, mas sofre divergências na SAFX08 onde aparece como BASE_OUTRAS_IPI.

## Causa Raiz:

A inconsistência nos campos entre as duas tabelas SAFX07 e SAFX08 levou a uma confusão sobre como atualizar o campo BASE_OUTR_IPI corretamente nas notas fiscais. É essencial que a informação seja aplicada de maneira uniforme em todos os itens de cada nota fiscal.

## Resolução:

Para corrigir a informação do campo BASE_OUTR_IPI nas tabelas SAFX07 e SAFX08, siga os passos abaixo:

  * Verifique se o indicador do campo 58 nas tabelas está preenchido com a opção '3 - outras'. Este passo é crucial porque o campo BASE_OUTR_IPI só será preenchido se a opção '3' estiver selecionada no campo indicador.


  * Se o indicador não estiver configurado corretamente, atualize-o para '3'.


  * Repita o procedimento para cada nota fiscal, garantindo que todas as informações estejam consistentes e corretas.


  * Após realizar as alterações, valide se os dados estão conforme o esperado realizando uma consulta no sistema.
