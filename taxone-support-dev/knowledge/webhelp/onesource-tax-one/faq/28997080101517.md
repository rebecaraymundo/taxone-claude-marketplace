Confira a seguir a regra para preenchimento dos campos "DATA_MOVTO" e "DATA_FATO_GERADOR" da SAFX53 na Interface SAP OBTI:

  1. Para o campo DATA_MOVTO:
     * Se o fato gerador for 1, a informação vem do campo DOCDAT das tabelas J_1BNFDOC.
     * Se o fato gerador for 2, a informação vem do campo PSTDAT das tabelas J_1BNFDOC.
     * Para outros tipos de fato gerador, segue a mesma lógica de buscar a data em campos específicos de acordo com o tipo.
  2. Para o campo DATA_FATO_GERADOR:
     * A origem segue a mesma lógica do campo DATA_MOVTO, porém, caso não haja o tipo de fato gerador e o período for Q, a data pode vir da BSAK-AUGDT ou J_1BNFDOC-DOCDAT.


Para configurar os tipos de Fato Gerador, acesse a parametrização "Tipos de Impostos - Outros" através da transação /n/MSAFX/PARAMETROS.
