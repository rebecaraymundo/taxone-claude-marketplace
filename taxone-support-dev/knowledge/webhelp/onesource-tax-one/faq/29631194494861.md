Caso enfrente problemas com a retenção de IR, onde a base do IR está diferente do Valor Bruto na geração do Registro R4020, resultando em uma base de cálculo reduzida, o problema pode estar na configuração inicial do sistema, onde a opção 'Considerar Valor Base igual ao Valor Bruto' esta desmarcada, fazendo com que o sistema considerasse o valor bruto ao invés da base de cálculo reduzida. Para resolver, siga as orientações:

  1. Acesse a tela de Dados Iniciais no sistema.
  2. Localize e desmarque a opção 'Considerar Valor Base igual ao Valor Bruto'.
  3. Salve as alterações realizadas.
  4. Regenere o Registro R4020 para que o valor correto da Base do Imposto seja considerado.
  5. Após seguir os passos acima, verifique se o valor da base de cálculo no Registro R4020 está correto. Caso persista algum problema, entre em contato com o suporte técnico para mais assistência.


**I****mportante:** Sempre verifique as configurações iniciais antes de proceder com gerações de registros fiscais e tributários para evitar discrepâncias nos valores reportados. Mantenha a documentação do sistema atualizada e consulte-a regularmente para assegurar o correto procedimento.
