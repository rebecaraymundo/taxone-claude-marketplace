Passo a passo para corrigir o erro "ORA-01756: quoted string not properly terminated" ao instalar o Interdados:

  1. O erro abaixo pode ocorrer quando é realizada uma nova instalação do Interdados, como por exemplo, instalação do pacote Versão: V11R01.001-046 sem ODI:

  2. Observe na mensagem de erro que o sistema informa o Oracle Database instalado no Servidor, onde neste caso, aponta para o**Oracle Database 23c Free, Release 12.0.0.0.0 – Developer-Release** :

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/20310887285517)

  3. Para a correção do problema, faça a correção para o Oracle Database 19c com Client 12c de 32Bits, já que o Oracle Database 23c Free, Release 12.0.0.0.0 – Developer-Release não foi homologado pela Thomson Reuters.

  4. Após a correção execute o instalador novamente.
