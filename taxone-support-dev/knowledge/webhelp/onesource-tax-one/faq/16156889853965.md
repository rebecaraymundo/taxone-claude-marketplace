Normalmente, esse problema ocorre pois o campo IND_FIS_JUR não está preenchido ou está com uma informação inválida. O campo deve ser preenchido com 1, 2, 3, 4 ou 5.

Para resolver o erro 90088: 'O indicador de pessoa Física/Jurídica não esta preenchido ou esta preenchido com informação inválida.', siga a orientação abaixo:

  * Preencha o campo com um valor válido (1, 2, 3, 4 ou 5).
