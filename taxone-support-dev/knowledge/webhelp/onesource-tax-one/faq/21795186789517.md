# [Básicos][Job Servidor] TAX ONE -  SAFX216 - Como corrigir o erro 92694?

O erro 92694 na tabela SAFX indica que o preenchimento da descrição no campo DSC_ITEM_APURACAO deve ser obrigatório. Siga os passos abaixo para corrigir o erro:

1\. Faça login no sistema que contém a tabela SAFX.
2\. Localize o campo DSC_ITEM_APURACAO dentro da tabela SAFX, onde os lançamentos são inseridos ou editados.
3\. Ao adicionar ou alterar um lançamento, assegure-se de que a descrição do item seja inserida no campo DSC_ITEM_APURACAO.
4\. Confirme se a descrição está presente e se cumpre com os requisitos do sistema. Se o campo estiver vazio, insira uma descrição adequada e clara para o lançamento.
5\. Salve as alterações e verifique se o Erro 92694 foi corrigido após o preenchimento da descrição.
