Passo a passo para corrigir o **Erro 91683** \- A data de validade da pessoa física/jurídica é anterior a data de inicio de vigência do código de atividade econômica informada:

  1. O problema ocorre pois o código do campo** COD_ATIVIDADE** possui uma data maior que o campo DATA_X04.

  2. Verifique a data de validade do **COD_ATIVIDADE **em [Básicos > Parâmetros > Preliminares > Atividade Econômica](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Basicos/Basicos_Parametros/Parametros_Menu_Preliminares/Param_Preli_Atividades_Economicas.htm>). Exemplo para corrigir o erro: Início de Validade **01/01/2021.**

  3. Altere a **DATA_X04 **para igual ou maior que a data de validade, ou então procure um **COD_ATIVIDADE** válido para esse **Cliente/Fornecedor:**
