Caso o cliente decida utilizar o envio das SAFX para o TaxOne por meio do TR File, há vários parâmetros disponíveis na namespace que podem ser configurados para otimizar tanto a geração e envio quanto o monitoramento, caso ocorram erros ou problemas no envio das cargas.

Passo a passo para configuração da Name Space para utilizar envio através do TR File:

  1. Acesse os **Parâmetros Gerais** , aba Configuração.
  2. Selecione a opção "**TR Files - Path Unix** " e informe o caminho do servidor Unix onde foi configurada a pasta para realizar a carga das SAFXs.
  3. No campo "**Timer - Importação Aut.:"** , informe o tempo de intervalo, em segundos, entre a geração das SAFXs e a importação das cargas. O tempo limite que pode ser utilizado é de 600 segundos.
  4. Após configurar esses dois parâmetros, o sistema estará apto a realizar o envio das extrações utilizando o TR File.
