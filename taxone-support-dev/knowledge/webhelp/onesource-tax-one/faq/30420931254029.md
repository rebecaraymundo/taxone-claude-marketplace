Esclarecimentos sobre a divergência na versão patch informada na interface:

  1. Após realizar a atualização dos ambientes, é possível verificar se a versão foi aplicada corretamente e se o ambiente foi atualizado.

  2. Durante esta verificação, pode ser encontrada uma divergência entre as informações nas telas "Parâmetros Gerais" - aba "Configuração" e a tela de "Mapeamento de Campos".


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/30420945999117)

  1. Esta divergência ocorre porque, na tela de "Parâmetros Gerais", a informação do Release - Patch refere-se à versão que foi instalada inicialmente no sistema, não sendo modificada quando são realizadas atualizações.
  2. Na tela de "Mapeamento de Campos", a informação referente ao Release - Patch refere-se à versão que o ambiente está utilizando no momento, sendo atualizada cada vez que uma atualização é realizada com sucesso.
