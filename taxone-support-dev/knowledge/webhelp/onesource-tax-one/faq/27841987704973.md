O objetivo deste artigo é orientar a correta extração do ICMS nas SAFXs de documentário fiscal.

Siga os passos abaixo para realizar a parametrização de ICMS:

  * Acesse o menu de parametrização de "Tipo de Impostos" no Interdados.
  * Verifique se a parametrização para o imposto ICMS está correta. Caso não esteja, ajuste para o tipo de imposto correto.
  * Realize novos testes para garantir que os valores de ICMS estão sendo exibidos corretamente.


Após ajustar a parametrização, realize testes adicionais para garantir que todas as configurações estão corretas e que os valores de ICMS estão sendo refletidos de forma precisa. Monitorize o sistema por alguns dias para confirmar a estabilidade da solução.
