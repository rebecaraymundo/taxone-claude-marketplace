Para corrigir o erro **90091** \- O Indicador de Conteúdo de Código de CGC/CPF deve ser preenchido de acordo com <1>, <2>, <3> ou <4>, siga os passos abaixo:

  1. No arquivo TXT, verifique se o campo **IND_CONTEM_COD** está em branco ou com "@".

  2. Informe o **Indicador do Conteúdo do Código** , de acordo com:

     1. 1 - Se no Campo de Código (02) o conteúdo for um Código para Pessoa Física/Jurídica que tenha CGC/CPF.

     2. 2 - Se no Campo de Código (02) o conteúdo é um CPF.

     3. 3 - Se no Campo de Código (02) o conteúdo é um CGC.

     4. 4 - Se a Pessoa Física/Jurídica não tenha CPF/CGC - casos especiais.
