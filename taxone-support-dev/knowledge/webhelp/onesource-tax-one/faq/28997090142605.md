Para consultar os usuários ativos cadastrados no módulo Administration do Tax One, siga as orientações:

  1. Acesse o módulo **Administration** no Tax One.
  2. Navegue até **Reporting**.
  3. Selecione **User Details** para visualizar o relatório de usuários ativos.
  4. O relatório exibirá os detalhes dos usuários ativos, como nome e status.
  5. Após verificar os usuários ativos, solicite as alterações necessárias ao administrador do sistema para manter a gestão de acessos atualizada. Continue monitorando regularmente as mudanças de equipe para atualizações futuras.


**Nota:** Considere estabelecer um processo regular de revisão de usuários ativos para garantir que o acesso ao sistema esteja sempre alinhado com a estrutura atual da equipe. Isso ajudará a manter a segurança e eficiência operacional.
