# [Básicos][Job Servidor] TAX ONE - SAFX2102 - Como corrigir o erro 91939 - Código de Aglutinação contido na Expressão de Ordem deve pertencer ao cadastro dos Códigos de Aglutinação do Balanço'?

Ao importar a SAFX2102, este erro pode ocorrer se não houver o código no cadastro, ou por causa da ordem dos dados na SAFX.

  1. Na SAFX, verifique se o código sinalizado no erro consta no cadastro.
  2. Verifique se este dado está na posição anterior no campo COD_CONTA_AGLUT. Se ele estiver no campo EXPRESSAO_ORD, o erro informado ocorre.
  3. Veja exemplo na imagem a seguir:
![](https://suporte.thomsonreuters.com.br/hc/article_attachments/19649400607629)
  4. O correto é estar conforme a imagem a seguir, para não ocorrer o erro do campo expressão de ordem:
![](https://suporte.thomsonreuters.com.br/hc/article_attachments/19649429657869)
Nota: O campo expressão de ordem não pode conter espaço entre os dados apresentados entre ponto-e-vírgula.
