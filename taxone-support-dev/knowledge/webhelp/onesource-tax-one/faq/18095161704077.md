Nota: Este erro pode ocorrer no momento em que um cliente novo, que esteja utilizando o Ambiente Interdados Oracle Cloud (V17), tenta fazer o processo de [Cadastro Tax One](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Interfaces/Interface_Oracle_V17R01_Cloud/Webhelp_Interface_Oracle_V17R01_Cloud/Manuais_portugues/Manuais_funcionais/Manual_de_usuario/V17R01_usuario_cadastrotaxone.htm>) ao clicar em Importar Definições de Tabelas.

Para solucionar este problema, siga os passos abaixo:

  1. Acesse o ambiente no qual apresentou problema, no menu **Opções > Parâmetros > Parâmetros de Configuração**.
  2. Ir até **API Importação de Dados** e verifique se a API URL está igual a:
**https://uat01.taxone.thomsonreuters.com/taxone-integrator**
     1. Se estiver diferente, substitua pela URL citada acima.
  3. Em **API Envio de Dados** , verifique se a API URL está igual
**https://uat01.taxone.thomsonreuters.com/safx**
     1. Se estiver diferente, substitua pela URL citada acima.
  4. Clique em Salvar e realize um novo teste.
