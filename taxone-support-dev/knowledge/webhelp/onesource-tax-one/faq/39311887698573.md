## Problema

Usuários estão enfrentando discrepâncias nos valores apresentados no relatório DCONT_SNT do SPED Contribuições, principalmente quando comparado ao valor contido nos documentos fiscais.

## Causa Raiz

A discrepância nas informações ocorre devido à falta de uma verificação detalhada entre o relatório sintético e o documento fiscal. O relatório sintético (DCONT_SNT) consolida os lançamentos, mas pode apresentar divergências em relação aos valores de base e impostos expostos nos documentos individuais.

## Resolução

Para corrigir o problema e verificar corretamente os valores apresentados: 1. Realize a limpeza e equalização inicial dos registros no sistema. 2. Após a equalização, gere os registros marcando as opções de relatórios desejados. 3. Para verificar os valores detalhados, emita o relatório analítico (DCONT_ANL) do SPED Contribuições. Este relatório fornece uma análise detalhada e individual dos valores de base e impostos, permitindo comparar com os documentos fiscais. 4. Compare as informações das notas fiscais com os valores apresentados no relatório analítico. Caso o problema persista: - Identifique uma nova nota fiscal ou lançamento que apresente discrepância entre os valores gerados. - Utilize sempre o relatório analítico (DCONT_ANL) para validar os dados, pois o relatório sintético apenas consolida os lançamentos.

## Próximos Passos

1\. Teste o procedimento indicado acima com outras notas fiscais que apresentam discrepância. 2. Verifique se os valores gerados estão consistentes e condizem com os documentos fiscais originais. 3. Caso os erros ainda persistam, entre em contato novamente com o suporte técnico, fornecendo evidências adicionais (como relatórios e documentos fiscais) para uma análise mais detalhada.

## Dicas e Recomendações

\- Sempre utilize o relatório analítico (DCONT_ANL) para validar os valores gerados, especialmente em casos de dúvidas ou inconsistências. - Realize procedimentos regulares de equalização e limpeza nos registros, garantindo que as informações estejam organizadas e consistentes no sistema. - Mantenha as evidências de discrepâncias documentadas e prontamente disponíveis para análise, como arquivos PDF ou imagens das notas fiscais e relatórios. - Consulte a documentação do sistema SPED Contribuições para entender melhor a funcionalidade dos relatórios sintético e analítico. Para mais informações, entre em contato com nosso suporte técnico ou consulte outros artigos na nossa base de conhecimento. ```
