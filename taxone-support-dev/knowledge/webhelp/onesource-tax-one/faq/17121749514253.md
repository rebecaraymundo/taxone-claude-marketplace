Para corrigir o erro **90120: 'O campo tipo de movimento de Entrada/Saída não está preenchido ou com informação inválida'** , siga os passos abaixo:

Nota: O erro ocorre quando o campo **Tipo de Movimento **está preenchido com informação inválida ou nula.

Para solucionar, preencha o campo deve ser preenchido com **1,2,3,4,5** ou **9.** Por ser campo chave, não pode estar nulo.
