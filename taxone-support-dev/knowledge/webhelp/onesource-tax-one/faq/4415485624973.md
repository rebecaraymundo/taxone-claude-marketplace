Passo a passo para corrigir o **Erro 92622** \- O Campo de Data de Prestação de Serviço com formato inválido:

  1. Verifique se o campo **DATA_PREST_SERV** está com um dos 3 valores aceitáveis:

     1. 00000000

     2. @

     3. Data no formato YYYYMMDD

  2. Caso não esteja, ajuste o arquivo para um dos três valores aceitáveis e o problema será solucionado.
