## Causa Raiz:

Após análise, identificou-se que os parâmetros nos dados iniciais estavam configurados apenas para buscar a capa da nota fiscal, limitando assim a geração de informações detalhadas necessárias para o Bloco D.

## Resolução:

  1. Acesse o módulo SPED Contribuições no sistema.
  2. Navegue até os parâmetros dos dados iniciais.
  3. Adjuste os parâmetros para incluir a busca por item na nota fiscal, além da capa.
  4. Salve as configurações alteradas.
  5. Tente gerar o Bloco D novamente e verifique se o problema foi resolvido.


## Próximos Passos:

Após realizar as mudanças sugeridas, é importante gerar o arquivo novamente e verificar se o Bloco D está sendo incluído corretamente. Monitore o sistema nas próximas gerações para garantir que não haja recorrências do problema.

## Dicas e Recomendações:

Recomenda-se revisar regularmente os parâmetros configurados no módulo SPED Contribuições para assegurar que todos os blocos necessários estão sendo gerados corretamente. Para qualquer dúvida ou necessidade de assistência adicional, entre em contato com o suporte técnico.
