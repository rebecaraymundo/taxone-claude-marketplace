Passo a passo para solucionar registros duplicados na SAFX113:

  1. Acesse a transação **/n/MSAFX/ PARAMETROS** e vá em **Tipos de Impostos**.
  2. Localize o tipo de imposto do ICMS configurado no documento fiscal.
  3. Para esse tipo de imposto ICMS, marque a opção "**Int. Dif. Aliq.**" (Diferencial de Alíquota).
  4. Salve as alterações realizadas.
  5. Execute novamente a SAFX113.
  6. Verifique se a duplicidade de linhas foi solucionada.


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/30775633007629)

A duplicidade de linhas na SAFX113 pode ocorrer devido ao uso de diferentes alíquotas (também usadas para PIS e COFINS) no mesmo campo, resultando em entradas duplicadas para o mesmo valor de base do ICMS. Marcar a opção "Int. Dif. Aliq." para o tipo de imposto ICMS permite que o sistema trate corretamente os casos com diferentes alíquotas, evitando a duplicação de registros na SAFX113 e possibilitando a validação das informações para o Registro C197 no EFD-ICMS/IPI SP.
