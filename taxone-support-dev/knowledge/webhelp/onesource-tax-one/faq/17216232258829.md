A mensagem de erro 'ORA- 06502:PL/SQL : numeric or value error : character string buffer too small', é apresentada devido ao campo ‘Núm. Recibo da Última Declaração’ conter a quantidade acima de 41 caracteres. Conforme o leiaute do Guia Prático do EFD Contribuições este campo deve contar até 41 caracteres.
