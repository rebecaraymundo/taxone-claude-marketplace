# Erro de estrutura - Gissonline de Paulinia SP, ausência da tag "CodigoTributacaoMunicipio" nos registros das notas fiscais.

## Causa Raiz:

A raiz do problema identificada é a falta de cadastro de códigos de tributação para o município no sistema GISS Online. Este cadastro é essencial para que a tag "CodigoTributacaoMunicipio" seja incluída corretamente nos registros das notas fiscais.

## Resolução:

Acesse o módulo GISS Online, na opção "Parâmetros por Município", e, no menu "Parâmetros > Código de Tributação por Município", configure os parâmetros para o município de Paulínia. Preencha o código do serviço utilizado no documento fiscal e o código de tributação conforme as exigências estabelecidas pelo município.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40379212221325)

Caso não haja, é necessário cadastrar os códigos de tributação utilizados. Insira o código seguindo o formato exigido pelo município, utilizando 9 dígitos.

Após o cadastro, gere novamente o arquivo e verifique se a tag "CodigoTributacaoMunicipio" foi incluída corretamente.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40379212222861).
