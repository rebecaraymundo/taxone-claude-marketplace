Esse problema na parametrização ocorre quando um usuário enfrenta dificuldades ao tentar importar cópias de parametrização de UAT para PRD no sistema Tax One, sendo que os arquivos não são encontrados mesmo após serem transferidos via FTP.

Normalmente, esse problema ocorre pelo uso de um usuário incorreto para importação dos arquivos no sistema Tax One. O usuário específico necessário para essa tarefa não estava sendo utilizado, resultando na falha de reconhecimento dos arquivos transferidos. Para resolver, siga as orientações:

1\. Acesse o FTP utilizando o usuário **INTG_TAXONE**.

2\. Realize a transferência dos arquivos necessários para a importação.

3\. No sistema Tax One, prossiga com o processo de importação dos arquivos agora reconhecidos.

4\. Confirme se a importação foi concluída com sucesso.
