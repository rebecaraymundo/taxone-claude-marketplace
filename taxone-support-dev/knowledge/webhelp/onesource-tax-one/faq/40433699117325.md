#

## Problema

Alguns registros não estavam sendo extraídos corretamente na geração de dados pela interface SAFX04, mesmo havendo clientes com dados disponíveis nos arquivos gerados pelo sistema.

##

## Causa Raiz

A causa raiz do problema foi identificada como resultado de uma parametrização específica que não estava adequada para capturar os registros de determinados clientes nas bases de dados relacionadas.

##

## Resolução

Para solucionar o problema, siga os passos abaixo:

  1. Acesse a transação /n/MSAFX/PARAMETROS e navegue até a aba "Parâmetros Gerais".
  2. Marque o parâmetro "SAFX04 - Modificações Adicionais".
  3. Realize a extração novamente para validar se o problema foi solucionado.
  4. Confirme que todos os dados de clientes estão sendo extraídos corretamente.


##

### Próximos Passos

Após implementar as mudanças e realizar os testes:

  * Confirme que todos os registros necessários estão presentes na extração.
  * Monitore as próximas extrações para garantir que o problema não reapareça.
  * Documente qualquer procedimento adicional identificado para futuras análises.


##

### Dicas e Recomendações

  * Mantenha a documentação atualizada sobre os parâmetros essenciais para a extração correta na interface SAFX04.
  * Reveja periodicamente as parametrizações e configurações aplicadas às interfaces.
  * Realize testes sempre que alterações forem feitas nas configurações de interface ou tabelas relacionadas.
  * Evidencie as boas práticas de análise utilizando filtros específicos para identificar discrepâncias nas extrações de dados.
