Quando o valor do diferencial de alíquota não é exibido no relatório, apesar de estar corretamente preenchido, isso ocorre devido à forma como o cálculo é realizado. O campo "Dif de Alíquota" calcula a diferença entre os valores "ICMS Destacado" e "ICMS Devido". Se ambos os valores são iguais, o campo "Dif de Alíquota" será exibido como zero. Para resolver, siga as orientações:

  1. Verifique os valores inseridos nos campos "ICMS Destacado" e "ICMS Devido".
  2. Confirme que os valores inseridos refletem corretamente as transações para as quais o relatório está sendo gerado.
  3. Se necessário, ajuste os valores para que o cálculo do diferencial de alíquota seja exibido corretamente.
  4. Para mais detalhes sobre como preencher esses campos, consulte o web help disponível em: [Link para o Web Help](<https://suporte.thomsonreuters.com.br/hc/pt-br/articles/21815373927949--B%C3%A1sicos-Report-Fiscal-O-campo-Valor-Diferen%C3%A7a-de-Al%C3%ADquota-n%C3%A3o-%C3%A9-exibido-no-relat%C3%B3rio-como-resolver>).
