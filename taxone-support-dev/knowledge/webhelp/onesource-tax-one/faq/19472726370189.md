Normalmente, esse problema ocorre pois o campo HISTPADRAO está preenchido, porém, não existe relacionamento de Tabela x Grupo para o estabelecimento informado na SAFX.

Para corrigir o erro 90477: 'Grupo de Centralizacao de Codigo de Historico Padrao nao esta preenchido ou invalido na data ou nao esta cadastrado ou Data nao preenchida.', siga a orientação abaixo:

  * Realize o relacionamento entre Tabelas x Grupos para a tabela Histórico Padrão no menu Parâmetros > Funcionais > Relac. entre Tabelas e Grupos.
