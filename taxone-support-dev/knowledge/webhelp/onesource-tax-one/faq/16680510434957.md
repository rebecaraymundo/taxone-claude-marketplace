Ao realizar a atualização da namespace de versões anteriores a 4008 pode ocorrer a falha Include Z_EHP_ZMSAFZ147, conforme exemplo abaixo:

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/16679984481933)

  1. Quando ocorrer esta falha, solicite análise pelo Abap devido a mudança na regra utilizada por este include. Após a análise e correção, realize novamente a importação das requests de atualização.


  1. No arquivo Report Output podem ser verificados mais detalhes, conforme descrição abaixo:


"Program /MSAFX/SAPLSAFE_PROCESSOS, Include Z_EHP_ZMSAFZ147===============E: Syntax error in line 000418

Different number of parameters in FORM and PERFORM(routine: F_TIPO_MATNR, number of formal paramete "


Abaixo um exemplo de como seria o esperado:


**Customizado** :

PERFORM f_tipo_matnr

USING ls_bseg-matnr

CHANGING gs_zsaft147-ind_pro.


**Exemplo Produto:**

PERFORM f_tipo_matnr

USING ls_1bnflin-matnr

ls_1bnflin-werks

CHANGING gs_zsaft147-ind_pro.

  1. Para realizar a modificação, confirme se o ambiente está parametrizado como Modificável. Esta verificação pode ser realizada através da SE03.


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/16679989787789)

  1. Passo a passo de como realizar a modificação:


  * Entre na SE38, informe o processo: /MSAFX/ZMSAFZ147.


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/16680005859981)

  * Selecione o botão ou apertre SHIT+F4 :


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/16680022359693)

  * Clique com o botão direito sobre a include e selecione **Operações de ampliação** , em seguida clique em **Modificar Implementação:**


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/16680022390669)
