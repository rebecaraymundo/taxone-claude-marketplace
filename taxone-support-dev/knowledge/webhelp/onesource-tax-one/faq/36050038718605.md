Configurando o Campo CFPS de 3 Posições para a Geração Correta de Arquivos Magnéticos

**Problema:**

Falta de dados no arquivo magnético CFPS, mesmo com a Nota Fiscal preenchida corretamente.

**Causa Raiz:**

O campo CFPS, que determina códigos fiscais de serviços, deve conter apenas três posições numéricas. No caso em questão, o campo estava preenchido com quatro posições, o que causou a falha na geração do arquivo.

**Resolução:**

Acesse o sistema na aba BÁSICOS.

Navegue até FERRAMENTAS e selecione Tabelas Internas.

Escolha Manter, depois Tabelas e finalmente Cadastro de Códigos Fiscais de Serviços.

Edite o campo CFPS para garantir que ele contenha apenas 3 posições numéricas.

Salve as alterações e regenere o arquivo magnético para verificar se o problema foi resolvido.

**Próximos Passos:**

Após ajustar o campo CFPS e regerar o arquivo, monitore se a geração ocorre sem erros. Se o problema persistir, entre em contato com o suporte técnico para uma análise mais aprofundada.
