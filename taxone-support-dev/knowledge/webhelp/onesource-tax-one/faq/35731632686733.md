O objetivo deste artigo é orientar a consultoria\cliente, sobre o processo de cadastro de empresas onde não sendo realizado de forma correta, pode acarretar em erros nos processos do interdados.




**Problema:**

Ao tentar extrair informações dos documentos fiscais pelas SAFX para todas as filiais da empresa, apenas os dados da matriz estavam sendo extraídos. Além disso, a mensagem de erro à seguir é apresentada ao executar o processamento.
Mensagem de erro: _EMPRESA XXXXXXXXX NAO CADASTRADA MSAFI_MSFSY_KDP1000_ORG_INFO OU CADASTRO DE EMPRESA SEM IDENTIFICACAO DE ORGANIZACAO DO EBS. ORG_ID=XXXXXXXXXX ORGANIZATION_ID= XXXXXXXXXXX FAVOR VERIFICAR._

**Causa Raiz:**

O problema foi causado pela presença de ORGANIZATION_ID duplicados cadastrados em diferentes estabelecimentos. Isso impedia a extração correta dos dados das filiais e a apresentação de erros durante o processo.

**Resolução:**

  * Inicie sessão na plataforma de cadastro de empresas.
  * Acesse o cadastro de Organizações de Inventário.
  * Remova as organizações de inventário duplicadas, deixando apenas uma para cada estabelecimento conforme o CNPJ correspondente. A aplicação não fazia essa checagem anteriormente por não utilizar o parâmetro. Com a criação da nova funcionalidade de envio FCI para o Oracle Cloud, o campo passou a ser validado com a carga 1000 e cadastro Taxone.
  * Salve e atualize as alterações no sistema.
  * Realize a tentativa de extração novamente para verificar se o erro persiste.
  * Se acaso o problema persistir, abra um chamado para o Suporte técnico da Thomson Reuters.


**Próximos Passos:**

Após realizar a correção, o usuário deve monitorar as próximas extrações para garantir que todos os dados estejam sendo coletados adequadamente pelas filiais. Caso o erro persista ou novos problemas ocorram, entre em contato com o suporte técnico para nova análise.

**Dicas e Recomendações:**

Mantenha os registros de cadastro das empresas sempre atualizados e verifique regularmente por duplicidades ou outras inconsistências que possam afetar os processos internos. Utilize a documentação e links de suporte disponíveis para garantir que as configurações sejam realizadas corretamente.
