A configuração padrão do TR-Transfer Files não separa automaticamente os logs de homologação e produção em pastas distintas quando configuradas no mesmo servidor.

Para evitar que os logs de ambos os ambientes sejam armazenados juntos, siga o passo a passo para configurar as pastas:

  1. Acesse o painel de controle do TR-Transfer Files.
  2. Nas configurações de conexão, defina pastas distintas para os ambientes de homologação e produção.
  3. Para garantir a separação adequada dos logs, recomenda-se utilizar usuários distintos para cada ambiente.
  4. Verifique os logs de tracking para confirmar que os registros estão sendo armazenados nas pastas corretas conforme os usuários configurados.


Próximos Passos: Após implementar a solução, monitore os logs de ambos os ambientes para garantir que estão sendo separados corretamente. Caso continue enfrentando problemas, entre em contato com o suporte técnico para uma revisão mais detalhada das configurações.

Dicas e Recomendações:

Considere revisar regularmente as configurações de usuários e pastas no TR-Transfer Files para evitar sobreposições ou erros de armazenamento de logs.
Mantenha a documentação atualizada conforme novas configurações ou alterações são realizadas.
