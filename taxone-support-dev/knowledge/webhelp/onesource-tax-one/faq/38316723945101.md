# [Estadual][GNRE]  O TAX One realiza a Emissão da GNRE com Código de Barras ?

O sistema Taxone não gera GNRE (Guia Nacional de Recolhimento de Tributos Estaduais) com código de barras.

Durante o desenvolvimento dessa funcionalidade, a inclusão do código de barras não foi possível devido a questões técnicas e legais.


Na época, as regras para impressão do código não estavam disponíveis.

O processo operacional para gerar GPS (Guia da Previdência Social), DARF (Documento de Arrecadação de Receitas Federais) e GNRE a partir do Taxone é apenas para conferência.
Para gerar esses documentos com o código de barras, é necessário utilizar o software fornecido pela Receita Federal.
