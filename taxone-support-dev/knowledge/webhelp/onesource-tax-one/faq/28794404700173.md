# [Básicos][Report Fiscal] TAX ONE - Como resolver a inconsistência na coluna "UF" do Relatório de Chaves de Documentos Fiscais Eletrônicos?

Quando a coluna "UF" no Relatório de Chaves dos Documentos Fiscais Eletrônicos apresentar "ERRO" apesar das informações serem compatíveis, normalmente, a causa do erro está no campo "Município" do cadastro do fornecedor, que deve ser preenchido corretamente para validar a UF no relatório. Para isso:

  1. Acesse o cadastro do fornecedor no sistema.
  2. Verifique se o campo "Município" está preenchido corretamente.
  3. Confirme se o estado do campo "Município" corresponde à UF da chave de acesso.
  4. Caso as informações estejam corretas, salve as alterações e gere o relatório novamente.
  5. Se persistir o erro, entre em contato com o suporte técnico.


**Importante:** É fundamental manter os cadastros de fornecedores sempre atualizados e completos para evitar problemas nos relatórios fiscais. Regularmente revise e atualize essas informações para garantir a precisão dos dados reportados.
