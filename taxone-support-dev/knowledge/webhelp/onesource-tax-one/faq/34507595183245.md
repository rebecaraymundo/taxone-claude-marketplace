# Como Resolver o Erro de Geração de EFD Contribuições Quando o Período Está Reaberto

Problema:

O usuário tenta gerar os registros para a EFD Contribuições de um mês futuro, mas encontra um erro indicando que um período anterior está reaberto e não pode ser fechado.

## Causa Raiz:

O status do período anterior está como 'reaberto', o que impede o fechamento deste período. Para alterar o status para 'realizado', é necessário gerar os registros deste período específico.

## Resolução:

  * Acesse o módulo **Obrigações** no sistema.


  * Selecione a opção **Geração dos Registros**.


  * Escolha o mês que deseja gerar, neste caso de exemplo o período 10/2023.


  * Clique em **Gerar** para iniciar o processo de Geração dos Registros.


  * Verifique se há erros no log de geração. Se estiver tudo correto, o status deve mudar para 'realizada'.


  * Com o status alterado, você pode agora fechar o período e prosseguir com a geração dos registros para o período 01/2025, conforme exemplo.


## Próximos Passos:

Após realizar os passos acima, monitore o sistema para garantir que o status dos períodos esteja correto e que não haja outros erros. Caso encontre dificuldades, entre em contato com o suporte.

## Dicas e Recomendações:

É recomendável verificar regularmente os status das obrigações e garantir que todos os períodos estejam devidamente fechados para evitar problemas na geração de novos registros. Consulte regularmente a central de ajuda para buscar orientações.
