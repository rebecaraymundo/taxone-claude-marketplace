A ausência de notas no relatório pode ser causada pela falta de equalização do período ou pela parametrização, ou ainda se a pessoa física/jurídica (PF/PJ) for do mesmo estado do estabelecimento para o qual o relatório está sendo gerado. Para resolver, siga as orientações:

  1. Verifique se as parametrizações estão como conforme:
     * O campo Estabelecimento deve estar preenchido.
     * O campo Base de Cálculo p/ Dif. Alíquota deve estar selecionado como '4 - Valor Informado (Sem Base de Cálculo)'.
  2. Após isso, proceda com a equalização do período e gere o relatório. Se a nota ainda não aparecer, verifique se a PF/PJ está cadastrada no mesmo estado do estabelecimento que está gerando o relatório.
