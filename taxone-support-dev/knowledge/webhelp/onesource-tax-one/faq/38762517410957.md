O objetivo deste artigo é orientar o usuário sobre como contornar possíveis problemas na integração de notas fiscais e seus respectivos itens para as SAFX07, 08 e 09.

**Problema:**

Após realizar uma programação conjunta para as SAFXs 07,08 e 09 o usuário encontrou problemas na integração do item das notas fiscais, SAFX08 e SAFX09:

![](https://suporte.thomsonreuters.com.br/attachments/token/Q5uzogGuDt55A9tqEVO1AZFT5/?name=image001.png)

**Causa Raiz:**

O motivo da inconsistência se dá pelo descompasso de processamento dos lotes enviados ao TaxOne, para cada SAFX um lote distinto é gerado, e o tempo de processamento e envio do lote para as SAFXs também são distintos, ocasionando problemas na integração dessas notas.

**Resolução:**

  * Realizar processamentos individuais para as SAFXs mencionadas, iniciando pela capa das NFs, SAFX07**;**


**Próximos Passos:**

Após realizar a alteração, recomendamos:

  * Realize processamentos individuais para as SAFX;
  * Verifique o resultado do processamento no TaxOne;


**Dicas e Recomendações:**

Para evitar problemas similares no futuro, siga essas melhores práticas:

  * Mantenha um registro atualizado da configuração do ambiente, incluindo alterações feitas nos parâmetros.
  * Estabeleça um processo de revisão para garantir que as configurações estejam corretas antes de realizar homologações ou testes.
  * Considere testar as alterações em um ambiente isolado para limitar potenciais impactos.
