## Introdução

Como gerar o registro J801 para fins de substituição com arquivo RTF

## Passo-a-passo

Módulo ECD

Parâmetros > Perfi

![1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/16173980939149)

É necessário ter um arquivo com a extensão em .RTF (Anexo)

Módulo ECD

Parâmetros > Dados Iniciais

![2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/16174014275213)

![3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/16173989628941)

Deverá ser preenchido todos os campos conforme a geração, e após deverá ser salvo os dados e os Dados Iniciais:

![4.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/16174028542861)

![5.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/16174006547725)

Módulo ECD

Geração > Meio Magnético

![6.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/16174014694925)

![7.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/16173981539213)

Algumas considerações:

\- Não deixar o arquivo RTF em aberto na máquina quando for realizado o vínculo nos Dados Iniciais.

\- Não retirar o arquivo RTF da pasta escolhida, pois poderá acarretar problemas na geração deste registro.
