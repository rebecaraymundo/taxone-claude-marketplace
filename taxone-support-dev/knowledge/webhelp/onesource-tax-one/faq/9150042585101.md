Pré-requisito: É necessário realizar os passos abaixo para novos usuários do produto ONESOURCE Tax One. Em caso de atualização em patch, este mesmo procedimento deve ser executado.

Para efetuar a Importação de Tabelas Acessórias, siga os passos abaixo:

  1. Realize o download das tabelas acessórias através do [link](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/inicio/downloads.htm>) disponibilizado no help do produto.

  2. Realize o Upload das tabelas desejadas via SFTP. Exemplo:

     1. Utilize a ferramenta Filezilla para importar o arquivo desejado.

     2. Não utilize o usuário integrador.

     3. O usuário do SFTP deve ser referente a conta do produto ONESOURCE Tax One para assim conseguir visualizar o arquivo importado.

  3. Para verificar a tabela importada, acesse o produto em Básicos > Ferramentas > Tabelas Internas > Importar > Tabelas Acessórias.
Nota: O estabelecimento desejado deve estar sempre selecionado, e não marcado como todos.

  4. Em **Selecionar Arquivos para Importação** , selecione o arquivo desejado, e após SELECT.

  5. Em **Importação de Tabelas** , informe se deseja recuperar apenas o arquivo selecionado ou todos. E, após preencha os parâmetros para importação.

  6. Depois clique em Importar. Aguarde o fim da importação e confira os dados da importação.
