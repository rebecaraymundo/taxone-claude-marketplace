Caso seja apresentado um erro no sistema ao tentar importar o arquivo SAFX148, que é necessário para a geração do F120 no contexto da EFD Contribuições. Esse erro pode ser devido ao preenchimento incorreto de campos específicos no arquivo SAFX148. Esse erro pode ser identificado no campo 'Mes/Ano de Aquisição dos Bens Incorp. ao Ativo Imobilizado' que não deve ser preenchido quando o 'Indicador de Operações Geradoras de Créditos' é diferente de 3. E, também, se o campo 'DATA_BAIXA' estiver preenchido com '000000'.

Para resolver, siga as orientações:

  1. Abra o arquivo SAFX148 em um editor de texto ou planilha.
  2. Verifique o campo 'DATA_BAIXA' e ajuste para '@' caso esteja preenchido com '000000'.
  3. Salve as alterações no arquivo.
  4. Tente importar o arquivo SAFX148 novamente no sistema.
