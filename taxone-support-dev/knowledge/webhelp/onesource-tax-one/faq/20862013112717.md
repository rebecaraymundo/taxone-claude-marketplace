# [Básicos][Job Servidor] TAX ONE - SAFX01 - Como corrigir o erro 90382?

Normalmente, o problema ocorre pois o código informado no campo CENTRO_DESPESA não está cadastro na tela Data Warehouse > Manutenção > Códigos > Centro de Despesa. Para corrigir o erro 90382: 'O Campo Código de Centro de Despesa não esta cadastrado', siga as orientações abaixo:

  1. Informe um código valido no arquivo TXT, ou
  2. Cadastre o código informado no arquivo TXT na tela Data Warehouse > Manutenção > Códigos > Centro de Despesa.
