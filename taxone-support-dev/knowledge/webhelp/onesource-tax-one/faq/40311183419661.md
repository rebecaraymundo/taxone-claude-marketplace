# [Básicos][OBI - OneSource Brasil Integration] Integração do Cancelamento de Notas Fiscais Eletrônicas via XML: Utilizando a SAFX337

**Problema:** O usuário tem dúvidas sobre como realizar a integração do cancelamento de Notas Fiscais Eletrônicas (NF-e) via XML no OBI, uma funcionalidade que anteriormente não era suportada.

**Causa Raiz:** Até o momento, não havia um procedimento padrão ou uma SAFX específica para a integração do cancelamento de NF-e baseada na chave da nota. Essa funcionalidade se tornou possível com a introdução da nova SAFX337.

**Resolução:** A integração do cancelamento agora é possível graças à introdução da **SAFX337** na **release 311.0.0 (24/09)** , que permite o procedimento baseado na chave da NF-e.

  1. **Desenvolvimento do Processamento:** Atualmente, ainda não existe um processamento padrão disponível para download. Portanto, é necessário **construir o processamento manualmente** no OBI.
  2. **Consulta ao Manual:** Utilize o **Manual de Layout SAFX** do Tax One para obter as informações sobre os campos obrigatórios da **SAFX337** (principalmente a chave da NF-e).
  3. **Construção Manual:** Crie o perfil de entrada, o perfil de saída e o processamento no OBI, mapeando os campos necessários para gerar a SAFX337.
  4. **Integração:** Integre o XML da nota com o OBI.
