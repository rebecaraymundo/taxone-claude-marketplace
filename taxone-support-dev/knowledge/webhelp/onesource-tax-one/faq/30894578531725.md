O objetivo deste artigo é auxiliar consultores e clientes na obtenção de informações e no mapeamento técnico necessários para a geração de obrigações acessórias no Oracle Cloud, com o desenho de solução (TO BE).

A dificuldade em localizar as informações técnicas necessárias para a geração dessas obrigações pode surgir devido à falta de familiaridade com os recursos disponíveis ou à complexidade intrínseca do sistema Oracle.

Passo a passo para resolução de problemas na geração de obrigações acessórias no Oracle Cloud:

  1. Acesse o link do Mapeamento Técnico do Interdados V17 para Oracle Cloud:
     * [Interdados - Mapeamento Técnico - V8.4 - Extrator ORACLE CLOUD](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Interfaces/Interface_Oracle_V17R01_Cloud/Webhelp_Interface_Oracle_V17R01_Cloud/Manuais_portugues/manuais_pdf_excel_ppt/TR_INTERFACE_INTERDADOS_V17_MAPEAMENTO_TECNICO_V8.4_EXTRATOR_ORACLE_CLOUD.zip> "Hyperlink \[https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Interfaces/Interface_Oracle_V17R01_Cloud/Webhelp_Interface_Oracle_V17R01_Cloud/Manuais_portugues/manuais_pdf_excel_ppt/TR_INTERFACE_INTERDADOS_V17_MAPEAMENTO_TECNICO_V8.4_EXTRATOR_ORACLE_CLOUD.zip\]")
  2. Para o Mastersaf DFe, consulte os seguintes links:
     * [Mastersaf Layout DFe V3 NFe e NFCe](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/DFE/DFEV3/Manuais/WebHelpDFEV3/assets/docs/Mastersaf_Layout_DFe_V3_NFe_NFCe_4_00v4_04_OCC.xlsx> "Hyperlink \[https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/DFE/DFEV3/Manuais/WebHelpDFEV3/assets/docs/Mastersaf_Layout_DFe_V3_NFe_NFCe_4_00v4_04_OCC.xlsx\]")
     * [Mastersaf Layout DFe V3 NFSe](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/DFE/DFEV3/Manuais/WebHelpDFEV3/assets/docs/Mastersaf_Layout_DFe_V3_NFSe_OCC.xlsx> "Hyperlink \[https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/DFE/DFEV3/Manuais/WebHelpDFEV3/assets/docs/Mastersaf_Layout_DFe_V3_NFSe_OCC.xlsx\]")
  3. Revise os documentos baixados e aplique as configurações necessárias no sistema Oracle Cloud.
  4. Após implementar as configurações, realize testes de geração de obrigações acessórias para verificar se está funcionando corretamente.
  5. Caso encontre inconsistências, abra um ticket no suporte técnico.


**Dicas e Recomendações:**

  * Familiarize-se com os manuais técnicos e participe de treinamentos oferecidos para melhor compreensão do sistema Oracle.
  * Mantenha o sistema atualizado e consulte regularmente a documentação disponível no site para prevenir problemas futuros.
