Caso enfrente erro de importação na SAFX2013 indicando que o CEST não existe, mesmo estando presente na tabela de SPED. Isso pode ocorrer devido à necessidade de manutenção manual na tabela TACES94, que é uma tabela acessória. A responsabilidade pela atualização dessa tabela é do cliente ou da consultoria, enquanto o suporte é fornecido para tabelas fixas. Para resolver, siga as orientações:

  1. Acesse o sistema e vá em Básicos > Ferramentas > Tabelas Internas > Manter > Tabelas > Código Especificador da Substituição Tributária.
  2. Realize o cadastro dos códigos necessários nesta seção.
  3. Após o cadastro, faça um novo teste de importação para verificar se o erro foi resolvido.
  4. Após realizar os passos acima, monitore o sistema para garantir que não ocorram novos erros de importação relacionados ao CEST. Caso o problema persista, entre em contato com o suporte técnico para mais assistência.


**Nota:** Mantenha a tabela TACES94 atualizada para evitar futuros problemas de importação. Considere revisar periodicamente todas as tabelas acessórias para garantir a consistência dos dados.
