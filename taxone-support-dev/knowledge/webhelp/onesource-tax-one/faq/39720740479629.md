# Como corrigir ausência de registro do imposto PCC na SAFX53

## Problema

O sistema Interdados não está gerando o registro do imposto PCC na SAFX53, apesar de todas as parametrizações estarem preenchidas corretamente.

## Causa Raiz

A ausência do registro do imposto pode ser atribuída à configuração incorreta na tela "Consulta Parametrização de Retenção de Impostos". O campo "Cod. Operação" estava incompleto ou referenciando outros impostos incompatíveis, como INSS.

## Resolução

Siga as etapas abaixo para corrigir o problema:

  1. Acesse a tela **Consulta Parametrização de Retenção de Impostos** no Interdados.
  2. Exclua os parâmetros que fazem referência ao imposto INSS e configure corretamente os códigos de operação.
\- Insira o código PCC para os impostos que o exigem.
\- Para o IR, configure apenas os códigos específicos que referenciam este imposto.
  3. Após realizar as alterações, processe uma nova carga no sistema.
  4. Verifique na SAFX53 se a linha referente ao imposto PCC foi gerada com sucesso.


## Próximos Passos

Depois de realizar os ajustes:

  * Confirme se o imposto PCC está corretamente listado na SAFX53.
  * Revise a parametrização da regra 6 para os impostos cadastrados na tela retenção de impostos, campo Cod.Operação
  * Testar o comportamento do sistema inserindo novas NFs que se enquadrem no cenário fiscal impactado para garantir que as modificações foram bem-sucedidas.
  * Documente qualquer novo comportamento e reporte à equipe de suporte caso surja outro problema.


## Dicas e Recomendações

Para evitar problemas semelhantes no futuro:

  * Antes de gerar uma nova carga, revise as parametrizações na tela "Consulta Parametrização de Retenção de Impostos" para certificar-se de que todos os impostos obrigatórios possuem configurações completas.
  * Revise as regras De/Para que contemplam os campos obrigatórios do layout do procedimento SAFX53.
  * Mantenha uma lista de referência das parametrizações padrão exigidas para cada imposto no sistema.
  * Verifique periodicamente os registros gerados pelo sistema para garantir que nenhum imposto foi omitido.
  * Considere criar documentações internas com cenários comuns de erro e respectivas soluções.


### Recursos Adicionais
