# Como lançar o código de obras/centro de custos no Documento Fiscal

###

## Resolução:

  1. Acesse o módulo **Básicos / Data Warehouse Manutenção Códigos Serviços**.
  2. Localize o código de serviço utilizado.
  3. Verifique se o tipo de serviço está preenchido como **Equiparado à Construção Civil**. Caso não esteja, realize a atualização necessária.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40549053077261)

4\. Acesse o modulo Básicos / Data Warehouse, menu: Manutenção > Documento Fiscal > Novo Documento Fiscal > Doc. Fiscal de Mercadoria e Serviço

Certifique-se de que a informação está corretamente preenchida no campo de envio apropriado, como o campo 81 - COD_CANAL_DISTRIB da SAFX07.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40549089789709)![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40549089794701)![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40549089795981)![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40549053088013)

##

5\. Após realizar as configurações e ajustes, realize o processo de Limpeza/ Equalização no Modulo Básicos > Data Warehouse no menu: Data Mart > Limpeza/ Equalização:

6\. Realize a geração do meio magnético e verifique se o documento Fiscal foi gerado no meio magnético:
