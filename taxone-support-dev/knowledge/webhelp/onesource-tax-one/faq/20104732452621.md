Este processo é realizado quando existe algum problema na geração de cargas, sendo necessário identificar algum erro no Log e localizar o lote gerado para análises.

Passo a passo para realizar a parametrização e gerar o arquivo JSON no Interdados V11R01:

  1. Acesse a tabela MSAFI_MSFTS_CONFIG_PROP e defina o valor 'S' para o parâmetro BACKEND.WRITE.LOG.
  2. Gere a carga desejada e anote o PST_ID criado para esta carga.
  3. Execute o seguinte Select:


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/20104663615117)

  1. Na última coluna, clique em CLOB para visualizar o conteúdo do arquivo JSON gerado.
  2. Copie todo o conteúdo exibido e salve em um arquivo de texto (.txt).
  3. Com o parâmetro BACKEND.WRITE.LOG definido como 'S', todas as cargas geradas terão o arquivo JSON criado.
  4. Após a análise, caso não queira mais gerar os arquivos JSON, acesse novamente a tabela MSAFI_MSFTS_CONFIG_PROP e defina o valor 'N' para o parâmetro BACKEND.WRITE.LOG.
  5. Lembre-se de desabilitar a geração após a análise para evitar o acúmulo desnecessário de dados no banco.
