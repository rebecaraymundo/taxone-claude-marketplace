**like like2 like4 likec as between from using || multiset**
**member submultiset.**

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/38267921686413)

O erro que esta causando a não geração é por conta de uma conta parametrizada incorretamente em **Parâmetros >Movimento Econômico >Contas Contábeis x Itens Movimento Econômico**

**Abaixo demonstro a parametrização incorreta.**

**Veja que a conta esta sem a aspas simples no inicio :**

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/38267921691021)

Após ajuste a DIME SC será gerada corretamente.
