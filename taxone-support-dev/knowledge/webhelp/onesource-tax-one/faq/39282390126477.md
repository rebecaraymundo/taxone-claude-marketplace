### **Sender e a Integração com Cofres de Senhas (API)**

**Problema:** O cliente precisa utilizar um cofre de senhas (como o CyberArk ou outros) para gerenciar credenciais, mas o Sender não tem suporte nativo para essa forma de autenticação.

**Causa Raiz:** Atualmente, o Sender opera exclusivamente com credenciais CIAM que precisam ser configuradas diretamente no arquivo `CONFIG`. Não existe um suporte nativo para integração direta com soluções externas de autenticação ou cofres de senhas.

**Resolução:** Para viabilizar a integração com um cofre de senhas, o cliente deverá desenvolver um integrador personalizado. Esse integrador será responsável por buscar as credenciais no cofre e, em seguida, utilizar o **OBI Gateway (API)** para enviar os dados ao TaxOne.

**Dicas e Recomendações:** O nosso portal **Customer Influencer** é o canal ideal para sugerir melhorias que podem impactar positivamente futuras versões do produto.
