Para realizar o Envio por e-mail - Comprovante Anual de retenção PIS/COFINS/CSLL, siga os passos abaixo:

  1. Em Federal > Obrigações de Tributos Federais > Relatórios > Comprovante Anual de Retenção, clique no ícone envelope.
  2. Preencha as informações na tela Enviar informe de Rendimento por E-mail e clique em **Enviar Declarações**.
  3. Clique no botão Download para consultar os dois arquivos gravados.
Nota: Para salvar os arquivos em alguma pasta, Exemplo: C:\temp\59697, coloque no mesmo diretório os seguintes arquivos:
     1. EnvioEmail.jar ( anexo)
     2. EnvioEmail.properties (anexo)
     3. Benecifiarios_comprovante_anual.csv
  4. Após, todos os arquivos que serão enviados. realize a extração dos arquivos zipados.


Nota: **Con** figure o arquivo** EnvioEmail.properties** com os dados para envio dos e-mails (Verifique com a equipe TI da empresa).

Abra o arquivo e preencha com informações abaixo (configure com os dados do cliente):

**email.tipoEmail=1**

Deixar 1
**email.smtp=smtp.gmail.com**

Informe o nome do servidor smtp
**email.porta=25**

Porta do servidor smtp
**email.from=** jose@gmail.com

Email do remetende
**email.usuario=** jose

Login do usuário de email
**email.senha=** 123456

Senha do Usuário de email
**email.usaSSL=N**

Indique se o servidor smtp usa SSL
**email.indAuth=N**

Indique se o servidor smtp precisa de autenticação
**email.indStartTls=N**

Indique se o servidor smtp usa TSL


**diretorio=C:\\\temp\\\enviarEmail\\\arquivos**

Diretório onde ficarão os arquivos a serem enviados e o arquivo enviar_email.csv. Dever usar \\\ entre os diretórios
**assunto=** Teste Envio de E-mail

Assunto que irá nos e-mails
**corpo=** Este email é para testar o envio....

Texto padrão que irá no corpo dos emails

Entre no **Command Prompt** , e acesse o primeiro diretório onde se encontra o arquivo EnvioEmail.jar.

Digite o comando **java -jar EnvioEmail.jar**

Exemplo o arquivo salvo:

C:\temp\59697

Finalmente, consulte o e-mail.
