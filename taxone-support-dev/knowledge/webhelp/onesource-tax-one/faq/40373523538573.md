## Problema:

Usuário relatou dificuldade ao importar um registro para o sistema fiscal devido a mensagem de erro indicando que o bem patrimonial não está cadastrado, apesar de estar presente no sistema.

## Causa Raiz:

O problema ocorreu devido à configuração errada da data no campo DATA_INI_CRED, que estava definida para uma data anterior à data de aquisição do bem patrimonial.

## Resolução:

  1. Acesse o registro do bem patrimonial no sistema.
  2. Verifique a data de aquisição do bem patrimonial.
  3. Ajuste o campo DATA_INI_CRED, assegurando que a data seja igual ou posterior à data de aquisição do bem.
  4. Salve as alterações feitas no registro.
  5. Proceda com a importação do registro novamente para verificar se o erro foi resolvido.
