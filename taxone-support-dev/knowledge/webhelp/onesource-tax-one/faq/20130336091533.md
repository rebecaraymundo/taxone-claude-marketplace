Passo a passo para fazer a gravação dos usuários de acesso de cada ambiente no FileZilla:


Ao subir uma SAFX para os ambientes no ONESOURCE Tax One, é necessário acessar com os dados de cada ambiente, fazendo as configurações necessárias.

  1. Para otimizar o processo de utilização da ferramenta, é possível deixar os logins já salvos, conforme a imagem a seguir.
  2. Para utilizá-los, apenas selecione o que desejar e clique em conectar.
