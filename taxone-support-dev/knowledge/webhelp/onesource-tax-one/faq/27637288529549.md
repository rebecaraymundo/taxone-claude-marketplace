Para integrar a autenticação SAML do Azure com o Tax One, siga as orientações:

O que é o SAML: A Linguagem de Marcação para Asserção de Segurança, ou SAML, é uma forma padronizada de informar aos aplicativos e serviços externos que um usuário é quem ele diz ser. A SAML possibilita a tecnologia de logon único (SSO) ao fornecer uma maneira de autenticar um usuário uma única vez e, em seguida, comunicar essa autenticação a vários aplicativos.

1\. Verifique se o serviço Single Sign On (SSO) está contratado para o Tax One. Se não, entre em contato com o Account Manager para adicioná-lo.

2\. Uma vez contratado o SSO, a equipe responsável fornecerá suporte durante a implantação, enviando instruções necessárias para a configuração.

3\. No Azure, cadastre o aplicativo utilizado para realizar as autenticações.

4\. Configure o redirecionamento no Azure para que após a autenticação SAML, o usuário seja redirecionado de volta para o Tax One.

5\. Após a configuração, monitore o processo de autenticação para garantir que a integração esteja funcionando corretamente. Caso encontre problemas, entre em contato com o suporte técnico.

Nota: É recomendável revisar periodicamente as configurações de segurança e autenticação para garantir que estão atualizadas e seguras. Para mais informações sobre a configuração do MFA e outras opções de autenticação, consulte a documentação oficial no link abaixo:
