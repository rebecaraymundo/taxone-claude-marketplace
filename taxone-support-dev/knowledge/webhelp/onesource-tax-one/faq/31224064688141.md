## Problema:

Os clientes encontram dificuldades para registrar uma nova Inscrição Estadual no mesmo estabelecimento e Unidade Federativa (UF), após o encerramento da inscrição anterior.

## Causa-raiz:

A mensagem "Já existe registro com a chave informada" é exibida porque já existe uma **Inscrição Estadual** cadastrada para essa UF, mesmo que ela esteja com a data de encerramento registrada. O sistema atual não suporta o cadastro de duas Inscrições Estaduais para a mesma UF no mesmo estabelecimento.
