## Problema

Notas fiscais foram indevidamente incluídas no relatório do mês 08 devido ao uso de parametrizações incorretas para o cálculo da média das saídas dos últimos 3 meses.

## Causa Raiz

O problema ocorreu porque foram inseridas parametrizações no campo "Nº de meses anteriores p/pesquisa das saídas", configurando períodos e/ou valores de forma inadequada, o que resultou na inclusão de notas indevidas no cálculo.

## Resolução

Siga os passos abaixo para corrigir o cálculo e gerar relatórios com as configurações desejadas:

  1. Acesse a área de parametrização do módulo utilizado para o cálculo de médias das saídas.
  2. Localize as parametrizações aplicadas anteriormente. Elas podem incluir configurações relacionadas a "Produtos Industrializados sujeitos ao FCI" e "Operações de Saída dos Produtos Industrializados".
  3. Exclua as parametrizações caso queira limpar o cálculo anterior.
  4. Reconfigure os intervalos ou parâmetros desejados, desta vez ajustando os CFOPs ou excluindo qualquer configuração que tenha incluído documentos indevidos no cálculo prévio.
  5. Sobrescreva o cálculo anterior ao executar novamente a rotina de geração, utilizando as novas parametrizações.
  6. Verifique o relatório gerado e confirme que as notas fiscais indevidas foram removidas e que apenas as desejadas foram consideradas.


## Próximos Passos

Após a realização do cálculo com as novas parametrizações:

  * Revise o relatório gerado para garantir que os dados correspondam à expectativa.
  * Se necessário, ajuste novamente as parametrizações e reexecute o cálculo.
  * Continue monitorando os relatórios gerados para validar os valores conforme os períodos de cálculo.


## Dicas e Recomendações

Para evitar problemas futuros:

  * Considere realizar uma revisão detalhada das parametrizações antes de executar qualquer cálculo de médias.
  * Utilize filtros específicos, como CFOPs, para garantir que apenas os documentos relevantes sejam considerados.
  * Documente as configurações realizadas para facilitar futuras revisões ou ajustes.
  * Consulte regularmente nossa base de conhecimento para aprender sobre boas práticas e atualizações do sistema.
