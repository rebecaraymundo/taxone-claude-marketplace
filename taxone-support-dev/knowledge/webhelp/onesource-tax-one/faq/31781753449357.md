Ao 'Gerar e Enviar XML' do evento R-4020 ocorre o erro com o código de mensagem DWERRXSD, cuja descrição é: "_NOK - NonParserException: cvc-complex-type.2.4.a: Invalid content was found starting with element '{http://www.reinf.esocial.gov.br/schemas/evt4020PagtoBeneficiarioPJ/v2_01_02:endExt}'. One of '{http://www.reinf.esocial.gov.br/schemas/evt4020PagtoBeneficiarioPJ/v2_01_02:frmTribut}' is expected_." A causa raiz desse problema é a falta de informações necessárias para preencher campos obrigatórios do evento. Normalmente, uma mensagem de alerta é apresentada durante a 'Geração Prévia' dos 'Movimentos', indicando a ausência dessas informações.

Passo a passo para resolver o erro ao 'Gerar e Enviar XML' de evento R-4020:

  1. Acesse o módulo _Data Warehouse > Manutenção > Retenções._
  2. Informe a 'Forma de Tributação' no respectivo 'Controle de Tributos'.
  3. Após realizar a correção, monitore o envio do evento para garantir que os dados estejam corretos no eCAC.


**Dica** : Consulte o log da 'Geração Prévia' dos movimentos no menu _EFD - Reinf > REINF > Geração Prévia > Movimentos > LOG_ para obter mais detalhes sobre as inconsistências que estão causando o erro na geração do XML.
