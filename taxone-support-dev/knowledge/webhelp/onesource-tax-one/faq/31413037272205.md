Caso enfrente problemas ao replicar os dados do ECD do ambiente de homologação para o ambiente de produção no Tax One, especificamente com a ausência de geração de algumas informações como I050 Plano de Contas, entre outros. Esse problema pode ocorrer devido ao status das contas no Plano de Contas, que foram marcadas como inativas, o que impede a correta geração dos dados. Para resolver, siga as orientações:

  1. Acesse o Plano de Contas no sistema Tax One.
  2. Verifique a situação de cada conta, garantindo que todas as contas utilizadas estejam marcadas como "Ativa".
  3. Após ajustar a situação das contas, realize um novo teste de replicação dos dados.
  4. Após ajustar o status das contas no Plano de Contas e realizar a replicação dos dados, é recomendável monitorar os próximos envios para garantir que o problema foi completamente resolvido. Caso o problema persista, entre em contato com o suporte técnico.


**Nota:** Regularmente verifique o status das contas no Plano de Contas para evitar problemas futuros na geração de arquivos. Mantenha a documentação e os registros de alterações atualizados para facilitar a identificação e resolução de problemas semelhantes.
