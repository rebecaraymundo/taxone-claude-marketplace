A tela "Identificador de Tipos de Crédito", carregada através da TACES 71, vem com códigos e parametrizações padrão que são úteis para começar a usar o sistema. No entanto, é possível e recomendável que o cliente faça cadastros e manutenções conforme a necessidade de sua empresa.


Passo a passo para incluir um tipo de crédito ou outra combinação que não esteja na tela padrão:

1\. Acesse SPED > EFD Contribuições > Parâmetros > Apuração > Identificador dos Tipos de Crédito


2\. Na tela de Identificador dos Tipos de Crédito, clique no botão "Personalizar parametrização".

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/17829811985421)

3\. Clique em "Incluir" para adicionar um novo tipo de crédito.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/17829860484621)

4\. Uma nova tela será aberta para que você preencha as informações desejadas.


5\. Após preencher os dados necessários, clique em "Confirmar" para salvar as alterações.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/17830038332557)

6\. O sistema exibirá uma mensagem confirmando que os dados foram atualizados.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/17830112718861)

Note que o botão que inicialmente dizia "Personalizar Parametrização" agora mudará para "Retornar Parametrização padrão Mastersaf" após qualquer inclusão de parametrização. Isso indica que alterações foram feitas. Se você selecionar esse botão, o sistema reverterá para as parametrizações padrão da tela, e todas as inclusões e alterações feitas serão perdidas.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/17830253914893)
