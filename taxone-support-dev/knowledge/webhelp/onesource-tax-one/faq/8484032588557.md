Para verificar o **log de importação das tabelas** , siga os passos abaixo:

  1. Verifique se o log foi importado com sucesso ou se apresentou erro e qual é a solução sugerida para a correção, sendo:

     1. Em Básicos > Job Servido > Controles > Relatórios > Relatório por Processo > Importação, na tela **Selecionar Processo** realize a busca do arquivo pelo número do processo, data, usuário.

     2. No campo **Descrição** é demonstrada a tabela para seleção, selecione o log desejado e clique em OK.

     3. É demonstrado o status da importação e a solução para o erro.

     4. Se a importação foi feita corretamente, é demonstrada a mensagem: Importação dos dados Finalizada com Sucesso.
