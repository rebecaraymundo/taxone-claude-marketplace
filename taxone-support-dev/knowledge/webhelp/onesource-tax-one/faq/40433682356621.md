##
Problema

Erro "Relational operator 'MOD' is not supported." ao aplicar a Request Principal do pacote de atualização ou instalação da Interface OBTI (print do erro abaixo):

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40967989495565)

##
Causa Raiz

A origem do erro esta está relacionado à funcionalidade "MOD" não ser suportada pela versão SAP ERP do cliente que ocasionaram erros em alguns trechos do código e em includes específicos.

##
Resolução

Atualizar para uma versão superior a **4108** que ja contempla o ajuste que elimina funcionalidade "MOD" de alguns trechos do código e includes.

  1. Realize a aplicação das Requests na ordem abaixo:
     1. Request Principa l(TRQ).
     2. Request Customizing(TDD)
     3. Request para Webservice
  2. Evite o uso de encapsulamento, conforme orientações da equipe de suporte, pois ele pode causar divergências nos dados transportados;
  3. Realize um monitoramento rigoroso do processo, incluindo a análise de logs gerados durante o transporte das Requests.


##
Próximos Passos

[](<https://suporte.thomsonreuters.com.br/instructions>) Realizar as seguintes ações após a aplicação das Requests:

  1. Verificar se o problema foi solucionado em todos os ambientes (DEV, QAS e PRD);
  2. Testar a funcionalidade atualizada para garantir que o problema de sintaxe foi corrigido e que as tabelas foram atualizadas corretamente;


##
Dicas e Recomendações

  * Mantenha backups regulares de seus ambientes antes de aplicar atualizações. Isso permitirá reverter processos caso algo não funcione conforme esperado;
  * Organize e valide a sequência de aplicação das Requests junto ao suporte técnico antes de realizar o transporte;
