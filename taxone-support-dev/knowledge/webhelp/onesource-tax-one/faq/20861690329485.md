Acesse esta FAQ no [web help do produto](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/WebHelp/mergedProjects/FAQs/Basicos_JS_SAFX01.htm#basicos_js_safx01_06>).

Normalmente, esse problema ocorre porque o código informado no campo CONTA_DEB_CRED não está cadastro na tela Data Warehouse > Manutenção > Códigos > Plano de Contas. Para corrigir o erro 90012: 'Conta Contábil não cadastrada ou data de validade posterior a data do lançamento', siga as orientações abaixo:

  1. Informe um código valido no arquivo TXT, ou
  2. Cadastre o código informado no arquivo TXT na tela Data Warehouse > Manutenção > Códigos > Plano de Contas.
