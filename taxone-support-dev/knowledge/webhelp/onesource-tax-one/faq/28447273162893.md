Passo a passo para resolver a divergência na quantidade de registros ao enviar para o TaxOne:

  1. Quando for identificado que a quantidade de registros extraídos no SAP é superior à importada no TaxOne, durante uma carga realizada via WebService, verifique se o retorno da extração contém alguma informação sobre erros.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/31576969639949)


  2. Caso encontre essa mensagem de erro, clique em "Detalhe" e confirme se a descrição do erro é "Obrigatório em Branco".

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/31576969642253)


  3. Se esse for o caso, acesse a Parametrização Geral > aba "Webservices", e marque a opção "**Envio de dados sem campo obrigatório no WS** ".

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/31576969643789)


  4. Após realizar essa parametrização, faça uma nova carga via WebService.
  5. Verifique se, com essa nova carga, a quantidade de registros enviados do SAP para o TaxOne está correta.
