### **Mensagem de usuário (User ID) já existe, no entanto ele não está cadastrado no ambiente**

**Problema:** Ao tentar cadastrar um novo usuário no módulo Administration, o sistema retorna uma mensagem informando que o **User ID (login)** já está em uso, mas o usuário não consta na lista de cadastros do ambiente atual.

**Causa Raiz:** A plataforma de login do Tax One é **única e compartilhada** entre todos os ambientes de todos os clientes. Por isso, cada User ID deve ser globalmente único. Se a mensagem de erro é exibida, significa que o login que você está tentando criar já está sendo utilizado por um usuário em **outro ambiente de um cliente diferente**.

**Resolução:** Devido à arquitetura de segurança da plataforma, não é possível identificar em qual ambiente o User ID está cadastrado. A solução imediata é:

  1. **Utilizar um User ID (login) diferente e único** para realizar o cadastro do novo usuário.


**Próximos Passos e Recomendações:**

  * Para evitar esse problema, sugerimos criar logins com alguma padronização que garanta a unicidade e evite conflitos com outros ambientes (por exemplo, utilizando o nome da empresa ou um prefixo identificador).
  * O login bem-sucedido na URL única da plataforma é o que direciona o usuário ao ambiente correto, o que reforça a necessidade de o User ID ser único em toda a plataforma.
