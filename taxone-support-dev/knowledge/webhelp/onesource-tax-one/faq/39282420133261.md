# Como realizar um refresh de dados do ambiente de Produção para o ambiente UAT

## Problema

É necessário realizar um refresh de dados do ambiente de produção para o ambiente UAT, mas há dúvidas sobre os passos e os riscos envolvidos no processo.

## Causa Raiz

O cliente deseja atualizar o ambiente UAT com os dados mais recentes do ambiente de produção para realizar testes e validações em um ambiente mais próximo da realidade de produção.

## Resolução

  1. Confirme se o procedimento está previsto no contrato da sua empresa. Para isso, entre em contato com o nosso time de CS - Customer Sucess para avaliação do seu contrato.
  2. Considere os impactos dessa operação. O principal risco é a sobreposição dos dados atuais do ambiente UAT com os dados provenientes do ambiente de produção. Certifique-se de que todos os dados e configurações importantes do ambiente UAT foram previamente salvos ou são dispensáveis.
  3. O processo consiste na clonagem completa do ambiente de produção, incluindo dados, parâmetros, logs, e outros recursos.
  4. Se o serviço estiver disponível em seu contrato, crie um time para o nosso time de suporte prosseguir com a clonagem do ambiente de Produção para o ambiente UAT.
