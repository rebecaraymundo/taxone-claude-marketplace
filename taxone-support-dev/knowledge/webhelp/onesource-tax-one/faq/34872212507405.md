## Este artigo tem como objetivo esclarecer como corrigir o erro o gerar o arquivo XML da Gissonline de ITU:
"A tag de Construção Civil é obrigatória e não foi informada no arquivo"

## Causa Raiz:

A causa do erro é a falta de preenchimento das tags `<tipos:CodigoObra>` e `<tipos:Art>` necessárias para a construção civil no XML.

## Resolução:

  * Acesse a capa da nota fiscal (SAFX07 - Campo 81) e preencha o campo "Canal Dist. / Obra". Este valor será usado para gerar a tag `<tipos:CodigoObra>`.


  * Vá até o cadastro do Canal de Distribuição / Obra (Básicos Data Warehouse Manutenção Cadastros Canais de Distribuição / Obras) e preencha o campo "Código ART". Este conteúdo será usado para a tag `<tipos:Art>`.


  * Verifique no cadastro do serviço se o campo Tipo de Serviço está selecionado como "Equiparado a Construção Civil".


## Próximos Passos:

Após inserir as informações corretas nas tags, gere novamente o arquivo XML e verifique se o erro foi corrigido. Caso persista, entre em contato com o suporte técnico.
