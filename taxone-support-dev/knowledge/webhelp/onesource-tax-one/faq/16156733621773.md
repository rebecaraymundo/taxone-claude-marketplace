Normalmente, esse problema ocorre pois o campo DAT_RE não está preenchido ou está preenchido com @.

Para corrigir o erro 91274: 'O campo Data do Registro de Exportação deve ser preenchido.', siga a orientação abaixo:

  * Preencha o campo DAT_RE com a data correta.
