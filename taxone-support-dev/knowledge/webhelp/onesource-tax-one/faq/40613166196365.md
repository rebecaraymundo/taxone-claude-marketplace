**Origem das informações para geração do campo :**


C100 | 23 | VL_BC_ICMS_ST | Valor da base de cálculo do ICMS substituição tributária | SAFX07 / SAFX08 | **64-Base ICMS Substituição / 61-Base ICMS Substituição ou 106-Base ICMS Origem/Destino / 90-Base ICMS Origem/Destino**
---|---|---|---|---|---
C100 | 24 | VL_ICMS_ST | Valor do ICMS retido por substituição tributária | SAFX07 / SAFX08 | **48-Valor ICMS Substituição / 52-Valor ICMS Substituição ou 107-Valor ICMS Origem/Destino / 91-Valor ICMS Origem/Destino**
---|---|---|---|---|---

**Importante:**
O valor só deve ser preenchido se o indicador de ST estiver marcado, ou seja, a coluna **IND_CRED_ICMSS (campo "APROPRIA")** deve ser igual a **“S”**.
Além disso, para documentos modelo **65 (NF-e para consumidor final)** , este campo deve ser deixado em branco (`||`).
