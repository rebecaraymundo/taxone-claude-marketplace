## Problema

Ao tentar gerar o arquivo SAFX2013 utilizando a parametrização de Tipo de Material por Centro, o sistema não está respeitando as parametrizações realizada

## Causa Raiz

A origem do problema esta na falta de parametrização completa

## Resolução

  1. Acesse o sistema SAP e vá até a transação /n**/msafx/saftr085 (** Especifico Tipo de Material e Utilização).
  2. Confirmar se as configurações para tipo de material, centro, classe Item estão corretos
  3. Certifique-se de que na transação parâmetros gerais, aba parametros gerais o parâmetro **Quebra arquivo por centro** está marcado.
  4. Salve as alterações realizadas.
  5. Realize um novo teste para verificar se o arquivo SAFX2013 está sendo gerado corretamente com base na parametrização.


## Próximos Passos

  1. Após realizar o teste, confirme se o arquivo está sendo gerado corretamente.
  2. Verifique outras configurações relacionadas caso o problema persista.
  3. Entre em contato com o suporte caso ainda enfrente dificuldades.
