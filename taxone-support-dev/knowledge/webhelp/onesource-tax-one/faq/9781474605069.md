Para realizar a geração de dados de integração de lançamentos contábeis para as tabelas de integração do Onesource ECF, siga os passos abaixo:

  1. Realize os Lançamentos Contábeis em [Básicos > Data Warehouse > Manutenção > Contabilidade > Diário Geral > Lançamentos Contábeis > Por Estabelecimento](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Basicos/Basicos_Data_Warehouse/Basicos_Data_Warehouse_Menu_Manutencao/Basicos_DW_Manutencao_Contabilidade.htm>).
Nota: Os registros de lançamentos contábeis são necessários para compor o arquivo TXT gerado.

  2. Realize a parametrização no módulo.
Nota: Para realizar a geração do arquivo é necessário que o módulo esteja devidamente parametrizado, conforme descrito nos seguintes artigos:
[Como parametrizar CNPJ da Central de Cadastros?](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/microcontent/FAQ_-_CONEXAO_ONESOURCE-08-05.mc#ECF_param_CNPJ_Central>)
[Como realizar a parametrização das Contas Contábeis?](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/microcontent/FAQ_-_CONEXAO_ONESOURCE-08-05.mc#ECF_param_Contas_Contabeis>)

  3. Realize a geração dos registros em [Conexão Onesource > ECF > Geração](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Conexao_Onesource_BR/ECF_Escrituracao_Contabil_Fiscal/ECF_Geracao/ECF_Geracao_Dados.htm>).
Nota: A geração pode ser realizada para inclusão ou exclusão. No sentido de geração de dados para exclusão, atente-se que para se gerar as informações de exclusão, primeiramente, deve ser realizada a exclusão dos registros de lançamentos contábeis, os quais farão parte do range de registros gerados, para que possa ser gerado os dados com leiaute de exclusão.

  4. Realize a validação da Integração no Onesource ECF em Painel > Painel de Controle e em Dados de Origem > Lançamentos Contábeis.


Nota: Caso ocorram erros na integração, estes podem ser visualizados através do Painel de Controle e também do relatório de erros e avisos de integração.
