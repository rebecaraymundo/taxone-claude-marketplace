O objetivo deste artigo é orientar o cliente à parametrizações referente ao extrator para a SAFX 2013.

**Próximos Passos:**

  * Instalar extrator LACLS_ExtractSpedMasterData no BIPublisher do Oracle Cloud.
Patch publicado no MOS: patch 37670050
For more details about how to run the deployment tasks, please refer to Cloud My Oracle Support Note KB8362, section 4.
**Diretório:**
Template: Shared Folders → Custom → Local Solution → BR → Fiscal Integrations
Data Model: Shared Folders → Custom → Local Solution → BR → Fiscal Integrations → Data Models

  * No Interdados, parametrizar o catálogo, **ativar o serviço JAVA** e parâmetros do extrator LACLS_ExtractSpedMasterData para SAFX2013 no Interdados V17.
Menu Procedimentos> Mais > Bip Procedimentos.
Menu Procedimentos> Editar Procedimento > Flag Serviço Java.
Menu Procedimentos> Mais > Parâmetros Extrator.

  * Executar o Extrator LACLS_ExtractSpedMasterData com os parâmetros:
<P_LEGAL_ENTITY> \-- informar Empresa ou null para todos
<P_BU_ID>\-- informar Business Unit ou null para todos
<P_LRU_ID> \-- informar Estabelecimento ou null para todos
<P_INV_ORG> \-- informar Organização de Inventário ou null para todos
**< P_DATASET>3 -- informar o valor 3 que representa Itens**
<P_PARTICIPANT_TYPE>informar null para todos
<P_LAST_UPDATE_FROM> \-- informar data de ou null para todos
<P_LAST_UPDATE_TO>\-- informar data ate ou null para todos
