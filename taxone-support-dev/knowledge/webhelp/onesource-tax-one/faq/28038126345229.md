Passo a passo para verificar se o New GL está ativo na Geração da SAFX01:

  1. Acesse a transação SE37 e execute a função /MSAFX/CHECK_NEWGL.
  2. Essa função irá verificar se o New GL está ativo no sistema.
     1. Se o New GL estiver ativo, as origens dos dados são as tabelas FAGLFLEXA e GLT0.
     2. Se o New GL estiver desativado, as origens dos dados são as tabelas BSEG e BKPF.


Dicas e Recomendações:

  * Mantenha as configurações do ambiente SAP atualizadas e revise-as regularmente para evitar problemas na geração de relatórios e interfaces.
  * Verifique sempre as tabelas de origem dos dados para garantir que as informações estejam corretas.
