Se os saldos de dezembro foram gerados com valores zerados no módulo ECF de Escrituração Contábil sem executar a reversão dos lançamentos de encerramento, siga as orientações:

  1. No módulo Conexão Onesource BR.
  2. Navegue até Parâmetros > Parametrização para Recomposição dos Saldos Contábeis.
  3. Realize a parametrização necessária conforme as instruções específicas do sistema ou consulte o manual do usuário.
  4. Após a parametrização, gere os saldos novamente para o período desejado.
  5. É necessário também processar o lote da ECF novamente após ajustar a parametrização.
  6. Após realizar as etapas de resolução, monitore os resultados para garantir que os saldos estão sendo gerados corretamente. Se o problema persistir, entre em contato com o suporte técnico para uma investigação mais aprofundada.


**Importante:** Regularmente verifique e atualize as parametrizações para evitar problemas similares no futuro. Considere também a revisão periódica dos procedimentos de fechamento contábil para assegurar que todas as etapas estão sendo executadas corretamente.
