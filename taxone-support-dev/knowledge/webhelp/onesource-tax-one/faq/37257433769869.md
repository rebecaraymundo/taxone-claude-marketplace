# [Municipal] [ISS] TAX ONE - Como parametrizar Tipo de Documento utilizado nas notas fiscais no sistema de geração para a Prefeitura de Curitiba.

Ao tentar gerar o arquivo do meio magnético com as notas fiscais de entrada para declaração junto à Prefeitura de Curitiba, pode ocorrer um erro relacionado ao parâmetro de **Tipo de Documento**.

Causa Raiz:

Esse erro ocorre devido à falta de parametrização adequada do **Tipo de Documento** utilizado nas notas fiscais dentro do sistema responsável pela geração dos relatórios para a Prefeitura de Curitiba.

Resolução:

Siga o passo a passo abaixo para corrigir a parametrização:

  1. **Acesse o sistema de suporte administrativo** referente ao ambiente Municipal de Curitiba.
  2. Navegue até o menu:
**Municipal Curitiba > Parâmetros > Tipo Documento Master SAF X Tipo Documento ISS - Curitiba**.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/37748174812941)

**3\. Verifique e ajuste** a parametrização do Tipo de Documento conforme necessário, garantindo que cada tipo utilizado nas notas fiscais esteja corretamente configurado.

**4\. Salve** as alterações realizadas.

5\. Tente novamente gerar o arquivo de notas fiscais.

Próximos Passos:

Após realizar o ajuste, monitore as próximas gerações de arquivos para garantir que o erro foi solucionado.
Se o problema persistir, entre em contato com o suporte técnico para assistência adicional.

Dicas e Recomendações:

  * **Revisão Periódica:** Recomenda-se revisar regularmente as parametrizações dos Tipos de Documento no sistema, evitando incidentes futuros.
  * **Consulta de Tipos Cadastrados:** Utilize o caminho informado para consultar todos os Tipos de Documento cadastrados e certifique-se de que suas configurações estejam sempre atualizadas.
