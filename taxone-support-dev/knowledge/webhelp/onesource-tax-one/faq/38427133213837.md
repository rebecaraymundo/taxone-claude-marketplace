Os arquivos auxiliares 201 são gerados em : **ICMS > DATA MART > Convênio ICMS 115 > Geração do Meio Magnético** e em **ICMS > DATA MART > Convênio ICMS 115 > Geração do Meio Magnético - Arquivos - Por aqui s****ão arquivos apenas para conferência.**

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/38446385476237)

**Auxiliares********são exatamente os mesmo gerados em ICMS > DATA MART > Convênio ICMS 115 > Geração do Meio Magnético,**

**incluindo mesmos nomes e mesmos layouts.**

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/38446385485325)

OBS: Para a geração do convênio 115 que é gerado flegando a opção "Convênio 60" que destaquei em laranja acima, a Opção no Formato do Convênio ICMS deve ser o Convênio 60. Isso ocorre porque o Convênio 60 trouxe atualizações relacionadas ao Convênio 115, fazendo com que ele prevaleça.


Gerando pela tela do Meio Magnético temos destacado em Rosa os arquivos para validação e em Laranja os arquivo auxiliares 201:

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/38446385488781)

Ficando alinhado que o comportamento do sistema é :



  * A geração pelo **meio magnético** apaga o hash/code
  * A geração pelos **arquivos auxiliares não** apaga o hash/code


Para garantir a correta execução, o fluxo recomendado é:


  1. Realizar a geração em :******ICMS > DATA MART > Convênio ICMS 115 > Geração do Meio Magnético**
  2. Importar o **Hash Code em : ICMS >****** DATA MART > Convênio ICMS 115 > **Importação Hash Code e Conferência do Registro de Saídas**
  3. Conferir os arquivos auxiliares em: **ICMS > DATA MART > Convênio ICMS 115 > Geração do Meio Magnético - Arquivos Auxiliares.**
