## Problema:

Um usuário expressou dúvidas relacionadas o TaxOne para o novo modelo de comunicação fiscal 62.

## Causa Raiz:

A incerteza surgiu devido à transição do modelo de comunicação 22 para o modelo 62. A falta de informações claras sobre como proceder com as mudanças necessárias gera questionamentos sobre a configuração adequada das interfaces.

## Resolução:

Siga os passos abaixo para garantir que a interface D365 e o TaxOne estejam preparados para o modelo de comunicação 62:

  * O sistema já se encontra adequado para receber esse modelo.
  * Acesse o documento [T1_Release_Notes](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/WebHelp/mergedProjects/Webhelp_ONESOURCE_TAXONE/Informativos/T1_Release_Notes.htm>) disponível nos informativos da empresa para verificar as especificações e as atualizações necessárias para suportar o modelo 62.
  * Implemente as mudanças conforme as diretrizes do documento [T1_Release_Notes](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/WebHelp/mergedProjects/Webhelp_ONESOURCE_TAXONE/Informativos/T1_Release_Notes.htm>)
  * Após realizar as alterações, faça uma verificação para garantir que a integração está funcionando corretamente.
