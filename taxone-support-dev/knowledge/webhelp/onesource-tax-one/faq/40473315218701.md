### Problema:

Quando realizado a extração de determinado processo é reportado a mensagem de erro: /MSAFX/SAFMESSAGES:534

### Causa Raiz:

Não foram selecionados dados para gerar os arquivos de saída ou inserts.

### Resolução:

  * Verificar se as parametrizações necessárias para a extração deste processo foram realizados.
  * Verificar se o mesmo tem alguma customização, caso tenha desabilitar e realizar novo teste.
  * Confirmar se deveria ser gerado alguma informação.


### Dicas e Recomendações

  * Verificar se os campos chaves estão com informação
  * confirmar se as tabelas de origem contem dados para extração
  * se continuar com a mensagem ou problema abrir chamado junto a equipe de suporte.
