Caso encontre uma dificuldade em gerar os dados no Relatório de Conferência mesmo seguindo o procedimento padrão de geração, isso pode acontecer devida a utilização da mesma data para geração e emissão do relatório. Para resolver, siga as orientações:

**Nota:** A data utilizada na emissão deve ser a data em que a solicitação de geração foi realizada.

  1. Acesse o sistema de geração de relatórios.
  2. No campo de emissão, ajuste a data para aquela em que foi realizada a solicitação de geração (exemplo: 26/07/2024).
  3. Execute o procedimento de geração novamente.
  4. Verifique se os dados são gerados corretamente.
