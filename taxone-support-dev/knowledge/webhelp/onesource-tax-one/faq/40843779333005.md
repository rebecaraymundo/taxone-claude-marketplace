# [Básicos][Gerais]ONESOURCE Tax One - Erro no Preenchimento das Bases de Tributação ICMS

PROBLEMA:

Erro no preenchimento dos Campos da SAFX08.

55 - TRIB_ICMS

56 - BASE_ICMS

82 - TRIB_ICMS_AUX

83 - BASE_ICMS_AUX

CAUSA RAIZ:

Erros no preenchimento dos Campos relacionados a ICMS causados pela indicação errada das Bases que serão tributadas.

RESOLUÇÃO:

Seguindo nosso Manual de Layout deve se Informar o campo da Tributação ICMS, de acordo com a Legislação de cada UF (Regulamento do Imposto sobre Circulação de Mercadorias e Serviços – de cada UF):

1 - Operações com Crédito/Débito ICMS;

2 - Operações com Isenção, Não Incidência, Imunidade Constitucional e Parcela de Redução Base de Cálculo;

3 - Outras.

Para cada item de um Documento Fiscal (Produto/mercadoria) o cliente deverá informar se a Operação é Tributada, Isenta ou Não Tributada ou Outras.

Esta informação é utilizada para a Escrituração dos Livros dos Convênios ICMS, e também para a Apuração do Imposto. Além disso, pode acontecer de uma Operação em que Parte do valor é Tributado e parte do Valor é Isento, e/ou outras combinações. Por isso, ao se definir as interfaces, deve ser analisada quais os dados que devem ser remetidos.

No caso de haver somente uma especificação referente a Operação devem ser informados os “Campos 55 e 56”.

No caso de uma operação, em que se necessita mais de uma especificação, ou seja, deve ser utilizado mais de Um Código de Tributação, além dos “Campos 55 e 56”, devem ser utilizados os “Campos 82 e 83”, para que todos os dados estejam contemplados para a Escrituração Fiscal.

Para indicar qual Base será tributada no campo 56 - BASE_ICMS é necessário que o campo anterior 55 - TRIB_ICMS esteja indicado com código: 1 - Operações com Crédito/Débito ICMS. (será preenchido BASE_ICMS)

Segue o link para Nosso Manual de Layout atualizado onde constam as informações referente aos campos da SAFX08: [Web Help - Tax One - Manuais_Tecnicos](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/WebHelp/mergedProjects/Webhelp_ONESOURCE_TAXONE/Manuais_Tecnicos/Manuais_Tecnicos.htm>)
