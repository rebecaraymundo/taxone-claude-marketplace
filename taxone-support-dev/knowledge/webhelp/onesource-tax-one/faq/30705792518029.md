**Pergunta** :

Acabei de adquirir o Tax One, é possível realizar as gerações das obrigações e guias diretamente pelo Tax Automation pela primeira vez?

**Resposta** : Não, a primeira execução deve ser realizada no Tax One. Dessa forma, os registros dos parâmetros utilizados ficarão gravados no banco de dados e quando o Tax Automation fizer uma execução, conseguirá resgatar os mesmos parâmetros obrigatórios da última execução.
