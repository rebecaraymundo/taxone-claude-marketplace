## Problema:

Usuários enfrentam erros durante a importação de dados no sistema Mastersaf DW/TAX ONE, relacionados a inconsistências nos campos de CPF para pessoas físicas.

## Causa Raiz:

O sistema verifica se os dados nos campos IND_FIS_JUR_DMED e COD_FIS_JUR_DMED correspondem a uma pessoa física e se o campo CPF/CPNJ está corretamente preenchido. Se o campo CPF está vazio ou possui erros, o sistema não consegue validar as informações, resultando em erro de importação.

## Resolução:

  * Acesse o módulo: BÁSICOS DATAWAREHOUSE Manutenção Cadastros Pessoas Físicas/Jurídicas no sistema Mastersaf DW/TAX ONE.


  * Verifique e certifique-se de que o CPF está corretamente cadastrado para cada pessoa física no campo COD_FIS_JUR_DMED.


  * Realize a importação novamente e verifique se o problema persiste.


## Próximos Passos:

Após ajustar as informações de CPF e realizar uma nova importação, monitore os resultados para garantir que não haja novos erros. Caso o problema continue, contate o suporte técnico para uma análise mais detalhada.

## Dicas e Recomendações:

Regularmente verifique e atualize os cadastros de pessoas físicas e jurídicas para evitar problemas de importação. Considere revisar o Manual de Layout disponível no sistema para entender melhor os requisitos de cada campo.
