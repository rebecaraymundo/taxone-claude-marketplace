**CAUSA RAIZ:**

Quando for solicitada a criação das credencial CIAM para conexão, é necessário indicar para qual dos serviços as credenciais serão utilizadas.

**OBTI = "WEBSERVICE DIRETO"**

\- Credenciais utilizadas no Integrador para processos Webservice direto com o Tax One, sempre que solicitadas o chamado deve ser direcionado para o Time de TECHOPS que fará a liberação.

TAXONE-INTEGRATOR = **INTERDADOS (Oracle) - *** Disponível somente para a V17

WS-CARGA-SAFX = **INTERFACE (Namespace SAP)**

**OBI**

\- Credenciais utilizadas na conexão com a ferramenta OBI, sempre que solicitadas o chamado deve ser direcionado para o Time de Contratos/Licenciamento.

Obs:. A criação do CIAM não possui custos para o cliente.
