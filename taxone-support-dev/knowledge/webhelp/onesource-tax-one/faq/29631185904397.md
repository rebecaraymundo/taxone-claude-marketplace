Caso enfrente erros de estrutura ao tentar parametrizar informações para municípios no campo "Modelo Msaf x Série". O problema pode estar na utilização de códigos incorretos para a parametrização dos campos mencionados, o que gera divergências e erros de estrutura nos relatórios gerados. Para resolver, siga as orientações:

  1. Acesse o sistema de parametrização para municípios.
  2. No campo "Parâmetros para Município - Parâmetros Modelo Msaf x Série", utilize o código **"01 - Nota Fiscal de Serviços Tributados -Série A"**.
  3. Após realizar as alterações, gere um novo relatório para verificar se o problema foi resolvido.
