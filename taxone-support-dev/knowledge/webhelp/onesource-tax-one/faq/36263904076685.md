# Como realizar o De x Para Contas Referenciais x Contas Empresa e Aglutinação em Massa no TaxOne.

## Problema:

Usuários precisam realizar o cadastro das Contas Referenciais x Contas Empresa (De x Para) e Aglutinação em massa, sem interferir nos dados de anos anteriores.

## Causa Raiz:

O sistema TaxOne não suporta diretamente a cópia ou duplicação de parametrização para diferentes anos, sendo necessário realizar exportação e ajuste das configurações para cada novo ano fiscal.

## Resolução:

  * Acesse o TaxOne e exporte a SAFX2101 existente na base.


  * Verifique e faça ajustes de inclusão de contas conforme necessário. A versão do plano referencial do governo para o novo ano (exemplo versão 11) pode apresentar contas novas.


  * Importe a SAFX2101 ajustada para o TaxOne aplicando a versão nova (exemplo versão 11).


  * Repita os passos para a SAFX2102 (Aglutinação), exportando dados, fazendo ajustes necessários e reimportando para a nova configuração dos códigos de aglutinação.


## Próximos Passos:

Após a importação bem-sucedida dos arquivos ajustados, realize testes para garantir que o plano referencial e a aglutinação estão funcionando corretamente para o ano fiscal desejado. Monitore os resultados nos primeiros dias para confirmar se todas as configurações estão de acordo com o necessário.

## Dicas e Recomendações:

É recomendável sempre fazer uma cópia de segurança das configurações atuais antes de realizar qualquer importação ou alteração nos sistemas fiscais. Mantenha comunicação constante com o suporte técnico durante o processo de ajuste e importação para resolver rapidamente quaisquer problemas que surjam.
