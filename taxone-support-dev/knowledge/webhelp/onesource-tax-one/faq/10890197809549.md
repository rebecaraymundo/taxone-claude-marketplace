Passo a passo para gerar o parâmetro **Série Msaf x Série** para todos os Módulos Municipais que utilizam este De x Para:

**Documento Fiscal**

  1. Acesse [Básicos > Data Warehouse > Manutenção > Documento Fiscal > Doc. Fiscal de Serviço](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Basicos/Basicos_Data_Warehouse/Basicos_Data_Warehouse_Menu_Manutencao/Basicos_DW_Menu_Manu_Doc_Fiscal.htm>).

  2. Na **Capa da Nota Fiscal** , no campo **Número NF **verifique qual **s****érie **foi informada.


**Parâmetro 'Série Msaf x Série'**

  1. Acesse [Municipal > Parâmetros para Município > Parâmetros> Modelo Msaf x Série](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Municipal/Parametros_Para_Municipio/Parametros_Municipio.htm>)

  2. Selecione a **UF **e o **Município.**

  3. Na tela **Relação Modelo MasterSaf X Série** :

     1. Informe a **Validade **do relacionamento entre as séries. A validade é do tipo “a partir de”.

     2. No campo **Série MSAF** digite a **série **utilizada nas notas fiscais cadastradas no Data Warehouse.

     3. No campo **Série** selecione as séries de acordo com o validador. O validador será identificado através do município previamente selecionado.
