# [SPED][EFD - ECD] TAX ONE - ECD - Registro 0150 e 0180 não são gerados na ECD como corrigir.

**Problema:**

Usuários estão enfrentando dificuldades para gerar corretamente os registros 0150 e 0180 no meio magnético da ECD no TaxOne após a importação de dados.

**Causa Raiz:**

O problema ocorreu porque o flag "Gerar Código do Participante (Safx39)" no módulo ECD > Parâmetros > Dados Iniciais estava desmarcado, impedindo que os registros 0150 e 0180 fossem gerados durante a criação do arquivo magnético.

**Resolução:**

Acesse o módulo ECD.

Navegue até Parâmetro e depois Dados Iniciais.

Verifique se o flag "Gerar Código do Participante (Safx39)" está desmarcado.

Caso esteja desmarcado, marque essa opção. Salve as configurações.

Refaça a geração do meio magnético para ECD e verifique se os registros 0150 e 0180 são gerados corretamente.

**Próximos Passos:**

Após realizar as alterações sugeridas e gerar novamente o meio magnético, é importante monitorar se os registros estão sendo criados conforme esperado. Caso o problema persista, entre em contato com o suporte técnico para uma análise mais detalhada.

**Dicas e Recomendações:**

Recomenda-se sempre verificar todos os parâmetros no módulo ECD antes de iniciar a geração do arquivo magnético, assegurando que todas as configurações necessárias estão corretas. Mantenha documentação atualizada sobre o processo para futuras consultas.
