## Problema

O usuário deseja saber se documentos cancelados no ERP são integrados com o sistema Taxone.

## Causa Raiz

O sistema Taxone apresenta um comportamento padrão em relação ao status dos documentos integrados, carregando apenas aqueles aprovados ou definidos como válidos, dependendo do tipo de documento (entrada ou saída) e das configurações específicas do produto.

## Resolução

Siga os passos abaixo para entender o comportamento da integração:

  1. Para documentos de entrada (SAFX07, SAFX08 e SAFX09): São integrados ao Taxone apenas os documentos com o status "aprovado", ou seja, aqueles cujo campo **G_INVOICES/APPROVAL_STATUS** não está definido como "CANCELLED".
  2. Para documentos de saída (SAFX07, SAFX08 e SAFX09): São integrados todos os documentos disponíveis no XML, exceto se houver uma regra configurada no produto para exclusão de determinados documentos.
  3. Verifique as configurações do sistema ERP e do Taxone para confirmar se existem regras específicas que impactam a integração.


## Próximos Passos

Após verificar os documentos integrados:

  1. Confirme no ERP o status dos documentos que devem ser integrados.
  2. Realize testes no Taxone para garantir que as regras de integração estão funcionando conforme esperado.
  3. Se necessário, ajuste as configurações no produto para atender às necessidades específicas do cliente.


## Dicas e Recomendações

  * Para evitar problemas futuros, verifique o status dos documentos no ERP antes de realizar a integração com o Taxone.
  * Documente qualquer regra personalizada aplicada no sistema para que os analistas possam realizar o suporte de forma mais eficiente.
  * Considere revisar periodicamente os documentos integrados para garantir que os dados estão consistentes entre os sistemas.
  * Para mais informações sobre o funcionamento da integração, consulte a documentação oficial do Taxone ou entre em contato com o suporte técnico.
