Caso enfrente problema ao acessar o TR Transfer e/ou criar um novo usuário, isso pode ocorrer devido a incompatibilidade da versão do Java instalada com a versão requerida pelo TR Transfer File. O sistema necessita do Java Amazon Corretto na versão 17, mas poderia estar configurado para uma versão anterior.

Para resolver, siga as orientações:

  1. Verifique a versão do Java instalada executando `java -version` no prompt de comando.
  2. Caso a versão não seja a Java Amazon Corretto 17, faça o download e instale a versão correta.
  3. Acesse as "Variáveis de Ambiente" do sistema.
  4. Na opção "Path", mova o caminho do Java versão 17 para o topo da lista.
  5. Confirme se a variável JAVA_HOME está apontando para o caminho do Java versão 17 e ajuste se necessário.
  6. Reinicie a máquina para aplicar as alterações.
  7. Após a reinicialização, verifique se os serviços relacionados ao TR Transfer estão ativos e tente acessar o sistema novamente.
  8. Após seguir os passos de resolução, o usuário deve verificar se consegue criar um novo usuário e acessar o TR Transfer sem problemas. Caso o problema persista, é recomendado entrar em contato com o suporte técnico para uma análise mais aprofundada.
