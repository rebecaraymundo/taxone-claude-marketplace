# [Municipal][ISS.WEB] TAX ONE - Ao gerar o Arquivo Magnético do ISS.WEB para Sertãozinho está levando notas de outros períodos, como resolver?

Caso enfrente problemas com a geração do arquivo magnético do ISS.WEB para o município de Sertãozinho, onde o cabeçalho e notas do arquivo gerado para o período de 07/2024 estão incorretamente mostrando informações do período de 05/2024. A causa do problema pode estar na configuração de geração do arquivo, onde a opção "Gerar Arquivos por Data de Emissão" não foi marcada, fazendo com que o sistema considere erroneamente as datas de emissão das notas ao invés do período selecionado para geração do arquivo. Para resolver, siga as orientações:

  1. Identifique a Nota de Serviço em Básicos > Data Warehouse > Manutenção > Documento Fiscal > Novo Documento Fiscal > Doc. Fiscal de Mercadoria e Serviço.
  2. Equalize no período desejado (07/2024).
  3. Gere o arquivo com a data desejada (07/2024) marcando a opção "Gerar Arquivos por Data de Emissão".
  4. Após seguir os passos de resolução, é importante realizar uma verificação detalhada do arquivo gerado para garantir que todas as informações estão corretas. Caso o problema persista, recomenda-se entrar em contato com o suporte técnico para uma investigação mais profunda.


**Nota:** É recomendável sempre verificar as opções de configuração antes de gerar arquivos importantes como o ISS.WEB. Certificar-se de que todas as configurações estão corretas pode evitar erros e a necessidade de regeneração de arquivos, economizando tempo e recursos.
