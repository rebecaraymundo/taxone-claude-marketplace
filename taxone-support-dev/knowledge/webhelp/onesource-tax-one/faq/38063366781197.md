# [Interfaces] [SAP – Namespace/OBTI] [Parametros] Erro de Conexão na Interface Namespace com Banco a Banco

Problema:

Erro identificado na interface Namespace após a mudança do banco de dados para o Taxone, resultando em alertas de conexão inexistente no banco de dados (conexão banco a banco - DBCO) que podem gerar confusões, mas não impactam no desempenho operacional do sistema.

Causa Raiz:

A causa do problema está na tentativa do sistema de verificar uma conexão com um banco de dados que não está mais em uso (pela Namespace), mas o sistema continua a verificar se há conexão, resultando no erro exibido na tela de Parâmetros Gerais, aba Configuração.

Resolução:

Para resolver e evitar que o erro de conexão continue aparecendo, siga os passos abaixo:

Acesse a interface, utilize a transação /n/msafx/parametros aba configuração.

Identifique as configurações que fazem referência ao banco de dados antigo, caso esta tenha sido modificada atualizar o endereço, se não utilizarem mais remover o endereço.

Salve as alterações realizadas e reinicie o sistema para verificar se o erro foi removido.

Próximos Passos:

Após realizar as alterações recomendadas, monitore o sistema para garantir que o erro não reaparece. Caso o erro persista ou você encontre dificuldades para alterar as configurações, entre em contato com o suporte técnico para assistência adicional.

Dicas e Recomendações:

Recomenda-se revisar regularmente as configurações do sistema para garantir que todas as conexões e referências a bancos de dados e outros serviços sejam atualizadas conforme mudanças ocorram no ambiente de TI. Isso evita alertas desnecessários e otimiza o desempenho do sistema.

Para mais informações ou assistência, não hesite em contactar o nosso suporte técnico.
