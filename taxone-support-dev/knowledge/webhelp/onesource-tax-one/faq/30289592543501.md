# [Básicos][Job Servidor] TAX ONE - Como resolver erro na Carga Automática SAFX08 devido ao uso de caractere especial?

Caso enfrente um erro denominado "INDICE_PROD_ACAB" durante a carga automática SAFX08, mesmo com o campo de caracteres dentro do limite permitido. Este erro pode ocorrer devido à presença de um caractere especial (pipe "|") no campo "DSC_RESERVADO1". Esse caractere é interpretado como um delimitador durante a importação, causando deslocamento das informações para campos subsequentes e resultando em erro. Para resolver, siga as orientações:

  1. Verifique todos os campos de entrada de dados na carga SAFX08.
  2. Remova o caractere pipe "|" ou qualquer outro caractere especial que possa estar sendo utilizado indevidamente.
  3. Consulte a aba de considerações no manual de layout para verificar quais caracteres são proibidos.
  4. Realize nova carga para verificar se o erro persiste.
