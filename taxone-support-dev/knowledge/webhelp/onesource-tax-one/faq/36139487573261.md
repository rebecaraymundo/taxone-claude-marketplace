## Resolução de Problemas de Rejeição de Arquivos TXT no SFTP Devido a Incompatibilidade de Formatos

### Problema:

Arquivos .txt são rejeitados pelo SFTP durante o processo de integração, apesar de conterem dados válidos.

### Causa Raiz:

A causa do problema foi identificada como uma incompatibilidade de formato. Os arquivos estão formatados para Unix, enquanto o sistema requer arquivos em formato Windows.

### Resolução:

  * Converter os arquivos .txt para o formato Windows.
  * Na opção de formatação, ajustar para a codificação ANSI.
  * Utilizar ferramentas adequadas, como, por exemplo, o Notepad++, para garantir que as configurações estejam corretas.


### Dicas e Recomendações:

Para evitar problemas de compatibilidade no futuro, sempre verifique o formato de arquivo exigido antes de iniciar o processo de integração. Consulte o manual do sistema, que define o formato de codificação padrão para a montagem de arquivos TXT como ANSI.

Utilizar ferramentas como o Notepad++ pode ser uma opção prática para verificação e ajuste da codificação e do formato dos documentos antes do envio.
