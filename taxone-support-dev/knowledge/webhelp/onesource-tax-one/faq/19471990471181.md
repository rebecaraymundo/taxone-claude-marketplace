Normalmente, esse problema ocorre porque os campos VLR_LANCTO, VLR_OPER_IND, DSC_RESERVADO1, DSC_RESERVADO2, DSC_RESERVADO3 possuem algum caractere que não é numérico.

Para corrigir o erro 90461: 'Valor Decimal ou Numerico com formato invalido.', siga a orientação abaixo:

  * Informe um caractere numérico válido ou informe @ para que o campo seja nulo.
