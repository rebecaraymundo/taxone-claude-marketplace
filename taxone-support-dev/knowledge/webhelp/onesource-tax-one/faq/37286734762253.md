Problema

Ocorreram problemas no processo de Instalar/ Atualizar a versão, após aplicadas as Requests algumas funcionalidades passaram a ficar não disponíveis e a versão ativa não mostrada corretamente na Transação MSAFX/SAFTR001A

Causa Raiz

A causa do problema foi a aplicação incorreta da ordem das requests. A Request de Customizing e a de WebServices foram aplicadas antes da Request Principal;

Resolução

Aplicar primeiramente a Request Principal (TRQ).

Em seguida, aplicar a Request de Customizing (TDD).

Por último, aplicar a Request de WebServices.

Próximos Passos

Após aplicar as Requests na ordem correta, execute os testes para garantir que todas as funcionalidades estão operando como previsto. Caso encontre alguma probleam não hesite em abrir um novo ticket de suporte.

Dicas e Recomendações

Antes de aplicar as Requests, confira a ordem correta indicada no documento de orientação disponibilizado no portal e salve regularmente backups do seu ambiente para evitar perda de dados em caso de falhas. Certifique-se de verificar a versão mais atualizada dos documentos de orientação antes de proceder com operações semelhantes no futuro.
