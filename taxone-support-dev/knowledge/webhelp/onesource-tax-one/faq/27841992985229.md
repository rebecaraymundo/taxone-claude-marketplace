Os códigos de receita são inseridos no sistema TAX ONE através da tabela TACES14, que é atualizada pelo cliente.

Passo a passo para fazer a inclusão do código de receita 0588-07 no sistema TAX ONE:

  1. Acesse o módulo '**Ferramentas** ' no sistema TAX ONE.
  2. Selecione a opção '**Tabelas Internas** ' e clique em '**Manter** '.
  3. Escolha a categoria '**Tabelas** ' e depois '**Código de Receita** '.
  4. Na lista de códigos de receita, adicione manualmente o código **0588-07.**
  5. Após inserir todas as informações, salve as alterações para atualizar o sistema.


Próximos passos:

  1. Verifique se o código**0588-07** está disponível para seleção ao realizar lançamentos no sistema.
  2. Caso o problema persista mesmo após a inclusão manual, entre em contato com o suporte técnico para obter assistência adicional.


Dicas e recomendações:

  * Mantenha a tabela de códigos de receita (TACES14) atualizada regularmente para evitar problemas de ausência de códigos.
  * Realize revisões periódicas dos códigos utilizados em sua empresa para garantir que todos estejam disponíveis no sistema TAX ONE.
