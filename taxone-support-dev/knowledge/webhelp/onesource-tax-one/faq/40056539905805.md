##

## Problema

Os registros de notas fiscais canceladas estão aparecendo como duplicados no sistema Mastersaf DW/TAXONE, mesmo estando configurado para não transmitir notas canceladas.

##

## Causa Raiz

A aparição de registros duplicados pode ser resultado de uma configuração inadequada nas interfaces do sistema ou na extração de dados, que não distingue corretamente o status das notas fiscais canceladas.

##

## Resolução

  1. Acesse a transação **/n/MSAFX/PARAMETROS** no sistema.
  2. Navegue até **Parametrização da Empresa** > **Nota Fiscal**.
  3. Habilite o parâmetro "**NF Est Recente Dupl (ZSAFE026)"** para realizar a extração apenas de notas ativas.
  4. Realize uma nova carga de dados para ver se o problema de duplicidade foi resolvido.


##

### Próximos Passos

Após realizar as configurações, é recomendado fazer um teste de extração de dados para garantir que as notas fiscais canceladas não estão mais sendo consideradas, evitando duplicidade no sistema. Monitore os próximos ciclos de transmissão para confirmar a solução do problema.

##

### Dicas e Recomendações

Assegure-se de revisar periodicamente as configurações de parâmetros no sistema para evitar futuras inconsistências nos registros. Considere a consulta com a equipe de suporte sempre que houver dúvidas ou novos problemas surgirem.
