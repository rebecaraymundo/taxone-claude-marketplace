## Problema:

O usuário está enfrentando um erro de importação na tabela SAFX2013, indicando que o CEST não existe, embora esteja presente na tabela de SPED.

## Causa-raiz:

O erro ocorre devido à necessidade de manutenção da tabela acessória TACES94 pelo cliente ou consultoria. A tabela TACES94 facilita a implantação de produtos através de cadastros básicos iniciais, mas requer atualizações manuais dos códigos CEST pelo cliente.

## Resolução:

Passo a passo para resolver o erro de importação na tabela SAFX2013:

  1. Acesse o módulo Básicos, menu Ferramentas > Tabelas Internas > Manter Tabelas > **Código Especificador da Substituição Tributária**.
  2. Realize o cadastro manual dos códigos CEST conforme necessário.
  3. Faça um novo teste de importação na tabela SAFX2013 e verifique se o erro foi resolvido.


## Próximos passos:

Após seguir as instruções de resolução, monitore o sistema para garantir que o erro não se repita. Caso o problema persista, entre em contato com o suporte técnico para assistência adicional.

## Dicas e recomendações:

É recomendado revisar e atualizar regularmente as tabelas acessórias para evitar problemas de importação no futuro. Mantenha-se atento às atualizações do sistema e participe de treinamentos oferecidos para melhorar o gerenciamento das tabelas internas.
