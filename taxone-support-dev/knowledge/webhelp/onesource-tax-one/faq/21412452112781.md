Passo a passo para geração do registro 1220 da DIMP:

1\. Acesse o módulo DIMP, menu Parâmetros > Movimento DIMP para realizar o **cadastro da movimentação financeira** (SAFX313).

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21412428688269)

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21412460034573)

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21412444212109)

**Nota:** O registro 1220, filho do registro 1200, não deve ser gerado para cancelamentos dentro do período de geração. A data de cancelamento deve ser no mês posterior à data da operação.

2\. Acesse o módulo DIMP, menu Geração > **Geração do Meio Magnético** para gerar o arquivo da DIMP.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21412452100621)

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21412416124557)

**ARQUIVO GERADO**

**Nota: **Será gerado um arquivo para cada UF à qual pertence o cliente.
