## **Configurando Inscrições Municipais para Geração do GISSONLINE em Diferentes Municípios**

### **Problema**

Como gerar o arquivo **GISSONLINE** para o município de **Diadema** , mesmo quando a filial possui inscrição municipal nesse município, mas está cadastrada com sede em **São Paulo**.

* * *

### **Causa Raiz**

A configuração do sistema não estava ajustada para permitir a geração de documentos fiscais para municípios diferentes do município cadastrado como sede da filial (São Paulo).

* * *

### **Resolução**

Para corrigir o problema e habilitar a geração do arquivo para Diadema, siga os passos abaixo:

  1. **Acesse o sistema** pelo caminho:
**Básicos → Data Warehouse → Manutenção → Cadastros → Inscrições Municipais**.
  2. **Localize a inscrição municipal** referente ao município de **Diadema**.
  3. **Configure corretamente os dados** :
     * Informe a inscrição municipal válida para Diadema.
     * Verifique se os campos obrigatórios estão preenchidos (CNPJ, município, regime, etc.).
  4. **Salve as alterações** e confirme que a inscrição está ativa para permitir a geração do arquivo.
  5. **Gere o arquivo GISSONLINE** selecionando o município de Diadema no processo de emissão.

Verifique se no módulo GISS online está selecionada a opção para geração no menu de Construção Civil, se aplicável.

Execute a geração do arquivo para o município desejado.

### **Próximos Passos**

Após realizar as alterações e gerar o arquivo, confirme se o documento foi criado corretamente e aceito pela **Prefeitura de Diadema**.
Caso identifique problemas:

  * Revise as configurações para garantir que todos os dados estão corretos.
  * Se o erro persistir, entre em contato com o **suporte técnico** , informando os detalhes da configuração e o arquivo gerado.


* * *

### **Dicas e Recomendações**

  * Revise periodicamente as **inscrições municipais** cadastradas no sistema para garantir que estão atualizadas e corretas, especialmente quando a filial atua em múltiplos municípios.
  * Essa prática ajuda a evitar falhas na geração de documentos fiscais e garante conformidade com as exigências municipais.
