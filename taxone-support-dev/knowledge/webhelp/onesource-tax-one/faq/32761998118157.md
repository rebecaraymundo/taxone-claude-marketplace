# Solução de Erro ao Iniciar Serviços TR Transfer Files Após Atualização do Java

## Problema:

O serviço TR Transfer Files não está reiniciando após uma atualização do Java realizada para a solução de um problema anterior relacionado à atualização de um patch.

## Causa Raiz:

A causa raiz do problema foi a incompatibilidade entre a nova versão do Java instalada e o serviço TR Transfer Files, que resultou em falhas ao tentar reiniciar o serviço.

## Resolução:

  * Reinicie a máquina onde o serviço TR Transfer Files está instalado.


  * Reinstale o serviço TR Transfer Files utilizando a nova versão do Java. Certifique-se de baixar o pacote de instalação atualizado do serviço através do link oficial: [Amazon Corretto 17](<https://docs.aws.amazon.com/corretto/latest/corretto-17-ug/downloads-list.html>).


  * Após a reinstalação, tente iniciar novamente o serviço TR Transfer Files.
