Para corrigir o erro **90163 - O código NBM não está cadastrado,** faça o seguinte

  1. Acesse o [Básicos > Data Warehouse > Manutenção > Códigos](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Basicos/Basicos_Data_Warehouse/Basicos_Data_Warehouse_Menu_Manutencao/Basicos_DW_Menu_Manutencao_Codigos.htm>).

  2. Cadastre o código **NBM**.
