Passo a passo para resolver o problema de não gravar registros na SAFX13:

  1. São necessárias duas configurações para esse cenário:

     * PARAMETRIZAÇÃO DADOS ORIGEM > ATIVO FIXO> LIVROS DO FA

     * PARAMETRIZAÇÃO DADOS ORIGEM > GL > LIVRO DO GL

  2. No Interdados, acesse _PARAMETRIZAÇÃO DADOS ORIGEM > ATIVO FIXO > LIVROS DO FA._
     * Nesta tela, faça a parametrização onde o código é o tipo do livro. Esta informação está disponível na origem do dado ou na tag BOOK_TYPE_CODE no XML gerado pelo extrator.
  3. Em seguida, acesse a parametrização no menu _PARAMETRIZAÇÃO DADOS ORIGEM > GL > LIVRO DO GL._
     * Nesta tela, é necessário parametrizar **apenas um livro GL no cadastro.** De acordo com a documentação da Oracle, somente um livro é necessário.
     * Caso o livro utilizado seja diferente do cadastrado, é necessária a parametrização do novo livro, deixando apenas um na tela como o livro principal.


Após realizar essas configurações nas parametrizações de dados origem para **Ativo Fixo** e **GL** , os registros da SAFX13 devem ser gravados corretamente durante a carga.
