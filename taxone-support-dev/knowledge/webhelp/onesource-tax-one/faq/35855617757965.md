# Extração de Relatório de Chave de Acesso dos Documentos na EFD

## Problema:

Os usuários necessitam extrair um relatório de chave de acesso dos documentos contidos na Escrituração Fiscal Digital (EFD), especificamente do Registro C100, para confrontar as chaves de acesso com a base de dados antes de gerar o meio magnético.

## Resolução:

  * Acesse o menu: Básicos > Data WareHouse > Relatórios > Conferências > Chave dos Documentos Fiscais Eletrônicos
