Para entender qual data considerar para os campos DATA_INI e DATA_FIM na tabela SAFX278, siga as orientações:

  1. Acesse a tabela SAFX278 no sistema.
  2. Para o campo DATA_INI, considere a data em que a parametrização do registro 0221 foi iniciada.
  3. Para o campo DATA_FIM, considere a data em que a parametrização do registro 0221 foi concluída.
  4. Salve as alterações e verifique se as datas estão corretas.
