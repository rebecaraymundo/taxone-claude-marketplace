## Problema:

O processo de importação de dados via OBI é concluído com sucesso, mas ao consultar o dashboard do tax automation, o integrador apresenta um erro de "estouro de campo" e o registro não é importado.

## Causa Raiz:

O erro ocorre devido à configuração incorreta dos delimitadores nos perfis de entrada e saída no OBI, que não correspondem ao formato esperado pelo sistema.

## Resolução:

  1. Acesse o cadastro do perfil de entrada associado ao processamento dos dados.
  2. Altere o delimitador do perfil de acordo com o delimitador do arquivo que será carregado no OBI. Exemplo TAB, XML, PIPE.
  3. Verifique e ajuste o delimitador do perfil de saída para "PIPE", que é o padrão para integração com o Tax One.
  4. Realize um novo teste de importação para confirmar que o erro foi resolvido.


## Próximos Passos:

Após ajustar os delimitadores e realizar a importação, verifique se o registro é importado com sucesso. Caso o erro persista, entre em contato com o suporte técnico para uma análise mais detalhada.
