# [Federal][CREDPIS]TAX ONE - Como resolver problemas com a geração de campos no módulo Credpis para os registros F120/F130?

O campo Conta Depreciação (Gr./Cód./Val.) dentro do cadastro do bem patrimonial é utilizado pelo módulo CREDPIS para inserir a conta contábil na tela de Bens Incorporados ao Ativo Imobilizado (F120/F130) do SPED Contribuições. Se o campo aparecer em branco, siga as orientações:

  1. Acesse o sistema e vá até a tela de cadastro do bem patrimonial (SAFX13) e informe a conta contábil no campo conta depreciação.
  2. Realize uma nova geração no módulo Credpis e verifique se os campos no registro F130 estão preenchidos corretamente.


**Atenção:**

1\. Após realizar as correções, monitore os próximos registros gerados para garantir a integridade dos dados. Caso o problema persista, entre em contato com o suporte técnico fornecendo as evidências necessárias para uma análise mais profunda.

2\. É recomendável revisar periodicamente o cadastro dos bens patrimoniais para evitar problemas futuros nas gerações de registros. Mantenha-se sempre atento às instruções detalhadas nos manuais do sistema e consulte a Central de Ajuda para artigos e guias que possam auxiliá-lo.
