Normalmente, esse problema ocorre porque o código informado no campo CENTRO_CUSTO não está cadastro na tela Data Warehouse > Manutenção > Códigos > Centro de Custo. Para corrigir o erro 90381: 'O Campo Código de Centro de Custo não esta cadastrado.', siga as orientações abaixo:

  1. Informe um código valido no arquivo TXT, ou
  2. Cadastre o código informado no arquivo TXT na tela Básicos > Data Warehouse > Manutenção > Códigos > Centro de Custo.
