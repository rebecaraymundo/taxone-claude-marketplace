Normalmente, esse erro ocorre devido ao campo DATA_OPERACAO estar com valor @, em branco ou 00000000.

Para corrigir o erro 90009: 'O Campo Data de Movimento nao esta preenchido ou preenchido com zeros.', siga a orientação abaixo:

  * Informe uma data no formato AAAAMMDD.
