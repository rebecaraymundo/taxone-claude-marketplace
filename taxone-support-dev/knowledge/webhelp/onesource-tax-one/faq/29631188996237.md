Caso enfrente dificuldade em emitir o relatório de notas de serviço após as recentes alterações na plataforma, apresentando o erro "não há processo para a data" ao tentar a emissão. Isso pode ocorrer devido a falta de correspondência entre as datas de geração e de emissão do relatório. Para resolver, siga as orientações:

Nota: O sistema requer que a data de emissão seja a mesma da geração do relatório.

  1. Em Gerenciais > Documentos Fiscais Analíticos > Conferência de Documentos Fiscais (INSS/ISS/PIS/COFINS).
  2. Selecione a opção "Geração" para criar um novo relatório.
  3. Após a geração, vá para a opção "Emissão".
  4. No campo data, insira a mesma data em que você gerou o relatório. Por exemplo, se você gerou o relatório em 05/08/2024, você deve inserir esta mesma data na emissão.
  5. Prossiga com a emissão do relatório.
  6. Após seguir os passos acima, verifique se o relatório foi emitido corretamente. Se o problema persistir, entre em contato com o suporte técnico para uma análise mais detalhada.


**Importante:** É recomendável sempre verificar as notas de atualização do sistema para se manter informado sobre quaisquer mudanças que possam afetar os processos que você realiza frequentemente. Além disso, garantir que as datas de geração e emissão sejam sempre correspondentes pode evitar problemas na emissão de relatórios.
