Passo a passo para resolver o problema de tela travando ao tentar abrir um registro de controle de tributos com caracteres especiais (acentos, cedilhas, parênteses, chaves, etc.):

Exemplo:
![](https://suporte.thomsonreuters.com.br/hc/article_attachments/26771507334413)

Este registrou subiu para o sistema, porém não é possível abri-lo em tela para fazer alteração.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/26771507335053)

  1. A solução para este tipo de caso é realizar a exclusão da tabela. Sugerimos que seja feita a extração da safx do período antes da exclusão.
  2. Após obter a SAFX, realize a parametrização da exclusão apenas para o dia daquele registro. Assim, a tabela ficará limpa para este período, permitindo que você reimporte a SAFX e ajuste conforme a necessidade.
