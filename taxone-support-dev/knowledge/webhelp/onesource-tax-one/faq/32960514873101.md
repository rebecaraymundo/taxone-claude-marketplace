# Resolução de Erro MS0029 Durante Envio de Eventos

## Problema:

Usuários enfrentam a rejeição de eventos de fechamento dos períodos com o erro MS0029, indicando duplicidade de débitos e que a apuração foi enviada por outra origem.

## Causa Raiz:

O erro MS0029 é uma mensagem do Governo indicando uma instabilidade no web-service do REINF, que é responsável por recepcionar o evento enviado pelo usuário. A duplicidade indicada não está relacionada a uma ação do usuário, mas sim a problemas de instabilidade do sistema governamental.

## Resolução:

  * Aguarde alguns minutos antes de tentar enviar o evento novamente.


  * Se o problema persistir, tente gerar novamente o R-2099.


  * Se o erro continuar, entre em contato com o SEFAZ através do "Fale Conosco" do REINF para mais assistência.


## Próximos Passos:

Após seguir os passos de resolução, monitore o sistema para verificar se o problema foi resolvido.

## Dicas e Recomendações:

É recomendável verificar frequentemente a estabilidade dos serviços do REINF através dos canais oficiais do Governo para evitar enviar eventos durante períodos de instabilidade conhecida.
