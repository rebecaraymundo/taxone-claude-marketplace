## Introdução

 _Esse artigo tem por objetivo mostrar os passos para solucionar o problema de falta de códigos de contribuição para geração do Bloco M_

 __**Os códigos precisam ser importados via TACES73, esta TACES é de responsabilidade do usuário importar.**__

## Passo-a passo

**1 - Baixar a TACES73 na Base de Conhecimento do ZENDESK**

Link: [Download Tabelas TACES](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/WebHelp/mergedProjects/Webhelp_ONESOURCE_TAXONE/inicio/downloads.htm>)

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/37247103366925)

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/37247054911245)

**2 - Realizar a importação da mesma no SFTP da empresa:**

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/37247103367437)

**3 - Realizar a importação no TAX ONE**

**Acessar Módulo Ferramentas > Tabelas Internas> Importar> Tabelas Acessórias**

**![](https://suporte.thomsonreuters.com.br/hc/article_attachments/37247054912909)**

**![](https://suporte.thomsonreuters.com.br/hc/article_attachments/37247054913677)**

**4 - Após importado verificar se os códigos estão disponíveis no Módulo SPED Contribuições**

**Acessar Módulo SPED Contribuições: Parâmetros > Apuração> Identificador dos Códigos de Contribuição**

**![](https://suporte.thomsonreuters.com.br/hc/article_attachments/37247054914573)**
