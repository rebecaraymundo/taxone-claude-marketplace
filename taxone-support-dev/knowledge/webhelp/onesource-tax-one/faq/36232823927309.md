# Duplicação de Parametrização do Plano Referencial da versão anterior para atual

## Problema:

Usuário deseja duplicar a parametrização do Plano Referencial x Plano de Contas de uma versão anterior para uma nova versão, sem ter que vincular manualmente conta a conta devido ao volume de informações.

## Causa Raiz:

O produto não possui uma funcionalidade de cópia ou duplicação automática de configurações entre diferentes versões, sendo necessária a realização de um processo manual ou ajuste através de importação.

## Resolução:

  * Acesse a versão 10 (exemplo) do plano referencial e exporte o arquivo SAFX2101, que contém as configurações desejadas.


  * Verifique o arquivo exportado para identificar se é necessário ajustar ou incluir novas contas, principalmente por conta de possíveis atualizações no layout pelo governo.


  * Importe o arquivo SAFX2101 ajustado para a versão 11 (exemplo) do novo plano referencial.


## Próximos Passos:

Após importar o arquivo, verifique se todas as contas e parametrizações estão corretas e funcionais. Realize testes de processamento para garantir que todos os dados estejam sendo utilizados conforme esperado. Se encontrar discrepâncias, ajuste manualmente ou entre em contato com o suporte técnico para apoio.

## Dicas e Recomendações:

Mantenha o registro das modificações realizadas durante o processo de importação e exportação para facilitar a resolução de possíveis problemas futuros.
