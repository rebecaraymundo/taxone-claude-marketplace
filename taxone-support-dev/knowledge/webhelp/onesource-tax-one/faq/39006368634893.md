## Problema

Os registros M100 e M500 gerados no EFD Contribuições estão apresentando os valores zerados, mesmo depois de realizar os procedimentos iniciais recomendados.

## Causa Raiz

O problema ocorre devido à configuração do método de apropriação de créditos nos dados iniciais. Quando o método de rateio é escolhido, o registro 0111 torna-se obrigatório. Caso os valores não sejam informados na tela "Rateio de Créditos Comuns", isso impactará diretamente na geração dos registros M100 e M500.

## Resolução

Siga os passos abaixo para corrigir o problema:

  1. Confirme as configurações nos parâmetros:
     * Em SPED EFD - Contribuições Parâmetros Perfil, verifique se os registros estão devidamente marcados.
     * No módulo Básicos Parâmetros Preliminares Empresa/Estabelecimento, na aba "Informações Específicas", certifique-se de que o campo "Natureza da Pessoa Jurídica" está preenchido no cadastro do estabelecimento/empresa.
  2. Verifique o método de apropriação de créditos nos dados iniciais:
     1. Confirme se o método de rateio foi selecionado.
     2. Se o método de rateio estiver ativo, certifique-se de que as informações foram inseridas na tela "Rateio de Créditos Comuns".
  3. Para escriturar os valores de rateio:
     * Acesse o menu: **Manutenção Receita Bruta Mensal - Rateio de Créditos Comuns**.
     * Insira os valores adequados e salve as configurações.
  4. Gerar os registros novamente no EFD Contribuições e verificar se os valores foram preenchidos corretamente.


## Próximos Passos

Após realizar os procedimentos acima:

  * Revisar os dados gerados para garantir que os valores nos registros M100 e M500 estejam corretos.
  * Realizar testes adicionais para confirmar que o problema foi resolvido completamente.
  * Caso persista alguma dúvida ou o problema não seja resolvido, entre em contato com o suporte técnico para assistência adicional.


## Dicas e Recomendações

  * Certifique-se de realizar escrutínios regulares no método de apropriação de créditos, garantindo que os valores necessários sejam informados corretamente.
  * Siga o guia prático do EFD Contribuições disponível no site oficial para entender melhor as configurações necessárias.
  * Utilize ferramentas de monitoramento para verificar a consistência dos dados configurados e evitar futuros problemas nos registros.
  * Acesse artigos adicionais ou KBA's relacionados ao EFD Contribuições na base de conhecimento da Thomson Reuters.
