**Problema:**

Alguns documentos não estão enviando corretamente os valores de base e retenção para o REINF, especificamente para o código de receita 8045.

**Causa Raiz:**

A natureza de rendimento válida para o código DARF 804506 é a 20010, conforme a tabela MOR da RFB (Receita Federal do Brasil).

**Resolução:**

Acesse o produto TAX One via tela Controle de Tributos.

Verifique a tabela MOR para confirmar que a natureza de rendimento correspondente ao código DARF 804506 é 20010.

Cadastre a natureza de rendimento correta no documento relevante.

Envie novamente o documento para o REINF.

**Próximos Passos:**

Após realizar os passos para correção, monitore a submissão do documento para garantir que os valores de base e retenção estão sendo enviados corretamente.

**Dicas e Recomendações:**

Para evitar problemas futuros com o envio de documentos para o REINF, sempre confira as tabelas oficiais da RFB relacionadas ao código de receita antes de realizar o cadastro de documentos. Mantenha-se atualizado com as últimas versões do software e documentações disponíveis na central de ajuda do Onesource Tax One.
