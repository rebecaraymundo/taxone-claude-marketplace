# [Básicos][Job Servidor] TAX ONE - Resolução de Duplicidade de Itens na Importação devido a Divergências nos Campos de Unidade de Medida

## Problema:

Durante a importação da SAFX08, milhares de itens foram duplicados no ambiente UAT devido a divergências nos campos relacionados à Unidade de Medida.

## Causa Raiz:

A causa raiz do problema foi identificada como divergências nos campos de Unidade de Medida entre diferentes itens enviados. O sistema considera esses campos na chave DISCRI_ITEM, que inclui COD_PRODUTO, NUM_ITEM e COD_UND_PADRAO, para determinar a unicidade dos itens.

## Resolução:

  * Verifique os campos de Unidade de Medida nos itens a serem importados e certifique-se de que eles estão uniformizados.


  * Se houver divergências, padronize os dados antes de realizar a importação para evitar duplicidades.


  * Confirme a correção fazendo uma nova importação no ambiente de teste antes de movê-la para o ambiente de produção.


## Próximos Passos:

Após implementar a solução, monitore os resultados das próximas importações para garantir que o problema de duplicidade não ocorra novamente. Caso o problema persista, entre em contato com o suporte técnico para investigação adicional.

## Dicas e Recomendações:

Considere a revisão regular dos campos de Unidade de Medida em seus dados para garantir a consistência. Isso ajudará a prevenir problemas de duplicidade e melhorará a integridade dos dados no sistema.
