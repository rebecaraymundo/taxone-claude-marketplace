Para corrigir o erro 90366: 'O Campo Codigo NCM nao esta cadastrado', verifique se o Código de NCM (COD_NBM) preenchido na SAFX13 está cadastrada no menu abaixo, com uma data de validade igual ou maior que a data do registro na SAFX13:

[Básicos > Data Warehouse > Manutenção > Códigos > Código NCM](<../Manuais_Funcionais/Basicos/Basicos_Data_Warehouse/Basicos_Data_Warehouse_Menu_Manutencao/Basicos_DW_Menu_Manutencao_Codigos.htm> "Hyperlink \[../Manuais_Funcionais/Basicos/Basicos_Data_Warehouse/Basicos_Data_Warehouse_Menu_Manutencao/Basicos_DW_Menu_Manutencao_Codigos.htm\]")
