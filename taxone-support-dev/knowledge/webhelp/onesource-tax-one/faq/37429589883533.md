# [Interfaces] [SAP – Namespace/OBTI] [Atualização de patch] Ajustes Necessários na Transição da Versão 40XX para 41XX

- **Criado:** 2025-06-17
- **Atualizado:** 2025-10-15

---

1\. Após a aplicação das requests de atualização para a **versão v4100** , a tela de versão ativa ainda exibirá a **versão 4099** como a ativa.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/37442641085197)

Para alterar essa informação, clique no botão '**Ativo** ' da **versão 4100** e depois em '**SALVAR** '. Em seguida, saia da transação e entre novamente.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/37442819461005)

**2.** Com a atualização para a versão V4100, os clientes que realizarem a atualização do pacote devem revisar as **VARIANTES** criadas para execução via JOB (SM36) no programa /MSAFX/SAFRP002. No campo P_VERS, é necessário atualizar para 41.

Não é necessário recriar a variante; basta atualizá-la para a versão atual.
