O **Calendário de Fábrica** impacta o campo DATA_VENCTO da tabela SAFX53, que é a data de vencimento. Portanto, é importante manter o calendário atualizado para garantir o cálculo correto das datas de vencimento.

Passo a passo para verificar e alterar o Calendário de Fábrica no SAP:

  1. Acesse a transação standard **SCAL**.
  2. Selecione a opção **"Calendário Fábrica"** e clique no **ícone de lápis** para editar.
  3. Verifique se as datas para o calendário utilizado não estão vencidas. Por exemplo, para o "BR - Calendário Brasileiro".
  4. Caso o calendário esteja vencido, selecione o **ID** correspondente e clique no**ícone de lápis** para modificar.
  5. Modifique o campo **"Válido até"** para a nova data desejada.
  6. Salve as alterações.
