**Objetivo:** Este processo é destinado exclusivamente para situações em que a tomada de crédito ocorre imediatamente após a entrada do bem no estabelecimento.

É necessário conter informações na **SAFX13** e **SAFX82** , sendo que é possível importar essas SAFX caso não deseje preenchê-las manualmente.

  * **SAFX13** (Arquivo de Cadastro de Bem), no agrupamento Básicos > módulo Data Warehouse > Manutenção > Cadastros > Bem Patrimonial
  * **SAFX82** (Arquivo Bens do Ativo Permanente - CIAP), no agrupamento Estadual > Ativo Permanente > Movimento > Manutenção dos Movimentos > Aquisições

**SAFX82 - Aquisições: Informações essenciais e funcionalidades**

Ao lidar com o arquivo SAFX82 - Aquisições, é essencial incluir informações críticas que são fundamentais para o cálculo adequado:

  * Tipo de Pessoa (Física ou Jurídica)
  * Código Fiscal
  * Modelo do Documento
  * Produto



Esses campos são indispensáveis para o processo de cálculo e devem ser preenchidos corretamente para garantir a precisão dos resultados.

No módulo de aquisições, os usuários podem:

  * Incluir novos dados de bens adquiridos
  * Alterar informações existentes
  * Excluir dados conforme necessário



Além disso, o sistema permite a consulta de baixas realizadas, facilitando o acompanhamento e a gestão dos bens.

**Apenas para conhecimento:** No módulo de alienações, é possível efetuar a baixa (total ou parcial) dos bens adquiridos. As baixas podem ocorrer por diversos motivos, como vendas, perecimento, extravio, entre outros. É essencial que essas ocorrências sejam registradas através da rotina específica para este fim. Para que o registro de uma alienação seja possível, é necessário que os bens já estejam devidamente cadastrados na tabela de aquisições. Este pré-requisito assegura a precisão e a integridade dos dados no sistema.

  * **Consulta de alienação:** Para consultar informações sobre uma alienação já registrada, clique em "ABRE".
  * **Cadastro de Novas Baixas:** Para registrar uma nova baixa, clique em Novo na barra de menu ou diretamente na pasta do campo **Identificador CIAP**.
**Nota** : Esta manutenção é referente à tabela de Arquivo Alienações Bens do Ativo Permanente - CIAP (SAFX83).


**Modelo de cálculo geral dos créditos**

Nesta rotina, há diversas funcionalidades essenciais para o gerenciamento eficaz dos bens do Ativo Permanente. As operações disponíveis incluem:

  * Inclusão: Adicionar novos registros ao cadastro de bens.
  * Alteração: Atualizar informações de bens já cadastrados.
  * Consulta: Visualizar detalhes dos bens registrados.
  * Além do gerenciamento de bens, esta rotina também abrange o registro das alienações realizadas em cada período de apuração. Isso permite uma visão completa e integrada das movimentações de ativos dentro do período especificado.


**Considerações importantes para geração de créditos:**

É importante ter a aquisição ativa desde o início do período de cálculo até o seu término. No exemplo dado, a aquisição deve estar ativa de 11/2022 a 11/2024.

Para fazer essa configuração, acesse o menu Movimento > Manutenção Dados de Cálculo > Modelo C > Modelo Cálculo Geral dos Créditos.

Para alimentar essa tela, é necessário realizar o Calculo do crédito - ICMS
