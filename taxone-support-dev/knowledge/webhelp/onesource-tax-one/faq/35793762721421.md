# [Básicos][Report Fiscal] TAX ONE -  Resolução de Erro em Relatório em Branco devido à Falta de Dados no Período Selecionado

#### Problema:

Usuário enfrenta o problema de gerar um arquivo de relatório que aparece em branco, mesmo após ajuste do mês de emissão para o período correto.

#### Causa Raiz:

A causa do problema foi a seleção de um período para o qual não havia dados registrados no sistema. Por isso, o relatório gerado não continha informações.

#### Resolução:

Após a análise via acesso remoto, verificou-se que o período correto com registros disponíveis é a partir de 12/2024. Segue o passo a passo para a solução:

  * Verifique a existência de registros para o período desejado no sistema.
  * Ajuste o mês de emissão do relatório para um período que contenha dados.
  * Regenere o relatório. Se o período escolhido for correto, o relatório deverá ser preenchido com as informações esperadas.
