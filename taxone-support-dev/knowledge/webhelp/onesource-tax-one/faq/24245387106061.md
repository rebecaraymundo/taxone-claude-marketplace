Passo a passo para executar uma carga de forma fracionada:

  1. Identifique o processamento a ser efetuado. Por exemplo, para o procedimento SAFX01 - LANÇAMENTOS CONTÁBEIS, acesse "Procedimentos" > "Nome do Procedimento: SAFX01 - LANÇAMENTOS CONTÁBEIS", clique em "MAIS" e então em "Bip Definições".

  2. Clique na aba "TEMPLATE".

  3. Localize a linha "SQLFINAL" na coluna "Chave Template" e clique em "Sql Template" (Nos '...').

  4. Insira o seguinte código:


SELECT Q.*

FROM (

SELECT ROW_NUMBER() OVER (ORDER BY PST_ID) AS NUMB

, x.*

from r_ked0001_20 x

) Q

WHERE NUMB BETWEEN 0 AND 50000 ORDER BY PST_ID

  * O comando deve ser alterado conforme a carga, por exemplo:


BETWEEN 0 AND 50000

  * Esse comando fará a carga do registro 0 até o 50000.


  1. Após a carga dessa forma, para carregar os próximos 50000 registros, altere o código conforme o exemplo:


BETWEEN 50001 AND 100000

  1. Após o término das cargas, retorne com o comando padrão:


SELECT * FROM r_ked0001_20
