# Resolução de Problemas de Acesso ao Transfer Files Após Atualização e Instalação do Java

## Problema:

Após a migração do ambiente e instalação da nova versão 3.0.2 do Transfer Files junto com o Java Amazon correto 17, a página de acesso ao sistema não está funcionando e nenhum erro é identificado nos logs.

## Causa Raiz:

A causa identificada para o problema de acesso foi a presença de variáveis de ambiente desatualizadas no JAVA_HOME, que ainda apontavam para a versão antiga do Java instalada na máquina.

## Resolução:

  * Verifique as variáveis de ambiente no sistema operacional onde o Transfer Files está instalado.
  * Localize a variável JAVA_HOME e verifique se o caminho está apontando para a instalação atual do Java, versão 17.
  * Caso o caminho esteja incorreto, atualize a variável de ambiente com o caminho correto para a versão atual do Java.
  * Após ajustar a variável, reinicie o serviço do Transfer Files e tente acessar a página novamente.
