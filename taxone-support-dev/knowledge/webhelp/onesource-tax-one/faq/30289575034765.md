Se o código de serviço não está aparecendo para ser marcado no agrupamento Tipo de Serviço, mesmo após ser cadastrado no módulo _Básicos > Data Warehouse_, pode ser que o código de serviço está vinculado a outro tipo de serviço. Isso pode impedir sua associação com o agrupamento desejado.

Passo a passo para vincular um código de serviço a um tipo de serviço específico no sistema:

  1. Acesse o módulo _SPED > EFD REINF._
  2. Acesse o menu _Parâmetros > Retenções Previdenciárias > Identificador do Tipo de Serviço > Por Código de Serviço._
  3. Clique no botão **"Relatório"** para gerar o **relatório de códigos de serviço**.

  4. Verifique se o **código de serviço** está vinculado a outro tipo de serviço. Por exemplo, se o código "11.02" está vinculado ao Tipo de Serviço 1000000005, ao invés de estar no Tipo de Serviço 1000000002.
     * Se estiver, desvincule o código desse tipo de serviço.
     * Vincule o código de serviço ao **Tipo de Serviço** correto.
     * Salve as alterações e verifique se o código aparece como esperado.
  5. Após realizar os passos acima, gere um novo relatório para confirmar que o código de serviço está corretamente vinculado ao grupo de Tipo de Serviço. Se o problema persistir, entre em contato com o suporte técnico para assistência adicional.


**Dicas e Recomendações:**

Regularmente verifique e revise os vínculos de códigos de serviço para garantir que estão corretamente associados, evitando problemas operacionais. Mantenha um registro das alterações realizadas para facilitar futuras auditorias e suporte.
