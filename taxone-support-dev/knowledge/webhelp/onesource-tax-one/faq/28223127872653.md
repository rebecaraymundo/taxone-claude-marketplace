## Problema:

Usuário enfrenta dificuldade ao tentar cadastrar um código de município que não está habilitado para inserção durante a importação da SAFX04.

## Causa-raiz:

O campo de inserção do código de município não estava habilitado devido a uma configuração no ambiente de produção. Além disso, a tabela de municípios não é atualizada automaticamente, requerendo atualizações manuais quando novos municípios são constituídos.

## Resolução:

  1. Verifique se está operando no ambiente correto (UAT ou Produção).


  1. Acesse o módulo Básicos > Data Warehouse > Manutenção > Cadastros > Município.


  1. Insira manualmente o código do município que deseja adicionar.


  1. Confirme se o município agora aparece na lista de municípios disponíveis.


## Próximos passos:

Após realizar o cadastro manual do município, é recomendado monitorar as próximas importações para garantir que o problema não persista. Caso haja novos municípios, repita o processo de cadastro manual conforme necessário.

## Dicas e Recomendações:

Para evitar futuros problemas com a falta de municípios na base de dados, recomenda-se verificar regularmente as atualizações disponíveis para a tabela de municípios e aplicá-las. Mantenha-se atento às comunicações sobre atualizações do sistema que possam incluir novos municípios automaticamente.
