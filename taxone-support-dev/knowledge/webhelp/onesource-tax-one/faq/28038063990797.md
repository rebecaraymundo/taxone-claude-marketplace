**Nota** : O link para download do certificado Comodo64.cer não está disponível nos manuais fornecidos para a instalação e configuração do WebService.

Segue o passo a passo para localizar e baixar o certificado Comodo64.cer necessário para configurar o WebService:

  1. Acesse o link para download dos pacotes de atualização: [[Interface SAP] OBTI – Thomson Reuters.](<https://suporte.thomsonreuters.com.br/hc/pt-br/articles/27513286162829--Interface-SAP-OBTI> "Hyperlink \[https://suporte.thomsonreuters.com.br/hc/pt-br/articles/27513286162829--Interface-SAP-OBTI\]")
  2. Faça o download do pacote de atualização mais recente.
  3. Após baixar o pacote, descompacte-o e localize a pasta "**WEBSERVICE - Certificado.zip** ".
  4. Dentro dessa pasta, você encontrará o arquivo **"Comodo64.cer"** , que é o certificado necessário.
  5. Se necessário siga as instruções fornecidas para importar e aplicar o certificado Comodo64.cer no SAP.
  6. Após instalar o certificado, verifique se o WebService está funcionando corretamente, realizando testes de conexão ou operações de teste.
  7. Caso encontre problemas após instalar o certificado, entre em contato com o suporte técnico para obter mais assistência.


O certificado Comodo64.cer é essencial para a configuração e utilização adequada do WebService, garantindo a segurança e autenticidade das comunicações. Seguindo esses passos, você poderá localizá-lo e instalá-lo corretamente.
