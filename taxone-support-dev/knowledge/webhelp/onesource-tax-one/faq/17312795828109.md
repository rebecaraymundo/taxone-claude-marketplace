**Nota:** Esta mensagem aparece pois o usuário que está extraindo as SAFX pode não ter permissão para gravar o arquivo no diretório informado ou no path default parametrizado.

  1. Na tela de Parametrização Geral > aba de Configuração:


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/25189029201677)

Atenção:

  * Path Windows: selecione a opção **Path Default** na tela de Extração em conjunto com a opção “**Desktop** ” em cargas online.
  * Path Unix: selecione o **Path Default** mas com a opção **Servidor Unix** marcado, essa opção é somente para cargas em **Background**.
