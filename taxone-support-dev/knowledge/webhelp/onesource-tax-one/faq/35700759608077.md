## Problema:

Usuário encontra erro ao gerar demonstrações contábeis no Taxone mesmo com saldo disponível na conta.

## Causa Raiz:

As contas não estão associadas aos códigos de aglutinação correspondentes no sistema Taxone.

## Resolução:

  * Acesse o módulo Taxone com suas credenciais de usuário.
  * Identifique a conta que possui saldo mas não está aparecendo nas demonstrações.
  * Associe esta conta ao código de aglutinação correto.
  * Repita a geração das demonstrações contábeis.


## Próximos Passos:

Após realizar a associação das contas aos códigos de aglutinação, gere novamente as demonstrações para verificar se o problema foi resolvido. Se o erro persistir, entre em contato com o suporte técnico para uma análise mais detalhada.

## Dicas e Recomendações:

Verifique regularmente se todas as contas estão devidamente associadas aos códigos de aglutinação para evitar futuros problemas nas demonstrações contábeis. Consulte o Tax Policies para mais informações sobre melhores práticas no uso do Taxone.
