Para preencher a Guia de Recolhimento com o Código de Receita Estadual, siga os passos abaixo:

  1. Realize o cadastro do Código de Receita Estadual da UF desejada antes de emitir a Guia de Recolhimento.
**Nota:** Este procedimento de cadastro é feito conforme a FAQ de Cadastro do Código de Receita Estadual.


  2. Em seguida, para utilizá-lo na Guia de Recolhimento, acesse Estadual > ICMS > Apuração > Apuração do ICMS > Lançamentos complementares > Apuração do ICMS Dif. Aliq. UF Origem/Destino (EC 87/15).


  3. Para o exemplo a seguir, utilizamos o código para ICMS da UF de SC: 2496 - ICMS - Resultante da Utilização de Crédito Presumido já cadastrado anteriormente.

UF: SC
N. Guia Recolhimento: 123456
Mês / Ano Refer.: 09/2022
Valor: 1.651,00
Data Vencimento: 05/10/2022
Código Obrigação: 02
Código Receita: 2496 - Resultante da Utilização de Crédito Presumido


  4. Em seguida, preencha os dados da tela com as informações necessárias para emissão da Guia de Recolhimento e informe o Código de Receita Estadual a ser utilizado.
