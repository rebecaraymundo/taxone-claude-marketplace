Após a execução da carga SAFX07, foi verificado que a Data de Pagamento da Nota Fiscal (NF) em alguns registros foi preenchida incorretamente com "ERROR", o que impede a importação desses registros para a tabela definitiva.


Para resolver o problema identificado é necessário ajustar o procedimento da SAFX07:

  1. Acesse o sistema Interdados.
  2. Navegue até a opção "Procedimentos".
  3. Selecione o procedimento "SAFX07".
  4. Clique em "Mais" para ver mais opções.
  5. Escolha "Layout" para modificar as configurações de layout.
  6. Localize o campo "DT_PAGTO_NF".
     1. No campo "Formato Origem", insira o formato de data "DD/MM/AAAA".
     2. No campo "Formato Destino", insira o formato de data "AAAAMMDD".
