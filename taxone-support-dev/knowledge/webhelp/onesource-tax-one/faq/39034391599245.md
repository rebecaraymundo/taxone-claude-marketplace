# Remoção Segura de Registros de Serviço Desatualizados no TaxOne

## Problema

Usuários encontram dificuldades ao decidir se podem remover códigos de serviço desatualizados dentro do sistema TaxOne, sem impactar outros registros.

## Causa Raiz

A confusão surge devido à coexistência de códigos de serviços com datas de validade distintas. Alguns desses códigos desatualizados são anteriores à data de adoção do sistema pela empresa do usuário.

## Resolução

Para excluir os registros de serviço desatualizados no TaxOne, siga os passos abaixo:

  1. Acesse o cadastro do serviço que deseja excluir.
  2. Verifique se o código de serviço não está associado a outros registros no sistema.
  3. Se não houver registros vinculados, clique no ícone da lixeira para realizar a exclusão.
  4. O sistema permitirá a exclusão apenas se não houver registros vinculados.
  5. Repita o processo para cada serviço que deseja excluir.
