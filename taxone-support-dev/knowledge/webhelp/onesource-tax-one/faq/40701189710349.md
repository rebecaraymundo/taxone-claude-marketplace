# [Básicos][Data Warehouse] TAX ONE - Erro ao Excluir Documento Fiscal com Data de Movimento Incorreta

## Problema:

Ao tentar excluir um documento fiscal com a data de movimento incorreta, o sistema apresenta uma mensagem de erro indicando que a exclusão não é possível.

## Causa Raiz:

O erro ocorre porque o período fiscal relacionado ao documento está fechado no sistema. Essa configuração impede alterações na base de dados para períodos já encerrados, conforme definido nas configurações de controle de fechamento.

## Resolução:

  1. No sistema, acesse o menu **Básicos > Ferramentas > Rotinas Acessórias > Fechamento Controle**.
  2. Verifique o status do período fiscal para o grupo ao qual o documento está associado.
  3. Se o período estiver fechado, reabra o período para permitir alterações. Siga os passos descritos no manual operacional do sistema para realizar essa operação com segurança.
  4. Após reabrir o período, tente excluir novamente o documento fiscal com a data de movimento incorreta.
  5. Finalize o procedimento e, se necessário, volte a fechar o período fiscal para evitar alterações indevidas.
