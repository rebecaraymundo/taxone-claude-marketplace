# [Interfaces] [ORACLE Interdados - V17R01 Cloud] [Geral] Problemas na integração dos itens de notas fiscais para a SAFX08 e SAFX09.

O Objetivo deste artigo é orientar o usuário referente à possíveis causas na não integração dos itens de notas fiscais para a SAFX08 e SAFX09.

**Problema:**

Itens de notas fiscais não estão sendo integradas ao TaxOne.

**Ações à serem adotadas:**

  * Verifique os parâmetros do extrator inseridos na tela do procedimento no Interdados;
  * Verifique no arquivo XML gerado pela extração dos dados, se os itens estão presentes no arquivo;
  * Verifique se a data do item está de acordo com o parâmetro cadastrado no procedimento do Interdados;
  * Verifique se há alguma regra nos parâmetros inseridos no extrator Oracle, direto no Oracle Cloud.


**Próximos Passos:**

  * Caso identificado o parâmetro que esteja impactando, certifique-se que não haverá outros impactos e o remova ou altere-o;
  * Processe uma nova carga e verifique se o problema persiste.


**Dicas e Recomendações:**

Para evitar problemas similares no futuro, siga essas melhores práticas:

  * Mantenha um registro atualizado da configuração do ambiente, incluindo alterações feitas nos parâmetros.
  * Estabeleça um processo de revisão para garantir que as configurações estejam corretas antes de realizar homologações ou testes.
  * Considere testar as alterações em um ambiente isolado para limitar potenciais impactos.


Para mais informações sobre configurações no sistema Interdados, consulte artigos relacionados ou entre em contato com a equipe de suporte.
