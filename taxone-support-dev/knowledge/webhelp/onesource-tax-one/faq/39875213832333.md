## Problema

O cliente deseja realizar a integração do sistema SAP com o Tax One via WebService, mas não possui as credenciais necessárias para autenticação.

##

## Causa Raiz

As credenciais de autenticação no Tax One para integração via WebService não são fornecidas automaticamente. Elas precisam ser criadas diretamente pelo cliente no próprio sistema Tax One.

##

## Resolução

Siga os passos abaixo para criar as credenciais necessárias para integração:

  1. Acesse o sistema Tax One utilizando o login de administrador ou outro perfil com permissões adequadas.
  2. Navegue até a seção de **Administração**
  3. Crie um novo usuário. Certifique-se de que o tipo de usuário seja configurado como **comum** e com acesso ao Tax One.
  4. Defina um nome de usuário e uma senha. Garanta que o usuário criado tenha as permissões necessárias para integração via WebService.
  5. Finalize o processo de criação e teste o login do usuário criado para garantir que ele pode acessar o Tax One sem problemas.
  6. Utilize essas credenciais (nome de usuário e senha) na configuração da Interface, acessando a transação /n/MSAFX/PARAMETROS > Parametrização Geral > WebServices.


##

### Próximos Passos

Após a criação das credenciais:

  * Configure o sistema SAP, adicionando as credenciais criadas no Tax One ao WebService de integração.
  * Realize testes iniciais de comunicação entre o SAP e o Tax One, verificando se os dados são transmitidos corretamente.
  * Monitore a integração por alguns dias para garantir que está funcionando sem falhas.
  * Em caso de dúvidas ou problemas durante a configuração, entre em contato com a equipe de suporte.


##

### Dicas e Recomendações

  * Para garantir segurança, utilize senhas fortes e troque-as regularmente.
  * Acompanhe a documentação técnica tanto do SAP quanto do Tax One para entender todos os recursos disponíveis no WebService.
  * Visite nossa base de conhecimento regularmente para acompanhar atualizações e novos artigos relacionados a integrações.
