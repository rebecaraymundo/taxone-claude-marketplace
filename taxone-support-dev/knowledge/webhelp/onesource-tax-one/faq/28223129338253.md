# [TR - Transfer Files - SFTP] TAX ONE - Como resolver problemas de Conexão com SFTP utilizando IP Dinâmico?

---

A causa do problema é a configuração dinâmica de IPs nos servidores, o que impede a utilização de um endereço IP fixo nas configurações de conexão. Para resolver, siga as orientações:

  1. Acesse o Prompt de Comando no Windows.
  2. Digite o comando `ping oms-sftp-prd.thomsonreuters.com`
  3. Pressione Enter para executar o comando e identificar o IP atual.
  4. Use o IP identificado para configurar temporariamente a conexão SFTP, até que uma solução permanente seja implementada.


**Atenção:** Após realizar a configuração, continue monitorando a conexão e consulte regularmente o IP para manter a operação sem interrupções. Considere implementar uma solução que não dependa de IPs dinâmicos para evitar problemas futuros.
