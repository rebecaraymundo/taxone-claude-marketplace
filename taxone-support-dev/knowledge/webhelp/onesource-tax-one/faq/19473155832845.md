Normalmente, esse problema ocorre porque o código informado no campo TIPO_LANCTO está igual a "X - Extemporâneo" e o campo DAT_LANCTO_EXTEMP está sem data. Para corrigir o erro 93185: 'Para o indicador de Lancamento igual a <X> (Lancamento Extemporaneo) o campo Data de Lancamento Extemporaneo deve ser preenchido.', siga a orientação abaixo:

  * Verifique se o lançamento realmente é extemporâneo e então informe a data no campo DAT_LANCTO_EXTEMP. Caso contrário, altere o campo TIPO_LANCTO para "N - Normal" ou "E - Encerrado".
