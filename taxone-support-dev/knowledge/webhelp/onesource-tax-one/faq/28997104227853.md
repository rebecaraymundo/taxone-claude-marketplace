O objetivo deste artigo é orientar o consultor\cliente sobre dúvidas de criação de usuários no Interdados, tanto ambiente UAT como PROD.

### Problema:

O usuário solicita a criação de um novo acesso no sistema Interdados, porém enfrenta dificuldades devido à política de usuário único no sistema.

### Resolução:

  * O sistema Interdados UAT possui uma política de usuário único, ou seja, apenas um usuário está disponível e ele permite realizar todas as operações e parametrizações necessárias.
  * Acesse o sistema Interdados com o usuário padrão fornecido pela sua organização.
  * Utilize as credenciais atuais para realizar as parametrizações e operações necessárias.
  * Se você enfrentar problemas com o acesso ou precisar resetar a senha, abra um novo ticket solicitando assistência específica para reset de senha.


### Próximos Passos:

Após seguir as instruções acima, monitore o uso do sistema para garantir que todas as funcionalidades estejam sendo acessadas sem problemas. Caso encontre qualquer dificuldade, entre em contato com o suporte técnico.

### Dicas e Recomendações:

Para evitar conflitos de acesso e garantir a segurança, sempre mantenha as credenciais de acesso confidenciais e em segurança. Evite compartilhar seu usuário e senha com outros colegas que não tenham permissão ou não devam utilizar o Interdados.
