Como Gerar o Registro D130 para o Modelo 57 do CT-e a partir de 2025

**Problema:**

O cliente está enfrentando dificuldades para gerar o registro D130 no arquivo da EFD ICMS IPI, mesmo tendo selecionado corretamente o perfil e utilizando o modelo 57 conforme as novas normas de 2025.

**Causa Raiz:**

A partir de 2025, houve uma alteração nas especificações técnicas para a geração dos registros na EFD. Conforme o Ato COTEPE/ICMS nº 164/2024, o modelo 57 do CT-e foi atualizado e requer que o campo SAFX07 esteja preenchido com a situação especial “G” para CT-e Simplificado ou “H” para Substituição CT-e Simplificado. O problema surge se essa condição não for atendida, impedindo assim a geração do registro D130.

**Resolução:**

Acesse o sistema.

Navegue até a tabela SAFX07 correspondente ao período de Fevereiro de 2025.

Verifique o campo 125 da tabela SAFX07 para os modelos de documento “57”.

Insira a situação especial “G” ou “H” conforme o caso indicado na normativa do Ato COTEPE.

Salve as mudanças e tente gerar o arquivo da EFD novamente.

**Próximos Passos:**

Após seguir as instruções acima, gere o arquivo e verifique se o registro D130 está incluído corretamente. Em caso de dificuldades repetidas, contate o suporte técnico para uma análise mais detalhada do problema.
