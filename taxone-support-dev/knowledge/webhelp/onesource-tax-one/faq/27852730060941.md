No que se refere às operações fiscais e regulamentações no setor de combustíveis, é essencial seguir diretrizes específicas para garantir a conformidade e a precisão dos registros. O **GUIA PRÁTICO** oferece orientações detalhadas sobre como as empresas devem proceder. A seguir, destacamos as principais informações relacionadas ao**REGISTRO C165** , que se aplica a operações com combustíveis:



**Registro C165: Operações com combustíveis (Código 01):**

  * Este registro deve ser apresentado por empresas do setor de combustíveis, como distribuidoras, refinarias e revendedoras, para operações de saída. Postos de combustíveis não estão obrigados a apresentar este registro.
  * Para um mesmo documento fiscal, não é permitido informar dois ou mais registros com a mesma combinação de conteúdo nos campos COD_PART e VEIC_ID.
  * Observações para o Documento Fiscal:
    * Movimento de Saída
    * Operação com combustíveis
    * Item de combustível (validado pelo código ANP no cadastro do produto, demonstrado nos passos posteriores)


Passo a passo para geração do Registro C165 - Operações com Combustíveis no SPED Fiscal:

  1. Acesse o módulo **Básicos > Data Warehouse**, menu _Manutenção > Documento Fiscal > Doc. Fiscal de Mercadoria e Serviço.

_![KBA TAX ONE - C165 - PRINT 1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/27852706977549)![KBA TAX ONE - C165 - PRINT 2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/27852730005133)![KBA TAX ONE - C165 - PRINT 3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/27852730017805)



  2. Faça a **equalização/limpeza** do período no módulo **Básicos > Data Warehouse**, menu _DATA MART > Equalização/Limpeza.

_![KBA TAX ONE - C165 - PRINT 4.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/27852706987533)


  3. Cadastro do produto (SAFX2013 - Campo 41) deve estar preenchido o Código ANP, na qual indicará que se trata de um item de combustível
  4. Acesse o módulo **Básicos > Data Warehouse**, menu _Manutenção > Códigos > Produto_. Certifique-se que o **Cadastro do produto (SAFX2013 - Campo 41)** contenha o Código ANP preenchido para identificar itens de combustível.

![KBA TAX ONE - C165 - PRINT 5.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/27852730020365)![KBA TAX ONE - C165 - PRINT 6.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/27852730024333)



  5. Acesse o módulo **Básicos > Data Warehouse**, menu _Manutenção > Cadastros Parâmetros > Pessoa Física/Jurídica por Ramo de Atividade_. Parametrize a **Pessoa Física/Jurídica por Ramo de Atividade** relacionado. O Ramo de atividade deve conter uma das opções: 94 - Refinaria, 104 - Distribuidor ou 106 - Transportador e Revendedor Retalhista.

![KBA TAX ONE - C165 - PRINT 7.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/27852730028813)


  6. Acesse o módulo**Básicos > Data Warehouse**, menu _Manutenção > Cadastros > Estabelecimentos X Pessoas Físicas/Jurídicas. _Parametriza uma Pessoa Física/Jurídica associado ao Estabelecimento (só é possível associar a Pessoa F/J com o indicador de estabelecimento).

![KBA TAX ONE - C165 - PRINT 8.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/27852730029965)


  7. Acesse o módulo **Básicos > Data Warehouse**, menu _Manutenção > Documento Fiscal > Informação de Transporte. _Cadastre as informações de transporte (SAFX50).

![KBA TAX ONE - C165 - PRINT 9.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/27852730035725)



  8. No módulo**SPED > EFD – Escrituração Fiscal Digital**, menu _Parâmetros > Perfil,_ cadastre e marque o perfil do SPED Fiscal com os registros C100 e C165. O Registro C165 será gerado quando a nota gravada no C100 tiver código de modelo = 01 ou 55).

![KBA TAX ONE - C165 - PRINT 10.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/27852707007885)![KBA TAX ONE - C165 - PRINT 11.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/27852707013645)


  9. Gere o meio magnético no módulo **SPED > EFD – Escrituração Fiscal Digital**, menu _Geração > Meio Magnético.

_![KBA TAX ONE - C165 - PRINT 12.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/27852707015181)


  10. Verifique se o Registro C165 foi gerado corretamente.
