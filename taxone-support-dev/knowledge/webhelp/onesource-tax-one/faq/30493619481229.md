Seguem informações sobre a conformidade do software TAX ONE com as diretrizes WCAG 2.1 para acessibilidade:

O WCAG 2.1 (Web Content Accessibility Guidelines 2.1) refere-se a um conjunto de diretrizes desenvolvidas pelo W3C (World Wide Web Consortium) com o objetivo de tornar o conteúdo Web mais acessível a pessoas com deficiências. Essas diretrizes são amplamente aceitas como padrão internacional para acessibilidade web.

As WCAG 2.1 definem três níveis de conformidade:

  * **Nível A** : O nível mais básico de conformidade. Alguns usuários não conseguirão acessar o conteúdo se esses critérios não forem atendidos.
  * **Nível AA** : O nível médio de conformidade. Trata da maior parte das barreiras de acessibilidade e é geralmente o nível recomendado para a maioria dos sites.
  * **Nível AAA** : O nível mais alto de conformidade. Trata de todas as barreiras de acessibilidade, mas pode não ser possível atingir este nível para todo o conteúdo.


Atualmente o TAXONE não é compatível com as diretrizes WCAG 2.1 de acessibilidade e não há planos de correção ou implementação previstos no roadmap de 2024 da empresa.

**Importante:** Recomendamos que o cliente monitore as atualizações futuras de produto, pois a inclusão de conformidade com WCAG 2.1 pode ser considerada em roadmaps futuros. Continue a verificar as atualizações de produto regularmente.

Encorajamos os clientes a expressarem a necessidade de acessibilidade como um requisito importante para futuras atualizações de produto. Isso ajudará a priorizar tais implementações. Para obter mais informações sobre acessibilidade e conformidade, visite [WCAG 2.1 Guidelines](<https://www.w3.org/WAI/standards-guidelines/wcag/>).
