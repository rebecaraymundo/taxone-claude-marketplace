Passo a passo para corrigir o** Erro 91025 -** Informações de Selo preenchida para produto não controlado por Selos:

  1. Preencha o campo **IND_CONTROLE_SELO** com “N” quando os seguintes campos estiverem vazios ou nulos (@):

     1. GRUPO_SELO

     2. SUB_GRUPO_SELO,

     3. COR_SELO

     4. SERIE_SELO


Nota: O erro ocorre porque o campo IND_CONTROLE_SELO está preenchido com "N" e os campos GRUPO_SELO, SUB_GRUPO_SELO, COR_SELO, SERIE_SELO estão preenchidos com alguma informação.
