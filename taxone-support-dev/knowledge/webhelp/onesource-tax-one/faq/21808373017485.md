Durante a configuração do WebService para ser utilizado no envio de carga automática do SAP para o TAxOne, é necessário realizar a configuração de tempo e ativação dos Jobs para o mesmo.


A configuração consiste em definir o intervalo de tempo que o Job será executado. A recomendação é que este intervalo seja verificado junto ao cliente para que não fique executando em curtos intervalos, consumindo recursos desnecessários, e nem em intervalos muito grandes para que não ocorra das cargas ficarem paradas no SAP esperando o horário para realizar o envio.

  1. Para realizar a configuração, entre na transação /N/MSAFX/parametros, selecione  _Parâmetros gerais > aba Controle Job. _


  1. Antes de modificar o tempo, verifique se o Job está parado (na coluna Status, deve estar em vermelho), selecione o tempo e coloque em minutos o intervalo que o Job deverá ser executado.


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/24320408157197)

  1. Cada Job da lista acima possui uma finalidade. Para facilitar a definição de intervalos, confira a relação de cada uma delas:


  * ENVIO WS BACKGROUND: será gerado o Job automaticamente, sem a necessidade de configurar pela SM36.
  * CONSULTA LOTE IMPORTAÇA O SAFX: dará o retorno para a Interface referente a carga efetuada no TAX ONE no log de Monitoramento.
  * ENVIO FILA IMPORTAÇA O AUTOMA TICA: fará o envio dos dados da SAFX para a tabela definitiva do TAX ONE.
  * CONSULTA LOTE IMPORTAÇA O DEFINITIVA> dará o retorno para a Interface referente ao resultado da importação efetuada no log de Monitoramento.
  * EXPURGO DE DADOS JSON: excluíra os dados das tabelas temporárias da Interface de acordo com o tempo definido na aba Web Service.
