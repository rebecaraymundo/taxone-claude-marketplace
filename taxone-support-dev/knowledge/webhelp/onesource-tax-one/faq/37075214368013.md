## Problema:

Após uma atualização na interface, os campos "COD_MEDIDA" e "COD_UNID_PADRAO" estão aparecendo em branco durante a extração dos dados do documentário fiscal no ambiente de qualidade.

## Causa Raiz:

A razão para que os campos mencionados estejam em branco está na configuração da tabela T006A. A unidade de medida "UNT", necessária no campo "COD_UNID_PADRAO", não está cadastrada para a linguagem PT (Português), mas sim apenas como "UN". Esta configuração impacta diretamente os campos "COD_MEDIDA" e "COD_UNID_PADRAO".

## Resolução:

  * Solicite ao consultor SAP ou ao responsável pelo sistema para adicionar a unidade "UNT" para a linguagem PT na tabela T006A.
  * Após o cadastro ser realizado, refaça a extração dos dados para validar se os campos estão preenchidos corretamente.


## Próximos Passos:

Após a alteração, é importante realizar uma nova extração de dados para confirmar se o problema foi resolvido. Caso os campos ainda apareçam em branco, entre em contato novamente com o suporte técnico para uma investigação mais detalhada.

## Dicas e Recomendações:

Recomenda-se sempre validar as configurações essenciais após qualquer atualização do sistema para evitar transtornos em operações regulares. Acompanhe também as notas de versão das atualizações para estar ciente de todas as mudanças implementadas.
