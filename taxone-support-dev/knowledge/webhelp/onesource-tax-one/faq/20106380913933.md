Nota: Este processo é realizado quando existe algum problema na geração de cargas, sendo necessário identificar algum possível erro no Log, assim como também localizar o lote gerado para análises.


Passo a passo para realizar a parametrização para geração do arquivo Json no Interdados V17R01:

  1. Acesse o Interdados e clique em "Opções > Propriedades".
  2. Na tela de "Configuração de Propriedades", utilize o campo "Filtro" para digitar "BACKEND.WRITE.LOG" e verificar se o parâmetro já existe.
     * Se o parâmetro existir, ele será exibido na tela.
  3. Uma vez encontrado o parâmetro, altere o valor de "N" para "S".
     * Após a alteração, clique no botão "Salvar", que será habilitado automaticamente.
  4. Realize uma carga no sistema e aguarde até que o status indique que a carga foi finalizada.
  5. Com a carga finalizada, clique no botão "Log".
  6. Na coluna "Mensagem", procure pela informação "TRANSMITINDO DADOS TAXONE".
     * Clique no botão com três pontinhos (...) para visualizar o arquivo JSON.
  7. Copie todo o conteúdo do arquivo JSON e salve em um arquivo com extensão ".txt".
  8. Lembre-se de que, com o parâmetro "BACKEND.WRITE.LOG" definido como "S", todas as cargas subsequentes gerarão o arquivo JSON.
  9. Para evitar a geração contínua do arquivo JSON e o acúmulo desnecessário de dados no banco de dados, repita o processo do passo 3 e altere o valor de volta para "N" quando necessário.
