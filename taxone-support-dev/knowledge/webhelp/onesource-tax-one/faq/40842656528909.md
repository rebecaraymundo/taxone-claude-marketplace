# [Básicos][OBI - OneSource Brasil Integration] - Lotes travados no Status FILA-TAXONE no Painel de Integração

**PROBLEMA:**

Os Lotes ficam parados no Painel de Integração no Status FILA-TAXONE.

**CAUSA RAIZ:**

O Status FILA_TAXONE significa que o processamento do Lote já saiu do OBI e está aguardando para ser executado no Tax One. Os lotes após serem processados pelo OBI, são enviados para importação no Tax One e nesse momento eles ativam o status FILA_TAXONE.

Caso ocorra alguma instabilidade, algum incidentes no sistema, ou erro na configuração do processo o lote pode ficar travado com este status. Outro motivo para que o lote fique marcado com este status é se o arquivo for muito volumoso, ou tenham outros processos sendo executados ao mesmo tempo no Tax One, o sistema vai manter o status FILA_TAXONE até a conclusão do processamento.

Após o término o status é alterado para FINALIZADO e posteriormente ENCERRADO.

**RESOLUÇÃO:**

Basta acompanhar o processamento do Lote para certificar que o mesmo será concluído. Neste momento o Status irá mudar para FINALIZADO e após para ENCERRADO.

Caso um lote trave no Status **FILA-TAXONE** e o processo não avance será necessário acionar o Suporte para uma avaliação detalhada do lote.
