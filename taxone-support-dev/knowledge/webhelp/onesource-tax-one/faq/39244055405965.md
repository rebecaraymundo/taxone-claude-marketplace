O objetivo deste artigo é orientar o usuário para criação de empresa no Interdados.

## Problema:

Usuário deseja cadastrar uma nova empresa no Interdados V17.

## Resolução:

O cadastro de empresas no Interdados segue alguns padrões que também são definidos no Oracle Cloud associando corretamente cada Empresa e Estabelecimento do Tax One ao respectivo Legal Entity/Establishment_ID no Interdados, por isso verifique no Oracle Cloud se a estrutura organizacional está correta.

## Próximos Passos:

Após conferência da estrutura no Oracle Cloud, realize a importação das tabelas de definição do TaxOne seguindo o seguinte caminho:

  * Acesse o Interdados e navegue até Cadastro TaxOne > Configuração e importação > Importar definições de tabela; o material disponibilizado no [Web Help - Cadastro TaxOne ](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Interfaces/Interface_Oracle_V17R01_Cloud/Webhelp_Interface_Oracle_V17R01_Cloud/Manuais_portugues/Manuais_funcionais/Manual_de_usuario/V17R01_usuario_cadastrotaxone.htm>)poderá apoiar caso necessário;
  * Execute o procedimento 1000 na tela de programação do Interdados; o material [Web Help - Carga 1000](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Interfaces/Interface_Oracle_V17R01_Cloud/Webhelp_Interface_Oracle_V17R01_Cloud/Manuais_portugues/Manuais_funcionais/Manual_de_usuario/V17R01_usuario_carga1000.htm>) também poderá apoiar;
  * Avance para a aba Cadastros no Interdados e siga os passos descritos no material [Web Help - Cadastros](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Interfaces/Interface_Oracle_V17R01_Cloud/Webhelp_Interface_Oracle_V17R01_Cloud/Manuais_portugues/Manuais_funcionais/Manual_de_usuario/V17R01_usuario_cadastros.htm>)


## Dicas e Recomendações:

  * Revise a estrutura cadastrada no Oracle Cloud;
  * Caso a empresa/estabelecimento esteja indisponível para cadastro, verifique o XML da carga 1000;
  * Documente e mantenha registros dos cadastros incluídos;
  * Consulte a documentação oficial para verificar boas práticas relacionadas à parametrização.
  * Caso tenha dúvidas, entre em contato com o suporte.


Para mais informações, recomendamos consultar os recursos adicionais sobre parametrizações e layouts na plataforma da empresa.
