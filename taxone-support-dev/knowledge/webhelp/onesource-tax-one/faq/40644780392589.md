## Introdução

 _Esse artigo tem por objetivo mostrar como habilitar o campo "Código de Pagamento" no relatório de Geração de Retenções de INSS_

 _**1 - Relatório Geração de Retenções de INSS**_

  * Módulo Federal> Obrigações de Tributos Federais> Outras Obrigações> Retenção de INSS - Serviços Prestados> Geração de Retenções de INSS


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40644773100173)

Se o campo Processar contribuições de INSS para o FUNRURAL estiver como "NÃO" não será habilitado o campo Código de Pagamento.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40644773101837)

Conforme está descrito no Manual Funcional do Módulo o código de pagamento apenas será habilitado se a opção de FUNRURAL for SIM.

Segue link do manual funcional do Módulo - [Link manual funcional](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/WebHelp/mergedProjects/Webhelp_ONESOURCE_TAXONE/Manuais_Funcionais/Federal/Obrigacoes_Tributos_Federais/Obrigacoes_Tributos_Federais.htm>)

![](/attachments/token/snijjkJbXTetQ7njjXUbZrd1E/?name=image.png)

Alterando a opção do FUNRURAL para sim será habilitado os códigos de pagamento.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40644780383245)
