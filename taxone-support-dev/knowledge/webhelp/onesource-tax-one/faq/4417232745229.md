Passo a passo para corrigir o **Erro 91899** \- Campo com conteúdo invalido. Este campo somente poderá conter a informação Sim se a pessoa for jurídica. Verificar campo 06 (CPF_CGC):

  1. Verifique se o campo **IND_CLIENTE_LIVRE** está preenchido corretamente, caso esteja, altere o campo **CPF_CGC** para um CNPJ válido.


Nota: O erro ocorre quando o campo **IND_PESSOA_LIVRE** está igual a "S", e o campo **CPF_CGC** está com o conteúdo de CPF ao invés de CNPJ. Caso o indicador esteja incorreto, altere o valor "N".
