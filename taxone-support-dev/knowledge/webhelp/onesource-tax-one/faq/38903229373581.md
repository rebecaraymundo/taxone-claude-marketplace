# [SPED][EFD - ECD]TAX ONE - ECD - Como gerar o registro I012 para o Livro R.

Para geração do registro I012 no arquivo do livro R, deverá informar o HASH CODE para o período no módulo: Sped Contábil / Menu: Parâmetros > Controle da Obrigação, selecione a última geração efetuada do período em questão, clique em "Informar Hash Code", selecione o período gerado e cadastre o "Cód. HASH" do livro A.



  1. Efetuar uma geração do meio magnético ECD "Tipo do Livro = A"
  2. Efetuar o cadastro do Hash Code para o período (Módulo; Sped Contábil / Menu: Parâmetros > Controle da Obrigação, selecione a última geração efetuada do período em questão, clique em "Informar Hash Code", selecione o período gerado e cadastre um número qualquer no campo "Cód. HASH"
  3. Efetuar uma nova geração do meio magnético ECD "Tipo do Livro = R

Registro gerado com sucesso em base do suporte seguindo os passos acima.
