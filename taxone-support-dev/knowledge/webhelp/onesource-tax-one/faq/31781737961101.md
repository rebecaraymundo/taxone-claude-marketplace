Caso enfrente dificuldade para realizar a exclusão e/ou inclusão em massa das contas referenciais no Tax One para corrigir cadastros duplicados ou errôneos. Isso pode acontecer devida a duplicidade no cadastro de contas referenciais causado pela importação repetida das mesmas contas contábeis com diferentes centros de custo, sem restrições na aplicação.

Para resolver, siga as orientações:

  * Em Básicos > Ferramenta > Rotinas Acessórias > Inicialização.
  * Selecione a opção Exclusão de Tabela Definitivas Movimentos SAFX 2101 para excluir os dados indesejados.
  * Reimporte as contas referenciais corretas, tomando cuidado para não marcar a opção que exclui todas as contas vinculadas que possam afetar saldos anteriores.
  * Após a exclusão e inclusão correta dos dados, é recomendável verificar se todas as contas referenciais estão corretamente cadastradas e se os totais estão devidamente ajustados. Continuar monitorando para qualquer discrepância nos cadastros futuros.


**Nota:** Para evitar futuras duplicidades, é recomendável revisar os procedimentos de importação e verificar se existem restrições ou validações que possam ser aplicadas para prevenir a importação de dados duplicados. Consulte regularmente a documentação disponível na central de ajuda para manter-se atualizado sobre as melhores práticas e funcionalidades do sistema.
