Como solucionar problemas de acesso aos ambientes UAT e PROD para novos clientes no Interdados Oracle Cloud V17R01:

  1. **Verifique o Status da Contratação:** Após a contratação do Interdados Oracle Cloud V17R01, confirme com o departamento comercial se o processo de contratação foi concluído.
  2. **Aguardar Liberação dos Ambientes:** A liberação dos ambientes UAT e PROD é realizada pelo time de Contratos (Entitlement e Licenciamento), que solicita a criação do ambiente ao Time de Techops que realiza a configuração desses ambientes.
  3. **Recebimento dos Dados de Acesso:** Uma vez que os ambientes estejam prontos, o Time de Techops enviará as URLs de acesso aos ambientes UAT e PROD para o time de Contratos, que por sua vez informará ao cliente.
  4. **Contato Inicial em Caso de Problemas:** Caso não receba os dados de acesso após a conclusão do processo de contratação, entre em contato com o departamento comercial para verificar o status do processo antes de prosseguir para o próximo passo.
  5. **Abertura de Chamado:** Caso já possua os dados de acesso (URL, usuário e senha), mas ainda assim encontrou dificuldades para acessar os ambientes UAT e PROD, abra um chamado para que o suporte técnico possa auxiliar na resolução do problema.
