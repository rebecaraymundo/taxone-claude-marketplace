Problema

Ao realizar extração ocorreu DUMP informando que a classe CL_REST_HTTP_CLIENT não pode ser encontrada (como no print exemplo abaixo):

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40781488307981)

The following syntax error was found in the program /MSAFX/SAPLSETOBI : "Le type 'CL_REST_HTTP_CLIENT' est inconnu.

Causa Raiz

Incompatibilidade de release: o ambiente não possui a classe CL_REST_HTTP_CLIENT porque a versão do SAP_BASIS é inferior a 7.31 SP16 e no ambiente foi aplicada Request para Webservice do pacote da Interface OBTI.



Resolução

Temos duas opções de resolução:

  1. Atualização para SAP_BASIS 7.31 SP16 ou superior.

  2. Abrir chamado solicitando uma Request de exclusão do objetos de WebService e aplicar a mesma nos ambientes onde a Request para Webservice foi aplicada indevidamente.


Próximos Passos

  * Após implementar a correção, monitorar o sistema para garantir que o erro não ocorra novamente
  * Sempre leia o Leia-me dos pacotes para entender pré-requisitos e impactos


Dicas e Recomendações

Planejar e executar a atualização para SAP_BASIS 7.31 SP16 ou superior.
