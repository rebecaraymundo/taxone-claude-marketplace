# Configuração do Registro 0200 no SPED Contribuições para Itens com Múltiplas Destinações

###

### Problema:

Usuários enfrentam desafios ao configurar o registro 0200 no SPED Contribuições para itens que possuem múltiplas destinações (ex: matéria-prima, uso e consumo, revenda).

### Causa Raiz:

De acordo com o Guia Prático do SPED, para itens com mais de uma destinação, o registro 0200 deve indicar o tipo de maior relevância, causando confusão sobre qual tipo escolher e como configurar corretamente no sistema.

### Resolução:

Para configurar corretamente o registro 0200 no SPED Contribuições no software TaxOne para itens com múltiplas destinações, siga os passos abaixo:

  * Acesse SPED > EFD Contribuições > Parâmetros > Dados iniciais.


  * Na aba Geral > Registros Bloco 0, marque a opção que permite adicionar o indicador ao código do produto --> "Considerar o Indicador no Código do Item"


  * No preenchimento, inclua o código do produto precedido pelo indicador de tipo de maior relevância (ex: "1-PA0015" para produto acabado).


  * Salve as configurações e gere o arquivo txt do SPED.

### Próximos Passos:

Após realizar esta configuração, monitore a geração dos registros no SPED para garantir que estão sendo reportados corretamente. Verifique também os próximos envios para assegurar consistência nas informações reportadas.

### Dicas e Recomendações:

Considere revisar periodicamente as configurações dos registros para garantir que estão de acordo com a legislação vigente e as práticas recomendadas. A atualização regular do sistema TaxOne e a consulta ao Guia Prático do SPED Contribuições podem ajudar a reduzir possíveis inconsistências.
