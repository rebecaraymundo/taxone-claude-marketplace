Caso enfrente problemas ao tentar aplicar permissões de grupo a usuários no módulo Específicos, onde as alterações não estão sendo refletidas conforme esperado. Isso pode ocorrer pela falta de reset nas permissões do usuário após a alteração nas configurações de grupo, o que impede que as novas configurações de grupo sejam aplicadas corretamente aos usuários. Para resolver, siga as orientações:

  1. Acesse o menu 'Users'.
  2. Selecione o usuário desejado.
  3. Navegue até 'Product Permissions' e selecione a opção 'TAX ONE'.
  4. Clique no botão 'Reset' para resetar as permissões do usuário.
  5. Salve as alterações e verifique se as restrições do grupo foram aplicadas corretamente ao testar o acesso do usuário.


**Importante:** Sempre que as permissões de grupo forem alteradas, um reset nas permissões dos usuários afetados deve ser realizado para garantir a aplicabilidade das mudanças.
