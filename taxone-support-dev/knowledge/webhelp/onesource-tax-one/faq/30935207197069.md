O objetivo deste artigo é demonstrar como o consultor ou cliente pode desativar o envio de tags em branco nos arquivos XML dos extratores do Oracle Cloud, com o intuito de reduzir o tamanho desses arquivos.

Passo a passo para desativar tags em branco no XML de Extratores Oracle Cloud:

  1. Acesse o Oracle Cloud e vá para a guia "**Modelo de Dados** " do Extrator.
  2. Clique em "**Propriedades** ".
  3. Na seção "**Opções de Saída XML**", desative a opção "**Incluir Tags Vazias para Elementos Nulos** ".
  4. Repita os passos 1 a 3 para todos os Extratores Oracle utilizados pela aplicação Interdados V17R01, caso necessário.
  5. Após desativar essa opção, as novas cargas relacionadas aos Extratores terão um ganho de performance devido à redução do tamanho do arquivo XML gerado.
