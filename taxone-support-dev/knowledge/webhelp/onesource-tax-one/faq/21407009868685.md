# [Municipal][Todos os Municípios] O Código do Município não é gerado no Arquivo, como resolver?

Acesse esta FAQ no [web help do produto](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/WebHelp/mergedProjects/FAQs/Municipal.htm#faq_municipal_19>).

Se a parametrização e a regra de geração foram seguidas corretamente, mas o código do Município ainda não for gerado no arquivo, isso pode ser devido à ausência do campo "Município Msaf" no cadastro do "Município ISS".


Para verificar e corrigir esse problema, siga os passos abaixo:

  1. Acesse Básicos > Data Warehouse > Manutenção > Cadastros > Município ISS > Manutenção.
  2. Verifique se o campo "Município Msaf" está devidamente preenchido no cadastro do "Município ISS".


Após a correção, o código do Município deverá ser gerado corretamente no arquivo dos Módulos Municipais.
