##

### Problema:

O usuário enfrentou um problema ao tentar concluir o processamento de importação/execução das tabelas SAFX112 e SAFX113 para as tabelas definitivas IMPX112 e IMPX113, onde não houve leitura adequada dos dados.

### Causa Raiz:

O erro no processo de importação foi identificado como sendo causado por um registro incorreto no campo DATA_FISCAL na tabela SAFX112, que não seguia o formato padrão de data AAAAMMDD.

### Resolução:

  1. Extraia a SAFX112 da tabela temporária utilizando o Job Servidor Exportação de Dados Tabelas Temporárias (SAFXs).
  2. Verifique se todos os registros no campo DATA_FISCAL estão no formato AAAAMMDD, conforme especificado no manual.
  3. Corrija quaisquer registros que não estejam nesse formato.
  4. Realize novamente o procedimento de importação/execução para verificar se o erro foi resolvido.
