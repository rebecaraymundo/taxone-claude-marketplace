Para esclarecer a localização dos servidores do produto Onesource Tax One, é importante notar que a base de dados é hospedada na infraestrutura da AWS (Amazon Web Services). Essa base de dados é distribuída entre dois servidores principais:


**AMERS:** Os servidores estão localizados nos Estados Unidos.

**EMEA:** Os servidores estão localizados na Europa.


Quando é realizado o acesso, o próprio sistema faz a validação das credencias e direciona o usuário para o servidor correto.

O Link para acessar as Bases de Dados são as seguintes:

**AMERS UAT01 e UAT02: **[Onesource Tax One - Base Homologação AMERS](<https://uat.onesourcetax.com/uat/home-login/UniversalLogin.aspx?OSTarget=https%3A%2F%2Fuat.onesourcetax.com&Logout=5>)

**AMERS PRD01 e PROD02:** [Onesource Tax One - Base Produção - AMERS](<https://www.onesourcetax.com/amer1/home-login/UniversalLogin.aspx?OSTarget=https%3A%2F%2Fwww.onesourcetax.com&Logout=5>)

**EMEA PROD03:** [Onesource Tax One - Base Produção - EMEA](<https://emea1.onesourcetax.com/emea1/home-login/UniversalLogin.aspx?OSTarget=https%3A%2F%2Femea1.onesourcetax.com&Logout=5>)
