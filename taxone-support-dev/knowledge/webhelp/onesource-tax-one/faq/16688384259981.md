Ponto de atenção ao ativar o trace:

  * Não abra as telas de parametrizações.
  * Não execute outros processos/extrações.
  * Na tela de Parametrização Geral deve estar flegada a opção 'Utiliza Geração RTTS'


Passo a passo para extrair o TRACE:

  1. Para realizar o TRACE do processo, acesse a transação ST05.

  2. Selecione as opções abaixo: SQL TRACE e RFC TRACE.


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/25186470184333)

  1. Clique no botão: “Activate Trace with Filter” para apresentar a tela a seguir:


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/25186470256141)

  1. No campo USER NAME informe o usuário SAP que irá rodar o processo de extração da MasterSAF.
  2. Em outra janela, abra a tela de EXTRAÇÃO da Namespace (/N/MSAFX/EXTRACAO) e informe os parâmetros e processo que apresentam a lentidão (Mesmo que esteja dando DUMP, rode o processo). Lembrando que a extração deve ser feita em BACKGROUND.

  3. Quando finalizar o processo com sucesso ou com DUMP, volte para a tela da transação ST05 e clique no botão “Deactivate Trace”abaixo para finalizar o TRACE, e após, clique no botão “Display Trace”.


A seguinte tela é aberta, informe o usuário do SAP:

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/25186470324237)

  1. Para finalizar exporte em formato .XLS os arquivos com os códigos em SQL e/ou ABAP gerados, compacte e envie no chamado.
