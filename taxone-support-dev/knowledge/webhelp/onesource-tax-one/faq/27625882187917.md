Passo a passo para gerar o Registro E250 do SPED Fiscal com código de receita 146-6 para contribuinte do Estado de São Paulo

Quando se tratar de ICMS ST com operação interna, e que o código de receita a ser gerado seja = 146-6 Substituição Tributária (contribuinte do Estado de São Paulo), será necessário seguir os seguintes passos:


1\. Cadastre manualmente a guia no módulo Estadual > ICMS, menu Apuração > Apuração do ICMS > Lançamentos Complementares > Apuração ICMS Subst. Tributária > Ajuste SINIEF, na aba**Guia:**


![KBA - E250 - SP - PRINT 1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/28949109447821)




2\. Realize a Apuração do ICMS

a. Geração, no módulo Estadual > ICMS, menu DATA MART > Apuração do ICMS > Geração:

![KBA - E250 - SP - PRINT 2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/28949109451533)
![KBA - E250 - SP - PRINT 3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/28949109456653)

b. Emissão, no módulo Estadual > ICMS, menu DATA MART > Apuração do ICMS > Emissão:


![KBA - E250 - SP - PRINT 4.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/28949109458061)
![KBA - E250 - SP - PRINT 5.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/28949093239437)




3\. Visualize a **Guia** no Livro de Apuração.


![KBA - E250 - SP - PRINT 6.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/28949093242381)

4\. Parametrize o perfil de geração da obrigação, com o Registro E250 marcado, no módulo SPED > Escrituração Fiscal Digital, menu Parâmetros > Perfil:


![KBA - E250 - SP - PRINT 7.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/28949093245709)



5\. Gere o Meio Magnético no módulo SPED > Escrituração Fiscal Digital, menu Geração > Meio Magnético > Gerar o SPED, na aba Parâmetros.


![KBA - E250 - SP - PRINT 8.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/28949109472653)

![KBA - E250 - SP - PRINT 10.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/28949109474701)



Registro E250 gerado:

![KBA - E250 - SP - PRINT 11.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/28949093252109)


6\. Verifique se o Registro E250 foi gerado corretamente, com o campo **05 - COD_REC** preenchido com '146-6' no arquivo TXT.


![KBA - E250 - SP - PRINT 12.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/28949109481869)
