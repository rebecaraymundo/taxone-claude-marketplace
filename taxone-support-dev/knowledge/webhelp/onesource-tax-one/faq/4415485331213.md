Passo a passo para corrigir o **Erro 91769** \- O Campo de Data de Nascimento Autônomo com formato inválido:

  1. Verifique se os campos **DATA_NASC_AUTONOMO **e **DATA_MOLESTIA** estão com um dos 3 valores aceitáveis:

     1. 00000000

     2. @

     3. Data no formato YYYYMMDD

  2. Caso não estejam, ajuste o arquivo para um dos três valores aceitáveis e o problema será solucionado.
