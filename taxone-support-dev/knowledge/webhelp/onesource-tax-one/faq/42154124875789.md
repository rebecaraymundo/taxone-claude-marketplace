## Problema

O usuário enfrenta dificuldades para excluir o cadastro de uma empresa filial específica no módulo Interdados.

## Causa Raiz

O sistema Interdados exige que as exclusões sigam uma ordem específica (inversa ao cadastro), e, em casos onde múltiplos estabelecimentos compartilham a mesma Organização de Inventário ou Unidade Operacional, a exclusão de apenas um estabelecimento não é permitida.

## Resolução

Para excluir um cadastro de empresa no Interdados, siga os passos abaixo:

  1. Acesse o módulo Interdados e localize o cadastro da empresa que deseja deletar.
  2. Siga a ordem inversa do cadastro para realizar a exclusão:
     * Organização de Inventário;
     * Unidade Operacional;
     * Estabelecimento;
     * Empresa.
  3. Verifique se há outros estabelecimentos que compartilham a mesma Organização de Inventário ou Unidade Operacional. Caso existam, será necessário excluir todos os cadastros associados.
  4. Após excluir os cadastros compartilhados, proceda com a exclusão da empresa desejada.
  5. Por fim, recadastre os itens necessários, como Unidade Operacional e Organização de Inventário, para os estabelecimentos que devem permanecer no sistema.


Se você precisar de ajuda em qualquer etapa, entre em contato com o suporte.

## Próximos Passos

Após aplicar a solução, recomendamos:

  * Realizar testes para garantir que a exclusão foi realizada corretamente.
  * Verificar se o recadastro das informações foi concluído com sucesso.
  * Monitorar o sistema para evitar inconsistências nos cadastros.


## Dicas e Recomendações

Para evitar problemas futuros, considere as seguintes melhores práticas:

  * Mantenha um controle detalhado dos cadastros antes de realizar exclusões, garantindo que não haja dependências ocultas.
  * Documente os passos realizados durante o processo de exclusão para facilitar futuras consultas.
  * Considere criar backups regulares do sistema antes de realizar alterações significativas.
