# [Municipal][BETHA]  Fazenda Rio Grande - Como ajustar o erro " Linha: 4 Mensagem: O município '0' informada para o item do documento fiscal não está cadastrada no banco de dados da prefeitura!"?

Quando apresentado o erro: 'Linha: 4 Mensagem: O município '0' informada para o item do documento fiscal não está cadastrada no banco de dados da prefeitura!', isso pode ocorrer devido ao não preenchimento do campo Município de ISS na SAFX07.

Após ajustar a configuração e gerar a nota fiscal, verifique minuciosamente se o campo "Código do Município" está sendo preenchido conforme esperado. Caso o problema persista, por favor, entre em contato com o suporte técnico para uma análise mais detalhada.

**Nota:** É recomendável revisar regularmente as configurações de campos críticos como o "Código do Município" para evitar inconvenientes futuros. Manter uma documentação detalhada das configurações e alterações realizadas pode ajudar na rápida identificação e resolução de problemas similares no futuro.
