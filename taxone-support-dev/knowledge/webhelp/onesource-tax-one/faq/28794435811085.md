Para que o registro F800 gere os valores corretos no Bloco M, a data de apropriação do crédito deve estar correta e os parâmetros de geração de registros devem estar configurados adequadamente. Siga as orientações:

  1. Verifique se a data de apropriação do crédito está correta e coincide com o período de apuração no sistema.
  2. Confira se todos os parâmetros de geração de registros estão configurados corretamente.
  3. Após verificar e ajustar as configurações, gere novamente os registros para o Bloco M.
  4. Se o problema persistir, verifique se há atualizações ou patches que precisam ser aplicados ao sistema.
  5. Após realizar os ajustes, monitore os próximos períodos para garantir que o problema foi resolvido. Caso os valores ainda não sejam apresentados corretamente, entre em contato com o suporte técnico para investigação adicional.


**Importante:** Assegure-se de manter o sistema sempre atualizado e de revisar regularmente os parâmetros de configuração. Além disso, mantenha registros detalhados dos lançamentos realizados para facilitar a verificação e correção de eventuais problemas.
