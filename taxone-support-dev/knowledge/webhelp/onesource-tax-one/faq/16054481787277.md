Normalmente, esse problema ocorre pois o Valor do Lançamento Contábil não esta cadastrado ou esta preenchido com zero.

Para corrigir o erro 90101: 'O campo Valor do Lançamento Contábil deve ser preenchido.', siga a orientação abaixo:

  * Preencha o campo VLR_LANCTO com um valor válido.

Veja que no arquivo, o campo esta preenchido com zero.
