# Como Salvar Relatórios R-9001 em Formato Excel

## Problema:

Os usuários estão enfrentando dificuldades para salvar relatórios R-9001 obtidos em formato CSV no formato Excel e solicitam a funcionalidade de geração automática no formato Excel.

## Causa Raiz:

Atualmente, os relatórios R-9001 são gerados pela RFB no formato CSV, que pode ser aberto em Excel. No entanto, o sistema não realiza a formatação automática em Excel e exige que os usuários utilizem o recurso "Texto para Colunas" do Excel para ajustar o conteúdo.

## Resolução:

Siga os passos abaixo para converter o arquivo CSV em formato Excel utilizando o recurso "Texto para Colunas" do Excel:

  1. Abra o arquivo CSV no Microsoft Excel.
  2. Selecione a coluna que contém os dados desformatados.
  3. Na barra de menu superior, clique em **Dados**.
  4. Selecione a opção **Texto para Colunas**.
  5. No assistente que aparecerá, escolha **Delimitado** e avance para a próxima etapa.
  6. Selecione **Tabulação** como o delimitador principal e avance.
  7. Finalize o assistente clicando em **Concluir**.
  8. Verifique se os dados foram corretamente formatados e salve o arquivo como Excel (**.xlsx**) utilizando **Arquivo Salvar Como**.


## Próximos Passos:

Após salvar o relatório no formato Excel, oriente os usuários a compartilhar o arquivo devidamente formatado.

## Dicas e Recomendações:

  * Reforce com os usuários que o arquivo CSV é compatível com Excel e que o processo de formatação é simples e eficiente ao seguir o passo a passo descrito acima.
  * Considere treinar os usuários que tenham menos familiaridade com o Excel em relação à funcionalidade "Texto para Colunas".
