# [Interfaces] [ORACLE Interdados - V17R01 Cloud] [Geral] Erro no Campo IDENT_FIS_JUR na Interface SAFX07 de Entradas

O objetivo deste artigo é orientar o usuário à respeito das possíveis causas que geram erro no campo IDENT_DIS_JUR e sua respectiva resolução na Interface SAFX07 de Entradas.

**Problema:**

Ao gerar a interface SAFX07 de entradas, o campo "06 - IDENT_FIS_JUR" está sendo exibido com o valor "@". Esse comportamento é diferente de gerações anteriores, nas quais o campo era preenchido corretamente.

**Causa Raiz:**

O problema ocorre devido à ausência de preenchimento do campo tag "G_SUPPLIERS" campo "SUPPLIER_TYPE" no arquivo XML, extraídos na origem dos dados (sistema responsável por alimentar a interface SAFX07). Quando esse campo está vazio, o processo de conversão para o formato JSON não consegue gravar a informação de maneira adequada, gerando inconsistências na interface.

**Resolução:**

Siga os passos abaixo para corrigir o problema identificado:

  * Verifique no XML de origem os campos das notas fiscais relacionadas (NF), como "SUPPLIER_TYPE". Certifique-se de que este campo esteja devidamente preenchido.
  * Caso o campo "SUPPLIER_TYPE" esteja vazio, corrija a informação no sistema de origem, alimentando o campo com o valor correto.
  * Verifique as parametrizações da regra De Para 18.
  * Aplique uma nova carga no sistema para enviar os dados corrigidos.
  * Valide novamente a interface SAFX07 para garantir que a informação no campo "06 - IDENT_FIS_JUR" foi preenchida corretamente.


**Próximos Passos:**

  * Confirme o preenchimento correto do campo "SUPPLIER_TYPE" antes de futuras cargas.
  * Realize testes para verificar se o problema foi de fato resolvido e que todas as notas fiscais estão sendo processadas corretamente.
  * Documente a alteração e compartilhe com a equipe envolvida, garantindo que boas práticas sejam aplicadas sistematicamente no processo de geração de interface.


**Dicas e Recomendações:**

Realize revisões periódicas no processo de geração de arquivos para identificar inconsistências antes de carregá-los no sistema.

Engaje as equipes técnicas responsáveis pelo sistema de origem e pela geração da interface para alinhamentos sobre campos obrigatórios e mapeamentos importantes.

Para mais informações sobre como configurar e corrigir problemas na interface SAFX07, consulte a nossa base de conhecimento ou entre em contato com o suporte técnico.
