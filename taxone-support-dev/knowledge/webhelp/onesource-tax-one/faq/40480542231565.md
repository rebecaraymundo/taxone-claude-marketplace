O objetivo deste artigo orientar o consultor\cliente sobre a crítica do interdados quando o cliente executa o programa e ocorre a mensagem de Versão do client não está compatível com a versão do patch.

### Problema

Ao realizar a instalação do interdados e o processo de usuário e senha do MSAFI e XXISV realizado, ao clicar no executável para iniciar o programa o cliente\consultoria se depara com a seguinte mensagem:
"A versão do client não está compatível com a versão do patch"
O sistema solicita atualizar o executável para a versão solicitada.
![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40480558097165)

### Resolução

Para resolver o problema em questão inicialmente o cliente\consultoria deverá realizar uma consulta da tabela MAFI_MSFTS_CTL_PATH.
Exemplo: select * from MAFI_MSFTS_CTL_PATH order by dsc_patch desc;
Verificar a como está a versão da linha que contenha o V11R010_046_CLIENTVB_VC11R02.0.111 e se realmente mostrar como aparece na versão do client da imagem do problema, o cliente\consultoria deverá realizar o procedimento a seguir.

1 - Alterar a linha da tabela MSAFI_MSFTS_CTL_PATCH de: V11R010_046_CLIENTVB_CV11R02.0.11 para V11R010_046_CLIENTVB_CV11R03.0.1 que é a versão esperada pelo sistema para o patch em questão.
2 - Abrir o executável e verificar se ele funcione normalmente. Se ocorrer uma mensagem informando que o Interdados foi colocado em estado de manutenção devido a aplicação do patch, deverá realizar o passo a seguir:
![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40480558097421)


### Próximos Passos

Após realizar os processos acima, o interdados está pronto para seguir os próximos passos descrito o manual.
Caso venha a ocorrer mais algum problema, um ticket deverá ser aberto ao time de Suporte Técnico com as informações e evidências destacadas para que uma nova análise seja realizada.

### Dicas e Recomendações

Assegure-se sempre de estar utilizando as versões de softwares homologadas para evitar problemas de compatibilidade e erros inesperados durante a instalação. Consulte regularmente as notas de versão fornecidas pelo suporte para ficar informado sobre quaisquer mudanças ou atualizações das homologações.
