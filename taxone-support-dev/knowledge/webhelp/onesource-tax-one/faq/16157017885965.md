Normalmente, esse problema ocorre pois o campo Data da DDE/DSE não está preenchido ou está preenchido incorretamente.

Para corrigir o erro 91287: 'O campo Data da DDE/DSE é inválido.', siga a orientação abaixo:

  * Preencha o campo com uma data válida.
