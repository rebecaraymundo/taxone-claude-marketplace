Para resolver o erro no registro 0175: "_ERRO: Não é permitida alteração da UF e/ou IE do participante. Deve ser criado um novo participante no registro 0150. Registro : 0175_ ", verifique as informações abaixo:

# Registro: 0175 da EFD-ICMS/IPI - Alteração da Tabela de Cadastro de Participante

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/34879461426445)

O registro 0175 deve ser preenchido obrigatoriamente sempre que ocorrerem alterações, dentro do período, nos seguintes dados do registro 0150:

NOME, COD_PAIS, CNPJ, CPF, COD_MUN, SUFRAMA, END, NUM, COMPL e BAIRRO.

É importante não utilizar o mesmo código para representar diferentes participantes em um único arquivo.

As informações deste registro são válidas até as 24 horas do dia anterior à data da alteração.

Em casos de alteração na **Inscrição Estadual** , deve-se criar um novo participante, e o registro 0175 não deve ser utilizado.

**Atenção:** O registro 0175 só deve ser utilizado para mudanças de endereço quando, no mesmo mês, houver emissão ou recebimento de duas ou mais notas fiscais em endereços diferentes para o mesmo participante.

Caso a mudança de endereço também envolva uma alteração na Inscrição Estadual, um novo registro 0150 deve ser criado para o novo participante, e o registro 0175 não deve ser utilizado.

ERRO: **Não é permitida alteração da UF e/ou IE do participante. Deve ser criado um novo participante no registro 0150. Registro : 0175**

## Resolução:

  1. Acesse o módulo Básicos > Data Warehouse, menu Manutenção > Cadastros > Pessoas Físicas/Jurídicas.
  2. Verifique o cadastro do Participante que está gerando o erro, e confirme se estão com a informação de Inscrição Estadual.
  3. Caso a Inscrição Estadual não esteja informada, preencha a informação e faça uma nova geração.
