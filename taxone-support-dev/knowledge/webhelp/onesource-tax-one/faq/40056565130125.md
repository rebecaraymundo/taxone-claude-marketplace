## Problema:

O campo 45-OBS_ICMS na SAFX08 da Namespace não está na descrição da origem na SE16N.

## Causa Raiz:

A principal causa de dificuldade para realizar a localização do campo OBS_ICMS se deve à forma como está parametrizado, este campo é composto por varias condições e verificação para realizar a composição do mesmos.

## Resolução:

Para que seja possível verificar a origem da informação sera necessário realizar a analise via código fonte referente ao processo.

  1. Acessar a transação SE38
  2. Informar o include: /MSAFX/LSAFE_PROCESSOSF01
  3. Buscar o campo itmtyp dentro da tabela j_1bnflin, esta é a primeira verificação para compor o campo. Atualmente temos mais de 200 linhas de código para compor este campo.
