Para solucionar o erro 92658: 'O campo UF e de procedimento obrigatório quando o indicador da apuracao = ICMS-ST' referente à abertura do layout SAFX218 e preenchimento do campo COD_ESTADO de forma simples, siga estes passos:

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21793926412813)

  1. Acesse o sistema que contém o layout SAFX218.
  2. Procure pelo campo COD_ESTADO no layout SAFX218.
  3. Preencha o campo COD_ESTADO com o valor adequado, se estiver vazio ou com valor incorreto.![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21793912825613)
  4. Salve as alterações e verifique se o erro foi corrigido.
  5. Faça um teste para garantir que o campo foi preenchido corretamente e que o erro não está mais ocorrendo.
