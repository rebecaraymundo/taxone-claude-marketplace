Para atender aos cenários em que no procedimento da SAFX04 não são enviados dados para o campo COD_MUNICIPIO, apresentado a falta de dados como @ (vazio):

  1. O Interdados não trata essa informação.
  2. A resposta para a situação está na planilha do DW (mesmo o cliente utilizando o Tax one), pois o Tax One possui um parâmetro o qual o município é preenchido de acordo com o CEP.
