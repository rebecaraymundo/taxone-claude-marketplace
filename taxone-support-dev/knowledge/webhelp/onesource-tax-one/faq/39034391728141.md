### **Disponibilidade da API de Cadastro de Usuários no módulo Administration**

**Problema:** A **API de Cadastro de Usuários no módulo Administration** ainda não está disponível para uso.

**Causa Raiz:** Essa API está em fase de desenvolvimento e será lançada em uma versão futura do produto, conforme o nosso cronograma oficial.

**Resolução:** No momento, a **API de Cadastro de Usuários no módulo Administration** não pode ser utilizada. Para acompanhar a previsão de liberação, consulte o **Roadmap** oficial no link: <https://roadmap.thomsonreuters.com/tax_taxone_2025.html>. Lembre-se que a data de lançamento pode variar, pois se trata de uma previsão.

**Próximos Passos:**

  * Acompanhe as atualizações diretamente no [Roadmap oficial](<https://roadmap.thomsonreuters.com/tax_taxone_2025.html>) ou nos **Release Notes** oficiais.
  * Para dúvidas, entre em contato com o suporte pelos canais oficiais.


**Dicas e Recomendações:**

  * Para que sua necessidade seja considerada no desenvolvimento, registre sua sugestão no portal **Customer Influencer**. Isso ajuda a equipe de produto a priorizar melhorias com base na demanda dos usuários.
