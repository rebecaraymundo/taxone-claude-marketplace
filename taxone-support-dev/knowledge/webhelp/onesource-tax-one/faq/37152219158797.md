Neste artigo abordaremos:
Registro E250 - Específico UF RJ: Campo 05 - Código Receita - Como alterar a geração do código 0230 para 100048?

Iniciaremos mostrando o cadastro do estabelecimento de geração do arquivo SPED Fiscal.

Cadastro do Estabelecimento: Básicos > Parâmetros > Preliminares > Empresa/Estabelecimento
O estabelecimento que fará a geração do arquivo é: LIMA

![KBA NOTA CAMPO 05 - REG E250 - RJ - PRINT 1.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/37925727212557)

A geração do Registro E250 deste artigo, é referente a UF RJ, para gerá-lo é necessário ter a guia para a UF e receita em questão. Neste caso a guia já se encontra gerada na GNRE Online. Veja a seguir:

Estadual > ICMS > Apuração > Guias de Recolhimento > Guias Recolhimento Tributos Estaduais (GNRE Online) > Manutenção
(Para entender a geração da guia basta acessar nossas FAQs da GNRE Online)

Veja que a guia está dentro do período 04/2025, que a receita é a 100048 - ICMS Substituição Tributária, e a UF favorecida é: RJ.

![KBA NOTA CAMPO 05 - REG E250 - RJ - PRINT 2.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/37925727212685)

Em seguida, iremos para o módulo SPED Fiscal.

Perfil: SPED > EFD - Escrituração Fiscal digital > Parâmetros > Perfil
Veja que o perfil que utilizaremos na geração é o FB - PERFIL 2025, e nele marcamos o Registro E250:

![KBA NOTA CAMPO 05 - REG E250 - RJ - PRINT 4.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/37925739495949)


Dados Iniciais: SPED > EFD - Escrituração Fiscal digital > Parâmetros > Dados Iniciais

Um ponto de atenção: o parâmetro "Considerar o Código da Receita da Guia de Recolhimento (UF=RJ)", quando marcado considera o código da receita utilizado na guia de recolhimento para preencher o campo 05 – COD_REC, com a receita 100048 e não 0230.
Faremos a geração no primeiro momento, com o parâmetro desmarcado:

![KBA NOTA CAMPO 05 - REG E250 - RJ - PRINT 3.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/37925727213453)

Vamos a primeira geração do arquivo:
SPED > EFD - Escrituração Fiscal digital > Geração > Meio Magnético > Gerar o SPED

![KBA NOTA CAMPO 05 - REG E250 - RJ - PRINT 5.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/37925727213837)

Na aba QUANTIDADE TOTAL DOS REGISTROS GRAVADOS, é possível ver que o registro E250 foi gerado:

![KBA NOTA CAMPO 05 - REG E250 - RJ - PRINT 6.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/37925727216269)

Com o arquivo txt aberto, podemos ver que no campo 05 do Registro E250, gerou com o código 0230:

![KBA NOTA CAMPO 05 - REG E250 - RJ - PRINT 7.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/37925727216525)

Voltando ao parâmetro dos Dados Iniciais, desta vez marcaremos o parâmetro Considerar o Código da Receita da Guia de Recolhimento (UF=RJ):

![KBA NOTA CAMPO 05 - REG E250 - RJ - PRINT 8.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/37925739497613)

E agora faremos uma nova geração do SPED:

![KBA NOTA CAMPO 05 - REG E250 - RJ - PRINT 9.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/37925739498253)

Vamos direto ao arquivo txt. Veja que desta vez, no campo 05, foi gerado o código 100048:

![KBA NOTA CAMPO 05 - REG E250 - RJ - PRINT 10.png](https://suporte.thomsonreuters.com.br/hc/article_attachments/37925739500813)
