## Problema:

O usuário não consegue visualizar as contas de Patrimônio Líquido na tela de manutenção das aglutinações na DMPL, apesar de existirem registros ativos dessas contas no plano de contas.

## Causa Raiz:

A causa do problema está relacionada à configuração do código de aglutinação. Na tela de DMPL, é necessário que o código de aglutinação vinculado à conta contábil no balanço patrimonial seja o mesmo usado na DMPL.

## Resolução:

  1. Verificar no balanço patrimonial o código de aglutinação associado à conta de Patrimônio Líquido de interesse.
  2. Garantir que o mesmo código de aglutinação esteja configurado na tela de manutenção das aglutinações na DMPL.
  3. Caso o código não esteja vinculado, realizar a vinculação e salvar as alterações.


## Próximos Passos:

Após realizar os passos acima, é recomendável testar a visualização do relatório de DMPL para garantir que as contas de Patrimônio Líquido estão aparecendo corretamente. Caso o problema persista, entre em contato com o suporte técnico para uma análise mais detalhada.

## Dicas e Recomendações:

Para prevenir futuras discrepâncias, sempre verifique as configurações de aglutinação antes de efetuar lançamentos contábeis ou criar novas configurações de relatório. Considere realizar revisões periódicas das configurações de contabilidade para assegurar a correta visualização e funcionalidade dos relatórios.
