# [Básicos][Job Servidor] TAX ONE - SAFX218 - Como corrigir o erro 92657?

O erro 92657: 'O campo UF só deve ser informado para os lancamentos referentes a Apuracao ICMS-ST' indica que o campo UF (Unidade Federativa) só deve ser preenchido para lançamentos relacionados à Apuração do ICMS-ST na tabela SAFX2016, siga as etapas abaixo para resolver o problema.

1\. Acesse o sistema que hospeda a tabela SAFX2016.

2\. Navegue até o campo COD_ESTADO na tabela SAFX2016.
3\. Confirme se o lançamento em questão refere-se à Apuração do ICMS-ST.
4\. Ao inserir ou editar dados para registros relacionados à Apuração do ICMS-ST, valide o campo COD_ESTADO:

  * Se o campo UF (Unidade Federativa) estiver vazio ou não estiver preenchido corretamente, ajuste-o de acordo com as regras de preenchimento.
  * Se o campo UF não deve ser informado para esse tipo de lançamento, verifique se está vazio.


5\. Salve as alterações realizadas e confirme se o Erro 92657 foi solucionado.

**Considerações Finais**

Certifique-se de que os registros relacionados à Apuração do ICMS-ST na tabela SAFX2016 tenham o campo UF preenchido corretamente ou vazio, conforme as regras de validação do sistema. Documente qualquer alteração realizada para referência futura.
