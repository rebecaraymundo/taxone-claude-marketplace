O objetivo deste artigo é orientar o usuário a respeito da compatibilidade dos provedores Java com o Interdados Oracle V11R01

**Dúvida:**

Os usuários necessitam saber se é possível substituir o Java Sun por outros provedores de Java em servidores que utilizam o Interdados Oracle V11R01.

**Resposta:**

O Interdados V11 depende do Java Sun 32 bits versão 1.8 para operar corretamente. Mudanças no provedor de Java podem causar incompatibilidades e problemas no funcionamento do sistema.

**Dicas e Recomendações:**

Certifique-se de manter todas as informações de seu sistema atualizadas e documentadas, incluindo versões e configurações de software. Verifique regularmente se há atualizações ou patches de segurança necessários para o Java utilizado.
