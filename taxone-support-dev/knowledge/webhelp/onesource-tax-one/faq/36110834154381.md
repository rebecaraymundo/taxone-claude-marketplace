# Resolução de Problemas com a Geração do Arquivo MOV001 no Onesource BR

## Problema:

O arquivo MOV001 não está sendo gerado no módulo Onesource BR, mesmo com lançamentos inseridos no ambiente.

## Causa Raiz:

O problema ocorre devido à falta de parametrização das contas contábeis que devem ser incluídas no arquivo MOV001 no módulo Conexão Onesource BR.

## Resolução:

  * Acesse o módulo **Conexão Onesource BR**.


  * Navegue até **Parâmetros > Parametrização das contas contábeis**.


  * Verifique se as contas que devem ser geradas no arquivo MOV001 estão listadas. Caso não estejam, selecione-as e adicione-as à lista.


  * Após a parametrização, tente gerar o arquivo novamente e verifique se os registros foram lidos corretamente.


## Próximos Passos:

Após seguir os passos de resolução, monitore o sistema para confirmar que o arquivo MOV001 é gerado corretamente. Se o problema persistir, entre em contato com o suporte técnico para assistência adicional.
