# [SPED][EFD - ECD]TAX ONE - ECD - Resolução de Problema com a Parametrização do Responsável Legal no Módulo ECD

#### Problema:

Usuários podem enfrentar dificuldades ao tentar parametrizar o responsável legal nos parâmetros de dados iniciais para a geração do sped contábil, onde o sistema da erro informando que o cadastro do responsável legal não existe ou está inativo, mesmo constando ativos nos cadastros.

#### Causa Raiz:

O problema ocorre devido à uma lentidão geral no sistema, que pode gerar inconsistências ao carregar alguns dados, como é o caso dos responsáveis legais não aparecerem ativos no módulo para a ECD. Isso pode estar relacionado com a performance do sistema ou com especificidades do cadastro de inatividade.

#### Resolução:

Siga os passos abaixo para corrigir a parametrização: Acesse o módulo de cadastro do responsável legal. Localize o campo chamado "Responsável Inativo". Desmarque e marque novamente essa opção, mesmo que não desejasse alterá-la inicialmente. Tente novamente carregar o responsável na tela do módulo ECD. Essa ação pode causar instabilidades temporárias, mas permite que o cadastro seja reconhecido no módulo desejado.

#### Próximos Passos:

Após seguir os passos recomendados, monitore o sistema para verificar se a alteração foi efetiva. Em caso de persistência do problema, entre em contato com o suporte técnico para uma análise mais detalhada.

#### Dicas e Recomendações:

Recomenda-se verificar periodicamente as atualizações do sistema para garantir que todas as funcionalidades estejam operando corretamente. Mantenha também o cadastro dos responsáveis legais atualizado e revisado para evitar futuros inconvenientes.
