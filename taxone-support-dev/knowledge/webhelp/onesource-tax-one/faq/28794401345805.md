**Problema:** O usuário enfrenta dificuldades para carregar a tabela SAFX04 via WinSCP para o sistema TaxOne, mesmo seguindo as instruções corretas de nomenclatura do arquivo.

**Causa raiz:** O problema ocorre porque a empresa não estava parametrizada corretamente para a carga da tabela SAFX04 no sistema.

Passo a passo para resolução:

  1. Acesse o sistema e vá até a seção de ferramentas.
  2. Verifique se a empresa está corretamente configurada para receber cargas da tabela SAFX04.
  3. Se não estiver, realize a parametrização adequada da empresa dentro das ferramentas do sistema.
  4. Após a parametrização, tente realizar a carga da tabela novamente via WinSCP.
  5. Próximos passos: Após realizar a carga, confirme se a tabela SAFX04 aparece corretamente no sistema Tax One. Monitore o sistema para verificar se outras tabelas estão sendo carregadas corretamente e se não há falhas recorrentes.


Dicas e recomendações:

Verifique sempre as configurações de parametrização antes de realizar cargas importantes no sistema.
Consulte a documentação técnica ou entre em contato com o suporte técnico se encontrar dificuldades ou inconsistências nas configurações.
