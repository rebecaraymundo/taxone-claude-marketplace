A Interface atende o S/4HANA através de views de compatibilidade do ECC apenas. Neste caso, o Basis deve configurar o SAP para que haja permissão de leitura nestas tabelas/views do ECC.

Pontos de atenção (quando utilizadas versões Clouds):

**Servidor Cloud da SAP:**

  * Cloud público: Não poderá utilizar a Namespace.
  * Cloud privado: Deverá avaliar com a SAP se há restrição e solicitar autorização para uso.


**Servidor Cloud do Cliente**

  * Não há restrições


Importante: A Interface Namespace/OBTI é certificada pela SAP para integração com o o S/4HANA.
