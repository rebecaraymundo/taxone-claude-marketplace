O erro "**Numeração do item inválida. Informar em ordem sequencial** " ocorre durante a validação do arquivo no PVA da EFD Contribuições quando há uma numeração incorreta dos itens em notas de serviço que não possuem crédito de PIS/COFINS. Esse problema surge porque alguns itens das notas de serviço estão sem as informações necessárias de PIS e COFINS, que são obrigatórias para a apuração por item. Durante a validação do arquivo, o sistema detecta a ausência dessas informações e isso causa o erro de numeração.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/32155628427661)

Passo a passo para resolver o erro "Numeração do item inválida. Informar em ordem sequencial" na EFD Contribuições para notas de serviço sem crédito PIS/COFINS:

  1. Verifique cada item da nota fiscal de serviço para confirmar se todos possuem as informações de PIS e COFINS preenchidas corretamente.
  2. Para os itens que não devem ter crédito de PIS e COFINS, verifique se o campo "**data de lançamento PIS/COFINS** " está devidamente preenchido. Caso esteja vazio, preencha a fim de ajustar o sistema para que eles não sejam considerados na geração do arquivo de validação.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/32155597742221)
_![](https://suporte.thomsonreuters.com.br/hc/article_attachments/32155628439949)_
  3. Equalize as informações no sistema após fazer os ajustes necessários.
  4. Gere novamente os registros e o meio magnético para a validação.
  5. Após implementar a solução, é importante realizar um novo teste de validação para garantir que o arquivo está sendo gerado corretamente sem erros. Monitore os próximos envios para confirmar que o problema foi resolvido definitivamente.


**Dicas e Recomendações:**

É recomendável revisar regularmente as configurações de itens em notas fiscais para garantir que todas as informações necessárias estão sendo capturadas corretamente, especialmente em relação aos créditos fiscais como PIS e COFINS. Isso ajuda a evitar erros de validação e problemas na apuração de impostos.
