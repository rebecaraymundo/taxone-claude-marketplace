Objetivo deste artigo é orientar a atualização de nova senha do Oracle Cloud no Interdados.

Passo a passo para alterar a senha do Oracle Cloud no Interdados:

  1. No Interdados, acesse _Opções > Parâmetros de Configuração_.
  2. Na página de parametrizações, role até encontrar a seção "INTEGRAÇÃO ORACLE CLOUD".
  3. Verifique se o "**Usuário Oracle Cloud** " está correto, de acordo com o utilizado para acesso ao Oracle Cloud.![](https://suporte.thomsonreuters.com.br/hc/article_attachments/28278632679309)No campo "**Senha Oracle Cloud** ", substitua a senha atual pela nova senha definida no processo de troca de senha de acesso do Oracle Cloud.
  4. Após alterar a senha, clique no botão "**SALVAR** " no final da página.
  5. Agora você pode executar uma carga para testar a nova senha.


Lembre-se de verificar se o "Usuário Oracle Cloud" está correto antes de substituir a "Senha Oracle Cloud" pela nova senha definida no Oracle Cloud.
