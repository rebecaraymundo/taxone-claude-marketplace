Para gerar registros para integração dos dados de empresa/estabelecimento no formato CAD0000 para posterior integração no Onesource ECF, siga os passos abaixo:

1\. Realize o cadastro da empresa/estabelecimento em [Básicos > Parâmetros > Preliminares > Empresa/Estabelecimento](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Basicos/Basicos_Parametros/Parametros_Menu_Preliminares/Param_Preli_Empresa_Estabelecimento.htm>) > Aba Informações Básicas.

2\. Realize as parametrizações do módulo de integração. Para realizar a geração do arquivo é necessário que o módulo esteja devidamente parametrizado, conforme no artigo: [Como parametrizar CNPJ da Central de Cadastros?](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/microcontent/FAQ_-_CONEXAO_ONESOURCE-08-05.mc#ECF_param_CNPJ_Central>)

3\. Realize a Geração dos Dados no menu [Geração](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Conexao_Onesource_BR/ECF_Escrituracao_Contabil_Fiscal/ECF_Geracao/ECF_Geracao_Dados.htm>).

Nota: Considerando a geração, poder ser realizada para inclusão ou exclusão. No sentido de geração de dados para exclusão, temos que nos ater ao princípio de que para se gerar as informações de exclusão, primeiramente, deve ser realizada a exclusão dos registros que deseje excluir, para que possa ser gerado o arquivo com leiaute de exclusão.
