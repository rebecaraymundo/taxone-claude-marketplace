# [SPED][EFD - REINF]TAX ONE - SPED - EFD - REINF - O Relatório de Conferência de Tributos Federais não apresenta o Controle de Tributos de algum estabelecimento, como resolver?

Segue a possível causa e a solução, para os casos em que não são gerados dados no Relatório de Conferência de Tributos Federais, ou então gerar os dados de algum(ns) estabelecimento(s), porém não gera de outro(s).

Passo a passo:

**1** \- Realize a emissão do relatório em Relatórios > Federais > Relatório de Conferência de Tributos Federais.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/20856163787149)

**2** \- Origem dos dados em Manutenção > Retenções > Controle de Tributos.

Este relatório tem como origem os dados de Controle de Tributos (SAFX53).

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/20856177174925)

**3** \- Inscrição Estadual em Parâmetros > Inscrições Estaduais.

Para que os dados do estabelecimento sejam considerados na emissão do relatório, é necessário que o estabelecimento possua o cadastro de Inscrição Estadual.

Se o estabelecimento em questão não possui 'Inscrição Estadual' e desejam que o mesmo apresente os dados no relatório, então devem efetuar um cadastro informando 'ISENTO' no campo 'Inscrição Estadual'.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/20856505938061)
