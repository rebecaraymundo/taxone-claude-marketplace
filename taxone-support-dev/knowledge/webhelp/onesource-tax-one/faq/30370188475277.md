Passo a passo para corrigir o problema de extração do campo SERIE_DOCFIS nas SAFX07, SAFX08 e SAFX09:

  1. Acesse o layout do procedimento SAFX07 ou SAFX08.
  2. Localize o campo SERIE_DOCFIS e dê um duplo clique sobre ele.
  3. No campo Regra De-Para, verifique se há alguma regra associada. Caso haja, remova e salve a configuração;
  4. Execute novamente o processo de carga após corrigir a parametrização.


**Nota** : Conforme o mapeamento técnico, o campo SERIE_DOCFIS não deve ter nenhuma regra associada.
