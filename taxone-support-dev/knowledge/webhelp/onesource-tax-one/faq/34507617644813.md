## Problema:

Usuário não consegue encontrar códigos específicos no campo "chave oficial IRF" ao parametrizar a Interface Namespace em Tipo de Impostos Outros.

## Causa Raiz:

Os códigos específicos não estavam previamente cadastrados no sistema para serem selecionados no campo solicitado.

## Resolução:

  * Acesse o sistema SAP ir na transação de **Parametros Gerais**.


  * Selecione **Tipo de Impostos (Outros)** e clique em **chav. Oficial**.


  * Clique em **novo** para adicionar um novo código.![](https://atendimentotr.zendesk.com/attachments/token/yshdl3cAacLRKaTiGvxPYvK5O/?name=image.png)


  * Informe o código que deseja adicionar (por exemplo, 2631).![](https://atendimentotr.zendesk.com/attachments/token/zDX1WSUMkfZOvOhqjTxTyYOxh/?name=image.png)

  * Selecione **Continuar sem indicações de projeto** para a configuração padrão.


  * Realize as parametrizações necessárias para cada código conforme sua necessidade específica.

![](https://atendimentotr.zendesk.com/attachments/token/rfvdjItgDn6fd7DQHdhd4b4lR/?name=image.png)
  * realizado estes passos sera exibido o novo codigo![](https://atendimentotr.zendesk.com/attachments/token/ZYJT77kgj0gGXMW2T2kaXWNAp/?name=image.png)


## Próximos Passos:

Após realizar o cadastro dos códigos necessários, verifique se eles aparecem corretamente no campo "chave oficial IRF". Caso haja algum problema, entre em contato com o suporte técnico para mais assistência.

## Dicas e Recomendações:

É recomendável revisar periodicamente as configurações de impostos no sistema SAP para garantir que todas as informações estejam atualizadas e corretas. Isso ajudará a evitar problemas futuros com a falta de códigos ou outras configurações inadequadas.
