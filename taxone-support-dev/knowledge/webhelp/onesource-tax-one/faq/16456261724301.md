Para fazer a extração do DUMP completo para análise do erro:

  1. Acesse a transação ST22.

  2. Informe os parâmetros necessários para visualização do Dump e clique no botão "Ínicio":


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/25175824557453)

  1. Selecione a linha do erro e clique no primeiro ícone para exibir o texto:


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/25175824582669)

  1. Para salvar o documento, clique em  _Sistema > Lista > Gravar > File Loca_ _l:_


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/25175841173645)
