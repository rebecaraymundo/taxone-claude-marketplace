Nota: Este erro é identificado ao realizar uma carga no Interdados Oracle V11R01 e a mesma é finalizada com a mensagem de erro: 'Erro: java.sql.SQLException: 500 null'.

Para corrigir o problema, siga o procedimento abaixo:

  1. Se você não tiver acesso ao **Banco de Dados** , entre em contato com seu DBA e solicite a execução da quary abaixo:
**select * from MSAFI_MSFTS_CONFIG**
  2. No campo **API_URL_TAX_WS** , verifique se a URL tem uma barra ( / ) no final. Se houver, remova a barra ( / ) através de update.

  3. Após o procedimento, refaça a carga.
