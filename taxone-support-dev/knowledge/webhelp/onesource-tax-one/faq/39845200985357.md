##

## Problema

Ocorre o DUMP abaixo ao realizar a extração do Documentário Fiscal via Banco a Banco:

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40751213218957)

An exception has occurred which is explained in more detail below. The exception is assigned to class "CX_SY_REF_IS_INITIAL" and was not caught in procedure "/MSAFX/GRAVAR_BANCO_MSAF""( FUNCTION)"

## Causa Raiz

A duplicidade é causada por uma nomenclatura duplicada nos campos 274 e 275 do layout, que afeta a inserção de dados na tabela /MSAFX/SAFTB057.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40751181166861)

##

## Resolução:

  1. Acesse a tabela /MSAFX/SAFTB057 usando a transação `SE16N` (print exemplo abaixo):
  2. Filtre os campos conforme necessário.
  3. Altere o valor da coluna FINAM para "026_8_IND_NF_ANT" e o valor da coluna NOME_ORACLE para "IND_NF_ANT" deverá ficar como ilustrado no print abaixo:

![](/attachments/token/MThEm8ddQo7al8MJUq32QPbas/?name=image.png)
  4. Após realizar essas alterações, tente novamente o envio da Documentário Fiscal via Banco a Banco.
