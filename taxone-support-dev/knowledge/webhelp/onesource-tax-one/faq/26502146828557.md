Passo a passo para geração do campo 25 - Valor do Ressarcimento da Ficha 3 do Ressarcimento ICMS-ST SP:

**Nota:** Este campo não será preenchido no caso de entradas (e suas devoluções) ou saídas para comercialização subsequente (e suas devoluções).

1\. Certifique-se de que o estabelecimento pertence à UF de São Paulo (SP) no módulo Básicos > Parâmetros, menu Preliminares > Empresa/Estabelecimento.

![1.jpg](https://suporte.thomsonreuters.com.br/hc/article_attachments/26502146771853)

2\. Realize o cadastro dos documentos fiscais de entrada e saída no módulo Básicos > Data Warehouse, menu Manutenção > Documento Fiscal > Documento Fiscal de Mercadoria e Serviço, aba Capa e Item.

Documento de Saída

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/26502146788493)

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/26502126539661)

Documento de Entrada

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/26502146792205)

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/26502126549901)

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/26502146798093)

3\. Execute a equalização no módulo Básicos > Data Warehouse, menu DATA MART > Equalização/Limpeza.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/26502146799629)

4\. Acesse o módulo Estadual > Ressarcimento de ICMS-ST SP, menu Parâmetros > Dados Iniciais para configurar os dados iniciais.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/26502126555405)

5\. Parametrize os produtos no módulo Estadual > Ressarcimento de ICMS-ST SP, menu Parâmetros > Produtos.
**Nota:** É possível realizar a parametrização por NCM e CEST.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/26502126556941)

6\. Parametrize os CFOPs de entrada e saída no módulo Estadual > Ressarcimento de ICMS-ST SP, menu Parâmetros > CFOP/ Natureza para a Ficha 3.

Saída

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/26502126557837)

Entrada

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/26502126559885)

7\. Gere os movimentos no módulo Estadual > Ressarcimento de ICMS-ST SP, menu Geração > Geração dos Movimentos.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/26502146810509)

8\. Calcule os saldos no módulo Estadual > Ressarcimento de ICMS-ST SP, menu Geração > Cálculo dos Saldos.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/26502146811789)

9\. Gere a Ficha 3 - Controle de Estoque no módulo Estadual > Ressarcimento de ICMS-ST SP, menu Relatórios > Ficha 3 - Controle de Estoque.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/26502146813197)

O campo 25 - Valor do Ressarcimento é preenchido da seguinte forma:

Campo **Valor Saída** (Valor total do ICMS suportado para o Enquadramento da Saída) - **Valor Confronto** (Valor Confronto ICMS Efetivo Saída - Campo 20 ou Valor Confronto-ICMS Efetivo Entrada - Campo 21).

Este campo não será preenchido no caso de entradas (e suas devoluções) ou saídas para comercialização subsequente (e suas devoluções).

Por exemplo, para uma nota de saída no enquadramento = 4, o campo será calculado conforme a fórmula acima.
