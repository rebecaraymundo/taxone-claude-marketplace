# [Básicos][Job Servidor] TAX ONE - SAFX01 - Como corrigir o erro 90383?

Acesse esta FAQ no [web help do produto](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/WebHelp/mergedProjects/FAQs/Basicos_JS_SAFX01.htm#basicos_js_safx01_12>).

Normalmente, esse problema ocorre porque o código informado no campo HISTPADRAO não está cadastro na tela Básicos > Data Warehouse > Manutenção > Códigos > Histórico Padrão.

Para corrigir o erro 90383: 'O Campo Codigo de Historico Padrao nao esta cadastrado', siga a orientação abaixo:

  * Informe um código valido no arquivo TXT, ou
  * Cadastre o código informado no arquivo TXT na tela Básicos >Data Warehouse > Manutenção > Códigos > Histórico Padrão.
