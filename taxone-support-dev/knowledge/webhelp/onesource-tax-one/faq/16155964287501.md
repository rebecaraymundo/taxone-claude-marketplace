# [Básicos][Job Servidor] TAX ONE - SAFX02 - Como corrigir o erro 90012?

Acesse esta FAQ no [web help do produto](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/WebHelp/mergedProjects/FAQs/Basicos_JS_SAFX01.htm#basicos_js_safx01_04>).

Normalmente, esse problema ocorre pois o código informado no campo COD_CONTA não está cadastro na tela Básicos > Data Warehouse > Manutenção > Códigos > Plano de Contas.

Para corrigir o erro 90012: 'Conta Contábil não cadastrada ou data de validade posterior a data de lançamento.', siga a orientação abaixo:

  * Informe um código valido no arquivo TXT, ou
  * Cadastre o código informado no arquivo TXT na tela Básicos > Data Warehouse > Manutenção > Códigos > Plano de Contas.
