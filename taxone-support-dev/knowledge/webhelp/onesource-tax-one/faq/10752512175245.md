Para realizar a parametrização da Operação por CFOP, siga os passos abaixo:


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/25613873696781) Nota: A parametrização de CFOP e Operação (entradas e saídas), tem como objetivo definir as operações a serem consideradas na geração dos movimentos de entradas, saídas e devoluções para RS.
Para cada uma destas operações o usuário deve informar os CFOP’s da entrada ou saída, conforme o caso. Assim, para cada operação são disponibilizados para seleção tanto os CFOP’s de entrada como de saída, para que o usuário possa informar os CFOP’s da operação principal.
A parametrização é utilizada na rotina de geração dos movimentos a partir das notas fiscais.

  1. Realize a **parametrização na rotina de geração dos movimentos de entradas e saídas** em Estadual > Ressarcimento de ICMS-ST RS > Parâmetros > Operações > CFOP.

  2. Para cada uma destas operações deve informar os **CFOP’s da entrada ou saída** , conforme o caso, bem como o CFOP de suas respectivas devoluções.


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/25613873696781)Nota: Quando existir na Tabela de Códigos Fiscais - SAFX2012 mais de uma ocorrência do mesmo CFOP, com datas de validade diferentes, será considerado para exibição apenas o registro de maior data de validade.

O mesmo código Fiscal de CFOP incluído não pode constar em outra parametrização de outra operação.
