Siga os requisitos de máquina dedicada conforme está no [Manual de Pré-Requisitos do Projeto, no tópico Configuração Máquina Dedicada.](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Interfaces/Mastersaf_DW/WEBHELP_ORACLE_EBS_Interdados_V11R01/Interdados_V11R01_EBS/Manuais_Tecnicos/Interface_Interdados_EBS_V11R010_000_PRE_REQUISITOS_PROJETO.htm> "Hyperlink \[../Interdados_V11R01_EBS/Manuais_Tecnicos/Interface_Interdados_EBS_V11R010_000_PRE_REQUISITOS_PROJETO.htm#conf_maquina_dedicada\]")

Configuração máquina dedicada:

  * Configuração Máquina Desktop dedicada.
  * Espaço em Disco: 1TB.
  * Memória: 16GB Ram.
  * Sistema Operacional: Windows Server 2003 Server ou superior.
  * Update do Windows devidamente atualizado.
  * Aplicação Interdados
  * Login administrador local
  * Acesso Internet
  * Banco de Dados: Oracle Database 19c
  * Oracle Client 12c 32Bits
  * Java JRE8 32Bits ou 64Bits Instalado


Esta configuração é válida para qualquer ambiente, seja Homologação, Dev ou ambiente Produtivo.
