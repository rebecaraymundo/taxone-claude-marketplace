Para corrigir o erro "Erro ao executar importação de definições de tabelas.404 null" ao tentar importar tabelas no Cadastro do Tax One, siga os passos abaixo:

1\. No Interdados Oracle Cloud V17R01, acesse _Cadastro Tax One > Configuração e Importação_.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21837704295565)

2\. Clique em **I mportar Definições de Tabelas**.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21837676618381)

3\. Neste momento o sistema apresenta o erro “**Erro ao executar importação de definições de tabelas.404 null.”**

**![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21837720569741)**

4\. Para corrigir o erro, acesse o menu principal, selecione "**Opções** " e depois **"Parâmetros de Configuração"**.

5\. Verifique o campo **"WEB SERVICE TAX ONE"** para assegurar que a URL da API de IMPORTAÇÃO DE DADOS esteja correta: [https://uat01.taxone.thomsonreuters.com/taxone-integrator](<https://uat01.taxone.thomsonreuters.com/taxone-integrator> "Hyperlink \[https://uat01.taxone.thomsonreuters.com/taxone-integrator\]").

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21837704304013)

6\. Caso a URL esteja incorreta, atualize-a para a URL fornecida acima.

7\. Retorne ao caminho I**nterdados > Cadastro TaxOne > Empresa** e tente adicionar a empresa manualmente mais uma vez.
