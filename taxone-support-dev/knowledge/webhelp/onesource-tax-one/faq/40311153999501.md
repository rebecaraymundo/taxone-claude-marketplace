## Problema:

Após a importação de alguns arquivos SAFX no sistema Tax One, o usuário não está conseguindo visualizar os logs de sucesso ou de erro, essenciais para a análise de possíveis problemas nos dados importados.

## Causa Raiz:

Este comportamento é causado pelo novo serviço de importação do Tax One, que cria automaticamente um novo job de importação automática. Registros que já passaram por um processo de importação e permanecem na tabela temporária não podem ser reimportados, o que leva à geração de logs vazios.

## Resolução:

Siga os passos abaixo para resolver o problema:

  1. Limpe (inicialize) a tabela temporária da SAFX.
  2. Realize uma nova carga de dados.
  3. Execute a importação novamente.
  4. Se os registros foram enviados via Webservice com parâmetros de Importação Automática, use o endpoint de Importação Automática para importá-los.
