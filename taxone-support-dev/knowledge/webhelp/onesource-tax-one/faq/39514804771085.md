### **Como Corrigir Inclusão Indevida de Notas Fiscais de Prestadores do Mesmo Município no Arquivo ISS.WEB**

* * *

#### **Problema**

Na geração do arquivo **ISS.WEB** , notas fiscais de prestadores localizados dentro do município estão sendo incluídas indevidamente, causando erros ao enviar o arquivo para o site da prefeitura.

* * *

#### **Causa Raiz**

O problema ocorre devido à parametrização ativa no **Módulo Municipal**. O parâmetro **“Prestador Dentro do Município”** está marcado sem o ajuste correto para excluir notas que atendem aos critérios exigidos pela legislação municipal.

* * *

#### **Resolução**

Para corrigir:

  1. Abra o sistema e acesse:
**Módulo Municipal → Parâmetro por Município → Parâmetros → Notas Fiscais Eletrônicas**
  2. Verifique se o parâmetro **“Prestador Dentro do Município”** está configurado corretamente.
  3. Ajuste conforme necessário e salve as alterações.
  4. ![](https://suporte.thomsonreuters.com.br/hc/article_attachments/39925261879693)
  5. Configure para desconsiderar notas fiscais com:

     * **Campo movto_e_s** diferente de 9 na tabela **safx07**.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/39925248838925)
     * Indicador de Tipo de Documento para Nota Fiscal Eletrônica (**IND_NF_ELETRONICA = 'S'**) na tabela **SAFX2005**.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/39925248843021)
     * Código do município do prestador igual ao código do município do estabelecimento.

  6. Após realizar as configurações acima, gere novamente o arquivo do ISS.WEB.
  7. Verifique se as notas fiscais dos prestadores dentro do município estão sendo corretamente excluídas do meio magnético.


### **Próximos Passos**

  * Finalize a integração do arquivo **ISS.WEB** com o site da prefeitura para validar a operação.
  * Caso o problema persista, revise as configurações realizadas e **abra um ticket** com evidências detalhadas para suporte técnico.


* * *

### **Dicas e Recomendações**

  * **Revise periodicamente** as parametrizações do módulo municipal para garantir conformidade com as exigências fiscais locais.
  * Antes de gerar o arquivo **ISS.WEB** , faça um **teste de validação** para confirmar que apenas notas válidas estão sendo incluídas.
  * **Mantenha a base de dados atualizada** e verifique se há notas fiscais com inconsistências que possam impactar o arquivo magnético.
  * Consulte nossa **base de conhecimento** para mais informações sobre parametrizações fiscais.
  * Em caso de dúvidas, **nossa equipe de suporte está à disposição** para ajudar!
