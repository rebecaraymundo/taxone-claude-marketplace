Passo a passo para gerar o parâmetro **Modelo Msaf x Modelo** para todos os Módulos Municipais que utilizam este De x Para:

Nota: O objetivo deste parâmetro é realizar o relacionamento entre os modelos informados no documento fiscal e os modelos solicitados para geração dos validadores.

**Documento Fiscal**

  1. Acesse [Básicos > Data Warehouse > Manutenção > Documento Fiscal > Doc. Fiscal de Serviço](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Basicos/Basicos_Data_Warehouse/Basicos_Data_Warehouse_Menu_Manutencao/Basicos_DW_Menu_Manu_Doc_Fiscal.htm>).

  2. Na **Capa da Nota Fiscal** , no campo **Modelo Doc.** verifique qual **t****ipo de Documento** foi informado.


**Parâmetro 'Modelo Msaf x Modelo'**

  1. Acesse [Municipal > Parâmetros para Município > Parâmetros> Modelo Msaf x Modelo](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Municipal/Parametros_Para_Municipio/Parametros_Municipio.htm>)[.](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Municipal/Parametros_Para_Municipio/Parametros_Municipio.htm>)

  2. Selecione a **UF **e o **Município.**

  3. Na tela **Relação****Modelo Msaf x Modelo** :

     1. Informe a **Validade **do relacionamento entre os modelos. A validade é do tipo “a partir de”. Essa data também é utilizada para buscar o grupo do modelo.

     2. No campo **Modelo Msaf** selecione os **modelos **utilizados nas notas fiscais cadastradas no Data Warehouse.

     3. No campo **Modelo **é apresentada uma lista com os modelos solicitados pelo validador em questão, selecione a opção desejada. O validador é identificado através do município previamente selecionado.
