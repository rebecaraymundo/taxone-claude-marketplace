### **Entendendo o Status "ERRO_TAXONE" nos lotes do OBI**

**Problema:** Um lote exibe o status **"ERRO_TAXONE"** ao ser consultado via API ou no Painel de Integrações do OBI. O significado não é compreendido pelo usuário, pois este status ainda não está documentado no manual oficial da API nem no Painel de Integração.

**Causa Raiz:** O status **"ERRO_TAXONE"** foi implementado recentemente para sinalizar lotes com falhas. Ele é exibido quando ocorre um dos seguintes problemas:

  * **Erro de Carga (Layout):** Falha na estrutura do arquivo enviado.
  * **Erro de Importação:** Falha na lógica de processamento dos dados dentro do Tax One (ex: incompatibilidade de código, campos obrigatórios ausentes).
