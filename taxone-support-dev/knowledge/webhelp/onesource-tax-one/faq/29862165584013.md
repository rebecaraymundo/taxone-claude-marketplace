Caso enfrente erro por conta de um Código de Receita incorreto, impedindo a transmissão de informações para o REINF, necessitando de lançamentos manuais. Para resolver, siga as orientações:

  1. Acesse o módulo Ferramenta > Tabelas Internas e selecione Manter.
  2. Escolha a opção **Tabelas** e, em seguida, **Código da Receita**.
  3. Localize o código incorreto (5987-04), exclua-o e adicione o código correto (5987-07).
  4. Após realizar as correções, verifique se a informação está sendo transmitida corretamente para o REINF. Caso persistam problemas, entre em contato com o suporte técnico.


**Importante:** Regularmente verifique as configurações e códigos cadastrados no sistema para evitar discrepâncias que podem afetar a transmissão de dados importantes. Mantenha-se atualizado com as últimas versões e atualizações do software para garantir a máxima eficiência e correção dos dados.
