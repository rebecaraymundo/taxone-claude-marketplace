Passo a passo para corrigir o** Erro 91170**: Indicador de Fabricação pelo Estabelecimento deve ser preenchido com <S> ou <N>:

  1. Preencha o campo **IND_FABRIC_ESTAB** , conforme o [manual de layout](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Tecnicos/Manual_Layout.htm>):

     1. S - Fabricado pelo Estabelecimento

     2. N - Não Fabricado pelo Estabelecimento.


Nota: O erro ocorre porque o campo IND_FABRIC_ESTAB não possui um dos valores definidos no manual de layout. Para corrigir o erro, altere o conteúdo do campo para um dos valores citados acima.
