# [Federal][CREDPIS] - Como resolver problemas de Cálculo de Parcelas no módulo CREDPIS?

O cálculo de parcelas no módulo CREDPIS não está sendo executado corretamente para aquisições realizadas em vários meses, impedindo a criação correta das parcelas para os períodos. A causa do problema é um campo não preenchido corretamente no cadastro dos bens (SAFX13), especificamente o campo "GRUPO DO BEM". Para resolver, siga as orientações:

1\. Acesse o cadastro dos seus bens no módulo SAFX13.

2\. Preencha o campo "GRUPO DO BEM" corretamente conforme necessário para cada bem.

3\. Após preencher o campo, retorne ao módulo CREDPIS e efetue novamente o cálculo das parcelas.

4\. Verifique se as parcelas foram corretamente criadas e se a informação foi inserida no F120/F130 do SPED contribuições.

5\. Após implementar a solução, monitore as próximas execuções do cálculo das parcelas para garantir que o problema foi completamente resolvido. Caso encontre qualquer inconsistência, entre em contato com o suporte para assistência adicional.

**Importante:** Recomendamos verificar regularmente as configurações dos campos nos cadastros utilizados no sistema para evitar problemas similares no futuro. Mantenha-se atualizado com as documentações e melhores práticas fornecidas na base de conhecimento.
