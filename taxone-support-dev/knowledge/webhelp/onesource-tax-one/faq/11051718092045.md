# [Estadual][Livros]TAX ONE - ICMS - Como realizar a alteração de status dos Livros Apuração do ICMS quando for apresentada uma mensagem de alerta que impede a confirmação?

Para realizar a alteração de status dos Livros Apuração do ICMS, para quando for apresentado a mensagem de alerta: '**A obrigação fiscal de código 108 está com a data de apuração inferior a data de fechamento para o estabelecimento'**..., siga os passos abaixo:

**Nota:** Essa situação trata-se de um comportamento esperado pela aplicação, pois foi efetuado algum fechamento para o grupo de tabelas. Após o fechamento, o usuário não pode alterar a base de dados relacionada ao grupo em questão até o período estipulado por ele.

  1. Acesse o controle deste fechamento em Básicos > Ferramentas > Rotinas Acessórias > Fechamento > Controle.


  2. O grupo pertinente aos livros de apuração é o **23 - Registro de Apuração do ICMS**. Observe se há um fechamento contemplando o período de apuração.


  3. Para que seja possível alterar o status da apuração, o controle de fechamento deve ser excluído ou a **Data Fechamento** deve ser alterada para um dia qualquer que seja anterior à data inicial do período da apuração que se deseja alterar o status.


  4. Após ajustado o controle de fechamento, efetue a manutenção no status do livro em Estadual > ICMS > Manutenção > Status das Obrigações.
