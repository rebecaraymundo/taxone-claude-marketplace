# [Municipal][Castro]  O estabelecimento não aparece para seleção na Opção de Geração " Arquivos de Entradas de Serviços ( Const. Civil / Utilities/ Telecom), como resolver?

Acesse esta FAQ no [web help do produto](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/WebHelp/mergedProjects/FAQs/Municipal.htm#faq_municipal_18>).

Quando o 'Estabelecimento' não aparece para seleção na opção de Geração de "Arquivos de Entradas de Serviços (Construção Civil / Utilities / Telecom)" no módulo Municipal, siga as orientações:

No menu Obrigações > Arquivo de Entradas de Serviços (Construção Civil / Utilities / Telecom) são listados apenas os estabelecimentos que não pertencem ao município que está sendo gerado e que têm o Código do Estabelecimento cadastrado no arquivo de Licenciamento.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21408827214477)

1\. Para que o município seja exibido para seleção, é necessário que o estabelecimento possua inscrições municipais eventuais cadastradas para o município em questão.

Isso pode ser feito em Básicos > Data Warehouse > Manutenção > Cadastros > Inscrições Municipais.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21408817106445)

2\. Realize o cadastro da Inscrição Municipal do estabelecimento para o município que será gerado.

Após realizar o cadastro da Inscrição Municipal, o estabelecimento deverá ser listado para seleção na geração dos arquivos municipais.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21408837106829)
