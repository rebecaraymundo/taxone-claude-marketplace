Foi constatado que, ao entrar em _Parâmetros Gerais_ e selecionar a aba Web Service, um erro de dump pode ser gerado, indicando que o endereço da URL não foi encontrado.


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21572437410317)

  1. A análise detalhada do dump revelou que uma das URLs fornecidas para a conexão do Web Service estava incorreta:


![](/attachments/token/lUZNioj8uUyR002LgIOtx1gdX/?name=image.png)

  1. Como não é possível acessar as configurações de URL diretamente através da opção "_Parâmetros Gerais > Web Service > Definir URL"_, a correção dessa informação deve ser feita utilizando a transação SE16n.


Para corrigir a URL:

  * Acesse a transação **SE16n.**
  * Insira a tabela**/MSAFX/SAFTB087**.
  * Selecione **"Atualização de visão"**.


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21572471671053)

  * Na tabela que se abre, clique em **Exibir/Modificar**.
  * Localize a URL incorreta e proceda com a modificação.
  * Selecione "**Salvar** ".
  * Realize um novo teste acessando _Parâmetros Gerais > aba Web Service_, para confirmar a correção.


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21572440910989)
