## Problema

Um usuário tenta excluir um código de pessoa física ou jurídica no sistema, mas enfrenta uma mensagem de erro informando que existem outros cadastros vinculados a esse código.

## Causa Raiz

O problema ocorre porque o código de pessoa física ou jurídica que o usuário está tentando excluir está vinculado a outros registros no sistema, como documentos fiscais.

## Resolução

  * Acesse o módulo Básicos.


  * Navegue até DataWarehouse > Manutenção > Cadastros > Pessoas Físicas/Jurídicas.


  * Localize o código de pessoa física ou jurídica que deseja excluir.


  * Verifique se existem documentos fiscais ou outros tipos de cadastros vinculados a este código.


  * Se existirem vínculos, será necessário desvincular ou gerenciar esses registros antes de proceder com a exclusão.


  * Após resolver os vínculos, tente excluir o código novamente.
