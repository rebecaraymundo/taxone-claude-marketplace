Passo a passo para corrigir o **Erro 92236** \- O Campo Indicador de Característica do Produto deve ser preenchido com @, 1, 2, 3, 4:

  1. Preencha o campo **IND_CARAC_PRODUTO** , conforme o [manual de layout](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Tecnicos/Manual_Layout.htm>):

     1. 1 - Combustível / Lubrificante Derivado do Petróleo

     2. 2 - Combustível / Lubrificante não Derivado do Petróleo

     3. 3 - Mineral

     4. 4 - Livros, Jornais, Periódicos ou Papel Imune


Nota: O problema ocorre porque o campo **IND_CARAC_PRODUTO** não possui um dos valores definidos no manual de layout. Para corrigir o erro, altere o conteúdo do campo para um dos valores citados acima.
