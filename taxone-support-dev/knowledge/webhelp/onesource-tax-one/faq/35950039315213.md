Como Popular o Campo 06 do Registro C100 para NF Complementares

**Problema:**

O campo 06 do Registro C100 não está sendo preenchido automaticamente em algumas Notas Fiscais Complementares.

**Causa Raiz:**

A origem do problema é que o Campo 125 (Situação Especial), que deve ser preenchido com o código '5 - NF Complementar ou CT-e de Complemento de Valores (modelo: 57)', não foi informado na capa do documento fiscal.

**Resolução:**

Abra o documento fiscal referente à Nota Fiscal Complementar.

Vá até a capa do documento fiscal.

Localize o Campo 125 (Situação Especial).

Preencha o campo com o código '5', que corresponde a 'NF Complementar ou CT-e de Complemento de Valores (modelo: 57)'.

Salve as alterações e regenere o documento fiscal.

O Campo 06 do Registro C100 agora deve estar populado corretamente como 'nota complementar - 06'.

**Próximos Passos:**

Após realizar essas correções, recomendamos executar uma verificação completa para garantir que o campo está sendo preenchido corretamente em todas as notas fiscais complementares futuras. Monitore os próximos registros para assegurar que o problema foi completamente resolvido.
