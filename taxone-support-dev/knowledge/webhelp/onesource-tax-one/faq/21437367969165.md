O agendamento de cargas é uma operação rotineira utilizada por clientes e consultorias para agendar suas cargas no sistema. Uma questão frequente é se é possível programar o agendamento para um período superior a 30 dias.


Para realizar um agendamento:

1\. Acesse a seção **'Processos\Programação'** e clique em '**Editar** ' (ícone do lápis) para acessar a tela de programação da carga SAFX selecionada.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21437419529101)

2\. Nesta tela, defina uma data inicial e uma data final para o agendamento. Se o intervalo entre a data inicial e a data final ultrapassar 30 dias, o sistema exibirá a mensagem: _'Período Máximo de 31 dias excedido'._

_![](https://suporte.thomsonreuters.com.br/hc/article_attachments/21437386923149)_

3\. Portanto, o usuário deve selecionar um intervalo de datas que não exceda 30 dias. Com base nesse parâmetro, não é possível realizar o agendamento de cargas para um período maior que 30 dias.
