# [Básicos][Job Servidor] TAX ONE - SAFX01 - Como corrigir o erro 90096?

Normalmente, esse problema ocorre porque o código informado no campo CONTRA_PART não está cadastro na tela Data Warehouse > Manutenção > Códigos > Plano de Contas. Para corrigir o erro 90096: 'A Conta Contábil de Contra-Partida não esta cadastrada.', siga as orientações abaixo:

  1. Informe um código valido no arquivo TXT, ou
  2. Cadastre o código informado no arquivo TXT na tela Data Warehouse > Manutenção > Códigos > Plano de Contas.
