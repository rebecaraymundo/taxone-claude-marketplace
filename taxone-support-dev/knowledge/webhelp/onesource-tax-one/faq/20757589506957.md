Nota: após o Basis importar as requests, se atente ao Manual de Atualização ou ao arquivo Leia-me com todas as informações necessárias para a conclusão da atualização.

  1. Além de verificar se o log da aplicação das requests está sem erros:


  * Na tela de versão, transação /N/MSAFX/SAFTR001A:
    * A primeira informação da versão se refere a Request Principal (Workbench)
    * A segunda informação de versão se refere a Request Customizing, que se refere ao update das tabelas acessórias.
    * As duas versões precisam coincidir e estar com o ícone de status verde.


![](https://suporte.thomsonreuters.com.br/hc/article_attachments/25153729058957)

  1. Caso o ícone de status esteja vermelho:


  * Faltou aplicar a request para o Mandante (cliente) que foi logado.
  * Faltou clicar no disquete para Salvar.
  * Falha no processo de encapsulamento para transporte de request.
