Se ao gerar relatórios por natureza do rendimento para notas fiscais de serviço o sistema indicar falta de parâmetro, mesmo após a configuração ter sido realizada, é possível que o problema esteja no campo 'Descrição Resumida' nos Parâmetros de Apuração do Identificador de Natureza da Receita. Verifique se a descrição está excedendo o limite permitido de 60 caracteres.

Passo a passo para resolver o problema na emissão de relatório por natureza do rendimento para notas fiscais de serviço:

  1. Acesse o módulo do **EFD Contribuições**.
  2. Clique em**Parâmetros Apuração Identificador de Natureza da Receita**.
  3. No campo '**Descrição Resumida'** , ajuste a descrição para não ultrapassar 60 caracteres. Isso pode ser feito abreviando palavras.
  4. Salve as alterações realizadas.
  5. Gere novamente os Registros e, posteriormente, faça uma nova geração do relatório.
  6. Após realizar os ajustes e gerar os relatórios, verifique se a emissão está sendo realizada corretamente sem erros. Caso o problema persista, entre em contato com o suporte técnico para uma análise mais detalhada.


**Dicas e Recomendações:**

É importante sempre verificar os limites de caracteres em campos de descrição para evitar erros de geração de relatórios e registros. Manter a documentação e parâmetros do sistema sempre atualizados facilita a identificação e solução de problemas.
