## Atualização Manual de CEP na TACES 12

### Problema:

Usuários precisam saber como atualizar manualmente o CEP na tabela TACES 12 quando um código não está disponível na base de dados atual.

### Causa Raiz:

A tabela TACES 12 não é atualizada automaticamente com os novos CEPs da base dos correios. É necessário inserir esses dados de forma manual caso um CEP específico não esteja presente.

### Resolução:

  * Acesse o sistema e vá até a seção **Datawarehouse**.


  * Navegue até a opção **Manutenção** e selecione **Cadastros**.


  * Escolha a opção **CEP** para adicionar um novo código postal.


  * Insira os detalhes necessários e salve as alterações.


### Próximos Passos:

Após inserir o novo CEP, verifique se ele está sendo reconhecido corretamente no sistema. Caso encontre algum problema, entre em contato com o suporte técnico para auxílio adicional.

### Dicas e Recomendações:

É recomendável verificar periodicamente a necessidade de atualização de novos CEPs para evitar inconvenientes no uso da tabela TACES 12. Mantenha registros das alterações realizadas para facilitar futuras consultas e possíveis ajustes.
