Ao fazer o ajuste em massa, ocorre o erro 'Arquivo Inválido', como no exemplo abaixo:

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/19806598220173)

O erro acontece quando o arquivo CSV não é salvo no padrão aceito pelo sistema.

Para corrigir, salve o arquivo como CSV (MS-DOS), conforme a imagem a seguir:

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/19806642287629)
