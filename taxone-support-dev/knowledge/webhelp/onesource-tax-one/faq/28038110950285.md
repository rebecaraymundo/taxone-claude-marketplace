Caso enfrente dificuldades na geração do Registro J800 na Escrituração Contábil Digital (ECD) quando a periodicidade do arquivo RTF é configurada de forma diferente da periodicidade das Demonstrações Contábeis, siga as orientações:

**Nota:** O sistema exige que a periodicidade de inclusão do arquivo RTF seja trimestral para que o registro J800 seja gerado corretamente junto ao meio magnético.

  1. No menu Parâmetros > Dados Iniciais > aba Demonstrações Contábeis.
  2. Configure a periodicidade do arquivo RTF armazenado para "Trimestral", que deve coincidir com a periodicidade das Demonstrações Contábeis.
  3. Realize o upload do arquivo RTF com esta configuração.
  4. Prossiga com a geração das Demonstrações Contábeis e do meio magnético para o mês de dezembro.
  5. Após realizar os passos acima, verifique se o registro J800 foi gerado corretamente no arquivo magnético. Caso o problema persista, recomenda-se entrar em contato com o suporte para uma análise mais detalhada ou agendar uma chamada para acompanhamento do processo.


**Importante:** É essencial que a periodicidade do arquivo RTF coincida com a periodicidade das Demonstrações Contábeis para evitar problemas na geração do registro J800. Mantenha-se atento às configurações dos parâmetros iniciais e consulte a documentação do sistema para quaisquer dúvidas adicionais.
