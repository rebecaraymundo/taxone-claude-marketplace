Se os valores existentes na apuração geral do PIS/COFINS estão sendo refletidos como zerados ao gerar o relatório sintético, isso pode ocorrer devido a uma configuração inadequada nos parâmetros do módulo EFD Contribuições. É necessário ajustar o identificador do tipo de operação e a natureza da base de crédito. Para resolver, siga as orientações:

  1. Acesse o módulo EFD Contribuições.
  2. Navegue até 'Parâmetros'.
  3. Ajuste o 'Identificador do tipo de Operação' para 'Por CFOP' ou 'Por Extensão de CFOP'.
  4. Configure o 'Identificador da Nat. da Base de Crédito' para 'Por CFOP', 'Por Extensão de CFOP' ou 'Por Código de Serviço'.
  5. Salve as alterações e realize um novo teste de geração do relatório.
  6. Após ajustar os parâmetros e gerar o relatório, verifique se os valores estão sendo apresentados corretamente. Caso o problema persista, entre em contato com o suporte técnico para uma verificação mais detalhada.


**Nota:** Para evitar problemas futuros de geração de relatórios, revise regularmente as configurações dos parâmetros no módulo EFD Contribuições e mantenha-se atualizado com as últimas instruções e diretrizes do sistema.
