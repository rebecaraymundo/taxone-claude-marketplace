Passo a passo para corrigir o **Erro 92595** \- Código de Instalação 2 da Pessoa Física/Jurídica inválido:

  1. Cadastre o **Código de Instalação** em[ Básicos > Ferramentas > Tabelas Internas > Manter > Tabelas > Tabela de Códigos de Instalação ANP T008](<https://webhelp.thomsonreuters.com.br/Site/HELPDW/V2R010/Conteudo/Demais_Empresas/Onesource_Tax_One/Help/WebHelp/Manuais_Funcionais/Basicos/Basicos_Ferramentas/Menu_Tabelas_Internas/Ferramentas_Menu_Tabelas_Internas_Manter.htm>).


Nota: O erro ocorre porque o código que está no campo **COD_INSTAL_ANP** não está cadastrado. Essa tabela é alimentada pela TACES98, caso o código não exista na tela é necessário cadastrá-lo.
