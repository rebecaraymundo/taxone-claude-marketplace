## Problema

O usuário deseja realizar pesquisas por faixa de datas nos critérios de relatórios do sistema Tax One.

## Causa Raiz

O sistema Tax One não possui funcionalidade para realizar pesquisas por faixa de datas como era possível no DW. Nos filtros disponíveis nas telas de definição de critérios, é permitido informar apenas uma única data específica.

## Resolução

Como o Tax One não permite a configuração de faixas de datas nos filtros de consulta de relatórios, o usuário deverá seguir o procedimento abaixo para realizar pesquisas baseadas em datas:

  1. Acesse a tela de definição de critérios do relatório desejado.
  2. Localize o campo para informar a data no filtro.
  3. Insira uma única data específica que deseja utilizar para a pesquisa.
  4. Confirme os critérios definidos e gere o relatório.


Para obter resultados de diferentes datas, será necessário repetir a pesquisa alterando a data no filtro.
