Caso enfrente um problema ao tentar inserir uma Descrição Complementar do Crédito no registro M105 quando a natureza de crédito é "13 - Outras Operações com Direito a Crédito", onde o campo fica desabilitado e não permite inserções, siga as orientações:

  * Verifique se o período que deseja editar está aberto para modificações.
  * Teste o acesso em um ambiente de produção para garantir que o problema não seja específico do ambiente de testes.
  * Se o problema persistir, entre em contato com o suporte técnico fornecendo seu login, senha (de forma segura) e detalhes adicionais do problema para uma análise mais aprofundada.
  * Enquanto aguarda o suporte, continue inserindo as informações necessárias diretamente no PVA como alternativa temporária.
  * Após implementar a resolução, monitore o sistema para verificar se o problema foi completamente resolvido. Caso contrário, reabra o ticket de suporte para mais assistência.


**Nota:** Para evitar problemas futuros, mantenha seu sistema sempre atualizado e verifique regularmente as permissões e configurações de acesso. Além disso, sempre verifique se o período que deseja trabalhar está adequadamente aberto para edições.
