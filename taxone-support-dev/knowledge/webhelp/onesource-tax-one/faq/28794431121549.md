Passo a passo para importar o Hash Code do Convênio 115 no Taxone - ICMS:

  1. Acesse o menu "**Importação Hash** " no sistema através do caminho: _ICMS > DATA MART > Convênio ICMS 115 > Importação Hash Code e Conferência do Registro de Saídas_.
  2. Informe o estabelecimento e o mês/ano de apuração desejados.
  3. Escolha a opção de importação: "**Via Servidor** " ou "**Client** ".
  4. Caso tenha escolhido a opção "**Via Client** ". Localize e selecione o diretório onde os arquivos Hash Code estão salvos.
  5. Clique no botão "**ABRE** " para visualizar os arquivos disponíveis.
  6. Selecione os arquivos desejados e clique em "Importar".


**Importante** : É necessário deixar todos os arquivos na mesma pasta e selecionar o arquivo "Y" juntamente com os demais arquivos. Dessa forma, a importação será realizada corretamente.
