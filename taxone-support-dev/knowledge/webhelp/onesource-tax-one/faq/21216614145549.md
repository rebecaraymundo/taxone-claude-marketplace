Segue uma rotina alternativa ao painel de controle de eventos para geração e envio do arquivo XML. Utilizando essa opção é possível ganhar celeridade no fluxo de geração e envio dos eventos com origem externa. Esta opção foi criada para atender principalmente o grande volume de eventos.

Importante: Essa opção é exclusiva para eventos com origem externa, e destina-se a geração e envio ao OneSource. Ou seja, as outras opções, como por exemplo, retificação de evento, deve ser executada a partir do painel de controle de eventos.

Segue passo a passo:

1\. Importação dos Arquivos Externos e Geração Prévia dos Movimentos

Os arquivos devem ser previamente importados para o produto e deve ser efetuada a geração prévia dos movimentos para o período.

Para importação dos arquivos e posterior geração prévia dos movimentos, pode ser verificado os seguintes artigos:

[TAX ONE - Importação de Arquivos Externos (JCP) XML dos Bancos](<https://suporte.thomsonreuters.com.br/hc/pt-br/articles/21162751356685>), e

[TAX ONE - Geração e envio de eventos relativos a arquivos externos (JCP) XML dos bancos](<https://suporte.thomsonreuters.com.br/hc/pt-br/articles/21165659218445>).

2\. Geração do arquivo XML e envio ao OneSource

  * Em REINF > Painel > Envio de Registros XML de Arquivos Externos - Grandes Volumes
