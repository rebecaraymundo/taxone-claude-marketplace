# Como parametrizar o Tipo Logradouro Msaf x Tipo Logradouro para geração correta do meio magnético

###

Ao gerar o meio magnético para o município de São José dos Campos utilizando o layout Nota Joseense, o campo obrigatório "Logradouro" (campo 16) não está sendo preenchido no meio magnético.

## Causa Raiz

O parâmetro "Tipo Logradouro Msaf x Tipo Logradouro" no módulo Municipal/Parâmetros para município deve ser realizada, para que a informação obrigatória do campo "Logradouro" seja gerada no meio magnético.

##

Abaixo estão os passos para resolver o problema:

Verifique o Tipo Logradouro utilizado no cadastro da PF/PJ SAFX04:

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40208480283021)

  1. Acesse o módulo **Municipal**.
  2. Navegue até **Parâmetros para município**.
  3. Localize o parâmetro **Tipo Logradouro Msaf x Tipo Logradouro**.
  4. Preencha o parâmetro de acordo com os manual e normas exigidas pela prefeitura de São José dos Campos.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40208470360205)

5\. Salve os parâmetros.

6\. Realize um teste gerando o meio magnético novamente e verifique se o campo 16 (Logradouro) está sendo corretamente preenchido no arquivo .txt.

![](https://suporte.thomsonreuters.com.br/hc/article_attachments/40208480289293)
