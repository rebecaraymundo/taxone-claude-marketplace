# Tela de Apuração do imposto para fazer a conciliação do PIS/COFINS (Liberação da versão nova para todos os clientes)

## Descricao

Liberar versão nova da Apuração para todos os clientes

## Release Notes

Informações Carta Patch - TAXONE 

  

Melhoria 

  

Objetivo: Nesta demanda, foi implementada uma melhoria, criando uma tela única, centralizando todos os registros a referente a apuração do PIS e COFINS.   

A tela unifica as informações que compõe o PIS e COFINS na apuração, permitindo uma melhor visualização e facilitando as manutenções necessárias, para a entrega da obrigação do EFD_Contribuições. 

  

  

Módulo:  > EFD – Escrituração Fiscal Digital das Contribuições 

  

Menu Impactado:  

  

>> SPED >> EFD – Escrituração Fiscal Digital das Contribuições 

  

Menu: Obrigações >>Manutenção >> Apuração Geral PIS/COFINS
