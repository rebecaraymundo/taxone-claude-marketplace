# [SPED] - TAXONE - EFD Contribuições - Notas de entrada de terceiros canceladas - Angular 13

## Descricao

Eu como usuário(a) fiscal do Modulo SPED – EFD_Contribuições. 

Gostaria que fosse feita uma melhoria na tela de Dados Iniciais, incluindo um flag, para que o cliente possa desprezar as Notas de entrada de terceiros canceladas no arquivo do EFD_Contribuições 

 

Informações Adicionais: 

Ticket: EFD Contribuições - Notas de entrada… – Thomson Reuters – Zendesk

## Release Notes

Informações Carta do Patch - Produto: TaxOne
 

Melhorias 

 

Nesta demanda, foi criado o parâmetro 'Considerar
Notas Fiscais de Entradas Canceladas – Modelo 55' na tela de Dados Iniciais, aba Específico. Ajustada a geração do registro C100 para considerar esse novo parâmetro. Por default, o parâmetro estará marcado, contemplando na geração do C100 as notas eletrônicas de entrada (modelo 55) cancelada (conforme já considerado na geração atual). Se desmarcado, essas notas serão desprezadas na geração.
 

 

Agrupamento: SPED Módulo: Parâmetros para Município Menus:  TAXONE → Sped → EFD – Escrituração Fiscal Digital das Contribuições →
Parâmetros à Dados Iniciais 

 TAXONE → Sped → EFD – Escrituração Fiscal Digital das Contribuições → Obrigações à Geração dos Registros
