# [REFINARIA MATARIPE] TKT94163 - TAX ONE - SCANC - Relatórios - Conferência dos Anexos VI, VI-M, VII, VII-M (Refinaria) - filtro do Mês/Ano Referência com problema

## Descricao

Cliente:
Vanda, boa tarde 

Conforme conversamos estou abrindo esse novo chamado para o problema reportado no ticket #90452. 

Na geração de relatórios no módulo SCANC, quando selecionamos a ultima data do mes pelo botão de calendario, só são apresentadas as notas dessa data, e com isso o relatório fica incompleto.
Temos que informar a primeira data do mês para trazer tudo. Porem, em outras obrigações quando se trata de data de referencia, apuração, sempre informamos a ultima data do mês. 

 

Suporte: 

Conforme evidenciado na aba Analysis, o filtro do Mês/Ano Referência só apresenta movimentação quando selecionado o último dia do mês. 

O cliente já havia relatado esse problema no através do TK  #90452  foi aberto demanda não simulada USER STORY 610805, pois não foi identificado problema.
