# INFOLEGIS 645/24 - B -  LAYOUT V1.3 - ATENDE.NET - SÃO JOSÉ/SC - DW/T1

## Descricao

A Prefeitura do Município de São José/SC disponibilizou o layout versão 1.3 do arquivo para importação de documentos de serviços do sistema Fiscal Web (Atende.Net).
O novo layout está disponível no endereço: 
 https://wiki.ipm.com.br/?download=202067&QR
 Histórico de revisões:
 Versão 1.0 - Criação do documento e padronização.
 Versão 1.1 - Alterado o item Layout do Registro Tipo 10.
 Versão 1.2 - Alterado o item Layout do Registro Tipo 30.
 Versão 1.3 - Alterada a tabela do item Situações Tributárias. 
 
 Cumpre observar que a informação do layout disponibilizado foi confirmado por telefone (47) 3531-1500 com Douglas da IPM Sistemas (empresa responsável pelo Fiscal Web/Atende.net). O acesso é feito no link  https://nfse-saojose.atende.net/autoatendimento/servicos/acesso-ao-sistema-fiscal-web-novo/detalhar/1 utilizando conta do Gov.br.
