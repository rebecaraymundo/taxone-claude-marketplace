# [BRASILAGRO] TKT114646 - TAXONE - EFD REINF - Erro ao realizar a geração prévia dos movimentos R-4020 - Produção Restrita.

## Descricao

**TKT114646 - BRASILAGRO - COMPANHIA BRASILEIRA DE PROPRIEDADES AGRICOLAS - ONESOURCE TAX ONE - SPED - EFD - REINF - Erro ao realizar a geração prévia dos movimentos R-4020 - Produção Restrita.** 

 
 Ao realizar a geração prévia dos moviemntos R-4020 é gerado o erro ="Já existe geração para os eventos R-4010 ou R-4020, de forma centralizada para este período de apuração << 0001 - BRASILAGRO - COMPANHIA BRASILEIRA DE PROPRIEDADES AGRICOLAS >> . Por isso, não será permitido efetuar essa geração descentralizada. Caso necessário, deverá ser cancelada a geração anterior destes eventos, no painel de eventos, ou efetuar a exclusão, no caso do evento já ter sido enviado para o governo com sucesso e só então, processar as gerações de forma descentralizadas".

## Root Cause

1)Atualizado em: 19/11/2024
2)Objeto Alterado:
 REINF_PGER_DIC_DADOS
 3)Causa do problema:
 Crítica ao processar as gerações de forma descentralizadas porque já existia geração centralizada porém elas estavam excluidas
 4)Solução aplicada:
 Retirada a crítica ao processar as gerações de forma descentralizadas porque a geração centralizada já estava excluidas
 5)Módulos impactados: Somente esse Módulo

## Release Notes

Reinf --> Geração Prévia --> Movimentos (R4020) 
 Ajuste para não considerar os eventos que foram rejeitados pelo governo para crítica de geração por centralizador ou centralizado
