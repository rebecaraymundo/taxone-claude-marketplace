# INFOLEGIS 1132/24 - B -  ISS - Três Corações/MG - Layout - DW/T1

## Descricao

A Prefeitura de Três Corações/MG disponibilizou atualização no Layout TXT do arquivo magnético de importação de notas fiscais para a Substituição Tributária do Sistema de Declarações Fiscais de ISS, desenvolvido pela E&L Produções de Software.

 O manual de integração está disponível no endereço:
 https://s3.amazonaws.com/el.com.br/nfse/layout_aceites_substituicao.pdf
 
 Tentamos diversos contatos com a Prefeitura de Três Corações/MG via telefone e email para obter informações sobre a data de alteração de layout, porém não tivemos nenhum retorno.
