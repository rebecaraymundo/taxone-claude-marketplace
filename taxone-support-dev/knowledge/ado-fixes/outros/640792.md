# [HEINZ BRASIL] TKT98874 - TAXONE - GASPAR - Gerando arquivo retificador mesmo não sendo

## Descricao

**TKT98874 - TAXONE - GASPAR (Ao gerar o Arquivo de Gaspar tanto prestador como tomador, o mesmo é gerado como arquivo retificador mesmo não sendo)** 

 Ao gerar o Arquivo para o município de Gaspar e validar ele no DES THEMA, ocorre o erro : " Não encontramos a declaração que você quer retificar" Isso devido ao arquivo esta sendo gerado com TAG de retificação mesmo não sendo retificador.

## Root Cause

Carta do Patch DW e Taxone: 

  

 

 Módulo: Municipal > Gaspar 

 Menu: Obrigações > Meio Magnético  

 Objetivo: Alteração de geração da tag <declaracao>, passando a considerar o preenchimento do campo Retificadora na geração, se o campo Retificadora estiver preenchido a tag será gerada como <declaracao retificadora>, caso contrário será gerada somente como <declaracao>.
