# [HUGHES] TKT116671 - TAXONE - ATIVO PERMANENTE - Relatório dos Créditos Extemporâneos - IEU

## Descricao

​**[HUGHES] TKT116671 - TAXONE - ATIVO PERMANENTE - Relatório dos Créditos Extemporâneos inicia o processo e porem não finaliza para estabelecimento com Inscrição Estadual Única.** 

 

CLIENTE:
 

 

 

Estou tentando tirar um relatório de crédito extemporâneo para poder conferir os valores, porém não aparece nada no relatório e não dá erro. 

Isso está acontecendo para estabelecimento com Inscrição Estadual Única. 

 

 

 

 
 
 
SUPORTE: 

 
O cliente está enfrentando dificuldades ao tentar gerar um relatório de crédito extemporâneo. Apesar de iniciar o processo de geração do relatório de cálculo, o mesmo não é concluído e não apresenta mensagens de erro. Em contato com a cliente foi informado que apenas estabelecimento com Inscrição Estadual Única não está gerando. Ao realizar testes em nosso ambiente de teste, a geração do relatório foi concluída com sucesso.
