# [APPLE ] TKT145169 - DW - EFD REINF - Reavaliação de regra R-4020 - Campo ideEvtAdic

## Descricao

O cliente possui o seguinte cenário: 

 

O evento esta sendo rejeitado pela mensagem: “O evento foi rejeitado pelo governo Motivo MS1576 - Para a Forma de Tributação para rendimentos de beneficiários no Exterior fmr Tribut de 40 a 50 não é permitida a informação do imposto de renda retido na fonte. O campo vlrIR não deve ser preenchido ou deve ser igual a 0.

## Release Notes

Informações p/a Carta do Patch: 

  

Módulo: EFD - REINF 
      Nesta demanda, será implementada uma alteração na geração do Registro R-4020,  A partir de agora, quando forem identificados dois os mais registros associados ao mesmo Beneficiário no exterior, serão demonstrados todas as informações de pagamento sem agrapamento, assim os códigos do campo "103 - Forma de tributação" serão demonstrados de acordo com o que for cadastrados nos controles de tributos.
