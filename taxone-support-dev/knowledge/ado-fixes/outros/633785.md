# [SANTA CRUZ] TKT97587 - TAXONE - EFD CONTRIBUIÇÕES - Erro ORA-00001: unique constraint - EPC_NAT_BASE_CRED_SERV_FPROC

## Descricao

** EFD Contribuições - Erro ORA-00001: unique constraint (USER_TAXONE_SUPORTE.PK_PRT_SERV_MSAF) violated ORA-06512: at USER_TAXONE_SUPORTE.EPC_NAT_BASE_CRED_SERV_FPROC, line 289** 
 Boa tarde!
Estamos com um problema na tela de cadastro de natureza de crédito por tipo serviço, a funcionalidade de replicação do cadastro para outros estabelecimentos não esta funcionando, acusando um erro no log do processo. 

Segue anexo evidências do cenário.

## Root Cause

Causa: Replicação para estab que já tem dados gravados ocorrendo em erro ORA - 00001 de duplicidade por Unique constraint.

 Solução: Tratamento com select prévio e de Exception NO_DATA_FOUND implementando insert para os demais estab's a serem replicados.
