# TKT151317 - MRV - TAX ONE - MUNICIPAL - NFSE ABACO - Manaus - Campo Natureza operação

## Descricao

TKT151317 - MRV - TAX ONE - NFSE ABACO - Manaus - Campo Natureza da Operação 

O cliente MRV, como já identificado em outras situações, possui Inscrição Eventual em alguns municípios e, por isso, precisa gerar uma mesma nota em dois arquivos referentes a municípios distintos. No caso específico deste ticket, o estabelecimento está localizado em Manaus, gerando o arquivo correspondente, porém o ISS é devido em Ribeirão Preto. 

 

O campo em questão é o "NaturezaOperação". Para o arquivo de Manaus, é necessário gerar o valor 28 - RECOLHIMENTO FORA. No entanto, atualmente está sendo gerado o valor 13 - RETENÇÃO ISSQN, mesmo que as parametrizações estejam configuradas para gerar o valor 28. Isso ocorre porque o indicador de retenção está marcado como "retido pelo tomador" e os campos de ISS retido estão preenchidos, o que atende às condições para geração do valor 13. O cliente precisa manter essa escrituração para que o arquivo de Ribeirão Preto seja gerado corretamente. 

 

Diante disso, solicitamos a avaliação da regra para permitir a geração do valor 28 no campo NaturezaOperação, mesmo quando o indicador de retenção estiver como "retido pelo tomador" e os campos de ISS retido estiverem preenchidos. 

 

Regra validada: https://trten.sharepoint.com/:w:/r/sites/REQ_MTZ/_layouts/15/Doc.aspx?sourcedoc=%7B82D3D3CF-04E3-49A1-BCBD-FE85303BD3EC%7D&file=MTZ_NFSE_ABACO_Geracao_do_Meio_Magnetico.docx&action=default&mobileredirect=true
