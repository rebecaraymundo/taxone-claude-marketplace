# TKT118481 - TAX ONE - ESTADUAL - ICMS - Código de observação não esta sendo replicado para todos os estab durante o processo de replicação

## Descricao

Há um problema com os parâmetros automáticos para a UF de Minas Gerais no sistema: o código de observação não está sendo replicado corretamente para todos os estabelecimentos durante o processo de replicação. Especificamente, o campo do código de observação não está sendo preenchido adequadamente, o que afeta a geração dos registros C195 e C197 do SPED fiscal.

## Root Cause

Causa: O campo do código de observação não está sendo preenchido adequadamente, o que afeta a geração dos registros C195 e C197 do SPED fiscal.

 Solução:  Ajuste para replicar também os dados para código param 17 e 18.
