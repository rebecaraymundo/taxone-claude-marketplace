# [Arcom] TKT 92160 - Tax One - SPED - SPED FISCAL - Registro 0221 não permite um novo cadastro de produto associado quando já existe cadastro associado a ele mesmo (alerta de validades diferentes)

## Descricao

Alegação do Cliente:
 

Registro 0221- CORRELAÇÃO ENTRE CÓDIGOS DE ITENS COMERCIALIZADOS - SAFX278
 

O Tax One não está permitindo inserir mais de 2 códigos de produto principal para o mesmo código de produto associado. Ao tentar inserir o segundo código o Tax One apresenta a seguinte mensagem de erro: “Foi encontrada parametrização para produto associado relacionado a outro produto principal, com datas de vigência em conflito com as datas de vigência informadas.” 

Conforme exemplo citado no Tutorial de Escrituração Fiscal da EFD ICMS-IPI para contribuintes do DF é permitido este cenário. Veja o exemplo da areia lavada. No guia prático não identificamos regra limitando este cenário. 

Realizamos testes de inserção de dados via tela (Parâmetros>Registro 0221>Produtos) e JSON com datas e quantidades diferentes e não obtivemos sucesso. 

Segue anexo print com os exemplos e embasamento legal. 

A obrigatoriedade do registro 0221, é aplicável para o cliente nas UF MG (15/02) e DF (20/02). 

 

Suporte 
O registro 0221 não está permitindo um novo cadastro de produto associado quando já existe cadastro associado a ele mesmo (alerta de validades diferentes), conforme demonstrado na simulação na aba analysis.

## Release Notes

Carta do Patch: DW E TAXONE 

 

Módulo: EFD - Escrituração Fiscal Digital  

  

Resumo da Alteração: Alteração na validação de inserção do Registro 0221. 

  

Descritivo das Alterações: Validar as datas inicial e final com base no produto pai preenchido, sendo necessário verificar se para o produto pai a ser importado a data inicial é maior ou igual a menor data inicial importada, bem como se a data inicial é menor ou igual a data final importada para o produto pai. 

 

Menu Job Servidor, Módulo: EFD - Escrituração Fiscal Digital  

          SPED > EFD - Escrituração Fiscal Digital  

  

- Módulos Impactados: EFD - Escrituração Fiscal Digital.
