## Root Cause

Ao realizar a geração do dos saldos o sistema considerava somente o ident_conta, nessa situação quando há mais de uma conta cadastrada e movimentada o sistema considerava somente um dos cadastros

## Release Notes

Módulo: SPED > ECD – Escrituração Contábil Digital 

Menu: Geração > Saldos
 
Realizado um ajuste na lógica de composição dos saldos contábeis para tratar corretamente situações em que existam múltiplos cadastros para a mesma conta contábil no plano de contas. Com essa melhoria, o sistema passa a considerar todos os cadastros como pertencentes à mesma conta, consolidando os valores de forma adequada e evitando divergências na geração dos saldos.
