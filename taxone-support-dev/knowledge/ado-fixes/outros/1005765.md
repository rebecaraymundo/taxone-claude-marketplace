# TKT163227 - SHOPEE - ONESOURCE TAX ONE - BÁSICOS -   Ao tentar gerar relatório está ocorrendo erro 'request 504'

## Descricao

Time, 
cliente está tentando gerar o relatório Obrigações Acessórias> ICMS> Resumo do ICMS p/ Data Mart> DATA, porém está ocorrendo o erro do print abaixo:

Cliente informa que apensar de estar gerando para um único dia, este é um exemplo da volumetria que eles possuem.

Por gentileza, poderia verificar?

(Acreditamos que o erro apresentado possa estar relacionado a volumetria do cliente.)

## Root Cause

Causa: Clientes com volumetria grande causando erros de timeout no relatório em tela.

 Solução:  Mudança para o relatório em Jasper e geração em PDF para atender clientes que possuem grande volumetria de dados.
