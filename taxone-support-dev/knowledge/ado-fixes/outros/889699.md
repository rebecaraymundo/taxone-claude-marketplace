# [DIMED] TKT146622 - TAX ONE - MUNICIPAL - ATENDE-NET - Campo Largo - Registro tipo 20 - Campo Alíquota

## Descricao

TKT146622 - DIMED - TAX ONE - MUNICIPAL - ATENDE-NET - Campo Largo - Registro tipo 20 - Campo Alíquota 

 

Atualmente para a geração do campo alíquota, o sistema busca a informação do campo "Alíquota ISS" do documento fiscal, porém o cliente informa que isso não é correto, visto que se tratando de documentos com ISS retido, o campo que deveria ler seria "Alíquota ISS Retido". Por tanto esta demanda tem essa finalidade, solicitar o ajuste na regra de leitura do campo no documento fiscal, para buscar também a informação do campo Alíquota ISS retido, e não somente do campo Alíquota ISS. 

 

Regra consultada: https://trten.sharepoint.com/:w:/r/sites/REQ_MTZ/_layouts/15/Doc.aspx?sourcedoc=%7BD9163964-DA47-43F4-9897-E83DB934AB2B%7D&file=MTZ-AtendeNet-Geracao_do_Meio_Magnetico.doc&action=default&mobileredirect=true
