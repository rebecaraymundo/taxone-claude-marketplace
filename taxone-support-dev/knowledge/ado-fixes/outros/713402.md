# TKT116994 - VERO S.A. - ONESOURCE TAX ONE - ESTADUAL - ICMS - Nomenclatura do arquivo gerado com espaço indevido

## Descricao

Ao gerar o arquivo magnético, o nome do arquivo não é aceito no validador, sendo necessário ajuste manual para que o validador do Convenio 115 aceite a importação. 

 

Por gentileza, verificar conforme anexo, os arquivos originais gerados pelo cliente em outro sistema e que já estão no formato ideal. Cliente solicita que seja no Taxone assumido o mesmo critério sem espaços no nome.

## Root Cause

Causa: Nome do arquivo continha espaços em branco na string gerando problemas no validador.

 Solução:  Utilização do comando TRIM na montagem de cada variável que compoe o nome do arquivo eliminando os espaços em branco.
