# [SIM REDE DE POSTOS] TKT131245 - TAX ONE - ESPECÍFICOS - I-SIMP - Município ANP Pinto Bandeira não lista na parametrização

## Descricao

TKT131245 - SIM REDE DE POSTOS - TAX ONE - ESPECÍFICOS - I-SIMP - Município ANP Pinto Bandeira não lista na parametrização 

 

Ao tentar realizar a parametrização Município x Município ANP, o município de Pinto Bandeira não é listado para parametrizar na coluna/campo Município ANP 

 

O cliente enfrenta este problema no Tax One, por isso foi verificado tanto no Tax One (produção e homologação), quanto no DW, em ambos não lista o município. Verificando a regra sobre este campo, informa que os dados devem ser recuperadas do Cadastro de Cidades (TFIX113).

## Solucao

SaaS

## Release Notes

Carta do
Patch – DW e Taxone: 

Módulo: Específicos > i-SIMP

Menu: Parâmetros > Município x Município ANP 

     Inclusão do município de Pinto Bandeira para o município
ANP, através da atualização da TFIX113 - Cadastro de Cidades.
