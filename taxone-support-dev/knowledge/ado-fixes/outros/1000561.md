# TKT164045 - CIRION - ONESOURCE TAX ONE - ESTADUAL  Movimento de Saídas - Utilities esta gerando alguns documentos com valor contábil 0,00, apenas para o ultimo documento o valor é gerado no relatório

## Descricao

Cliente relatou que ao Gerar o relatório de Movimento de saídas - Utilites, alguns documentos estão sendo gerados com valor contábil = 0,00, apenas para o ultimo documento o valor contábil esta sendo gerado corretamente. 

 

Relatório gerado no Modulo Estadual > Operacional> Conferências> Movimento de Saídas - Utilities

## Root Cause

Causa: Campo relativo ao valor_tot_nota possui expressão de visualização não compatível com o TaxOne.

 Solução:  Internalização do controle para o PL/SQL do relatório para ser compatível nos ambientes do DW e TaxOne.
