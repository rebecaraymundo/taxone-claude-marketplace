# [PLUXEE BENEFICIOS] TKT111554 - DW DESIF - Módulo 1 - Registro 0410 - Campo 03 - Código de Inscrição Municipal

## Descricao

**[PLUXEE BENEFICIOS BRASIL S.A.] TKT111554 - DW - Municipal - DESIF - Módulo 1 - Registro 0410 - Campo 03 - Código de Inscrição Municipal da Dependência sendo preenchido com o CNPJ invés da Inscrição Municipal** 

Ao realizar a geração da DESIF, do módulo 1 - Módulo Demonstrativo Contábil, no registro 0410 - Balancete Analítico Mensal, no campo 03 está sendo gerado com o CNPJ, porém conforme o manual de layout e a regra, este campo deve ser preenchido com a inscrição municipal:

REGRA (https://trten.sharepoint.com/:w:/r/sites/REQ_MTZ/_layouts/15/Doc.aspx?sourcedoc=%7B65E44564-2D60-4503-ABEC-4955856BA12B%7D&file=MTZ-DES_IF-Geracao_do_Meio_Magnetico-Bancos.doc&action=default&mobileredirect=true):
