# [SOUZA CRUZ] TKT148029 - TAX ONE - BÁSICOS - JOB SERVIDOR - Erro de importação - Data Fiscal - SAFX07

## Descricao

Fazendo cópia da demanda 892869 pois o problema não está resolvido.  

O parametro  menu Preliminares > Parametrização para Data Fiscal para que a nota de saída seja preenchida pela data_saida_rec está ativo, mas o JobServidor não está respeitando esse parametro. Segue link da call: Thomson Reuters - Reunião _148029 - SOUZA CRUZ SA-20251127_095918-Gravação de Reunião.mp4 

 

Prezados, o cliente relata que, mesmo com a opção 'Parametrização para Data Fiscal' ativada, o sistema TaxOne não está considerando essa configuração durante a importação da X07, quando há limitação de período ativo. 

 

Opção limita período marcado 

  

 

  

  

Opção limita período desmarcado 

  

 

  

Pelo log sistema está considerando a data de emissão, pois a data saída recebimento é julho 

  

 

  

Precisamos importar algumas notas fiscais para fechamento do período 07/2025 e encontramos um erro na importação da safx07. As notas de saída onde a data de emissão é 30/06/2025, porém a data fiscal(data saída recebimento) esta com data 01/07/2025 não são importadas. Quando colocamos no job limita período, e colocamos o período de 01/07/2025 a 31/07/2025 o sistema não acha registros. Quando tiramos a opção limita período ele da erro, dizendo que as notas não podem importadas pois período esta fechado. Mas as notas são escrituradas no mês 7. Aparentemente o sistema está considerando a data de emissão e não data fiscal, mesmo com o parâmetro de data_fiscal para notas de saída no menu parâmetros. 

 

------------ 

 

O relatório de parametrização para data fiscal, SAFX e demais arquivos estarão em anexo.  

O teste de suporte estará no campo "Detailed Simulation of the Problem"

## Root Cause

Falha de produto.

## Solucao

SaaS

## Release Notes

Carta do Patch:  DW e TAXONE 

Módulo: Básicos > Job Servidor 

Menu: Importação > Importação > Execução 

       Ajuste na importação da tabela SAFX07 (Capa de Documento Fiscal), passando a considerar o parâmetro Parametrização da Data Fiscal, quando marcado o período informado na importação (quando a opção limitar período estiver marcada) buscará pelo campo data_saida_rec, sem o parâmetro marcado a data de emissão continuará sendo considerada.
