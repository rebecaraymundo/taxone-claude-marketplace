# [AGCO DO BRASIL] TKT121937 - 128581-TAXONE - JOB SERVIDOR - ERRO NA IMPORTAÇÃO SAFX13 CAMPO VLR_AQUIS

## Descricao

Cliente abriu chamado com crítica de Erro não tratado pelo sistema na importação da SAFX13. 

Validamos que o erro só ocorre quando o campo VLR_AQUIS está preenchido. 

Solicitamos apoio para validação e correção do erro. 

 

Att:. 

Douglas Fernandes

## Root Cause

Causa: Formatação de campos no arquivo incompatível com a interpretação dentro da procedure para valores zerados e nulos.

 Solução: Ajuste na interpretação dentro da procedure para valores zerados e nulos de campos numéricos.

## Release Notes
