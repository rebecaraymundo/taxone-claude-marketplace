# [CJ DO BRASIL] TKT118560 - TAX ONE - BÁSICOS - REPORT FISCAL - PRODUÇÃO - ERRO NO RELATÓRIO DE ORDEM DE PRODUÇÃO

## Descricao

No dia 31/10/2024 foi aberta demanda 702905 solicitando correção no Relatório de Ordem de Produção no Módulo do Report Fiscal. 

Na ocasião foi relatado que o relatório estava abreviando valores impedindo a conferência correta dos itens. 

Cliente retornou afirmando que o relatório continua apresentando os mesmos erros. 

Solicitamos apoio para esta situação. 

Att:. 

Douglas Fernandes

## Root Cause

Causa: Arquivo salvo como *.CSV ou *.XLSX não trazia todos os 6 caracteres decimais apresentados no relatório em tela.

 Solução: Aplicação de formula para montagem do valor com as casas decimais corretas com o campo em string.
