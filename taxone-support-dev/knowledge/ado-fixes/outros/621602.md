# REINF - R2010 - Retificação REINF Legado

## Descricao

Eu como usuário(a) fiscal do Registro R2010 da EFD Reinf Gostaria que fosse implementada a possibilidade de processar eventos de retificação, cujo o evento original foi enviado por outros sistemas, mas que as notas fiscais (legado) tenham sido importadas a nossa solução fiscal. Para possibilitar a retificação dos registros Reinf
 
 Funcionalidade Existente:  Agrupamento: SPED Módulo: EFD - Reinf Menu: REINF - Geração Prévia - Movimentos R-2010 
 Detalhes Adicionais 
- Solução de contorno para possibilitar a retificação de eventos que originalmente não foram enviados pelo DW/TaxOne; 
- Considerar apenas R-2010 como piloto para o processo; 
- Premissa Processo 
- Importação de XML via OneourceBR; 
- Tabelas SAFX07/08 importadas do período à ser retificado  
- Cliente Bayer apoiando na validação do processo;
