# [RAÍZEN] # - TAXONE - Municipal - Manaus - NFSE ABACO - NF Sem ISS

## Descricao

Eu como usuário(a) fiscal do Município de Manaus - NFSE Abaco Gostaria que fosse ajustado para trazer todas as notas, ou apenas as notas com retenção Para que o cliente possa definir via parâmetro 
 Funcionalidade Existente:  Agrupamento:  Módulo: Municipal Menu: Manaus 
 Detalhes Adicionais 
- Cenário apenas constante em alguns municípios, não há ato legal, mas é totalmente devido o cliente considere apenas mandar notas com retenção, sem necessidade de mandas as demais, devemos permitir isso. 
- Cenário ajustado na Nota Carioca

## Root Cause

Solicitada a nclusão de um parâmetro para definir se serão geradas as notas com retenção de ISS, ou sem retenção ou todas as notas

## Release Notes

Carta do
Patch - DW e TAX ONE: 

Módulo: Municipal >  > Manaus - NFSE Abaco 

Menu: Obrigações > Arquivo de Entradas de Serviços 

Inclusão de um novo parâmetro na tela de
geração, permitindo selecionar se o arquivo irá conter todas as notas fiscais,
apenas as notas fiscais com retenção de ISS, ou somente as notas fiscais sem
retenção de ISS.
