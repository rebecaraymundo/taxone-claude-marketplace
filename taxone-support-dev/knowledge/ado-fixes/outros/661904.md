# TKT 100361 [INTELBRAS] TAXONE - FEDERAL - OTF - DCTF - Ficha de Pagamento/Compensação - Fase 02

## Descricao

Analise DCTF  Com relação a geração e entrega da DCTF via solução TR, através da SAFX537 é possível importar as informações para a Ficha Débitos, porém, para a importação de informações para a tela de Pagamentos não existe uma SAFX disponível. No cenário Intelbras, temos um volume grande de pagamentos para informar na DCTF, o que torna inviável utilizar a solução oferecida pela plataforma caso o preenchimento tenha que ser feito manualmente.
Gostaria de solicitar a disponibilização de uma SAFX para que possamos fazer a importação dos dados de pagamentos e assim utilizar essa opção oferecida pela solução Tax One.
