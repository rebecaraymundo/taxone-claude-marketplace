# TICKET 92884 - MCCAIN DO BRASIL - TAX ONE - Básicos - Ferramentas - Sistema não esta permitindo inserir novo codigo de Classe de Vencimento - PROD e UAT

## Descricao

Na tentativa de inclusão de classe de vencimento de SC 11992 no módulo Ferramentas - Classe de vencimento o sistema não está habilitando o campo UF para preenchimento, impossobilitando a inclusão do novos códigos.

## Root Cause

Causa: Datawindow do tipo GRID apresenta problemas para gravação do registro em TaxOne

 Solução: Datawindow migrada para tabular, ajuste no alinhamento dos campos e dropdown's.

 N/A
