# [MRV] TKT154619 - TAX ONE - MUNICIPAL - CAJAMAR 'GEISWEB'-  Geração do meio magnético, opção gerar o arquivo "Quebrar o arquivo por data de Emissão"

## Descricao

Cliente necessita que o meio magnético para o Munícipio de Cajamar, seja incluído a opção "Quebrar arquivo por data de Emissão". 

 

Atualmente no modulo, para este município, esta disponível somente a opção Gerar movimento a partir "Data de emissão".

## Solucao

SaaS

## Release Notes

Carta do
Patch: DW e TAXONE 

Módulo
Municipal > Cajamar 

Menu
 Obrigações > Meio Magnético 

     Inclusão
de um parâmetro na tela de geração, para fazer a quebra do arquivo por
data de emissão do documento fiscal.
