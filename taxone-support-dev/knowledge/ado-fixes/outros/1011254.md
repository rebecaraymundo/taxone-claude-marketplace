# TKT166328 - FEMSA - ONESOURCE TAX ONE - BÁSICOS - DATA WAREHOUSE - Relatório trazendo informações incoerentes

## Descricao

Time, 

Cliente informa que o Relatórios > Conferências > Documento Fiscal - ICMS/IPI - Formato XLS está trazendo informações erradas. 

Ocorre que se o cliente flega apenas a regra "30078 - Nao existe item para esta capa", o relatório não traz nenhuma informação, pois de fato, não existe capa sem item ( período testado 02/2025) 

Porém se marcar todas as regras, aparece erros de capa sem item, mesmo que não haja. 

O comportamento nesse relatório está trazendo resultados inconsistentes. Esse comportamento é intermitente, ora acontece, ora não. 

Relatório gerado para todos os estabelecimentos, marcando apenas a opção 'Não existe item para esta capa' 

 

 

 

 

 

 

Agora marcando todas as opções: 

 

O relatório gerado trás informações de notas que possuem seus itens devidamente no sistema, como por exemplo a nota 000296
