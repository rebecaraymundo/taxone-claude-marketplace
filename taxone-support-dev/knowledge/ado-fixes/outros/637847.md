# [ADATA ELECTRONICS] TKT98043 - TAXONE - DATA WAREHOUSE - Tela Data Mart  - Caracteres inválidos

## Descricao

Cliente ADATA ELECTRONICS BRAZIL S/A relata que na tela do Tax One está aparecendo erro de caracteres em vários Módulos do sistema.

Solicitamos apoio ao Time de TechOps para verificar se o Charset do DUMP está no formato correto. Mas solicitaram direcionar para Sustentação pois para correção desse cenário precisão saber qual objeto que devem compilar no banco de dados. 
 Att:. Douglas Fernandes

## Root Cause

Causa: Cliente ADATA ELECTRONICS com procedure apresentando caracteres especiais de UTF-8.

 Solução: Recompilação da procedure com encoding ANSI na base de UAT do cliente.
