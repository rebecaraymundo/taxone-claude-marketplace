# [LASA] TKT114808 - TAXONE - RESSARCIMENTO DE ICMS-ST - PR - Cálculo do Ressarcimento - NF Devolução

## Descricao

[TKT114808 - LOJAS AMERICANAS S/A - ONESOURCE TAX ONE - ESTADUAL - RESSARCIMENTO DE ICMS-ST - PR - Cálculo do Ressarcimento / Complemento desconsiderando documento fiscal de saída quando existe devolução correspondente}
 

 

No processamento do Ressarcimento ICMSST - Paraná, identificamos que, por algum motivo, o sistema está desconsiderando notas fiscais de venda a consumidor final quando ocorre uma devolução. 

Pela análise que fizemos, aparentemente foi desconsiderada porque o sistema entendeu que a venda estava gerando estoque negativo. Entretanto, analisando as ultimas entradas do material, não era para ter sido considerado negativo. 

 

Inconsistência similar a este caso foi tratado no #10939 (pacote desvinculado). Após a aplicação do pacote desvinculado, o centário que haviamos reportado no TK #10939 foi corrigido. Porém, ao enviar o novo arquivo para a SEFAZ, outros materiais passaram a apresentar erro similar. 

 

Linhas retornadas com erro na validação junto a SEFAZ PR: 

Linha Registro Campo Mensagem Regra 

40696 1220 N_ITEM_REF Chave de acesso referenciada nao foi declarada no arquivo G18d-40 

67666 1220 N_ITEM_REF Chave de acesso referenciada nao foi declarada no arquivo G18d-40 

93056 1220 N_ITEM_REF Chave de acesso referenciada nao foi declarada no arquivo G18d-40 

 

Considerando o material que consta na linha 40696 (cod_produto = '4263650'), todas as notas fiscais de saida deveriam ter sido apresentadas no Arquivo ADRCST, pois o total de saidas resulta na quantidade de 1887, e as entradas = 4097, considerando a busca desde Out/2021.
