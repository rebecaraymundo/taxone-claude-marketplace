## Descricao

**TKT116704 - HUGHES TELECOMUNICACOES DO BRASIL LTDA. - ONESOURCE TAX ONE - SPED - EFD - ESCRITURAÇÃO FISCAL DIGITAL - Registro G126 não gera para o estabelecimento centralizado na geração por Inscrição Estadual Única** 
 
 Quando geramos o SPED Fiscal - Registro G126, marcando o parâmetro "Geração Inscrição Estadual Única", o G126 é gerado somente com os dados do estabelecimento centralizador, para o estabelecimento centralizado não gera.

## Release Notes

Ecd --> Geração --> Meio Magnético --> Gerar o Sped 
 Ajuste na geração do Sped para gerar o Registro G126 dos estabecimentos centralizados.
