# [SHOPEE] TKT93317 - TAXONE - BASICOS - DATA WAREHOUSE - O relatório de Documentos Duplicados não é gerado no ambiente da Shopee.

## Descricao

O relatório localizado em Básicos > Data Warehouse > Relatórios > Conferências > Documentos Duplicados está apresentando o erro 504 durante a geração. 
 Este problema já estava sendo tratado na demanda 623140, mas após a correção o cliente efetuou um novo teste e o relatório ainda não é gerado para os estabelecimentos com mais volume de dados. 
 Para estabelecimentos menores o relatório e gerado (Exemplo PR). Mas para outros com mais dados continua com o Erro 504.
