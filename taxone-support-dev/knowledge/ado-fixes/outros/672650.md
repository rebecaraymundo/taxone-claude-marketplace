# [SANTANDER] TKT104954 - DW - REINF - Eventos não retornados para base do DW

## Descricao

**TKT - 104954 - DW - REINF - Os eventos estão no OS com status Retornado do Governo com Sucesso e não estão na base do DW.  Apoio N3** 

 
 Os eventos estão no Onesource com status "Retornado do Governo com Sucesso", mas os eventos não estão na base do DW.

## Release Notes

REINF --> Envio de Eventos - Grandes Volumes --> Família 4000
 Gerando em uma sessão
 REINF --> Geração Prévia --> Movimento Abrindo um novo executável desse módulo e gerando em outra sessão após início da geração em grandes volumes
 
 Nesse cenário a geração prévia apaga os eventos que estão sendo enviado para o governo e grava outros registros na tabela de ocorrência ocasionando falta de sicronia no processo. Criado um controle de concorrência para bloquear uma geração enquanto a outra estiver gerando.
