# [HUGHES] TKT151115 - TAX ONE  - ICMS - Convênio ICMS 115 - Hash code não importa quando a descrição do arquivo está com "c" minúsculo

## Descricao

TKT151115 - HUGHES - TAX ONE - ESTADUAL - ICMS - Convênio ICMS 115 - Hash code não importa quando a descrição do arquivo está com "c" minúsculo 

 

Quando o arquivo hash code é gerado validador, o mesmo gera com C maiúsculo, e no momento só está importando com c minúsculo, onde o cliente está tendo que fazer o ajuste manual para conseguir fazer a importação. 

 

Conforme alinhamento entre o Marcelo, e o próprio cliente, é que tenha um tratamento no Tax One, para que o arquivo possa ser importado independente do C ou c.
