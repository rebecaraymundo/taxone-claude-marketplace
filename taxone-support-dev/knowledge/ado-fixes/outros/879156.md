# TKT145245 - FOSNOR - ONESOURCE TAX ONE - MUNICIPAL - CAMPO ALEGRE DE LOURDES - Avaliar a regra para não considerar notas fiscais quando o prestador é no mesmo município do estabelecimento.

## Descricao

Validador: SAATRI NFSE 

Município: Campo Alegre de Lourdes 

 

Avaliar a regra para não considerar notas fiscais quando o prestador é no mesmo município do estabelecimento ( Campo Alegre de Lourdes) 

 

Foi emitido o Infolegis nº 2252/25 da inclusão do município onde foi mencionado: 

 

Na regra de negocio não consta regra especifica para o município de Campo Alegre de Lourdes 

 

 

 

 

Hoje as notas ficais do mesmo município do estabelecimento está sendo considerado no arquivo, existe a parametrização de Notas Fiscais Eletrônicas, porem não funciona para esse município. 

Avaliar o município e o validador para considerar essa parametrização.

## Root Cause

Alteração da regra de negócio.

## Solucao

SaaS

## Release Notes

Carta do
Patch - DW e TAX ONE:
 

 

   Módulo: Municipal > CAMPO ALEGRE DE LOURDES 

 
 Menu: Obrigações > Meio Magnético 

     Nesta demanda será liberado alteração na regra de geração do meio
magnético que permitirá ao usuário definir, via parametrização, se as Notas Fiscais
Eletrônica de Entradas (Tomador domiciliado no
município do Estabelecimento), deverão ser consideradas no arquivo Meio
Magnético, para o validador SAATRI NFSE. 

Para
maiores detalhes consultar o Help do módulo. Localização: Municipal >
Parâmetros por Município > Parâmetros > Notas Fiscais Eletrônica.
