# [DIAGEO] TICKET 104169 - TaxOne - ICMS > Operacional > Listagens da Apuração > Apuração icms dif alíquota UF orig/des - EC 87/15. Relatório sem movimento gerado com data 00/00/0000

## Descricao

Quando o usuário gera o relatório de diferencial de alíquota por meio do TAX ONE e não há movimento, o período não está sendo preenchido. 
 
 
 No entanto, é importante que esse campo seja preenchido para facilitar a geração de evidências para fins de histórico e conformidade, pois o cliente pede para controle de compliance.

## Root Cause

Causa: Datawindow sem dados com data 00/00/0000 no relatório.

 Solução:  Criação de datawindow Dummy para demonstrar somente dados do cabeçalho quando não houver registros a serem recuperados pelo relatório.

 N/A
