# [REFINARIA DE MATARIPE] TKT119298 - TAXONE - SPED FISCAL -  Registro H005

## Descricao

**TKT119298 - REFINARIA DE MATARIPE S.A. - TAX ONE - SPED Fiscal -  Geração mensal do Registro H005 quando não há movimento para CNAEs específicos** 

O cliente precisa gerar o registro H005 mensalmente mesmo que não haja movimento no período, devido ao CNAE que possui: 4681801. 

 

Após realizar testes, identificamos que o TAX ONE não gera este registro quando não há registro na SAFX52 para o período. 

 

Fizemos a consulta junto a base tributária, que nos retornou: 

 

"Maria, boa tarde! 

 

De acordo com GP da EFD ICMS/IPI 3.1.6, as empresas que exerçam as atividades descritas na CNAE-Fiscal sob os códigos 4681-8/01 e 4681-8/02 deverão apresentar o registro H005, mensalmente, a partir de julho de 2012, não vinculada a apresentação do registro a movimentação do estoque. 

 

Att. 

Tatiane Assis" 

 

Verificando a regra do Bloco H, referente ao registro H005, tem um trecho que faz menção a situação de quando não há registro na SAFX52: 

 

Caso não existam registros na SAFX52 para a data informada, o procedimento deve ser: - gravar o registro H005 com o campo VL_INV = zero; - gravar mensagem no log de erros (mensagem de número 338 da planilha de erros);

## Solucao

SaaS

## Release Notes

Menu: Geração >> Meio Magnético >> Gerar o SPED 
 Correção na geração do Registro H005 para ser gerado quando não tiver registro na X52.
