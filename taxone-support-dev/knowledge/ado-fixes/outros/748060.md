# [RAIZEN] - TKT118752 - DW - SPED - REINF - O relatório da família 4000 foi alterado a forma de geração impactando o cliente RAIZEN

## Descricao

Análise suporte 
 Conforme solicitado foi aberto a demanda para avaliar a situação do Cliente RAIZEN, referente ao relatório da família 4000 do REINF, o qual foi alterado a forma de geração e impactou a operação do cliente.

## Root Cause

Causa: Rotina utilizava arquivo JMsafDW.bat ao qual o cliente reclama de ser impedido de executa-lo em seu ambiente devido ao seu anti virus.

 Solução:  Rotina refeita para executar o programa java JMsafDW.jar a partir do DW Powerbuilder.

 N/A
