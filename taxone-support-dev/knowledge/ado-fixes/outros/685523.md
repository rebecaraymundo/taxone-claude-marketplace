# [HUGHES] TKT109266 - TAXONE - Municipal > Louveira TAG de tributação deixou de ser obrigatória

## Descricao

O cliente informou que o campo 'Tributação' na tag <nf> das Notas Fiscais Recebidas já não é mais obrigatório. Atualmente, para a cidade de Louveira, nosso sistema emite o arquivo para o validador NFe Cidades.   
 
Ele nos enviou um manual referente a Louveira, que anexei a Demanda

 
 Seria possível avaliarmos a regra e incluirmos uma opção para que este campo seja gerado em "branco"? Lembrando que, em XML, campos não preenchidos não são gerados.
