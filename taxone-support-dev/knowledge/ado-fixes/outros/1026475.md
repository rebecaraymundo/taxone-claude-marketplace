# TKT170564 - TAX ONE - ESTADUAL - ICMS - Relatório Apuração ICMS Dif. Alíquota UF Orig/Dest - EC 87/15

## Descricao

Operacional >  Listagens da Apuração >  Apuração ICMS Dif. Alíquota UF Orig/Dest - EC 87/15 
 O relatório não está apresentando todas as movimentações para as UF quando informada a UF para selecionar o estabelecimento, considerando apenas a UF informada para exibição dos dados. 
 
 
Se selecionada a UF, por exemplo SC, a listagem de estabelecimento, apresenta somente estabelecimentos dessa UF, porém os dados do relatório também apresentam somente da UF de SC, quando deveria exibir todas as UFs em que o estabelecimento possuir movimentação de valor destino (no caso de NFs de saída).
 
 
 
 Se selecionada TODAS as UFs e o estabelecimento, os dados são gerados corretamente, porém a exibição das UFs não deveria estar limitada a geração dos dados e sim somente a seleção do estabelecimento.

## Root Cause

O relatório estava aplicando incorretamente o filtro de UF tanto nos estabelecimentos quanto nas movimentações fiscais. Isso fazia com que operações interestaduais com diferencial de alíquota EC 87/15 fossem excluídas quando a UF de destino era diferente da UF parametrizada, impedindo a visualização completa das operações sujeitas à retenção de ICMS entre diferentes estados. O filtro deveria ser aplicado apenas para selecionar os estabelecimentos da UF desejada, mas as movimentações deveriam incluir todas as UFs de destino.

## Release Notes

Carta do Patch - DW e TAX ONE: 
  
 
Módulo impactado: Estadual -> ICMS  
 
Menu: Operacional -> Listagens da Apuração -> Apuração ICMS Dif. Alíquota UF Orig/Dest - EC 87/15  
 
Ajuste para que o filtro por UF considere apenas a lista de estabelecimentos, não interferindo na geração dos dados do relatório, que deve exibir todas as UFs de destino, uma em cada página.
