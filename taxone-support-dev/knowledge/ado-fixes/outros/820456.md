# TKT134024 - FRETEBRAS - ONESOURCE TAX ONE - SPED - EFD - ESCRITURAÇÃO FISCAL DIGITAL - Registro 1400 não está gravando o código do produto

## Descricao

Ao gerar o SPED Fiscal com o parâmetro "Calcular Valores por Produto" selecionado em Dados Iniciais, o sistema não está considerando o código do produto. Mesmo após a inserção manual dessa informação por meio da tela de manutenção, o código não é gravado corretamente. 

Esse comportamento pode ser verificado na simulação anexa, evidenciando uma falha no processo de gravação ou no tratamento do parâmetro selecionado.

## Root Cause

Causa: Grupo, Indicador e código de produto não estão sendo gravados.
https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/795435
 
 Solução: Ajuste no processo de gravação para produto indicado ou não na janela.

## Release Notes

Módulo: SPED - EFD

 Resumo da Alteração: Ajuste no processo de gravação da janela relativo a dados de produto.
 
 Descritivo das Alterações: Ajuste no processo de gravação da janela relativo a dados de produto.

 Módulos Impactados: SPED - EFD
