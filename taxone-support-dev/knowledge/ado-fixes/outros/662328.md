# [LOJAS UNIAO] TKT101806 - TAXONE - SPED FISCAL = Para UF de SP o campo 05 - Cod_Rec do Registro E250 deve ser gerado de acordo com Portaria CAT 147/2009

## Descricao

Complementando, no Call que realizamos dia 12/06, a Samara e Fabiana mostrou a Portaria Cat 125 e explicou que o site não aceita mais o codigo 100099 para pagar a DARE. O site aceita o código 247-1 na DARE.

## Release Notes

Carta
do Patch: DW e TAXONE 

   
 Menu: Sped, Módulo EFD-Escrituração Fiscal Digital 

          Geração
> Meio Magnético > Gerar o SPED 

  

     Alteração da
regra de geração do Registro E250, campo 05 - COD_REC para a UF igual a SP,
para converter os códigos de receitas gerados, para os códigos disponíveis para
a DARE SP.
