# TKT164543 - COPA ENERGIA - ONESOURCE TAX ONE - SPED - REGISTRO E115 (GERAL) - PRÉ GERAÇÃO - Alteração regra de preenchimento do registro E115 para a UF do RJ

## Descricao

Conforme alinhado, após validação das bases legais para a geração do registro E115 (geral) da UF RJ, será necessário um ajuste na regra de preenchimento do registro, maiores detalhamentos abaixo: 

 

De acordo com o Manual de preenchimento de documentos fiscais e escrituração – Benefícios Fiscais, a orientação para preenchimento do registro E115 tanto na hipótese de isenção, quanto na de redução de base de cálculo é de que o contribuinte deve identificar o(s) benefício(s) utilizado(s) da seguinte forma:  

  

• Campo 02: preencher com o código da Tabela 5.2 correspondente ao dispositivo legal; 

• Campo 03: preencher com o valor 0; 

• Campo 04: não preencher 

 

Esperado o preenchimento conforme print abaixo: 

 

 

Atualmente o sistema esta preenchendo a informação para a UF do RJ conforme o print abaixo: Teste realizado na base de UAT do estabelecimento CAMILA para o período 10/2025. 

 

 

Pontos observados: 

1º O sistema esta preenchendo o campo 03 com os valores dos documentos fiscais que atendem os requisitos parametrizados; 

2° Mesmo com o flag "Não Preencher Descrição Complementar do Ajuste" desmarcado, o sistema esta trazendo no campo 04 a descrição para um código e para o outro não.

SaaS

## Release Notes

Carta do Patch – DW e TAXONE: 

  

Módulo: SPED - Escrituração Fiscal Digital 

  

Resumo da alteração: Alteração da regra de preenchimento do campo 03-VL_INF_ADIC do registro E115 do SPED-EFD. 

  

Descrição das alterações: 

  

- Criação de um parâmetro na tela de Informações Adicionais da Apuração, que vai definir a forma de preencher o campo 03-VL_INF_ADIC do registro E115: 

  

      Módulo SPED - Escrituração Fiscal Digital, 

            Menu Parâmetros > Registros E115/1925 > Informações Adicionais da Apuração (Registro E115/1925) 

  

- Alteração da pré-geração do Registro E115 para fazer o tratamento do parâmetro criado: 

  

       Módulo SPED - Escrituração Fiscal Digital 

           Menu Pré-Geração > Registro E115 (Geral) 

  

ATENÇÃO: Antes da criação do parâmetro,  o campo 03-VL_INF_ADIC do registro E115 não era preenchido para estabelecimentos do RJ.  A partir da liberação desse parâmetro, o preenchimento do campo 03-VL_INF_ADIC do registro E115 vai ser feito
