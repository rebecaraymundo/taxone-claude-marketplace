# INFOLEGIS 2176/24-C - ISSQN - SÃO JOSÉ DO RIO PRETO/SP - Layout XML (GissOnline) - DW/T1

## Descricao

Conforme informações disponibilizadas pela Prefeitura de São José do Rio Preto, a partir de 01/11/2024 foi implantada a nova versão da GissOnline, que substitui o sistema de escrituração fiscal (GissOnline - https://portal.gissonline.com.br) e o sistema de emissão de NFS-e (Ginfes - https://sjrp.ginfes.com.br).

 Conforme informações do Manual, visando facilitar o registro de serviços comprados (serviços tomados), dando maior autonomia e agilidade nas declarações de serviços, a Gissonline disponibiliza ao contribuinte a opção de importação de arquivos XML de notas fiscais recebidas. Este recurso permite que o contribuinte realize a declaração fiscal eletrônica gerando
 num mesmo arquivo XML até 50 notas fiscais, e posteriormente realizar o upload dos arquivos gerados quantas vezes for necessário. O arquivo possui um layout próprio, baseado no modelo ABRASF 2.04, trata-se de um arquivo XML com codificação UTF-8.
 
 O novo sistema está disponível no endereço: https://sjrp.giss.com.br/#/
