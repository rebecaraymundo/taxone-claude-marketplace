# TKT171350 - MUNDIAL - ONESOURCE TAX ONE - BÁSICOS - FERRAMENTAS - SISTEMA NÃO ESTA SALVANDO A INFORMAÇÃO NO CAMPO 'FIM VIGÊNCIA' NA TABELA DE NCM

## Descricao

SISTEMA NÃO ESTA SALVANDO A INFORMAÇÃO NO CAMPO 'FIM VIGÊNCIA' NA TABELA DE NCM. 

TESTE REALIZADO EM AMBIENTE DE PRODUÇÃO DO SUPORTE 

EMPRESA = 076 

ESTABELECIMENTO = BR001 

 

PASSO A PASSO 

1° Acessar Básicos -> Ferramentas -> Tabelas Internas -> Manter -> Tabelas 

 2° Escolho a opção Códigos de Ativ Prod e Serv Sujeitos à Contrib. Rec.Bruta 

 3° Procuro pela NCM 3924 e informo a data fim 31/12/2024 

 4° Clico no Lápis para poder editar o registro, coloco a data de fim vigência e fecho. 

 5° Clico em salvar, ele mostra a mensagem que foi salvo 

 6° Procuro novamente a NCM 3924 e a data de fim vigência está branco 

 

Evidências em anexo

## Root Cause

Causa: Datawindow's do tipo GRID no taxone tem problemas de identificação de linhas alteradas impedindo a gravação de informações alteradas.

 Solução: Passar as 71 datawindow's utilizadas por essa janela para datawindow do tipo TABULAR.
