# [LINDEMANN] TKT111166 - TAXONE - ICMS - Valor de saldo devedor gerado indevidamente no Livro de Apuração do ICMS ST

## Descricao

Favor verificar o erro que esta aparecendo no livro de apuração do ICMS / ICMS ST, onde indevidamente esta trazendo no relatório saldo devedor de ICMS ST, sendo que o cliente possui apenas saldo credor do período anterior (31/07) atualizado no mês 08/2024. 

Segue dados para acesso ao TAX ONE PRD: 

Usuário: Suporte-Thomson-Lindemann
Senha: Gsw@2024

## Root Cause

Causa: Valor de Saldo Devedor (Debito Menos Credito) gerando valores negativos

 Solução:  Ajuste na procedure de cálculo para nesses casos zerar o valor quando negativo.
 
 Ações preventivas/Boas práticas:
 N/A
