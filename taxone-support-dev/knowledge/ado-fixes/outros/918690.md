# [MRV] TKT152902 -  TAX ONE - MUNICIPAL - SPE - Macaé  - "Divergência no Preenchimento do Campo "Código de Serviço Municipal" para a Prefeitura de Macaé"

## Descricao

Recebemos um contato do cliente informando que, para a Prefeitura de Macaé, o campo "Código de Serviço Municipal" deve ser preenchido com o mesmo valor utilizado no campo "Código de Serviço Lei 116/03". Atualmente, o sistema está utilizando outro campo para essa informação. 

o sistema está utilizando o valor do campo destacado em amarelo. 

  

 

 

 

Foi solicitado um teste de validação com a Eunice, mas o cliente preferiu não realizar devido à insatisfação com o comportamento atual. 

 Segundo o manual da prefeitura, os campos devem seguir a estrutura da Lei Complementar 116/03 (campo 28) e da Lei Complementar 282/2018 de Macaé (campo 29). Não foram identificadas alterações legislativas recentes que justifiquem a mudança no comportamento do sistema. 

Teste realizado foi anexado para referência.

## Root Cause

Alteração de legislação.
