# TKT 110501 - TAXONE - ECD - Geração - Saldos - O relatório de consulta ou geração de saldos gerados para uma empresa está listando o cabeçalho de outra.

## Descricao

O Cliente está gerando os saldos contabeis da empresa e o relatório de confêrencia está trazendo dados de cabeçalho de outra empresa. São duas matrizes diferentes e mesmo estando na empresa selecionada e os parametros de geração tbm, traz o relatório com cabeçalho de outra Matriz e os dados gerados da empresa selecionada.

## Release Notes

Carta do Patch DW e Taxone: 

  

  

Menu SPED, Módulo: ECD 

          Geração > Saldos 

   Alteração da recuperação dos dados de estabelecimento no cabeçalho do relatório, corrigindo inconsistência possível quando há estabelecimento com o mesmo número para empresas diferentes.
