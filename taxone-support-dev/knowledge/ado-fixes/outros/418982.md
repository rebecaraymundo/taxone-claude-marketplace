# [MFS-89760] BÁSICOS - JOB SERVIDOR - Validação da Importação de Tabelas com campo Código de Produto com 60 posições em SAFX2093 a SAFX265 - REQ/DEV_PL/QA

## Descricao

Essa MFS tem como objetivo validar, nas soluções DW e TaxOne, os processos de Carga / Importação / Importação Batch / Importação Automática e Exportação, das tabelas listadas abaixo, cujo campo COD_PRODUTO aumentou o tamanho para 60 posições:
 

 

- SAFX2093 
- SAFX214 
- SAFX215 
- SAFX234 
- SAFX236 
- SAFX237 
- SAFX26 
- SAFX260 
- SAFX261 
- SAFX265  
 
No início do projeto foi aberta uma demanda por SAFX. Com o andamento do projeto identificamos que seria mais produtivo entregar 10 SAFX's por demanda. Com isso atualizamos escopos de demandas e encerramos as que foram unificadas.
 
 
Ações do BA: 
    1) Fazer requisito com o levantamento de campos a serem alterados das SAFX. 

    2) Alterar o Manual de Layout SAFX e Planilha de Alteração de SAFX; 

    3) Abrir demanda para o time de Interfaces (SAP e Oracle) 

    4) Abrir demanda para o time de Conteúdo para publicar os manuais na base de conhecimento. 

Ações do Desenvolvedor: 

 

 

   1) Confirmar se as Cargas das SAFX's estão com o campo de Produto alterado para 60 posições. (PB) 

   2) Alterar a procedure de Exportação da SAFX, para gravar o campo no arquivo txt com 60 posições. 

   3) Alterar as TFIX96 e TFIX97 

   4) Tira as evidências das alterações de todas as SAFX nos ambientes de DEV e RC.  

  5) Evidências dos objetos de Carga das SAFX's mostrando o codigo do produto com 60 posições. (PB) 

 

 

Ações do QA:
 

   1) Testar o fluxo completo (carga, importação, exportação) no DW e Taxone de apenas uma SAFX.
