# TKT130657 - LKM TECNOLOGIA LTDA - ONESOURCE TAX ONE - SPED - EFD - REINF - Erro ao gerar eventos com datas extemporânea 

## Descricao

Cliente reporta erro no log ao gerar um evento original com datas extemporânea, veja abaixo o erro no log: 

 
 Creio que esse problema seja em relação a volumetria, pois na base UAT esse erro não ocorre.

## Release Notes

REINF -->  Geração Prévia --> Movimentos (R-2010) 
  Erro ORA-01422 ocorreu devido à importação de arquivos legados, resultando na criação de identificadores diferentes para o mesmo CNPJ do prestador. Isso aconteceu porque um identificador foi gerado pela importação de dados legados e outro pelo sistema. Para resolver esse problema, foi feita uma alteração na geração do evento R-2010, de modo que as informações legadas sejam ignoradas durante o processo de geração.
