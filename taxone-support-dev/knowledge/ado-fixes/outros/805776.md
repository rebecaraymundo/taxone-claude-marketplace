# [HUGHES] TKT131966 - TAX ONE - ESTADUAL - ICMS - Convênio ICMS 115 - Importação do hash code

## Descricao

TKT131966 - HUGHES - TAX ONE - ESTADUAL - ICMS - Convênio ICMS 115 - Importação do hash code 

 

Ao realizar a importação do hash code, é exibida a mensagem no log: 

 

Código do erro: 101086 

Mensagem de erro: "Arquivo de Controle relativo ao arquivo Mestre não identificado no diretorio informado" 

Conteúdo: CE05206385002881221A2304N01c.001

## Root Cause

Causa: Processo de montagem do nome do arquivo de retorno ('Controle') não conseguia reconhecer o arquivo correto.

 Solução: Refeito o parcialmente o processo de montagem do nome do arquivo a partir do arquivo Mestre.

## Solucao

SaaS
