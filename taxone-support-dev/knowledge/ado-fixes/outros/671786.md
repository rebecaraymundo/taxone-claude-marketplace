# [YAMAHA]TKT106933 - DW - SPED Fiscal - Registros 0200 e 0221 - sendo gerados indevidamente

## Descricao

Demanda aberta conforme alinhamento com Andrea para fins de tratamento do Registro 0221. 

O Registro 0221 está sendo gerado quando existe movimentação do ítem nas operações que não é obrigatório o Registro C170. Como exemplo as entradas formulário próprio (E/S = 4) ou saída de NFe emissão própria, cujo C170 não deve ser gerado.
