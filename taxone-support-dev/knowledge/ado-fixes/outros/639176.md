# Servidor de Relatórios - TAXONE - Básicos - Report Fiscal - Conferência de Documentos Fiscais (ICMS/IPI/PIS/COFINS) - Report Server SEM MOVIMENTO (DEV)

## Descricao

Ramo: MFS575311: Menu: Gerenciais >> Documentos Fiscais >> Analíticos >> Conferência de Documentos Fiscais (ICMS/IPI/PIS/COFINS) - Report Server Aplicar tratamento do período sem movimento para o relatório.  Tratamento para Período sem Movimento:
 
- Arquivo PDF incluir texto “SEM MOVIMENTO” no
cabeçalho. Cabeçalho deve conter os dados de estabelecimento, período como
ocorre para um período sem movimento. 

- Arquivo EXCEL incluir a coluna
“PERIODO_COM_MOVIMENTO” e preencher com “NÃO” para períodos sem movimento. 

 ***** A coluna “PERIODO_COM_MOVIMENTO” deve ser incluída ao final do relatório!! *******
