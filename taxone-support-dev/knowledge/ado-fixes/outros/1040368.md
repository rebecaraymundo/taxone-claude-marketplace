## Root Cause

O arquivo MOV0001 utiliza o caractere pipe ('|') como separador entre os campos do layout. Quando algum campo cadastrado no sistema contém esse mesmo caractere no seu conteúdo (por exemplo, em descrições ou observações), o arquivo gerado passa a ter separadores extras não previstos, quebrando a estrutura esperada do layout. Isso faz com que o sistema Onesource BR interprete o arquivo como inválido, pois a quantidade de colunas por linha não corresponde ao formato definido.
