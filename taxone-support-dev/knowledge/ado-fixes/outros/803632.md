# [DROGARIA SAO PAULO] TKT133145 - MASTERSAF DW - BÁSICOS - DATA WAREHOUSE - Botão "Selecionar Item" da tela de Notas Fiscais não está filtrando corretamente.

## Descricao

Botão "Selecionar Item" da tela de Notas Fiscais não está filtrando corretamente. 

Quando clicamos em Selecionar Item, a tela de pesquisa é aberta, mas os filtros não funcionam corretamente. 

A simulação do problema está no vídeo anexo. 
Pelo TRACE identifiquei que o problema ocorre pois no x08_itens_merc.ident_fis_jur está sendo passado 0 ao invés do IDENT correto.

## Root Cause

Causa: Botão "Selecionar Item" da tela de Notas Fiscais não está filtrando corretamente. Segundo o trace realizado a query dinâmica não considera o ident_fis_jur da nota para pesquisa.
 
MFS: https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/672643 

 Solução: Troca do campo ident_fis_jur para ident_fis_jur_dec na query dinâmica de pesquisa de itens devido a modificação realizada pela MFS672643 que incluiu o identi_fis_jur_dec (decimal) e não alterou o processo de pesquisa do objeto abaixo identificado para recuperação de itens com a nova variável criada.
