# [SUMITOMO] TKT120632 - TAXONE - SPED - EFD - ESCRITURAÇÃO FISCAL DIGITAL DAS CONTRIBUIÇÕES - F100 esta considerando sobre Tabela de Receita Bruta Mensal Para Fins de Rateio (0111).

## Descricao

SITUAÇÃO DO CLIENTE: O cliente utiliza o método automático (rateio de crédito nos dados iniciais). Ao gerar o relatório, percebeu que o registro F100 foi considerado na receita bruta do registro 0111. No entanto, de acordo com o cliente, os registros F100 não devem ser incluídos na receita bruta para rateio, pois são usados para escriturar aluguéis, juros, entre outros. Esse tipo de escrituração não é considerado receita direta no contexto da movimentação fiscal. 

 

O cliente forneceu embasamento legal, e por isso encaminhei a questão para nossa base tributária. Segundo a equipe, a inclusão do F100 pode ou não ser apropriada, dependendo de como o cliente utiliza o F100.
