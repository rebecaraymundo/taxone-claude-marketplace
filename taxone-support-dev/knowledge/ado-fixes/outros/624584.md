# [MARTINS]TKT94343 - DW - ESTADUAL - ICMS - Ao realizar a manutenção manual nos Débitos Especiais após a importação da Safx309 apresenta a seguinte mensagem: Falta valor obrigatório para coluna na linha 

## Descricao

O cliente faz a SAFX309 manualmente e importa mensalmente os dados de várias filiais ao mesmo tempo. No momento que o usuário importa a SAFX tem alguns lançamentos que ele não tem o valor do débito especial fechado, para este cenário ele importa com o valor de 0,01. Quando o usuário tem o valor apurado corretamente ele insere o valor manualmente via tela e ao salvar apresenta a seguinte mensagem de erro: “Falta valor obrigatório para coluna na linha 2. Por favor, entre com o valor.” 

Segue anexo evidências.

## Root Cause

Causa: Campos requeridos não preeenchidos.

 Solução: Checagem antes da gravação dos dados obrigatórios de preenchimento das linhas de débitos especiais.
