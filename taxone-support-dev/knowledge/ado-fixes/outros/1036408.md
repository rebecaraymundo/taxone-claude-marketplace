# [CLARO] - DW - BASICOS - JOB SERVIDOR - SAF_IB_X334 - erro ORA-54013: INSERT operation disallowed on virtual columns

## Descricao

importação batch diária da SAFX334, no ambiente da Claro, está apresentando o erro

## Root Cause

O erro ocorre porque a tabela SAF_IB_X334 possui uma ou mais colunas virtuais (provavelmente a coluna ROWID_S), e o processo de importação batch diária está tentando inserir dados diretamente nessas colunas. O Oracle não permite operações de INSERT em colunas virtuais, pois elas são calculadas automaticamente pelo banco de dados. Isso faz com que todas as tentativas de importação batch falhem com o erro ORA-54013, impedindo a carga diária de dados no ambiente da Claro.
