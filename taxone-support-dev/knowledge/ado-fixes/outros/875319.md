# TKT144570 - MRV - ONESOURCE TAX ONE - MUNICIPAL - ISSQN WEB - Avaliar a regra para não considerar notas fiscais quando o prestador é no mesmo município do estabelecimento.

## Descricao

Validador: ISSWEB 

 

Avaliar a regra para não considerar notas fiscais quando o prestador é no mesmo município do estabelecimento ( SERRA) 

 

Hoje as notas ficais do mesmo município do estabelecimento está sendo considerado no arquivo, existe a parametrização de Notas Fiscais Eletrônicas, porem não funciona para esse município. 

Avaliar o município e o validador para considerar essa parametrização.

## Root Cause

Alteração da regra de negócio.

## Solucao

SaaS

## Release Notes

Carta do Patch - DW e TAX ONE:
 

   Módulo: Municipal > Serra 

   Menu: Obrigações > Meio Magnético 

 

     Nesta demanda será liberado alteração na regra de geração do meio magnético que permitirá ao usuário definir, via parametrização, se as Notas Fiscais Eletrônica de Saída e/ou de Entradas (Prestador/Tomador domiciliado no município do Estabelecimento), deverão ser consideradas no arquivo Meio Magnético, para o validador ISS WEB. 

 

Para maiores detalhes consultar o Help do módulo. Localização: Municipal > Parâmetros por Município > Parâmetros > Notas Fiscais Eletrônica.
