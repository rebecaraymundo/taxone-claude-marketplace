# [MRV] TKT145429 - TAXONE - NOTA JOSEENSE  Código do município do prestador não esta sendo gerado conforme tabela do IBGE.

## Descricao

Ao validar o meio magnético, foi identificado o seguinte erro: "Código do município do prestador inválido". 

 

Após análise do arquivo gerado, constatou-se que o código do município não está sendo gerado de acordo com a tabela do IBGE. 

 

Exemplo: O código do município de Belo Horizonte, conforme a tabela do IBGE, é 3106200. No entanto, o valor que está sendo enviado para o meio magnético é 316200. 

 

O problema ocorre em todos os municípios. 

 

 

 

 

 

Cadastro PF/PJ: 

 

EX: No município de Alfredo Wagner apresenta no arquivo o código 42705, onde deveria ser apresentado 4200705. 

 

 

 

 

 

 

Ex: Para o município de Montes Claros (MG), o código do prestador preenchido no arquivo está correto e corresponde ao código da tabela do IBGE: 3143302.

## Root Cause

Alteração de regra de negócio.
