# [FEMSA] TKT148251 - TAX ONE - DATAWAREHOUSE - Erro na execução do relatório de capa x item

## Descricao

Caros, boa noite! 

Em 19/08/2025, o usuário que está realizando as cargar para o Tax One, no ambiente (emea1.onesourcetax.com) informou que fez a geração do relatório de capa x item referente aos dias 01 até 15/02/25. 

O tempo de execução foi de 3h23min e não gerou nenhum registro. 

Este mesmo relatório no DW demora menos de 10 minutos para gerar o mês todo. 
 

 
 

 
 Log Explorer | Datadog

## Root Cause

A solução para o erro JasperReports foi implementada com sucesso. O problema estava na incompatibilidade de tipos entre a definição do campo no relatório e o tipo de dados no banco.

## Solucao

SaaS

## Release Notes

corrigido problema no relatório de a incompatibilidade de tipos entre a definição do campo Modelo no relatório e o tipo de dados no banco.
