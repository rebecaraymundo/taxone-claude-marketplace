# [MRV] TKT156277 - ONESOURCE TAX ONE - MUNICIPAL - SIGISS - LONDRINA Campo Base de cálculo, não esta realizando a soma dos itens do documento fiscal

## Descricao

Ao escriturar documentos fiscais que possuem mais de um item (Cód serviço), o valor da base de calculo dos itens não esta sendo somada, levando para o meio magnético somente o valor da base de calculo do primeiro item, e ao submeter o arquivo no validador, esta apresentando os seguintes erros: 

 

Base de Cálculo não pode ser menor que o Valor da NF / Valor da Base de Cálculo não pode ser menor que 50% do Valor da NF. 

 

Exemplo documento fiscal: 

 

 

 

 

 

 

 

 

 

 

 

 

ITEM 01: 

 

 

 

 

 

 

 

 

ITEM 02: 

 

 

 

 

 

 

 

 

ITEM 03: 

 

 

 

 

 

 

 

 

Arquivo magnético: 

 

Considerou somente o valor da base de calculo 3787,67, correspondente ao preenchimento do item 01.

## Root Cause

Falha de produto.

## Solucao

SaaS

## Release Notes

Carta do Patch – DW e TAXONE: 

Módulo Municipal > Londrina 

Menu  Obrigações > Meio Magnético 

     Alteração de preenchimento do campo Base de Cálculo de ISS, passando a somar os valor caso haja mais de um item de serviço na mesma NF.
