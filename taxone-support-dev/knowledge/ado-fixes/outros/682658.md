# [HUGHES] TKT 109245 - TAXONE - Município > Jaboatão dos Guararapes: "Campo do valor de ISS Retido esta considerando o valor do ISS Normal. 

## Descricao

Assunto: Problemas na Integração dos Valores de ISS Retido para o arquivo de Jaboatão dos Guararapes

 Recentemente, a cliente entrou em contato para relatar um problema: os valores de ISS retido não estavam sendo corretamente transferidos para o arquivo gerado para o município de Jaboatão dos Guararapes. Ao testar e analisar as regras do sistema, identifiquei algumas inconsistências, que conversei com Bruna Ribeiro. Abaixo estão os pontos principais identificados:
 
 Validador de Dados: O validador atual usado para Jaboatão dos Guararapes é o Remessa XML, enquanto que o sistema está apresentando o DMSE - Declaração Mensal de Serviços Eletrônica como validador em tela.
 
 Transferência de Valores: O valor do ISS Retido que consta na nota não está sendo corretamente incluído na geração do arquivo pois ele esta considerando o valor preenchido no campo Valor ISS da SAFX08 ao invés de considerar o valor do campo Valor ISS Retido.
 
 Os detalhes dos meus testes foram adicionados à aba 'Análises da Situação' para revisão e acompanhamento.​
