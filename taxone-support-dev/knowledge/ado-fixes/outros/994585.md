## Root Cause

O erro ocorre porque o sistema tenta inserir dados na tabela SAFX08 informando valores para **todas** as suas colunas de uma vez, sem especificá-las individualmente. O problema é que essa tabela possui **colunas calculadas automaticamente pelo banco de dados** (colunas virtuais), que são preenchidas de forma automática e não permitem que valores sejam inseridos diretamente nelas. Ao tentar gravar dados nessas colunas calculadas, o banco de dados rejeita toda a operação de importação com erro.
