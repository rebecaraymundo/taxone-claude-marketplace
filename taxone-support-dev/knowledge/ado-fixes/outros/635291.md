# [SHPX LOGISTICA] TKT94586 - TAXONE - Inclusão de informação no Manual de Layout

## Descricao

**Básicos- Inclusão de informação no Manual de Layout - PROD - SHPX LOGISTICA LTDA.** 
Verificamos que quando é feito um processo de Importação da Tabela SAFX09, mesmo se for preenchido o campo BASE_ISS o valor não é importado se não for preenchido o Campo TRIB_ISS. O valor não é apresentado (Importado) para o Tax One. Somente com o Campo TRIB_ISS preenchido o campo é apresentado (Importado) na tela do Tax One.  
 No Manual de Layout não existe informação da necessidade do preenchimento do Campo TRIB_ISS para mostrar o campo BASE_ISS após a importação. Consultando a regra percebemos que o campo TRIB_ISS não é obrigatório e por este motivo solicitamos apoio com esta avaliação.

Solicitamos apoio para incluir a orientação da necessidade de preenchimento do campo TRIB_ISS ou a retirada da regra para que o valor do campo BASE_ISS possa ser mostrado após importação da Tabela SAFX09.
