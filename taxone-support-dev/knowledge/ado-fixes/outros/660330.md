## Descricao

Cliente necessita realizar a geração da DES SP, porém ao gerar ocorre o erro :  SAF_DES_TXT: ORA-29280: invalid
directory path
 
  Devido a esse erro, o arquivo não é gerado. Esse problema ocorre tanto no ambiente UAT do cliente quanto no nosso ambiente UAT interno. Atualmente, dispomos de apenas uma opção de diretório para seleção, que é a unidade u:\

## Root Cause

Causa: No taxOne janela ainda exibe a funcionalidade de escolha de diretório o que no T1 é direto na nuvem.

 Solução: Verificação se ambiente é TaxOne deixando invisível a opção de escolha de diretório.
