# [ROBIEL] TKT106298 - TAXONE - SPED - REINF - Cálculo da CPRB > Geração dos ajustes da CRPB não localiza os dados.

## Descricao

No módulo REINF temos rotinas para inserir de forma automatica dados na tela de contribuição previdenciaria sobre a receita bruta, isso a partir de notas conforme parametrização de CFOPs, indicador de atividade economica e código de receita. Verificado que a geração dos cálculos para buscar as receitas (saídas) é efetuada com sucesso, porém a geração dos ajustes não localiza os dados.
  Aba analysis consta simulação detalhada.
