# TKT139595 - GSR SERVIÇOS EM INFORMÁTICA LTDA - MASTERSAF DW - SPED - EFD - ESCRITURAÇÃO FISCAL DIGITAL - Erro na Pré-Geração do Ressarcimento ICMS-ST (MG): Error message is ORA-00001: restrição exclusiva (.) violada

## Descricao

Descrição do Problema: 

Durante a execução da rotina no módulo SPED Fiscal > Pré-Geração > Ressarcimento ICMS-ST (Específico MG) > Pré-Geração, o sistema apresenta a seguinte mensagem de erro no log: 

"Error message is ORA-00001: restrição exclusiva (.) violada." 

 

Inicialmente, o cliente estava utilizando o patch 304. Foi solicitado que atualizasse a base para verificar se o erro havia sido corrigido ou se a mensagem de erro se tornaria mais amigável ao usuário. 

Após a atualização, no entanto, passou a ocorrer um novo erro no log: 

"Erro na gravação das Saidas. (EFD_CALC_RESSARC_MG_C185_C330.gravaSAI)ORA-24381: erro(s) no DML do array" 

 

Solicito apoio para análise e identificação da causa raiz de ambas as mensagens de erro, a fim de permitir que o cliente prossiga com a geração do arquivo do SPED Fiscal/Ressarcimento.

## Root Cause

Causa: Quando identificado linhas iguais o processamento era interrompido por erro ORA de duplicidade.

 Solução: Ajuste no processamento para mitigar duplicidades durante o processamento.

On-Premise
