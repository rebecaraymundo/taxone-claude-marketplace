# TKT147904 - MRV - ONESOURCE TAX ONE - SPED - EFD - ESCRITURAÇÃO FISCAL DIGITAL DAS CONTRIBUIÇÕES - Gerando 1800 para SCP sem movimento

## Descricao

Situação do cliente: O cliente possui vários SCPs cadastrados, mas apenas um deles apresenta movimentação. No entanto, ao gerar os registros, o sistema está aplicando o registro 1800 para todos os SCPs, inclusive para aqueles sem movimentação.  

 

De acordo com o time de informações, os dados são separados para cada SCP, considerando apenas as movimentações específicas de cada um. 

﻿﻿﻿

## Solucao

SaaS

## Release Notes

Menu: Obrigações >> Geração dos Registros 
 Correção na geração do registro 1800 para gerar por SCP.
