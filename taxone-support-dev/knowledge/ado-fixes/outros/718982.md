# TKT116762 - TAX ONE - MUNICIPAL - GISSONLINE - ITU - Avaliar a geração da tag <tipos:CodigoMunicipio> para considerar o local da prestação do serviço.

## Descricao

Cliente: 

A retenção
de ISS no documento fiscal escriturado no TAX ONE, indica o município de ITU,
mas no arquivo gera o código de município do prestador. Desta forma, a retenção
não é calculada no sistema da prefeitura de ITU. 

 
Suporte: 

Avaliar a geração
da tag<tipos:CodigoMunicipio> para considerar o local da prestação. Pela
a regra atual é considerado o código do município do prestador, porem o manual de
layout a informação é considerar o “Código de identificação do município
conforme tabela do IBGE (Local de prestação do serviço)”. No caso do cliente o local de prestação foi no município de ITU. 

 

Regra: 

 

 

Manual de layout: 

 

 

 Simulação aba Analysis

## Root Cause

A documentação de requisitos solicitava desde sempre a recuperação da descrição do município para o campo, o programa porém recuperava o código do município.
