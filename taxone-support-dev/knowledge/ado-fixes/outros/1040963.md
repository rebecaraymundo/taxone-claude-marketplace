# [T1 - Emergencial] Relatório de Detalhamento por Produto - Geração – Report Fiscal / Report Server

## Descricao

ID ou URL da demanda aberta pelo(s) cliente(s): https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/1040462
 
 Cliente(s) atendido(s):  Savegnago e Bimbo
 
 Data limite para aprovação desta solicitação: 05/02/2026 
 Problema reportado: O Relatório de Detalhamento por ProdutoNão estava sendo gerado para o cliente em uma falha de performance durante a etapa de Geração. O problema ocorreu quando a geração foi acionada para um período de quatro dias, envolvendo simultaneamente 91 estabelecimentos. 
 Solução aplicada: 
Para contornar o problema e garantir a disponibilização das informações ao cliente, adotou-se a estratégia de disponibilizar os arquivos gerados em quebras por estabelecimento, no formato Excel, diretamente na etapa de geração. 

Com essa alteração: 

- O processo não realizará mais a população da tabela utilizada na etapa de emissão; 
- Consequentemente, o cliente não utilizará mais o processo de emissão, passando a obter o relatório exclusivamente via geração, já com os arquivos segmentados.   
 Detalhes sobre a estratégia de testes:  https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/1040806
