# TKT95184 - TAX ONE - BÁSICOS - REPORT FISCAL - Relatório Diferencial de Alíquotas p/ DATA MART p/ Documento (Formato Excel e resumido) não é gerado quando o período não tem movimento

## Descricao

Foi
liberado no TAXONE apenas, módulo Report Fiscal, os relatórios Diferencial de
Alíquota p/ DataMart, opções Documento – Formato EXCEL e Documento Resumido.
Menu: 

- Obrigações
Acessórias>> ICMS>> Diferencial de Alíquota p/ DataMart>>
Documento - Formato EXCEL - Report Server >> Geração/Emissão 
- Obrigações
Acessórias>> ICMS>> Diferencial de Alíquota p/ DataMart>>
Documento (resumido) - Report Server  
  

O objetivo
dessa MFS é fazer o tratamento para período SEM MOVIMENTO; 

Vamos aproveitar atualizar o Manual Funcional do TAXONE para inclusão dessas funcionalidades. 

MDOC: https://dev.azure.com/tr-ggo/Mastersaf%20Documentation/_workitems/edit/628906 

 

Esses relatórios não estão disponíveis no MasterSAF DW, portanto seu manual não deve ser
alterado.
