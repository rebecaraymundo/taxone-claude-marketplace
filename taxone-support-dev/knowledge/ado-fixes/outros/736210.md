# [ALCOA] TKT 119795 - TAXONE - CIAP - Modelo C Report Server : Não esta listando todos os meses como no antigo relatório Modelo C "

## Descricao

**[ALCOA] TKT119795 - TAXONE - CIAP -  Relatórios >> Emissão Livro >> Modelo C Report Server : "Parte 3 não esta listando todos os meses como no antigo relatório Modelo C "** 

 Consultora entrou em contato conosco questionando porque o novo relatório Modelo C Report Server não sai a parte 3 como era anteriormente.  Em conversa com a Liliane e Andrea fui informada que o relatório Modelo C Report Server deve ser gerado da mesma forma que era o anterior e isso não esta ocorrendo.  
 Segue uma demonstração de como era gerado anteriormente:  
 
EMISSÃO DO LIVRO PELO MENU MODELO C 

 
 
 
 
 
na parte 3 vem com todos os meses do ano: 

 
 
EMISSÃO DO LIVRO PELO MENU MODELO C – REPORT SERVER 

 
 
 
Na parte 3 do livro é demonstrado apenas o mês vigente e não
mês a mês como na opção anterior : 

 
 
 
 Ela informa que precisa  que o relatório REPORT SERVER seja gerado de forma ordenada
por data e com todos os meses do ano sendo demonstrados na parte 3.

## Root Cause

Causa: Relatório utiliza a data inicial como argumento para recuperação de somente o intervalo digitado de datas.

 Solução:  Modificado para a partir da data final e caso a data inicial o dia seja "01" a janela considerará a pesquisa desde de 01/01/ do ano indicado nada data final do período.
