# [ALCOA ] TKT106063 - Tax One - Report Fiscal - Erro no cabeçalho do Relatório ICMS Substituicao tributaria por codigo Fiscal - CFOP - UAT - ALCOA ALUMINIO S/A

## Descricao

O Relatório de ICMS Substituicao tributaria por codigo Fiscal - CFOP  não mostra o Título baseado na seleção ENTRADAS quando não há movimento para o período. 
 Solicitamos apoio para avaliar a situação reportada. 
 Att:. Douglas Fernandes

## Root Cause

Causa: INdicador de relatório de Entrada ou Saída não funcionando quando não há registros a serem recuperados no relatório.

 Solução: Utilização do argumento para definição do relatório mesmo quando não houver registros a serem recuperados.
