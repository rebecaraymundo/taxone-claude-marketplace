# TKT139660 - SYNGENTA - ONESOURCE TAX ONE - BÁSICOS - Importação safx154

## Descricao

Cliente informa que ao importar a safx154, a maioria dos registros estão sendo desconsiderados, por conta do campo NUM_ITEM, onde está ocorrendo apenas a importação da primeira linha de cada insumo, as demais linhas são ignoradas e isso não ocorria antes.

## Root Cause

Levantamento incorreto dos campos chaves para ser informado no campo IND_VALID_OBI para as SAFXs.
