# [EQUINOR] TKT104841 - TAXONE - DW - Controle de Tributos - FALHA FECHAMENTO TELA

## Descricao

**TKT 104841 - Tax One - Básicos - FALHA FECHAMENTO PERÍODO MÓDULO FERRAMENTAS PARA SAFX53 - UAT - EQUINOR BRASIL ENERGIA LTDA.** 

 Federal> Controle de Obrigações Federeais > Atualização> Status das Obrigações> Por Estabelecimento
 
 
 Cliente relata falha ao tentar realizar o fechamento de período para a SAFX53. 
Informa que o processo permite a edição das informações em tela mesmo após fechamento ter sido realizado e isso trás insegurança e desconfiança na confirmação das informações.

Já havia sido aberta uma Demanda (665320) para entendimento do processo, mas devido as queixas da cliente estamos abrindo esta outra demanda solicitando N3 Apoio para validação das evidências onde são apresentadas as percepções de falha no processo. 
 Att:. Douglas Fernandes.
