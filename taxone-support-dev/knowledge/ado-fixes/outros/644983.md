# [Roadmap] Implementação de testes que atendam a importação/carga de tabelas acessórias

## Descricao

US criada para estudo/implementação de testes que atendam carga/importação que dependem de arquivos externos, atualmente carregados via filezilla. Não há serviços para subir esses arquivos.

   Nesse carregamento feito pelo Filezilla o usuário pode utilizar seu própio usuário ou então usar o usuário integrador fornecido e então colocar o arquivo na pasta, depois disso o arquivo passa a aparecer para selecionar na tela do Tax One. 

Credenciais do usuário integrador para utilizar no Filezilla e estabelecer a conexão com o T1 para importar o arquivo desejado:

Credenciais de um usuário próprio (base de automação TAXONE OCLOUD3 (WAN)):

  Analisar a viabilidade de um teste na importação das tabelas acessórias possivel de ser replicável sem que aconteça erros na regressão automatizada, visto que o diretório onde o arquivo alocado pelo Filezilla é limpo periódicamente.

  Em anexo, há um dos arquivos externos que deverá ser utilizado para executar a importação/carga desse cenário de teste.
