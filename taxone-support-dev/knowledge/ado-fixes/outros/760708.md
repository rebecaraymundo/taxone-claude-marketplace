# [GRUPO CASAS BAHIA S.A.] TKT105372 - MELHORIAS - VIA VAREJO - SPED - EFD -  Registro 0221 c/ Parametrização Centralizada por Estabelecimento na SAFX278 (QA)

## Descricao

Ramo: MFS663925 
 Objetivo da Demanda: Permitir que a Parametrização por Produto p/ registro 0221 (SAFX278), de um estabelecimento, possa ser utilizada para outros estabelecimentos. O problema relatado pela Via Varejo é a alta volumetria na  tabela, caso ela tenha que carregar todos os seus estabeledimentos (1.500 de estabelecimentos). A quantidade de produtos para gerar o 0221 para um estabelecimento do DF é 1.500.000 produtos. Se todos esses produtos tiverem que ser parametrizados para os 1.500 estabelecimentos, a volumetria da tabela ficaria em 2.2 bilhões.  
 Hoje apenas o DF está exigindo a entrega desse registro.  Reunião realizada junto a Via Varejo 19/07/2024, a proposta foi: 
- A Via Varejo carregar todos os estabelecimentos do DF para gerar o 0221, já que essa UF é a única que está rejeitando os arquivos sem o registro 0221. 
- Acompanharmos a adesão das outras UFs. 
- Entregarmos a melhoria no produto em janeiro/2025 onde a parametrização de um estabelecimento poderá ser considerada para a geração do 0221 de outros estabelecimentos.   
 Ticket: https://atendimentotr.zendesk.com/agent/tickets/101311
 Escopo da Solução:  
- Criar um parâmetro na tela de Dados Iniciais do Sped, onde o cliente informa se deseja utilizar a parametrização do Produto p/ 0221 de outro estabelecimento. E selecione qual é o estabelecimento detentor da parametrização;  
- Ou criar uma tela nova de Centralização de Estabeleicmentos, como a de Inscrição Estadual Única, conforme print abaixo. 
- Alterar a Geração do Sped, para o registro 0221 para utilizar a parametrização do produto do estabelecimento mencionado em Dados Iniciais, caso o novo parÂmetro esteja flegado. 
- Ou se for criada a tela de Centralização dos Estabelecimentos, utilizar a parametrização do Centralizador, caso o estabelecimento um Centralizado. Senão, continuar utlizando sua própria parametrização.

## Release Notes

Carta do Patch – DW e TAXONE: 

  

 Resumo da Alteração: Inclusão de um novo parâmetro na tela de Dados Iniciais para permitir que a Parametrização por Produto do registro 0221 de um estabelecimento, seja utilizada para outros estabelecimentos. 

Solução Funcional:

- Inclusão do parâmetro "Utilizar a Parametrização por Produto do Estabelecimento" que deve ser preenchido com o estabelecimento detentor da Parametrização por Produto para geração do registro 0221. Campo não é obrigatório. 

  Menu Estadual, Módulo: EFD – Escrituração Fiscal Digital
           Parâmetros à Dados Iniciais

- Alteração de regra de geração dos registro 0221, para contemplar o novo parâmetro "Utilizar a Parametrização por Produto do Estabelecimento". Se estiver preenchido, considerar a Parametrização por Produto do estabelecimento informado no novo parâmetro. Senão, considerar a Parametrização por Produto do estabelecimento foco da geração.
