# [INX] TKT137242 -TAX ONE - MUNICIPAL - CABREUVA - DES XML - tag <ISSDevido></ISSDevido> está gerando sem valor.

## Descricao

Ao realizar a geração do arquivo Magnético do ISS – CABREUVA o arquivo não gera a informações da tag <ISSDevido>0,00/ISSDevido>, ocorrendo erro no validador. 

 

ISSDevido será o valor do ISS pago pelo prestador 

 

 

Na documentação de regra não consta informação nessa tag para Cabreúva. 

 

 

 

Após teste preenchendo o valor manual no arquivo, submetemos no validador e foi importado com sucesso.

## Root Cause

Foi criada nova regra de negócio para preenchimento da TAG ISSDevido para o município de Cabreúva.
