# [DELOITTE] TKT108689 - DW - PARÂMETROS PARA MUNICIPIO - REALIZAR A ATUALIZAÇÃO DOS CÓDIGOS DESIG (TACES102)

## Descricao

Foi realizar uma carga da safx2116 (referente a DESIF), porém ao importar apresentou o erro :
Código de tributação DESIF invalido, pois não pertence a tabela de código de tributação DESIF - Anexo 6 ( em anexo print das telas com o código que precisa ser). 

Hoje na TACES102 - não existe o código 17.11.01-000, mas na tela de entrega da desif esse códgio está liberado e precisa ser utilizado pela a Pluxee.
