# [MAGALU] TKT 111196 - DW - ICMS > Data mart > Apuração do ICMS " erro no livro de Apuração de ICMS P9 -  ORA - 12899 : value too large for column "MASTERSAF", "ICA_PARAM_GER_APUR", "NUM_VERSAO" (actual: 4, maximum: 3)"

## Descricao

Ao tentar emitir o livro de apuração de ICMS P9 para empresas do  RS.   ORA - 12899 : valor muito grande para a coluna
 "MASTERSAF", "ICA_PARAM_GER_APUR", "NUM_VERSAO" (real: 4, máximo: 3)
 
 O erro ocorre tanto do caminho :  ICMS > Data mart > Apuração do ICMS > Ajuste SINIEF
 
 
 Quanto no caminho :  ICMS > Data mart > Apuração do ICMS > Emissão (após fazer a geração)  
 
 
 Cliente esta na versão 297.1.1 informou não poder atualizar.

## Release Notes

Carta do
Patch - DW: 

Módulo: Estadual
> ICMS 

Menu: DATA
MART > Apuração do ICMS > Ajuste SINIEF 

Menu: DATA
MART > Apuração do ICMS > Emissão 
      Ajuste na geração
e na emissão do livro de apuração, para corrigir o erro de gravação no banco.
