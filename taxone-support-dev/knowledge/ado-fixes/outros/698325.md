# [AGRO ASSAI] TKT109909 - TAXONE - ESTADUAL - RESSARCIMENTO ICMS ST SP - O cálculo do Valor de Confronto não foi realizado para este item de devolução.

## Descricao

Cliente: 
 Estou com
problema no cálculo do valor de confronto. O caso agora é a falta dos valores de confronto
para NF de devolução 143482. A NF referenciada 143222 foi emitida no mesmo
período e, por acaso, é o primeiro movimento do mês para os dois itens em
questão. Já tentei fazer a equalização usando diferentes períodos. 
 
 
  
Suporte 

  

A cliente
relatou que, no mês 04/2024, houve uma nota fiscal de saída seguida por uma
devolução que anulou a operação. A nota fiscal de entrada foi registrada no mês
anterior. Orientamos a cliente a preencher o campo 107 da SAFX08 (Valor da
Carga Tributária Retida) no documento de saída. Durante os testes, verificamos
que o valor da devolução é apresentado na ficha 3. No entanto, a cliente não
concorda com o preenchimento deste campo, alegando que o problema não está
relacionado a isso, mas sim à equalização. 

  

Dado o
histórico de tickets abertos sobre essa questão, solicitamos um apoio para
resolver a situação. 

 

 Regra utilizada: 
 
 
 Orientação prevista no manual de layout da CAT42 
 
 
 Double check: Vanderleia Simulação Aba analysis
