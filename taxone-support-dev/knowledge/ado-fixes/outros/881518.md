# [MRV] TKT146329 - TAX ONE - DDS NATAL - Menu “Arquivo de Entradas de Serviços (Const. Civil/Utilities/Telecom)” não existe para o município de Natal

## Descricao

TKT146329 - MRV - TAX ONE - MUNICIPAL - DDS NATAL  - Menu “Arquivo de Entradas de Serviços (Const. Civil/Utilities/Telecom)” não existe para o município de Natal 

O estabelecimento gerador é do município de Parnamirim, porém possui Inscrição Municipal no município de Natal, neste caso, precisa gerar o arquivo dentro do módulo de Natal, pelo menu “Arquivo de Entradas de Serviços (Const. Civil/Utilities/Telecom)”, porém não tem este menu no módulo de Natal: 

 

 

 

Precisa ser incluído o menu, para possibilitar a geração de um estabelecimento de outro município, mas que tem a IM para o município de Natal.
