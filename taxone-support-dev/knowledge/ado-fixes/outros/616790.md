# [GE POWER ] TKT92137 - TAX ONE - Básicos - Parametrização de relatório solicitando campo que não existe

## Descricao

Relatório está solicitando a seleção de estabelecimento, porém na parametrização do mesmo não possui o campo.

Basicos > Report Fiscal > Gerenciais > Contabil > Geral > Razão (Exportação de Dados)

## Root Cause

Causa: Query utilizando a função lib_global.get_cod_usuario() que retorna nulo ao invés do usuário.

 Solução:  Troca da query de carga de estabelecimento.
 
 Ações preventivas/Boas práticas:
 N/A
