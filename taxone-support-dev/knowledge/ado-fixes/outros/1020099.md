## Root Cause

O problema ocorre porque o sistema não está configurado corretamente para processar o ano calendário 2025 (ano-base 2024). As parametrizações necessárias para este período não foram registradas no banco de dados, fazendo com que o módulo Federal/OTF não encontre as configurações adequadas ao tentar gerar os arquivos de Informe de Rendimento. Adicionalmente, há falhas no tratamento de arquivos que podem impedir a criação dos diretórios de saída e a cópia dos arquivos gerados para seus destinos corretos.
