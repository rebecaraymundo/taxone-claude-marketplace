# [APPLE] TKT119931 - TAXONE - EFD REINF - Erro de tarja vermelha ao gerar o relatório

## Descricao

T119931 - APPLE COMPUTER BRASIL LTDA - ONESOURCE TAX ONE - SPED - EFD - REINF - Relatórios > Relatório de Detalhes do Painel  Erro de tarja vermelha ao gerar o relatório  - UAT** 

 

Estamos em projeto de Migração do Mastersaf para o Taxone. 

Estamos na fase de testes dos usuários. 

 

Ao tentar gerar o relatório de detalhes do painel ocorre uma mensagem de erro na tela (erro inesperado) e o relatório não é gerado. Tentamos gerar em duas máquinas e algumas vezes, mas o erro persiste e o relatório não é gerado. 

 

Anexo erro.

## Root Cause

Causa: Devido a leitura errada do SQL dentro da datawindow do relatório, a conversão do um campo date para string  TO_CHAR(:PERIODO, 'MM/YYYY') apresenta erro no tennat da Apple.

 Solução: Alteração do SQL do campo Período para  TO_CHAR(TO_DATE(:PERIODO, 'DD/MM/YYYY'), 'MM/YYYY').
