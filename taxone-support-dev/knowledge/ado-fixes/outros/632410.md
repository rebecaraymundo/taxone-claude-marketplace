# Servidor de Relatórios - TAXONE - Básicos - Report Fiscal - Relatório Documentos Fiscais > Analiticos > Mercadoria   (SEM MOVIMENTO) (QA)

## Descricao

Menu: Gerenciais >> Documentos Fiscais >> Analíticos >> Mercadoria
 Aplicar tratamento do período sem movimento para o relatório. Tratamento para Período sem Movimento:
 
- Arquivo PDF incluir texto “SEM MOVIMENTO” no
cabeçalho. Cabeçalho deve conter os dados de estabelecimento, período como
ocorre para um período sem movimento. 

- Arquivo EXCEL incluir a coluna
“PERIODO_COM_MOVIMENTO” e preencher com “NÃO” para períodos sem movimento. 

 

Fazer o Manual Operacional para essa funcionalidade.

## Release Notes

Carta do Patch – TAXONE APENAS: 

  

Módulo: Report Fiscal 

  

Resumo da alteração: Melhoria no tratamento dos relatórios analíticos de mercadoria via Report server quando período não tem movimento. 

Módulo Report Fiscal
 

 

 

Menu: 

         

   Gerenciais >> Documentos Fiscais >> Analíticos >> Mercadoria >> Geração / Emissão 

 

No caso de não existir movimento para o estabelecimento e período apurado, os relatórios emitidos em PDF apresentaram o texto "SEM MOVIMENTO" no cabeçalho. Já os emitidos em Excel, a primeira coluna "PERIODO_COM_MOVIMENTO" será preenchida com "não".
