# TKT165893 - COPA ENERGIA - ONESOURCE TAX ONE - SPED - REGISTRO E115 - Registro E115 especifico para RS consta uma linha do CFOP 6921(no caso de específico para RS) e situação com código de benefício não movimentado no período

## Descricao

Análise de geração de registro E115 com CFOP 6921 zerado – RS 

 

Descrição da Demanda: 

Identificada a geração de registro no arquivo E115, específico para o estado do Rio Grande do Sul, contendo o CFOP 6921 com valor zerado (|E115|RS052999|0|6921|). 

 

Verificado que não houve movimentação para esse CFOP no período, não existindo nenhuma Nota Fiscal com o código de benefício destacado na SAFX08, campo 255. 

 

Código de benefício RS052999  

Solicita-se: 

- Análise da causa da geração do registro E115 com CFOP 6921 e valor zerado. 
- Ajuste no processo de geração, se necessário, para evitar inclusão de registros sem movimentação.  

 

Evidência anexada: 

|E115|RS052999|0|6921| 

 

Observações adicionais: 

Não há movimentação para o CFOP 6921 no período analisado.
