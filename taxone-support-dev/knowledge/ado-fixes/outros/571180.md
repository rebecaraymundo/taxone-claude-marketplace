Solicitação do Cliente Magalu: 

 
"Necessidade: inclusão de campos para análise dos valores de: confronto, a ressarcir e completo.
        Natureza de operação
         Valor Ressarcimento
         Valor Complemento
 Necessitamos ter um relatório no módulo MVA PR similar ao que temos na CAT 42/2018, ficha 3 - Controle de Estoque"
 
 Embora o cálculo do módulo do Ressarcimento PR seja diferente do modelo da CAT42, o objetivo do cliente é possuir uma rastreabilidade das notas fiscais  consideradas para o cálculo e sua composição. 
 O PM deu GO para essa solicitação.  Será interessante fazer uma conversa com a cliente para avaliar as informações que eles precisam.
