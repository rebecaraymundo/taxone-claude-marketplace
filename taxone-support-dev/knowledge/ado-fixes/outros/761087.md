# [HAPVIDA] TKT121311 - DW - EFD CONTRIBUIÇÕES FINANCEIRA/ASSEMELHADA - O campo 4 (COD_TIPO_CON) do registro 0110.

## Descricao

​​**[HAPVIDA] TKT121311 - DW - EFD CONTRIBUIÇÕES FINANCEIRA/ASSEMELHADA - O campo 4 (COD_TIPO_CON) do registro 0110 no SPED Contribuições Financeira/Assemelhada está gerando um valor fixo de 1.** 

 

Situação: O campo 04 (COD_TIPO_CON) do registro 0110 está gerando um valor fixo de 1, que corresponde à opção de alíquotas básicas. No entanto, o cliente precisa que seja entregue com a opção 2, que se refere a alíquotas diferenciadas. 

Após verificar a regra, constatei que o campo está fixo em 1 conforme o layout atual. Contudo, a base tributária  confirmou  que permite a entrega com a opção 2. Portanto, solicito a revisão da regra aplicada a este campo.
