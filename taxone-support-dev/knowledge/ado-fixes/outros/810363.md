# [GSK] TKT127091 - DW - SPED - EFD - Parametrização Registro E115 Geral RJ - não salva em alguma situação - clone not a bug indevido

## Descricao

TKT127091 - GSK - MASTERSAF DW - SPED - EFD - ESCRITURAÇÃO FISCAL DIGITAL - Parametrização Registro E115 Geral RJ - não salva em alguma situação
 
Ao parametrizar o Registro E115 Geral - RJ, a rotina apresenta um comportamento inesperado: ela salva apenas o último produto inserido, mesmo quando se tratam de produtos distintos, desde que possuam a mesma data de validade. Esse comportamento foi confirmado através de uma simulação realizada.
