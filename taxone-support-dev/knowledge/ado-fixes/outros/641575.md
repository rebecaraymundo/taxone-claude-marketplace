# [SUCAFINA BRA] TKT99581 - TAXONE - VARGINHA - Erro geração Notas Fiscais Emitidas

## Descricao

**TKT99581 - TAXONE - VARGINHA ( Ao gerar o arquivo de Varginha ocorre o erro no log de geração :Erro no tratamento de dados das Notas Fiscais Emitidas--ORA-06502: PL/SQL: numeric or value error: character string buffer too small)** 
Ao gerar o Arquivo para o município de Varginha ocorre o erro no log de Geração : " ="Erro no tratamento de dados das Notas Fiscais Emitidas (MUN_BETHA_GERA_TOMAD.setDadosItem).
			Chave: 001-2002-01/12/2023-1831-000083662//. ORA-06502: PL/SQL: numeric or value error: character string buffer too small" Porém ao realizar o mesmo teste em nosso ambiente UAT com os mesmos dados da ciente o erro não ocorre.  O erro ocorre para a cliente em qualquer período que ela gera.

## Release Notes

Informações para a Carta do Patch: DW e TAX ONE 

Agrupamento: Municipal 

Módulo: Varginha - MG   
Objetivo: Readequação do tamanho da variável TIPO_NOTA, visando comportar o tamanho máximo possível de preenchimento, passando de uma para dez posições.
