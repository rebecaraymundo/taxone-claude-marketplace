# TKT174091 - TEX COURIER -  ONESOURCE TAX ONE -  ICMS > Apuração > Lançamentos Complementares > Código de Ajuste Sped fiscal  "Ao cadastrar código de Ajuste ocorre o erro: Código de Ajuste deve ser preenchido" - Copy

## Descricao

Anteriormente foi aberta uma demanda sobre o assunto e foi encerrado com Not a Bug - 789004 -- 

 

Consultor entrou em contato informando que não consegue cadastrar código de ajuste na tela ICMS > Apuração > Lançamentos Complementares > Código de Ajuste Sped fiscal porque ocorre a mensagem : Código de Ajuste deve ser preenchido, conforme orientação da demanda anterior informamos que o posicionamento de nossa equipe técnica sobre o formato de cadastro na tela em questão seria após preencher cada campo, pressione a tecla "TAB" ou clique fora da área de preenchimento para que o sistema registre a informação. Mas ela alega que mesmo fazendo isso o problema persiste, em nosso ambiente não ocorre isso. 

 

Irei colocar a gravação da tela em anexo a demanda. 

 

Em conversa na reunião de N3 com o Daniel, a janela é JAVA e ela precisa de um tratamento unitário de pesquisa e gravação unitário.  

Com isto fui orientada a abrir esta demanda.

## Root Cause

O erro **'Código de Ajuste deve ser preenchido'** ocorre porque a tela de seleção (zoom) utilizada para buscar e vincular um Código de Ajuste SPED Fiscal não está retornando corretamente o valor escolhido pelo usuário para o campo correspondente. Com isso, o sistema interpreta o campo como vazio e bloqueia o salvamento do registro, mesmo quando o usuário já selecionou um código válido. O problema está na forma como a janela de pesquisa repassa a informação selecionada para o formulário principal de Lançamentos Complementares.
