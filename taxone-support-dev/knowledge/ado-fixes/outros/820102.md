# [SOUZACRUZ] T1 PROD02 AMERS- PKG_INI_TAB_FPROC

## Descricao

Lentidão PKG_INI_TAB_FPROC executando MAIS de 5hs cliente [USER_TAXONE_SOUZACRUZ] Taxone PROD02 AMERS: 

 
A query ofensora é:   SELECT ROWID FROM SAFX113 WHERE RTRIM(COD_EMPRESA) = :"SYS_B_0" AND RTRIM(COD_ESTAB)   = NVL(:"SYS_B_1", RTRIM(COD_ESTAB)) 
 
 
-----------------------------------------------------------------------------

| Id  | Operation         | Name    | Rows  | Bytes | Cost (%CPU)| Time     |

-----------------------------------------------------------------------------

|   0 | SELECT STATEMENT  |         |    53 |  1113 |  2874   (5)| 00:00:01 |

|*  1 |  TABLE ACCESS FULL| SAFX113 |    53 |  1113 |  2874   (5)| 00:00:01 |

-----------------------------------------------------------------------------
 

The predicate RTRIM("SAFX113"."COD_EMPRESA")=:B1 used at line ID 1 of the   execution plan contains an expression on indexed column "COD_EMPRESA". This   expression prevents the optimizer from selecting indices on table   "USER_TAXONE_SOUZACRUZ"."SAFX113". 

  
 Ash report em anexo.

## Root Cause

Causa: Problemas de performance indicados por Techops
 
 Solução: Retirada dos comandos  RTRIM de empresa e estab para melhoria de performance da query.
