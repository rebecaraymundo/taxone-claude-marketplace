# TKT128432 - COMPANHIA BRASILEIRA DE METALURGIA E MINERACAO - TAX ONE - Básicos - Relatório Cód. Fiscal de Operação - Report Server não está trazendo notas com CFOP 1933 e 2933.

## Descricao

Mesmo após realizar as parametrizações solicitadas na demanda 794754, o cliente informa que o problema ainda persiste na geração do relatório, não trazendo os CFOPs parametrizados.

 Report Fiscal no caminho “Obrigações Acessórias > ICMS > Resumos do ICMS p/ DataMart. > Cód. Fiscal de Operação - Report Server”

## Root Cause

Causa: Recuperação do estabelecimento selecionado estava depois do select que busca o preenchimento do parâmetro 64.

 Solução: Colocar a recuperação do parâmetro de estabelecimento antes do select de recuperaçào do parâmetro 64 que determina a recuperação ou não das notas de serviço.
