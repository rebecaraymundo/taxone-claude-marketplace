# [COMPANHIA BRASILEIRA DE METALURGIA] TKT105716 - DW - EFD FISCAL - Registro E113

## Descricao

**TKT 105716 - DW - SPED - SPED FISCAL - Registro E113 não é gerado para as notas associadas manualmente o número do item de vinculação fica no formato número com casas decimais** 

 "Estamos com problema na geração da EFD ICMS/IPI - 062024 onde o registro E113 dos itens vinculados nos códigos de ajustes da apuração de ICMS não estão sendo informados após geração da EFD.

## Root Cause

Causa: Campo com o número de discriminação com um espaço a esquerda.

 Solução: Ajuste no campo para excluir espaços a esquerda.
 
 Ações preventivas/Boas práticas:
 N/A

## Release Notes

Módulo: ESTADUAL ICMS

 Resumo da Alteração: Ajuste no valor de número de discriminação dos documentos vinculados manualmente (para registros novos).
 
 Descritivo das Alterações: Ajuste no valor de número de discriminação dos documentos vinculados manualmente (para registros novos).
 
 Funcionalidade Alterada: ESTADUAL ICMS > APURAÇÃO > APURAÇÃO DO ICMS > LANÇAMENTOS COMPLEMENTARES > APURAÇÃO DO ICMS > AJUSTE SINIEF -  (DOCUMENTOS FISCAIS VINCULADOS)
 
 Módulos Impactados: ESTADUAL ICMS

OBS: Favor informar aos clientes que este ajuste somente corrige novos registros. Para registros já gravados, será necessário excluir-los e inseri-los novamente para que a janela corrija o problema.
