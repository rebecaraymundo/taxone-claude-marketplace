# [WILPORT] TKT122704 - TAXONE - Municipal – Parâmetros por Município – Parâmetros – Serviços Msaf x Serviços - Inclusão 17.01

## Descricao

**[WILPORT] TKT122704 - TAXONE - Municipal – Parâmetros por Município – Parâmetros – Serviços Msaf x Serviços'  "Não tem o código  de serviço 17.01 para cadastro"** 

 

 

Ao realizar a parametrização no módulo "Parâmetros para Município > Serviços Msaf x Serviços", constatei que o código de serviço 17.01, que é necessário, não estava disponível para parametrização, nem no sistema Tax One nem no DW. 

Código do Serviço Prestado: Este campo deve ser preenchido com o código de serviço apropriado, que pode ser encontrado na tela 'Municipal – Parâmetros por Município – Parâmetros – Serviços Msaf x Serviços', correspondente ao serviço indicado na nota fiscal. O código deve ser inserido sem separadores (pontos) e alinhado à esquerda. Os espaços restantes devem ser preenchidos com espaços em branco. Este procedimento é necessário apenas se o prestador do serviço for de fora do município em questão; caso contrário, o campo deve ser totalmente preenchido com espaços em branco. Preencha o campo com espaços à direita até completar o tamanho total especificado. 

Adicionalmente, o cliente nos forneceu uma lista de serviços de São Luís na qual consta o código 1701, indicando a necessidade de inclusão deste código no sistema para futuras parametrizações.

## Root Cause

Causa: Causa: Ajuste de Regra Inclusão de novos itens TFIX85.txt 
 Solução:  inclusão dr itens no arquivo ex: "133	17.01	Assessoria ou consultoria de qualquer natureza, nao contida em outros itens desta lista; Analise, exame, pesquisa, coleta, compilacao, e fornecimento de dados e informacoes de qualquer natureza, inclusive cadastro e similares. " entre outros.
