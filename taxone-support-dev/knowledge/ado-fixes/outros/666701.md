# Servidor de Relatórios - TAXONE - Ressarcimento ICMS-ST - SP - Aplicar tratamento SEM MOVIMENTO no Report Server  (QA)

## Descricao

Tratamento do período SEM MOVIMENTO p/ relatório da Ficha 3 migrado para o Report Server  Localização: Módulo: Estadual >> Ressarcimento ICMS-ST SP   Menu: Relatórios >> Ficha 3 - Controle de Estoque   Ramo: MFS666701 Descrição: Aplicar tratamento do período sem movimento para o relatório.  Tratamento para Período sem Movimento:
 
- Arquivo PDF incluir texto “SEM MOVIMENTO” no
cabeçalho. Cabeçalho deve conter os dados de estabelecimento, período como
ocorre para um período sem movimento. 

- Arquivo EXCEL incluir a coluna
“PERIODO_COM_MOVIMENTO” e preencher com “NÃO” para períodos sem movimento. 

 ***** A coluna “PERIODO_COM_MOVIMENTO” deve ser incluída ao final do relatório!! *******

## Release Notes

Carta do Patch – TAXONE APENAS: 

  

Módulo: Ressarcimento ICMS-ST SP 

  

Resumo da alteração: Melhoria no relatório da Ficha 3 migrado para o Servidor de Relatórios para tratamento em períodos sem movimento. 

 

Módulo Estadual >> Ressarcimento ICMS-ST SP 
Menu: Relatórios >> Ficha 3 - Controle de Estoque >> Emissão  
No caso de não existir movimento para o estabelecimento e período apurado, os relatórios emitidos em formato PDF, apresentarão o texto "SEM MOVIMENTO" no cabeçalho. Quando emitidos em formato Excel, a última coluna denominada "PERIODO_COM_MOVIMENTO" apresentará a informação "não".
