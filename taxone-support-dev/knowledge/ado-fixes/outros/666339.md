Tratamento do período SEM MOVIMENTO p/ relatório de Divergência de Valores entre Capa x Item - Doc. Fiscal  Localização: Módulo: Básicos >> Data Warehouse  Menu: Relatórios >> Conferências >> Divergência de Valores entre Capa x Item - Doc. Fiscal  >> Geração / Emissão Ramo: MFS666344 Descrição: Aplicar tratamento do período sem movimento para o relatório.  Tratamento para Período sem Movimento:

cabeçalho. Cabeçalho deve conter os dados de estabelecimento, período como
ocorre para um período sem movimento.

“PERIODO_COM_MOVIMENTO” e preencher com “NÃO” para períodos sem movimento. 

 ***** A coluna “PERIODO_COM_MOVIMENTO” deve ser incluída ao final do relatório!! *******
