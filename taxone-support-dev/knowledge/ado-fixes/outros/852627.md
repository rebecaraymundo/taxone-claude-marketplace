# [MRV] TKT 141653 - TAX ONE - MUNICIPAL - SERRA, TERESINA, SÃO LUÍS "Correção de Mensagem no Log de Geração: De 'Modelo Msaf x Série' para 'Modelo Msaf x Modelo' visto que após Parametrização do Modelo a mensagem some"

## Descricao

Situação ocorre para SERRA, TERESINA, SÃO LUÍS, conforme conversado com Rogério Ohashi vinculo a mesma demanda por se tratar da mesma situação. 

O registro de log para a geração esta exibindo a mensagem "Modelo Msaf x Série" de forma equivocada. 

Esta mensagem deveria especificar "Modelo Msaf x Modelo", uma vez que se refere à interação entre modelos , pois a serie esta parametrizada e após a parametrização do modelo a mensagem some do log. 

A mensagem inicial acaba causando confusão quanto ao processo que esta sendo realizado. 

 

Após a correta parametrização do modelo, a mensagem no log some. 

 Isso indica que o modelo está corretamente configurado e que a interação esperada entre modelos está sendo efetivamente registrada. 

Esse ajuste é crucial para a clareza e precisão dos registros de log, facilitando o monitoramento e a análise dos processos por parte dos usuários e mantenedores do sistema.
