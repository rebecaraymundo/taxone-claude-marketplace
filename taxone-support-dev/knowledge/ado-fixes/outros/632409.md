# Servidor de Relatórios - TAXONE - Básicos - Report Fiscal - Relatório Diferencial de Alíquota p/ DataMart>> Códgio Fiscal de Operação - CFOP  (SEM MOVIMENTO) (QA)

## Descricao

Menu: Obrigações Acessórias>> ICMS>> Diferencial de Alíquota p/ DataMart>> Códgio Fiscal de Operação - CFOP - Report Server
 Aplicar tratamento do período sem movimento para o relatório. Tratamento para Período sem Movimento:
 
- Arquivo PDF incluir texto “SEM MOVIMENTO” no
cabeçalho. Cabeçalho deve conter os dados de estabelecimento, período como
ocorre para um período sem movimento. 

- Arquivo EXCEL incluir a coluna
“PERIODO_COM_MOVIMENTO” e preencher com “NÃO” para períodos sem movimento. 

 ***** A coluna “PERIODO_COM_MOVIMENTO” deve ser incluída ao final do relatório!! *******

## Release Notes

Carta do Patch – TAXONE APENAS: 

  

Módulo: ICMS 

  

Resumo da alteração: Melhoria no tratamento dos relatórios de Diferencial de Alíquota p/ DataMart - Report server quando período não tem movimento. 

 

Módulo ICMS 

 

Menu: 

         

   Obrigações Acessórias>> ICMS>> Diferencial de Alíquota p/ DataMart>> Códgio Fiscal de Operação - CFOP - Report Server 

 

No caso de não existir movimento para o estabelecimento e período apurado, os relatórios emitidos em PDF apresentaram o texto "SEM MOVIMENTO" no cabeçalho. Já os emitidos em Excel, a primeira coluna "PERIODO_COM_MOVIMENTO" será preenchida com "não".
