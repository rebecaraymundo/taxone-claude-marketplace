# [SUMITOMO] TKT122917 - TAXONE - REPORT FISCAL - Conferência de Documentos Fiscais (ICMS/IPI/PIS/COFINS) - Report Server

## Descricao

**[SUMITOMO] TKT122917 - TAXONE - REPORT FISCAL - Erro ao gerar relatório Conferência de Documentos Fiscais (ICMS/IPI/PIS/COFINS) - Report Server** 
Ao realizar a geração do relatório Conferência de Documentos Fiscais (ICMS/IPI/PIS/COFINS), o log do mesmo apresenta "Erro na atualizacao - TMP_CONF_DOC_FIS_RS: ORA-12899: value too large for column USER_TAXONE_SUMITOMORUBBER.TMP_CONF_DOC_FIS_RS.NUM_AUTENTIC_NFE (actual: 46, maximum: 44)." 
Foi realizado a conferência do campo informado NUM_AUTENTIC_NFE, porém nenhum dos dados passa de 44 caracteres.

## Release Notes

Carta para o
patch: 

Essa alteração tem
como objetivo aumentou o tamanho do campo NUM_AUTENTIC_NFE para 80 posições. 

  

Módulo Impactado:
Report Fiscal 

Menu: Gerenciais > Documentos Fiscais > Analíticos > Conferência de Documentos Fiscais (ICMS/IPI/PIS/COFINS) - Report Server > Emissão
