# [MRV] TKT149803 - TAX ONE - MUNICIPAL - SERRA - ISS.WEB - Avaliar a regra para gerar o campo " Regime Simples" para o Município de Serra - ES

## Descricao

Solicitação de avaliação da regra de negócio referente à geração do arquivo do município de Serra. Atualmente, o campo "Regime do Simples" não está sendo preenchido. No entanto, conforme a necessidade do cliente, esse campo deve receber um dos seguintes valores: 

- (N) Não Simples 
- (S) SINAC 
- (M) SIMEI (aplicável apenas para declarações de serviços tomados) 
- 
  
Solicita-se análise e ajuste da regra de negócio para garantir o correto preenchimento do campo conforme as opções acima. 

 

Na regra esse campo é gerado apenas para outros municípios: 

 

 

 

Arquivo do cliente: 

 

 

 

Manual de layout anexo.

## Root Cause

Alteração de regra de negócio.

## Solucao

SaaS

## Release Notes

Carta do
Patch - DW e TAX ONE: 

   Módulo: Municipal > Serra 

 
 Menu: Obrigações > Meio Magnético 

 
   Nesta demanda será liberado alteração na regra de geração do meio
magnético campo “Regime Simples”, conforme orientação do Manual de Layout da
Prefeitura de Serra, para o validador ISS WEB.
