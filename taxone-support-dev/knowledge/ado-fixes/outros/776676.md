# TKT127018 - COSAN RAIZEN - MASTERSAF DW - REPORT FISCAL - GERENCIAIS  - CONTABIL  -  Geral  - Balancete Relatório Balancete esta gerando contas duplicadas.

## Descricao

Situação do Cliente: 

 

O cliente está enfrentando um problema onde a conta contábil está cadastrada duas vezes, cada uma com validades diferentes. Isso resulta na duplicação da conta quando é gerado o relatório de balancete. O comportamento esperado é que o sistema utilize o último "IDENT CONTA". 

Mesmo ao inativar uma das contas contábeis, o relatório continua gerando entradas para ambas as validades.
