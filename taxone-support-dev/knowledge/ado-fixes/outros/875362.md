# [MRV] TKT145204 - TAX ONE - DES XML - Menu “Arquivo de Entradas de Serviços (Const. Civil/Utilities/Telecom)” não existe para o município de Limeira

## Descricao

TKT145204 - MRV - TAX ONE - MUNICIPAL - DES XML - Menu “Arquivo de Entradas de Serviços (Const. Civil/Utilities/Telecom)” não existe para o município de Limeira 

 

O estabelecimento gerador é do município de Campinas, porém possui Inscrição Municipal no município de Limeira, neste caso, precisa gerar o arquivo dentro do módulo de Limeira, pelo menu “Arquivo de Entradas de Serviços (Const. Civil/Utilities/Telecom)”, porém não tem este menu no módulo de Limeira: 

 

 

 

 

Precisa ser incluído o menu, para possibilitar a geração de um estabelecimento de outro município, mas que tem a IM para o município de Limeira.
