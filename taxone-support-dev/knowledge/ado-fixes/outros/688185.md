# [HUGHES] TKT107938 - DW - ATIVO PERMANENTE - Movimento - Saldos a Creditar(II) 

## Descricao

**TKT 107938 - TAX ONE - Estadual>Ativo Permanente>Movimento>Saldos a Creditar(II)-Report Server>Geração:"Ao selecionar a opção Cód. Bem e Incorporador, evitar a adição de uma barra no arquivo EXCEL emitido quando o incorporador não estiver presente."** 

 
 
 Cliente relatou que ao gerar o relatório de Saldos a Creditar (II) no Report Server, ao selecionar as opções 'Cód. Bem' e 'Incorporador', ocorre a inserção de uma barra ao lado do código (ex: 000990094000002/) mesmo na ausência do incorporador. Esta questão foi discutida com Andrea Rocha, e com isso abro essa demanda para ajustar o sistema de modo que a barra não seja incluída quando o dado do incorporador não estiver disponível conforme orientado. Atualmente, o sistema insere automaticamente a barra, independentemente da presença do incorporador. Conforme acordado com Andrea, a barra será mantida apenas quando o código do incorporador estiver presente.
