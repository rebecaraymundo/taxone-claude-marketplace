# [IVG] TKT127843 - TAXONE - TAX AUTOMATION - Status das importações ficando como impedido

## Descricao

Verificamos que alguns estão ficando impedido após algumas tarefas apresentar o status PROBLEMA, analisando os logs do datadog, verificamos que existe um erro no processamento de alguns jobs, dando erro de constraint. 

 

 O erro ocorre aleatoriamente, conversando com o time técnico e analisando o código do taxone, nós do time do automation, acreditamos que o erro ocorre na concorrência do sequence, uma vez que, é disparado diversos jobs ao mesmo tempo acaba ocorrendo conflito na numeração, dando erro no processamento.
