# TKT136844 - DELOITTE (EATON LTDA) - ONESOURCE TAX ONE - BÁSICOS - JOB SERVIDOR - Relatório de LOG de Importação trazendo digito "a mais" na relação de notas

## Descricao

Cliente entrou em contato pois, no relatório de LOG de importação, encontrou um digito "a mais" (número 1) a frente do documento fiscal; 

 

 

 

 

 

Utilizamos algumas SAFX, mas não conseguimos repetir o cenário do cliente 

Ao que tudo indica, esse número '1' é o 'ident_fis_jur_w'
 

 

 

 
Errado:

  
 
 if  (num_lote_w = pnum_lote OR pnum_lote IS NULL) then
 
           PQTD_DEL := PQTD_DEL + 1;
 
           msg_erro_w := 'Registro ignorado por estar em duplicidade:'||cod_empresa_w||' - ';
 
           msg_erro_w := msg_erro_w||cod_estab_w||' - '||movto_e_s_w||' - ';
 
           msg_erro_w := msg_erro_w||norm_dev_w||' - '||cod_docto_w||' - '||ident_fis_jur_w;
 
           msg_erro_w := msg_erro_w||num_docfis_w||' - '||serie_docfis_w;
 
           saf_at_log(PNUM_PROCESSO, seq_log_det_proc.nextval, 1, msg_erro_w, null, null, 99910, null);
 
         end if;
 
  
 
 Correto:
 
  
 
 if  (num_lote_w = pnum_lote OR pnum_lote IS NULL) then
 
           PQTD_DEL := PQTD_DEL + 1;
 
           msg_erro_w := 'Registro ignorado por estar em duplicidade:'||cod_empresa_w||' - ';
 
           msg_erro_w := msg_erro_w||cod_estab_w||' - '||movto_e_s_w||' - ';
 
           msg_erro_w := msg_erro_w||norm_dev_w||' - '||cod_docto_w||' - '||ident_fis_jur_w||' - ';
 
           msg_erro_w := msg_erro_w||num_docfis_w||' - '||serie_docfis_w;
 
           saf_at_log(PNUM_PROCESSO, seq_log_det_proc.nextval, 1, msg_erro_w, null, null, 99910, null);
 
         end if;

## Root Cause

Causa: Concatenação de informações da mensagem de erro/crítica do log sen sepador entre ident_fis_jur e num_docfis.

 Solução: Colocação de ífem '-' para separar as duas informações ident_fis_jur e num_docfis.
