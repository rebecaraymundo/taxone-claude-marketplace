## Descricao

Análise do ambiente  

 
- Ambiente Cloud: Sim 
 
- Ambiente que
     acontece o problema: UAT 
 
- Link do
     ambiente/cenário: Não 
 
- Acesso ao
     ambiente: Não, havendo necessidade de acesso é possível simular no
     ambiente do cliente.  
  

Cliente informa que ao
gerar as aquisições a partir das NFs está ocorrendo erro no tamanho do campo
compl_descr_bem da tabela APT_Aquisicao igual SAFX08 campo 21-Tamanho 255.
Necessitamos dessa alteração, pois as UF AM e SC exigem a descrição do XML. 

  

Erro na geração das
aquisições a partir das NFs está ocorrendo, pois o Campo Descrição Complementar
do Item (SAFX08-21-Tamanho 255) está com o tamanho maior do que é permitido
para Geração das Aquisições a partir das NFs (APT_Aquisicao-Tamanho 50). 

  

Mapeamento SAFX’s: 

 
 
Erro na geração dos registros de Aquisição do ativo permanente: 

  

SAF_CIAP_AQUISICAO: Ora 12899 Value too large for column 

  

"USER-TAXONE_BEM_KLABIN". "APT_AQUISICAO"
"COMPL_DESCR_BEM" (Actual: 68, Maximun: 50) 

 

 

SAFX08 campo 21 tamanho 255 e ao gerar a
APT_Aquisicao o tamanho é 50,  

  

Teste tamanho alterado para 50 e gerou
normalmente.
