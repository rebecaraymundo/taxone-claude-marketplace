# TKT170878 - MAGAZINE LUIZA - MASTERSAF DW - SPED -  - Erro no log Tipo I050 - Erro ao inserir dados na tabela SPED_I050_WORIO RK. ORA-00060: deadlock detected while waiting for resource, na geração do Meio Magnético - Copy

## Descricao

O cliente Magazine Luiza está enfrentando um problema ao gerar o meio magnético, especificamente o erro "Tipo I050 - Erro ao inserir dados na tabela SPED_I050_WORK. ORA-00060: deadlock detected while waiting for resource". Esse erro ocorre ao tentar gerar o período de 06/2024 a 12/2024. De acordo com o cliente, o problema começou a aparecer após a aplicação do Patch 315.0.0. 

Realizei testes com o Patch 315.01 e o problema não ocorreu. Por esse motivo, solicito apoio para identificar a causa do problema.

SaaS
