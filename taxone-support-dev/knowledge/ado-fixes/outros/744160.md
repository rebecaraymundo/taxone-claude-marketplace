# [LOJAS RENNER S.A.] TKT120047- DW - SPED - EFD - REINF - Relatórios - Relatório de Retençoes Previdenciarias  Esta gerando todas as notas das filiais na matriz.

## Descricao

Prezados, boa tarde! 

 

Com a atualização para o PATCH 301.0.0 o relatório previdenciário sofreu modificações, onde no campo de estabelecimento ( Coluna C ) esta sendo informado o Centralizador ( 991 ) , e não o estabelecimento de escrituração da Nota Fiscal, bem como o campo de CGC ( Coluna X ).  

Poderiam verificar esse questão, pois fica inviável realizar conciliação em 600 filiais sem esse campo chave especificado no relatório. 

 

No aguardo, 

- Vinicius Ferreira
