# TKT130140 - AGCO - ONESOURCE TAX ONE - MUNICIPAL - DMST-e  Atualização da TFIX83 para a tag <serie> município de Canoas

## Descricao

Cliente: 

Estamos tratando o tema de "SERIE", porque a mesma não existe no De-para para a serie que temos para as NFSE no Tax One. 

Sendo assim, conforme manual anexo, pagina 31, temos a seria "S", permitida pelo validador da prefeitura, ondem sem o depara, a informação não vai no layout do XML, gerando erro. 

 

Suporte: 

 

Atualizar a TFIX 83 - Cadastro das Séries/Modelos das Obrigações  

O manual de layout do município solicita a série "S". Na regra não consta a serie solicitado pelo manual de layout. 

Avaliar também quando não existir a série preenchida no documento e não existir a parametrização a tag <serie> deve ser gerada mesmo não constando informação, pois sem ela no arquivo é gerado erro no validador.
