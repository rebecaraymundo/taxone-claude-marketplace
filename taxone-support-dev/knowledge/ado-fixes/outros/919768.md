# TKT153004 - IFOOD - ONESOURCE TAX ONE - BÁSICOS - REPORT FISCAL - Relatório trazendo informação de outro campo no campo NUM_DOCFIS

## Descricao

Relatório está trazendo a informação do campo NUM_DOC_SER ao invés do campo NUM_DOCFIS 

 

 

 

 

Analisando a documentação de requisitos, o campo NUM_DOCFIS deve retornar os dados do campo número 8 do manual layout, que é o campo NUM_DOCFIS 

 

 

Manual layout 

 

 

 

Porém quando uma nota tem o campo NUM_DOCFIS_SERV preenchido na SAFX07, está retornando os dados desse campo no relatório. 

 
 

 

Por gentileza, poderiam verificar.

## Root Cause

Causa: Campo referente a NUM_DOCFIS esta demosntrado o valo de NUM_DOC_SERV.

 Solução: Ajuste na função COALESCE do campo para demonstrar o valor correspondente ao título do campo.
