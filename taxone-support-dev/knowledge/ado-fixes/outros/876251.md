# TKT145037 - YARA - ONESOURCE TAX ONE - BÁSICOS - DATA WAREHOUSE - Sistema não está permitindo a inclusão de NBM que não existe na base UAT.

## Descricao

Sistema não está permitindo a inclusão de NBM que não existe na base UAT. 

Vide anexos. 

Foi solicitado o select abaixo (ADO 875623) ao time de TechOps que retornou sem linhas. 
SELECT * FROM X2043_COD_NBM WHERE COD_NBM IN ('73151190', '39172290');

## Root Cause

Causa: Mensagem de falta de preenchimento de campo sendo visualizada na janela mesmo com o campo preenchido.

 Solução: Ajuste no processo de recuperação dos dados gravados de forma correta os campos chaves.

## Solucao

SaaS
