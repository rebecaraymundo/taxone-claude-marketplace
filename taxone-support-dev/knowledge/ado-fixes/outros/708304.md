- Modelo Msaf x Modelo  
- Efetuar uma pesquisa com uma data anterior a do cadastro na x2024; o produto não vai relacionar nada. 
- Sem sair da tela efetuar uma nova pesquisa com data valida, ou seja ,  que exista na x2024, o produto continuara sem recuperar nada.  

 

Resultado encontrado: O produto não está fazendo a limpeza da tabela após a primeira consulta com dados inválidos, ou seja, se efetuamos uma consulta com uma data que não possui cadastro na tabela x2024 o produto não recupera dados, mas se nessa mesma tela colocarmos uma data valida, o produto continua sem recuperar nada, pois fica com a memoria da primeira consulta em tela. `Para correta recuperação dos dados se faz necessário nesse caso , sair e entrar novamente no menu.
 

Resultado esperado: Que a tabela temporária dessa tela, seja limpa a cada pesquisa.
