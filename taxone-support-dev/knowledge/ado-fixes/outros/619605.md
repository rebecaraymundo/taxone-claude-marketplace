# [Internal Bug] T1 - Municipal - Fortaleza - EF-E - Ao tentar executar um Meio Magnetico, o processo fica agarrado no status iniciado e depois fica como interrompido

## Passos de Reproducao

URL do ambiente de testes utilizado: https://qa.onesourcetax.com/
  
 
 Steps: 
- Preencher o campo  Empresa -> 076 
- Preencher o campo Estabelecimento -> 27422 
- Preencher o campo Agrupamento -> Municipal 
- Preencher o campo Modulo -> Fortaleza - EF - E 
- Acessar o menu Obrigacoes --> Meio Magnetico 
- Preencher o campo Data Inicial -> 01/10/2022 
- Preencher o campo Data Final -> 31/10/2022 
- Marca o estabelecimento -> 01CE 
- Clicar no botao executar  
- Clicar na aba Processos  

 

Resultado encontrado: ao clicar no botão executar o processo nunca termina a situação sempre fica Iniciado e depois de um tempo ela vai para interrompido
 

Resultado esperado: O processo deve terminar e a Situaçao mudar para encerrado
 

Evidência da falha:
 

 

 Testes que falharam:    
VALIDADOR EF-e Fortaleza
