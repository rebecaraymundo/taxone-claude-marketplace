## Descricao

[TEX COURIER S.A.] TKT110272 - TAX ONE - Municipal - São Paulo NFTS - Após inserir a data de validade, e depois alterar, para buscar os serviços, o sistema não respeita a nova data inserida

Quando acessamos a tela de NFTS - Código de Serviços Pessoa Jurídica e inserimos uma data de validade, ao tentar editar a data, o sistema não aplica a alteração.

Este problema ocorre tanto com com o cliente, quanto conosco.
Gravei um vídeo demonstrando o problema.

## Release Notes
