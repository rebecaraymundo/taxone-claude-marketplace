# TKT171940 - FIAT - ONESOURCE TAX ONE - BÁSICOS - ADMINISTRATION - Usuário com Universal ID extenso não funciona corretamente.

## Descricao

O cliente Fiat está tentando cadastrar um usuário com o ID luisa.ferreirafournier@external.stellantis.com, que possui 46 caracteres. O usuário é cadastrado normalmente, mas ao acessar o Tax One, a lista de Empresas não é cadastrada, mesmo com todas as permissões cadastradas corretamente. 

 

Simulei esse problema no ambiente do suporte e quando o Universal ID é muito extenso, ocorre esse mesmo problema que o cliente está enfrentando.
