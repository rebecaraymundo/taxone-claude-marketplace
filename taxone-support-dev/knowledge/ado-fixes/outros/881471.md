# [CBD] TKT146201 - DW - ATIVO PERMANENTE - Erro ORA-01461 ao Solicitar Cálculo de Crédito - ICMS

## Descricao

Ao executar a rotina de Cálculo de Crédito - ICMS, o sistema apresenta o seguinte erro: 

"Erro ao gravar Estabelecimentos Selecionados no LOG: ORA-01461: é possível ligar um valor LONG apenas para inserção em uma coluna LONG." 

 

Esse erro está impedindo o cliente de utilizar os créditos referentes ao período, impactando diretamente a operação fiscal. 

Para apoiar na análise técnica, o cliente ativou o trace da rotina, e o resultado foi anexado à demanda para investigação detalhada. 

Apesar da atualização da base para o patch 309.0.0, o erro persiste.﻿

## Solucao

On-Premise

## Release Notes

Menu: Ativo Permanente >> Cálculo do Crédito - ICMS 
 Correção na geração para não ocorrer mais o erro quando seleciona todos os estabelecimentos.
