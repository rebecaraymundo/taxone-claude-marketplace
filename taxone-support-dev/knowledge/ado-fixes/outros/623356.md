# [ATVOS] TKT93577 - TAXONE - SPED FISCAL - Registro 1400 - Erro no Relatório de Manutenção

## Descricao

**Registro 1400 - Relatório de Manutenção do Registro 1400 não está gerando informações.** 
Cliente 
 
Não estamos conseguindo gerar o relatório de "Manutenção do Registro 1400". 

Apesar de termos registros gerados no período, no momento que geramos o relatório, está sendo exibida a mensagem de que "Não há registros" e o relatório gera em branco. 

Suporte 

Cliente realizou a inclusão do registro e ao consultar o relatorio é apresentado a seguinte mensagem " Não a registro". 

No arquivo magnetico o registro é gerado com sucesso. 

Simulação na aba Analysis

## Root Cause

Causa: O campo cod_item_part_mun não é obrigadtório mas esta sendo obrigatório para exibir dados do relatório.

 Solução: Alterado o SQL do relatório para permitir trazer registros mesmo que o campo cod_item_part_mun esta nulo.
