# [LM WIND] TKT90331 - TAXONE - Municipal - Ipojuca - Obrigações - Meio Magnético ( o campo <tpPais> do XML esta sendo gerado incorretamente) 

## Descricao

Ao realizar a geração do XML de Ipojuca, o campo <tpPais> esta saindo com o nome do País e não com o código conforme aponta no manual.

## Release Notes

Carta do Patch: DW E TAXONE 

 

Módulo: MUNICIPAL
