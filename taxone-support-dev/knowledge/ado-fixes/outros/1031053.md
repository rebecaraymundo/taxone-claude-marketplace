# TKT167372- TAX ONE - JOB SERVIDOR - PROD01 -  Erro de CHUNKING na base do cliente quando os processos são feitos pelo Webservice

## Descricao

Cliente: LIBBS FARMACEUTICA
Ambiente: PROD01
ID: BI0 

Ao realizar importação via Webservice o Sistema está apresentando erro: "INVALID TABLE FOR CHUNKING"

OBS: Apenas quando as tabelas vem pelo Webservice o erro ocorre.

## Root Cause

O erro ocorre porque o Job Servidor do TAX ONE executa múltiplas importações via Webservice de forma paralela e simultânea, tentando realizar operações de inserção em massa (chunking) nas mesmas tabelas SAFX ao mesmo tempo. O Oracle não permite que várias sessões façam chunking paralelo na mesma tabela simultaneamente, resultando no erro ORA-29491. Além disso, algumas procedures estão sendo chamadas dentro de contextos de transação que já possuem operações DML pendentes, o que também impede o chunking adequado das tabelas temporárias utilizadas na importação.
