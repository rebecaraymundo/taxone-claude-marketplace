# TKT 137960 - MRV - ONESOURCE TAX ONE - BÁSICOS - JOB SERVIDOR - Erro importação SAFX2064 -- Campo 77 - (NAT_PESSOA_JUR) -- Opção 05

## Descricao

Cliente entrou em contato pois ao realizar a importação da X2064 para cadastro de estabelecimento, TaxOne acusa a seguinte mensagem: 

 

 

 

Analisando o restante do LOG e a SAFX, nota-se que no campo (NAT_PESSOA_JUR) o cliente inseriu a informação 05. Validando com o Manual de Layout, notei que os valores possíveis para este campo são: 

 

00 - Sociedade Empresária em Geral (não participante de SCP como sócia ostensiva); 01 – Sociedade Cooperativa (não participante de SCP como sócia ostensiva); 02 – Entidade sujeita ao PIS/PASEP exclusivamente Folha de Salários; 03 – Pessoa jurídica em geral participante de SCP como sócia ostensiva; 04 – Sociedade cooperativa participante de SCP como sócia ostensiva. 

 

Ou seja, não há opção 05. 

 

Verifiquei a proc com o código de erro gerado, retornando o seguinte: 

 

 

 

Porém, ao entrar no TaxOne > Básicos > Parâmetros > Preliminares > Empresa/Estabelecimento 

Ao ir na aba "Informações Especificas", no campo "Natureza da Pessoa Jurídica" há como cadastrar o estabelecimento com a opção 05. 

 

 

 

Precisamos que a validação seja corrigida para ler também este campo quando o valor for 05.

## Root Cause

Existe um tipo 5, mas o módulo criticava quando o  usuário escolhia esse tipo
