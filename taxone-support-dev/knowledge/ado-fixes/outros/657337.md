# [Conexão OnesouceBR]: DW/T1 - Conexão OnesouceBR - ECF-Escrituração Contábil - Exclusão Arquivo Vazio

## Descricao

Eu como usuário(a) fiscal do Integração de Arquivos ECF da Solução Fiscal para OnesouceBR.
 Gostaria que os arquivo sem movimentação não fossem gerados Para sanar situações de fila e processamento, onde arquivos mesmo vazios são gerados sem que haja necessidade e camel não trafega arquivos com menos de 5 bytes. 
 Funcionalidade Existente:  Agrupamento: Conexão Onesouce BR Módulo: ECF-Escrituração Contábil Menu: Geração > Geração dos Dados 
 Detalhes Adicionais 
- Arquivo sem informação não devem ser gravados 
- Prever Mensagem ao cliente

## Release Notes
