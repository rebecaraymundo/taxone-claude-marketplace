# TKT161704 - ZOETIS - TAX ONE - Básicos - Erro na equalização

## Descricao

TKT161704 - ZOETIS - TAX ONE - Básicos - Erro na equalização: 

"Erro nao previsto pelo sistema. ESTAB.: LIMASP GRAVA_DWT_ITENS_MERC: ORA-06502: PL/SQL: numeric or value error: character string buffer too small" 

 

 

 

A causa do erro está no campo "Número do item da NFCom anterior", que está preenchido com o valor "NULL". No Manual de Layout SAFX, informa que este campo aceita até 4 posições, porém conforme visto em call com o Jorge, este campo está aceitando até 3. Neste caso, a demanda tem por objetivo ajustar o tamanho do campo, para que esteja de acordo com o Manual de Layout: 

Campo 267 - SAFX08:

## Solucao

SaaS

## Release Notes

Menu: DATA MART > Equalização/Limpeza 
 Correção para não ocorrer mais o erro de ORA-06502.
