# [MHWIRTH]TKT86113 - TAXONE - EFD CONTRIBUIÇÕES - Registro C500 - Campo 11 - VL_ICMS - Reavaliar regra para também fazer leitura do campo ICMS Não escriturado do item das notas.

## Release Notes

Carta do Patch - DW e TAXONE

 

Menu SPED,
Módulo: Escrituração Fiscal Digital das Contribuições
        Menu
Obrigações > Geração dos Registros

    Nesta demanda será
liberada alteração na regra de geração do Campos 11 – VL_ICMS
do  registro C500, para considerar no cálculo o Valor de ICMS Não
Escriturado, para isso foi disponibilizado na tela de Dados Iniciais o parâmetro
“Considerar valor de ICMS Não Escriturado no Campo 11 - VL_ICMS”, se o
parâmetro estiver marcado o sistema passará a considerar os valores dos campos “43-Valor do
ICMS” da SAFX08 + o campo “80- VLR_ICMS_NDESTAC”. Caso estiver desmarcado
continuará preenchendo o Campo 11 – VL_ICMS com o valor do campo “43-Valor do
ICMS” da SAFX08.  

Para maiores detalhes consultar Help do módulo. 

Módulos Impactados >> Escrituração Fiscal Digital das Contribuições

Localização: SPED >> Escrituração Fiscal Digital das Contribuições
>> Obrigações >> Geração dos Registros
