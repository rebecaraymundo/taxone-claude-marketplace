# TKT158784 - ROYAL CANIN - TAX ONE - ESTADUAL - ATIVO PERMANENTE - Erro ao salvar ajuste do campo "Valor Total do Crédito Mensal" na Manutenção Dados de Cálculo

## Descricao

TKT158784 - ROYAL CANIN - TAX ONE - ESTADUAL - ATIVO PERMANENTE - Erro ao salvar ajuste do campo "Valor Total do Crédito Mensal" na Manutenção Dados de Cálculo 

O cliente está ajustando o campo "Valor Total do Crédito Mensal" em Estadual > Ativo Permanente > Movimento > Manutenção de Dados de Cálculo > Modelo D/Modelo C (BA, PR) > Crédito. Após inserir um novo valor, ao salvar e retornar à tela, o ajuste não é salvo. 

Em nossa base não foi possível reproduzir o comportamento. Em reunião com o Daniel (N3 – Não Simulado), foi apontado tratar-se de uma intermitência, possivelmente relacionada ao tempo de resposta da janela/cliente. Diante disso, alinhamos a abertura de demanda para correção do erro. 

 

Anexei o vídeo com o erro.

## Root Cause

Causa: Valores imputados na janela não estão sendo gravados na base de dados do sistema.

 Solução:  Ajuste no processo de gravação da janela para aceitar todos os dados digitados no momneto da gravação.
