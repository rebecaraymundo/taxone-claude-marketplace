# TKT147107 - TIMKEN - ONESOURCE TAX ONE - ESTADUAL - DIME SC - Registro 32 – Quadro 11: O campo "Saldo Devedor" (Item 170) não deve ser gerado quando houver saldo credor

## Descricao

Durante a validação do arquivo, identificamos um erro relacionado ao preenchimento do campo "Saldo Devedor" (Item 170) no Registro 32 – Quadro 11. 

O cliente possui apenas saldo credor, e conforme as regras de validação, o campo "Saldo Devedor" não deve ser gerado nesse cenário. No entanto, o sistema está gerando o campo com valor zero, o que resulta em erro na validação do arquivo. 

Solicitamos análise e correção do comportamento para que o campo "Saldo Devedor" (Item 170) não seja apresentado quando houver exclusivamente saldo credor.
