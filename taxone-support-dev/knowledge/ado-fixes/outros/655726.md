# [MARLIM AZUL] TKT101840 - TAXONE - MACAÉ - Erro campo 28 - Código do Serviço Municipal 

## Descricao

Municipio de Macaé 
 O campo 28 - Código do Serviços Municipal está apresentando erro no validador da prefeitura, referente o campo 28 - codigo do Serviço Municipal. 
 
 
 O layout de Macaé, menciona em um dos campos do layout, o chamado código de serviço da lista municipal. 
 
 Foi realizado um alteração manual no arquivo nas posições 676 a 695, colocando o CNAE e o mesmo foi validado. 
 Conforme call realizado com o cliente e o Denilson a regra necessita alterar a regra para buscar nesse campo o CNAE de acordo com o codigo do serviço.
