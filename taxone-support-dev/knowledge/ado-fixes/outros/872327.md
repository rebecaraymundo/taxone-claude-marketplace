# TKT142604 - SANTA CRUZ - ONESOURCE TAX ONE - ESTADUAL - RESSARCIMENTO DE ICMS-ST - PR -  REGISTRO 1115, não é possível excluir/alterar o registro uma vez preenchido os campos obrigatórios no documento fiscal

## Descricao

Olá, identificamos na rotina de geração do ressarcimento ICMS ST- PR que, uma vez informado em um documento fiscal os dados para geração do registro 1115 não há possibilidade de excluir/alterar o registro. 

 

Campos de preenchimento obrigatório conforme regra: 

- Período de geração for maior ou igual a 05/2025; 

- Somente Notas de entrada (MOVTO_E_S <> “9”); 

- Campo Identificador Único da Guia de Recolhimento deve estar preenchido (campo 276 da SAFX08); 

- Campo Situação Especial - Substituição Tributária” = ‘1’ (campo 132 da SAFX08).
