# TKT132063 - SPAL INDUSTRIA BRASILEIRA DE BEBIDAS S/A - DW - Básicos - Relatório de Validação de Sequência trazendo notas inutilizadas para algumas filias

## Descricao

Cliente inform que para algumas empresas  o relatório não esta sendo gerado corretamente, trazendo notas denegadas como ausentes.

Estamos com um erro no relatório de validação de sequencia em que o mesmo considera as notas fiscais já inutilizadas.
Não ocorre para todas as empresas/filiais, apenas algumas.

## Root Cause

Erro de PGA durante execução maior que um mês.

## Release Notes

Carta do Patch - DW e TAXONE 

 

Menu Básicos - Módulo: Report Fiscal.  

  

Obrigações Acessórias > Documentos Fiscais > Validação da Sequência de Documentos - Formato Excel 

  

  Limitar o período de execução do relatório em 31 dias.
