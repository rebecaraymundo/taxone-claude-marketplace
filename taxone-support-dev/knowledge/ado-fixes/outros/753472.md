# TKT122725 - APPLE COMPUTER BRASIL LTDA - TAX ONE - BÁSICOS - DATA WAREHOUSE - Alterar registros em tela safx04 dando erro.

## Descricao

Cliente subiu um registro com informações erradas para safx04 e ao tentar realizar a alteração via tela, não está sendo possível, dando mensagem de erro:
090084 - CPF/CNPJ para este Tipo de Registro não deve ser Preenchido'

Identificamos um cadastro incorreto na safx04. Trata-se de um residente/domiciliado no exterior. UF = EX
O cadastro veio com campo CPF/cnpj preenchido e com campo Indicador do código' para = Código 

Tentamos alterar a informação do campo Indicador do código' para = Especial e apagar o conteúdo do campo cpf/cnpj, porém o sistema apresenta a mensagem:
090084 - CPF/CNPJ para este Tipo de Registro não deve ser Preenchido'

## Root Cause

Causa: Cliente subiu um registro com informações erradas para safx04 e ao tentar realizar a alteração via tela, não está sendo possível, dando mensagem de erro:
090084 - CPF/CNPJ para este Tipo de Registro não deve ser Preenchido'
 
 Solução: Ajuste no processo de validação da janela para permitir a limpeza do campo CPF/CNPJ e troca do conteúdo do campo "Indicador do Código".
