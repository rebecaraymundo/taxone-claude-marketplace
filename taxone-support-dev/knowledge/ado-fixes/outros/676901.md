# INFOLEGIS 375/24 - A - ISSQN - FRANCA/SP - Migra Sistema (SILTEC) - DW/T1

## Descricao

Conforme informações da Prefeitura de Franca/SP, consiste na migração do sistema  “GISSONLINE”, para novo Sistema “ISS Web” da Sil Tecnologia, para gerenciamento do ISSQN, a partir de 30 de julho de 2024, conforme informações disponibilizadas no Portal da Prefeitura: https://www.franca.sp.gov.br/noticias/financas/novo-sistema-de-administracao-tributaria-e-fiscal-emissor-de-notas-fiscais-eletronicas-de-servico-nfs-e

 O manual de layout do arquivo para importação de documentos de serviço está disponível no endereço: https://www.franca.sp.gov.br/index.php?option=com_phocadownload&view=file&id=14221:layout-importacao
