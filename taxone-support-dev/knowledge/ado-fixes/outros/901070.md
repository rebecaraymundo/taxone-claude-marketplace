# [RAIZEN] TKT149526 -  TAX ONE - MUNICIPAL - GISS ONLINE Paulinia -  "Código de Município causando o erro :  Atividade informada no campo Código de Tributação do município não existe"

## Descricao

Ao gerar o arquivo referente ao município de Paulínia no sistema Tax One e realizar a validação no portal da Giss Online, ocorre o seguinte erro: 
"Atividade informada no campo Código de Tributação do Município não existe."
Identifiquei que o problema está relacionado à parametrização do código de atividade.  

 

Especificamente, há uma inconsistência entre o código de atividade vinculado ao serviço (Serviço x COD_ATIVIDADE) e a extensão exigida pela prefeitura de Paulínia. Essa divergência impede a validação do XML, resultando na rejeição completa do arquivo. 

 

Para resolver a situação, é necessário ajustar a parametrização de forma que o código do serviço seja corretamente vinculado ao código de atividade com a extensão exigida pelo município, conforme evidenciado nos documentos anexos. (totalizando 9 dígitos)  

Documentos anexados para análise: 

- Evidência da parametrização atual; 
- Arquivo XML original (com erro); 
- Arquivo XML ajustado manualmente (validado com sucesso); 
- Teste realizado no ambiente de suporte.
