# [LOJAS UNIAO] TKT117329 - ONESOURCE TAX ONE - MUNICIPAL - BRAGANÇA PAULISTA - Validador DES XML - Notas Fiscais sem retenção do ISS não devem ser consideradas no arquivo.

## Descricao

Cliente 

Estamos com
erro ao importar o Meio Magnético de Bragança Paulista.

No XML deve ser gerado somente as notas com retenção de ISS.

Está causando críticas na prefeitura ao na tag base de calculo ao gerar no xml
notas sem retenção de ISS. 

 

 

  

Suporte 

Atualmente,
o arquivo gerado no validador DES XML inclui notas fiscais
"com" e "sem" retenção de ISS, resultando em críticas na
prefeitura ao na tag base de calculo durante a importação no site da
prefeitura. Esses erros ocorrem porque as notas sem retenção de ISS estão sendo
incluídas no arquivo. 

  

Por favor
avaliar uma funcionalidade no módulo que permita a geração de arquivos
contendo exclusivamente notas fiscais com retenção de ISS.

## Release Notes

Carta do Patch - DW e TAXONE 

 

Menu Municipal - Módulo: Bragança Paulista.  

  

Menu Obrigações > Meio Magnético > Gerar. 

  

Alteração da regra - Desconsiderar
as notas fiscais que não possuírem valores de ISS.
