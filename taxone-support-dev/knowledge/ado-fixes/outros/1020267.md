# TKT169035 - PROFARMA - TAX ONE - Job Servidor  - Importação SAFX341 - Registro 1400 SPED-EFD - Campo Valor Total

## Descricao

Olá,  

Cliente efetuou o carregamentos dos valores apurados para os códigos IPM RJREC00001 e RJREC00002 porem ao consultar a informações geradas na tela de manutenção , verificamos que o valor total nao está sendo atualizado, sendo assim não é gerado o respectivo registro no sped fiscal. 

Os valores não estão atualizando e somente atualizam quando entramos manualmente na tela de manutenção e clicamos no "Outras Deduções:" 

Abaixo print do lançamento simulando o exemplo dos valores carregados, para o registro RJREC00002 realizado o procedimento de acessar os dados carregados e clicado no campo "outras deduções" até o sistema atualizar o valor total. 

 

 

 

Gravação extensa, caso necessite basta solicitar que encaminho.

## Root Cause

Falha de produto.

## Solucao

SaaS
