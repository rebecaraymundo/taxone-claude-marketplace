# [DEMANDA INTERNA] [DW] - Basicos - Data Warehouse - Menu: Manutenção > Cadastros > Cadastro Pessoas Físicas/Jurídicas - Não é possível cadastrar um novo Participante

## Descricao

No ambiente de AC está ao tentar cadastrar um novo participante aparece o Erro abaixo 
 obs.:  verificada a sequence amarrada na tabela X04 e está OK, o problema não é acarretado devido a problemas na sequence. 
 -------------------------------------------- 
 Foi verificada e chegamos a conclusão que a Base de Dados de AC atingiu um numero de sequence com 12(doze) dígitos, devido a essa quantidade de dígitos o powerbuilder não consegue carregar a sequence para ser utilizada no insert pois a variavel está como Long(32bits), a melhoria deve ser efetuada colocando a varialvel que carrega a sequence para LongLong(64bits) conseguindo assim efetuar o insert corretamente na tabela X04.
