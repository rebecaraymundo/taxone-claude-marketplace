# TKT 168325 - IRB BRASIL - ONESOURCE TAX ONE - BÁSICOS - REPORT FISCAL - Relatorio nao considera documentos com classificação "I - Invoice"

## Descricao

Prezados, boa tarde. 

Estamos com o chamado 158425 aberto referente as Invoices que devem ser escrituradas na Nota Carioca, nesse chamado nos foi orientado a preencher o campo COD_CLASS_DOC_FIS com "I" (Invoice). 

Agora ao gerar o relatório abaixo para conferência do ISS, essas notas não estão vindo. 

Report Fiscal >> Gerenciais >> Documentos Fiscais >> Analíticos >> Conferência de Documentos Fiscais (INSS/ISS/PIS/COFINS) >> Geração 

Busquei outras opções de relatório, mas não encontrei nenhuma que venha essas notas. 

Atenciosamente,

## Solucao

SaaS

## Release Notes

Carta do Patch - DW e TAX ONE: 
  
 
Módulo impactado: Básicos > Report Fiscal  
 
Menu: Gerenciais > Documentos Fiscais > Analíticos > Conferência de Documentos Fiscais (INSS/ISS/PIS/COFINS) > Geração  
 
Alteração na regra da geração do relatório para passar a considerar o código “I” (Internacional/Invoice - proveniente do campo 12 da SAFX07) quando o parâmetro “Considerar Notas Fiscais de Serviço não escrituradas” estiver marcado.
