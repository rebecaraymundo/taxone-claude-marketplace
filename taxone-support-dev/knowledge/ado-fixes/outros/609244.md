# [VECTOR TRANSPORTES] TKT89908 - TAX ONE - UAT - Básicos - Report Fiscal - Erro ao selecionar tipo movimento no relatório de NF por CFOP

## Descricao

Cliente informou que o erro foi corrigido no relatório de NF por CFOP - modelo 2, porém o erro também está ocorrendo no Relatório de NF p/ CFOP.

Caminho: 
- Básicos >Report Fiscal> Obrigações Acessórias>  
- ICMS/IPI p/ DataMart>  
- Relatório de NF p/ CFOP 

Erro ao selecionar opção de movimento no relatório de NF por CFOP:

Isso está ocorrendo apenas neste cliente.

Obs: Foi feita uma correção para este cliente na demanda 591163 onde ocorria o mesmo problema.

Por gentileza, verificar.

## Root Cause

Causa: Data fim não estava sendo recuperada de forma correta pela aplicação

 Solução: Criada uma função para recuperação dos parâmetros de recuperação dos CFOP e retrieve da listagem de CFOP's válidos para o período de pesquisa.
