  Fazer a emissão do relatório Modelo C no Report Server. Módulo: Estadual >> Ativo Permanente Menu: Relatórios >> Emissão Livro >> Modelo C - Report Server
 Tomar como base o relatório disponível no menu: Relatórios >> Emissão Livro >> Modelo C. Os seguintes critérios devem ser atendidos: 
- Disponibilizar o relatório em Excel e PDF. 
- Tratamento do relatório para período SEM MOVIMENTO

performance em relatórios, estamos disponibilizando o livro Modelo C do CIAP via
Report Server:

performance melhor.

C original será removida do módulo.
