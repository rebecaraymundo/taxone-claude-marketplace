# TKT98543 [HNK] TAXONE - SPED - EFD Contribuicoes - Erro no campo DATAFIM do Registro 0205

## Descricao

Boa tarde! 
 Cliente relata que o campo DATAFIM do registro 0205 do EFD Contribuicoes esta assumindo valor maior que a data do periodo de geracao. 
 Exemplo: geracao de efd de 05/2023 gera o 0205 com data futura de 07/2023.

## Release Notes

Menu: Obrigações >> Geração dos Registros 
 Correção para não gerar o registro 0205, quando a data da alteração for posterior a data fim gerada.
