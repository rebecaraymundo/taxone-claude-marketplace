# Servidor de Relatórios - TAXONE - Básicos - Report Fiscal - Diferencial de Alíquotas por CFOP / Documento -  QA

## Descricao

Obrigações Acessórias >> ICMS >> Diferencial de Alíquota p/ DataMart >> CFOP/Documento  
 Incluir tratamento do período SEM MOVIMENTO.

## Release Notes

Carta do
Patch – TAXONE APENAS: 

  

Módulo: Básicos >> Report Fiscal 

  

Resumo da
alteração: Disponibilizado
relatório de Diferencial de Alíquota p/ DataMart >> CFOP/Documento no
Servidor de Relatórios. 

Dando
continuidade às melhorias de performance em relatórios, estamos
disponibilizando o relatório emitido via Report Server: 

Obrigações Acessórias >> ICMS >> Diferencial de Alíquota p/ DataMart >> CFOP/Documento - Report Server 

Sobre Período Sem
Movimento: 

No caso de não existir
movimento para o estabelecimento e período, os relatórios emitidos em PDF
apresentaram o texto "SEM MOVIMENTO" no cabeçalho. Já os emitidos em
Excel, a primeira coluna "PERIODO_COM_MOVIMENTO" será preenchida com "não". 

 

Informamos que o relatório original (Diferencial de Alíquota p/ DataMart >> CFOP/Documento) continua disponível, mas recomendamos o uso da nova opção, visto
que em breve o original será removido do módulo.
