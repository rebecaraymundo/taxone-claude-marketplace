# [HNK] TKT91316 - TAXONE - Lentidão na consulta dos Lançamentos Contábeis

## Descricao

Cliente abriu chamado informando baixa performance na navegação do Tax One. 
 Solicitamos apoio para avaliar esta demanda.

Att:. Douglas Fernandes

## Root Cause

Causa: Para pesquisas utilizando num_lancamento a query de recuperação de lançamentos gravados não possuia boa performance na tabela X01_CONTABIL.

 Solução: Criado DDL para criação de novo índice por num_lamncamento na tabela X01_CONTABIL.
