# TKT162487 - ACUMULADORES MOURA - ONESOURCE TAX ONE - BÁSICOS - JOB SERVIDOR - Importações em loop

## Descricao

O cliente Moura continua enfrentando um problema crítico: processos de importação entram em loop e permanecem em execução indefinidamente. 

Esse incidente já foi tratado nas seguintes demandas: 

- https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/890851 
- https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/915633 
- https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/920496 
- https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/962249   
Apesar de existirem quatro demandas no histórico, cada uma delas contém diversos registros indicando que o problema foi corrigido. No entanto, após algum tempo, o erro volta a ocorrer, demonstrando que a solução aplicada não foi definitiva. 

Solicitamos que seja analisada a causa raiz dessa recorrência e que seja implementada uma solução definitiva para este problema. 

Relato do parceiro que está atendendo o cliente: 

"Pessoal !!
