# Ticket 105261- DW/T1 - SPED - SPED CONTRIBUIÇÕES - Obrigações - apuração - F800( M100/M500) 

## Descricao

Situação:  F800, avaliar a regra dos Créditos PIS/COFINS transferidos para forma dedução de crédito de forma manual. 
 Neste situação atual do cliente, ele faz a geração do F800 de forma automática, porem ele utiliza o fleg dos Dados iniciais, para a utilização do crédito de forma manual. E uma vez que este fleg esta ativo as linhas do M100/M500 ficam zeradas e bem como o F800. 
  
 Continuação da demana https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/659565
