# [MARLIM AZUL] TKT99906- TAXONE - MACAÉ - ATUALIZAÇÃO DO LAYOUT - Erro campo 23 - Regime Especial de Tributação

## Descricao

**TKT99906- TAXONE - MUNICIPAL - MACAÉ (VALIDADOR SPE) ATUALIZAÇÃO DO LAYOUT - Erro campo 23 - Regime Especial de Tributação** 

 Houve atualização do layout, e por consequência, quando o cliente importou o arquivo na prefeitura, ocorreu o erro:
 
"Detalhe: Campo Regime Especial () inválido."

Visto que este campo não existe no layout da regra para Macaé, também observamos que o campo 22 possui uma descrição diferente, e além do campo 23 que é novo, há o campo 24 - Cidade  de incidência do Serviço.

Será necessário que a regra atenda conforme o layout atualizado.
