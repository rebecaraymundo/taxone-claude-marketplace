# [PS] [KLABIN] TKT97656 - TAXONE - REPORT FISCAL - DIFAL - Relatório Documento - Formato EXCEL- Report Server com erro

## Descricao

Boa noite,  
 Cliente reporta que pós atualização MFS-627351 aplicado em 22/04 o relatório não está apresentando todos os documentos, fizemos a execução em excel e em PDF e ocorre o mesmo erro. Caminho:
Obrigações Acessórias > ICMSDiferencial de Alíquota p/ DataMart >Documento - Formato EXCEL Report Server > Emissão 
 
 tela de Básicos > Ferramentas > Parâmetros do Sistema > Parâmetros por Estabelecimento
 
 
 
 

Anexo relatórios gerando no ambiente de UAT da Klabin. Geramos o detalhado que esta com o erro e o resumido. Comparando os dois relatórios os documento fiscais não são os mesmos que foram gerados. As poucas notas fiscais que estão no relatório detalhado, não constam no resumido.

## Root Cause

Causa: Loop do cursor com inicialização em ponto errado.

 Solução:  Inicialização antes do fetch do Resumo_c do bulk collect.
