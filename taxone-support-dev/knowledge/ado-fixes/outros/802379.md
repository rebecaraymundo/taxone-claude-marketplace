# TKT 129247 - SODEXO DO BRASIL - MASTERSAF DW - SPED - EFD - ESCRITURAÇÃO FISCAL DIGITAL - O Registro E113 não foi gerado no caso de inscrição estadual única.

## Descricao

Conforme abertura de ticket para que seja analisado qual é motivo Sped Fiscal a partir do mês 02/2025 não está apresentando o registro E113. 

 

 

Solicitação de apoio para identificar o motivo pelo qual o Registro E113 não está sendo gerado no caso de inscrição estadual única para o SPED Fiscal no ambiente do cliente Sodexo. 

Até o momento, verifiquei todas informações da base  conforme simulação anexa, mas ainda não consegui determinar a causa do problema.
