# [MRV] TKT150635 - TAX ONE - MUNICIPAL - NOTA JOSEENSE Campo Local de prestação do serviço, não é gerado conforme tabela do IBGE.

## Descricao

Ao gerar o meio magnético para o município de São José dos campos / SP, o campo 28 "Código do município do Local da Prestação do Serviço" não é gerado conforme código da tabela do IBGE, ocasionando erro ao submeter o arquivo no validador do município. 

 

Exemplo: O código do município de Belo Horizonte, conforme a tabela do IBGE, é 3106200. No entanto, o valor que está sendo enviado para o meio magnético é 316200. 

 

O problema ocorre em todos os municípios.
