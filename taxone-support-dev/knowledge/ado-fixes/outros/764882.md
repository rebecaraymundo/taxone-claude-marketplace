# [HEINZ BRASIL S.A.] TKT125068 - TAXONE- Após migração do Relatório de Apoio à Apuração de Serviços para o servidor, mudou formatação dos campos com data

## Descricao

Relatório mudou o formato dos campos da data, gerando problema nos processos do cliente:

Obs: Cliente comentou ter outros relatórios com a mesma mudança, provavelmente os relatórios que foram migrados para o servidor de relatórios estão apresentando esse problema.

## Root Cause

Causa: Formatação de data do sistema não esta aplicada a sessão do cliente.

 Solução:  Inclusão do Alter Session NLS_DATE_FORMAT para formatação dos campos data com o formato "DD/MM/YYYY".
