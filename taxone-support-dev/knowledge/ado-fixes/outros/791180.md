# [VERIFONE] TKT126278 - TAXONE - ICMS - P9 resumo dos CFOPs entradas/saídas não é apresentado em PDF

## Descricao

Foi liberada a correção, porém o cliente devolveu relatando que a correção não surtiu efeito. 
Efetuado novos testes na base do cliente, uma vez que é homologação e realmente continua com o problema. 

>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>> 

Após efetuar a geração e emissão, o P9 não apresenta o resumo dos CFOPs entradas/saídas em PDF. 

Inicia a partir dessa página: 

 

 
Double check: Fabiana/Elisabeth

## Root Cause

Causa: Formatação da data no ambiente do cliente esta divergente com o padrão adotado pelo TaxOne.

 Solução: Inclusão das funções f_get_data_formatada e f_get_periodo_mes_anopara formatação das datas no SQL do relatório para adequar o campo dat_apuracao para o formato DD/MM/YYYY.
