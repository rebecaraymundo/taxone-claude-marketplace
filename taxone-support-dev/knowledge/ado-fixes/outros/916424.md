# [TICKET] TKT152480 - MASTERSAF DW - SPED - EFD - REINF - Apoio

## Descricao

Eventos R4080 - Transmitidos com valores divergentes 

 

Precisamos de apoio com base nas informações abaixo. 

Mensalmente enviamos uma media de 210 mil eventos R4080 para a Ticket Serviços, este mes apos a geração dos eventos constatamos que o volume estava dentro do esperado e o fluxo de transmissão dos eventos seguiu normalmente, ocorre que os valores esperados não correspondem ao que de fato foi importado para o DW, temos um valor de imposto 3x menor do que o esperado, com a mesma quantidade de eventos. 

Fizemos uma validação nas informações que foram importadas para o DW (DOC1), e identificamos que foram carregados corretamente. 

Como o relatorio de conferencia dos eventos 4000 não esta funcionando, obtivemos de voces um select para conferencia dos eventos gerados e identificamos que ha uma divergencia consideravel nos valores que foram gerados com os valores que foram importados para o DW 

Precisamos de apoio para entender se o problema encontrado foi no momento da geração dos eventos R4080, que por algum motivo não somaram os valores em sua totalidade, ou se há algum outro problema afetando nossa transmissão. 

Necessitamos tambem saber se é possivel reprocessar todos os eventos, mudando seu status para que sejam eventos de retificação, fazendo assim uma nova transmissão de tudo que ja foi enviado para a Receita.
