# [VULCABRAS] TKT89386 - DW - Básicos - Apresentação de Caracteres Especiais Módulo REINF - PRD 

## Descricao

O cliente abriu chamado relatando erro de caracteres especiais em todos os relatórios do módulo REINF.
 
 Versão Patch aplicada 289.1.0 
 Verificado que todo o módulo do REINF está apresntando o erro. 
 Att:. Douglas Fernandes

## Root Cause

Causa: Encoding das procedures fora do padrão ANSI causando caracteres estranhos nos relatórios.

 Solução:  Re-Encoding das procedures nos padrões ANSI de desenvolvimento.
 
 Ações preventivas/Boas práticas:
 N/A

## Solucao

On-Premise
