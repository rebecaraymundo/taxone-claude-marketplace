# [MRV] TKT142866 - TAX ONE - MUNICIPAL - NOTA CARIOCA - Ajuste de Regra para Preenchimento de código do Benefício

## Descricao

Regra do Registro Tipo 40 – Campo Código do Benefício Vai Preencher com o campo Benefício Fiscal da tela Parâmetros – Serviço x Benefício Fiscal referente ao serviço cadastrado na nota fiscal. 

Porém no manual de layout é destacado que o campo Código do Benefício somente deve ser preenchido quando o Tipo de Tributação de Serviço for 03 e 04  

 

 

 

 

Atualmente quando se tem o parâmetro em Parâmetros – Serviço x Benefício Fiscal  mesmo quando o Tipo de Tributação de Serviço é 01,02,05 ou 06 esta sendo levado o campo Código do Beneficio.  

 

 

Casando erro no validador. 

 

Abro esta demanda solicitando ajuste da regra porque quando for a situação 3/4, o beneficio deve ser preenchido, mas para as demais situação não.

SaaS
