# [GENERAL MOTORS] TKT 102472 - DW - JOB SERVIDOR - Importação Batch está mantendo o Status de Concorrência

## Descricao

**TKT102472 - DW - JOB SERVIDOR - Importação Batch está mantendo o Status de Concorrência mesmo após conclusão da importação.** 
O problema é que a Importação Batch está mantendo o Status de Concorrência da equalização como S, mesmo após conclusão da importação.  
Isso impossibilita importação manual ou ajustes manuais nos documentos, pois o sistema entende que existe um processo de equalização em andamento.

Cliente informou que continua travado o processo, mesmo ele tendo feito as correções que foram solicitadas antes e aplicado as boas práticas também. O problema está ocorrendo tanto na base de teste quanto na base de produção.

 

Por gentileza, auxiliar.
