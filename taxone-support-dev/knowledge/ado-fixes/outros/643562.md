# [DEMANDA INTERNA]  TAXONE - Codigo Fonte de Objetos que apresentam caracteres invalidos

## Descricao

Precisamos do código fonte dos objetos contidos no arquivo anexo (listados abaixo). Estes objetos no esquema TEMPLATE estão com caracteres que deveriam ser acentuados (ou cedilha) substituídos pelo caractere "¿".

O objetivo é verificar se está correto no código fonte original dos objetos e, caso positivo, a partir deles, fazer a correção no esquema TEMPLATE e nos esquemas dos clientes também.

## Root Cause

Causa: Procedures com caracteres inválidos no Taxone e/ou DW. (ex: ¿)

 Solução: Retirada dos caracteres inválidos para recompilação dos objetos ANSI.
