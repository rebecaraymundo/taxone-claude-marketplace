# TKT118897 - 22203-TAX - FERRERO DO BRASIL INDUSTRIA DOCEIRA E ALIMENTAR LTDA - DW - SPED FISCAL -  Erro: PowerBuilder application execution error (R0036)

## Descricao

Ao gerar o relatório dos Parâmetros por CFOP/Natureza de Operação - NF's Saída c/ direito a Ressarcimento (e Devoluções) ocorre o erro: de PowerBuilder application execution error (R0036)

## Root Cause

Causa: Open da janela de relatório não possui o campo t_8.text em sua datawindow principal.

 Solução: Teste para verificar a existência ou não do campo t_8.text para que a janela possa abrir normalmente.
 
 Ações preventivas/Boas práticas:
 N/A
