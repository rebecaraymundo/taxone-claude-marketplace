# [HNK] TKT102312 - TAXONE - EFD CONTRIBUIÇÕES - Erro no campo DATAFIM do Registro 0205

## Descricao

a tarde! 
 Cliente relata que o campo DATAFIM do registro 0205 do EFD Contribuicoes esta assumindo valor maior que a data do periodo de geracao. 
 Exemplo: geracao de efd de 05/2023 gera o 0205 com data futura de 07/2023.

## Release Notes

Informações p/a
Carta do Patch: TAXONE e DW 

  

Módulo: EFD – REINF e EFD – Escrituração Fiscal Digital das
Contribuições

       
Nesta demanda, será liberada alteração na geração do Campo 04 do
Registro 0205 quando a Data Fim do código anterior for igual à data
fim da geração, nesse caso, o sistema passará a gravar com a data fim de geração
-1 (D-1).   

Módulos Impactados >> EFD – Escrituração Fiscal Digital das
Contribuições 

Localização: SPED >> EFD – Escrituração Fiscal Digital das
Contribuições >> Menu Manutenção >> Geração dos Registros
