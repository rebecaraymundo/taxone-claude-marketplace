# TKT167121 - ONESOURCE TAX ONE - GISS ONLINE - Parâmetro por Município - Serviços Lei 116 x Serviços Municipais x Código Atividade " Serviço Não Exibido na Parametrização – Falha na Origem do Cadastro"

## Descricao

Ao tentar realizar a parametrização na tela Parâmetros > Serviços > Serviços Lei 116 x Serviços Municipais x Código Atividade, verificamos que, no campo Serviços, não estava sendo exibido o serviço necessário com a validade desejada (a partir de 2010). 

Com base nisso, identificamos que a origem do campo Serviço nessa tela vem do cadastro realizado em: 

GISS ONLINE > Parâmetros por Município > Parâmetros > Cadastro dos Códigos de Serviços Municipais da GISS ONLINE. 

Porém, constatamos uma inconsistência: essa tela não está permitindo a criação de novos códigos de Serviço. 

Abro esta demanda após alinhamento com a Alessandra Navatta. 

Como observação, seria interessante disponibilizar um material com as regras de preenchimento da tela Parâmetros > Serviços > Serviços Lei 116 x Serviços Municipais x Código Atividade, descrevendo detalhadamente as origens e dependências de cada campo para facilitar futuras parametrizações.
