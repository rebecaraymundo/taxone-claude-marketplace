# [MERITOR] TKT128095 - TAXONE - REPORT FISCAL - Relatório gerando data no formato americano para opção em excel

## Descricao

Time, cliente notou que ao gerar o relatório no formato Excel, os campos datas estão vindo com formato americano.
 
Básicos> Report Fiscal

Ambas as opções estão apresentando o mesmo problema.

Ao gerar os relatórios de rotina, notamos que ao salvar em Excel, o formato de data /hora segue com o formato americano.
As minhas configurações navegadoras e tela inicial do Taxone, já estão no formato Português, como orientado por vocês.

 
Poderiam nos apoiar nessa correção de formato de data e hora?
Pois isso irá impactar nas análises e conferências do cliente.

## Root Cause

Causa: Datas com formatação contendo dados de hora, minuto e segundo quando salvo em formato Excel.

 Solução:  Ajuste da formatação das datas quando salvo em formato Excel demonstrando em formato DD/MM/YYYY.
