# TKT139613 - STARRETT - ONESOURCE TAX ONE - MUNICIPAL - GISS ONLINE - ITU - Estabelecimento não aparece para seleção na geração do meio magnético do município de Itu. - Copy

## Descricao

O estabelecimento não aparecem para seleção na geração do meio magnético do município de Itu - GISS ITU. 

 

A parametrização do município no cadastro do estabelecimento foi realizado.

## Root Cause

Causa: Estabelecimento não aparece para seleção na geração do meio magnético do município de Itu mesmo com o SQL recuperado na base do cliente validar que o SQL traz sim os dados mas não demonstra em tela.

 Solução: Refeito o SQL de recuperação dos estabelecimentos por UF e município da GISS.
