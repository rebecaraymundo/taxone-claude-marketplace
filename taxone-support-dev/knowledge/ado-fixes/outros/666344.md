# Servidor de Relatórios - TAXONE - Básicos - Data Warehouse - Conferência de Documentos Fiscais (ICMS/IPI/PIS/COFINS) - Tratamento SEM MOVIMENTO (QA)

## Descricao

Tratamento do período SEM MOVIMENTO p/ relatório de Divergência de Valores entre Capa x Item - Doc. Fiscal  Localização: Módulo: Básicos >> Data Warehouse  Menu: Relatórios >> Conferências >> Divergência de Valores entre Capa x Item - Doc. Fiscal  >> Geração / Emissão 
 Descrição: Aplicar tratamento do período sem movimento para o relatório.  Tratamento para Período sem Movimento:
 
- Arquivo PDF incluir texto “SEM MOVIMENTO” no
cabeçalho. Cabeçalho deve conter os dados de estabelecimento, período como
ocorre para um período sem movimento. 

- Arquivo EXCEL incluir a coluna
“PERIODO_COM_MOVIMENTO” e preencher com “NÃO” para períodos sem movimento. 

 ***** A coluna “PERIODO_COM_MOVIMENTO” deve ser incluída ao final do relatório!! *******

## Release Notes

Carta do Patch – TAXONE APENAS: 

  

Módulo: Data Warehouse 

  

Resumo da alteração: Melhoria no tratamento do relatório de Divergência de Valores entre Capa x Item - Doc. Fiscal (via Servidor de Relatórios) quando a período sem movimento. 

 

Módulo Básicos >> Data Warehouse
 

Menu: Relatórios >> Conferências >> Divergência de Valores entre Capa x Item - Doc. Fiscal  >> Geração / Emissão 

 

 

No caso de não existir movimento para o estabelecimento e período apurado, o relatório emitido em formato PDF, apresentará o texto "SEM MOVIMENTO" no cabeçalho. Quando emitido em formato Excel, a última coluna "PERIODO_COM_MOVIMENTO" apresentará a informação "não".
