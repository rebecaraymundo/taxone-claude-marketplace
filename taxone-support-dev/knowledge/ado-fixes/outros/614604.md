# [IGREJA UNIVERSAL] TKT91033 - TAXONE - REINF - Log de geração prévia dos movimentos informa R-4010 gerado com sucesso mas não lista eventos no painel.

## Descricao

Em 01/2024 tem 9922 eventos R-4010 recebidos pelo fisco com sucesso (normais, não são jcp). 

 O cliente identificou alguns cpfs que não geraram, quando eu submeto a geração prévia dos movimentos R-4010 para 

ver se é criado os faltantes, o log aponta que Geração do evento R4010 finalizada com sucesso, porém

no painel não lista nenhum evento R-4010 aguardando geração do XML. 

 Busquei um exemplo de controle de tributos que não gerou e esta informado a devida pfj com CPF, natureza de rendimento, data fiscal e data de fato gerador são iguais. 

 O tema foi escalado devido a falta desses eventos para concluir o valor a ser pago na dctf. 

Precisaria de um apoio para buscarmos a solução desse tema.

## Release Notes
