# [ELEVADORES OTIS] TKT96857 -TAXONE - MUNICIPAL - SÃO JOSÉ DOS CAMPOS - CAMPO 24 - Código de serviço Prestado -  RN34

## Descricao

**SÃO JOSÉ DOS CAMPOS - Campo código de serviço Prestado gerando sem o ponto (4 posições) e o correto conforme manual, é 5 posições** 

Identificamos um problema no sistema referente à geração do campo 'Código de Serviço' no arquivo de São José dos Campos. Verificamos que o ponto de separação, necessário para a correta validação, não está sendo incluído. Esse erro está afetando a entrega da documentação à Prefeitura, pois o código é gerado sem a mencionada pontuação. É essencial que a inclusão do ponto seja implementada no sistema para assegurar a conformidade com os requisitos de validação.

## Release Notes

Carta do Patch - DW e TAX ONE: 

 Módulo: Municipal > São José dos Campos
     Menu: Obrigações > Meio Magnético Alteração da
regra de geração das notas fiscais de serviços tomados, que passa a gerar o
campo 24-Código de Serviço Prestado com 5 posições, passando a considerar o ponto, conforme definido no Manual Layout.
