# TKT160433 - SEARA - ONESOURCE TAX ONE - BÁSICOS - OBI - ONESOURCE BRASIL INTEGRATION - PROD14 -  A importação da SAFX266 não lê os registros quando a origem dos dados é o OBI.

## Descricao

O cliente SEARA ALIMENTOS está enfrentando problemas ao efetuar a importação da SAFX266 via OBI. Quando a origem da carga é o OBI, a importação lê 0 registros. No entanto, se o mesmo arquivo for enviado via SFTP, a importação automática lê os registros normalmente. 

 

Ocorre o mesmo cenário atendido na ADO: User Story 897751 TKT148018 - FORD - ONESOURCE TAX ONE - BÁSICOS - OBI - ONESOURCE BRASIL INTEGRATION - A importação da SAFX44 não lê os registros...
