# [Dootax API/Integração] DE/PARA de Códigos de Empresa Tax One/Dootax-Desenvolvimento

## Descricao

Objetivo

 

Criar, em caráter de urgência, uma tabela específica para INSERT, a fim de atender à demanda de alinhamento técnico com o cliente e garantir a continuidade dos testes na solução contratada do cliente DOOTAX. Neste momento, será utilizada essa tabela de INSERT para atender o cliente Raízen em regime de "toque de caixa", evitando a interrupção dos testes, em conformidade com o prazo de entrega acordado em reunião para o dia 19/12. 

Regra de negócio
Com base nos dados disponibilizados pelo cliente, será criada uma tabela de DE/PARA entre os códigos utilizados no Tax One e os códigos utilizados no Dootax. As conversões realizadas por meio dessa tabela terão efeito exclusivamente sobre as empresas/estabelecimentos do cliente Raízen. Todos os arquivos JSON gerados para "Guias de Pagamento" deverão, obrigatoriamente, considerar os relacionamentos DE/PARA definidos na nova tabela, aplicando-se apenas ao cliente Raízen. 

Ressalva
A demanda nº  será responsável pela criação de uma tela de conversão DE/PARA, a ser disponibilizada ao cliente, para que este realize as conversões conforme o padrão TR. 

Dados
Link do DE/PARA para INSERT. De_Para_SAFTB003_004_.zip
