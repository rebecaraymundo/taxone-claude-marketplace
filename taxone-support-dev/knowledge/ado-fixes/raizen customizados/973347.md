# [T1/API-Contabilização] Desenvolvimento  de Tela de Estorno -Desenvolvimento

## Descricao

O objetivo é criar uma tela específica que permita ao usuário realizar o processo de estorno de forma prática e eficiente. O estorno é uma operação contábil fundamental, utilizada para corrigir ou anular lançamentos realizados de forma incorreta ou que necessitam de ajustes. 

Essa operação é executada por meio de uma técnica de lançamento inverso, que consiste na reversão dos valores originalmente registrados. Nesse processo: 

- Um débito é convertido em crédito; 
- Um crédito é convertido em débito.   Regra de Negócio:
