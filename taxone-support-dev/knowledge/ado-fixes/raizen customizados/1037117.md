# Dootax API Integração Chamado #171637 Erro no preenchimento do Parâmetro em tela

## Descricao

Objetivo: 
 Trata-se da correção do erro reportado pelo usuário ao tentar criar um parâmetro de guia de pagamento para ICMS_NORMAL. 
Atualmente, o TaxOne exibe a mensagem informando que “o cadastro já existe”, porém o parâmetro nunca foi criado pelo usuário.

Esse comportamento incorreto impede a continuidade do processo. 

Impacto: O erro impede o usuário de configurar os Parâmetros necessários para a geração das guias de pagamento, bloqueando todo o fluxo de criação e envio das guias. 
 Jornada do usuário: 
 Manutenção> Guias de Pagamentos>Parâmetros
