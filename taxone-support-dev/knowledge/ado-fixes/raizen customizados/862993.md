# BUG - Ajuste Tela PB -Descrição de regras e layouts preenchimentos campos parâmetros

## Passos de Reproducao

Identificado alguns ajustes de tela a serem realizados
conforme mencionado abaixo. 

 - Ajuste Descrição parâmetros Regra de Vencimento 

Altera na tela de seleção da regra de vencimento as
seguintes informações: 

Campo POSTO REGRA VENCTO para “REGRA VENCIMENTO” e campo VW
REGRA VENCTODESCRICAO PARA DESCRICAO VENCIMENTO 

 

 - Ajuste Descrição parâmetros Regra de Imposto No menu Guia de Pagamentos/Parâmetros no parâmetro “Regra Imposto” alterar o nome da descrição “REGRA IMPOSTO DESCRIÇÃO” para “DESCRIÇÃO IMPOSTO” 
 - Ajuste layout de campos dos parâmetros de guias das regras Código Arrecadação, Cod Receita, Det Receita e Convenio.
 
  Campo CODIGO ARRECADACAO É um campo de tamanho 20 e está aceitando preenchimento de mais de 20
caracteres. Ajustar o parâmetro para: 1- Limitar o campo para quantidade de
caracteres do layout. 
  Campo COD RECEITA É um campo com tamanho de 20 caracteres e está aceitando preenchimento
de mais de 20 caracteres e um campo numérico aceitando preencher com Letras. 1- Limitar o campo para quantidade de
caracteres do layout e deixar como numérico. 
 Campo DET_RECEITA Apesar de ser um campo ainda sem tratamento. É um campo com tamanho de
20 caracteres e está aceitando preenchimento de mais de 20 caracteres e um
campo numérico aceitando preencher com Letras. 1- Limitar o campo para quantidade de
caracteres do layout e deixar como numérico. 
 Campo CONVENIO Apesar de ser um campo ainda sem tratamento. É um campo com tamanho de 30
caracteres e está aceitando preenchimento de mais de 30 caracteres. 1- Limitar o campo para quantidade de
caracteres do layout. 
 Campo Regra Vencimento É um campo com tamanho de 8 caracteres e está aceitando preenchimento de
mais de 8 caracteres. 1- Limitar o campo para quantidade de
caracteres do layout.
