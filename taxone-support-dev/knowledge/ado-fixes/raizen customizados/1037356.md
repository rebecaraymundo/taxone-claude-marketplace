# [Dootax APIS- Integração]Requisito/Desenvolvimento/Teste – Tela De/Para Código de Empresa

## Descricao

Objetivo
 
Criar uma tela de De/Para que permita ao usuário informar manualmente os códigos de Empresa e Estabelecimento.

Esses códigos serão utilizados para sobrescrever (reverter) os códigos padrões atribuídos automaticamente pelo sistema, garantindo que a geração do DARF e do JSON utilize exatamente os códigos corretos em situações excepcionais ou personalizadas. Regras de Negócio 
O usuário poderá informar manualmente códigos de empresa e estabelecimento sempre que necessário, para garantir a correta geração ou reversão do DARF. 

Se o usuário não preencher os campos manuais: 

- O sistema deve utilizar os códigos padrões atribuídos pelo processo automático.  
Se o usuário preencher qualquer um dos campos: 

- O sistema deve substituir o código padrão pelo código informado manualmente. 
- O sistema deve utilizar esses códigos durante:

- A preparação e envio do JSON
