# [T1/API-Contabilização]-Testes(QA)- Validação tela Parâmetros ref Ajuste/Outros Lanctos da Apuração/Complemento Lançtos Contábeis - Diferença valores/Geração da Guia/Alteração de status/Geração do arquivo Json e Relatório do status de ICMS_Normal

## Descricao

Objetivo 

Desenvolver uma tela de parametrização de Ajustes e Outros lançamentos de contas contábeis no módulo de Contabilização, permitindo ao usuário configurar, importar, gerenciar e exportar parâmetros contábeis relacionados à apuração de impostos (ICMS). Essa funcionalidade visa garantir a correta integração e contabilização dos dados fiscais, seja de forma automática ou manual.
 

A tela de status é projetada para proporcionar aos usuários uma visão clara e atualizada do status dos arquivos JSON gerados até o envio. Essa tela desempenha um papel crucial na gestão e no monitoramento dos processos de lançamentos contábeis, garantindo transparência e controle efetivo sobre as operações realizadas. Todas as informações geradas na tela da geração serão espelhadas no Relatório de Status.
 

A geração do arquivo tem como objetivo armazenar e organizar informações relevantes às operações realizadas pelo usuário no formato JSON 
É desenvolver uma tela específica para a realização de lançamentos complementares de impostos como ICMS, ICMS-ST, DIFAL, FCP, IPI e PIS/COFINS, destinada à geração do arquivo JSON. Essa tela será utilizada para contabilizar ajustes e outros lançamentos que exigem a inclusão de valores adicionais, permitindo ao usuário completar a contabilização de forma precisa e adequada. A funcionalidade foi projetada para que os valores necessários sejam inseridos manualmente pelo usuário, sem dependência dos valores provenientes das apurações de impostos como ICMS, ICMS-ST, DIFAL, FCP, IPI e PIS/COFINS.
 
Proposta Inicial:
 

 Ao clicar no menu Outros Lançamentos da Apuração, o sistema deverá abrir uma janela modal (pop-up) sobre a tela principal, permitindo ao usuário realizar as parametrizações de Outros Lançamentos da Apuração sem perder o contexto da tela de seleção de dados. 

Repita a mesma orientação que foi aplicado para o Ajustes da Apuração. 
 
 
Regras de Negócio 

Essas operações incluem lançamentos contábeis das seguintes categorias: 

- Ajust
