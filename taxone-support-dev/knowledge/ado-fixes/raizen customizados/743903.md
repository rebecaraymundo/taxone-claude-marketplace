# [Dootax APIS- Integração]Guias de pagamento de IPI - Desenvolvimento

## Descricao

Objetivo: Implementar a funcionalidade para gerar a Guia de IPI 
 Parâmetros : 
 Tabelas Utilizadas: 
-  ITEM_AP_DISCR;  ITEM_APURAC_CALC   Dados Necessários: 
 
- Dados da Empresa: Informações cadastrais da empresa responsável pela apuração. 
- Dados dos Estabelecimentos: Detalhes dos estabelecimentos vinculados à empresa.   Apuração: 
 
- TABELA=ITEM_APURAC_DISCR-Campo: COD_EMPRESA 
- TABELA =ITEM_APURAC_DISCR-: COD_ESTABELECIMENTO 
- TABELA =ITEM_APURAC_CALC Campo: COD_TIPO_LIVRO =002 
- TABELA= ITEM_APURAC_DISCR-COD_OPER_APUR = 013  
- STATUS   Descrição do Período de Apuração 
 Funcionalidade:
 
 TABELA = ITEM_APURAC_DISCR- CAMPO: DAT_APURACAO 
 Exibe o período selecionado pelo usuário para apuração. 
 Formato do Campo: 
- yyyy - mmm (onde yyyy representa o ano e mmm representa o mês abreviado, como Jan, Feb, Mar, etc.).Regras de Formatação:

- O sistema deve formatar automaticamente a data conforme o período selecionado, exibindo apenas o mês e o ano.
- Detalhes diários não devem ser incluídos.Regras de Limitação:
- O período selecionado não pode exceder um mês.
- Caso o usuário selecione um período maior, o sistema deve exibir a seguinte mensagem de erro:
- "Seleção inválida: o período de apuração deve ser de no máximo um mês."
- Caso o usuário selecione um período que ultrapasse esse limite, o sistema deve exibir a seguinte mensagem de erro: "Seleção inválida: o período de apuração deve ser de no máximo um mês."

Inclusão dos Impostos no Campo "Grupo de Impostos:

- IPIInclusão dos Impostos no Campo "TIPO":

- IPIPreenchimento do Campo "UF":
- Todos os impostos IPI utilizam:
Tabela TACE (Tabela de Código Tax Payments - DOOTAX), campo UF_FAVORECIDA = FEDERAL.
Regra de Data de vencimento: 

UF= Federal

- IPI = Dias uteis
- IPI = Antecipado
- Interação do Usuário:
- O usuário deve selecionar o tipo de imposto IPI.
- O usuário deve informar quantidade de dias que deseja antecipar ou dias uteis
- 
Cálculo Automático:

- O sistema identifica o vencimento regular
