# [Dootax APIS- Integração] Desenvolvimento/QA- Geração Guia de pagamento -SCANC -RAIZEN - parte 2

## Descricao

Objetivo
 

Incluir no sistema o processo de geração da Guia de SCANC e disponibilizar a visualização de seus dados no relatório "Status de Guias". 

Regras de Negócio 

- A geração da Guia de SCANC deverá seguir o layout de integração definido na planilha layout_integracao_txt - entrada. 
- A função interna responsável pela integração/geração deverá receber e retornar as informações em formato JSON, em conformidade com o layout especificado.  
- Link da documentação Dootax:  
- MTZ: MTZ-ICMS-Geracao_Guia de Pagamento - SCANC.doc  
Jornada do Usuário 

- Acessar o Tax One; 
- Navegar até: Módulo Básico > Guias de Pagamentos > Geração > Status > Relatórios; 
- Efetuar a Geração das Guias na  "Status de Guias"
 
- Visualizar os dados das Guias de SCANC geradas no relatório "Status de Guias".  
Finalidade
Essa funcionalidade garante que os dados fiscais sejam organizados de forma eficiente e acessível, permitindo: 

- o armazenamento em tabelas específicas; e 
- a exportação em formato JSON para integração com os sistemas Dootax Raízen e Tax Payments – TR via OBI.  Print de telas: Parâmetro
