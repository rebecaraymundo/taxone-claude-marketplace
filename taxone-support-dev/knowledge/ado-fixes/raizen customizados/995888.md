# [Dootax APIS- Integração]Desenvolvimento Guias de pagamento geração do JSON/Relatório de Status do ICMS-ST

## Descricao

Objetivo

Incluir no sistema o processo de geração da Guia de ICMS-ST e disponibilizar a
visualização de seus dados no relatório "Status de Guias". 

Regras de Negócio 

 
- A geração da Guia de ICMS-ST
     deverá seguir o layout de integração definido na planilha layout_integracao_txt
     - entrada. 
 
- A função interna responsável
     pela integração/geração deverá receber e retornar as informações em
     formato JSON, em conformidade com o layout especificado.  
 
- Link da documentação
     Dootax:  
- MTZ: MTZ-ICMS-Geracao_GUIA_ICMS-ST (1).doc  
Jornada do Usuário 

 
- Acessar
     o Tax One; 
 
- Navegar até: Módulo
     Básico > Guias de Pagamentos > Geração > Status > Relatórios; 
 
- Efetuar a Geração das Guias na  "Status de Guias"
 
- Visualizar os dados das Guias
     de ICMS-ST geradas no relatório "Status de Guias".  
Finalidade

Essa funcionalidade garante que os dados fiscais sejam organizados de forma
eficiente e acessível, permitindo: 

 
- o armazenamento em tabelas
     específicas; e 
 
- a exportação em formato JSON para
     integração com os sistemas Dootax Raízen e Tax
     Payments – TR via OBI.  Print de telas: Parâmetro 
 Tela de Geração: 
 Tela de Status
