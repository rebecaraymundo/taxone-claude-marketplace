# [T1/API-Contabilização] SAFX 345 -Importação Extra Apuração Contabilização- Desenvolvimento/Testes

## Descricao

Objetivo:  
A tabela SAFX 345 foi
desenvolvida especificamente para carregar informações relacionadas aos
impostos de contabilização de extra apuração. O principal objetivo desta tabela
é viabilizar o correto processo de contabilização no sistema SAP, contemplando
os impostos do grupo de impostos "Extras Apurações", tanto em âmbito
estadual , em alinhamento com as parametrizações contábeis já
existentes no módulo de Contabilização do Tax One. 

  

Com isso, a SAFX 345 passa
a ser o repositório específico para os lançamentos de extra apuração que
envolvem contabilização, assegurando que todas as informações necessárias
estejam disponíveis para o SAP com o detalhamento exigido pelos processos
fiscais e contábeis, reduzindo falhas de parametrização, evitando ausência de
dados para determinados impostos e mantendo a consistência entre o Tax One, os
livros fiscais e o sistema SAP. 

 

Link de acesso do documento: 

MTZ_Carga_Importacao_SAFX_Extra_Apuracao.docx
 

 

Regras de Negócio: 
 
- Todos os campos obrigatórios devem ser preenchidos, caso contrário o sistema deve exibir mensagem indicando o campo faltante. 
- A chave de identificação (CHAVE_ID) deve ser única para cada registro e é obrigatória. 
- O campo "Grupo de Imposto" deve indicar se o registro é Federal ou Estadual. 
- O campo "Imposto" deve ser preenchido conforme o grupo de imposto, podendo ser: IPI (Federal), ICMS-Próprio, ICMS-ST, ICMS-Antecipado, FCP, DIFAL (Estadual). 
- Devem ser informados o código da empresa e do estabelecimento. 
- Os campos de período (mês) e exercício (ano) da apuração são obrigatórios. 
- O código e a descrição de ajuste devem ser preenchidos conforme a apuração dos tributos. 
- Os campos de débito e crédito exigem código e descrição da conta conforme o plano de contas, além dos valores dos lançamentos. 
- O login do usuário responsável e o CNPJ do estabelecimento devem ser informados    Jornada do Usuário  
- Acessar o Tax One; 
- Navegar até: Módulo Básicos > Job Servidor >
