# [API de Integração de Contabilização]Testes -SAFX344 de Parâmetro Outros Lançamentos

## Descricao

Objetivo 

A tabela SAFX (inserir número de reserva) foi desenvolvida especificamente para carregar informações relacionadas aos Outros Lançamentos da Apuração que não estão disponíveis no sistema Tax One. O principal objetivo desta tabela é facilitar a geração do JSON necessário para a integração com o sistema Tax Payments. 

Adicionalmente, a funcionalidade de importação associada à SAFX 344 tem o propósito de trazer dados exclusivos para o Grupo de Impostos 'Extras Apurações'. Estes dados, ausentes no Tax One, são essenciais para viabilizar a correta geração das guias de pagamento. 

 

Regras de Negócio / Comportamentos: 

- A importação da SAFX344 deve seguir as seguintes premissas de negócio e
comportamentos: 
- Layout e fontes de dados: o arquivo deve
obedecer ao Manual de Layout SAFX(inserir número) e às estruturas de referência
utilizadas pelo sistema: SAFX2002 (Plano de Contas), SAFX2003 (Centro de
Custo), SAFX2006 (Natureza da Operação) e SAFX147 (Operação de Crédito).  
 
 
Resultado da Importação: 

Caso a importação seja realizada com sucesso, os dados serão preenchidos
nas seguintes telas: 

- Módulo: Manutenção
>> Contabilização Apuração >>Seleção de Dados/ Parâmetros >>
Paramentos de Outros Lançamentos 
- Menu: Acesso
Principal >> Parâmetros  

Jornada do Usuário: 

- O usuário fará a importação da tabela e o preenchimento de dados conforme desejado.  
Regra Geral para validação
