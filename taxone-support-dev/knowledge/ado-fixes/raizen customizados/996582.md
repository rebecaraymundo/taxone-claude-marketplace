# [T1/API-Contabilização] Desenvolvimento inclusão de regra para da Geração do arquivo JSON ICMS Normal/ST/Difal/Antecipado/FCP/IPI/Extra Apur - Lançamentos Contábeis de Ajustes da Apuração/Lançamento complementar/Estorno

## Descricao

Objetivo 

A geração do arquivo tem como objetivo armazenar e organizar informações relevantes às operações realizadas pelo usuário no formato JSON 

Regras de Negócio 

Essas operações incluem lançamentos contábeis das seguintes categorias: 

- Ajustes da Apuração – Focado na apuração do ICMS -Antecipado. Utiliza a tela de Parâmetros Ajustes da Apuração para buscar os dados na apuração. 
- Complemento de Lançamentos Contábeis – Referente ao ICMS - Antecipado, utilizando a tela de Lançamento Complementar. 
- Estorno – Referente a lançamentos enviados anteriormente e retornados via SAFX, utilizando a tela de Estorno/SAFX de Retorno. 
- Status e Relatório de Status  MTZ: MTZ_Tela de Geração Arquivo Jason Contábil.docx  Link:  Layout de saida de Contabilização.xlsx MTZ: MTZ_Tela de Status Contábeis.docx MTZ_Tela de Estorno – Lançamentos Contábeis arquivo JSON.docx
 Layout_Relatório de Status Contábeis.xlsx MTZ_Tela de Complemento de  Lançamentos Contábeis - Manual Diferença de valores.docx
 
 Jornada do usuário:

- Acessar a tela de Geração de Arquivo JSON no sistema. 
- Informar os filtros/parametrizações necessários (por exemplo: período de apuração, empresa, filial, tipo de lançamento contábil, etc., conforme o caso). 
- Selecionar o tipo de lançamentos contábeis. 
- Confirmar a geração do arquivo JSON = ICMS -Antecipado 
- O sistema consolida e grava os dados contábeis em formato JSON. para execução na tela de Status.  Protótipo: 
 
 -------------------- 
Objetivo: 

Incluir o campo Número de Documento Contábil, retornado pela SAFX de Retorno nº 347, na lógica de Lançamento Complementar, impedindo que o usuário prossiga com o Lançamento Complementar quando não houver número de documento disponível na tela. 

Regra de Negócio:  

- O campo Número de Documento Contábil deve ser preenchido automaticamente pelo sistema com o valor retornado da SAFX de Retorno nº 347, campo NRO_DOC_CONTABIL, considerando os filtros existentes na tela: 
- Empresa 
- Estabelecimento 
- Gr
