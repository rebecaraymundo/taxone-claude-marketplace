# Regra campo val_item_discrim na geração das guias

## Passos de Reproducao

Para alguns tipos de imposto como por exemplo o ICMS INCENTIVADO na regra do programa GERA_GUIAS_PAGTO_FPROC no código de geração das guias, o campo val_item_discrim esta realizando alguns cálculos quando temos algum lançamento de incentivo na deduções como mostrado anexo.  
  Hoje se nos lançamentos de deduções no livro estiver 500 reais este valor ao gerar a guia entra na regra do programa apenas 10% do valor de 500 que seria 100 reais e este valor sai na guia para pagamento Temos também esta regra para outros tipo de impostos como o Gera_Fundo_Icms_Normal. 
 Avaliar se será necessário ter nesta regra este tipo de calculo!!
