# [T1/API-Contabilização] - Desenvolvimento de Tela Seleção de Dados e menus PARTE III

## Descricao

Objetivo: 

Desenvolvimento de uma Tela de Contabilização 

O objetivo é criar uma tela de contabilização que permita a apuração automática ou manual dos seguintes impostos: ICMS, ICMS-ST, ICMS-Antecipado, Difal, FCP, IPI, PIS e COFINS. Após a apuração, o usuário poderá selecionar e processar os dados contábeis de maneira eficiente. Além disso, a tela oferecerá funcionalidades para abrir modais que direcionam para as telas de parametrizações, essenciais para a correta integração e contabilização das informações. 

 
 
  
Localização da Tela:  
 
  
 
- 
Agrupamentos: Básico  
 
- 
Módulo: Manutenção >> Contabilização >>Seleção de Dados/ Parâmetros >> Geração/Execução >> Relatórios  
 
- 
Menu: Acesso Principal >> Seleção de Dados/ Parâmetros
