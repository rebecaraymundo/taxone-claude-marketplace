# TAXONE Datamart automático - Impedir o disparo manual ou desativar o automático por meio de parametrização sem conversão

## Descricao

Criar uma parametrização interna na aplicação e por cliente que:  
- Inative os acesso dos usuários disparem o Datamart manualmente 
- Inative o acesso dos usuários desativarem o Datamart Automático  Premissa: parametrização dos itens 1 e 2 acima deve ser online, sem necessitar de conversão 
  Para ativar/desativar  Nova coluna na tabela PRT_PAR_MSAF do schema do cliente IND_CTRL_DATAMART_AUTOM (0 desligado, 1 - ligado) 
Primeiros clientes selecionados 
- MARISA 
- ATLASSCHINDLER 
- AVENUEHOCHE 
- SODEXO 
- TICKETSOLUCOES 
- FEDEX 
- LOCCITANE 
- MAGAZINE LUIZA
