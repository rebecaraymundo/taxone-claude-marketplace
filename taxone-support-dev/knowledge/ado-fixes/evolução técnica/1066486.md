# TAX ONE - Melhorias Extrator - Abertura por Tenant - SP03/04 - CH

## Descricao

Conforme analise cenário Stelantis e uso do programa
      
1) Filas por Tenant
 

- Alterar o serviço receiver abrir conector filas por Tenant da mesma forma que temos no consumer (OBI) 
- Alterar o serviço extractor abrir conector filas por Tenant da mesma forma que temos no consumer (OBI)
  
 Para a alteração do receiver existem 4 opções para buscar os tenants, já que a rotina não conecta com o banco de dados: 
 1) Criar uma fila no rabbitmq alimentada a cada 1h pelo extractor com a lista dos tenants ativos e o receiver ler essa fila; 

 2) Fazer o receiver conectar com o banco para ler a tenants_connection a cada 1h;
 
 3) A cada 1h o TDM mandar uma lista de tenants ativos que usam TDM em uma nova fila do rabbit e o receiver ler essa fila;
 
 4) Criar um novo endpoint no taxone_admin para listar os tenants ativos e o receiver ler esse endpoint. (essa foi a escolhida)
