# [TAXONE][AUDIT][TRIGGER]

## Descricao

Favor promover para  o produto taxone a trigger e as alterações abaixo 
 alter table prt_par_msaf add dt_ult_alteracao  date;
alter table prt_par_msaf add usr_ult_alteracao varchar2(128);
  
 CREATE OR REPLACE TRIGGER trg_upd_prt_par_msaf
before update
 ON prt_par_msaf
 FOR EACH ROW
 DECLARE
 --
 -- OBJETIVO: auditar update da tabela par_prt_msaf 
 -- DATA    : Dez/2025
 -- AUTOR   : Alceste
 --
 BEGIN
       :new.dt_ult_alteracao  := sysdate;
 	  :new.usr_ult_alteracao := DBMS_UTILITY.OLD_CURRENT_SCHEMA;
 End;
