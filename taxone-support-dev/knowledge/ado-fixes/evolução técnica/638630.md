# [FILA EXEC]: T1 - Federal - IPI - PKG_PERDCOMP_MM_FPROC

## Descricao

como usuário(a) fiscal do IPI Gostaria que fosse implementado o processo de FILA EXEC para o package PKG_PERDCOMP_MM_FPROC Para favorecer o processo de geração, utilizando a solução de filas. 
 Funcionalidade Existente:  Agrupamento: Federal Módulo: IPI Menu: Meio Magnético > IN 320/03 PER/DCOMP > Geração Arquivo Reintegra 
 Detalhes Adicionais 
    Objetos para alteracao_lib_proc_executar
     - controle CPU, qtd processos unicos, qtds processos totais
     - atualização flags para limpeza automática (INCIADO, INICIALIZADO, FILA-EXEC e não termina)
 Cenário atual:
  - Pendencias na planilha de evolução técnica com > Objetos para alteracao_lib_proc_executar (dúvidas com Pessanha) Mitigar: Criação jobs no DB em massa, posso causar lentidão generalizada
