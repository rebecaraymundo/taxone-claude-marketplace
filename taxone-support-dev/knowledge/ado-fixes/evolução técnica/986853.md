# TKT160814 - EQUINOR - ONESOURCE TAX ONE - BÁSICOS - DW - Mensagem de erro, na tela de Item da Nota Fiscal, não permite trocar de abas.

## Descricao

Quando selecionamos o item e ele abre na tela de INF. Gerais,
depois selecionamos qualquer outra aba, o sistema emite a mensagem abaixo: 

“Existem
erros presentes no formulário, corrija-os para permitir a troca de abas.” 

 

 

Ao tentarmos editar o campo quantidade com a mesma
quantidade que foi importada o sistema informa que quantidade é inválida 

 
Porém, de acordo com manual layout, o campo
permite 011V006

## Solucao

SaaS

## Release Notes

Na tela de Nota Fiscal/Item de serviço quando importado um valor quebrado, a tela não permitia a troca de aba, mesmo não realizando nenhuma alteração.

Foi identificado que o campo Quantidade na aba "INF. Gerais" não permitia valores quebrados, foi realizado uma alteração para que um valor decimal seja aceito
