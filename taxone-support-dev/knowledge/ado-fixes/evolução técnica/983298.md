# TAXONE - Implementacao Tela Acompanhamento Importações SAFX  -  SP24

## Descricao

Conforme analise do ImporterSAFX  
- avaliar criação da tela de acompanhamento de importações   
 
 Motivo: Melhoria Processos TAXONE 
 Resultado Esperado:
    
 
  
 
 Estudo técnico - apenas apresentação a validação da proposta no Sprint 14 
 --- 
 
Filtro

- Período (data_ini e data_fim)

- Situação

  3 - Em fila

  7 - Validando Duplicidade

  4 - Importando

  9 - Interrompido

  8 - Pausado

  Todos

Primeiro Grid

- Checkbox para seleção

- Número do lote

- Número do Job

- Número Prioridade

- Data Inicio Recebimento

- Tempo de Processamento

- Status 

  

Segundo Grid

- SAFX

- Qtde Registros 

  

Ações

- Opção "Voltar para Fila", para mudar o status para 3, dos lotes com status 4 e 7 com mais de 2h de processamento.

- Opção "Interromper", para cancelar um lote com status 3, ou com status 4 e 7 com mais de 2h de processamento, mudar para 9 (Interrompido).

- Opção "Pausar", para colocar em Pausa (8) um lote com status 3.

- Opção "Retirar Pausa", mudar o status para 3 os lotes em Pausa(8).

- Opção "Priorizar", mudar o num_prioridade para 400 para lotes com status 3.

- Opção "Despriorizar", mudar o num_prioridade para 999 para lotes com status 3. 

  

  

Tela de visualização da Fila dos Lotes 

  

- Essa tela ficará no módulo Job Servidor

  Menu: Controles > Acompanhamento da Fila de Importação

- Listar todos os lotes com status 

  3 (Em Fila de processamento), 

  4 (Importando), 

  7 (Validação de duplicidade), 

  8 (Em Pausa), 

  9 (Interrompido)

  - ordenado pelo num_prioridade (do menor para o maior) e num_job (do menor para o maior).
