# TAXONE Datamart automático - Impedir por tela que o usuário desative o datamart automático e dispare manualmente - SP02

## Descricao

Com base nas parametrizações abaixo   
- Inative os acesso dos usuários disparem o Datamart manualmente 
- Inative o acesso dos usuários desativarem o Datamart Automático  Básicos > Data Warehouse > DATA MART > Equalização/Limpeza - Impedir que o usuário dispare manualmente FERRAMENTAS >> Parametros do sistema >> Parametros de DATAMART Automatico - Impedir que desative a configuração do Datamart Automático  Considerar o valor  Na tabela PRT_PAR_MSAF coluna IND_CTRL_DATAMART_AUTOM (0 desligado, 1 - ligado) 
Primeiros clientes selecionados 
- MARISA 
- ATLASSCHINDLER 
- AVENUEHOCHE 
- SODEXO 
- TICKETSOLUCOES 
- FEDEX 
- LOCCITANE 
- MAGAZINE LUIZA
