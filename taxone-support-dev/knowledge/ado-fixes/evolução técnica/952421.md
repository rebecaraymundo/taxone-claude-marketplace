# [CAMEL] TAX ONE - Arquivo e ZIP nomes diferentes - SP22

## Descricao

Solicitado uma mudança nos nomes dos arquivos para um tipo especifico de relatorio, onde a entrega exige um nome especifico.

O arquivo deve ser entergue no sftp seguindo a regra: 
 Arquivo TXT: CodRegulado_DataReferencia.txt Arquivo ZIP: 110_CodRegulado_DataReferencia_AAAAMMDDHHMM.zip 
 ​Para realizar essa alteração foi acordado que o arquivo vai ser gerado da seguinte forma para o Camel diferenciar esse novo processo dos outros já existente: 
 Arquivo de entrada: TR_ISIMP_CodRegulado_DataReferencia.txt 
 Assim o camel deve montar a estrutura desejada, exemplo mostrado abaixo:

Entrada:
           TR_ISIMP_7700_20250831.txt 
 Saida:
          110_7700_20250831_202508311558.zip            ------ 7700_20250831.txt
