# TKT148018 - FORD - OBI -  Fixes importação SAFXs - 701, 702, 325, 326 - SP25

## Descricao

O cliente Ford está enfrentando problemas ao efetuar a importação da SAFX44 via OBI. Quando a origem da carga é o OBI, a importação lê 0 registros. No entanto, se o mesmo arquivo for enviado via SFTP, a importação automática lê os registros normalmente. 
 

DEMANDA DIVIDIDA para entrega em grupos 

 

- Alteração apenas nos objetos de banco de importação, mesmo fluxo de entrada,  
- qualquer origem vai executar mesmos objetos logo teste de um fluxo somente é suficiente (Webservice, OBI, SFTP)
