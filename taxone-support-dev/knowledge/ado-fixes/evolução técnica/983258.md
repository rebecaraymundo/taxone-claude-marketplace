# TKT149931 - SECULUS-SEMEAR - ECF - ESCRITURAÇÃO CONTÁBIL FISCAL - ECF - ESCRITURAÇÃO CONTÁBIL FISCAL - Erro de formato invalido na Integração do arquivo MOV0001 do T1 para o Onesource.

## Descricao

Processamento no OS está acusando que o formato do arquivo txt está invalido, porém cliente efetuou a integração via TAXONE, ou seja, não manipulou, criou ou formatou o arquivo para integração diretamente na tela do Onesource. 

Ao verificar geraçao dos dados dos lançamentos gerados pelo cliente no T1, e baixar o TXT ue foi integrado para o Onesource, o formato está conforme manual de layout (UTF- 8 (SEM BOM)). 

 

Precisamos de apoio referente a esse caso, para entender por qual motivo, a integração destes dados está ocorrendo com erro no Onesource. 

 

OBS: Foi aberta a demanda 900897 para o time do ONESOURCE, que analisou e verificou que o problema está no T1.
