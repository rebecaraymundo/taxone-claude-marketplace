# [INTERDADOSV17]: Conexão via TaxOne

## Descricao

Eu como usuário(a) fiscal do Interdados V17 Gostaria que fosse realizada a chamada do produto Interface via módulo TaxOne, onde se faz necessário a criação de módulo para essa chamada. Para buscar a simplificação dos acessos na jornada do cliente. Abaixo a sugestão: 
 Funcionalidade Nova:  Agrupamento: Basicos Módulo: Interdados V17 *Descrição Manager: 

 

Acesso ao Integrador Oracle Interdados V17: Módulo de Extração e Parametrização da Interface com o ERP - Oracle Cloud Fusion 

Este módulo oferece funcionalidades avançadas para a extração e parametrização de dados, facilitando a integração entre o sistema ERP Oracle Cloud Fusion e o Tax One.
