# [CAMEL] - Rotina de Stop não esta funcioando corretamente.

## Descricao

Foi identificado que a rotina de Stop de rotas não esta funcionando corretamente. 
 Objetivo da rotina:
Parar uma rota especifica de um cliente, logo quando ele colocar um arquivo o mesmo não deve ser movimentado para o NAS ou para o SFTP.

Deve ativar uma rota que foi parada e a movimentação de arquivos deve voltar a acontecer.
