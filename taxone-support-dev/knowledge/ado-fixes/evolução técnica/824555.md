# TAX ONE - LIB_IMPORT - Processo Importação Habilitar Paralelismo  - SP11

## Descricao

Conforme incidente da NISSEI
   

- ajustar processo para fazer o paralelismo de cargas  
 Esta demanda tem por objetivo possibilitar a rotina de importacao via Servico ImporterSafx paralelizar as atividades tb por threads dentro do BD a partir de um parametro que contera a qtde de registros minimos que permitira esta divisao de tarefas.

O parametro esta definido como coluna na tabela prt_par_msaf.qtde_regs_safx_thread, seu valor default sera de 500.000 e quando estiver = 0, significa que a rotina NUNCA ira paralelizar as tarefas no BD. 
 Vale ressaltar que a paralelizacao que existe atualmente no BD continua mantendo suas premissas e atendem somente as tabelas que estao parametrizadas para multithread, ou seja, a paralelizacao NAO estara disponivel para todas as SAFXs, apenas as q ja atendiam os criterios de paralelizacao multithread.
