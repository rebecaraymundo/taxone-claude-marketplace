# [Automation][PROD02] - Substituição de URL para o padrão PROD02

## Descricao

Overview:  
  
 

Após a implantação do novo "shape" em prod02, é necessário ajustar as URLs do frontend de modo que este aponte para o padrão de rotas dos serviços disponibilizados no novo shape, i.e. subsituir oms-mastersaf-taxautomation por oms-mastersaf-taxautomation-02. 

 

 
  
Proposta inicial:  
  
Substituir as constantes oms-mastersaf-taxautomation por oms-mastersaf-taxautomation-02 no processo de geração do pacote para prod02, semelhante ao que é feito para os serviços T1.
