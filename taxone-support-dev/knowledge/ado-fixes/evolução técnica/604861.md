# TAXONE - T1ICMS - Incluir filtro de data na tela de Ajustes das NFs Vinculadas (Tipo 1)

## Descricao

Identificado pelo monitoramento interno erro de timeout na tela de Ajustes das NFs Vinculadas (Tipo 1) pelo volume de registros do cliente SANTA CRUZ. 
 O timeout ocorreu devido a necessidade de um COUNT para fazer a páginação na Grid. 
 Para tratar essa situação está sendo proposto duas restrições técnicas que possui impacto funcional: 
- Não carregar automaticamente os docs pra seleção ao abrir a tela 
- Limitar a busca a 1 mes no maximo;  Alinhado e validado com a Liliane Assaf - BA Estadual

- Avaliar índice na tabela 
- Revisar PL/SQL (abaixo)  Referências https://github.com/tr/taxone_modules_t1icms/blob/master/src/main/java/com/thomsonreuters/t1icms/repository/ItemApuracDiscrAjusteRepository.java
 
 PL/SQL X07 SELECT COUNT ( X07.COD_EMPRESA ) 
FROM ( 
    SELECT X07_DOCTO_FISCAL.COD_EMPRESA, X07_DOCTO_FISCAL.COD_ESTAB, X07_DOCTO_FISCAL.DATA_FISCAL, X07_DOCTO_FISCAL.MOVTO_E_S, X07_DOCTO_FISCAL.NORM_DEV, X07_DOCTO_FISCAL.IDENT_DOCTO, X07_DOCTO_FISCAL.IDENT_FIS_JUR, X07_DOCTO_FISCAL.NUM_DOCFIS, X07_DOCTO_FISCAL.SERIE_DOCFIS, X07_DOCTO_FISCAL.SUB_SERIE_DOCFIS, X04_PESSOA_FIS_JUR.IND_FIS_JUR, X04_PESSOA_FIS_JUR.COD_FIS_JUR 
    FROM X04_PESSOA_FIS_JUR, X07_DOCTO_FISCAL 
    WHERE X07_DOCTO_FISCAL.IDENT_FIS_JUR = X04_PESSOA_FIS_JUR.IDENT_FIS_JUR AND ? = X07_DOCTO_FISCAL.COD_EMPRESA AND X07_DOCTO_FISCAL.COD_ESTAB IN ( 
        SELECT CCC.COD_ESTAB 
        FROM ESTABELECIMENTO CCC 
        WHERE CCC.COD_EMPRESA = ? AND CCC.COD_ESTAB = ? 
        UNION ALL SELECT C.COD_ESTAB 
        FROM ICP_INSC_EST_CENTR C, ESTABELECIMENTO 
        WHERE NVL ( ESTABELECIMENTO.IND_APUR_ICMS, ? ) = ? AND C.COD_EMPRESA = ESTABELECIMENTO.COD_EMPRESA AND C.COD_ESTAB_CENTR = ESTABELECIMENTO.COD_ESTAB AND ESTABELECIMENTO.COD_EMPRESA = ? 
        UNION ALL SELECT CC.COD_ESTAB 
        FROM CENTRAL_ESCRITURA CC, ESTABELECIMENTO 
        WHERE NVL ( ESTABELECIMENTO.IND_APUR_ICMS, ? ) <> ? AND CC.COD_EMP_CENTRAL = ESTABELECIMENTO.COD_EMPRESA
