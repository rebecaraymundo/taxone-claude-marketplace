# TAXONE - OBI importer e consumer - Possibilitar o Direcionamento de Alguns Tenants para Novo Processo Importação - 354

## Descricao

Implementar pontos discutidos na reuniao com time da Evolução Técnica sobre o novo processo de importação de SAFXs (usando um novo componente) 
- Alterar tabela CONNECTION_CONFIG do schema TENANTS_CONNECTION para informar qual(is) tenant(s) será(ão) direcionado(s) para testar o novo processo de importação 
- Alterar OBI consumer para direcionar o processo de importação de lotes de SAFX para o novo processo de importação somente para o(s) tenant(s) configurado(s) e não mais enviar para a fila de importação   Resultado esperado: 
- testar o novo processo de importação para lotes oriundos do OBI apenas para algum(ns) tenant(s) pré-configurado(s). 
- não impactar o processo de importação atual para lotes oriundos do OBI para os demais tenants.
