# [HUGHES] TKT126690 - TAX ONE - SPED - EFD - ESCRITURAÇÃO FISCAL DIGITAL -  Seleção de perfil no meio magnético do Sped Fiscal

## Descricao

Ao inserir um código de perfil específico (161, 105) e gerar o sped, o sistema seleciona automaticamente o primeiro perfil da lista (230), independentemente do código inserido pelo cliente. Esse comportamento sugere um problema de filtragem ou na interface, que não está respondendo adequadamente às entradas do usuário. 

Realizei uma chamada com a cliente para acompanhar o processo, uma vez que o problema não se manifesta quando replicamos o procedimento aqui. Testei com uma empresa que não possui Inscrição Estadual Única e com uma que possui, considerando que ela está marcando essa opção, mas o problema não ocorreu em nossa base UAT. 

Como medida temporária, ela estava ciente da seleção manual do perfil desejado. No entanto, enfatizou a necessidade de uma solução permanente, visto que este problema já foi reportado anteriormente. 

Também realizei testes nos ambientes UAT e de produção, e o problema não foi observado nesses contextos. 

Anexei gravações das telas para facilitar o entendimento do problema relatado.

## Root Cause

Causa: Ao selecionar o perfil desejado a janela trás como seleção o item 1 do perfil como o selecionado ao invés do escolhido pelo usuário.

 Solução: Ordenação do conteúdo do perfil pelo campo cod_perfil.
