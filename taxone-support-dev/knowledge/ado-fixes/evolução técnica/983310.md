## Release Notes

Foi incluído um novo campo na tela de parametrização da Importação Automática.
Esse parâmetro serve para o usuário informar se deseja limpar as tabelas temporárias (SAFX) após a importação.
Nome do campo: Limpa Tabela
Os valores permitidos são:
- Sim: Apaga os registros importados e mantém os registros com erro na tabela safx. - Não: Mantém todos os registros na tabelas safx. - Mesmo com Erro: Esse é o Default do campo. Apaga todos os registros da tabela safx.
