# [COMPANHIA BRASILEIRA] TKT109261 - TAXONE - JOB SERVIDOR - Importação da SAFX07 - Campo SCP - DEV

## Descricao

**[COMPANHIA BRASILEIRA] TKT109261 - TAXONE - BÁSICOS - JOB SERVIDOR - Importação da SAFX07 não esta preenchendo o campo SCP mesmo estando preenchido no arquivo.** 

 

 

Identificamos que o Taxone não está gravando na tabela x07_docto_fiscal, a informação do código SCP informado na tabela temporária SAFX07. 

A informação já está presente na Safx07 antes do processo de carga/importação, porém ao consultarmos a nota, é possível averiguar que o campo não vem preenchido com a informação já presente na tabela temporária. No lugar o Taxone está abrindo uma aba de seleção manual do código SCP que está cadastrado na Safx2057. Este formato manual de seleção torna-se inviável frente à quantidade de notas que precisam ter o código SCP informado para declaração da Efd Contribuições. 

Em anexo seguem prints da interface demonstrando que o SCP já está no arquivo antes da importação, print do taxone demostrando que no mesmo o campo não foi preenchido e a própria interface para análise e testes se necessário.
