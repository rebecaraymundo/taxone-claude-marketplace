# [DW/TAX ONE] Construtor de Relatórios necessita de Criação de FK na DWT_ITENS_MERC e X326_ORIG_ICMS_MONO  (DEV/QA) ** Hot Fix S05 **

## Descricao

Para atendimento ao Construtor de Relatorios precisamos criar FK nas seguintes tabelas: 
- FK_SAF_3480: FK na tabela X326_ORIG_ICMS_MONO campo IDENT_UF_ORIG referenciando campo IDENT_ESTADO da tabela ESTADO 
- FK_SAF_3481: FK na tabela DWT_ITENS_MERC campo IDENT_PRODUTO referenciando campo IDENT_PRODUTO da tabela X2013_PRODUTO;
  
 

Conforme alinhado com Alcestes é necessário criar Indices IDX_DWT_ITENS_MERC_PRODUTO e IDX_X326_ORIG_ICMS_MONO_ESTADO  

 CREATE INDEX IDX_DWT_ITENS_MERC_PRODUTO 

    ON DWT_ITENS_MERC (IDENT_PRODUTO) INVISIBLE; 

  

CREATE INDEX IDX_X326_ORIG_ICMS_MONO_ESTADO

    ON X326_ORIG_ICMS_MONO (IDENT_UF_ORIG) INVISIBLE;
