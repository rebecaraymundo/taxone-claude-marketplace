## Descricao

Objetivo: Atendimento a versão 1.19 para o Ressarcimento de SC, que vale a partir de Janeiro de 2026. Para isso: 
- Atualizar TFIX 67 A 76 com a estrutura p/ essa versão;
 
- Disponibilizar a versão 1.19 no Perfil;
