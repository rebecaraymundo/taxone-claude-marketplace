# [Internal Bug] T1 - Específicos - Combustíveis E Derivados De Petróleo - A pesquisa da grid Estabelecimento não esta funcionando 

## Passos de Reproducao

URL do ambiente de testes utilizado: https://qa.onesourcetax.com/
  
 
 Steps: 
- Acessar o agrupamento -> Específicos 
- Acessar o módulo -> Combustíveis e Derivados de Petróleo 
- Ir no Menu -> Movimento de Refinaria -> Geração dos Anexos VII e VII-M 
- Preencher o campo Mes/Ano Referencia -> 10/2023 
- Marcar o checkbox Inscrição Estadual Única 
- Preencher o campo Pesquisar UF -> GO - Goias 
- Clica no botão Pesquisar onde se encontra uma lupa na grid estabelecimento 
- Clicar no botão Critério 
- Clicar em editar  
- Preencher a coluna A Cod Estab||' N' com o valor -> OS2800_N 
- Clicar em Pesquisar  
Resultado encontrado: O sistema não esta realizando o filtro na grid 
 

Resultado esperado: O sistema deve filtrar a grid de acordo com o filtro aplicado
 

Evidência da falha:
 

 

 Testes que falharam:    
Geracao Anexo VII e VII-M - Empresa RSA2 - DIE GLOCKE ELETROMAGNETICS
