# [Data Warehouse]: DW/T1 - Ajuste de execução simultânea do DATAMART - Lock de banco (QA)

## Descricao

Eu como usuário do Tax One gostaria que houvesse um controle das execuções da equalização Data Mart de forma a impedir que mais de um processo seja executado para mesma empresa, estabelecimento e período.    Funcionalidade Existente:  Agrupamento: Básicos Módulo: Data Warehouse Menu: DATA MART > Equalização/Limpeza E Módulo: Ferramentas Menu: Rotinas Acessórias > Equalização/Limpeza

## Release Notes

Informações Carta Patch - Produtos: DW e TAXONE 
  
 Objetivo: Implementamos um controle nas execuções da equalização Data Mart, localizada nos módulos Data Warehouse e Ferramentas, para impedir a execução simultânea de processos para a mesma empresa, estabelecimento e período, evitando problemas de lock em tabelas no Banco de Dados.
 
  Agrupamento: Básicos Módulo: Data Warehouse Menu: DATA MART > Equalização/Limpeza e Módulo: Ferramentas Menu: Rotinas Acessórias > Equalização/Limpeza
