# [Internal Bug] T1 - Estadual - Meio Magnético - Produto não é localizado por conta do usuário não preencher campos porém nenhum pop-up indica ação necessária

## Passos de Reproducao

Credenciais de acesso para reprodução da falha:
      
- Tennant / Database: T1 QA (tenant VO4)   
- Usuário | Senha: acesso próprio 
- Empresa: 076 
- Estabelecimento: login Todos, ao acessar tela pode ser qualquer um, porém utilizamos o '00' 
- Período: independe  
 Pré-requisito(s): (optional)
 
 Steps: 
- Acessar Meio Magnético sem logar com estabelecimento 
- Navegar até o menu: Parâmetros >> Parâmetros de Tributos - Registro 75 (Sintegra) 
- Preencha o campo de COD_PRODUTO  
- PRODUTO 60 POSIÇÕES 0123456789012345678901234567890123456789
 
- Após Tab, dados do produto não são trazidos  
- Preencha o campo GRUPO_PRODUTO  
- 02 
- Após seleção, campo COD_PRODUTO será limpado
  
- Preencha novamente o campo de COD_PRODUTO  
- PRODUTO 60 POSIÇÕES 0123456789012345678901234567890123456789
 
- Após Tab, dados do produto não são trazidos  
- Preencha o campo IND_PRODUTO 
- Acabado: '01' no banco 
- Imediatamente após selecionar a opção '01', dados do produto são trazidos
