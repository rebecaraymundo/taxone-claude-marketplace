# [IDT]: DW / T1 - Básicos - Ferramentas - Consulta Pré-Vendas (DEV)

## Descricao

Eu como usuário(a) fiscal do ONESOURCE Tax One Gostaria que fosse criado um framework que consulte detalhes de documentário fiscal Para estimativa de nosso time de pré-vendas de motor de cálculo (IDT) via aplicação 
 Funcionalidade Nova:  Agrupamento: Básicos > Ferramentas Módulo: Relatório Menu: IDT Detalhes Documentos 
 Detalhes Adicionais 
- Hoje é executada uma query paliativamente que busca informações de tabelas temporária DWT 
- O framework deve buscar informações das tabelas definitivas X07, X08, X09 
- Normalmente o período de consulta é de 1 ano, por este motivo essa query e consulta deve ser performática 
- Liberaremos no produto, mas sem carta release, pois a finalidade é estimativa do time de pré-vendas 
- Tela de Geração: 
- Empresa: Todas ou Uma a Uma 
- Estabelecimento: Todos ou Um a Um 
- Período: DD/MM/AAAA à DD/MM/AAAA (com base na data fiscal)  
- Query hoje utilizada, deve ser considerado que: 
- Seja performática a consulta no frame via aplicação 
- A consulta é sempre 1 ano para estimativa 
- Os dados da query estão voltados a tabelas DWT, que dependem do DataMart, o que não é algo que devemos replicar no frame. Os tabelas e campos devem ser convertidos em X07, X08 e X09...
