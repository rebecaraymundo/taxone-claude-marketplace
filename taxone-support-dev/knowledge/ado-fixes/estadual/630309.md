# 93901 -  TAX ONE - ESTADUAL - Menu DATA MART - EMISSÃO EM EXCEL DOS LIVROS APURAÇÃO/ENTRADA/SAÍDA - Quando não existe dados no período deverá gerar o arquivo com a informação "SEM MOVIMENTO".

## Descricao

Tratamento do Período SEM MOVIMENTO:
 

1) Livros P1; P2; P9 devem gerar o Arquivo EXCEL sem movimento, quando não existe movimentação no período.    

A MFS627745 passou a gravar o arquivo Excel com os dados do estabelecimento para o período sem movimento. 

https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/627745. 

A MFS630309 continuará o tratamento, para incluir o texto "SEM MOVIMENTO" numa das colunas do excel. 

 

2) Os livros P1, P2 e P9 gerados em Arquivo PDF, já foram ajustados para tratar o período SEM MOVIMENTO pela MFS (https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/625795) e https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/627745.
 

 

 

-- 

Texto original da abertura da Demanda pelo Suporte: 

Boa tarde,  O livro registro de entradas modelo P1 não está gerando dados do mês 02/2024, favor verificar. 

Os 02 estabelecimento 11154 e 11155 não tiveram movimentos no mês 02/2024, então no processo de geração deveria aparecer 01 PDF (1 página) "SEM MOVIMENTO" para cada estabelecimento. 

Fiz o processo de geração e emissão no módulo = Estadual / ICMS / Datamart / Emissão Livro de Entradas Modelo P1 /Ajuste SINIEF (Relatório do Servidor) e não está gerando o PDF.

## Release Notes

Carta do Patch – TAXONE APENAS: 

  

Módulo: Estadual >> ICMS 

  

Resumo da alteração: Tratativa dos Livros de Entrada, Saída e Apuração do ICMS para
períodos sem movimento. 

  

No caso de não existir movimento para o estabelecimento e período apurado, os relatórios emitidos em PDF apresentam o texto "SEM MOVIMENTO" no cabeçalho.  

Já para relatórios emitidos em Excel, foi criada uma coluna nomeada como "PERIODO_COM_MOVIMENTO", e será preenchida com SIM ou NÃO.
