# TKT137551 - TAM - ONESOURCE TAX ONE - ESTADUAL - CIAP - Alteração DIFAL SP p/ fornecedores do Simples Nacional (QA)

## Descricao

FUNCIONALIDADE: Geração das Aquisições do CIAP 

 

Localização: 

Módulo: Estadual > Ativo Permanente 

Menu: Movimento > Geração das Aquisições a partir das NF’s  

 

 

Descrição: 
 
Ajustar o Cálculo do Diferencial de Alíquota para SP, quando o fornecedor é do Simples Nacional. 

 

               Base de Cálculo = Valor Contábil do Item / (1 – Alíquota Interestadual) 

               Valor do Débito = Base de Cálculo * Alíquota Interna 

               Valor do Crédito = Base de Cálculo * Alíquota Interestadual 
 ------------------------------------------------------------------------------------------------------------
 Descrição do Suporte: 

 

Verificamos que atualmente o sistema está realizando o cálculo do Diferencial de Alíquotas (DIFAL) de forma divergente da prevista na legislação de SP, nas operações com fornecedores optantes do Simples Nacional. 

 

Situação reportada: UF de SP 

 

 

 
Ajustar o Cálculo do Diferencial de Alíquota para SP, quando o fornecedor é do Simples Nacional. 

 

               Base de
Cálculo = Valor Contábil do Item / (1 – Alíquota Interestadual) 

               Valor do
Débito = Base de Cálculo * Alíquota Interna 

               Valor do
Crédito = Base de Cálculo * Alíquota Interestadual

## Release Notes

Carta do Patch - DW e TAX ONE:

Módulo: Estadual > Ativo Permanente 

Menu: Movimento > Geração
das Aquisições a partir das NF’s  

                                       

Nesta demanda será liberada alteração da regra do cálculo do campo “Valor
diferencial de Alíquota” para tratar o cálculo do Diferencial de Alíquota em
operações realizadas por contribuintes optantes pelo Simples Nacional, para os
estabelecimentos que pertençam ao estado de São Paulo. 

O parâmetro 92
(Parâmetros por UF -> Módulo ICMS), para a UF de São Paulo (SP), em
operações de optantes pelo Simples Nacional, somente a opção (“Valor
Contábil do Item”) é válida para a emissão do DIFAL Base Dupla. 

Para a Recuperação das alíquotas (Interna e
Interestadual), caso as mesmas não estejam informadas no Item da
nota, Necessário parametrização das Alíquotas no módulo
Data Warehouse, na opção "Alíquota por CFOP e
Produto" ou "Alíquota Padrão". 

  

Módulos Impactados: Ativo
Permanente.
