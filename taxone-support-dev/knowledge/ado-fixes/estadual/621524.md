# DW/TAXONE - ESTADUAL > Controle de Mercadorias Remessa/Retorno - Relatório de Pendência de Mercadorias em Poder de Terceiros por Estabelecimento- Incluir período em Data de Remessa - QA parte 1

## Descricao

O cliente Chevron, migrou do DW para o TaxOne e trouxe toda a sua base.  Quando o cliente vai gerar o relatório de Pendência de Mercadorias em Poder de Terceiros por Estabelecimento, a tela de geração possui apenas o campo Data de Remessa até:  e recupera todas as notas fiscais com mais de 20 anos.  O cliente gostaria de emitir o relatório somente a partir de determinado período. 
Como sugestão pode-se alterar o processo para considerar um período inicial e final para a Data de Remessa.
