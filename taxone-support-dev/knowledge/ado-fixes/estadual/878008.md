# INFOLEGIS 182/25 - Z - 122 - iSIMP - Orientações (REQ, DEV, QA)

## Descricao

O
projeto SIMP – Sistema de Informações de Movimentação de Produtos – tem por
objetivo instrumentalizar os processos regulatórios da ANP, monitorando, de
forma integrada, os dados de produção e movimentação de produtos na cadeia do
downstream; permitindo assim o acompanhamento do mercado; a disponibilização de
estatísticas para a sociedade e o apoio técnico ao Ministério de Minas e
Energia (MME) e ao Conselho Nacional de Política Energética (CNPE).  

Consiste na obrigatoriedade de entrega à ANP,
de um arquivo eletrônico, utilizado para o envio de dados sobre as atividades
dos agentes regulados informantes, relativos a um exclusivo e determinado mês
de referência. 

A ANP disponibiliza a Tabela de Códigos de
Apoio – 01/08/2025. 

A
nova versão da Tabela está disponibilizada no site:  

https://csa.anp.gov.br/downloads/manuais-isimp/tabela-apoio-isimp-vigente.zip 

Também
foram disponibilizadas orientações para envio de carga SIMP através do sistema
DPP (Válido a partir de 04/08/2025): 

- Como solicitar acesso ao sistema DPP para envio de
movimentações SIMP? 
- Tabelas de apoio ao I-SIMP (Planilha Dinâmica) 
- Manual para consulta das atualizações mensais nas planilhas
dinâmicas nas tabelas de apoio à carga SIMP no DPP 
- Orientações básicas para acessar uma carga SIMP no sistema DPP 
- Manual para envio de movimentações através do arquivo
".txt" 
- Manual para envio de movimentações através do arquivo
".xml" 
- Baixe o modelo de planilha ".xlsx" para envio de
movimentações pelo sistema DPP 
- Baixe o exemplo de planilha ".xlsx" preenchido para
envio de movimentações pelo sistema DPP 
- Workshop realizado em 16/07/2025 
- FAQ - Perguntas Frequentes da carga SIMP no DPP
