# [HUGHES] TKT131966 - TAX ONE - ESTADUAL - ICMS - Convênio ICMS 115 - hash code não está reconhecendo

## Descricao

TKT131966 - HUGHES - TAX ONE - ESTADUAL - ICMS - Convênio ICMS 115 - Importação do hash code 

Após importar o hash code o sistema não está reconhecendo e dá a Mensagem de erro: "Arquivo de Controle relativo ao arquivo Mestre não identificado no diretorio informado" 

 

 

 
Ocorre erro: 
Código do erro: 101086 

Mensagem de erro: "Arquivo de Controle relativo ao arquivo Mestre não identificado no diretorio informado"

## Solucao

SaaS

## Release Notes

Carta para DW e TAX
ONE:  

  

Módulo: ICMS 

 

Informamos que foi realizada a correção no processo de importação do Hash Code no módulo ICMS, contemplando a funcionalidade "Importação Hash Code e Conferência do Registro de Saídas". 

A correção foi aplicada no seguinte caminho do sistema:
 

Menu: ICMS >> Data Mart >> Convênio ICMS 115 >> Importação Hash Code e Conferência do Registro de Saídas 

 

Módulos Impactados:
ICMS
