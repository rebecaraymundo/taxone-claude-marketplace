# Criação de Trigger para eliminar a TFIX97 (QA)

## Descricao

Criação de uma trigger para carregar a tabela CAT_LAYOUT_HIST, a partir da CAT_LAYOUT_IMP.  Dessa forma a TFIX97 poderá ser eliminada, mantendo apenas a TFIX96. 
  TFIX96 - carrega a CAT_LAYOUT_IMP TFIX97 - carrega a CAT_LAYOUT_HIST

  DESCRICAO 
  DESCRICAO 
 
 
  NOME_CAMPO 
  NOME_CAMPO

  MFS_ALTERA 
  MFS_ORIGEM 
 
 
  DSC_COMENTARIO 
  DSC_COMENTARIO 
 
 
  IND_ALT

    Domínio de valores:

    - A/C: Alteração de campo em SAFX;

    - A/D: Alteração de campo Descrição;

    - A/T: Alteração de campo Tamanho;

    - I/C: Inclusão de campo em SAFX;

    - C/S: Criação de SAFX

    - C/E: Crítica de Erros 
    
 
 Essas tabelas podem ser conferidas a partir dos seguintes relatórios: 
- Relatório do Manual de Layout  
            Localização: Menu Básicos à Job Servidor 

                                   Menu Carga à Manual de Layout SAFX 
            Esse relatório é montado a partir da TFIX96, tabela cat_layout_imp. 
- 
Relatório de Alterações de SAFX 

Localização: Menu Básicos à Job Servidor 
                      Menu Carga à Alterações de SAFX  
Esse relatório é montado a partir da TFIX97, tabela cat_layout_hist.
