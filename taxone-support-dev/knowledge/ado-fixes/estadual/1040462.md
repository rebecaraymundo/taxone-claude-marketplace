# [T1] Solução Emergencial - Relatório de Detalhamento por Produto - Geração – Report Fiscal / Report Server (QA)

## Descricao

Emergencial para solução na Geração do Relatório de Detalhamento por Produto (Devido a alta volumetria de dados): 
 Report Fiscal > Gerenciais > Documentos Fiscais> Detalhamento por produto > Geração
 
 
 O Relatório de Detalhamento por ProdutoNão estava sendo gerado para o cliente em uma falha de performance durante a etapa de Geração. O problema ocorreu quando a geração foi acionada para um período de quatro dias, envolvendo simultaneamente 91 estabelecimentos.  
  
Solução Implementada 

Para contornar o problema e garantir a disponibilização das informações ao cliente, adotou-se a estratégia de disponibilizar os arquivos gerados em quebras por estabelecimento, no formato Excel, diretamente na etapa de geração. 

Com essa alteração: 

- O processo não realizará mais a população da tabela utilizada na etapa de emissão; 
- Consequentemente, o cliente não utilizará mais o processo de emissão, passando a obter o relatório exclusivamente via geração, já com os arquivos segmentados.
