# TAXONE - Job Servidor - Nova Tela do Relatório de Log da Importação - Remoção do Menu Antigo (REQ/DEV/QA)

## Descricao

Módulo Job Servidor
 O objetivo dessa MFS é retirar o menu original (tela ofensora) do TAXONE:
  - Menu: Controles >> Relatórios >> Relatório por Processo >> Importação
   
 A MFS-635958 criou a nova tela e disponibilizou-a no menu:   - Menu: Controles >> Relatórios >> Relatório por Processo >> Importação (Nova Tela)
 https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/635958
 
 Com a retirada da tela original, o menu da nova tela deve ser renomeado para:   Menu: Controles >> Relatórios >> Relatório por Processo >> Importação  
 Alterar manual operacional do módulo no Taxone para retirar a opção original. 
 A alteração não pode refletir no MasterSAF! só TAXONE!

## Release Notes

Carta do Patch – TAXONE
APENAS: 

 
 

Módulo: Básicos – Job Servidor 

  

 

Resumo da alteração: : 

Visando melhorar a performance e usabilidade do nosso produto em maio/2024, disponibilizamos uma nova tela para emissão dos Relatórios dos Processos de Importação. 

 

Menu: Controles > Relatórios > Relatórios por Processo > Importação (Nova Tela)
 

 

   Com isso estamos removendo a tela original, deixando apenas a nova opção, que passa a ser acessada em:

 
Menu: Controles > Relatórios > Relatórios por Processo > Importação 
   
 

 Maiores detalhes consultar
o Manual Operacional no WebHelp.
