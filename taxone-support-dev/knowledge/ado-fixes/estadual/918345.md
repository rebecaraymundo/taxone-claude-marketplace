# TKT146119 - INX - ONESOURCE TAX ONE - ESTADUAL - ICMS - Ajuste no cálculo do difal para as aquisições do Simples Nacional para a UF de SP. Regra para os livros 108 E 165 (QA)

## Descricao

Olá pessoal, 

Após constantes questionamentos dos clientes referente a rotina de cálculo do difal de uso/consumo e ativo imobilizado quando aquisições de fornecedores optantes pelo simples nacional, retornamos uma solicitação de analise dos decretos e legislações para o time tributário. 

 

Abaixo o retorno solicitando uma adequação no cálculo, solicitamos apoio para avaliação da regra, atualmente a rotina do cálculo atual tem impactado a geração do difal de um modo geral.

## Release Notes

Informações para a Carta do
Patch DW e TAX One:

Módulo: ICMS

Resumo da Alteração: Ajuste nas regras do cálculo do Débito do
Diferencial de Alíquota (DIFAL) para contemplar o tratamento do cálculo da Base
Dupla especificamente para o estado de São Paulo (SP), em operações realizadas
por contribuintes optantes pelo Simples Nacional. 

  

Cálculo
a Ser Considerado para o Simples Nacional - SP: 

  

 
- Base de Cálculo
     Crédito (Considerada para o cálculo do Valor de Crédito)

     Valor Contábil do Item / (1 – Alíquota Interestadual)  
  

 
- Valor do Crédito:

     Base de Cálculo Crédito * Alíquota Interestadual  
  

 
- Base
     de Cálculo Débito (Considerada para o cálculo do Valor de Débito)

     Valor Contábil do Item / (1 – Alíquota
     Interna)  
  

 
- Valor do Débito:

     Base de Cálculo Débito * Alíquota Interna  
  

 
- Valor Cálculado = Valor do Débito –
     Valor do Crédito  
  

 
- Valor Diferencial
     Alíquota = Valor Calculado
