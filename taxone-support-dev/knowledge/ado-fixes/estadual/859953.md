# TKT137551 - TAM - ONESOURCE TAX ONE - ESTADUAL - ICMS - Apuração do ICMS (livro 108) - Alteração DIFAL SP p/ fornecedores do Simples Nacional (QA)

## Descricao

FUNCIONALIDADE: Apuração do ICMS (livro 108) 

 

Localização : 

Menu Estadual, Módulo: ICMS 

Mestresaf: 

Menu DATA MART > Apuração do ICMS > Ajuste SINIEF  

 

          DATA MART > Apuração do ICMS > Ajuste SINIEF – Multiempresa
           

          DATA MART > Apuração do ICMS > Geração 

Táxon: 

          DATA MART > Apuração do ICMS > Geração 

 

Descrição : 
 
Ajustar o Cálculo do Diferencial de Alíquota para SP, quando o fornecedor do Simples Nacional. 

 

               Base de Cálculo = Valor Contábil do Item / (1 – Alíquota Interestadual) 

               Valor do Débito = Base de Cálculo * Alíquota Interna 

               Valor do Crédito = Base de Cálculo * Alíquota Interestadual 
 ------------------------------------------------------------------------------------------------------------
 Descrição do Suporte: 
Verificamos que atualmente o sistema está realizando o cálculo do Diferencial de Alíquotas (DIFAL) de forma divergente da previsão na legislação de SP, nas operações com fornecedores optantes do Simples Nacional. 

 

Situação reportada: UF de SP

SaaS

## Release Notes

Informações para a Carta do
Patch DW e TAX One: 

Módulo: ICMS

Resumo da Alteração: Ajuste nas regras do cálculo do Diferencial de
Alíquota (DIFAL) para contemplar o tratamento do cálculo da Base Dupla
especificamente para o estado de São Paulo (SP), em operações realizadas por
contribuintes optantes pelo Simples Nacional. 

  

Descrição
das Alterações: 

Para atendimento ao Cálculo
do DIFAL com Base Dupla para SP opreações com optantes do Simples Nacional,
atentar-se aos seguintes pontos: 

1) Parametrização p/ UF
Módulo Estadual>> ICMS Menu: Manutenção >> Parâmetros p/UF Marcar o
parâmetro 92 e definir a opção “Valor Contábil do Item” (campo 64) ". 

2) Para a Recuperação das
alíquotas (Interna e Interestadual), caso as mesmas não estejam informadas
no Item da nota, Necessário parametrização das
Alíquotas no módulo Data Warehouse, na opção "Alíquota por CFOP
e Produto" ou "Alíquota Padrão". 

3) Parametrização p/
Lançamentos Automáticos Módulo Estadual>> ICMS Menu: Manutenção >>
