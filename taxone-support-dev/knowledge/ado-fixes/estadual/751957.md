# [REPORT SERVER] [ICMS] [P1, P2, P9] INFOLEGIS 2242/24 - CNPJ Alfanumérico - Alteração na Máscara E Validação do CNPJ (QA)

## Descricao

Objetivo:  

Alteração em massa nos
objetos Report Server da máscara do CNPJ para contemplar caracteres
alfanuméricos. 

Alteração na função de validação
do CNPJ para considerar alfanumérico. 

A função SAF_VALIDA_CGC responsável pela validação
Digitos Verificadores do CNPJ, com CNPJs alfanuméricos oficialmente
válidos.  

Essa função foi alterada
pela https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/741454 para
contemplar CNPJ numérico e alfanumérico.         
                     
                     
                     
                     
                   
   

Usar o gerador de CNPJ: 
   

https://toolshub.com.br/gerador-cnpj-filial 

https://geradorbrasil.com/gerador-de-cnpj-alfanumerico/ 

  

Levantamento Report Server
realizado em  20/01/2025: REPORT_SERVER_CGC_CNPJ.xlsx.
