## Release Notes

Carta do Patch – TAXONE APENAS: 

 

 

Módulo: Estadual >> Ativo Permanente
  
 
 

  

Resumo da alteração: Melhoria no tratamento dos relatórios dos Saldos a Creditar e Saldos a Creditar II via Report Server, para período sem movimento. 

Opções: 
 

Movimento >> Saldos a
Creditar - Report Server 

Movimento >> Saldos a
Creditar II - Report Server 

 

 

 

No caso de não existir movimento para o estabelecimento e período apurado, os relatórios emitidos em PDF apresentaram o texto "SEM MOVIMENTO" no cabeçalho. Já os emitidos em Excel, a primeira coluna "PERIODO_COM_MOVIMENTO" será preenchida com "não".
 

 

 
Informamos que os relatórios originais continuam disponíveis, mas recomendamos
o uso das novas opções, visto que em breve os relatórios originais serão
removidos do módulo.
