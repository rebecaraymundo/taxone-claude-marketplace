# TAXONE - Relatórios - Report Fiscal - Remoção dos relatórios atendidos pelo Servidor de Relatórios - liberados em novembro - QA

## Descricao

DEV: MFS594326 QA: MFS617351 
 Conforme orientação do PM, os relatórios do TaxOne que foram substituídos pelo Servidor de Relatórios devem ser removidos a partir de 1 mês após a sua liberação.
  
 Em 28/11/2023 foram liberados no módulo do Report Fiscal para substituir os relatórios abaixo que devem ser removidos: 
 - Gerenciais >> Documentos Fiscais >> Analíticos
     >> Mercadoria - Gerenciais >> Documentos Fiscais >> Analíticos
     >> Detalhamento por Produto (Via Servidor) - Gerenciais >> Estoque >> Movimentação
     >> Movimento do Estoque 
 REMOÇÃO APENAS NO TAXONE - DW SE MANTÉM! 
 O desenvolvedor, além de remover as opções acima, deverá: 
- Verificar se após a criação dos relatórios via Report Server, os relatórios originais sofreram alguma atualização que não realizada na opção do Repor Server; 
- Caso tenha ocorrido, fazer a alteração na opção do Report Server;  O tester deverá: 
- Certificar que os relatórios originais se mantém no DW; 
- No Tax ONE, certificar que os relatórios do Repor Server, correspontentes aos antigos que estão sendo removidos, estão funcionando; 
- Certificar que os relatórios antigos foram removidos do Tax One.  Observação Importante:   O relatório do Detalhamento do Produto (opção Servidor de Relatórios) está sendo atualizado para incluir informações de Monfasia, através  das MFS's:  https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/611483 e https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/610751 
 Portanto o relatório "Gerenciais >> Documentos Fiscais >> Analíticos >> Detalhamento por Produto (Via Servidor)" só pode ser removido após a entrega dessas duas MFS's.
