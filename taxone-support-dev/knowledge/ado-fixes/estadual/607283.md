# [SHOPEE] Registro 1400 - Código 1202 - UF Alagoas e Código PI004 - UF Piauí (ENTRADAS)

## Descricao

Essa demanda foi aberta para incluir os códigos 1202 (AL) e PI004 (PI) na Regra de Geração do Registro 1400, conforme solicitação do cliente SHOPEE, para Serviço de Transporte. 
 
- 1202 - TRANSPORTE (RODOVIÁRIO, 
FERROVIÁRIO, AÉREO, FLUVIAL E 
MARÍTMO) 
- PI004 - Prestadores de serviço de transporte rodoviário intermunicipal e interestadual de passageiros e de cargas
  Obs.:  A princípio o tratamento será feito para as operações de entrada, atendendo a solicitação do cliente. 
 O tratamento para operações de Saída será feito na MFS-607284.

## Root Cause

Causa: Caso o usuário cadastre mais de dois itens de movimentação econômica na janela a mesma "pagina" a informação porém não é possível trocar da primeira página. 

 Solução: Refazer o layout da datawindow e aumenta-la para que a API de paginação do T1 possa funcionar corretamente.
