# [IPIRANGA] TAX ONE - BÁSICOS - JOB SERVIDOR - Criação de SAFX339 - Parametrização de CNAE x CNAE ANP - I-SIMP (QA)

## Descricao

Ramo: MFS786236 REQ 786234 DEV 786235,
910413
 QA 786236 
 
 Eu como usuário(a) fiscal do ramo de Combustíveis. 
 Gostaria que fosse disponibilizada uma SAFX para que consiga carregar a parametrização de CNAE x CNAE ANP. Hoje o Tax One disponibiliza uma tela de manutenção para imputar essa informação, no módulo Específicos >> I-SIMP, Menu Parâmetros >> CNAE x CNAE ANP. Mas como são muitos registros para serem incluídos manualmente, a inserção manual se torna inviável.  
 Funcionalidade Existente:  Agrupamento: Básicos Módulo: Job Servidor Menu: Carga > Execução
Importação > Execução 

Importação Batch > Execução 
             Exportação Dados > Execução 
  Detalhes Adicionais 
 
Criação da SAFX e disponibilizar as funcionalidades de Carga, Importação Normal e Batch e Exportação. 
 

 
  
 

TFIX30 (tabela PRT_MODULOS_MSAF) 

 

DexPara CNAE: 

TACES01 (tabela ATIV_ECONOMICA) 

TFIX119 (PRT_CNAE_OBRIG) 

DExPara (PRT_CNAE_MSAF_OBRIG) 

 
 Ver nova orientação passada pelo Time de Evolução Técnica 01/08/25

TODAS as atuais e novas SAFXs deverao possuir um campo virtual denominado chave_safx<respectiva numeracao> e um indice denominado IX_CHAVE_SAFX<respectiva numeracao>.

Exemplo: SAFX07 deverá ter um campo chamado chave_safx07 e um indice denominado IX_CHAVE_SAFX07(chave_safx07).

Resumo da alteração :  Disponibilizamos a nova interface SAFX339, que
permite realizar a importação da
Parametrização que correlaciona o Código Nacional de Atividade Econômica (CNAE)
com um código de Atividade Econômica específico de uma Obrigação.. 

Essa
parametrização é utilizada no atendimento ao i-SIMP cuja tela de manutenção está
disponível no menu Parâmetro >> CNAE x CNAE ANP.
