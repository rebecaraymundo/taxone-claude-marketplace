# TKT168001 - MRV - ONESOURCE TAX ONE - MUNICIPAL - São Leopoldo - Campo valor tributável esta sendo gerado erroneamente

## Descricao

O cliente MRV possui um estabelecimento em Porto Alegre, mas possui inscrição eventual para São Leopoldo, onde está realizando a geração do meio magnético. O cliente informou que o campo "Valor Tributável" está sendo gerado de forma incorreta. Atualmente, a regra de preenchimento desse campo está definida da seguinte forma: 

 

 

 

Quando o cliente possui preenchimento no campo "VLR TRIBUTO ISS", como no caso da MRV, esse valor é automaticamente populado no campo "Valor Tributável" do meio magnético. 

 

No entanto, ao consultar o manual de layout do município, é exemplificado que o campo "Valor Tributável" corresponde ao cálculo: Base de Cálculo - Valor de Dedução = Valor Tributável. 

 

 

 

 

Poderiam, por gentileza, avaliar o preenchimento deste campo com base no exemplo disponibilizado pelo município? 

 

Pois como o cliente deve preencher o campo Valor tributo ISS no documento fiscal, a regra atual não atende a geração do campo corretamente. 

 

 

Obs: Anteriormente o campo havia sido tratado através da demanda https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/911545, onde após a liberação, estava sendo gerado corretamente.
