# INFOLEGIS 725/24 - B - ISSQN - BATATAIS/SP - Migra Sistema (BETHA) - DW/T1

## Descricao

Consiste na migração do sistema  “SIAPPA”, para novo Sistema Web “Livro Eletrônico Fly” na Prefeitura de Batatais/SP, para gerenciamento do ISSQN, a partir de 23 de janeiro de 2023, conforme informações disponibilizadas no Portal da Prefeitura: https://www.batatais.sp.gov.br/servicos/empresa/nota-fiscal-eletronica-fly-enota-para-empresas

 O acesso ao Livro Eletrônico para realizar a declaração dos serviços tomados está disponível no endereço eletrônico: https://e-gov.betha.com.br/livroeletronico2/02022-038/login.faces?lastUrl=/selecaodemodulo.faces
 
 Cumpre observar que a informação do layout a ser utilizado foi disponibilizado pelo cliente, não estando disponível no Portal da Prefeitura. Conforme contato anterior com a empresa Betha, fornecedora do sistema “Livro Eletrônico Fly”, a Betha disponibiliza mais de um formato de layout do Livro Eletrônico, de acordo com o contrato com a Prefeitura. A Betha não divulga informações sobre qual layout a Prefeitura utiliza.
 
 Tentamos contato com a Prefeitura de Batatais/SP (E-mail: financas@batatais.sp.gov.br e Telefone: 16 3660-3400), porém não tivemos nenhum retorno.
