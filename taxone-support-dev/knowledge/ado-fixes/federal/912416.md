# INFOLEGIS  512/25 - A - ISSQN - GRAMADO/RS - Migra Sistema (Atende Net)

## Descricao

A
Prefeitura de Gramado/RS alterou o sistema “BETHA” para o Sistema “Atende Net”,
que contempla a entrega de informações ao fisco municipal, via Internet, sobre
os serviços tomados, a partir de 29 de março de 2024. 

O novo sistema está disponível no endereço: 

https://nfse-gramado.atende.net/autoatendimento/servicos/nfse 

  

Cumpre observar que a informação sobre o layout a ser utilizado deverá
ser disponibilizado pelo cliente. O acesso ao sistema e manuais é restrito ao
contribuinte com usuário e senha.
