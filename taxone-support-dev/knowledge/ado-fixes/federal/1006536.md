# TKT162743 - MRV - ONESOURCE TAX ONE - MUNICIPAL - SIGISS - Taubaté Reavaliação de regra para o campo Alíquota do Simples Nacional para buscar do campo Alíquota ISS Retido

## Descricao

Estamos trabalhando com o municipal de Taubaté e quando o cliente foi validar o meio magnético, se deparou com erro de alíquota. 

Pelo print do erro e regra do manual, se o Prestador de Serviço for Optante do Simples Nacional informar a Alíquota do Simples Nacional - Padrão Nacional, decimal separado por vírgula. (obrigatório). 

Trata-se de uma nota que tem retenção. 

Obs: A solicitação da MRV é para a regra considerar também o campo de aliquota retido, porque a MRV não preenche o campo de aliquota normal.
