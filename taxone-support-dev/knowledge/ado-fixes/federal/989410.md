# TKT160721 - DIMED - TAX ONE - MUNICIPAL - CAÇADOR - Tags <ValorIssRetido> <BaseCalculo> e <Aliquota> não estão sendo gerados

## Descricao

TKT160721 - DIMED - TAX ONE - MUNICIPAL - CAÇADOR - Campos <ValorIssRetido> <BaseCalculo> e <Aliquota> não estão sendo gerados 

 

O cliente está realizando a geração do arquivo e, para as notas com retenção de ISS, os campos <ValorIssRetido>, <BaseCalculo> e <Aliquota> não estão sendo preenchidos. 

De acordo com a regra, esses campos devem buscar inicialmente os valores de: Valor Base ISS, Valor ISS e Alíquota ISS. 

Quando esses campos não estiverem preenchidos, o sistema deve então buscar as informações em: Valor Base ISS Retido, Alíquota ISS Retido e Valor ISS Retido, desde que o campo <IssRetido> seja igual a '1'. 

Porém, mesmo atendendo a essas condições, os campos não estão sendo gerados conforme a regra. 

 

Regra validada: MTZ_DIR_Geracao_do_Meio_Magnetico.docx
