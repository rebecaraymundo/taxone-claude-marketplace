# [TAX ONE] - Básicos - Data Warehouse - Relatório de notas duplicadas 

## Descricao

Eu como usuário(a) fiscal do Relatório de Notas Fiscais Duplicadas
 Gostaria que o relatório fosse refaturado via servidor de relatórios (report Server) Para que o usuário pudesse ter uma melhor experiência no que tange performance 
 Funcionalidade Existente:  Agrupamento: Básicos Módulo:  Data Warehouse Menu: Relatórios > Conferências > Documentos Duplicados 
 Detalhes Adicionais 
- Demanda surgiu oriunda de problemas de performance com clientes Klabin e GE, refaturamento o relatório que era gerado em tela para framework, e com isso limitamos o período há 1 mês para geração. A geração ocorre, mas a critica dos clientes é que 1 mês não é suficiente, com isso precisamos redirecionar ao report server. 
- Abaixo relato do cliente Klabin:    Cliente reclama que após uma atualização de patch o relatório de notas duplicadas não está permitindo gerar o período maior que 1 mês, por exemplo de 01/05/2024 À 19/06/2024.

## Root Cause

Bom dia !

Conforme verificado não foi possível reproduzir na base Ocloud (VO4)

## Release Notes

Informações Carta Patch - Produtos: TAXONE 
 Ajuste 
 Objetivo: Migramos para o servidor de relatório o relatório 'Documentos Duplicados' do modulo Data WareHouse. Desta forma, haverá uma melhoria na performance do relatório. 
  Agrupamento: Básicos Módulo: Data Warehouse Menu: Relatórios > Conferências > Documentos Duplicados
