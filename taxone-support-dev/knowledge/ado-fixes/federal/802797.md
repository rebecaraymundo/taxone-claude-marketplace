# INFOLEGIS 871/25 - B - ISSQN - BOTUCATU/SP - Migra Sistema (FIORILLI)

## Descricao

A Prefeitura de Botucatu/SP
alterou o sistema “SIGEP” para o Sistema “Fiorilli”, que contempla a entrega de
informações ao fisco municipal, via Internet, sobre os serviços tomados.
Conforme informações por telefone (14-3811 1467), o novo sistema entrou em
vigor a partir de maio de 2024. 

O novo sistema está disponível no endereço: 

http://siabotucatu.dcfiorilli.com.br:8089/issweb/home 

  

Cumpre observar que a informação sobre a alteração e layout a ser
utilizado foi disponibilizado pelo cliente. O acesso ao sistema e manuais é
restrito ao contribuinte com usuário e senha.
