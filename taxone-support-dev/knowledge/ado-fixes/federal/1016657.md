Serviços
Tomados Município de Paranaguá - PR. 

  

 Na geração do
arquivo municipal eventual de Paranaguá, identificamos que o sistema está
preenchendo o campo de situação tributária do ISS com o código 14 (NTRIB) – Não
Tributada. 

  

Contudo, de acordo com
o layout definido pela Prefeitura de Paranaguá, o código correto a ser
informado para esse cenário deveria ser o código 01 (TIRF) – Tributada
Integralmente com Imposto Sobre Serviços Retido na Fonte (ISSRF). 

  

Importante salientar
que essa geração trata-se inscrição eventual. 

  

Abaixo evidenciaremos o
processos realizados, juntamente com a Especificação de Paranaguá – PR.

 Especificação:
 
 
Após análise identificamos
que o problema está na regra de negócio sistêmica do TaxOne,
especificamente na RN35-R (Regra de Situação Tributária para Notas
Recebidas). 

  

Regra correta para
Paranaguá: 

Para Notas Recebidas
com ISS Retido: 

   

SE:   

- Campo IND_TP_RET =
"2" (ISS Retido)   

- Campo VLR_ISS_RETIDO
> 0   

- Local da prestação =
Paranaguá 

 ENTÃO: cod_sit_tributária = 1 (TIRF)   

  

Seguindo a mesma
lógica da RN35-R1 (municípios como Cascavel, Pinhais, etc). 

 

Necessitando assim de
uma verificação técnica de ajuste de regra.
