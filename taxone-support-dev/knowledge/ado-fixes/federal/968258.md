POUSO ALEGRE/MG - Migração
 

 

A Prefeitura de Pouso Alegre/MG alterou o sistema “SIGISS” para o Sistema “Atende Net” da IPM Sistemas, para gerenciamento do ISSQN, a partir de 14 de outubro de 2025. 

  

O novo sistema está disponível no endereço: 

https://nfse-pousoalegre.atende.net/ 

  

 

Adicionalmente, o hub de atualização com manuais, vídeos e webinários pode ser consultado em: https://pousoalegre.atende.net/subportal/atualizacao-do-sistema-fiscal-web 

 
NOVO HAMBURGO - Migração
  
A Prefeitura de Novo Hamburgo/RS alterou o sistema “ISS NET” para o Sistema “Atende Net” da IPM Sistemas, para gerenciamento do ISSQN, a partir de 04 de março de 2023.  

   

O novo sistema está disponível no endereço:  

https://nfse-novohamburgo.atende.net/autoatendimento/servicos/nfse  

   

Adicionalmente, o hub de atualização com manuais, vídeos e webinários pode ser consultado em: https://novohamburgo.atende.net/subportal/mudanca-sistema-nota-fiscal-eletronica
