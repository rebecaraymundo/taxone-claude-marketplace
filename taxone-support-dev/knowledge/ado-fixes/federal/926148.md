# DW/T1 - INFOLEGIS 712/25 - B - ISSQN - CATANDUVA/SP - Migra Sistema (RLZ)

## Descricao

A
Prefeitura de Catanduva/SP divulgou a migração do sistema “ISS Web” para o
sistema “RLZ”, utilizado para Gerenciamento Eletrônico do ISSQN, a partir de 22
de abril de 2025. 

O
Novo Sistema de Emissão de Nota Fiscal Eletrônica e Escrituração do ISSQN no
Portal do Cidadão Online tem as seguintes funcionalidades: 

- Emissão de
NFS-e; 
- Escrituração
de serviços tomados; 
- Escrituração
de prestadores de outro município; 
- Geração de
guias do ISSQN.  
  

O novo sistema está disponível no endereço: https://cidadaoonline.catanduva.sp.gov.br/app/pages/login-nfe 

Cumpre
observar que a informação sobre o layout a ser utilizado deverá ser
disponibilizado pelo cliente. O acesso ao sistema e manuais é restrito ao
contribuinte com usuário e senha.

  O novo sistema está disponível no endereço: https://cidadaoonline.catanduva.sp.gov.br/app/pages/login-nfe
