# TKT169519 - EATON - ONESOURCE TAX ONE - MUNICIPAL - CAXIAS DO SUL Erro ao importar o meio magnético no validador, somente um item é inserido

## Descricao

O cliente está realizando a escrituração de documentos fiscais referentes a serviços tomados. Alguns desses documentos possuem mais de um item com o mesmo código de serviço. O lançamento dos documentos fiscais é feito dessa forma para a contabilização das despesas. 

 

No entanto, ao inserir manualmente o mesmo código de serviço no documento fiscal, o sistema Tax One não permite essa inserção. Por outro lado, ao importar os documentos fiscais por meio do SAFX, é possível realizar o lançamento com vários itens utilizando o mesmo código de serviço. 

 

 

 

Ao verificar o arquivo de meio magnético gerado, observa-se que ele cria uma linha para cada item com o mesmo código de serviço. Porém, ao submeter o arquivo no validador, apenas o primeiro item do documento fiscal é considerado. Além disso, ao tentar reimportar o meio magnético, o arquivo é rejeitado e os demais itens não são incluídos. 

 

 

 

 

 

 

 

 

 

Como o documento fiscal possui múltiplos itens com o mesmo código de serviço, o cliente espera que os valores desses itens sejam somados automaticamente, conforme ocorre em outros municípios. 

 

Contudo, ao consultar o manual de layout do município e regra de geração do meio magnético, não há nenhuma orientação sobre como a estrutura do arquivo deve ser gerada: se deve conter apenas um item consolidado ou se os itens devem ser somados. 

 

Poderiam avaliar, como podemos tratar essa situação?
