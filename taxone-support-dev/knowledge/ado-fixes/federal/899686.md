# TKT149866 - MRV - TAX ONE - MUNICIPAL - ISS WEB Cecam -  Menu “Arquivo de Entradas de Serviços (Const. Civil/Utilities/Telecom)” não existe para o município de Santana de Parnaíba

## Descricao

TKT149866 - MRV - TAX ONE - MUNICIPAL - ISSQN WEB -  Menu “Arquivo de Entradas de Serviços (Const. Civil/Utilities/Telecom)” não existe para o município de Santana de Parnaíba 

 

O estabelecimento gerador está localizado no município de São Paulo, mas possui Inscrição Municipal em Santana de Parnaíba. Nesse caso, o arquivo deve ser gerado no módulo de Santana de Parnaíba, pelo menu “Arquivo de Entradas de Serviços (Const. Civil/Utilities/Telecom)”. No entanto, esse menu não está disponível no módulo de Santana de Parnaíba 

 

 

 

 

 

É necessário incluir o menu para possibilitar a geração do arquivo de um estabelecimento de outro município que possua Inscrição Municipal em Santana de Parnaíba.
