# INFOLEGIS 1411/24 - C - ISSQN - BARCARENA/PA (ISS Net_Nota Control) - Migra Sistema - Ajuste de Regra

## Descricao

A Prefeitura de Barcarena/PA alterou o Sistema de Emissão de Notas Fiscais Eletrônicas de Serviço e Declaração de Serviços, para a plataforma ISS. Net Online – Nota Control
O novo sistema está disponível no endereço:
https://www.issnetonline.com.br/barcarena/Online/Login/Login.aspx?ReturnUrl=%2fbarcarena%2fonline%2f

O manual com layout está disponível no endereço:
https://www.notacontrol.com.br/static-issnetonline/AjudaExterno/Docs/OnlineExternoOficial.pdf

Em contato por telefone (91 99230-3908) com Juliane da Prefeitura de Barcarena/PA foi informado que as NFSe dos Prestadores do próprio município, não devem ser declaradas, visto que são importadas automaticamente pelo sistema no momento da geração da mesma pelo Prestador no sistema ISS Net Online – Nota Control. Também foi informado que o sistema foi alterado a partir de janeiro/2023.
 Observações Importantes:Conforme análise da demanda e INFOLEGIS 1411/24 - C - ISSQN - BARCARENA/PA (ISS Net_Nota Control), não se trata de uma alteração de layout e sim um ajuste de regra! Pois no requisito pois existem duas regras de tamanho campo para tipo de documento desse layout, um com 9 posições e outra com 20. Cliente está querendo informar o tipo de documento com 20 posições. 
 
Vou ajustar a demanda 705567 para regra.

## Release Notes
