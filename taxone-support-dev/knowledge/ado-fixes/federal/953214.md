# [MRV] TKT155979 - TAX ONE - MUNICIPAL - SIAP.NET - JACAREI -  Geração Eventual - Erro ao validar arquivo

## Descricao

TKT155979 - MRV - TAX ONE - MUNICIPAL - SIAP.NET - JACAREI - Geração Eventual - Erro ao validar arquivo 

O arquivo que é gerado pelo Tax One, com aspas, está ocasionando erro na validação do arquivo. No caso deste cliente - MRV, que faz a geração eventual, o estabelecimento pertence a município de São José dos Campos - SP, e está gerando pelo módulo de Jacarei - SP. 

No Manual de Layout do município não há orientações sobre este ponto, no entanto, o cliente fez o contato com a prefeitura que confirmou as aspas indevidas (consulta em anexo) 

 

Realizamos um acesso remoto, segue link demonstrando a importação e o erro: AGENDA TICKET 155979 - MUNICIPAL JACAREI - ERRO LAYOUT IMPORTAÇÃO-20251021_153227-Gravação de Reunião.mp4 

O arquivo com aspas foi importado no validador, e ocorreu erro: 

 

"Quantidade de colunas não está de acordo com a documentação, verifique se os textos estão entre aspas (ex: "Texto com quebra de linha ou uso de , ou ; deverá estar entre aspas") ou se não existe algum delimitador ( , ou ; ) "sobrando." 

 

 

 

Quando o arquivo corrigido sem as aspas foi importado, não ocorreu mais erro: 

 

 

 

Na regra, temos a informação na geração do arquivo de Jacarei, que o formato é CSV, e o separador de campos é a vírgula
