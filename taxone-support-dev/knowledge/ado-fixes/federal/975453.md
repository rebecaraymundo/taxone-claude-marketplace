# TKT158541 - MRV - ONESOURCE TAX ONE - MUNICIPAL - NOTA JOSEENSE - SÃO JOSE DOS CAMPOS/CAMPINAS e TERESINA

## Descricao

Validador: NOTA JOSEENSE 

1) Município do estabelecimento: São Jose dos Campos e Campinas. 

Avaliar regra campo 39 Motivo da não retenção e 41 Exigibilidade do ISS quando existe situações onde o estabelecimento recebe 2 notas do mesmo fornecedor porem uma NF a retenção é dentro do município e a outra NF é fora do município. 

Foram analisadas as seguintes Notas Fiscais: 

- NF 4468: Local da prestação de serviço: São José dos Campos e Campinas ( município do estabelecimento) 
- NF 4469: Local da prestação de serviço: Taubaté  
Ambas as notas possuem o cadastro do município como Campinas (Cadastro SAFXX04). Ao parametrizar para o código de serviço 0702, o parâmetro se aplica às duas notas, independentemente do local da prestação. 

Regra do Campo 39 - Motivo da não retenção - 'A’ (Não Incidência no Município): 

 

 

Regra do Campo 40 - exigibilidade - '2’ (Não Incidência): 

 

Hoje os campos gera incorreto para uma NF, exemplo: 

 
2) Município do estabelecimento: Teresina
 
 Análise Suporte 
 Durante a validação dos arquivos magnéticos gerados para o Município de Teresina, foi identificado que o sistema está gerando registros de notas fiscais duplicados. A análise dos arquivos demonstrou que, para notas fiscais que possuem múltiplos itens com o mesmo código de serviço, o sistema está criando uma linha para cada item, sem consolidar os valores. 
 Incluir o município de Teresina na regra especifica: 
  
 
Por favor avaliar.
