# [AGCO] TKT150251 - TAX ONE - MUNICIPAL - DMSTE - CANOAS - Avaliar a regra para não considerar notas fiscais quando o prestador é no mesmo município do estabelecimento.

## Descricao

Validador: DMSTE-e 

Município: Canoas 

Avaliar a regra para não considerar notas fiscais quando o prestador é no mesmo município do estabelecimento ( Canoas) 

Conforme informado pelo time de Base Tributária as as operações registradas em NFS-e serão lançadas automaticamente na Declaração Mensal de Serviços Eletrônica (DMS-e), módulo integrante do Sistema da NFS-e. 

 

 

 

Na regra de negocio não consta regra especifica para o município de Canoas. 

 

 

 

Hoje as notas ficais do mesmo município do estabelecimento está sendo considerado no arquivo, existe a parametrização de Notas Fiscais Eletrônicas, porem não funciona para esse município. 

Avaliar o município e o validador para considerar essa parametrização.
