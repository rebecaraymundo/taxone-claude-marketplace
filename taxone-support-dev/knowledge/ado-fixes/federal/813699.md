# [EFD REINF] DW/T1 - REINF – Ajustes tabela STG_REINF_OUT

## Descricao

Eu como usuária do modulo EFD REINF, gostaria que ajustasse a tabela; 

 

1) STG_EVENTOS_REINF_OUT e incluir o valor default 0 na coluna LIDO. 

  

ALTER TABLE STG_EVENTOS_REINF_OUT ALTER
COLUMN LIDO SET DEFAULT '0'; 

  

É necessário também um update nessa tabela para todos os tenants
para mudar caso algum valor já esteja salvo na tabela como NULL. 

  

UPDATE STG_EVENTOS_REINF_OUT  SET
LIDO = '0' WHERE LIDO IS NULL; 

 

2)  Inclusão de um índice na coluna
"LIDO" da tabela STG_EVENTOS_REINF_OUT  

 

Demanda do Onesource: User Story 818366: INTEGRADOR -
Ajuste nas queries da tabela STG_EVENTOS_REINF_OUT 

 

 

Essa necessidade veio de encontro com o atendimento a
Delloite/Meli, onde foi identificado um ajuste que melhorou significativamente
a performance deles para envio de eventos ao Onesource. (Equipe Onesource)
