# INFOLEGIS 874/25 - D - ISSQN - CABO FRIO/RJ - Migra Sistema (ISS DIGITAL_TG Sistemas)

## Descricao

A
Prefeitura de Cabo Frio/RJ substituiu o sistema “SIGISS” pelo antigo sistema já
utilizado pela prefeitura “ISS Digital” da “TG Sistemas” para gestão do
Imposto Sobre Serviços (ISSQN), a partir de 20 de fevereiro de 2025. 

A Secretaria de Fazenda criou uma seção no portal com documentações, manuais e
vídeos explicativos para facilitar na transição dos sistemas acessando o link https://fazenda.cabofrio.rj.gov.br/manuais-iss-nfs-e/
