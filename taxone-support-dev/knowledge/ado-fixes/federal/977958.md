# TKT160073 - MRV - ONESOURCE TAX ONE - MUNICIPAL - DES-E - Incluir a opção de geração do meio magnético - Arquivo de Entradas de Serviços (Construção Civil/Utilities/Telecon)

## Descricao

Título: Ajuste no Menu de Obrigação para Geração do Meio Magnético – Inscrição Municipal Eventual (São Gonçalo) 

Descrição do Problema: 

O cliente reportou que, no menu "Obrigação" do sistema, não está disponível a opção de geração do meio magnético – Arquivo de Entradas de Serviços (Construção Civil/Utilities/Telecom) para inscrições municipais eventuais, especificamente para o município de São Gonçalo. 

Contexto: 

- Cliente possui inscrição municipal eventual para São Gonçalo. 
- Ao tentar gerar o meio magnético, não aparece a opção correspondente para inscrição municipal eventual no caminho: Obrigações > Arquivo de Entradas de Serviços (Const. Civil/Utilities/Telecom). 
- No validador anterior, a opção estava disponível, conforme evidência apresentada pelo cliente.  
Solicitação: 

Solicitamos análise e ajuste no sistema para que a opção de geração do meio magnético – Arquivo de Entradas de Serviços (Construção Civil/Utilities/Telecom) seja disponibilizada para inscrições municipais eventuais.
