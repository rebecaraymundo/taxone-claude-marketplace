# INFOLEGIS 1408/25 - B - ISSQN - ITATINGA/SP - Migra Sistema (ISS ELETRÔNICO CECAM) - DW/T1

## Descricao

A Prefeitura de Itatinga/SP substituiu o sistema “GEISWEB” pelo novo sistema “ISS Eletrônico” da “CECAM” para gestão do Imposto Sobre Serviços (ISSQN), a partir de 06 de novembro de 2023.

 O novo sistema está disponível no endereço:
 https://itatinga-prefeitura.cecam.com.br/issweb/login/login.aspx 
 
 Cumpre observar que a informação sobre o layout a ser utilizado deverá ser disponibilizado pelo cliente. O acesso ao sistema e manuais é restrito ao contribuinte com usuário e senha.
