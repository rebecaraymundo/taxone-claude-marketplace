# DW/T1 - INFOLEGIS 460/25 - B - ISSQN - LAGOA SANTA/MG - Migra Sistema (BETHA)

## Descricao

A Prefeitura de Lagoa Santa/MG divulgou a migração do “NFe Cidades” para o sistema “E-Nota” da “Betha”, utilizado como novo Sistema Tributário Municipal, a partir de 07 de outubro de 2025.

 Para escrituração dos serviços tomados é utilizado o Livro Eletrônico.
 O novo sistema e o layout do arquivo estão disponíveis no endereço:
 https://nota-eletronica.betha.cloud/#/inicio
