# TKT175667 - MRV - ONESOURCE TAX ONE - MUNICIPAL - GISS ONLINE V2 - CAMPO DOS GOYTACAZES CAMPO CÓDIGO DO MUNICIPIO ESTA SENDO GERADO COM O CÓDIGO DO IBGE, INCORRETAMENTE

## Descricao

Ao gerar o meio magnético para o município de Campos dos Goytacazes, a tag CódigoMunicipio está sendo gerada incorretamente. De acordo com o manual de layout, este campo deve ser preenchido com o código do IBGE referente ao local de prestação do serviço, que, neste caso, é Campos dos Goytacazes. Contudo, o sistema está gerando o campo com o código do IBGE do município do prestador dos serviços.  

 

 

Empresa / Estabelecimento: 

 

 

 

 

 

 

 

Documento fiscal: 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Cadastro PF/PJ 

 

 

 

 

 

Ao gerar o meio magnético, o campo código do municipio esta sendo gerado com o Código do IBGE de Belo Horizonte ao invés de Campo dos Goytacazes 

 

 

 

Ao submeter o arquivo no validador, ocasionou o seguinte erro: 

 

 

 

 

 

Regra atual para Geração do campo: 

 

 

 

 

 

 

 

Gostaria de solicitar que seja avaliado a geração do campo, para que seja gerado com o código IBGE do local de prestação dos serviços, conforme manual de layout.
