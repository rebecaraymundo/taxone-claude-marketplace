# TKT161274 - TAX ONE - MUNICIPAL - NFSE GIAP PRESCON - Cotia - Reavaliação da regra para geração eventual não considerar as notas do mesmo município

## Descricao

Solicitação Suporte 
 Para o validador NFSE - GIAP PRESCON, município Cotia, 

O cliente está gerando o arquivo como eventual (Arquivo de Entradas de Serviços (Const. Civil/Utilities/Telecom) e nele estão sendo consideradas notas em que o prestador é do mesmo município de geração do arquivo, o que não é correto. 

Neste caso, para gerações eventuais, em que o cadastro do estabelecimento é outro município, a geração não atende a regra para desconsiderar os documentos em que o Prestador está no mesmo município em que o tomador. 

A solicitação é para uma reavaliação da regra de requisitos.

## Release Notes

Informações p/a Carta do Patch: DW e TAX ONE 

Ajuste de regras 

Objetivo: Ajustado o processo de geração para o município de Cotia, do validador NFSE GIAP PRESCON, para que sejam desconsideradas as notas fiscais eletrônicas emitidas por prestadores estabelecidos dentro do município da inscrição eventual.  

  

Layout Padrão: XML 

Módulo Impactado 

Módulo: Municipal > Cotia/SP 

Menu: Geração 

 

- Geração do Arquivo de Entradas de Serviços (Const. Civil/Utilities/Telecom)
