# [Raizen] - T1 - Básicos - Report Fiscal -  Gerenciais -> Documentos Fiscais -> Analíticos -> Relatório de Apoio à Apuração de Serviços DOCNUM+Multiempresa

## Descricao

Eu como usuário(a) fiscal do Relatório de Apoio à Apuração de Serviços do Tax One Gostaria que fosse realizada melhoria do relatório considerando a inclusão do novo campo DOCNUM e a geração para multiempresas. Para facilitar a analise em detalhe do usuário  
 Funcionalidade Existente 
 Agrupamento: Básicos Módulo: Report Fiscal Menu: Gerenciais -> Documentos Fiscais -> Analíticos -> Relatório de Apoio à Apuração de Serviços   
 Detalhes Adicionais:  
- Campo a ser adicionado  DOCNUM  -   Numero de Controle (X07 - NUM_CONTROLE_DOCTO) 
- Geração para todas as empresas

## Release Notes

Informações Carta Patch - Produto: TAXONE 

Melhoria 

Objetivo: Ajustes na tela de geração do Relatório de Apoio à
Apuração de Serviços, incluído a opção Geração Multiempresa, alteração do
campo Estabelecimento, passando para Empresa/Estabelecimento, inclusão do campo
DOCNUM na tela de parâmetro, para geração no Relatório de Apoio à Apuração de
Serviços, com a finalidade de auxiliar  na conferência das notas fiscais.
