# TKT163566 - MRV - ONESOURCE TAX ONE - MUNICIPAL -  DES XML - ITAPEVA -  Realiação de regra para geração do campo Base de Cálculo no meio magnético.

## Descricao

Para o municipal de Itapeva, o validador exige o preenchimento de valor de base de cálculo, mesmo em casos em que não tem alíquota e valor de ISS retido na nota, pelo fato da retenção ter ocorrido pelo prestador. 

Se não informado a Base de cálculo, ocorre erro no validador, impedindo que a nota seja importada na prefeitura. 

 

O cliente realizou testes no validador, pegando a mesma informação do campo "ValorNota" e colocando no campo "BaseCalculo" e o registro foi validado com sucesso. 

 

Foi solicitado a abertura do chamado, para avaliar a possibilidade de quando não tiver Base/Alíquota/Imposto, ser preenchida a mesma informação que consta no campo valor total. 

 

Encaminho as evidências para análise.
