# [REINF] - REINF - Painel - Painel de Controle de Eventos - Incluir mensagem informativa

## Descricao

Eu como usuário(a) fiscal do REINF. Gostaria que fosse incluso uma mensagem informativa no Painel de Controle de Eventos, afim de orientar o cliente, para que quando se tratar da geração e o envio do XML  de grandes volumes, deve ser utilizado o Menu REINF > Envio de Eventos - Grandes Volumes. 
 Essa demanda foi criada, afim de orientar o cliente na forma de geração dos eventos, devido a erro de duplicidade relatado pelo cliente Universal.  
 Funcionalidade Existente:  Agrupamento: SPED Módulo: EFD - REINF Menu: REINF > Painel > Painel de Controle de Eventos
   Detalhes Adicionais
 
 Analise foi feita pela demanda https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/621217.
