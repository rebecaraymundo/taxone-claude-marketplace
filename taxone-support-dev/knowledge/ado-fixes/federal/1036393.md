# TKT171834 - MRV - ONESOURCE TAX ONE - MUNICIPAL - ISS WEB - ITAQUAQUECETUBA-  Arquivo de Entradas de Serviços (Const. Civil / Utilities/Telecom) Falha na Consolidação de Itens no Arquivo Magnético de Itaquaquecetuba

## Descricao

Durante a validação dos arquivos magnéticos gerados para o Município de Itaquaquecetuba, foi identificado que o sistema está gerando registros de notas fiscais duplicados quando a nota tem mais de um item.﻿ 

 

 

A análise dos arquivos demonstrou que, para notas fiscais que possuem múltiplos itens com o mesmo código de serviço, o sistema está criando uma linha para cada item, sem consolidar os valores. 

 

Em nossa regra atualmente não consta nada mencionando que deve agrupar : 

 

 

Porém cliente informa que esta incorreto e causando erros na validação. 

Manual de layout em anexo. 

 

Cliente não tem mais de um cadastro para o código de serviço utilizado  :
