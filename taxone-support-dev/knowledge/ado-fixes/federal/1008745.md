# TKT166705/165309 - TAX ONE - MUNICIPAL - - ISS WEB - ITAQUAQUECETUBA - Inclusão na regra dos campos Regime Simples e Versão do Layout para Itaquaquecetuba e Campo alíquota gerando de forma incorreta

## Descricao

1) Análise Suporte - Inclusão do Campo "Regime Simples" 
 A solicitação é para alteração da regra de requisitos, para inclusão do município de Itaquaquecetuba na  regra do campo "Versão Layout" com geração como 03, e inclusão do município na regra de geração do campo "Regime Simples" que atualmente não é gerado. 
 Conforme manual de Layout, a versão do layout deve ser gerada como 03, e deve ser gerado o campo "Regime Simples". 
 Realizamos testes junto a MRV no validador da Prefeitura, e incluindo manualmente a versão do layout e o campo Regime Simples, o arquivo foi validado sem erros.
 
 Todos os testes realizados estão na aba "Analysis". 
 
 2) Análise Suporte - Erro Campo Alíquota, formato errado 
 Geração eventual para Itaquaquecetuba: 
 Este município utiliza o validador ISS WEB.  
 O formato da alíquota no arquivo está gerando incorretamente. Para exemplo de uma nota com alíquota de 5%, Conforme regra de requisitos, para Itaquaquecetuba: deveria gerar 00500, e está gerando 50. 
 Foi efetuado testes no validador da prefeitura e quando informado no arquivo a alíquota de 00500, o arquivo é validado com sucesso (Todos os testes efetuados estão na  aba "Analysis"). 
 Obs: A geração para Itaquaquecetuba é eventual (menu: Arquivo de Entradas de Serviços (Const. Civil / Utilities/Telecom))
