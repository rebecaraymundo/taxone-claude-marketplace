# INFOLEGIS 188/26 - E - ISSQN - CAMPO GRANDE/MS - Migra Sistema (DSF)

## Descricao

A
Prefeitura de Campo Grande/MS substituiu o sistema “DMS” pelo novo sistema da “DSF
– Desenvolvimento de Sistemas Fiscais” para gestão do Imposto Sobre Serviços
(ISSQN), a partir de 15 de dezembro de 2025. 

  

O novo sistema está
disponível no endereço: 

https://nfse.campogrande.ms.gov.br/notafiscal/paginas/portal/index.html#/ 

  

Cumpre
observar que a informação sobre o layout a ser utilizado deverá ser
disponibilizado pelo cliente. O acesso ao sistema e manuais é restrito ao
contribuinte com usuário e senha.
