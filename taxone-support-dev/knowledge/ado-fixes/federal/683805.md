# [Server Report]: Report Fiscal - Obrigações Acessórias > IPI > Relatório de Documentos Fiscais de Saída > Emissão

## Descricao

Eu como usuário(a) fiscal dos Relatórios do TaxOne. Gostaria que fosse viabilizado no Servidor de Relatórios o desenvolvimento do Relatório IPI sobre o Relatório de Documentos Fiscais de Saída, deve ser realizado em excel e pdf. 
 Para facilitar a visualização do usuário. 
 Funcionalidade Nova:  
 Agrupamento: Básicos Módulo: Report Fiscal Menu: Obrigações Acessórias > IPI > Relatório de Documentos Fiscais de Saída Detalhes Adicionais:  
- Detalhes com equipe de Evolução Técnica sobre os pontos de desenvolvimento (jasper, boas práticas desenv.) 
- Deve-se levar em consideração o nível de complexidade da construção (1sem - 1mês) 
- Deve ser viabilizado nos formatos em excel + pdf

## Release Notes

Informações Carta Patch - Produtos: TAXONE 
 Ajuste 
 Objetivo: Visando uma melhor performance, migramos o Relatório de Documentos Fiscais de Saída > Emissão para a funcionalidade de servidor de relatórios, do produto TAXONE. A partir desta demanda, será possível continuar gerando o relatório no formato PDF e agora também em formato Excel.  
  Agrupamento: Básicos Módulo: Report Fiscal Menu: Obrigações Acessórias > IPI >> Relatório de Documentos Fiscais de Saída > Emissão
