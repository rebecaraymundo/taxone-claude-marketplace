# TKT175665 - MRV - ONESOURCE TAX ONE - MUNICIPAL - GISS ONLINE V2 - CAMPO DOS GOYTACAZES CAMPO ALIQUOTA ESTA SENDO GERADA INCORRETAMENTE

## Descricao

Ao gerar o meio magnético para o município de Campo dos Goytacazes o campo alíquota esta sendo gerado incorretamente, em desacordo com o manual de layout: 

 

 

 

Geração do campo Alíquota no meio magnético: 

 

 

 

Ao submeter o arquivo no validador da Giss Online, ocorre o seguinte erro: 

 

 

 

Ao consultar a Regra de Geração do campo, encontrei a seguinte regra: 

 

O município de Campo dos Goytacazes, não esta abrangido para ser gerado no formato conforme indicado no manual de layout.

## Root Cause

O sistema possui uma regra de formatação específica para o campo **Alíquota** no arquivo de meio magnético do GISS Online, porém o município de **Campos dos Goytacazes (RJ)** não estava incluído nessa regra. Por isso, o campo era gerado em um formato diferente do exigido pelo validador da prefeitura local, fazendo com que o arquivo fosse rejeitado. A ausência do código de identificação do município na lógica de formatação condicional é a causa do problema.
