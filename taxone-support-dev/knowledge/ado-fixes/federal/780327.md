# EFD REINF – Arquivos JCP –  Gerando um novo registro como 'Original' e não como 'Retificadora' - Quando o Status foi igual 4

## Descricao

Essa demanda é originaria da demanda https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/777857, que foi liberada como emergencial para cliente Klabin. 

 

Porem há necessidade de aplicar para todos os cliente! Pois trata de um problema legado
