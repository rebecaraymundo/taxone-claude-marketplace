# TKT164337 - MRV - ONESOURCE TAX ONE - MUNICIPAL -  ATENDENET - NOVO HAMBURGO -  Reavaliação de regra para o campo valor tributável quando tem redução de base de cálculo.

## Descricao

Liria, conforme solicitado no chamado 153839, estou registrando esse novo chamado para que avaliem o campo valor tributável, quando tem redução de base de cálculo, conforme evidência enviada pelo cliente. 

As notas enviadas como exemplo são 000464125 e 000464144.  

Nota: Todos os municípios que utilizam o mesmo validador, precisa de tratamento para o campo em questão. 

  

Pedimos ao cliente verificar a inclusão de uma regra na Interface para trazer os campos já corretos na SAFX09. Mas o cliente não aceitou: 

"No cliente quando 2 campos são preenchidos, e algum momento foi identificado que tem validadores pegam informações de um campo ou de outro. Por gentileza, verifique se é possível incluir na regra para quando tiver valor de dedução preenchido, considerar o valor total para esse validador no momento de gerar o meio magnético. Não podemos deixar zerado o campo de Base, uma vez que ele está preenchido na origem."
