# TKT173304 -  TAX ONE - MUNICIPAL - SIMPLISS - BLUMENAU "Inconsistência na Geração do Código de Natureza de Operação para geração Arquivo de Entradas de Serviços (Const. Civil/Utilities/Telecom)"

## Descricao

Conforme conversado com a Rosemeire abro essa demanda. 

Encaminho para avaliação do time de Produto a solicitação realizada pela cliente Maria Eunice Fernandes, referente ao comportamento do campo cd_natureza_operacao na geração do arquivo magnético para o município de Blumenau. 

Durante o projeto em andamento, foi identificado que, para notas sem retenção, o sistema tem preenchido o campo Código da Natureza de Operação com o valor “2”, conforme regra atual que considera as informações do campo COD_MUNIC_MSAF (SAFX2097) associado ao COD_MUNIC_ISS (SAFX07). 

Entretanto, conforme evidências apresentadas pela cliente, o validador municipal não aceita o valor “2” para operações Eventuais, ainda que o campo “Tipo Retenção” venha automaticamente preenchido pela interface como “Retido pelo Prestador” — comportamento padrão para outros municípios, sem causar inconsistências. 

 

Em teste interno, foi possível identificar que: 

- Quando o campo Tipo Retenção está preenchido como “Retido pelo Prestador”, o sistema gera o cd_natureza_operacao = 2. 
- Quando o campo Tipo Retenção não está preenchido, o sistema gera o código = 1.  
 

 

A cliente solicita ajuste na regra, alegando que: 

- O município de Blumenau exige “1” para natureza de operação quando a prestação e origem são Blumenau, mesmo em notas Eventuais. 
- O valor “2” gerado atualmente ocasiona erro no validador, inviabilizando o envio. 
- O campo de retenção não pode ser removido da interface, pois é preenchido automaticamente e não gera problemas em outros municípios já homologados. 
- Trata-se de uma exceção específica para casos de empresas Eventuais, para os quais a regra atual não contempla adequadamente. 
- Dada a maturidade do projeto, não é viável alterar a interface; sendo necessária uma condição adicional via regra de negócio.
