# TKT160880 - MRV - TAX ONE - MUNICIPAL - ISS.WEB - Criação de regra para o município de MOGI DAS CRUZES-SP/ CARIACICA-ES - para não gerar notas de prestador do mesmo município

## Descricao

TKT160880 - MRV - TAX ONE - MUNICIPAL - ISS.WEB - Criação de regra para o município de MOGI DAS CRUZES-SP/ CARIACICA-ES - para não gerar notas de prestador do mesmo município - Geração eventual 

 

O cliente está gerando o arquivo e nele estão sendo consideradas notas em que o prestador é do mesmo município de geração do arquivo, o que não é correto. 

 

Na regra de geração normal tem algumas particularidades pra alguns municípios mas em nenhuma delas cita o município de Mogi das Cruzes, e não tem regra geral de como não gerar notas em que o prestador é do mesmo município.

A inclusão desta regra também será necessário para geração eventual do município de Cariacica - ES, que tem o mesmo validador. 

REGRA VALIDADA: https://trten.sharepoint.com/:w:/r/sites/REQ_MTZ/_layouts/15/Doc.aspx?sourcedoc=%7B3D9A789E-8018-4C02-B357-D55525281FCC%7D&file=MTZ-ISS_WEB-Geracao_do_Meio_Magnetico.doc&action=default&mobileredirect=true

## Release Notes

Informações p/a Carta do Patch: DW e TAX ONE 

Ajuste de regras 

Objetivo: Ajustado o processo de geração para os municípios de Mogi das Cruzes e Cariacica, do validador ISS WEB, para que sejam desconsideradas as notas fiscais eletrônicas emitidas por prestadores estabelecidos dentro do município da inscrição eventual.
