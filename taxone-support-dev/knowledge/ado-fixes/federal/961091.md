# [MRV] TKT157221 - TAX ONE - MUNICIPAL - ISS.WEB - FRANCA Avaliar a regra para gerar o campo " Regime Simples" para o Município de Franca - SP

## Descricao

O campo "Regime do Simples" não está sendo preenchido corretamente. No entanto, conforme a necessidade do cliente, esse campo deve receber um dos seguintes valores: 

 

- (N) Não Simples 
- (S) SINAC 
- (M) SIMEI (aplicável apenas para declarações de serviços tomados)  

 

Solicita-se análise e ajuste da regra de negócio para garantir o correto preenchimento do campo conforme as opções acima. 

 

Na regra esse campo é gerado apenas para outros municípios: 

 

 

 

Código PF/PJ: 

 

 

 

 

 

 

 

 

Documento fiscal: 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Meio magnético gerado: 

 

Conforme manual de layout, o campo deveria gerar ''N'', esta sendo gerado como 1.
