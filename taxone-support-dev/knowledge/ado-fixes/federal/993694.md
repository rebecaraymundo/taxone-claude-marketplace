# TKT159323 - MRV - ONESOURCE TAX ONE - MUNICIPAL - BETHA - Avaliar a regra para realizar o DEPARA nas serie utilizada nas notas fiscais.

## Descricao

Avaliar a regra para realizar o DEPARA nas serie utilizada nas notas fiscais. 

 

Módulo Municipal > Parâmetros por Municípios, existe a possibilidade de Fazer um “De/Para” de série, porém para o município Itatiaia não funciona. 

 

A regra de negócio é a seguinte: 

  

Para o campo Documentos – Série, deve-se utilizar o valor do campo SERIE_DOCFIS da tabela DWT_DOCTO_FISCAL. 

 

 

 

Como o cliente MRV tem diversos municípios a séria escriturada no Tax One é padrão, porem cada validador tem a série especifica.
