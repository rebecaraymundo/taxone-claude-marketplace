# [MUNICIPAL]: T1/Automation - POC - Implementação por Validadores Municipais - Situação Especial Eventual/CC/Utilities - Validação Final T1 + Automation

## Descricao

Eu como usuário(a) fiscal das obrigações municipais.   
 1) Gostaria que fosse realizado implementação da automação Geração do Meio Magnético para Situação Especial por validadores municipais no Tax Automation, dando a possibilidade do cliente realizar o geração de determinado município no que compete o validador.  
  Dessa forma simplificaria o processo de implementação de municipal, sendo assim um ganho para o produto aumento o portfólio de prefeituras atendidas. 
 2) Gostaria que fosse ajustado a nomenclatura dos arquivos do meio Magnético e situação especial para cada município gerado via Tax Automation dos Validadores implementados.
 
  “Código Empresa + Código Estabelecimento + Município + MMAAAA”, como exemplo, teríamos Barretos:  
“931 0105 – BARRETOS 102024” 

A exceção ocorre para Curitiba que possui um padrão diferente “PMC_10_2024” (mas não atenderíamos) 
  

3)Gostaria fosse inclusa uma mensagem de LOG Exemplo: Nenhum registro encontrado no cursor C_SERV_RECEB_TOMADOS para os parâmetros informados. Empresa: 076, Estabelecimento: CAMGBE, Período: 01/11/2014 a 30/11/2014.
 

  Nenhum registro encontrado para processamento no período informado. Arquivo Sem Movimento", para os arquivos gerado no período em que não tiveram dados na sua geração   
 
  
4) Validadores a serem contemplados
