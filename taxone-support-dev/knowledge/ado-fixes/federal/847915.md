# [Lucro Presumido]: T1 - Básicos - Report Fiscal - Tela Recuperação de Valores (Tela) - BA - Alinhamento

## Descricao

Eu como usuário(a) fiscal dos Report Fiscal  Gostaria que fosse implementada uma nova tela a fim de realizar a parametrização da recuperação de valores para o novo relatório de Lucro Presumido IR/CS com Sinergia PIS/COFINS Para suprir a necessidade do usuário em delimitar o mapeamento das contas que farão parte do relatório mencionado. 
 Funcionalidade Nova:  Agrupamento: Básicos  Módulo: Report Fiscal Menu:  Relatórios > Lucro Presumido              > Recuperação de Contas 
 Detalhes Adicionais 
- Detalhamento Campos dos campos: 
- Código Empresa: 
- Código Estabelecimento: 
- PRT: Usuário deverá selecionar o parâmetro desejado: 01 - Lucro Presumido IR/CS com Sinergia PIS/COFINS 
- Código Conta: Conforme seleção da tabela X2002 
- Descrição da Conta Contábil: Conforme seleção da tabela X2002 
- Indicador da Natureza: (não traz da 2002) Escolha do usuário, conforme: 
- Ativo 
- Passivo 
- Despesa 
- Receita Patrimônio Liquido  
- Recuperação Valores (permitirá o usuário, no processo de geração do relatório, capturar as informações de saldo - balancete): Escolha do usuário, conforme: 
- Débito 
- Crédito 
- Movimento 
- Saldo  
- Exceção BC: A fim de definir que a conta selecionada não deverá participar do cálculo a base do PIS/COFINS
