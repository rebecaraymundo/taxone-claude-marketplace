# [EFD REINF] DW/T1 - REINF – Criação Índices de tabelas STG

## Descricao

Eu como usuária do modulo EFD REINF. 
https://atendimentotr.zendesk.com/agent/tickets/131617
 
Gostaria que fosse avaliado e aplicado nos produtos do DW e T1, a melhoria feito no cliente Assai/Sendas, onde
foi identificado uma grande demora para execução de uma query pelo integrador
do reinf, fazendo com que os eventos não retornassem ao DW. 
 
Solução: Para resolver esse caso criamos 5
índices que resolveram o problema. 

 

Após
o DBA do cliente criar os índices, no mesmo momento, o integrador (karaf) já
efetuou a busca dos status e retornou para a STG_IN e consequentemente para o
Painel de Eventos do DW. 

  

CREATE INDEX IX_STGEVENTIN_PROT on
STG_EVENTOS_REINF_IN(PROTOCOLO_ONESOURCE); 

CREATE INDEX IX_STGEVENTIN_IDODT on
STG_EVENTOS_REINF_IN(ID_EVENTO_ONESOURCE,DT_CRIACAO); 

CREATE INDEX IDX_REINF_IN_OUT_MAX ON
STG_EVENTOS_REINF_IN (ID_STG_EVENTOS_REINF_OUT, ID_STG_EVENTOS_REINF_IN DESC); 

CREATE INDEX
IDX_REINF_CONTROLE_PROC ON STG_EVENTOS_REINF_CONTROLE
(ID_STG_EVENTOS_REINF_IN, PROCESSANDO, STATUS_PROC, PROCESS_DATE); 

CREATE INDEX
IDX_REINF_OUT_NAFILA_DT ON STG_EVENTOS_REINF_OUT
(ID_STG_EVENTOS_REINF_OUT, TIPO_AMBIENTE, NAFILA, DT_CRIACAO);
