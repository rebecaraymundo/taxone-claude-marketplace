# INFOLEGIS 176/24 - A - ISSQN - RIO NEGRO/PR - Migra Sistema (IPM SISTEMAS) - DW/T1

## Descricao

A Prefeitura do Município de Rio Negro/PR alterou o sistema de gerenciamento do ISSQN, passando a utilizar o Fiscal Web -  Atende Net da empresa IPM Sistemas desde outubro de 2020.
A informação de que atualmente a prefeitura utiliza a Solução da IPM Sistemas foi confirmada por ofício encaminhado direto ao contribuinte e informações disponibilizadas no endereço: https://rionegro.atende.net/subportal/atualizacao-do-sistema-fiscal-web
 O manual de Layout do Arquivo para Importação de Documentos de Serviços está disponível no endereço:
 https://rionegro.atende.net/subportal/atende.php?rot=1&aca=119&ajax=t&processo=viewFile&file=BDC5CE0874F7C3EBF97729E6FE3E5C5B9CC95F3B&sistema=WPO&classe=UploadMidia
