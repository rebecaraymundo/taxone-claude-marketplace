# INFOLEGIS 2207/25 - A - ISSQN - ITIRAPINA/SP - Migra Sistema (SIGISSWEB) - DW/T1

## Descricao

A
Prefeitura de Itirapina/SP alterou o sistema “CONAM” para o Sistema “SIGISSWEB”,
que contempla a entrega de informações ao fisco municipal, via Internet, sobre
os serviços tomados, a partir de 01 de dezembro de 2024. 

O novo sistema está disponível no endereço: 

https://itirapina.sigissweb.com/ 

  

O acesso a documentação para integração está disponível no endereço: https://downloads.sigissweb.com/MaterialParaIntegracao.rar

## Release Notes

Informações p/a Carta do Patch: DW e TAX ONE 

Atualização Legal 

Objetivo:  A Prefeitura de
Itirapina/SP migrou do validador DES XML, para o Sistema validador “SIGISSWEB -
Sistema Integrado de Gerenciamento do ISSQN”, que contempla a entrega de
informações ao fisco municipal, via Internet, sobre os serviços tomados, a
partir de 01 de dezembro de 2024. 

Layout Padrão: TXT

Manutenções Realizadas:

do Cadastro dos Validadores Municipais – (PRT_VALIDADOR_MUNIC) 

2) Inclusão na TFIX85 – Cadastro dos Códigos de Serviços Municipais – (PRT_SERVICO_OBRIG)
