# INFOLEGIS 256/25 - C - ISSQN - TERESINA/PI - Migra Sistema (DSF) - DW/T1

## Descricao

A
Prefeitura de Teresina/PI alterou o sistema “ISSO – ISS Online” para o Sistema “Nova
Nota Fiscal Teresina” da DSF – Desenvolvimento de Sistema Fiscais Ltda., que contempla
a entrega de informações ao fisco municipal, via importação de arquivo .TXT,
sobre os serviços tomados.  

Na
comunicação publicada no site da prefeitura sobre esta transição, não foi
informada a data exata em que a migração foi efetivada. 

O
novo sistema e manuais estão disponíveis no endereço: 

https://notafiscal.teresina.pi.gov.br/notafiscal/paginas/portal/#/  

Cumpre observar que as informações sobre o
layout estão disponíveis apenas mediante autenticação com usuário e senha e foi
disponibilizada pelo cliente no ticket.

## Release Notes

Informações p/a Carta do Patch: DW e TAX ONE 

Migração do Municipio de Teresina/PI 
Objetivo: Informar que a Prefeitura de Teresina/PI publicou instituindo a migração para o novo Sistema Eletrônico de Gestão do ISSQN, incorporando a funcionalidade de geração de arquivo magnético por meio do validador "NOTA JOSEENSE" disponibilizado pela “DSF” Desenvolvimento de Sistema Fiscais Ltda., substituindo o validador "ISS online". 
O novo sistema está disponível no endereço: https://notafiscal.teresina.pi.gov.br/notafiscal/paginas/portal/#/ 

 

Layout Padrão: TXT 

Módulo Impactado 

Módulo: Municipal > Teresina 

Menus: Obrigações 

- Geração do Meio Magnético 
- Arquivo de Entradas de Serviços (Const. Civil / Utilities /Telecom)   
  

Manutenções Realizadas: 

 

 

- Inclusão de dados na TFIX83 - Cadastro das Séries/Modelos das Obrigações – (PRT_SERIE_OBRIG). 
- Inclusão e alteração de dados na TFIX105 – Atualização do Cadastro dos Validadores Municipais – (PRT_VALIDADOR_MUNIC).
