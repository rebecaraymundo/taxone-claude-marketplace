# TKT172916 - TAX ONE - MUNICIPAL - Itatiaia - Betha  "Ajuste no Campo i_motivos_alt_aliquota – Erro na validação de Itatiaia"

## Descricao

Durante a validação do meio magnético do município de Itatiaia, a key user encontrou um erro de estrutura relacionado ao campo i_motivos_alt_aliquota. 

ERRO : Mensagem: O id passado como parâmetro não tem os tipos esperados (MotivosAltAliquotaDao)! 

 

 

Ao analisar o caso, verificamos que o Tax One está gerando esse campo com o valor “000”. 

 

 

Contudo, quando não há informação para esse campo, ele não deve ser preenchido. 

Realizamos um teste editando o arquivo e removendo os “000”. Após essa alteração, o arquivo foi importado com sucesso pela Prefeitura, conforme confirmado pela cliente. 

 

 

Diante disso, solicito o ajuste no comportamento do campo i_motivos_alt_aliquota, para que ele não seja preenchido quando não houver informação válida. 

Pelo o que analisei ele esta respeitando a regra RN40 e deveria respeitar a regra RN40.a 
 

 

 

Arquivos enviados pela cliente: 

- Arquivo com erro 
- Arquivo ajustado e validado com sucesso 
- Evidências em print destacando o campo
