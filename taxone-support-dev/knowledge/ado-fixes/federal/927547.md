# DW/T1 - INFOLEGIS 463/25 - C - ISSQN - AMPARO/SP - Migra Sistema (NF-e CIDADES)

## Descricao

Conforme informações publicadas pela Prefeitura de Amparo/SP e confirmada em contato telefônico, foi confirmado que o sistema de Gerenciamento do ISSQN foi alterado para o sistema “NF-e Cidades” desde 1°/11/2023, sem a publicação de novo ato normativo, disciplinando o novo sistema.
O novo programa contendo o sistema de Declaração Eletrônica de Serviços está disponível no endereço eletrônico: https://www.nfe-cidades.com.br/landing-page 
 Cumpre observar que a informação do layout utilizado deverá ser disponibilizado pelo cliente. As informações sobre manuais e layouts somente são disponibilizadas para o contribuinte através de acesso ao portal com usuário e senha.
