# INFOLEGIS 832/24 - B - ISSQN - APUCARANA/PR - Migra Sistema (IPM SISTEMA) - DW/T1

## Descricao

Conforme informações da Prefeitura de Apucarana/PR, o município passou por uma atualização na versão do sistema Fiscal Web da IPM Sistemas, que é responsável pela Emissão de Notas e Escrituração Fiscal. A atualização para a nova versão ocorreu no dia 22/02/2022. 
 
 O manual de layout do arquivo para importação de documentos de serviço está disponível no endereço: http://www.apucarana.pr.gov.br/site/wp-content/uploads/2022/02/ImportacaoDocumentosFiscais.pdf
 
 Os manuais da obrigação estão disponível no endereço:
 http://www.apucarana.pr.gov.br/site/manual-de-integracao-nfs-e/
