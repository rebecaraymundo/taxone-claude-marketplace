# TKT168532 - MRV - ONESOURCE TAX ONE - MUNICIPAL - ATENDENET - ARAPONGAS - Reavaliação da regra para o campo Código da Situação Tributária para Arapongas 1234

## Descricao

Análise Suporte 

 

Referente ao campo "Código da situação tributária da declaração do serviço do Registro Tipo 20" do Layout da ATENDE NET 

 

A solicitação é para incluir o município de Arapongas na mesma regra especifica que foi criada para Nova Hamburgo, especifica para a geração eventual (“Arquivos de Entrada de Serviços (Const.Civil / Utilities/ Telecom)”. 

 

Atualmente Arapongas está gerando pela regra geral, que não tem o código "1" e alguns outros códigos que pede o manual de Layout. Como a MRV utiliza a geração eventual para este município, a solicitação é para inclusão na mesma regra de Nova Hamburgo.
