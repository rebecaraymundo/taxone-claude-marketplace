# [REINF] - T1/DW - Geração de eventos via JCP - Trava p/ Período Fechado

## Descricao

Eu como usuário(a) fiscal do Módulo EFD_REINF 

 

Gostaria que fosse ajustado a Geração Previa para os
eventos externos de JCP, com uma trava para quando o período estiver fechado,
bem como para Envio de Registros XML de Arquivos Externos - Grandes
Volumes 

 

Funcionalidade Existente:  

Agrupamento: SPED 

Módulo: REINF 

Menu: Geração Prévia > Movimentos 

Menu: Envio de Eventos - Grandes Volumes >
Arquivo de Origem Externa 
 

 

Detalhes Adicionais do cálculo:
 

Essa trava deve funcionar com o mesmo proposito do já é feito para a SAFX53

## Release Notes

Informações p/a Carta do Patch:
DW e TAX ONE 

Melhoria 

Objetivo: Incluir uma validação na geração prévia do JCP (Arquivos
Externos) dos eventos R-4010 (Pagamentos/Créditos a Beneficiário Pessoa Física)
e R-4020 (Pagamentos/Créditos a Beneficiário Pessoa Jurídica), para informar
que o período já está fechado e não prosseguir com o processamento dos eventos. 

Funcionalidade Existente:  

Agrupamento: SPED 

Módulo: REINF 

Menu: Geração Prévia > Movimentos 

- R-4010 - Pagamentos/Créditos a
Beneficiário Pessoa Física - Arquivos Externos 
- R-4020 - Pagamentos/Créditos a
Beneficiário Pessoa Jurídica - Arquivos Externos
