# INFOLEGIS 1005/25 - B - ISSQN - JOAO MONLEVADE/MG - Migra Sistema (E&L)

## Descricao

A
Prefeitura de Monlevade/MG divulgou a migração do sistema “ISS Intel” para o
sistema “E&L”, utilizado para Gerenciamento Eletrônico dos dados econômico-fiscais,
para declaração das operações de serviços tributáveis ou não tributáveis, para
processamento eletrônico de dados de suas declarações mensais, para
recolhimento do imposto devido pelos serviços tomados e ou prestados. 

A
Nota Fiscal Eletrônica será automaticamente gravada na escrituração do
prestador de serviço. 

O responsável tomador dos serviços sujeitos ao imposto, deverá escriturar
mensalmente, pelo sistema tributário, as notas fiscais dos serviços tomados,
bem como os demais documentos fiscais com seus respectivos valores.  

O novo sistema está disponível no endereço: 

https://mg-joaomonlevade-pm-nfs.cloud.el.com.br/#/login 

Cumpre observar que a informação sobre a alteração e layout a ser
utilizado foi disponibilizado pelo cliente.

## Release Notes

Informações p/a Carta do Patch: DW e TAX ONE 

Migração do município João Monlevade/MG

O novo sistema está disponível no endereço: https://mg-joaomonlevade-pm-nfs.cloud.el.com.br/#/login
