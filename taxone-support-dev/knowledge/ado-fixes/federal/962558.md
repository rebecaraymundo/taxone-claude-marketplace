# [MRV] TKT157551 - TAX ONE - MUNICIPAL - ISS ONLINE - ARAÇATUBA Erro no campo Local do Serviço, quando o estabelecimento possui Inscrição Municipal Eventual

## Descricao

Gostaríamos que fosse avaliada a situação do cliente MRV para verificar se há alguma tratativa que permita que o campo seja gerado corretamente. 

 

O estabelecimento está localizado no município de Ribeirão Preto e possui uma Inscrição Municipal Eventual em Araçatuba. Quando o cliente gera o "Arquivo de Entradas de Serviços (Const. 

Civil / Utilities / Telecom)", o documento fiscal é gerado com o Local do Serviço indicado como "F- Fora do município", quando deveria ser "D- Dentro do município" para evitar erros ao submeter o arquivo no validador da prefeitura de Araçatuba. 

 

Como o campo "Município ISS" foi preenchido como Araçatuba, onde o ISS será devido, o cliente acredita que o campo "Local do Serviço" deveria ser gerado como "D". 

 

Ao analisar a regra para preenchimento do campo, observamos que ele é gerado com base no município do estabelecimento, razão pela qual o campo não é gerado conforme o esperado.
