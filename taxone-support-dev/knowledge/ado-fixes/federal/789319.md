# DW/T1 - INFOLEGIS 566/25 - A - ISSQN - PRAIA GRANDE/SP - Migra Sistema (GissOnline)

## Descricao

Saída esta pela demanda QA: https://dev.azure.com/tr-ggo/Mastersaf%20Fiscal%20Solutions/_workitems/edit/789109 
 Consiste na migração do Sistema “ISSNet Online” para o sistema “GissOnline”, utilizado para gestão e controle do ISSQN a partir de 1° de novembro de 2024, na Prefeitura de Praia Grande/SP.
  
 Os contribuintes do ISS, tomadores de serviços, responsáveis tributários e representantes legais que realizam todas as operações relativas ao ISS (tributo municipal) terão que se cadastrar, obrigatoriamente, no portal Giss Online.
 
 O novo sistema está disponível no endereço: https://praiagrande.giss.com.br/#/
 
 Cumpre observar que a informação sobre a alteração e layout a ser utilizado foi disponibilizado pelo cliente.
