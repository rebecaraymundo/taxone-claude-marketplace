# [MUNICIPAL]: T1 - POC - Implementação por Validadores Municipais - Telas de Parâmetros das TFIX83 e TFIX84

## Descricao

Eu como usuário(a) fiscal das obrigações municipais. 

Gostaria que fosse ajustado as nomenclaturas dos menus
das telas de parâmetros do Tipo de Documento e Serie, reorganizasse as telas
agrupando. Também tratar a retiradas as restrições de recuperação das UFs na
abertura das telas de parâmetros. 
   
  
Agrupamento: Municipal 

Módulo: Parâmetro para Município 

Menu: Parâmetros > Modelo > Passa a ser Cadastro dos Tipos de Documento das Obrigações 

- Cadastro Modelo > Alterar nomenclatura para Cadastro Tipo de Documento e remover o submenu Modelo Msaf x Modelo.
   
- Puxar as telas Tipo Docto Msaf x Tipo Docto e Tipo Docto Msaf x Espécie para agrupar no novo menu Cadastro dos Tipos de Documento das Obrigações.  
  

                                                    

Agrupamento: Municipal 

Módulo: Parâmetro para Município 

Menu: Parâmetros >  Série > Passa a se Cadastro das Séries/Modelos das Obrigações 

- Cadastro Série > Manter a nomenclatura do cadastro da série.

   
- Puxar as telas Série Msaf x Série e Serie Msaf x Modelo Msaf x Serie  
 para agrupar no novo menu Cadastro das Séries/Modelos das Obrigações.

Ajustes/Melhorias

Objetivo: Nesta demanda, estamos liberando os ajustes
de nomenclaturas dos menus das telas de manutenção da TFIX83 - Cadastro das
Séries/Modelos das Obrigações e TFIX84 - Cadastros dos Tipos de Documentos das
Obrigações no módulo Municipais / Parâmetros para Município. Reordenamos os
menus, criando os agrupamentos Cadastro dos Tipos de Documento das Obrigações e
Cadastro das Séries/Modelos das Obrigações.  

Também realizado uma melhoria na retirada das restrições de
recuperação das UFs na abertura das telas de parâmetros.
