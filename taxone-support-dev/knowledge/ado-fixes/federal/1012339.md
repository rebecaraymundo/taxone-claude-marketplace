# TKT167491 - MRV - ONESOURCE TAX ONE - MUNICIPAL - ISS WEB CAMACARI - CAMAÇARI - AUSENCIA FLEGUE QUEBRA POR DATA DE EMISSAO

## Descricao

Cliente solicita que na Geração do meio magnético Arquivo de Entradas de Serviços (Const. Civil / Utilities/Telecom, seja incluído a opção de Quebra por data de emissão, assim como é possível realizar quando utilizar a opção Geração Meio magnético. 

 

Flegue disponível na Opção > Meio Magnético: 

 

 

Ausência, quando o arquivo é gerado na Opção Arquivo de Entradas de Serviços (Const. Civil / Utilites / Telecom)
