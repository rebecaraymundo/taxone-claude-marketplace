# INFOLEGIS 2176/25 - D - GISSONLINE - LAYOUT XML - Versão 2.2 - DW/T1

## Descricao

Conforme informações passadas pela empresa Eicon, responsável pela solução Gissonline, foi disponibilizada a versão 2.2 do Manual Guia CST, que é utilizado em todos os municípios que já utilizam a nova versão da GISS.

 Conforme informações do Manual, visando facilitar o registro de serviços comprados (serviços tomados), dando maior autonomia e agilidade nas declarações de serviços, a Gissonline disponibiliza ao contribuinte a opção de importação de arquivos XML de notas fiscais recebidas. Este recurso permite que o contribuinte realize a declaração fiscal eletrônica gerando num mesmo arquivo XML até 50 notas fiscais, e posteriormente realizar o upload dos arquivos gerados quantas vezes for necessário. O arquivo possui um layout próprio, baseado no modelo ABRASF 2.04, trata-se de um arquivo XML com codificação UTF-8.
 
 As datas de disponibilização, vigência e competência para a utilização do layout devem ser consideradas a partir da adesão dos municípios à nova GISS.
