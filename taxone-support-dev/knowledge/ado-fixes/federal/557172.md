# [Internal Bug] T1 (Angular 13) - Municipal - Itatiba - Finteliss - Campos com tamanho insuficientes [Teste 035] / Problema existente

## Passos de Reproducao

URL do ambiente de testes utilizado (Tax One): https://qa.onesourcetax.com/
  
 Credenciais de acesso para reprodução da falha: 
- Tennant / Database: V04 
- Empresa: 076
 
- Estabelecimento: ITATIB - ITATIBA - Estabelecimento de Itatiba 
- Período:  
 Steps:
 
- Acessar o sistema utilizando empresa e estabelecimento citados acima 
- Acessar Municipal > Itatiba 
- No modal exibido, escolher a opção 'FINTELISS' e clicar no botão 'OK' 
- Acessar o menu 'Obrigações' > 'Meio magnético'  
- Preencha os campos de data com o período entre '01/10/2021 e 31/10/2021' 
- Preencha o campo 'Número do Lote' 
- Clique no botão 'Executar' 
- Após fechar a mensagem informativa, acessar a aba 'Processos' 
- Clique no botão 'Listar' 
- Observe a listagem de informações dos processos já executados  

 

Resultado encontrado: as informações dispostas nas colunas 'Início Execução' e 'Fim Execução' possuem fonte maior do que as demais colunas e não exibe integralmente a informação.
 

Resultado esperado: as informações devem ser exibidas de forma integral para que o usuário faça a leitura via tela e também aplique fonte dentro do padrão.
