# TKT161759 - MRV - TAX ONE - MUNICIPAL - SIGIS - Araras e Sumaré - Ajustar o preenchimento dos campos Nome Prefeitura e CNPJ Prefeitura do Registro 0

## Descricao

TKT161759 - MRV - TAX ONE - MUNICIPAL - SIGISS - Araras e Sumaré 

Tratar o erro dos campos "Nome Prefeitura" e "CNPJ Prefeitura" do Registro 0 

Ao gerar o arquivo de ISS para o município de Araras, os campos "Nome Prefeitura" e "CNPJ Prefeitura" do Registro 0 estão sendo preenchidos com base nos "Dados Iniciais", que hoje só podem ser parametrizados uma única vez. 

Isso causa problema na geração eventual, pois essas informações (nome e CNPJ da prefeitura) variam conforme o município para o qual o arquivo está sendo gerado. 

No caso específico deste cliente, os "Dados Iniciais" estão parametrizados para Santa Bárbara d'Oeste, pois ele de fato gera arquivos para esse município. Porém, o cliente também gera arquivos para outros municípios que utilizam o mesmo validador. Assim, por exemplo, na geração especial do arquivo ISS para o município de Araras, o Tax One está levando no arquivo as informações de "Nome Prefeitura" e "CNPJ Prefeitura" referentes à Santa Bárbara d'Oeste, em vez de Araras. 

Será necessário ajustar a regra para que os "Dados Iniciais" possam ser parametrizados por município (e não apenas por estabelecimento), permitindo que "Nome Prefeitura" e "CNPJ Prefeitura" sejam corretamente preenchidos conforme o município do arquivo gerado.
