# [EFD_Contribuições]: DW/T1 - Registro 0120: Identificação de EFD-Contribuições Sem Dados a Escriturar

## Descricao

Eu como usuário(a) fiscal do Módulo EFD_Contribuições. 

Gostaria que fosse ajustado o código na geração
do Registro 0120: Identificação de EFD-Contribuições Sem Dados a
Escriturar. 

Para os fatos geradores ocorridos a partir de 01 de agosto
de 2017, o Registro "0120 - Identificação de EFD_Contribuições Sem Dados a
Escriturar" é de preenchimento obrigatório, quando na escrituração não
constar registros referente a operações geradoras de receitas ou de créditos,
ou seja, a escrituração estiver zerada, sem dados 

Hoje no código não está respeitando a regra de geração, está
gerando para ambos com movimento e sem movimento. 
Causa: O Registro 0120, não deve ser gerado quando tiver movimentação, nos Blocos "A", "C", "D", "F"  ou "P.
  
Funcionalidade
Existente:

Módulo: EFD-Escrituração Fiscal
Digital das Contribuições 

Menu: Manutenção > Registro do
Bloco 0 > Identificação dos períodos dispensados (0120) 

Menu: 
Obrigações > Geração dos Registros
