# INFOLEGIS  1978/25 - B - ISSQN - IBIRITÉ/MG - Migra Sistema (Siap.Net) - DW/T1

## Descricao

Consiste na migração do sistema “E&L” para o sistema “Siap.Net – ISS Online”, utilizado para processamento eletrônico de dados das declarações dos serviços contratados e/ou prestados na Prefeitura Municipal de Ibirité/MG, a partir de 21 de junho de 2022. 
O novo sistema está disponível no endereço:
 https://pmibirite.geosiap.net.br/pmibirite/issonline/iss.login.php
 
 Cumpre observar que a informação sobre a alteração e layout a ser utilizado foi disponibilizado pelo cliente. O acesso ao sistema e manuais é restrito ao contribuinte com usuário e senha. Tentamos contato com a Prefeitura, porém sem sucesso.
