# INFOLEGIS 727 - B - ISSQN - BENTO GONÇALVES/RS - MIGRA SISTEMA (ELOTECH)

## Descricao

A
Prefeitura de Bento Gonçalves/RS alterou o sistema “Atende.net” para o Sistema
“Oxy ISS” da “Elotech”, que contempla o gerenciamento do ISSQN, a partir de 01
de fevereiro de 2025. 

O
programa contendo o sistema de informatização e escrituração eletrônica do ISSQN
está disponível no endereço eletrônico: 

https://bentogoncalves.oxy.elotech.com.br/iss/home 

  

 

Cumpre observar que a informação sobre o layout
a ser utilizado deverá ser disponibilizado pelo cliente. Os manuais
disponibilizados no portal não contêm layout para fins de entrega da DMS.
Tentamos contato com a Prefeitura, porém sem sucesso.
