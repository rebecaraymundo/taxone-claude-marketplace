# [OCI] TAX ONE - Erro ao iniciar o backend no OCI devido a dependência não necessária do SecretsManager

## Descricao

Estamos enfrentando um problema ao tentar iniciar o backend no Oracle Cloud Infrastructure (OCI). A aplicação lança um erro relacionado à dependência do SecretsManagerServiceImpl. O erro ocorre devido à tentativa de criação de um bean para o awsSecretsManager, que falha por falta de uma região especificada.
 
 Detalhes do Erro: Erro Reportado: org.springframework.beans.factory.UnsatisfiedDependencyException
Causa: Falha ao criar o bean secretsManagerServiceImpl devido a uma dependência não satisfeita no secretsManagerClient.
Exceção Aninhada: java.lang.IllegalArgumentException: region must not be blank or empty 
 Contexto:
Repositório: taxsami_artifacts
 Classe Afetada: SecretsManagerServiceImpl
 Arquivo de Configuração: connection.properties
 Configurações:
  auth.db.type=db  cloud.type=localstack 
 Solução Proposta:
É necessário modificar a implementação para que a dependência da classe SecretsManagerServiceImpl seja carregada apenas quando for realmente necessário trabalhar com o SecretsManager. Isso pode ser feito através de uma configuração condicional ou de um perfil específico que somente ativa essa dependência em ambientes apropriados.
