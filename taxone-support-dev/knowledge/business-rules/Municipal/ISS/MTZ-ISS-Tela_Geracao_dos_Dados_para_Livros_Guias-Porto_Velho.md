#### Cód\.

### Descrição

### DR

RN01

__Estab\. Centralizador:__

Este campo exibe o estabelecimento referente ao município parametrizado para a geração\.

OS4140

RN02

__Tipo do Livro:__

Este campo exibe o tipo de livro que foi parametrizado para o estabelecimento\. 

OS4140

RN03

__Formato Geral:__

Deverá estar sempre selecionado como  default, a opção “Sintético”\.

OS4140

RN04

__Formato Específico:__

Este campo deverá estar marcado como  default, com o município que foi parametrizado como formato específico na tela de Manutenção de Municípios ISSQN\. Neste caso deverá ter efetuado a Apuração do ISSQN antes\.

OS4140

__Informações Sobre o recolhimento do ISSQN Retido__

RN05

__ISSQN Retido:__

Neste campo deve\-se informar o valor de ISSQN Retido\. Esse valor é informado manualmente pelo usuário\.

OS4140

RN06

__Data do Recolhimento:__

Este campo é referente a Data de Recolhimento, que é inserida manualmente pelo usuário\.

OS4140

RN07

__Banco/Código:__

Este campo é referente ao número do Banco, que é inserido manualmente pelo usuário\.

OS4140

RN08

__DAM:__

Este campo é referente ao número do documento, que é inserido manualmente pelo usuário\.

OS4140

RN09

__Estabelecimentos:__

Este campo é referente aos estabelecimentos que estão relacionados ao município que foi parametrizado para a Geração\.

OS4140

RN10

__Período da Apuração:__

Referente a data inicial e data final da apuração do livro\.

OS4140
