# Regras Gerais

**Origem das informações para geração:**

Este relatório deve ser gerado de acordo com as regras já previstas, para o relatório no formato padrão.
Porem as informações devem ser geradas em formato TXT, com separação dos campos com pipe “|”, para ser gravado e incluido em excel.
