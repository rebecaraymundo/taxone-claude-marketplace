__Coluna Empresa / Estabelecimento: __

Originalmente esta coluna exibia apenas o estabelecimento, mas foi alterada na __OS4538__ para mostrar também o código da empresa,__ __com objetivo de facilitar a identificação da origem dos estabelecimentos, quando o código da programação em questão for uma programação multiempresa\.

Esta coluna é preenchida com o código da empresa \+ código do estabelecimento \+ razão social do estabelecimento\.
