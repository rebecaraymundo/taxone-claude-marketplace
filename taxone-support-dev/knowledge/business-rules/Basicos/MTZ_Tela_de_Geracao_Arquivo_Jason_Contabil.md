## Introdução


O principal objetivo da tela de Geração de Arquivo JSON é consolidar e gravas os dados contábeis em formato JSON. Isso facilita a integração com o sistema OBI para a requisição do sistema SAP, utilizando APIs dos impostos ICMS, ICMS-ST, ICMS-Antecipado, Difal, FCP, PIS/COFINS e IPI.

## Log


Se não houver nenhum lançamento para Ajuste da Apuração, Outros Lançamentos, Lançamentos Complementares, Estorno ou Contabilização automática:
Não gerar arquivo.
Informar ao usuário: "Nenhum lançamento elegível para estorno no período de apuração informado."
Se faltar parametrização contábil para algum caso específico:
Registrar no log e não gerar o lançamento correspondente.
Informar resumo ao usuário (ex.: "5 lançamentos não gerados por falta de parametrização contábil.").
