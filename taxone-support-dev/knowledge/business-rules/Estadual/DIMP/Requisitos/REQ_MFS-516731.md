### <a id="_Toc128559555"></a>Relatório de Movimentação DIMP

__Localização__: Menu Estadual, módulo DIMP, menu Relatórios à Relatório de Movimentação DIMP 

 

__Descrição__

Incluir os campos abaixo no relatório:

- Nat\. Jurídica Origem Operação

ind\_nat\_jur 

- Ind\. Tipo PIX

ind\_tp\_pix

- ID Pedido

id\_pedido 

- Não Gerar Registros 1100/1500

ind\_nao\_gera\_reg

- Motivo p/ Não Gerar Registros 1100/1500

      DSC\_MOTIVO\_NAO\_GERA\_REG

- Data Liquidação

DT\_LIQUIDACAO

Esses campos devem ser incluídos após o campo “ID Transação Complementar”\.
