# ESPECIFICOS PIN SINAL: Solicitação De Informacao

## Descricao do Padrao
Clientes reportam problemas relacionados a: Geração Pin Sinal; PIN SINAL; PIN Sinal substituído pelo SIMNAC 

## Causa Raiz
Solicitação De Informacao — classificacao baseada em 4 tickets resolvidos.

## Resolucao / Orientacao
Resolucoes aplicadas nos tickets de referencia:
Cintia, boa tarde
&nbsp;
Para geração do PIN SINAL, devem ser carregados os dados da&nbsp;SAFX502&nbsp;(Veículo de Transporte).
&nbsp;
Deve observar também as SAFX, os dados da aba&nbsp;Informação de Transporte são carregados na&nbsp;SAFX50&nbsp;e os dados da aba&nbsp;Veículo de Transporte são carregados na&nbsp;SAFX502
&nbsp;
- Para a geração do Pin Sinal é necessário associar um registro de Informação de transporte a cada documento fiscal.
O campo “via de transporte” não é de preenchimento...
